## 第1部分：对照《民法典》人格权编提取备考提纲


### 一、人格权基本原理（民法典第989-1001条）

| 要点 | 法典依据 | 核心规则 |
| :--- | :--- | :--- |
| 人格权不得放弃、转让、继承 | 第992条 | 强制性规定 |
| 姓名/名称/肖像的许可使用 | 第993条 | 明星代言、加盟 |
| 人格权请求权不适用诉讼时效 | 第995条 | 停止侵害、排除妨害、消除危险、消除影响、恢复名誉、赔礼道歉 |
| **人格权侵害禁令（预先保护）** | **第997条** | **新增，从未考过** |
| 侵权责任裁量因素 | 第998条 | 行为目的、方式、后果、影响范围等 |
| 合理使用（新闻报道/舆论监督） | 第999条 | 不承担侵权责任 |
| 拒不承担责任的执行方式 | 第1000条 | 公告/公布法律文书，费用由行为人负担 |


### 二、一般人格权（民法典第109条）

| 类型 | 内容 |
| :--- | :--- |
| **人身自由权** | 非法拘禁（合理时间不构成侵权） |
| **人格尊严权** | 兜底作用；可依据习惯处理（祭奠权案） |


### 三、物质性人格权（民法典第1002-1004条）

| 权利 | 侵权形态 | 关键判断 |
| :--- | :--- | :--- |
| **生命权** | 死亡（自杀不构成侵权；协助自杀构成侵权） | 看人死没死 + 怎么死的 |
| **身体权** | 完整性、行动自由 | 外部器官受损；假肢看是否发挥人体组织功能 |
| **健康权** | 正常运转和发挥 | 内部器官/机能受损 |


### 四、精神性人格权

#### （一）姓名权与名称权（民法典第1012-1017条）

| 权利 | 主体 | 侵权形态 |
| :--- | :--- | :--- |
| **姓名权** | 自然人 | 干涉、**盗用**（重点）、假冒 |
| **名称权** | 法人/非法人组织 | 商号侵权 |

**客体扩大解释**：笔名、艺名、网名等（具有一定知名度）。

#### （二）肖像权（民法典第1018-1022条）

| 要点 | 规则 |
| :--- | :--- |
| **客体** | 可识别的外部形象（不限于面部） |
| **内容** | 制作、使用、公开、许可他人使用 |
| **侵权形态** | 丑化、污损、伪造、制作、使用、公开 |
| **许可使用解释规则**（**新增**） | 有利于**肖像权人**的解释（第1021条） |
| **许可使用任意解除权**（**新增**） | 无约定期限 → 随时解除（第1022条） |

**肖像权 vs 著作权**：行使发表、复制、发行、出租、展览等权利须经肖像权人同意。

**合理使用**：科研、新闻报道、职务行为、风景旅游场所拍照、寻人启事等。

#### （三）名誉权（民法典第1024-1028条）

| 要点 | 规则 |
| :--- | :--- |
| **主体** | 所有民事主体（自然人可主张精神损害赔偿；法人不能） |
| **客体** | 品德、声望、才能、信用等社会评价（信用≠独立权利） |
| **侵权形态** | 侮辱、**诽谤**（捏造散布虚假事实，造成社会评价降低） |
| **侵权须指向特定人** | 不特定群体不能告 |

#### （四）荣誉权（民法典第1031条）

| 要点 | 规则 |
| :--- | :--- |
| **主体** | 所有民事主体 |
| **客体** | 光荣称号、荣誉称号 |
| **侵权形态** | 非法剥夺、诋毁、贬损 |
| **英雄烈士保护** | 结合死者人格利益保护 |

#### （五）隐私权（民法典第1032-1033条）

| 要点 | 规则 |
| :--- | :--- |
| **主体** | **自然人独有** |
| **客体** | 私生活安宁、私密空间、私密活动、私密信息 |
| **侵权形态** | 刺探、侵扰、泄露、公开（重点是**侵扰私生活安宁**） |


### 五、个人信息保护（民法典第1034-1039条）

| 要点 | 规则 |
| :--- | :--- |
| **核心判断标准** | **可识别性** |
| **个人信息 vs 隐私** | 个人信息是大圈，隐私是小圈（姓名是个人信息但不是隐私） |
| **侵权形态** | **过度处理**（收集、存储、使用、加工、传输、提供、公开） |
| **免责情形** | 自行公开、维护公共利益 |

### 第一段：人格权基本原理概述

好，接下来咱们来看第五部分，人格权编。这一部分在考试当中往往在客观题中考察，主观题基本上不涉及，在客观题中也就一道选择题，要么是单选题一分，要么是多选题两分，结合侵权责任来考察。

把书翻到三百七十五页，来看专题四十，人格权。

先来看人格权的基本原理。第一个问题，人格权**不能放弃、不能转让、也不能继承**，特别是不得放弃（民法典第992条）。

第二，**许可使用**（民法典第993条）。虽然人格权往往不直接体现财产内容，但有些权利（如姓名、肖像）可以许可他人使用——这就是**明星代言**。名称权（商事主体的权利）也可以许可使用——就是实践中的**加盟**。

第三，权利受到侵害后的问题。人格权（姓名权、肖像权、名誉权、隐私权等）受到侵害后，侵权责任的承担有其特点，特别是**消除影响、恢复名誉、赔礼道歉**。物权、财产权受到侵害不能要求赔礼道歉，但人格权受到侵害往往可以。并且消除影响、恢复名誉、赔礼道歉，以及停止侵害、排除妨害、消除危险，这些**不适用诉讼时效期间的限制**（民法典第995条），原因在于侵权行为一直在持续。

第四，**预先保护（人格权侵害禁令）** （民法典第997条，**新增，从未考过**）：民事主体有证据证明行为人正在实施或者即将实施侵害其人格权的违法行为，不及时制止将使其合法权益受到难以弥补的损害的，有权依法向人民法院申请采取责令行为人停止有关行为的措施。

> **预先保护例题**：老王发现李佳在写小说《梦想和中央电视台女主持人的故事》，说准备三个月后发到网上。老王不想火。
>
> **问**：老王现在可以怎么办？
>
> **答**：可以向法院申请**人格权侵害禁令**，要求李佳停止写作。

第五，**侵权责任承担的裁量**（民法典第998条）：承担多大责任，要看侵权行为的形态、范围等。比如同样发一条微博，你的微博关注者5个，老孟的微博关注者100多万，侵权的社会影响范围完全不同。

第六，**合理使用**（民法典第999条）：为公共利益实施新闻报道、舆论监督等行为的，可以合理使用民事主体的姓名、名称、肖像、个人信息等——不用承担侵权责任。

> **合理使用例题**：疫情期间，老孟拍到马峰不戴口罩随地吐痰，发到网上。马峰能否追究侵权责任？
>
> **答**：**不可以**。这是舆论监督，属于合理使用（高铁霸座案同理）。

第七，**责任承担**（民法典第1000条）：行为人因侵害人格权承担消除影响、恢复名誉、赔礼道歉等民事责任，拒不承担的，法院可以采取在报刊、媒体上公告或公布生效法律文书的方式执行，产生的费用由行为人负担。


### 第二段：一般人格权

一般人格权包括两种：**人身自由权**和**人格尊严权**（民法典第109条）。

**人身自由权**：主要就是非法拘禁。超市怀疑顾客偷东西，报警后解决纠纷耗时30分钟——这是**合理时间**，不构成侵权。但如果把顾客拉到保安室扣了三天三夜，则构成侵权（民法上侵权，刑法上犯罪）。

**人格尊严权**（民法典第109条，兜底作用）：法律没有特别规定时，依据**习惯**处理（前提：不违背公序良俗）。

> **人格尊严权例题（祭奠权案）** ：兄弟二人不和，父亲去世，老大知道但不告诉老二，擅自办完丧事。老二得知后把老大告到法院。
>
> **问**：法院如何处理？
>
> **答**：民法典虽未规定“祭奠权”，但祭奠是对逝去先人的怀念，是风俗习惯，不违背公序良俗，法院可依据习惯判决老大赔偿老二（侵犯人格尊严权）。

**注意**：不能随便用一般人格权。如妻子擅自终止妊娠，不能说是侵犯丈夫的人格尊严权（这是生育权的范畴）。


### 第三段：具体人格权——物质性人格权

具体人格权分为两类：**物质性人格权**（以人的肉体或身体为载体，自然人独有）和**精神性人格权**。

物质性人格权包括三项：**生命权、身体权、健康权**（民法典第1002-1004条）。

#### （一）生命权

**侵权形态**：看人死没死。没死肯定不侵犯生命权。但人死了也不一定侵犯生命权——要看怎么死的。

- **受害人自杀**：行为人不构成侵权
- **协助自杀**：构成侵权，甚至构成犯罪（如安乐死）

> **生命权例题**：楼下群众起哄，怂恿跳楼者“你跳不跳，我等着吃饭呢”，跳楼者受刺激跳楼身亡。
>
> **问**：起哄群众是否承担侵权责任？
>
> **答**：**不承担**（受害人自杀，行为人不构成侵权），但可能承担行政处罚责任。

#### （二）身体权

**侵权形态**：**完整性**和**行动自由**（民法典第1003条）。

**假肢的判断**：看是否发挥人体组织的功能。
- 正在人体中发挥组织功能（不可拆卸）→ 侵犯**身体权**
- 可以自由拆卸放在一旁 → 侵犯**财产权**

> **假肢例题**：老孟装了假腿（不可拆卸），打架时对方一棍子打掉。侵犯了什么权？
>
> **答**：**身体权**（装配物已构成身体不可分离的一部分）。
>
> 如果老孟晚上把假牙拆下来泡在水杯里，你把水杯抢走扔掉——侵犯**财产权**。

**典型侵权形态**：偷剪他人头发指甲、医生误摘病人器官、手术留下疤痕等。

#### （三）健康权

**侵权形态**：身体机能不能**正常运转和发挥**（民法典第1004条）。

> **健康权例题**：一拳把耳朵**打聋了**（耳朵还在但听不见）→ 侵犯**健康权**；一拳把耳垂**咬掉了**（外部器官缺失）→ 侵犯**身体权**。
>
> **身体权 vs 健康权**：
> - 外部器官受损→身体权
> - 内部器官/机能受损→健康权
> - 摘除肾脏→同时侵犯身体权（完整性破坏）+健康权（机能不能正常运转）

**掰手腕与自担风险**（民法典第1176条）：自甘风险行为，只要没有故意或重大过失，不侵权。


### 第四段：精神性人格权（一）——姓名权与名称权

**姓名权**（自然人所独有，民法典第1012条）和**名称权**（法人、非法人组织所有，民法典第1013条）。

**客体扩大解释**（民法典第1017条）：具有一定知名度的笔名、艺名、网名等也受保护（“你们抓的是鲁迅，关我周树人何干”）。

**侵权形态**（民法典第1014条）：**干涉、盗用、假冒**。重点是**盗用**。

- **盗用**：未经允许，为谋取不正当利益擅自使用他人姓名——**仅用姓名，未用身份**（如培训机构用老师名字招生，实际老师不知情）
- **假冒**：既用姓名又用身份（冒名上大学、冒名领取稿酬）

**不构成侵权**：披露真实姓名、曾用名（名字起出来就是为了被叫的）。


### 第五段：精神性人格权（二）——肖像权

肖像权是自然人所独有（民法典第1018条）。

**肖像的客体**（民法典第1018条第2款）：**可以被识别的外部形象**，不限于面部。有可识别性即构成肖像。

> **肖像识别性例题**：把老孟照片打马赛克，但露出欧米茄手表。法考圈都知道拿欧米茄手表的是孟老师——**仍可识别**，侵犯肖像权。
>
> **乔丹案**：剪影无法体现乔丹个人特征，不具备可识别性——**不构成侵害肖像权**。

**肖像权的内容**（民法典第1018条第1款）：制作、使用、公开、许可他人使用（明星代言）。

**肖像权许可使用的两个新增考点（从未考过）** （民法典第1021条、第1022条）：

1. **解释规则**（第1021条）：对肖像许可使用合同中肖像使用条款的理解有争议的，应当作出**有利于肖像权人**的解释。
2. **任意解除权**（第1022条）：对肖像许可使用期限**没有约定或约定不明确的**，任何一方当事人可以**随时解除**合同，但应当在**合理期限之前**通知对方。

**侵权形态**（民法典第1019条第1款）：**丑化、污损、伪造、制作、使用、公开**六种。

- **丑化**：葛优躺案（用葛优瘫坐形象做宣传）
- **污损**：结婚现场前男友烫坏照片眼睛嘴唇
- **伪造**：AI换脸技术
- **制作**：制作老师表情包
- **使用**：用他人照片作微信头像
- **公开**：未经同意公开合影

**肖像权与著作权的冲突**（民法典第1019条第2款）：肖像作品权利人（摄影师）行使**发表、复制、发行、出租、展览**等权利时，必须经肖像权人同意（与公开有关的权利都要经同意）。

**合理使用（不构成侵权）** （民法典第1020条）：科研、新闻报道、职务行为（公安机关抓逃犯）、风景旅游场所拍照、寻人启事等。


### 第六段：精神性人格权（三）——名誉权、荣誉权、隐私权

#### （一）名誉权（民法典第1024条）

所有民事主体（自然人、法人、非法人组织）都有名誉权。区别：自然人可主张**精神损害赔偿**；法人/非法人组织不能。

**客体**：名誉——品德、声望、才能、信用等社会评价。信用是名誉的一部分，没有单独的“信用权”。

**侵权形态**：**侮辱、诽谤**。重点是**诽谤**：捏造散布虚假事实，造成他人社会评价降低。

**侵权须指向特定人**：笼统地说“某机构老师讲课是狗屎”——不能告；如果指名道姓说“孟老师讲课是狗屎”——可以告。

**文学作品与新闻报道**（民法典第1027条、第1028条）：以真人真事或特定人为描述对象的文学作品，不能含有侮辱诽谤内容。新闻媒体需严格审核，网络媒体侵权后应及时更正删除。


#### （二）荣誉权（民法典第1031条）

所有民事主体都有荣誉权（光荣称号、荣誉称号）。

**侵权形态**：**非法剥夺、诋毁、贬损**。

**英雄烈士保护**：结合死者人格利益保护（狼牙山五壮士、邱少云、董存瑞、雷锋等）。


#### （三）隐私权（民法典第1032条）

自然人独有。

**客体**：私生活安宁 + 私密空间、私密活动、私密信息。

**侵权形态**（民法典第1033条）：**刺探、侵扰、泄露、公开**。重点是**刺探**和**侵扰**，特别是**侵扰私生活安宁**。

> **侵扰隐私权例题**：培训机构学员信息被泄露，陌生同学深夜发五万条“我爱你”短信。
>
> **问**：是否侵权？
>
> **答**：**构成**。发垃圾短信侵犯私生活安宁（隐私权）。

**典型侵权**：宾馆安装摄像头、家门口摄像头拍邻居玄关、私家侦探跟踪、人肉搜索等。


### 第七段：个人信息保护

个人信息（民法典第1034条）：以电子或其他方式记录的，与已识别或可识别的自然人有关的各种信息。

**个人信息 vs 隐私**：个人信息是大圈，隐私是小圈。姓名是个人信息但不是隐私。

**核心判断标准**：**可识别性**（是否能够识别到具体个人）。

**侵权形态**：**过度处理**（民法典第1035条）——收集、存储、使用、加工、传输、提供、公开个人信息，不得过度。

> **个人信息保护例题**：天鹅公司将售房过程中收集的购房者姓名、身份证号码、电话、家庭住址等信息打包销售。
>
> **问**：是否侵权？
>
> **答**：**构成侵权**（侵害个人信息权益）。身份证号码具有可识别性。甚至可能构成侵犯公民个人信息罪。

**免责情形**：自行公开、为维护公共利益（舆论监督）。

**信息处理者的义务**：不得泄露篡改；信息有误时权利人可以要求更正（民法典第1037条）。


### 第八段：本专题总结

翻回三百七十五页，人格权的体系：

| 类型 | 具体权利 | 主体 | 侵权形态/要点 |
| :--- | :--- | :--- | :--- |
| **一般人格权** | 人身自由权、人格尊严权 | 自然人 | 兜底作用 |
| **物质性人格权** | 生命权 | 自然人 | 死亡 |
| | 身体权 | 自然人 | 完整性、行动自由 |
| | 健康权 | 自然人 | 正常运转和发挥 |
| **精神性人格权** | 姓名权 | 自然人 | 干涉、**盗用**、假冒 |
| | 名称权 | 法人/非法人组织 | 商事主体商号 |
| | 肖像权 | 自然人 | 丑化、污损、伪造、制作、使用、公开 |
| | 名誉权 | 所有民事主体 | 侮辱、**诽谤** |
| | 荣誉权 | 所有民事主体 | 非法剥夺、诋毁、贬损 |
| | 隐私权 | 自然人 | 刺探、侵扰、泄露、公开 |
| **个人信息** | 个人信息权益 | 自然人 | 可识别性、不得过度处理 |
