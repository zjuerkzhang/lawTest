## 第1部分：对照《民法典》合同权利义务终止提取备考提纲


### 一、清偿——以物抵债（合同编通则司法解释第27条、第28条）

| 要点 | 规则 |
| :--- | :--- |
| **性质** | **诺成性合同** |
| **新旧债关系** | **新旧两债并存**（只有履行新债，旧债才消灭） |
| **选择权** | 债权人催告后享有选择权（可选择要求履行新债或旧债） |
| **诉讼中的以物抵债** | 法院出具**确认文书**，**不发生物权变动效力** |
| **无权处分** | 以物抵债协议有效，相对人可能**善意取得** |
| **以物抵债 vs 流押条款** | 履行期限**届满后**→有效；履行期限**届满前**+“归你所有”→无效（流押条款） |


### 二、清偿抵充（民法典第560条、第561条）

**数项债务的清偿顺序**（第560条）：
1. **约定抵充**（意思自治）
2. **指定抵充**（债务人指定，保护债务人）
3. **法定抵充**（保护债权人）：①已到期债务 → ②无担保/担保最少的债务 → ③负担较重的债务（利息高的） → ④到期在先的 → ⑤按比例

**费用利息本金的抵充顺序**（第561条）：**费→利→本**（保护债权人）


### 三、合同解除（民法典第562-566条）

#### （一）解除的法定情形（第563条）

1. 因不可抗力致使不能实现合同目的
2. 履行期限届满前，当事人一方明确表示或以行为表明不履行主要债务（逾期违约）
3. 当事人一方迟延履行主要债务，经催告后在合理期限内仍未履行
4. 当事人一方迟延履行债务或有其他违约行为致使不能实现合同目的（根本违约）
5. 法律规定的其他情形

#### （二）解除权的行使（第565条）

- **通知解除**：合同自**通知到达对方时**解除
- **诉讼/仲裁解除**：起诉状副本/仲裁申请书副本**送达对方时**解除
- 对方有异议的，任何一方可请求法院或仲裁机构**确认解除行为的效力**（确认之诉）

#### （三）解除的除斥期间（第564条）

- 有法律规定或当事人约定→依规定/约定
- 无规定/约定→自知道或应当知道解除事由之日起**1年**，或经对方催告后在**合理期限**内不行使的，权利消灭

#### （四）解除的效力（第566条）

- **尚未履行的**——终止履行
- **已经履行的**——可请求恢复原状或采取其他补救措施，并有权请求赔偿损失（**违约责任**）
- 合同因违约解除的，**不影响违约责任的承担**
- **不影响担保责任**


### 四、抵销（民法典第568条、第569条；合同编通则司法解释第57条）

**法定抵销要件**：
1. 互负债务
2. 标的物种类、品质相同
3. 均已到期

**不得抵销的情形**（通则解释第57条，新增）：
- 因侵害自然人人身权益，或者故意、重大过失侵害他人财产权益产生的损害赔偿债务，**侵权人**主张抵销的，**不予支持**（防止暴力讨债）
- 已过诉讼时效之债 → **不一定**（看债务人是否提诉讼时效抗辩权）


### 五、提存（民法典第570-574条）

**适用情形**（第570条）：债权人无正当理由拒绝受领 / 债权人下落不明 / 债权人死亡未确定继承人等。

**法律效果**（第571-574条）：
- 提存成立 → **视为债务人已履行债务**
- 提存物所有权、风险、孳息归**债权人**
- 提存部门负妥善保管义务
- 提存物领取权：自提存之日起**5年**内不行使，归国家所有（但**债权人未履行对债务人的到期债务**的除外）


### 六、免除（民法典第575条）

- 有相对人的意思表示，必须向债务人作出
- 原则：免除通知到达债务人，**债消灭**
- 例外：债务人在合理期限内**明确拒绝**的，**债不消灭**


### 七、混同（民法典第576条）

原则：债权债务同归一人 → **债消灭**。例外：**损害第三人利益的除外**（如权利质权中的债权不消灭）。

### 第一段：合同权利义务终止概述

好，接下来咱们来看专题十八，叫做合同的权利义务终止，又叫债的权利义务终止。这个专题是客观题和主观题都要考到的。在旁边拿笔写上**主观题**。

合同权利义务终止的方式一共六种：**清偿、解除、抵销、提存、免除、混同**。其中前三种是考试的重点，后三种了解即可。


### 第二段：清偿——代物清偿与以物抵债（主观题★★★★★）

先来看第一个，叫做清偿。清偿其实说的就是**履行**。履行是一个行为，清偿是一个结果。比如欠一万块钱，一年期满后把钱给了你——这就是清偿。

考试重点在书的二百三十四页，叫做**代物清偿、以物抵债**和**清偿抵充**。代物清偿、以物抵债旁边写上**主观题**。

#### （一）以物抵债协议的概念

以物抵债的“债”要进行扩大解释——可能是约定之债、法定之债、自然之债（包括已过诉讼时效之债）。“物”就是七大标的物——房、地、车、设备、股权、债权、票据——只要有交换价值的东西都可以来以物抵债。

以物抵债是个协议（财产性合同），受民法典合同编限制。根据以物抵债协议是在**履行期限届满前**签订的还是**履行期限届满后**签订的，分两类：
- **履行期限届满后的清偿型以物抵债协议**——目的就是为了还债（本专题讲授）
- **履行期限届满前的担保型以物抵债协议**——本质是担保（非典型担保，放到借款合同部分讲）

#### （二）清偿型以物抵债协议的六个问题

| 问题 | 规则 |
| :--- | :--- |
| **1. 性质** | **诺成性合同**（不是实践性合同） |
| **2. 效力** | 原则**合法有效** |
| **3. 新旧债关系** | 原则**新旧两债并存**——只有履行了新债，旧债才消灭 |
| **4. 选择权归属** | **债权人享有催告后的选择权**（债权人催告后债务人不履行，债权人可选择要物或要钱） |
| **5. 诉讼中达成的以物抵债协议** | 法院出具的法律文书是**确认文书**，**不发生物权变动效力**（未考过） |
| **6. 无权处分** | 拿他人之物抵债——以物抵债协议合法有效，受让人可能**善意取得**（未考过） |

> **以物抵债例题（历年真题）** ：王某向丁某借款100万元后无力清偿，遂提出以自己所有的一幅古画抵债，双方约定第二天交付。
>
> **分析**：旧债=100万借款合同（金钱之债）；新债=以物抵债协议（古画）。新债是手段，旧债消灭是目的。
>
> **问**：下列说法正确的是？
>
> A. 双方约定以古画抵债，原借款合同失效，王某只能以交付古画履行债务——**错**（新旧两债并存，尚未履行新债，旧债未消灭）
>
> B. 双方交付古画的行为属于履行借款合同义务——**正确**（交付古画后，100万借款债消灭）
>
> C. 王某有权在交付古画前反悔，提出继续以现金偿付——**错**（选择权在**债权人丁某**，不在债务人王某）
>
> D. 古画交付后如果被鉴定为赝品，王某应承担瑕疵担保责任——**正确**（违约责任）
>
> **答案：B、D**

#### （三）以物抵债 vs 流押条款的区别

**看时间**：履行期限**届满后**签订 → 清偿型以物抵债，**有效**（债务人诚信表现）；履行期限**届满前**签订 + “直接归你所有” → **流押条款无效**（剥夺标的物市场定价权，过度担保）。


### 第三段：清偿抵充（主观题已考三次★★★★★）

**清偿抵充**：同一债务人欠同一债权人数笔同类债务，债务人部分清偿，所还的到底是哪一笔债？

**三步规则**（民法典第560条）：

1. **约定抵充**（有约从约）——意思自治
2. **指定抵充**（保护债务人利益）——债务人在清偿时指定抵充哪一笔（转账备注）
3. **法定抵充**（保护债权人利益）——三步走：
   - 第一步：先抵**已到期**的债务（未到期的债权视为无债权）
   - 第二步：再抵**无担保**或担保最少的债务（普通债权优先保护）
   - 第三步：最后抵**负担较重**的债务（利息高的优先）

> **清偿抵充例题**：胡某于2006年3月10日向李某借款100万，期限3年（2009年3月10日到期）；2009年3月30日再借100万，期限3年（2012年3月30日到期）。两笔借款均先后由王某保证。胡某于2010年2月归还借款100万，未作指定。
>
> **问**：这100万还的是哪一笔？
>
> **答**：**第一笔**（2006年借款）。第一笔已到期，第二笔未到期——法定抵充第一步直接解决。

**费利本规则**（民法典第561条）：清偿不足以清偿全部债务时，**费利本**顺序——先抵**费用**，再抵**利息**，最后抵**本金**（保护债权人利益）。

> **费利本例题**：孟某向马某借款100万，年息10%，到期后拒绝归还，马某起诉支付各项费用2万（总额112万）。强制执行中孟某清偿50万，双方无特别约定。
>
> **问**：这50万如何抵充？剩余债务多少？
>
> **答**：先抵费用2万，再抵利息10万，最后抵本金38万（50-2-10=38）。未偿还本金62万，继续计算年息10%的利息。


### 第四段：解除权——概述与五个核心考点

第二个每年绝对必考事项——**解除权**。

解除权是**例外**，不得成为常态。基于鼓励交易原则和合同严守原则，法律要限制当事人解除合同。

**五个核心考点**（民法典第562-566条）：

| 考点 | 规则 |
| :--- | :--- |
| **1. 对象法定** | **尚未履行或尚未履行完毕且合法有效**的合同才可以被解除（无效合同无解除权；已履行完毕无解除权） |
| **2. 程度法定** | 必须达到**根本违约**的程度（合同目的无法实现）。轻微瑕疵不准解除合同。 |
| **3. 情形法定** | 双方解除（协议解除）或单方解除（约定解除权、法定解除权） |
| **4. 行权法定** | 行权必须**明示**，不能用行为抗议。**或诉或裁或通知**——通知为主，诉讼仲裁为辅。 |
| **5. 效力法定** | 合同解除**三个不影响**：不影响合同效力、不影响违约责任、不影响担保责任。**解除权+违约责任** |


### 第五段：合同解除的两种方式

#### （一）双方解除（协议解除/合意解除）

事后双方当事人又达成一个新的协议来解除原来的合同。完全是意思自治，尊重当事人的新选择。

#### （二）单方解除

**约定解除权**（民法典第562条第2款）：合同成立时**事前**约定解除事由，事由发生时享有解除权的一方可以解除合同。

**法定解除权**（民法典第563条，每年必考★★★★★）：

四种情形：

| 情形 | 示例 | 解除权主体 | 能否追究违约责任 |
| :--- | :--- | :--- | :--- |
| **不可抗力** | 地震、洪水 | **双方**均可 | ❌ 不能（不可抗力是法定免责事由） |
| **逾期违约** | 迟迟未开工，交房无望 | **非违约方**单方 | ✅ 可以 |
| **迟延履行+催告** | 迟延履行主要债务，经催告后合理期限内仍未履行 | **非违约方**单方 | ✅ 可以 |
| **从给付义务违反（根本违约）** | 合同目的无法实现（如承诺的养老设施没有） | **非违约方**单方 | ✅ 可以 |


### 第六段：法定解除权——从给付义务违反的典型例题

> **例题（经典真题）** ：甲房产开发公司在交给购房人张某的小区平面图和项目说明书中都标明有一个健身馆。张某看重小区健身方便决定购买一套商品房，并与甲公司签订了购房合同。张某收房时发现小区**没有健身房**。
>
> **问**：下列说法正确的是？
>
> A. 甲公司不守诚信构成根本违约，张某有权退房——**正确**（解除权）
>
> B. 甲公司构成欺诈，张某有权请求甲公司承担违约责任——**正确**（撤销权→缔约过失责任）
>
> C. 甲公司恶意误导，张某有权请求甲公司双倍返还购房款——**错**（没有惩罚性赔偿）
>
> D. 张某在退房和要求甲公司承担违约责任之间只能选择一种——**错**（解除权可以同时退房+追究违约责任）
>
> **答案：A、B**


### 第七段：解除权 vs 撤销权的区别

| 对比项 | 撤销权 | 解除权 |
| :--- | :--- | :--- |
| **行权方式** | **或诉或裁**（不能通知） | **或诉或裁或通知**（通知为主） |
| **效果** | **向前发生效力**（有**溯及力**）→ 自始无效 | **向后发生效力**（**无溯及力**）→ 向未来终止 |
| **责任** | **撤销权 + 缔约过失责任** | **解除权 + 违约责任** |
| **对象** | 效力有瑕疵的合同（可撤销） | **合法有效**的合同 |

> **理解关键**：撤销权有溯及力→合同自始不存在→只能追究缔约过失责任。解除权无溯及力→合同一直有效→可追究违约责任。


### 第八段：解除权的行使与效力

**行权方式**（民法典第565条）：
- **通知解除**：合同自**通知到达对方时**解除
- **诉讼/仲裁解除**：起诉状副本或仲裁申请书副本**送达对方时**解除（本质还是通知）
- 对方不同意解除的，任何一方均可请求法院或仲裁机构**确认解除行为的效力**（**确认之诉**）

**解除权的除斥期间**（民法典第564条）：法律规定或当事人约定 → 按法律规定或约定；无规定/约定的，自知道或应当知道解除事由之日起**1年**内行使。

**解除的效力**（民法典第566条）：
- 尚未履行的——**终止履行**
- 已经履行的——根据履行情况和合同性质，可以请求**恢复原状**或采取其他补救措施
- **并有权请求赔偿损失**（违约责任）
- 合同因违约解除的，**不影响违约责任的承担**
- 担保人**继续承担担保责任**


### 第九段：抵销——法定抵销与约定抵销

第三个重点——**抵销**。互负债务的双方当事人将两项债务互相抵冲，提高偿债效率。

**法定抵销的积极要件**（民法典第568条）：
1. 互负债务
2. 标的物种类、品质**相同**（一般都是金钱之债）
3. **均已到期**（未到期的债权视为无债权）

**法定抵销的消极要件（核心考点）** ——民法典合同编通则司法解释第57条（新增，从未考过★★★★★）：

| 债务类型 | 能否抵销 | 理由 |
| :--- | :--- | :--- |
| **侵权之债——侵权人主张抵销** | ❌ **不能** | 防止暴力讨债（切鼻子、切手指、抄家） |
| **侵权之债——受害人主张抵销** | ✅ **可以** | — |
| **已过诉讼时效之债** | **不一定** | 取决于债务人是否提诉讼时效抗辩权 |


### 第十段：提存——了解

提存：因债权人的原因导致债务人履行困难，债务人可将标的物交付提存部门（公证部门），从而完成债务清偿。

**适用情形**（民法典第570条）：
1. 债权人无正当理由拒绝受领（最常见——标的物降价了）
2. 债权人下落不明
3. 债权人死亡未确定继承人等

**不适宜提存的标的物**：鲜活易腐烂物品 → 可**变卖后提存价款**。

**提存的法律效果**（民法典第571-574条）：
- 提存成立，**视为债务人已履行债务**
- 提存后应及时通知债权人
- 提存物所有权、风险、孳息均归**债权人**

**提存物的领取权（新增，未考过）** ：自提存之日起**5年**内不行使，归国家所有。**例外**：债权人未履行对债务人的到期债务（没给钱的，不能归国家）。


### 第十一段：免除——民法典第575条

**免除**：债权人抛弃债权的行为。**有相对人的意思表示**——必须向债务人作出。

**修正的单方行为说**（民法典第575条）：
- **原则**：免除通知到达债务人，**债即消灭**（无需债务人同意）
- **例外**：债务人在合理期限内**明确拒绝**的，**债不消灭**

> **例题**：李某向王某借款，某日王某对赵某表示：“李某生活困难，这钱他不用还了。”
>
> **问**：下列说法正确的是？
>
> A. 王某的意思表示向赵某作出时生效——**错**（必须向债务人作出）
>
> B. 王某免除债务的意思表示需李某受领——**正确**
>
> C. 李某无须向王某偿还债务——**不一定**（李某可行使拒绝权）
>
> D. 王某的意思表示不产生债务免除的效果——**不一定**
>
> **答案：B**


### 第十二段：混同——民法典第576条

**混同**：债权债务同归一人。原则：债**消灭**。例外：**损害第三人利益的除外**（如权利质权中，债权消灭会损害质权人利益，则债权不消灭）。


### 第十三段：本专题总结

| 制度 | 核心规则 | 法典依据 |
| :--- | :--- | :--- |
| **清偿——以物抵债** | 新旧两债并存；债权人有催告后选择权；诉讼中达成的以物抵债协议是确认文书 | 合同编通则司法解释第27、28条 |
| **清偿——清偿抵充** | 约定抵充→指定抵充→法定抵充（到期→无担保→利息高）；费利本 | 第560、561条 |
| **解除权** | 对象法定+程度法定+情形法定+行权法定+效力法定；解除权+违约责任 | 第562-566条 |
| **抵销** | 互债同类到期；侵权之债的侵权人不得抵销；自然之债抵销不确定 | 第568、569条；通则解释第57条 |
| **提存** | 因债权人原因履行困难；5年领取权；例外（未付款不能归国家） | 第570-574条 |
| **免除** | 有相对人的意思表示；债务人有权拒绝（拒绝则债不消灭） | 第575条 |
| **混同** | 原则消灭；损害第三人利益的除外 | 第576条 |
