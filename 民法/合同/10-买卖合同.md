## 第1-1部分：对照《民法典》买卖合同提取备考提纲


### 一、买卖合同的性质

| 分类维度 | 性质 |
| :--- | :--- |
| 单务/双务 | **双务合同**（双方互负对待给付义务） |
| 有名/无名 | **有名合同**（典型合同） |
| 有偿/无偿 | **有偿合同**（需给付代价） |
| 诺成/实践 | **诺成合同**（意思表示合意即成立） |
| 要式/不要式 | **不要式合同**（不需要特定形式，商品房买卖合同除外） |


### 二、无权处分的买卖合同（民法典第597条）

- 无权处分**不影响**买卖合同效力（物债两分区分原则）
- 因出卖人未取得所有权或者处分权致使标的物所有权不能转移的，买受人可以解除合同并请求出卖人承担**违约责任**


### 三、风险负担（民法典第604-611条）

| 规则 | 法典依据 | 内容 |
| :--- | :--- | :--- |
| **交付主义（原则）** | 第604条 | 交付前出卖人承担，交付后买受人承担 |
| **违约在先（根本违约）** | 第610条 | 买受人拒绝接受或解除合同的，风险由出卖人承担 |
| **在途货物买卖** | 第606条 | **合同成立时**风险转移给买受人（出卖人明知毁损灭失的除外） |
| **代办托运（未约定交付地点）** | 第607条 | **货交第一承运人时**风险转移给买受人 |

**两个不影响**（第609条、第611条）：
- 风险负担**不影响**从给付义务的违约责任
- 风险负担**不影响**出卖人的违约责任


### 四、孳息归属（民法典第630条）

- **交付前**归**出卖人**
- **交付后**归**买受人**


## 第1-2部分：对照《民法典》三类特殊买卖合同提取备考提纲


### 一、试用买卖合同（民法典第637-640条）

| 要点 | 规则 |
| :--- | :--- |
| **本质** | **附生效条件的买卖合同** |
| **试用期** | 有约从约；无约由**出卖人**确定 |
| **买受人的选择权** | **形成权**（单方意思表示即可产生效果） |
| **视为购买** | ①付款；②出卖/出租/设定担保；③试用期届满**不作表示** |
| **使用费** | **无约免费试用** |
| **风险负担** | 试用期内由**出卖人**承担（交付只是占有，不是履行交付义务） |


### 二、动产所有权保留买卖合同（民法典第641-643条）

| 要点 | 规则 |
| :--- | :--- |
| **适用对象** | **动产**（不动产不适用，因涉及过户税收） |
| **所有权转移** | 买受人**付清全款**时才转移 |
| **取回权** | 出卖人享有（买受人未付款、经催告仍不支付，且付款<75%） |
| **限制取回权** | 买受人已支付价款达到总价款**75%**以上 |
| **回赎权** | 买受人享有（取回后消除事由可回赎） |
| **登记对抗** | 未经登记，**不得对抗善意第三人**（第641条第2款） |


### 三、分期付款买卖合同（民法典第634条）

| 要点 | 规则 |
| :--- | :--- |
| **定义** | 总价款**至少分三次**支付 |
| **行权前提** | 未支付到期价款达到全部价款**五分之一**（20%），经催告仍未支付 |
| **出卖人权利** | ①**加速到期权**（请求支付全部价款）；②**解除权**（解除合同，追究违约责任） |


### 四、一物数卖（买卖合同司法解释）

| 标的物类型 | 排序规则 |
| :--- | :--- |
| **不动产** | **登记**优先 |
| **特殊动产**（车、船、飞机） | **交付 > 登记 > 合同成立在先** |
| **普通动产** | **交付 > 先付款 > 合同成立在先** |

### 第一段：典型合同概述

好，接下来我们来看第二分编，叫做典型合同。咱们的民法典当中规定了十九类典型合同，加上原来的旅游合同的司法解释，接下来我们会看到二十类的合同。但二十类的合同不是每一个都考的。

我把它分成了三类：
- **第一类（必考）** ：五个——**买卖、赠与、借款、租赁、保证**
- **第二类（常考）** ：四个——**融资租赁合同、保理合同、建设工程施工合同、委托合同**
- **第三类**：剩下的合同类型，大家只要抓住其中的重点就可以

首先翻过来先来看专题二十，买卖合同。在它旁边写上**主观题**。买卖合同肯定是每年必考类型的合同。


### 第二段：买卖合同的性质

先来看买卖合同的基本原理。它有两个问题，第一个就是买卖合同的性质。

我们为什么要学它？因为后边的这些合同，谁跟买卖合同的性质不一样，谁就是考点。买卖合同是一个**参照物**。

**买卖合同的性质**：**双务、有名、有偿、诺成、不要式合同**。

- **双务**：双方互负对待给付义务
- **有名**：典型合同，法律明文规定
- **有偿**：需要给付代价
- **诺成**：一诺千金，意思表示合意合同即成立
- **不要式**：不需要特定形式（不需要非得书面）。除非法律特别规定，如商品房买卖合同必须有书面形式【补充：城市房地产管理法第41条】。

> **注意**：人家问你“合同的性质”，说的是它是双务/单务、有名/无名、诺成/实践、要式/不要式。人家问你“效力如何”的时候才说有效/无效/待定/撤销。


### 第三段：无权处分的买卖合同

**无权处分**是指以自己的名义处分他人或共有的财产。基于**物债两分区分原则**，买卖合同是买卖合同，所有权变动是所有权变动。**一句话：无权处分不影响合同效力，仅影响合同履行**（民法典第597条）。

> **原因**：买房需要四步动作（合同、交钱、交房、过户）——前三步债权效力，第四步物权效力。签买卖合同不会自动发生所有权变动。
>
> **效果**：
> - 保护**买受人**：如果买受人最终不能取得标的物所有权，可以追究无权处分人的**违约责任**
> - 保护**标的物所有权人**：如果所有权人不同意，就不发生物权变动，合同无法履行


### 第四段：风险负担——交付主义（核心考点★★★★★）

**风险负担**：在合同订立后，标的物因**不可归责于任何一方的事由**（不可抗力）而发生的意外毁损灭失的损失由何方负担。不可抗力一定是风险（洪水、海啸、地震、泥石流、雷击、失火）。

**原则：交付主义**（民法典第604条）——**交付之前由出卖人承担，交付之后由买受人承担**。

> **买房例题**：买房四步动作（合同、交钱、交房、过户）。交房前一天失火房子烧毁——风险由**出卖人**（开发商）承担（退本金及利息）；交房后两天失火——风险由**买受人**承担（购房款照付）。

**例外一：违约在先**（民法典第610条）：因标的物不符合质量要求，致使不能实现合同目的（**根本违约**），买受人拒绝接受标的物或者解除合同的，标的物毁损灭失的风险由**出卖人**承担。

> **例题**：开发商交房时房屋主体结构质量不合格，买受人拒绝签收。两天后发生地震房子没了——风险由**出卖人**（开发商）承担。

**例外二：在途货物买卖**（民法典第606条）：标的物已在运输途中，出卖人寻找买受人出卖在途货物。**风险自合同成立时起由买受人承担**。

> **注意**：如果出卖人在订立合同时**已经知道**标的物已经毁损灭失却仍然出卖的，风险由**出卖人**承担（非善意，无需保护）。

**例外三：代办托运**（民法典第607条）：买受人委托出卖人代办托运。未约定交付地点的，**货交第一承运人时风险转移**给买受人。出卖人没有义务负责运输。

> **综合例题（在途货物+代办托运）** ：青山公司与绿水公司约定，青山公司将2000台电脑交付顺丰公司承运，运输至广东方向，具体地点由绿水公司另行通知（**代办托运**，未约定交付地点）。3月3日青山公司将电脑交付顺丰公司（**货交第一承运人，风险转移给绿水公司**）。3月6日绿水公司将电脑出卖给蓝天公司，并指示顺丰公司将货运至蓝天公司仓库（**在途货物买卖**，3月6日合同成立时**风险转移给蓝天公司**）。
>
> **问一**：3月4日因雷击200台电脑灭失，风险由谁承担？
>
> **答**：**绿水公司**（3月3日已货交第一承运人，风险已转移）。
>
> **问二**：3月8日因雷击200台电脑灭失，风险由谁承担？
>
> **答**：**蓝天公司**（3月6日在途货物买卖合同成立，风险已转移）。
>
> **问三**：如3月5日电脑已因泥石流全部毁灭，绿水公司知情，却仍于3月6日将货物出卖给蓝天公司，风险由谁承担？
>
> **答**：**绿水公司**（出卖人订立合同时已知标的物毁损灭失，风险由出卖人承担）。


### 第五段：两个不影响

**第一，风险负担不影响从给付义务**（民法典第609条）：出卖人未交付有关标的物的单证和资料（从给付义务），**不影响**标的物毁损灭失的风险的转移。主给付义务（交付标的物）是风险转移的标志，从给付义务（发票、说明书、合格证等）违约了，追究违约责任就行——**一码归一码**。

> **例题**：甲将货物交付给乙（主给付义务已履行），但未交付产品合格证和原产地证明文件（从给付义务）。当晚山洪爆发货物全部毁灭。
>
> **问一**：风险由谁承担？
>
> **答**：**乙**（交付主义，已交付）。
>
> **问二**：乙是否还要支付剩余货款？
>
> **答**：**要**（风险由乙承担，货款照付）。
>
> **问三**：甲是否构成违约？
>
> **答**：**构成**（未交付单证资料，从给付义务违约），应承担违约责任。
>
> **答案：A、B正确；C（解除合同）错（不构成根本违约）；D（补交货物）错（风险由乙承担，无需补交）**

**第二，风险负担不影响违约责任**：标的物毁损灭失的风险由买受人承担，**不影响**因出卖人履行债务不符合约定，买受人请求其承担违约责任的权利。


### 第六段：孳息归属

**孳息归属规则**（民法典第630条）：**交付之前归出卖人，交付之后归买受人**。与风险负担规则一致——**交付主义**。

> **孳息例题**：老孟养鸡，将两万只鸡卖给乙，约定明天上午10点交鸡。今天晚上鸡下了两万颗蛋——归**老孟**（交付前）；第二天10点交付后，当天晚上又下了两万颗蛋——归**乙**（交付后）。

> **经典例题（海螺珍珠案）** ：苏某在海鲜市场张某处购买了一只大海螺（**买卖合同**），后交给海河大饭店加工（餐饮服务合同/委托加工）。厨师剥开发现海螺里有一颗大珍珠。
>
> **问**：珍珠归谁所有？
>
> **答**：**苏某**。张某与苏某之间的买卖合同，孳息（珍珠）归买受人苏某（已交付）；苏某与饭店之间是餐饮服务合同/委托加工，不发生物权变动。


### 第七段：本部分总结

**买卖合同风险负担的核心规则**：

| 情形 | 风险转移时间 | 法典依据 |
| :--- | :--- | :--- |
| **原则（交付主义）** | **交付时** | 第604条 |
| **违约在先（根本违约）** | **出卖人承担** | 第610条 |
| **在途货物买卖** | **合同成立时** | 第606条 |
| **代办托运（未约定交付地点）** | **货交第一承运人时** | 第607条 |

### 第一段：三类特殊买卖合同概述

好，接下来咱们来看三类特殊买卖合同，分别是**试用买卖合同**、**动产所有权保留买卖合同**和**分期付款买卖合同**。【修正：原文“适用买卖合同”应为“试用买卖合同”，民法典用的是“试用买卖”（第637-640条），下同】


### 第二段：试用买卖合同

试用买卖合同，它在法考以来考得比较少，原因在于它所有的考点都已经考过了。试用买卖是干什么用的？就是**新产品推广**（体验式营销）。商家允许老百姓去体验，体验得好就买，不好就不买。

**本质**：**附生效条件的买卖合同**（民法典第637条）——生效条件就是买受人的认可。

#### （一）试用期（民法典第637条）

三步确定：
1. **有约从约**
2. **无约补充**
3. 补充不能，由**出卖人确定**（为什么？如果由买受人确定，买受人可以试用一辈子，对出卖人不公平）

#### （二）买受人的选择权——形成权

买受人对是否购买享有选择权，单方意思表示即可产生法律效果。

**意思表示方式**（民法典第638条）：
- **明示**：直接表示购买
- **默示——积极行为**：付款；或将标的物**出卖、出租、设立担保物权**（“非试用行为”）【修正：原文“非适用行为”】
- **默示——消极行为**：试用期限届满，买受人**对是否购买未作表示**的，**视为购买**

#### （三）使用费（民法典第639条）

**有约从约，无约免费试用**——试用是体验式营销，不能收费。

#### （四）风险负担（民法典第640条）

试用买卖中，标的物在试用期内毁损灭失的风险由**出卖人**承担。因为试用期内交付只是**占有**，不是履行买卖合同主要义务的交付。

> **试用买卖模型（国美电器做菜机器人案）** ：国美电器为推广新型做菜机器人，与孟某签订试用买卖合同，将机器人带回家使用15天。
>
> **问一**：买卖合同是否成立？
>
> **答**：**成立**（附生效条件的买卖合同，条件成就前成立但未生效）。
>
> **问二**：试用期如何确定？
>
> **答**：有约从约，无约由**出卖人**确定。
>
> **问三**：第8天孟某将机器人**出卖**给邻居王某，可否推定为购买？
>
> **答**：**可以**（出卖是非试用行为，视为同意购买）。此时不是无权处分——孟某占有标的物，出卖行为触发**简易交付**，瞬间从出卖人处取得所有权，再卖给王某是**有权处分**，王某**继受取得**，与善意取得无关。
>
> **问四**：第10天孟某决定不买，将机器人归还，国美电器可否请求支付使用费？
>
> **答**：**不能**（无约免费试用）。
>
> **问五**：第10天孟某所在地遭遇暴雨，机器人毁损，风险由谁承担？
>
> **答**：**出卖人**（国美电器）。试用期内交付只是占有，不是履行交付义务。


### 第三段：动产所有权保留买卖合同（非典型担保之首）

**非典型担保合同**一共四个：动产所有权保留买卖合同、让与担保合同、动产融资租赁合同、有追索权的保理合同。这是第一个。

**为什么存在所有权保留？** 买受人付款能力较弱，出卖人又想卖。双方约定：标的物可以先交付，但在买受人**付清全款前**，标的物所有权仍归**出卖人**——法律允许这种特别约定（民法典第641条）。

**为什么只适用于动产？** 不动产买卖需要过户登记（收税），如果约定付清房款所有权就转移，就等于逃税，违反税法强制性规定——约定无效。

#### （一）核心考点：一个数字、一个权利、一个对抗力

| 核心要素 | 内容 |
| :--- | :--- |
| **一个数字** | 总价款的**75%** |
| **一个权利** | **取回权**（出卖人享有） |
| **一个对抗力** | **未经登记，不得对抗善意第三人**（民法典第641条第2款） |

#### （二）取回权（民法典第642条）

**适用情形**：
1. 买受人未按约定支付价款，经**催告**后在合理期限内仍未支付（**法定催告义务**）
2. 买受人未完成约定条件
3. 买受人将标的物出卖、出质或作出不当处分（第三人未善意取得的）

**限制**：买受人已支付价款达到总价款**75%**以上的，出卖人**不得行使取回权**（民法典第642条第2款）。

#### （三）回赎权（民法典第643条）

出卖人行使取回权后，买受人在双方约定或出卖人指定的合理回赎期限内，消除出卖人取回标的物的事由的，可以请求**回赎**标的物。

未回赎的，出卖人可以以合理价格出卖标的物，所得价款扣除未支付的价款及必要费用后，**多退少补**。

> **动产所有权保留模型（孟某卖轿车案）** ：孟某将轿车以50万元卖给马某，约定马某每月付2万，**付清全款前孟某保留轿车所有权**。双方未办理所有权保留登记。孟某3月3日交付轿车（交付不发生物权变动）。
>
> **问一**：3月3日轿车所有权归谁？
>
> **答**：**孟某**（特别约定，付清全款才转移所有权）。
>
> **问二**：马某对轿车的占有是有权占有还是无权占有？
>
> **答**：**有权占有**（存在合法有效的买卖合同）。
>
> **问三**：马某支付20万后不再付款（50万×75%=37.5万，20万<37.5万），经孟某催告后仍不付款，孟某有何权利？
>
> **答**：**取回权**。出卖人履行了法定催告义务后，买受人仍未履行，出卖人有权行使取回权。
>
> **问四**：孟某行使取回权后，马某又及时补交拖欠的购车款，马某有何权利？
>
> **答**：**回赎权**（从未考过，重点关注）。
>
> **问五**：马某支付20万后不再付款，一怒之下将轿车卖给曹某，曹某善意取得。孟某能否对曹某行使取回权？
>
> **答**：**不能**。未经登记不得对抗善意第三人，曹某善意取得后，孟某只能追究马某的违约责任。
>
> **问六**：马某支付40万后不再付款（40万>37.5万），孟某能否行使取回权？
>
> **答**：**不能**。已支付价款达到75%以上，取回权被限制。但所有权仍未转移（需付清全款）。

#### （四）所有权保留 vs 超级动产抵押权

| 对比项 | 所有权保留（非典型担保） | 超级动产抵押权（典型担保） |
| :--- | :--- | :--- |
| **性质** | 具有担保功能的**合同** | **抵押权**（典型担保物权） |
| **交付后果** | **不转移所有权**（付清全款才转移） | **转移所有权**（先卖后押） |
| **担保功能** | **取回权**（强力保护出卖人） | **优先受偿权**（千年老二） |
| **登记效果** | 未经登记不得对抗善意第三人 | 十日内登记有超级优先权 |


### 第四段：分期付款买卖合同

**分期付款买卖合同**：买受人将应付总价款在一定期限内**至少分三次**向出卖人支付的合同（民法典第634条）。

**目的**：促进消费，让付款能力较弱的买受人能提前消费。

#### （一）核心考点：一个数字、两个权利

| 核心要素 | 内容 |
| :--- | :--- |
| **一个数字** | 总价款的**80%** |
| **权利一** | **变更权**（加速到期权）——要求买受人一次性付清剩余价款 |
| **权利二** | **解除权**——解除合同，追究违约责任 |

**行权前提**：买受人未支付到期价款的金额达到全部价款的**五分之一**（即20%），经**催告**后在合理期限内仍未支付的（民法典第634条）。

#### （二）两线三区（所有权保留+分期付款结合时）

当买卖合同中**同时约定**所有权保留和分期付款时：
- **75%线**（所有权保留）：超过此线 → 取回权被限制
- **80%线**（分期付款）：超过此线 → 加速到期权和解除权被限制

| 支付比例 | 可用权利 |
| :--- | :--- |
| **<75%** | 取回权 + 加速到期权 + 解除权（**三选一**） |
| **75%-80%** | 加速到期权 + 解除权（取回权被限制，**二选一**） |
| **>80%** | 只能追究**违约责任**（特殊权利均被限制） |

> **分期付款+所有权保留模型（众森公司与徐工集团案）** ：众森公司向徐工集团采购50台铲车，价款100万，**分期付款**支付，同时约定**付清全款前铲车所有权归徐工集团**。双方未办理登记。
>
> **情形一**：众森公司支付**60万**后不再付款（<75%<80%）。徐工集团有何权利？
>
> **答**：**三选一**——①行使取回权；②行使变更权（加速到期，要求一次性付清剩余40万）；③行使解除权（解除合同，追究违约责任）。
>
> **情形二**：众森公司支付**78万**后不再付款（>75%但<80%）。徐工集团有何权利？
>
> **答**：**二选一**——①行使变更权（加速到期）；②行使解除权。取回权被限制（已超75%）。
>
> **情形三**：众森公司支付**95万**后不再付款（>80%）。徐工集团有何权利？
>
> **答**：只能追究**违约责任**（请求继续支付剩余5万）。
>
> **情形四**：众森公司将其中一台铲车卖给了**不知情**的海马公司并交付，徐工集团未登记。徐工集团能否向海马公司行使取回权？
>
> **答**：**不能**。未经登记不得对抗善意第三人，海马公司可能**善意取得**。


### 第五段：一物数卖（多重买卖）——排序

#### （一）不动产一物数卖

**核心规则**：看**登记**——办理过户登记的买受人取得所有权（民法典第209条）。其他买受人只能追究出卖人的**违约责任**。

> **一房数卖例题**：孟某将房屋卖给马某（合同，未交房未过户）；又将房屋卖给曹某（合同，曹某入住但未过户）；再将房屋卖给徐某（合同，**当即办理过户**）。
>
> **问一**：曹某知悉孟某已将房屋卖给马某，能否主张合同无效？
>
> **答**：**不能**。知悉≠恶意串通≠无效。孟某卖房时仍是所有权人，**有权处分**。
>
> **问二**：马某能否请求继续履行过户？
>
> **答**：**不能**（法律上不能履行——特定之债标的物已发生物权变动）。马某只能追究孟某的**违约责任**。
>
> **问三**：房屋所有权归谁？
>
> **答**：**徐某**（办理了过户登记）。

#### （二）动产一物数卖

**排序规则**（买卖合同司法解释第9条、第10条）：

| 动产类型 | 排序规则 |
| :--- | :--- |
| **特殊动产**（机动车、船舶、航空器） | ①**交付** > ②**登记** > ③**合同成立在先** |
| **普通动产** | ①**交付** > ②**先付款** > ③**合同成立在先** |

> **动产一物数卖排序例题**：甲出售一台挖掘机（**普通动产**），分别与乙、丙、丁、戊签订买卖合同：
> - 3月1日与乙签合同（胁迫，可撤销，但撤销前有效），约定货到付款（**仅合同**）
> - 4月1日与丙签合同，丙支付**20%货款**（**先付款**）
> - 5月1日与丁签合同，丁支付**全部货款**（**也付款，但丙在先**）
> - 6月1日与戊签合同，甲将挖掘机**交付**给戊（**交付**）
>
> **排序**：**戊（交付）> 丙（先付款）> 丁（后付款）> 乙（仅合同）**


### 第六段：买卖合同总结

翻回二百六十页，买卖合同每年必考：

| 制度 | 核心规则 |
| :--- | :--- |
| **无权处分的买卖合同** | 不影响合同效力，仅影响合同履行（物债两分） |
| **风险负担** | **交付主义**（第604条）；在途货物=合同成立时；代办托运=货交第一承运人时 |
| **两个不影响** | 风险负担不影响从给付义务，不影响违约责任 |
| **孳息归属** | **交付主义**（第630条） |
| **试用买卖合同** | 附生效条件的合同；无约免费试用；出卖人承担风险 |
| **动产所有权保留** | **75%** + **取回权** + **登记对抗** |
| **分期付款买卖** | **80%** + **加速到期权** + **解除权** |
| **一物数卖** | 不动产看**登记**；动产看**交付 > 登记/付款 > 合同成立在先** |
