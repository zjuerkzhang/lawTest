## 第1部分：对照《民法典》运输、技术、保管、仓储合同提取备考提纲


### 一、运输合同（民法典第十九章，第809-842条）

#### （一）强制缔约（第810条）

从事公共运输的承运人不得拒绝旅客、托运人通常、合理的运输要求。

#### （二）客运合同——承运人的责任

| 损害类型 | 归责原则 | 法典依据 | 免责事由 |
| :--- | :--- | :--- | :--- |
| **人身损害** | **无过错责任** | 第823条 | ①旅客自身健康原因；②旅客故意或重大过失 |
| **自带物品** | **过错责任** | 第824条 | 承运人无过错则免责 |
| **托运物品** | **无过错责任** | 第824条 | 不可抗力等 |

#### （三）货运合同——承运人责任（第832条）

货物毁损、灭失的免责事由：①不可抗力；②货物本身自然性质或合理损耗；③托运人、收货人的过错。

#### （四）承运人留置权（第836条）

托运人或收货人不支付运费、保管费等费用的，承运人对相应运输货物享有**留置权**（可约定排除）。


### 二、技术合同（民法典第二十章，第843-887条）

#### （一）无效情形

1. **非法垄断技术**（限制创新、搭售）→ 无效（第850条）
2. **侵害他人技术成果** → 无效（第850条）

#### （二）技术开发合同

| 类型 | 专利申请权归属 | 法典依据 |
| :--- | :--- | :--- |
| **委托开发** | 归**研究开发人**（受托人）；委托人可免费实施+优先受让权 | 第859条 |
| **合作开发** | 归**共有**；**一票否决权**（一方不同意则不得申请） | 第860条 |

#### （三）技术秘密成果的使用、转让、许可（第861条）

- 有约从约
- 无约：**均有**使用权和转让权
- 可以**普通许可**他人使用（无需同意）
- **独占或排他许可**须征得对方同意或追认，否则无效


### 三、保管合同（民法典第二十一章，第888-903条）

| 要点 | 规则 | 法典依据 |
| :--- | :--- | :--- |
| **认定** | 购物、就餐、住宿时将物品存放在指定场所 → **视为保管** | 第888条 |
| **性质** | 可有偿可无偿；**实践性合同** | 第890条 |
| **寄存人** | 可**随时领取**保管物（任意解除权） | 第899条 |
| **保管人留置权** | 享有留置权（可约定排除） | 第903条 |
| **无偿保管人责任** | 无故意或重大过失 → 不赔偿 | 第897条 |


### 四、仓储合同（民法典第二十二章，第904-918条）

| 要点 | 规则 |
| :--- | :--- |
| **性质** | 商事合同；**一定有偿**；**诺成性合同** |
| **与保管合同区别** | 保管：民事/可有偿可无偿/实践性；仓储：商事/一定有偿/诺成性 |

### 第一段：运输合同概述

好，接下来咱们来看专题三十，运输合同。这个合同是偶尔才会考察到的，所以咱们抓住它的重点就行了。就在三百四十一页运输合同的体系当中的四个绿色的点，把它抓住就行了。

先来了解一下什么叫运输合同。运输合同是承运人将旅客或货物从启运地点运输到约定地点，旅客、托运人或收货人支付票款或运输费用的合同（民法典第809条）。运输合同包括两种：**客运合同**和**货运合同**。我们重点考察对象是**客运合同**。

**强制缔约理论**（民法典第810条）：从事公共运输的承运人，不得拒绝旅客、托运人通常、合理的运输要求。出租车司机不能拒载——理论上如此，实际大家都懂。【补充：与供用电合同原理相同，都有提供公共服务的职能】


### 第二段：客运合同——承运人的救助义务

客运合同的两个考点：

**第一，承运人的救助义务**（民法典第822条）：承运人在运输过程中，应当尽力救助患有急病、分娩、遇险的旅客。

比如坐火车时广播响起：“五号车厢有人倒地，请医护人员前来救助。”这就是承运人的义务。履行好了没事，没履行就要承担责任。


### 第三段：客运合同——承运人的责任与免责事由（核心考点）

**人身损害赔偿**（民法典第823条）：承运人承担**无过错责任**（不考虑有没有过错，原则上都要赔）。

**两个免责事由**：
1. **旅客自身健康原因**造成的（如心脏不好坚持坐飞机，起飞时死亡——航空公司免责）
2. **旅客故意或重大过失**造成的（如失恋后吃安眠药死在火车上——铁路公司免责）

**财产损害赔偿**（民法典第824条）：

| 财产类型 | 归责原则 | 说明 |
| :--- | :--- | :--- |
| **自带物品**（随身携带） | **过错责任** | 需看承运人有没有过错 |
| **托运物品** | **无过错责任** | 托运物品不在旅客控制范围内，承运人承担无过错责任 |

**免票、持优待票或经许可搭载的无票旅客**：不因免票或打折而免责（民法典第823条第2款）。

> **客运合同例题（唐某小宝案）** ：唐某带领三周岁小宝乘坐客运班车，给小宝办理了免票手续。乘车途中，客运班车与蒋某驾驶的轿车相撞发生交通事故。唐某轻伤（医药费2000）且手机摔坏（修理费5000），小宝脑震荡（医药费20万）。
>
> **分析**：
> - **唐某人身损害（2000）**：承运人承担**无过错责任** → 必须赔（除非有免责事由，本案没有）
> - **唐某手机损失（5000）**：承运人承担**过错责任** → 需看司机有无过错
> - **小宝人身损害（20万）**：承运人承担**无过错责任** → 必须赔（不因免票而免责）
> - **连带责任**：唐某不能请求客运公司和蒋某承担连带责任（不构成共同侵权）
>
> **答案**：A（若班车司机能证明对事故没有过错，对于唐某的手机损失可以免责——正确）；B（小宝有权请求客运公司承担赔偿责任——正确）。D项（连带责任）错误。


### 第四段：货运合同

货运合同就一个重点——**承运人的留置权**（民法典第836条）：托运人或者收货人不支付运费、保管费以及其他运输费用的，承运人对相应的运输货物享有**留置权**。但当事人可以约定排除（如鲜活易腐烂物品）。

**承运人的免责事由**（民法典第832条，从未考过）：货物毁损、灭失，承运人不承担责任的情形：
1. **不可抗力**
2. **货物本身的自然性质或者合理损耗**
3. **托运人、收货人的过错**

**不可抗力下的运费处理**（民法典第835条）：货物在运输过程中因不可抗力灭失，未收取运费的，承运人**不得请求支付运费**；已经收取运费的，托运人**可以请求返还**。


### 第五段：运输合同专题总结

三百四十一页，四个绿色考点：
1. **强制缔约**
2. **承运人的责任和免责事由**（最主要）
3. 托运人的任意解除权（了解）
4. 承运人的留置权


### 第六段：技术合同概述

接下来咱们来看专题三十一，技术合同。这个合同考得越来越少，主要难度在于结合了知识产权。未来学完知识产权后再回头看会好很多。

技术合同的标的物是**技术**，主要包括五种合同类型：**开发、转让、许可、咨询、服务**。考试重点是前三个：**开发、转让、许可**。

**要式合同**（民法典第843条）：技术开发合同、技术转让合同、技术许可合同应当采用**书面形式**。咨询和服务合同不要求书面形式。


### 第七段：技术合同无效的情形

**第一，非法垄断技术**（民法典第850条，民法典技术合同司法解释第10条）：限制技术竞争和发展、限制创新的约定无效。

重点情形：①限制对方在合同标的技术基础上进行新的研究开发；②**搭售**（要求接受并非实施技术必不可少的附带条件，如购买非必须的技术、原材料、产品、设备等）。

**第二，侵害他人技术成果**（民法典第850条）：通过不正当手段从他人处获得技术成果再转让的，技术转让合同**无效**（不管受让人是否知情）。

> **侵害他人技术成果模型**：甲公司向乙公司转让了一项技术秘密（不公开）。经查，该技术秘密是甲公司通过不正当手段从丙公司获得的。
>
> **第一问**：甲公司与乙公司之间的技术转让合同效力如何？
>
> **答**：**无效**（无论乙公司是否知情，都违反法律强制性规定）。
>
> **第二问**：乙公司可否继续使用该技术秘密？
>
> **答**：乙公司**不知情**（善意）→ 可以继续使用；乙公司**知情**（非善意）→ 不能继续使用。
>
> **第三问**：乙公司是否需要支付费用？
>
> **答**：善意 → 向丙公司支付**合理费用**；非善意 → 与甲公司构成**共同侵权**，承担**连带赔偿责任**。
>
> **第四问**：乙公司是否需要履行保密义务？
>
> **答**：**需要**（不管知不知道是偷来的，都要保密）。
>
> **第五问**：合同无效后民事责任如何处理？
>
> **答**：民法典第157条——恢复原状，缔约过失责任。乙公司不知情 → 甲公司单独承担；乙公司知情 → 双方各自承担。


### 第八段：技术开发合同——委托开发

技术开发合同是就新技术、新产品、新工艺、新材料及其系统的研究开发所订立的合同（民法典第851条）。分两种：**委托开发**和**合作开发**。

**委托开发**（民法典第859条）：相当于代理制度。研究开发人（受托人）完成的发明创造，申请专利的权利属于**研究开发人**（事实行为，谁完成归谁）。委托人可以**免费实施**该专利，并享有**优先受让权**。

> **委托开发例题**：甲乙均为丙研究所工作人员，二人受丁公司委托，利用业余时间合作研究了一种新型电池，各方未就专利申请权的归属作出约定。
>
> **问**：该发明的专利申请权属于谁？
>
> **答**：**甲和乙**（研究开发人，事实行为谁完成归谁）。


### 第九段：技术开发合同——合作开发

**合作开发**（民法典第860条）：相当于共有制度。专利申请权属于**合作开发的当事人共有**。

**核心规则**：**一票否决权**——合作开发的当事人一方**不同意**申请专利的，另一方或者其他各方**不得申请专利**。

> **合作开发例题**：甲乙丙三人合作开发一项技术，合同中未约定权利归属。技术开发完成后，甲丙想要申请专利，而乙主张通过商业秘密保护（不想公开）。
>
> **问**：甲丙能否申请专利？
>
> **答**：**不能**。合作开发中，一方不同意申请专利的，一票否决。


### 第十段：技术秘密成果的使用、转让和许可（有约从约，无约均有）

技术秘密成果衍生出三个权利：**使用权**、**转让权**、**收益分配权**。

**有约从约，无约均有**（民法典第861条）：当事人均有使用和转让的权利，包括**自己使用**和以**普通实施许可**方式许可他人使用。

**限制**：以**独占**或**排他**实施许可的方式许可他人使用的，必须征得对方**同意**或事后**追认**，否则无效。

| 许可类型 | 可以使用的人数 | 是否需要对方同意 |
| :--- | :--- | :--- |
| **普通实施许可** | n个人（随意） | **不需要** |
| **排他实施许可** | 2个人（你和对方） | **需要**同意 |
| **独占实施许可** | 1个人（只有你） | **需要**同意 |

> **技术秘密成果例题**：甲公司与乙公司签订技术开发合同，未约定技术秘密成果的归属（委托开发）。甲公司按约支付经费后，乙公司交付技术成果资料。后甲公司在未告知乙公司的情况下，以**普通使用许可**方式许可丙公司使用该技术；乙公司在未告知甲公司的情况下，以**独占使用许可**方式许可丁公司使用该技术。
>
> **问**：下列哪一说法正确？
>
> A. 该技术成果的使用权、转让权仅属于乙公司——**错**（无约均有）
>
> B. 该技术成果的使用权、转让权仅属于甲公司——**错**（无约均有）
>
> C. 甲与丙签订的许可使用合同无效——**错**（普通许可有效）
>
> D. 乙与丁签订的许可使用合同无效——**正确**（独占许可须征得对方同意）
>
> **答案：D**


### 第十一段：技术转让合同与技术许可合同

**技术转让合同**（民法典第862条）：合法拥有技术的权利人，将现有特定的**专利、专利申请权、技术秘密**相关权利让与他人所订立的合同。

**技术许可合同**（民法典第867条）：将现有特定的**专利、技术秘密**相关权利许可他人实施使用所订立的合同（许可相当于“租”给对方用）。

**核心义务**：**保密义务**（不管什么类型都要保密）。

> **专利实施许可例题**：甲公司与乙公司签订专利实施许可合同，约定乙公司在专利有效期限内**独占实施**甲公司的专利技术，并特别约定乙公司**不得擅自改进**该专利技术。后乙公司根据消费者反馈，在未经甲公司许可的情况下对专利技术做了改进，并采取了保密措施。
>
> **问**：下列哪一说法正确？
>
> A. 甲公司有权自己实施改进技术——**错**（独占许可中，连甲公司自己都不能用）
>
> B. 甲公司无权要求分享改进技术——**正确**（谁改进的归谁）
>
> C. 乙公司改进侵犯了甲公司的专利权——**错**（“不得擅自改进”的约定无效）
>
> D. 乙公司改进技术属于违约行为——**错**（约定无效，不存在违约）
>
> **答案：B**


### 第十二段：技术咨询合同与技术服务合同（了解）

**技术咨询合同**（民法典第878条）：委托人按照受托人符合约定要求的咨询报告和意见作出决策所造成的损失，由**委托人承担**（但当事人另有约定的除外）。——就像律师提供咨询意见，最终决定权在委托人自己。

**技术服务合同**（民法典第881条）：不包括承揽合同和建设工程合同。


### 第十三段：技术合同专题总结

技术合同体系：**开发、转让、许可、咨询、服务**五种。

重点掌握：委托开发（专利申请权归研究开发人）、合作开发（一票否决权）、技术秘密成果（无约均有，独占/排他须同意）。


### 第十四段：保管合同

专题三十二，保管合同和专题三十三仓储合同放在一起讲，因为是一类合同。保管合同是**民事合同**，仓储合同是**商事合同**。

**保管合同的认定**（民法典第888条）：寄存人到保管人处从事**购物、就餐、住宿**等活动，将物品存放在指定场所的，**视为保管**（超市自助存取包、宾馆寄存行李）。

**性质**：
- **可有偿可无偿**（有偿保管 vs 无偿保管）
- **实践性合同**（自保管物交付时成立，民法典第890条）

**寄存人的任意解除权**（民法典第899条）：寄存人可以**随时领取**保管物。

**保管人的留置权**（民法典第903条）：寄存人未支付保管费或其他费用的，保管人对保管物享有**留置权**。但当事人可以约定排除。

**无偿保管人的责任**（民法典第897条）：保管人证明自己没有**故意或重大过失**的，不承担赔偿责任（因为没拿钱，要求不能太高）。

> **无偿保管例题**：孟某因装修房屋，将一批古书交朋友徐某**无偿保管**。徐某将古书置于床下。一日，楼上住户家水管冻裂，水流至徐某家，致古书严重受损。
>
> **问**：徐某是否承担赔偿责任？
>
> **答**：**不承担**。徐某系无偿保管，无故意或重大过失。责任应由楼上住户承担。


### 第十五段：仓储合同

仓储合同是保管人储存存货人交付的仓储物，存货人支付仓储费的合同（民法典第904条）。

**与保管合同的区别**：

| 对比项 | 保管合同 | 仓储合同 |
| :--- | :--- | :--- |
| **类型** | 民事合同 | **商事合同** |
| **是否有偿** | 可有偿可无偿 | **一定有偿** |
| **合同性质** | **实践性合同**（交付时成立） | **诺成性合同**（意思表示合意即成立） |
| **解除权** | 寄存人可随时领取 | 无约定保管期间，双方均有任意解除权 |
