# 合同的原理与分类

## 第1部分：对照《民法典》合同编相关条款提取备考提纲

| 分编 | 内容 | 法典依据 |

### 二、合同的学理分类（八组）

#### （一）单务合同 vs 双务合同

| 类型 | 区分标准 | 典型合同 |
| :--- | :--- | :--- |
| **双务合同**（原则） | 有**对待给付**义务 | 买卖合同 |
| **单务合同**（例外） | 无对待给付义务 | **赠与合同** |


#### （二）有名合同 vs 无名合同（民法典第467条）

| 类型 | 含义 | 法律适用 |
| :--- | :--- | :--- |
| **有名合同**（典型合同） | 法律赋予特定名称并设有规则 | 适用各该章规定 |
| **无名合同**（非典型合同） | 法律未赋予特定名称 | 适用通则 + **类推适用**最相类似的典型合同 |


#### （三）有偿合同 vs 无偿合同

| 类型 | 区分标准 | 典型合同 |
| :--- | :--- | :--- |
| **有偿合同**（原则） | 有**对价**（要钱） | 买卖、租赁等 |
| **纯粹无偿合同**（例外） | 无对价 | 赠与、担保、借用 |
| **可有偿可无偿**（★★★） | 根据约定 | **委托、保管、自然人借贷** |


#### （四）诺成性合同 vs 实践性合同（核心考点★★★★★）

| 类型 | 成立条件 | 典型合同 |
| :--- | :--- | :--- |
| **诺成性合同**（原则） | 意思表示合意即成立（一诺即成） | 买卖、租赁、赠与 |
| **实践性合同**（例外） | 意思表示合意 + **交付标的物** = 成立 | 定金、自然人借贷、保管、借用【补充：民法典第586条、第679条、第890条、第889条】 |


#### （五）要式合同 vs 不要式合同

| 类型 | 区分标准 | 典型合同 |
| :--- | :--- | :--- |
| **不要式合同**（原则） | 不需要特定形式 | 买卖合同、赠与合同 |
| **要式合同**（例外） | 需要特定形式（如**书面**） | **担保合同**（抵押、质押、保证、定金）【补充：民法典第400条、第427条、第469条、第586条】 |


#### （六）本约合同 vs 预约合同（核心考点★★★★★）

| 类型 | 含义 | 核心规则 |
| :--- | :--- | :--- |
| **本约合同** | 最终欲签订的合同 | — |
| **预约合同** | 约定将来签订本约的合同 | 违反预约 → 追究**违约责任**，但**不得强制缔约**（不得请求继续履行）【补充：民法典第495条】 |


#### （七）射幸合同 vs 确定合同

| 类型 | 区分标准 | 典型合同 |
| :--- | :--- | :--- |
| **确定合同**（原则） | 标的物在合同成立时确定 | 买卖、租赁 |
| **射幸合同**（例外） | 标的物发生与否**不确定** | **保险合同**、彩票合同、抽奖式有奖销售 |


#### （八）一时性合同 vs 继续性合同

| 类型 | 区分标准 | 典型合同 |
| :--- | :--- | :--- |
| **一时性合同**（原则） | 一锤子买卖 | 买卖 |
| **继续性合同**（例外） | 时间持续，对时间因素要求高 | **租赁合同**、供用电水气热力合同 |

### 第一段：合同编体系概览

好，接下来咱们来看第四部分，叫做合同编。书的第一百八十六页，我们先来看体系：它一共有三个分编——**第一分编通则**、**第二分编十九类典型合同**、**第三分编准合同（无因管理和不当得利）**。准合同我们在前边债的基本原理部分已经讲完了，所以这个部分我们继续重点学习的是**通则**和**典型合同**。

合同编的通则就是一个合同从生到死的流程，一共有八章。其中有两章每年绝对必考：**第五章合同的保全**和**第八章违约责任**。学习的时候要重点关注这两章。


### 第二段：适用范围与合同的学理分类概述

先来看一百八十七页专题十二，合同的基本原理，就两个问题：第一个适用范围（了解），第二个是重点——**合同的学理分类**。

**适用范围**：合同编所说的合同主要是**财产关系的协议**。有关**身份关系**的协议（如离婚协议、收养协议、监护协议），先要受到民法典其他各编（婚姻家庭编、总则编）的调整，只有在其他编没有规定且与合同编性质不相违背的情况下，才能参照适用。

**合同的学理分类**：一共有**八组**分类——单务合同和双务合同、有名合同和无名合同、有偿合同和无偿合同、诺成性合同和实践性合同、要式合同和不要式合同、本约合同和预约合同、射幸合同和确定合同、一时性合同和继续性合同。

考试怎么考？**一个选项**，在多选题中嵌入一个选项问合同的性质如何。每一组分类中都是**原则 + 例外**，考试不考原则、考例外，不考常态、考变态。书的一百八十七页下方带绿色字体的部分就是考点。


### 第三段：第一组——单务合同 vs 双务合同

第一组，单务合同和双务合同。区分标准就四个字：**对待给付**——看看合同双方当事人有没有对待给付关系。

**原则：双务合同**——双方互负债务，有对待给付义务。典型：**买卖合同**。

> 买卖合同中有两个标的物（表+钱），卖方权利=请求付款（买方的义务），买方权利=请求交表（卖方的义务），双方互为权利人和义务人。

**例外：单务合同**（少数派，考试重点）。典型：**赠与合同**。

> 老孟给刘某送大白兔奶糖、鸡蛋、巧克力，刘某收下了。老孟说“你看你都接受了这么多礼品，就当我女朋友呗”，刘某说“滚”。请问咋办？没办法！人家收下了也不一定非得当你女朋友作为回报，人家没有义务。赠与人只有义务没有权利，受赠人只有权利没有义务。


### 第四段：第二组——有名合同 vs 无名合同（民法典第467条）

**有名合同**（典型合同）：民法典及相关民商事法律赋予了特定名称并设有特定规则。后边的十九类典型合同都是有名合同。

**无名合同**（非典型合同，考试重点）：法律没有明文规定名称和规则的合同。因为民商事活动具有复杂性、多样性，不可能通过十九个合同把所有生活都囊括掉。

**核心考点（民法典第467条）** ：无名合同适用合同编通则的规定，并**可以参照适用**合同编典型合同或其他法律最相类似的合同规定——**类推适用**。

> **无名合同例题（房屋换钢琴）** ：孟某将房无偿提供给马某居住，马某无偿教孟某的女儿弹钢琴。
>
> **分析**：既不像租赁合同（租赁是有偿的），也不像劳务合同（单纯请家教），是个“骡子”式的混合合同。
>
> **问**：房屋漏雨产生纠纷，法院怎么处理？
>
> **答**：类推适用租赁合同规则——出租人有适租义务，要提供适合居住的标准。
>
> **问**：马某教钢琴教得不好，怎么处理？
>
> **答**：类推适用劳务合同规则。

**其他无名合同**：培训合同、美容服务协议、储蓄合同、让与担保合同等。


### 第五段：第三组——有偿合同 vs 无偿合同

区分标准：**对价**——要不要钱。

**原则：有偿合同**——买卖、租赁等。

**例外：无偿合同**（少数派）：

**三类纯粹无偿的**：赠与合同、担保合同、借用合同。

**三类可有偿可无偿的**（考试重点★★★★★）：
1. **委托合同**（绝对必考）——律师事务所收费委托 vs 法律援助无偿委托
2. **保管合同**——收费停车场 vs 不收费停车场
3. **自然人之间的借款合同**——可要利息可不要利息，没约定视为无利息

**区分意义**：**注意义务程度不同**——有偿合同注意义务更高，违约责任的追究更重。


### 第六段：第四组——诺成性合同 vs 实践性合同（核心考点★★★★★）

**区分标准**：合同成立的标志不同。

**原则：诺成性合同**（不要物合同）——只要双方当事人意思表示合意，合同即成立（一诺即成、一诺千金）。如买卖合同、租赁合同、赠与合同。

**例外：实践性合同**（要物合同）——意思表示合意 + 交付标的物 = 合同成立。少数派，整个民商法中只有**四个**：

| 与钱有关 | 与物有关 |
| :--- | :--- |
| **定金合同** | **保管合同** |
| **自然人之间的借款合同** | **借用合同** |

**重点：两个与钱有关的**（定金、自然人借贷）【补充：民法典第586条、第679条】。

> **定金合同例题**：孟某与宝马4S店签订书面定金合同，约定第二天孟某交付定金10万元，4S店将一台宝马轿车出卖给孟某。第二天孟某反悔不想买了。
>
> **问**：4S店能否追究孟某的违约责任？
>
> **答**：**不能**。定金合同是实践性合同，自交付定金时成立。没交付定金之前，孟某可以随时反悔，既不需要承担违约责任，也不需要承担缔约过失责任。理由有二：①给付定金一方（老百姓）往往处于劣势地位，法律赋予其无责任反悔权；②定金法则的适用以交付定金为前提，没交付就无法适用。

> **自然人借款合同例题**：孟某向马某借款100万，双方签订书面借款合同，约定第二天马某将100万元交付给孟某。第二天马某反悔了。
>
> **问**：孟某能否追究马某的违约责任？
>
> **答**：**不能**。自然人之间的借款合同是实践性合同，自提供借款时成立。交付借款之前，马某随时可以反悔，不用承担任何责任。理由：具有互助性，法律不得强迫任何人做一个好人。

**两个与物有关的**：保管合同、借用合同（了解即可）。


### 第七段：第五组——要式合同 vs 不要式合同

**区分标准**：是否需要特定形式（如书面形式）。

**原则：不要式合同**——买卖合同、赠与合同等（口头即可）。

**例外：要式合同**（考试重点）——**担保合同**（抵押合同、质押合同、保证合同、定金合同）必须采用**书面形式**。给别人做担保往往是不利的，要让人慎重考虑，签字画押。


### 第八段：第六组——本约合同 vs 预约合同（核心考点★★★★★）

**这是考试绝对重点！**

**一交易两合同**：买一套房却可能签两份合同。之所以不直接签本约，是因为签订本约的条件尚不具备。

**最典型的场景：商品房买卖中的认购书/预定书**（民法典第495条）。开发商未取得预售许可证前，不能签商品房预售合同（本约），否则合同无效。于是先签一份**预约合同**（认购书），规避行政监管。

**预约合同的三个必备要素**：当事人、标的物、时间（合同成立即生效）。

**核心考点（民法典第495条）** ：违反预约合同构成违约，可以追究**违约责任**，但**不得强制缔约**（不得请求继续履行）。因为行为无法直接强制。

> **例题（商铺认购书案）** ：甲公司未取得商铺预售许可证，便与李某签订商铺认购书（预约合同），约定李某支付认购金即可取得优先认购权。双方约定了认购面积和房价，但对楼号、房型未做约定。李某依约支付了认购金。甲公司取得预售许可证后，未通知李某前来认购，将商铺售罄。
>
> **问**：下列哪一表述是正确的？
>
> A. 认购书无效，因甲公司未取得预售许可证即对外销售——**错**（预约合同不是本约，不需要预售许可证）
>
> B. 认购书不成立，因合同内容不完整——**错**（三要素齐备，合同成立）
>
> C. 甲公司未履行通知义务，构成根本违约——**正确**（合同目的无法实现）
>
> D. 甲公司需承担继续履行的违约责任——**错**（不得强制缔约，只能赔钱）


### 第九段：第七组——射幸合同 vs 确定合同

**区分标准**：标的物的内容和范围是否在合同成立时就确定。

**原则：确定合同**——买房（房确定）、买车（车确定）。

**例外：射幸合同**（发生与否不确定）——典型：**保险合同**（保险事故是否发生不确定）、彩票合同、抽奖式有奖销售合同。


### 第十段：第八组——一时性合同 vs 继续性合同

**区分标准**：时间因素。

**原则：一时性合同**——一锤子买卖（买房、买车）。

**例外：继续性合同**——履行对时间因素要求高，时间持续。典型：**租赁合同**（有租期，最长不超过20年）、供水电气热力合同。


### 第十一段：八组分类总结

八组分类中，重点掌握**三个**：
