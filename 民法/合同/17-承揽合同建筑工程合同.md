## 第1部分：对照《民法典》承揽合同与建设工程合同提取备考提纲


### 一、承揽合同（民法典第770-787条）

#### （一）定义（第770条）

承揽合同是承揽人按照定作人的要求完成工作，交付工作成果，定作人支付报酬的合同。

**核心属性**：**强烈的人身信赖关系**。

#### （二）主要工作 vs 辅助工作（第772条、第773条）

| 工作类型 | 能否交由第三人 | 是否需要定作人同意 | 法律效果 |
| :--- | :--- | :--- | :--- |
| **主要工作** | 原则上**不能** | **需要**同意 | 擅自转承揽 → 定作人可**解除合同** |
| **辅助工作** | **可以** | **不需要** | 承揽人就第三人工作成果向定作人负责 |

#### （三）两大权利

| 权利 | 权利主体 | 法典依据 | 说明 |
| :--- | :--- | :--- | :--- |
| **任意解除权** | **定作人** | 第787条 | 完成工作前可随时解除，但应赔偿损失 |
| **留置权** | **承揽人** | 第783条 | 定作人未支付报酬或材料费的，承揽人可留置工作成果 |


### 二、建设工程合同（民法典第788-808条）

#### （一）定义与形式（第788条、第789条）

建设工程合同是承包人进行工程建设，发包人支付价款的合同。应当采用**书面形式**。

**特殊身份要求**：承包人须有**建筑业企业资质**，无资质不能签合同。

#### （二）四大禁止行为（第791条）

1. 禁止发包人**肢解发包**
2. 禁止承包人**全部转包**
3. 禁止承包人**肢解后全部分包**
4. 禁止承包人分包给**不具备相应资质的单位**
5. 禁止**分包单位再分包**

**主体工程亲自完成**：主体结构的施工必须由总承包人**亲自完成**，即使经发包人同意转包也**无效**。

#### （三）无效情形（建工司法解释一第1条、第2条、第3条）

1. 承包人**未取得资质或超越资质**（无资质即无效）
2. **挂靠**（没有资质的实际施工人借用有资质的建筑施工企业名义）→ 不合格时承担**连带责任**
3. 应当招标而未招标或**中标无效**
4. **阴阳合同**（背离中标合同实质性内容→黑合同无效，中标合同有效）
5. 起诉前未取得**建设工程规划许可证**等审批手续（起诉前取得的除外）；发包人能办不办，又以未办为由主张无效的，法院**不予支持**

#### （四）无效合同的处理（建工司法解释一第6条）

| 工程质量 | 处理规则 |
| :--- | :--- |
| **合格** | **参照**最后一份合同约定**折价补偿** |
| **不合格 + 可修复** | 修复后合格 → 参照补偿 |
| **不合格 + 不可修复** | **没有权利**（不能主张补偿） |

#### （五）诉讼地位（建工司法解释一第15条、第43条）

- **工程质量不合格**：发包人可将总承包人、分包人、实际施工人作为**共同被告**（连带责任）
- **工程质量合格**：实际施工人可以发包人为被告主张权利，法院应追加转包人或违法分包人为**无独立请求权的第三人**，发包人在欠付工程价款范围内承担责任

#### （六）建设工程价款优先受偿权（第807条，建工司法解释一第35-42条）

| 要点 | 规则 |
| :--- | :--- |
| **性质** | **法定优先权**（债权优于物权的例外） |
| **目的** | 保护农民工等建筑工人的工资权益 |
| **主体** | 与发包人直接订立建设工程施工合同的**承包人**（总承包人、装饰装修工程承包人） |
| **前提** | 工程质量**合格** |
| **范围** | 仅限于**工程款本身**（利息、违约金等**不享有**优先受偿权） |
| **行权方式** | ①与发包人协议折价；②申请法院拍卖 |
| **行权期间** | **十八个月**除斥期间（**新增，从未考过**），自发包人**应当给付建设工程价款之日**起算 |

### 第一段：承揽合同概述

好，接下来咱们来看专题二十八，叫做承揽合同。这个合同是偶尔才会考察一次的，不是考试的重点。咱们把它的**一个属性、两个权利**抓住就可以了。

先来看什么叫承揽合同。承揽合同是承揽人按照定作人的要求完成工作，交付工作成果，定作人支付报酬的合同（民法典第770条）。主要包括六种：**加工、定作、修理、复制、测试、检验**。考试重点是前三种：**加工、定作和修理**（修车、修手表、修机械设备、定做礼服）。


### 第二段：承揽合同的核心——人身信赖关系

这里边就**一个属性、两个权利**。最重要的属性就是**强烈的人身信赖关系**。

以定制礼服为例：结婚时刘某要穿婚纱，我配西装。她千挑万选选中A公司来定做婚纱——就是看中人家手艺好。所以承揽人得亲自完成主要工作。

**主要工作**：我选择你就是因为信任你的手艺，承揽人得**亲自完成**。如果转交给第三人，必须征得定作人的同意。擅自转承揽 = **根本违约**，定作人可**解除合同**追究违约责任。

**辅助工作**（如修花边、安扣子、装拉链）：可以由承揽人自己完成，也可以交由第三人完成，**不需要征得定作人同意**。但承揽人要就第三人的工作成果向定作人负责。


### 第三段：承揽人的两大权利

**第一个权利——定作人的任意解除权**（民法典第787条）：定作人在承揽人完成工作前可以**随时解除**合同。原因：如果定作人觉得做这个东西没用了，让人家继续做无用功对整个社会也不利。解除后，承揽人已完成的工作部分的报酬应当支付。

**第二个权利——承揽人的留置权**（民法典第783条）：定作人未向承揽人支付报酬或者材料费等价款的，承揽人对完成的工作成果享有**留置权**。


### 第四段：建设工程合同概述

接下来咱们来看专题二十九，建设工程合同。这个合同是第二梯队的常考类型，特别是主观题经常会拿它做基础法律关系。

这个合同的特色：**对上**有社会公共利益和国家利益的保护（楼倒倒、桥塌塌会损害不特定人民群众的人身财产安全）；**对下**有弱势群体的保护（农民工利益保护）。

**合同订立方式**：建设工程施工合同通过**招投标**方式订立（竞争性缔约）。

**特殊身份要求**：必须有**建设工程施工企业资质**，没有资质签不了合同。国家为了保障工程质量，对施工企业有资质限制（一级、二级、特级资质）。

**合同形式**：建设工程合同应当采用**书面形式**（民法典第789条）。


### 第五段：四大禁止行为（民法典第791条）

法律规定了四大禁止行为（因为实践中普遍存在）：

| 序号 | 禁止行为 | 说明 |
| :--- | :--- | :--- |
| **1** | **禁止发包人肢解发包** | 不能把一栋20层的教学楼切成小块分别发包给不同企业 |
| **2** | **禁止承包人全部转包** | 卖工程（“工程猴子”），自己啥事不干——**主体工程必须由总承包人亲自完成**，即使经发包人同意也无效 |
| **3** | **禁止承包人肢解后全部分包** | 全部分包 = 转包 |
| **4** | **禁止承包人分包给不具备相应资质的单位** | 无资质不能干 |
| **5** | **禁止分包单位再分包** | 分包单位不能再分包给第三人 |

> **主体工程亲自完成**（民法典第791条第2款）：总承包人经发包人同意，可以将自己承包的部分工作交由第三人完成，但**主体结构**的施工必须由总承包人**亲自完成**。即使经发包人同意，将主体结构转包的，转包合同**无效**。

> **建设工程施工合同模型（甲大学建教学楼）** ：甲大学欲建设一栋20层教学楼，乙、丙、丁有资质，戊、己无资质。
>
> **第一问**：甲大学可否将20层教学楼肢解五份发包？
>
> **答**：**不可以**（禁止发包人肢解发包）。
>
> **第二问**：如乙公司中标，乙公司可否将该建设工程全部转包给丙公司？
>
> **答**：**不可以**（禁止承包人全部转包）。主体工程必须由总承包人亲自完成，即使经发包人同意也无效。
>
> **第三问**：如乙公司中标，乙公司可否肢解后全部分包给丙公司和丁公司？
>
> **答**：**不可以**（全部分包 = 转包）。
>
> **第四问**：如乙公司中标，乙公司可否分包给无资质的戊公司或己公司？
>
> **答**：**不可以**（必须有资质）。
>
> **第五问（重点）** ：如乙公司中标，经甲大学同意，乙公司将部分非主体工程分包给丙公司，丙公司可否再分包给丁公司？
>
> **答**：**不可以**（禁止分包单位再分包）。丙公司是违法分包人，丁公司是实际施工人，合同无效。
>
> **第六问**：如乙公司将建设工程转包或违法分包，甲大学可否解除合同？
>
> **答**：**可以**。根本违约 → 非违约方甲大学享有**法定解除权**。


### 第六段：建设工程施工合同无效的六种情形

建设工程施工合同到考场上大胆选无效，一般它都是无效的。

| 序号 | 无效情形 | 说明 |
| :--- | :--- | :--- |
| **1** | **无资质或超越资质** | 无资质即无效（重点考无资质） |
| **2** | **挂靠** | 没有资质的实际施工人借用有资质的建筑施工企业名义——**连带责任**（干的活不合格时承担连带责任） |
| **3** | **陪标** | 应当招标而未招标或中标无效（暗箱操作） |
| **4** | **阴阳合同** | 另行订立的合同实质性变更中标合同内容（工程范围、工期、工程质量、工程价款等）→ 黑合同无效，中标合同有效 |
| **5** | **先上车后补票（补正规则）** | 发包人起诉前未取得建设工程规划许可证等审批手续→无效；但**起诉前**取得的除外。发包人能够办理而不办理，又以未办理为由主张合同无效的，法院**不予支持**（权利滥用） |

> **阴阳合同例题**：中标合同约定工程款1亿，另行签订合同约定8000万（变更了工程价款这一实质性内容）。
>
> **分析**：中标合同（白合同）**有效**，另行订立的合同（黑合同）**无效**。

> **先上车后补票例题**：发包人在起诉前未取得建设工程规划许可证，但**能够办理却恶意不办理**，自己跑到法院主张合同无效。
>
> **答**：法院**不予支持**（权利滥用，不能自我检举揭发）。


### 第七段：无效合同的处理（民法典第793条）

合同无效了，但工程质量才是重点。人家活干了，农民工利益要保护。

| 工程质量 | 处理 |
| :--- | :--- |
| **合格** | **参照**（不是“按照”——因为合同无效）最后一份合同的约定**折价补偿** |
| **不合格 + 可修复** | 修复后合格 → 参照最后一份合同折价补偿 |
| **不合格 + 不可修复** | **没有权利**，连补偿都拿不到（干活不合格，白干） |

> **无效合同处理例题（2017年真题）** ：甲房地产开发公司将项目主体工程发包给乙企业，乙企业一直未取得建筑施工企业资质（**无资质→合同无效**）。该项目主体工程已封顶完工。
>
> **问**：下列哪些说法正确？
>
> A. 该建设工程施工合同无效——**正确**（无资质即无效）
>
> B. 因项目主体工程已封顶完工，故合同不应认定为无效——**错**（不能因为活干完了就变合法）
>
> C. 该项目主体工程经竣工验收合格，乙企业可以参照合同约定请求甲公司**折价补偿**——**正确**（无效+合格→参照补偿）
>
> D. 该项目主体工程经竣工验收不合格，甲公司可以请求乙企业承担赔偿责任——**正确**
>
> **答案：A、C、D**


### 第八段：建设工程施工合同中的诉讼地位（债权人代位权诉讼的应用）

> **模型**：甲大学（发包人）欠乙公司（总承包人）钱，乙公司欠丙公司（违法分包人）钱，丙公司欠丁公司（实际施工人）钱。丁公司直接把甲大学告到法院去——**债权人代位权诉讼**。

**两种情形**（民法典第793条、建工司法解释一第15条、第43条）：

| 工程质量 | 谁告谁 | 规则 |
| :--- | :--- | :--- |
| **不合格** | 发包人告总承包人、分包人、实际施工人 | **共同被告**（连带责任，突破合同相对性） |
| **合格** | 实际施工人告发包人 | 法院应当追加转包人或违法分包人为**无独立请求权的第三人**；判决发包人在**欠付工程价款范围内**对实际施工人承担责任（突破合同相对性） |


### 第九段：建设工程价款优先受偿权（法定优先权，民法典第807条，核心考点★★★★★）

**本质**：债权优于物权（例外）——目的是对**农民工等建筑工人的工资权益予以优先保护**。

**地位**：它是**法定优先权**（法律规定，银行即使是物权人也得排在后面）。

**主体**：与发包人直接订立建设工程施工合同的**承包人**（总承包人；装饰装修工程的承包人也可主张，但限于具备折价或拍卖条件的装饰装修工程）。

**前提**：工程质量**合格**（跟合同有效无效没关系，跟工程有没有竣工没关系，烂尾了也能要）。

> **法定优先权例题**：大项公司（无资质）与中森公司签订建设工程施工合同，无资质 → 合同**无效**，但工程验收**合格**。
>
> **问**：大项公司是否有权主张建设工程价款优先受偿权？
>
> **答**：**有权**。工程质量合格即可主张优先受偿权，与合同是否有效无关。


### 第十段：法定优先权的范围与行权期间

**范围**：仅限于**工程款本身**（民法典第807条，建工司法解释一第35条）。利息、违约金、损害赔偿金等**不享有**优先受偿权。

> **例题**：发包人欠承包人工程款2000万，利息200万，违约金100万。
>
> **问**：法定优先权的范围是多少？
>
> **答**：**2000万**（只有工程款本身），利息和违约金是**普通债权**，没有优先受偿权。

**行权方式**（民法典第807条）：①与发包人**协议**将建设工程折价；②**申请法院拍卖**。

**行权期间**（建工司法解释一第41条，**新增，从未考过★★★★★**）：**十八个月**的除斥期间，自**发包人应当给付建设工程价款之日**起算。

> **法定优先权综合模型（中铁十八局案）** ：众森公司开发幸福小区，与中铁十八局签订建设工程施工合同（主体工程，工程款2000万），与中铁七局签订装饰装修工程施工合同（装饰装修工程，工程款1000万）。两工程均验收合格，交付后众森公司资金链断裂，将五栋别墅抵押给工商银行贷款2000万。产生利息200万和100万。法院拍卖别墅得3300万。
>
> **第一问**：中铁十八局是否有权优先受偿？
>
> **答**：**有权**（与发包人直接订立合同的总承包人）。
>
> **第二问**：中铁七局是否有权优先受偿？
>
> **答**：**有权**（装饰装修工程的承包人，具备折价或拍卖条件）。
>
> **第三问**：前提是什么？
>
> **答**：工程质量**合格**。
>
> **第四问**：优先受偿权的范围是多少？
>
> **答**：中铁十八局**2000万**（工程款），中铁七局**1000万**（工程款）。利息合计300万**没有**优先受偿权。
>
> **第五问**：行权期间？
>
> **答**：**十八个月**的除斥期间，自**发包人应当给付建设工程价款之日**起算。
>
> **第六问**：排序？
>
> **答**：中铁十八局 > 中铁七局 > 工商银行。**债权优于物权**（法定优先权例外）。
>
> **第七问**：3300万如何分配？
>
> **答**：中铁十八局拿走2000万，中铁七局拿走1000万，工商银行拿走300万。工商银行剩余未受偿部分转为**普通债权**。


### 第十一段：本专题总结

翻回三百三十四页，建设工程合同的体系：

| 考点 | 核心内容 |
| :--- | :--- |
| **四大禁止行为** | 禁止肢解发包、禁止全部转包、禁止分包给无资质单位、禁止分包单位再分包 |
| **无效情形** | 无资质、挂靠、陪标、阴阳合同、先上车后补票 |
| **无效处理** | 合格→参照最后一份合同折价补偿；不合格→修复或没有权利 |
| **诉讼地位** | 不合格→发包人告一串（共同被告）；合格→实际施工人告发包人（无独三） |
| **法定优先权** | 债权优于物权；工程款本身；十八个月除斥期间（自应付之日起算） |
