# 合同的订立

## 第1部分：对照《民法典》合同订立相关条款提取备考提纲


### 一、要约与要约邀请

| 概念 | 定义 | 法典依据 | 关键区别 |
| :--- | :--- | :--- | :--- |
| **要约** | 希望与他人订立合同的**意思表示** | 第472条 | ✅ 有法律约束力；内容**具体确定** |
| **要约邀请** | 希望他人向自己发出要约的**表示** | 第473条 | ❌ 无法律约束力；内容不一定具体确定 |

**商业广告和宣传**：原则要约邀请；内容具体确定（三要素齐全）→ 视为**要约**（第473条第2款）。


### 二、要约的生效、撤回、撤销与失效

| 制度 | 时间节点 | 核心规则 | 法典依据 |
| :--- | :--- | :--- | :--- |
| **生效** | 到达受要约人时 | **到达主义**（非对话方式） | 第474条、第484条 |
| **撤回** | 要约**生效前** | 撤回通知**先于或同时**到达 | 第475条、第477条 |
| **撤销** | 要约**生效后**，**承诺作出前** | 撤销通知在承诺作出前到达 | 第476条、第477条 |
| **不得撤销** | — | ①定期限 ②有名式 ③有理由 | 第476条 |
| **失效** | — | ①被拒绝 ②被撤销 ③期满未承诺 ④实质性变更 | 第478条 |


### 三、承诺

| 制度 | 核心规则 | 法典依据 |
| :--- | :--- | :--- |
| **定义** | 受要约人**完全同意**要约的意思表示 | 第479条 |
| **方式** | 通知（原则）；**行为**（例外，履行抵销规则） | 第480条、第490条 |
| **生效** | **到达**要约人时生效 | 第484条 |
| **撤回** | 承诺可以**撤回**（通知先于或同时到达） | 第485条 |
| **承诺迟延**（内因） | 视为**新要约** | 第486条 |
| **承诺迟到**（外因） | **有效**（除非要约人及时反对） | 第487条 |


### 四、合同成立的时间与地点

| 合同类型 | 成立时间 | 法典依据 |
| :--- | :--- | :--- |
| **承诺生效时**（原则） | 承诺到达时 | 第483条 |
| **要式合同**（书面） | **最后一方签章/按指印时** | 第490条 |
| **要物合同**（实践性） | **交付标的物时** | — |
| **招标投标** | **中标通知书到达中标人时** | 第490条（参照） |
| **拍卖** | **落槌/电子交易系统确认时** | — |


### 五、合同条款

| 条款类型 | 核心考点 | 法典依据 |
| :--- | :--- | :--- |
| **必备条款** | 三要素：**当事人、标的物、数量**（数量最重要） | 第470条 |
| **争议解决条款** | **独立性**——合同不成立/无效/被撤销/确定不发生效力，不影响其效力 | 第507条 |
| **格式条款** | ①判断：**有没有谈**；②提示说明义务；③无效情形；④**不利解释规则** | 第496-498条 |

**格式条款核心规则**（民法典第496-498条）：
- 判断标准：**未与对方协商**
- 提供方义务：**主动提示**（异常条款）+ **被动说明**
- 违反后果：条款**不成为合同内容**（不成立）
- 无效情形：①人身损害免责无效；②故意/重大过失财产损害免责无效；③不合理免责/加重/限制；④排除主要权利
- 解释规则：**不利于提供方**的解释


## 第2部分：近年考点与2026年预测

| 知识点 | 是否涉及主观题 | 近年考情 | 2026年可能考试方式 | 举例 |
| :--- | :--- | :--- | :--- | :--- |
| **要约 vs 要约邀请——商业广告** | 客观题 | **2023年考过** | 给出“商业广告/网购信息”场景，问是要约还是要约邀请 | 甲在二手平台发布“出售自行车，1000元，先到先得”——要约（内容具体确定） |
| **要约的撤销——不得撤销情形** | 客观题 | 常考 | 给出“定期限/有名式/有理由”场景，问能否撤销 | 甲发要约“请于10日内答复”，3日后反悔——**不得撤销**（定期限） |
| **承诺的迟延 vs 迟到** | 客观题 | 常考 | 给出“承诺逾期”场景，问原因及效力 | 甲发要约“4月8日前回复”，乙4月9日回复——承诺**迟延**（内因），视为新要约 |
| **实质性变更——砍价** | 客观题 | **2023年考过** | 给出“变更价款”场景，问是承诺还是新要约 | 甲发要约1000元，乙回复900元——**实质性变更**，为新要约/反要约 |
| **格式条款——判断（有没有谈）** | 客观题 | **每年必考** | 给出“打印条款/手写条款”场景，问是否为格式条款 | 合同中有打印条款和手写条款——**打印条款**可能为格式条款，手写条款为非格式条款 |
| **格式条款——提示说明义务** | 客观题/主观题 | 常考 | 给出“未提示异常条款”场景，问条款是否成立 | 保险合同争议解决条款未加粗提示——条款**不成为合同内容** |
| **格式条款——不利解释规则** | 客观题 | 常考 | 给出“机票退票手续费计算标准有歧义”场景，问如何解释 | 机票“加收票价50%手续费”，“票价”有歧义——适用不利解释规则，退500元 |
| **格式条款——无效情形** | 客观题 | 常考 | 给出“美容卡余款不退”场景，问条款是否有效 | 美容服务协议“如单方放弃服务，余款不退”——**无效**（不合理限制消费者权利） |
| **争议解决条款的独立性** | **✅ 主观题★★★★★** | 常考 | 给出“合同无效但争议解决条款有效”场景，问法院是否有管辖权 | 借款合同无效，但合同中约定“纠纷由某法院管辖”——争议解决条款**独立有效** |
| **履行抵销规则（行为承诺）** | 客观题 | 偶有考查 | 给出“一方已履行主要义务+对方接受”场景，问合同是否成立 | 甲向乙下订单未签书面合同，乙发货甲收货——合同**成立**（履行抵销） |

> **老师原话总结**：
> - “要约是**意思表示**（有法律约束力），要约邀请是**表示**（无法律约束力）。”
> - “商业广告和宣传——**内容具体确定就是要约**（三要素：当事人、标的物、数量）。”
> - “承诺的迟延是**内因**（受要约人有过错）→ 视为新要约；承诺的迟到是**外因**（受要约人无过错）→ 承诺有效。”
> - “格式条款——**有没有谈**是判断标准。没谈的就是格式条款。”
> - “格式条款——**不跟你谈的条款，打不打招呼都要提示**；没提示 → 条款不成为合同内容。”
> - “争议解决条款——**独立性**，合同死了它还活着（民法典第507条）。”
> - “格式条款解释——**不利于提供方**（民法典第498条）。”

**合同订立 vs 合同成立**：订立是一个**过程**，成立是一个**结果**。


### 第二段：要约与要约邀请的区别（核心考点）

一百九十七页下方引出了第一个问题：**要约和要约邀请的区别**。

#### （一）要约（民法典第472条）

**要约**：希望与他人订立合同的**意思表示**（有法律约束力）。

**两个构成要件**：
1. **内容具体确定**——三要素：**当事人、标的物、数量**
2. **表明经受要约人承诺，要约人即受该意思表示约束**——不是闹着玩的

**要约的典型类型**：自动售货机、自助寄包机、ATM自助存取款机、自助停车机——这些都是**现货要约**。

#### （二）要约邀请（民法典第473条）

**要约邀请**：希望他人向自己发出要约的**表示**（**无法律约束力**）。

**重点：商业广告和宣传**
- **原则上**：商业广告和宣传是要约邀请（只描述商品的好，不具体确定数量）
- **例外**：内容**具体确定**（三要素齐备）→ 视为**要约**

> **网购规则**：根据电子商务法，网店上发布的商业广告和宣传，**一律视为要约**（内容具体确定，默认数量为一件）。即使商家特别声明“不是要约”，该声明也无效。

> **例题（2023年真题）** ：甲在校内二手平台上发布信息：“出售自己的自行车，价款一千元”，附上发票和照片，并附言“诚信出售，先到先得”。
>
> **问**：该信息是要约还是要约邀请？
>
> **答**：**要约**。三要素全部具备：当事人（甲）、标的物（自行车）、数量（一辆）。
>
> **乙看到后**通过微信联系甲**出价九百元**——这是**实质性变更**（变更价款），不构成承诺，而构成**新要约/反要约**。
>
> **几天后**乙给甲发微信：“就按你说的价来，一千一百元。”——乙显然想买，但写错了价格。
>
> **规则**：根据合同条款解释规则（民法典第142条、第466条；合同编通则司法解释第1条），应当探究当事人的**内心真意**（误在无害真意规则），结合磋商过程判断。正确答案：**合同成立，价格为一千元**。


### 第三段：合同订立的程序

一般情形下采用**一对一谈判性缔约**（要约+承诺，意思表示合意即成立）——公平但效率低。

商人采用**一对多竞争性缔约**——效率高。主要有三种：

| 方式 | 要约邀请 | 要约 | 承诺 | 合同成立时间 |
| :--- | :--- | :--- | :--- | :--- |
| **招标投标** | 招标公告 | 投标文件 | 中标通知书 | **中标通知书到达中标人时** |
| **现场拍卖** | 拍卖公告 | 出价 | 落槌 | **拍卖师落槌时** |
| **网络拍卖** | 拍卖公告 | 出价 | — | **电子交易系统确认成交时** |


### 第四段：要约的运行规则

#### （一）要约的生效

- **对话方式**：相对人**知道**其内容时生效
- **非对话方式**：**到达**相对人时生效（**到达主义**——到达你的可控范围即可，不管你是否看见）

#### （二）要约的撤回 vs 撤销

| 概念 | 时间节点 | 条件 | 效果 |
| :--- | :--- | :--- | :--- |
| **撤回** | 要约**生效前**（发出后→到达前） | 撤回通知**先于或同时**到达 | 阻止要约生效 |
| **撤销** | 要约**生效后**（到达后→承诺作出前） | 撤销通知在**承诺作出前**到达 | 使已生效的要约失效 |

**不得撤销的三种情形**（民法典第476条）：
1. **定期限**——确定了承诺期限（如“请于十日内答复”）
2. **有名式**——明示要约不可撤销
3. **有理由**——受要约人有理由认为要约不可撤销，并已为履行合同做了合理准备工作（交易习惯）

#### （三）要约的失效（民法典第478条）

1. **被拒绝**
2. **被依法撤销**
3. **承诺期限届满**，受要约人未作出承诺
4. **受要约人对要约的内容作出实质性变更**（砍价 → 实质性变更 → 原要约失效，产生新要约）


### 第五段：承诺的规则

**承诺**：受要约人**完全同意**要约的意思表示（有法律约束力）。

#### （一）承诺的方式

- 原则：**通知**
- 例外：**行为**（**履行抵销规则**——采用合同书形式订立合同，在签字或盖章前，当事人一方已经履行主要义务，对方接受的，合同成立）【补充：民法典第490条】。

> **例题**：买卖手表，你直接冲过来把钱拍在桌上，把手表拿走，我也没反对——合同成立（履行抵销规则）。

#### （二）承诺的撤回

承诺可以**撤回**（在到达要约人之前撤回），但**原则上不能撤销**（因为承诺到达时合同即成立）。

#### （三）承诺的迟延 vs 承诺的迟到（核心考点）

| 概念 | 原因 | 性质 | 法律效果 |
| :--- | :--- | :--- | :--- |
| **承诺的迟延** | **内因**（受要约人**自身原因**导致逾期） | 受要约人**有过错** | **视为新要约/反要约** |
| **承诺的迟到** | **外因**（第三人原因导致逾期） | 受要约人**无过错** | **承诺有效**（除非要约人及时表示反对） |

> **例题**：孟某的要约中载明“请于四月八日前回复”，周某四月九日才回复——这是**承诺的迟延**（内因，周某自己晚了），视为新要约。
>
> 如果孟某的要约载明“请于四月十五日前回复”，但由于邮政公司原因导致四月二十日才到达——这是**承诺的迟到**（外因），**承诺有效**。


### 第六段：意思表示（了解）

| 概念 | 含义 | 是否有法律约束力 |
| :--- | :--- | :--- |
| **意思表示** | 将内心意欲发生法律效果的意思表示于外部 | ✅ 有 |
| **表示**（要约邀请） | 无产生法律效果意图的表示 | ❌ 无 |

**默示表示**：
- **积极的默示**（行为）——履行抵销规则
- **消极的默示**（沉默）——**原则上不产生法律效果**，除非有法律规定、当事人约定或交易习惯（如法律拟制的“视为拒绝”）。


### 第七段：合同成立的时间和地点（了解）

| 合同类型 | 成立时间 |
| :--- | :--- |
| **要式合同**（书面形式） | **最后一方签章/按指印时** |
| **要物合同**（实践性合同） | **交付标的物时** |
| **非要式非要物合同** | **承诺到达时** |

**成立地点**：最后一方签章/按指印的地点为合同成立地。

> **例题**：张某和李某采用书面形式签订买卖合同。双方在甲地谈妥主要条款，张某于乙地签字，李某于丙地按指印，合同在丁地履行。合同签订地点为**丙地**。


### 第八段：合同的内容——条款

#### （一）必备条款（合同成立要件）

**三要素**：**当事人、标的物、数量**。数量最重要，法律无法对数量进行推定。

#### （二）主要条款——决定合同类型

价款/报酬决定合同类型（如价款为0→赠与，价款为正→买卖）。

#### （三）争议解决条款（主观题★★★★★）

**独立性**（民法典第507条）：合同不成立、无效、被撤销或确定不发生效力的，**不影响**合同中有关争议解决方法的条款的效力。

#### （四）格式条款（客观题★★★★★）

**格式条款**：当事人为了重复使用而预先拟定，并在订立合同时**未与对方协商**的条款。

**判断标准**：**有没有谈**。

- 不跟你谈的就是格式条款
- 手写条款 > 打印条款（格式条款提供方以“未实际重复使用”或“依据示范文本”为由否认的，法院不予支持）
- 明确约定“不是格式条款”的条款——**无效**（格式条款是强制性规定，不可约定排除）

**格式条款提供方的义务**：
1. **主动提示义务**——对**异常条款**（如争议解决条款）用文字、符号、字体等明显标识提示
2. **被动说明义务**——对概念、内容及法律后果进行解释说明

> **举证责任**：格式条款提供方承担（民法典第496条）。
>
> **电子合同**：仅以设置了“弹窗勾选”等方式主张履行了提示义务的，**不予支持**。

**违反提示说明义务的后果**：对方没有看到该条款 → 意思表示未合意 → 该条款**不成为合同的内容**（不成立）。

**格式条款无效的情形**（民法典第497条）：
1. 免除造成对方**人身损害**的责任（人身无价，免责无效）
2. 免除因**故意或重大过失**造成对方财产损害的责任
3. **不合理地**免除或减轻其责任、加重对方责任、限制对方主要权利（用日常生活经验判断）
4. **排除**对方主要权利（如排除起诉权、仲裁权、撤销权、解除权）

**不利解释规则**（民法典第498条）：对格式条款有两种以上解释的，应当作出**不利于提供格式条款一方的解释**。

> **例题（美容服务协议）** ：甲与乙公司订立美容服务协议，约定服务期半年，服务费预收后逐次计扣。协议格式条款中载明“如甲单方放弃服务，余款不退”。
>
> **问**：该条款是否有效？
>
> **答**：**无效**。该条款不合理地限制消费者权利。甲虽单方放弃服务构成违约，但不能以此为由完全不退余款。

> **例题（机票退票手续费）** ：甲买了一张北京飞广州的机票，原价2000元，打折价1000元。后甲临时有事没去，航空公司按“加收机票价格50%做手续费”扣款。航空公司认为“机票价格”指原价2000元，扣1000元；甲认为指实际支付的1000元，应退500元。
>
> **问**：应如何处理？
>
> **答**：适用**不利解释规则**，应作出不利于格式条款提供方（航空公司）的解释，**退500元**。


### 第九段：本专题总结

翻回第一百九十七页，合同的订立这个专题就一个选择题，四个考点：

1. **要约和要约邀请的区别**——重点看商业广告和宣传，内容是否具体确定
2. **合同订立的程序**——重点是一对多的竞争性缔约（招标投标、拍卖）
3. **合同成立的时间和地点**——了解
4. **合同内容**——重点掌握**争议解决条款**（独立性）和**格式条款**
