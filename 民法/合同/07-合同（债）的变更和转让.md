## 第1部分：对照《民法典》债的变更与转让提取备考提纲


### 一、合同的变更（民法典第543条、第544条）

| 要点 | 规则 |
| :--- | :--- |
| **定义** | 主体不变，内容变 |
| **条件** | **协商一致**（民法典第543条） |
| **内容约定不明** | **推定为未变更**（民法典第544条） |
| **与违约责任的关系** | 合同变更**不影响**违约责任（同意变更不等于同意免除违约责任） |


### 二、债权转让（民法典第545-550条）

#### （一）三个要件

| 要件 | 规则 |
| :--- | :--- |
| **合法有效的债权** | 不要求到期，已过诉讼时效的债权也可转让 |
| **具有可让与性** | 依据性质不得转让/当事人约定不得转让/法律规定不得转让 |
| **让与人与受让人达成协议** | 内部效力——协议生效即产生 |

#### （二）可让与性的特殊规则（民法典第545条）

| 债权类型 | 约定不得转让的效力 |
| :--- | :--- |
| **非金钱债权** | **不得对抗善意第三人**（第三人不知情的，可主张权利） |
| **金钱债权** | **不得对抗第三人**（即使第三人知情，也需保障货币流通性） |

#### （三）通知规则（民法典第546条）

- 债权转让**无需债务人同意**，仅需**通知**
- **未经通知，对债务人不发生效力**
- 通知人：让与人或受让人（受让人通知时需提供取得债权的证据）

#### （四）表见让与（从未考过★★★★★）

债权人将债权转让的事实通知债务人后，即使让与并未发生或让与无效（合同不成立、无效、被撤销或确定不发生效力），债务人基于对让与通知的信赖而向受让人履行的，依然有效——**债务人只需履行一次**。

#### （五）法律效果（民法典第547-549条）

**内部效力**（第547条）：受让人取得与债权有关的**从权利**（如担保物权），从权利随主权利一并转让，不因未办理转移占有或登记而受影响。

**抗辩权移转**（第548条）：债务人对让与人的抗辩，可以向受让人主张（最重要的是诉讼时效抗辩权）。

**抵销权**（第549条）：债务人可以向受让人主张抵销（情形一：债务人对让与人的债权先到期或同时到期；情形二：两个债权基于同一合同关系产生）。

**费用负担**（第550条）：因债权转让增加的履行费用，由**让与人**负担。


### 三、债务承担（民法典第551条、第552条）

#### （一）免责的债务承担（民法典第551条）

| 要点 | 规则 |
| :--- | :--- |
| **定义** | 原债务人**退出**法律关系 |
| **核心规则** | **需经债权人同意** |
| **债权人未作表示** | **视为不同意**（新增，从未考过） |
| **对担保人的影响** | 未经担保人**书面同意**，担保人**不再承担担保责任**（民法典第391条） |

#### （二）并存的债务承担（债务加入）（民法典第552条）

| 要点 | 规则 |
| :--- | :--- |
| **定义** | 原债务人**不退出**，第三人加入 |
| **核心规则** | **通知**即可（无需债权人同意） |
| **责任承担** | 第三人与债务人承担**连带债务** |


### 四、债权债务概括承受

| 情形 | 规则 | 法典依据 |
| :--- | :--- | :--- |
| **法人合并与分立** | 债权债务法定概括承受 | 民法典第67条 |
| **买卖不破租赁** | 所有权变动不影响租赁合同效力——新所有权人承受出租人权利义务 | 民法典第725条（**绝对重点**） |
| **限定继承** | 继承人仅在遗产范围内承担被继承人债务 | 民法典第1161条 |

### 第一段：合同的变更——主体不变，内容变

好，接下来咱们来看专题十七，叫做合同的变更和转让，又叫债的变更和转让。这个专题是客观题和主观题都会考察到的，特别是客观题每年必考事项。

先来看第一个问题，叫做合同的变更。合同的变更，它说的就是**主体不变，内容变**。合同的内容条款发生了变化。比如签订一个手机买卖合同，约定价格一万块钱，履行之前你对着我说“九千行吗”，我说“行”——价格从一万变九千，主体没变，内容变了，双方一致同意。

**合同变更的条件**：重点是**协商一致**，绝对不能单方变更。单方变更合同构成违约要承担违约责任。

**核心考点**：合同变更**不影响违约责任**（民法典第543条）。即使同意变更合同内容（如变更履行期限），也不等于同意免除对方的违约责任。

> **例题**：开发商与购房人约定2025年1月1日交房，但开发商交不了房，双方协商将交房日期变更为2025年8月1日。8月1日交房后，购房人主张迟延交房的违约金。开发商抗辩“你同意变更合同了”。
>
> **问**：开发商要不要承担违约责任？
>
> **答**：**要承担**。购房人同意变更交房日期，并不等于同意免除违约责任。

**当事人对合同变更内容约定不明的，推定为未变更**（民法典第544条）。


### 第二段：债权转让——卖债权

**债权转让**，又叫债权让与，几乎每年必考事项。它主要解决的是**银行处置不良资产**的问题，也应用于保理等领域。

**本质**：债权转让就是卖债权或卖债权凭证（购物券、月饼券、电影票等）。


### 第三段：债权转让的构成要件

**要件一：存在合法有效的债权**——债权**不要求到期**，未到期的债权可以转让，已过诉讼时效的债权也可以转让（民法典第545条）。

**要件二：被让与的债权具有可让与性**（民法典第545条，核心考点）

哪些债权不能转让？

1. **根据债权性质不得转让**——具有**人身专属性**的债权（如劳务之债、竞业禁止协议、专门为特定人画肖像的合同等）
2. **当事人约定不得转让**——重点！需区分**金钱债权 vs 非金钱债权**
   - **非金钱债权**（财务之债、劳务之债）：当事人约定不得转让，**不得对抗善意第三人**。即第三人不知情的，可主张权利；第三人知情的，债务人可拒绝履行。
   - **金钱债权**（货币之债）：当事人约定不得转让，**不得对抗第三人**（即使第三人知情）。因为货币的流通性优先（民法典第545条第2款）。
3. **法律规定不得转让**——如文物等。

**要件三：让与人与受让人达成债权转让协议**——协议一经达成，即在二者之间产生内部效力。

**要件四：通知债务人**（民法典第546条，核心要件）

债权转让原则上仅需**通知**债务人，不需债务人同意。通知人可以是原债权人（让与人），也可以是受让人（受让人通知时需提供取得债权的证据）。

**未经通知，该转让对债务人不发生效力**（外部效力不产生）。


### 第四段：债权转让的特殊问题——表见让与（从未考过★★★★★）

**表见让与**：债权人将债权转让的事实通知债务人后，即使让与并未发生或让与无效（合同不成立、无效、被撤销或确定不发生效力），债务人基于对让与通知的信赖而向受让人履行，依然有效。

> **核心原理**：债务人很被动，他没有参与债权转让协议的签订，不能因为债权人和受让人之间的合同效力瑕疵而受到损害。保护债务人的履行利益，债务人只需要履行一次。


### 第五段：债权转让的法律效果

**内部效力**：受让人取得与债权有关的**从权利**（如利息债权、担保物权），从权利随主权利一并转让，**不因未办理转移占有或登记手续而受影响**（民法典第547条）。

**外部效力**：

1. **债务人的抗辩权移转**（民法典第548条）：债务人接到债权转让通知后，债务人对让与人的抗辩，可以向受让人主张。最重要的是**诉讼时效抗辩权**——债权转让不得使债务人的处境恶化。

2. **债务人的抵销权**（民法典第549条）：债务人可以向受让人主张抵销。
   - 情形一：债务人接到债权转让通知时，债务人对让与人享有债权，且债务人的债权**先于转让的债权到期或同时到期**。
   - 情形二：债务人的债权与转让的债权基于**同一合同关系**产生（如租赁合同中的租金债权和维修费债权）。

> **租赁合同抵销例题**：孟某将房屋出租给马某，租期内冰箱坏了，孟某拒绝维修，马某自行修理花费2000元。后马某欠孟某租金1万元。孟某将1万元租金债权转让给曹某并通知马某。曹某向马某主张1万元时，马某可向曹某主张抵销2000元。
>
> **分析**：两个债权均基于同一房屋租赁合同产生。债权转让不得使债务人的处境恶化——在没有转让之前，马某本来就可以对孟某主张抵销，转让后当然也可以对曹某主张。


### 第六段：债权转让 vs 债权代位权诉讼（区分重点）

| 对比项 | 债权转让 | 债权代位权诉讼 |
| :--- | :--- | :--- |
| **是否有合同** | ✅ 有债权转让协议（当事人约定） | ❌ 无法定债权转让协议 |
| **原债权人是否退出** | ✅ **退出**法律关系（全部转让时） | ❌ **不退出**法律关系 |
| **第三人的诉讼地位** | **可以**追加为无独立请求权第三人 | **应当**追加为无独立请求权第三人 |


### 第七段：债务承担——免责的债务承担

**债务承担**，又叫债务转移。

**免责的债务承担**：原债务人**退出**法律关系，将债务全部或部分转移给第三人（民法典第551条）。

**核心规则：需经债权人同意**（同意规则，不是通知规则）。

> **为什么需要同意？** 债务人的信用和履行能力对债权人影响巨大——把债务从“老实巴交的孟某”转让给“要饭的马某”，对债权人影响巨大。

**协议签订的三种方式**：
1. 债务人与第三人签订债务承担协议 → 需**事后征得债权人同意**
2. 债权人与第三人直接签订债务承担协议 → 债权人**签字行为表明同意**
3. 三方共同签订协议 → 债权人签字表明同意

**债权人未作表示的，视为不同意**（新增，从未考过）。

**担保人的影响**（民法典第391条）：债权人允许债务人转移全部或部分债务的，**未经担保人书面同意**，担保人**不再承担相应的担保责任**。


### 第八段：债务承担——并存的债务承担（债务加入）

**并存的债务承担（债务加入）** ：原债务人**不退出**法律关系，第三人加入进来共同承担债务（民法典第552条）。

**核心规则：不需要债权人同意，仅需通知**（因为对债权人有利——多了一个债务人）。

**两种方式**：
1. 第三人与债务人签订债务加入协议 → **通知债权人**
2. 第三人直接向债权人表示愿意加入债务 → 债权人未在合理期限内明确拒绝

**责任承担**：第三人在其愿意承担的债务范围内和债务人承担**连带债务**。

> **债务加入 vs 第三人代为履行的区别**：
> - **债务加入**：有协议，第三人成为合同当事人（连带债务人）
> - **第三人代为履行**：无协议，第三人不是合同当事人（不加入法律关系）


### 第九段：债权债务的概括承受（了解）

**债权债务概括承受**：债权和债务一并转让（民法典第67条）。

**常考形式**：
1. **法人的合并与分立**——债权债务法定概括承受
2. **买卖不破租赁**（民法典第725条，**绝对重点**）：租赁物在承租人按照租赁合同占有期限内发生所有权变动的，**不影响租赁合同的效力**——新的所有权人承受出租人的权利和义务。
3. **限定继承**（民法典第1161条）：继承人仅在遗产范围内承担被继承人的债务。
