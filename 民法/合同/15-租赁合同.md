## 第1-1部分：对照《民法典》租赁合同提取备考提纲


### 一、租赁合同的定义与性质（民法典第703条）

| 性质 | 说明 |
| :--- | :--- |
| **有偿合同** | 承租人必须支付租金 |
| **诺成合同** | 意思表示合意即成立 |
| **负担行为** | 只移转使用权，不发生物权变动 |


### 二、租赁合同的期限与形式（民法典第705条、第707条）

- **最长期限**：不得超过**二十年**；超过部分**无效**（第705条）
- **租期六个月以上**：必须采用**书面形式**（第707条）
- **租期不满六个月**：可书面可口头（不要式）


### 三、备案登记（民法典第706条）

备案登记**不影响**租赁合同的效力（行政管理目的）。


### 四、风险负担（民法典第729条）

- 租赁物因不可归责于承租人的事由毁损、灭失的，承租人可请求**减少租金**或**解除合同**
- **出租人承担风险**（所有权主义）


### 五、出租人的义务——维修义务（民法典第712条、第713条）

- **有约从约**
- **无约出租人修**
- **承租人导致的自行修**（第713条第2款，新增，从未考过）


### 六、承租人的义务

| 义务 | 法典依据 | 核心规则 |
| :--- | :--- | :--- |
| **禁止添附** | 第715条 | 未经同意不得改善或增设他物；违反可请求恢复原状或赔偿损失 |
| **禁止转租** | 第716条、第718条 | 未经同意转租构成根本违约，出租人可解除合同 |
| **支付租金** | 第722条 | 迟延支付须经**催告**后方可解除合同（法定催告义务） |


### 七、承租人的权利

| 权利 | 法典依据 | 核心规则 |
| :--- | :--- | :--- |
| **法定解除权** | 第731条 | 危及安全或健康可随时解除（人身价值优先） |
| **买卖不破租赁** | 第725条 | 所有权变动**不影响**租赁合同效力（公示在先原则） |
| **优先购买权** | 第726条 | 仅限房屋；同等条件；四个例外 |
| **续租权** | 第732条 | 共同居住人/共同经营人可继续租赁 |
| **优先承租权** | 第734条 | 仅限房屋；同等条件（新增） |

## 第1-2部分：对照《民法典》租赁合同提取备考提纲


### 一、转租（民法典第716条、第718条、第719条）

#### （一）合法转租（经出租人同意）

| 要点 | 规则 |
| :--- | :--- |
| **租金差价** | 不构成不当得利（有法律上的原因） |
| **次承租人的合法利益** | 次承租人可**代为履行**欠付租金（第719条） |
| **出租人起诉次承租人** | 可主张侵权、恢复原状，**不能主张违约责任**（无合同） |
| **承租人起诉次承租人** | 可主张**违约责任**（有转租合同） |

#### （二）擅自转租（未经出租人同意）

- 构成**根本违约**，出租人可**解除**与承租人的租赁合同（第716条）
- 出租人知道或应当知道承租人转租，但在**六个月内**未提出异议的，视为同意转租（第718条）


### 二、无效租赁合同

| 情形 | 规则 | 补正 |
| :--- | :--- | :--- |
| **违法建筑** | 租赁合同**无效** | 一审法庭辩论终结前取得合法手续的可补正 |
| **临时建筑超期** | 超期部分**无效** | 一审法庭辩论终结前取得延期手续的可补正 |


### 三、一房数租（排序规则）

**合法占有 > 备案登记 > 合同成立在先**（最高人民法院城镇房屋租赁合同纠纷司法解释第5条）


### 四、装饰装修与扩建（司法解释第9-14条）

| 行为 | 有无同意 | 租赁期满后处理 |
| :--- | :--- | :--- |
| **装饰装修** | 经同意 | 有约从约，无约**不得请求补偿** |
| **装饰装修** | 未经同意 | 出租人可请求**恢复原状**或**赔偿损失** |
| **扩建** | 经同意 + 合法手续 | 费用由**出租人**负担 |
| **扩建** | 经同意 + 无合法手续 | 按**过错**分担 |
| **扩建** | 未经同意 | 费用由**承租人**负担，可要求恢复原状 |


### 五、不定期租赁合同（民法典第707条、第730条、第734条）

| 产生方式 | 法典依据 |
| :--- | :--- |
| 租期六个月以上未采用书面形式，又无法确定租期 | 第707条 |
| 对租期没有约定或约定不明，又不能协商确定 | 第730条 |
| **租赁期限届满，承租人继续使用，出租人未提出异议** | 第734条（**考最多**） |

**法律效果**：双方当事人均享有**任意解除权**，但应当在**合理期限之前通知**对方（第730条）。

### 第一段：租赁合同概述

好，接下来咱们来看专题二十五，租赁合同。这个专题也是客观和主观都会考察的，在旁边写上**主观题**。

租赁合同和前面的买卖合同、赠与合同最大的区别就是：它只是**移转标的物使用权**的合同，它不发生物权变动，仅发生一个债权债务关系，是个最典型的**负担行为**。

**租赁合同的定义**（民法典第703条）：出租人将租赁物交付承租人使用、收益，承租人支付租金的合同。

重点要强调的是租赁合同**一定是有偿合同**。如果住人家房子不要钱，那就变成了**借用合同**了。


### 第二段：租赁合同的期限与备案

**租期限制**（民法典第705条）：租赁合同最长期限不得超过**二十年**。超过二十年的，**超过部分无效**（民法典第156条部分无效的应用）。

> **注意**：居住权可以有期限，但**不受二十年限制**（可能超过二十年）；租赁合同的租期，一次性的租期最长不得超过二十年。到期后可以重新订立一个新的租赁合同。

**备案登记**（民法典第706条）：备案登记制度只是一个行政管理目的，不影响合同效力。老百姓没有去办理房屋租赁合同备案登记，确实违法，但**违法不一定无效**。备案登记不影响合同效力。

> **备案登记例题**：甲将房屋出租给乙，租期三年，未办理租赁合同备案登记。
>
> **问**：租赁合同是否有效？
>
> **答**：**有效**。备案登记不影响合同效力（民法典第706条）。


### 第三段：租赁合同的形式与风险负担

**合同形式**（民法典第707条）：
- **租期不满六个月**：可以书面，也可以口头（不要式合同）
- **租期六个月以上**：必须采用**书面形式**（要式合同）

**风险负担**（核心区别★★★★★）：

| 合同类型 | 风险负担规则 | 法典依据 |
| :--- | :--- | :--- |
| **买卖合同** | **交付主义**（交付前出卖人承担，交付后买受人承担） | 第604条 |
| **租赁合同** | **所有权主义**（**出租人**承担风险） | 第729条 |

**原因**：买卖合同移转标的物所有权（发生物权变动）；租赁合同只移转使用权（不发生物权变动）。标的物是谁的，谁承担风险。

> **租赁合同风险负担例题**：我把房出租给你，租期三年，月租金一万块，你一次性付清36万。你住了第一年没事，第二年北京洪水房子没了。
>
> **问**：剩下的租金我要不要退？
>
> **答**：**要退**。出租人承担风险（所有权主义）。


### 第四段：出租人的义务——维修义务（核心考点）

出租人的义务中，最重要的就是**维修义务**（民法典第712条、第713条）。

**三句话规则**：

1. **有约从约**：租赁合同明确约定由谁维修，就按约定
2. **无约出租人修**：出租人负有法定的维修义务（标的物是出租人的）
3. **承租人导致的自行维修**：因承租人过错导致租赁物损坏的，承租人自己修（**新增，从未考过**，民法典第713条第2款）

> **承租人导致损坏例题**：承租人租住房屋期间，与女朋友打架，女朋友拿刀砍人，承租人躲闪，刀砍坏了门和玻璃。
>
> **问**：谁修？
>
> **答**：**承租人修**。承租人自己导致的损坏，自己承担。


### 第五段：承租人的义务——禁止添附与禁止转租

#### （一）禁止添附（民法典第715条）

承租人未经出租人同意，不得对租赁物进行改善或增设他物（贴墙纸、贴瓷砖、装空调等）。未经同意的，出租人可以请求承租人**恢复原状**或**赔偿损失**。

#### （二）禁止转租（民法典第716条、第718条，核心考点★★★★★）

承租人经出租人同意，可以将租赁物转租给第三人。

**承租人未经出租人同意转租的**——构成**根本违约**，出租人可以**解除合同**（解除的是出租人与承租人之间的合同，不是承租人与次承租人的合同）。

> **禁止转租例题**：我把房出租给你，租期三年，合同没有允许你转租。你在第二年擅自转租给马峰。
>
> **问一**：我能否解除合同？
>
> **答**：**能**。未经同意转租构成根本违约（法定解除权）。
>
> **问二**：我解除的是哪个合同？
>
> **答**：解除的是**我和你的合同**（合同相对性原理）。你与马峰的合同因基础丧失而无法继续履行。


### 第六段：承租人的义务——支付租金

承租人**无正当理由未支付或迟延支付租金**的，出租人可以请求承租人在**合理期限内支付**；承租人逾期不支付的，出租人可以**解除合同**（民法典第722条）。

**必须履行法定催告义务**（民法典第563条第3项）：不能承租人一迟延支付租金就立刻赶人走，必须经催告后在合理期限内仍不支付，才能解除合同。


### 第七段：承租人的权利（一）——法定解除权

承租人的五大权利中，考试重点是三个绿色的。

**法定解除权**（民法典第731条）：租赁物危及承租人的安全或者健康的，即使承租人**订立合同时明知**该租赁物质量不合格，承租人仍然可以**随时解除合同**。

> **人身价值高于财产价值例题**：我把刚装修完的房子租给你，你明知房屋刚装修可能有质量问题仍同意租住。住了一个月后你皮肤红肿出疹，头晕目眩。
>
> **问**：你能解除合同吗？
>
> **答**：**能**。即使你订立合同时明知质量不合格，仍可随时解除（人身价值高于财产价值）。


### 第八段：承租人的权利（二）——买卖不破租赁（民法典第725条，核心考点★★★★★）

租赁物在承租人按照租赁合同**占有期限内**发生所有权变动的，**不影响**租赁合同的效力。

**核心规则：公示在先原则**——看**入住（交付）** 和**登记**谁在先，谁在先保护谁。

| 情形 | 规则 |
| :--- | :--- |
| **先租后卖**（承租人入住在前，所有权变动在后） | **买卖不破租赁**（承租人无需搬出） |
| **先卖后租**（所有权变动在前，承租人入住在后） | **买卖破租赁**（承租人需搬出，追究出租人违约责任） |

> **租赁 vs 买卖不破租赁例题**：甲公司与乙公司签订租赁合同，约定将一台机器出租给乙公司，约定**一周后交付**（未交付，乙公司没有使用利益）。租赁合同成立次日，甲公司就同一台机器与丙公司订立买卖合同，约定以**占有改定**方式交付（丙公司已取得所有权）。一周后，甲公司将机器交付给乙公司使用。
>
> **问**：乙公司能否主张买卖不破租赁？
>
> **答**：**不能**。公示在先原则——丙公司的占有改定交付在先，乙公司的实际交付在后。丙公司取得所有权，乙公司只能追究甲公司的违约责任。
>
> **答案：B、C、D**——B不适用买卖不破租赁；C丙公司已取得机器所有权；D乙公司有权请求甲公司承担违约责任。


### 第九段：承租人的权利（三）——房屋承租人的优先购买权（主观题重点★★★★★）

房屋承租人在出租人出卖租赁房屋时，在**同等条件**下享有优先购买权（民法典第726条）。

**一个原则，四个例外**——考试考例外：

| 例外情形 | 说明 |
| :--- | :--- |
| **1. 房屋共有人行使优先购买权** | 物权优先于债权（共有人优先于承租人） |
| **2. 出租人将房屋卖给近亲属** | 情感因素优先（八类近亲属） |
| **3. 承租人在15日内未明确表示购买** | 视为放弃（法律拟制） |
| **4. 第三人善意购买并已办理登记手续** | 买卖合同有效，出租人赔偿承租人损失 |

> **优先购买权排序例题**：孟某、马某按20%和80%份额共有一间房屋，二人将房屋出租给曹某，曹某经同意后转租给徐某。现孟某欲转让自己的份额。
>
> **问**：马某、曹某、徐某优先购买权的顺序？
>
> **答**：**马某（共有人，物权）> 徐某（次承租人，实际居住）> 曹某（承租人）**。价值位阶：物权效力 > 居住利益 > 一般债权。

> **优先购买权受侵害例题**：甲与乙订立房屋租赁合同，半年后甲将该出租房屋出售给丙，**未通知乙**。乙以优先购买权受到侵害为由，请求法院判决甲丙间的房屋买卖合同无效。
>
> **问**：下列哪一表述正确？
>
> A. 甲出售房屋无需通知乙——**错**（需通知，诚信原则）
>
> B. 丙有权根据善意取得规则取得房屋所有权——**错**（甲是有权处分，与善意取得无关）
>
> C. 甲侵犯了乙的优先购买权，但甲丙间的合同有效——**正确**
>
> D. 甲出售房屋应当征得乙的同意——**错**（只需通知，无需同意）
>
> **答案：C**


### 第十段：承租人的权利（四）——优先承租权（民法典第734条，新增）

租赁期限届满，房屋承租人享有在**同等条件**下优先承租的权利。

> **注意**：优先承租权也是在**同等条件**下行使（如租金、支付方式等）。如果别人出更高的租金或更优的支付条件，承租人无法主张优先承租。


### 第十一段：续租权（民法典第732条）

承租人在租赁期间死亡的，与其生前**共同居住的人**或者**共同经营的人**可以按照原租赁合同继续租赁该房屋。

> **续租权例题**：我把房出租给张三，张三与妻子、儿子共同居住。租期内张三死亡。
>
> **问**：张三的妻子和儿子能否继续居住？
>
> **答**：**能**（共同居住人享有续租权）。


### 第十二段：承租人的五大权利总结

翻回三百一十三页，承租人的五大权利：

| 权利 | 适用标的物 | 核心规则 |
| :--- | :--- | :--- |
| **法定解除权** | 所有租赁物 | 根本违约；人身安全可随时解除 |
| **买卖不破租赁** | 所有租赁物 | 公示在先原则（入住 vs 登记） |
| **优先购买权** | 仅**房屋** | 同等条件；四个例外 |
| **续租权** | 房屋（共同居住/经营人） | 承租人死亡后继续租赁 |
| **优先承租权** | 仅**房屋** | 同等条件（新增） |

### 第一段：转租概述

好，接下来咱们来看第四个问题，叫做转租。

转租就两种模式：第一种是经过出租人同意的，叫**合法转租**；第二种是没有征得出租人同意的，叫**擅自转租**，又叫**违法转租**。考试的重点先是这个**擅自转租**。

转租是指出租人将租赁房屋租给承租人，承租人又把承租房租给次承租人的租赁法律关系。前一个租赁关系叫**本租**，后一个租赁关系叫**转租**。


### 第二段：合法转租

先来看三百一十七页，第一种经过出租人同意的转租，就是合法转租。这里边重点就是**坚守合同相对性原理**。

> **合法转租模型**：2024年3月2日，孟某将房屋出租给马某，租期五年，月租金1万元。一年后，马某**经过孟某同意**将房屋转租给曹某，租期四年，月租金1.5万（未超过剩余租期）。马某是二房东，每月赚5000元差价。2025年3月6日，曹某**擅自将房屋的一堵墙拆除**。
>
> **第一问**：孟某同意转租后，能否向马某每月收取1.5万租金？
>
> **答**：**不可以**。基于合同相对性，孟某和马某的租金是1万，不是1.5万。
>
> **第二问**：马某每月多收的5000元差价租金是否构成不当得利？
>
> **答**：**不构成**。因为征得了出租人同意，有法律上的原因（合法转租），不符合不当得利的构成要件。
>
> **第三问（重点）** ：如马某欠付孟某两个月租金2万元，孟某欲解除与马某的房屋租赁合同，曹某如何救济自己的权利？
>
> **答**：曹某可以**代为履行**（替马某还钱），因为曹某作为次承租人具有**合法利益**（民法典第719条）。这就是**第三人代为履行抗辩权**——曹某代马某支付欠付的租金，以阻止孟某行使解除权。代为履行后，曹某可以找马某追偿。
>
> **第四问**：如孟某欲直接起诉曹某，共有几种选择？
>
> **答**：三种救济途径——①告曹某**侵权**（侵犯房屋所有权）；②请求**恢复原状**（物权请求权）；③侵权+恢复原状并用。**不能追究违约责任**，因为孟某与曹某之间无合同（合同相对性）。
>
> **第五问**：如孟某欲起诉马某，共有几种选择？
>
> **答**：三种救济途径——①告马某**违约**；②请求**恢复原状**；③违约+恢复原状并用。**不能追究侵权责任**，因为墙不是马某拆的。
>
> **第六问**：马某承担责任后，能否向曹某主张权利？
>
> **答**：**可以**。马某基于与曹某之间的合法有效的转租合同追究曹某的**违约责任**。


### 第三段：擅自转租（违法转租）

翻过来三百一十八页，考的更多的肯定是违法转租，也叫擅自转租。

核心要点：出租人对于承租人有一个**选择权**。承租人擅自转租构成**根本违约**，出租人可以行使**解除权**。基于合同相对性，出租人只能解除与承租人的合同，不能解除承租人与次承租人的合同。

> **擅自转租模型**：2024年3月2日，孟某将房屋出租给马某，租期五年，月租金1万元。一年后，马某**擅自**将房屋转租给曹某（未经孟某同意），租期四年，月租金1.5万。2025年3月6日，曹某擅自将房屋一堵墙拆除。3月10日，孟某发现，欲请求曹某搬离房屋。
>
> **第一问**：孟某是否有权解除与马某的租赁合同？
>
> **答**：**有权**。根据民法典第716条，承租人未经出租人同意转租的，构成根本违约，出租人可以解除合同。
>
> **第二问**：孟某解除与马某的租赁合同后，是否可以请求曹某搬离房屋？
>
> **答**：**可以**。一旦孟某解除与马某的合同，马曹之间的转租合同就无法继续履行（标的物要被孟某收回）。
>
> **第三问**：如曹某拒不搬离，孟某如何救济？
>
> **答**：可以在两项债权请求权中**择一选择**——①主张**占有使用费**（不当得利）；②主张**侵权责任**。注意：这里不能叫“租金”，因为租赁合同已被解除，只能叫“占有使用费”。
>
> **第四问**：曹某如何救济自己的权利？
>
> **答**：曹某基于合法有效的转租合同追究马某的**违约责任**。


### 第四段：无效租赁合同的情形

租赁合同一般情况下都是有效的，但何时无效？——看**标的物**。

**情形一：违法建筑**（民法典第153条，最高人民法院城镇房屋租赁合同纠纷司法解释第2条、第3条）：出租人就未取得建设工程规划许可证或未按照建设工程规划许可证规定建设的房屋，与承租人订立的租赁合同**无效**。

**效力瑕疵的补正**：在一审法庭辩论终结前取得合法手续的，可以认定有效（**相对无效**）。

**情形二：临时建筑**：经批准修建的临时建筑，在核准使用期限内租赁有效；超出核准使用期限的租赁合同**无效**（司法解释第3条）。

> **无效租赁合同例题**：居民甲经主管部门批准修建了一排**临时门面房**，核准使用期限为两年。甲将其中一间租给乙开餐馆，租期两年（在核准期内，租赁合同**有效**）。期满后未办理延长使用期限手续，甲又将该房屋出租给丙，并签订了一年的租赁合同（超出核准期，租赁合同**无效**）。
>
> **问**：下列哪些说法正确？
>
> A. 甲与乙的租赁合同无效——**错**（有效）
>
> B. 甲与丙的租赁合同无效——**正确**（违法建筑）
>
> C. 甲无权将该房屋继续出租给丙——**正确**（违法建筑不能出租）
>
> D. 甲无权向丙收取该年的租金——**正确**（租金前提是租赁合同合法有效，只能主张占有使用费）
>
> **答案：B、C、D**


### 第五段：一房数租——排序

一房数租，谁先获得租赁权？**三步排序**（最高人民法院城镇房屋租赁合同纠纷司法解释第5条）：

1. **已经合法占有**租赁房屋的（先入住/先交付）→ **第一位**
2. 都未占有的，**已经办理备案登记**的 → **第二位**
3. 都未占有也未备案的，**合同成立在先**的 → **第三位**

> **一房数租例题**：刘某欲出租房屋，2016年5月与孟某签订合同A，租期一年，孟某**随即入住**（合法占有）。2016年9月，刘某又与马某签订合同B，租期一年，并与马某办理了**备案登记**手续。
>
> **问**：下列哪些选项是正确的？
>
> A. 孟某因合法占有而具有优先权——**正确**（先下手为强）
>
> B. A合同虽然未办理备案登记，但仍然有效——**正确**（备案登记不影响合同效力）
>
> C. A合同因成立在先而有优先权——**错**（不是因为成立在先，而是因为合法占有）
>
> D. B合同因备案而具有优先权——**错**（备案排第二，合法占有排第一）
>
> **答案：A、B**


### 第六段：装饰装修和扩建

#### （一）装饰装修

承租人经出租人同意装饰装修的，租赁期间届满时，对装饰装修物的处理：**有约从约，无约不得请求出租人补偿**。

> **装饰装修例题**：我把房出租给你，租期三年。你说墙发黄想贴墙纸，我说“你贴呗”。你贴了38万的墙纸。三年租期届满，你要求我补偿38万。
>
> **问**：我是否应当补偿？
>
> **答**：**不需要**。有约从约，无约不得请求出租人补偿。正常情况下没人会贴这么贵的墙纸，就算我同意，我也没想到你会贴这么贵的。

#### （二）扩建

承租人经出租人同意扩建，但双方对扩建费用没有约定的：
- **办理合法建设手续的**：费用由**出租人**负担
- **未办理合法建设手续的**：费用由双方按照**过错**分担

> **扩建例题**：甲将其邻街房屋和院子出租给乙作为汽车修理场所，**经甲同意**，乙先后两次自费扩建多间房屋作为汽车间。乙在又一次扩建报批过程中发现甲出租的全部房屋均**未经过城市规划部门批准，属于违章建筑**（租赁合同无效）。
>
> **问**：下列哪些说法正确？
>
> A. 租赁合同无效——**正确**（违法违章建筑）
>
> B. 因甲乙对于扩建房屋都有过错，应分担扩建房屋的费用——**正确**（双方都有过错）
>
> C. 因甲未告知乙租赁物为违章建筑，乙可解除租赁合同——**错**（无效合同不存在解除权问题）
>
> D. 乙可继续履行合同——**错**（无效合同不存在继续履行问题，只能承担缔约过失责任）
>
> **答案：A、B**


### 第七段：不定期租赁合同（民法典第707条、第730条、第734条）

不定期租赁合同有三种产生方式：

1. **租期六个月以上，未采用书面形式，又无法确定租期的**（民法典第707条）——视为不定期租赁
2. **对租期没有约定或约定不明确，事后又不能协商确定的**（民法典第730条）——视为不定期租赁
3. **租期届满，承租人继续使用，出租人没有提出异议的**（民法典第734条）——原租赁合同继续有效，但为**不定期租赁**（考最多）

**不定期租赁的法律效果**：双方均有**任意解除权**。但出租人解除的，应当在**合理期限之前通知**承租人（给搬家时间）。

> **不定期租赁例题**：我把房出租给你，租期三年。三年研究生毕业了，你没搬走，我每个月照常收租金，你每个月照常给租金，我不反对。
>
> **问**：此时是什么租赁合同？
>
> **答**：**不定期租赁合同**（租期届满后，以行为方式表明继续租赁）。双方均有任意解除权。


### 第八段：租赁合同专题总结

翻回三百零九页，租赁合同的体系：

| 考点 | 核心规则 |
| :--- | :--- |
| **最长期限** | 二十年，超过部分无效 |
| **备案登记** | 不影响合同效力（违法不一定无效） |
| **风险负担** | **所有权主义**（出租人承担） |
| **维修义务** | 三句话：有约从约，无约出租人修，承租人导致的自己修 |
| **转租** | 合法转租 vs 擅自转租（擅自转租=根本违约，出租人可解除） |
| **无效情形** | 违法建筑、临时建筑超期；一审法庭辩论终结前可补正 |
| **一房数租** | 合法占有 > 备案登记 > 合同成立在先 |
| **装饰装修** | 有约从约，无约不得请求补偿 |
| **扩建** | 经同意+合法手续→出租人负担；经同意+无合法手续→按过错分担 |
| **不定期租赁** | 双方均有任意解除权；租期届满继续使用为最常见情形 |
