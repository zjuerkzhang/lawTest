# 债的分类

## 第1部分：对照《民法典》债的分类提取备考提纲


### 一、债的分类体系总览

| 分类标准 | 类型 | 法典依据 |
| :--- | :--- | :--- |
| **发生原因** | 法定之债、意定之债、自然之债 | — |
| **主体人数** | 单一之债、多数人之债 | 民法典第517-520条 |
| **标的物性质** | 财务之债、劳务之债、货币之债 | 民法典第579条 |
| **标的物选择** | 简单之债、选择之债 | 民法典第515-516条 |


### 二、法定之债（民法典第三编第三分编）

#### （一）无因管理之债（民法典第979-984条）

**构成要件**（第979条第1款）：
1. 没有法定或约定的义务
2. 为避免他人利益受损失而管理他人事务
3. 不违反本人明示或可推知的意思（例外：违反法律或公序良俗的不影响）

**管理人的权利**（第979条第1款）：
- 请求受益人偿还**必要费用**
- 因管理事务受到损失的，请求受益人给予**适当补偿**

**管理人的义务**（第981-983条）：
- 方法有利，不得擅自中断
- 及时通知、等待指示
- 及时转交和报告

**管理人的报酬**：**无报酬请求权**（无因管理是学雷锋，不能要报酬）

#### （二）不当得利之债（民法典第985-988条）

**构成要件**（第985条前段）：
1. 一方获得利益
2. 他方受到损失
3. 获益与受损之间有因果关系
4. 没有法律根据

**不当得利返还的排除**（第985条但书）：
1. 为履行道德义务进行的给付
2. 债务到期之前的清偿
3. 明知无给付义务而进行的债务清偿

**返还范围**：
- **善意得利人**（第986条）：仅返还**现存利益**
- **恶意得利人**（第987条）：返还**所受利益**，利益不存在的**赔偿损失**
- **无偿转让的第三人**（第988条）：相应范围内承担返还义务

**返还内容**：原物及**孳息**（天然孳息 + 法定孳息）


### 三、自然之债

| 类型 | 法典依据 | 核心规则 |
| :--- | :--- | :--- |
| **已过诉讼时效之债** | 民法典第192条 | 债权存在但丧失强制执行力；自愿履行不得请求返还 |
| **赌债等违法之债** | — | 无请求权；已给付不得请求返还 |
| **彩礼** | 彩礼纠纷司法解释 | 根据“登记+共同生活”综合判断 |
| **限定继承之债** | 民法典第1161条 | 继承人仅在遗产范围内承担清偿责任 |


### 四、意定之债——单方允诺之债（悬赏广告）

**悬赏广告**（民法典第499条）：悬赏人以公开方式声明对完成特定行为的人支付报酬的，完成该行为的人可以请求其支付。

**性质**：法考采**单方允诺说**——悬赏广告一经发出即发生法律效力。

**特点**：
- 完成行为的人**无须知晓**悬赏广告的存在
- **无民事行为能力人**亦可主张报酬


### 五、多数人之债（民法典第517-520条）

| 类型 | 核心规则 | 法典依据 |
| :--- | :--- | :--- |
| **按份之债** | 各债务人按份额负担债务；各债权人按份额享有债权 | 民法典第517条 |
| **连带之债** | 债权人可请求任一债务人履行全部债务；**对外连带、对内按份** | 民法典第518-520条 |


### 六、依标的物分类的债（民法典第579条）

| 类型 | 特点 | 核心规则 |
| :--- | :--- | :--- |
| **特定之债** | 标的物不可替代 | 交付前毁损灭失 → **免除继续履行义务** |
| **种类之债** | 标的物可替代 | 交付前毁损灭失 → **不免除继续履行义务** |
| **劳务之债** | 人身依附性 | **不得请求直接强制执行** |
| **货币之债** | 金钱给付 | **金钱之债永不消灭** |


### 七、选择之债（民法典第515-516条）

**选择之债**：标的有多项而债务人/债权人可选择其一履行【补充：民法典第515条】。

**选择权性质**：**形成权**（单方意思表示即可产生法律效果）【补充：民法典第516条】。

### 第一段：债法总则概述

好，接下来我们来看第三部分，叫债法总则。它和后边的第四部分合同编结合在一起，在我们的整个客观题考试当中一共占**二十分**左右。

首先，我们来看书的一百七十一页专题十一，叫债的基本原理。这个专题分成了两个小节：**债的法定分类**和**债的学理分类**。考试重点考的是第一节——**债的法定分类**。【修正：原文“债的学历分类”】

债权根据发生原因不同分成了三类：**约定之债、法定之债和自然之债**。


### 第二段：自然之债——四种类型

先来看书的一百七十二页的**自然之债**。自然之债就是既不是当事人约定的，也不是法律规定的。

#### （一）已过诉讼时效之债（民法典第192条）

已过诉讼时效之债，债权本身依然存在。如果债务人提了诉讼时效抗辩权，法院将判决驳回债权人的诉讼请求；如果债务人不提，法院将判决债权人胜诉。如果债务人自愿履行，不构成不当得利。

#### （二）赌债等违法之债

赌债、赌资、嫖资等违法之债——如果输钱没给，不能起诉要求给付；如果给了，也不能起诉要求返还。

#### （三）彩礼（2025年刚考过★★★★★）

彩礼根据当地风俗习惯产生。判断是否返还，核心看两个要素：**是否办理结婚登记**、**是否共同生活**。

| 情形 | 规则 |
| :--- | :--- |
| **未办登记 + 未共同生活** | 原则上**可以**请求返还 |
| **未办登记 + 已共同生活** | 法院根据实际情况综合判断，**不一定全额返还** |
| **已办登记 + 未共同生活** | 原则上**可以**请求返还（以离婚为前提） |
| **已办登记 + 已共同生活** | 原则上**不予支持**返还；但共同生活时间**较短**且彩礼数额**过高**的，法院可酌情确定返还比例 |

**彩礼 vs 赠与的区分**（《最高人民法院关于审理涉彩礼纠纷案件适用法律若干问题的规定》第3条第2款）：
- **彩礼**：以缔结婚姻为目的给付的大额财物 → 可能返还
- **赠与**：节日、生日等特殊纪念意义时点给付的**价值不大**的礼物礼金，或为表达增进感情的**日常消费性支出** → **不返还**

#### （四）限定继承之债——超出遗产范围的债务（民法典第1161条）

**限定继承**：继承人必须在继承遗产的范围内清偿被继承人的债务，超出的部分变为**自然之债**。继承人愿意还就还，不愿意还就不用还；如果还了，不能基于不当得利要回来。

> **例题**：甲的父亲去世，留有1000万元遗产，甲是唯一继承人。乙告知甲，其父亲生前尚欠自己借款2000万元。
>
> **分析**：1000万元遗产范围内，甲必须偿还；超出的1000万元是**自然之债**，甲愿意还就还，不愿意还就不用还。债权人不得强制执行甲的**个人财产**。


### 第三段：单方允诺之债（2024年考过）

**单方允诺之债**最典型的是**悬赏广告**（民法典第499条）。

悬赏广告的性质——**法考采单方允诺说**：只要悬赏人公开声明完成特定行为支付报酬，完成行为的人即可请求支付报酬。

**单方允诺说的优势**（有利于完成行为的人）：
1. **无民事行为能力人亦可主张报酬**——六岁小孩捡到狗狗送回来，也要给报酬
2. **不知悬赏广告存在亦可主张报酬**——事后才看到悬赏广告，也可以要求支付

> **例题（方某行李案）** ：方某将一行李遗落在出租车上，立即发布寻物启事，言明愿意2000元现金酬谢返还行李者。出租车司机李某发现该行李及寻物启事后与方某联系，现方某拒绝支付2000元给李某。
>
> **分析**：方某对李某享有**物权请求权**（返还行李），李某对方某享有**债权请求权**（2000元酬金）。物权优先于债权，李某应先返还行李，再起诉要求支付酬金。只要方某发布了悬赏广告，就必须支付酬金。

**其他单方允诺之债**：
- 假一罚十的承诺
- 自然之债的同意履行（同意履行多少范围内形成单方允诺之债）


### 第四段：无因管理之债（法定之债★★★★★）

**无因管理** = 学雷锋做好事、见义勇为、助人为乐——**没有法定或约定的义务，为避免他人利益受损失而管理他人事务**。

#### 构成要件（民法典第979条）

| 要件 | 内容 |
| :--- | :--- |
| **无因** | 没有法定或约定的义务 |
| **管理意思** | 主观上具有为避免他人利益受损而管理他人事务的意思 |
| **管理行为** | 客观上实施了管理他人事务的行为 |
| **不违反本人意思** | 不违反本人明示或可推知的意思（例外：违反法律或公序良俗的不影响） |

**不构成无因管理的情形**：
- **误信管理**：误把他人事务当成自己的事务而管理（如误把他人的牛当成自己的牛饲养）——不构成无因管理，但可主张不当得利
- **图谋私利**：纯粹为了自己的利益——不构成无因管理
- **好意施惠**：顺便帮忙扫雪、顺路投递邮件——不构成无因管理

**主观上兼顾为自己利益**——**不影响**无因管理的构成（如“恐殃及池鱼”而救火）【补充：民法典第979条】

**无因管理是“要行为不要效果”** ——有没有效果不重要，有没有救人的行为才重要。不能要求每一个雷锋都是专业人员。

**不当无因管理**（违反本人意思的事务管理）——应当承担赔偿责任。

> **例题（卖鱼案）** ：甲正在市场卖鱼，突闻其父病危离去。邻摊菜贩乙代为叫卖，以比甲原价高出5元的价格卖出200斤，多卖1000元据为己有；后将余下100斤以每斤3元低价卖出，亏了700元。
>
> **分析**：多拿的1000元构成**不当得利**，亏损的700元构成**不当管理**应赔偿。乙应当把多拿的钱还回去，还得赔700元。

**法律效果（管理人的权利）** ：
- ✅ **必要费用请求权**（如打车送医的费用）
- ✅ **所负债务请求权**（如垫付的医药费）
- ✅ **所受损害请求权**（如救人时受伤的医疗费）
- ❌ **没有报酬请求权**（无因管理是学雷锋，不能要报酬）

**管理人的义务**：
- **方法有利，不得擅自中断**——要么别管，要么好好管（不能用祈祷代替送医）
- 及时通知、等待指示
- 及时转交和报告


### 第五段：不当得利之债（法定之债★★★★★）

**不当得利** = 人在屋中坐，财从天上来——**不劳而获**。

#### 构成要件（民法典第985条）

| 要件 | 内容 |
| :--- | :--- |
| **一方获益** | 得利人取得了利益 |
| **一方受损** | 受损失的人受到了损失 |
| **因果关系** | 获益与受损之间有因果关系 |
| **无法律根据** | 没有法律上的原因 |

#### 不当得利返还的排除（民法典第985条但书）

| 排除情形 | 举例 | 是否构成不当得利 |
| :--- | :--- | :--- |
| **为履行道德义务进行的给付** | 养子女对生父母给付生活费 | ❌ 不构成 |
| **债务到期之前的清偿** | 提前还清未到期债务 | ❌ 不构成 |
| **明知无给付义务而进行的债务清偿** | 替人买单（明知没有义务） | ❌ 不构成 |

**误认的非债清偿**（与“明知的非债清偿”不同）——**构成不当得利**。

> **例题（夫妻共同债务）** ：甲乙是夫妻，共同欠丙1万元。甲还了1万元但未告知乙，乙误以为甲没还，又还了1万元。丙共收到2万元。
>
> **分析**：丙构成**不当得利**，应当返还多收的1万元本金及利息。

#### 法律效果——返还范围

| 类型 | 返还范围 |
| :--- | :--- |
| **善意得利人** | 仅返还**现存利益**（不知情且无过错，不承担风险）【补充：民法典第986条】 |
| **恶意得利人** | 返还**所受利益**，利益不存在的**赔偿损失**【补充：民法典第987条】 |

**返还内容：原物及孳息**（天然孳息如小牛小羊，法定孳息如利息）。


### 第六段：债的学理分类（了解即可）

#### （一）单一之债 vs 多数人之债

- **单一之债**：债权人和债务人各为一人
- **多数人之债**：债权人或债务人一方为二人以上

**多数人之债**向下分为：
- **按份之债**：各债权人/债务人按份额享有/负担债权债务（有约从约）【补充：民法典第517条】
- **连带之债**：债权人可以请求任何一个债务人履行全部债务（对外连带、对内按份）【补充：民法典第518条】

> **连带之债的核心规则**：对外**连带**，对内**按份**，人人有份。

#### （二）财务之债、劳务之债、货币之债（看标的物）

| 类型 | 特点 | 核心考点 |
| :--- | :--- | :--- |
| **特定之债**（财务之债的子类） | 标的物**独一无二**，不可替代 | 交付前毁损灭失 → **免除继续履行义务**，只能赔钱 |
| **种类之债**（财务之债的子类） | 标的物**具有可替代性** | 交付前毁损灭失 → **不免除继续履行义务** |
| **劳务之债** | **人身依附性** | **不得请求直接强制执行**（如演出合同、授课合同）【补充：民法典第579条】 |
| **货币之债** | 金钱之债 | **永不消灭**，欠债还钱天经地义【补充：民法典第579条】 |

#### （三）简单之债 vs 选择之债

- **简单之债**：标的物只有一个，无可选择
- **选择之债**：标的物有两个以上，可选择其一【补充：民法典第515条】

**选择权是形成权**（单方意思表示即可产生法律效果）。


### 第七段：本节总结

**法定分类（第一节）** ——每年必考，轮番考察：
- 2024年：**单方允诺之债**（悬赏广告）
- 2025年：**自然之债**（彩礼）
- 2026年预测：**无因管理**和**不当得利**之间挑一个

**学理分类（第二节）** ——了解即可，重点掌握：
1. **连带之债**——对外连带、对内按份（十二字规则）
2. **财务之债 vs 劳务之债 vs 货币之债**——特别是特定之债和劳务之债
3. **简单之债 vs 选择之债**——了解即可
