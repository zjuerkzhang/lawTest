# 合同的履行

## 第1部分：对照《民法典》合同履行相关条款提取备考提纲


### 一、电子合同的履行（民法典第491条、第512条）

| 标的物类型 | 交付/履行时间 | 法典依据 |
| :--- | :--- | :--- |
| **商品** | **签收时间**为交付时间 | 第512条第1款 |
| **服务** | **凭证载明的时间**；不一致的以**实际提供服务的时间**为准 | 第512条第2款 |
| **在线传输**（电子书等） | **进入对方指定系统且能够检索识别时** | 第512条第3款 |

**合同成立时间**：提交订单成功时（第491条第2款）。

**合同形式**：网购为**书面形式**（第469条第3款）。


### 二、涉他合同（民法典第522条、第523条）

#### （一）向第三人履行的合同（利他合同，第522条）

| 类型 | 要件 | 第三人权利 | 法典依据 |
| :--- | :--- | :--- | :--- |
| **不真正利他合同** | 无“直接请求”约定 | 受领权、拒绝受领权（2个） | 第522条第1款 |
| **真正利他合同** | 有“**直接请求**”约定 | 受领权、拒绝受领权 + **独立的履行请求权** + **独立的违约责任请求权**（4个） | 第522条第2款 |

**抗辩权的移转**：债务人对债权人的抗辩，可以向第三人主张（债的同一性原理）。

**第三人的权限**：第三人**无权撤销或解除**合同（第三人不是合同当事人）。

#### （二）由第三人履行的合同（第523条）

第三人不履行债务或履行不符合约定的，**债务人应当向债权人承担违约责任**。

**第三人只有义务，没有权利**。


### 三、第三人代为履行（民法典第524条；合同编通则司法解释第30条）

**构成要件**：
1. 合同**未约定**第三人具有履行义务
2. 债务人不履行债务
3. 第三人对履行该债务具有**合法利益**
4. 根据债务性质、合同约定或法律规定，未明确排除第三人代为履行

**具有合法利益的第三人**：
- 次承租人（转租中代付租金）
- 母公司（为子公司代偿）
- 抵押物买受人（代偿抵押债务以消灭抵押权——抵除权）
- 近亲属（配偶、父母、子女等）

**法律效果**：第三人代为履行后，发生**法定债权转让**——债权人对债务人的债权**法定转让**给第三人，第三人可以向债务人追偿。

**债权人的拒绝权**：第三人对履行该债务具有合法利益的，**债权人无正当理由不得拒绝**（民法典第524条第2款）。


### 四、双务合同履行抗辩权（民法典第525-528条）

| 抗辩权 | 适用情形 | 行使主体 | 法典依据 |
| :--- | :--- | :--- | :--- |
| **同时履行抗辩权** | 无先后顺序 / 约定同时履行 | 双方均可 | 第525条 |
| **先履行抗辩权** | 有先后顺序，后履行一方先被请求 | **后履行一方** | 第526条 |
| **不安抗辩权** | 有先后顺序，先履行一方有**确切证据**证明对方可能无法履行 | **先履行一方** | 第527条、第528条 |

**不安抗辩权的程序**：中止履行 → **及时通知**对方（诚信原则）→ 对方在合理期限内未恢复履行能力且未提供适当担保的 → 视为以行为表明不履行主要债务 → 可**解除合同**并请求**承担违约责任**。

**抗辩范围**：全部不履行→全部抗辩；部分不履行→部分抗辩（相应抗辩）。

**性质**：三大抗辩权均为**暂时性抗辩权**（延期抗辩权），不同于诉讼时效抗辩权（永久性抗辩权）。


### 五、情势变更（民法典第533条）——主观题重点

**构成要件**：
1. 合同**成立后**，履行过程中
2. 合同基础条件发生**无法预见的、不属于商业风险的重大变化**
3. 继续履行对一方**明显不公平**

**处理程序**：
1. 受不利影响的一方与对方**重新协商**（前置程序）
2. 合理期限内协商不成的 → 请求**人民法院或仲裁机构变更或解除合同**

**限制规则**：
- 当事人**均主张变更**合同的 → 法院**不得解除**合同
- 当事人**约定排除**情势变更适用的 → **该约定无效**

**情势变更 vs 商业风险**：
- 商业风险：价格**正常**波动 → 风险自担
- 情势变更：价格**异常**波动 → 可主张情势变更

**情势变更 vs 不可抗力**：
- 不可抗力（第180条）：不能预见、不能避免且不能克服 → **免责事由**
- 情势变更（第533条）：不能预见、不属于商业风险 → **变更或解除合同**
- 二者可以重合（如新冠疫情）

| **情势变更** | **✅ 主观题** | **2022年考过** | 给出“合同履行中价格异常波动”的场景，问能否变更或解除合同 | 合同履行中原材料价格暴涨4倍，继续履行对卖方明显不公平。问：卖方如何救济？（先协商，协商不成请求法院变更或解除合同） |

| **情势变更——排除约定无效** | 客观题 | 偶有考查 | 给出“合同约定排除情势变更适用”的条款，问效力 | 合同中约定“无论发生什么都按原价履行”。问：该约定是否有效？（无效，排除情势变更的约定无效） |

> **老师原话总结**：
> - “涉他合同的履行——向第三人履行的合同，**每年必考**。记住‘给女朋友送花’的模型：**2个权利还是4个权利？看‘直接请求’四个字**。”
> - “第三人代为履行的本质——**无约定、主动履行、合法利益、法定债权转让**。”
> - “第三人代为履行与‘由第三人履行’最大的区别——**有没有约定**。”
> - “双务合同三大抗辩权——**同时履行抗辩权、先履行抗辩权、不安抗辩权**，都是**暂时性抗辩权**。”
> - “情势变更——**两个不**：不能预见、不属于商业风险。**先协商，协商不成请求变更或解除**。”

### 第一段：合同的履行概述

好，接下来咱们来看专题十五，合同的履行，又叫债的履行。我们前边给大家提到过，咱们国家的民法典因为没有债法总则这一编，所以民法典合同编通则担负着债法总则的功能。债的一些共通的东西都在这一编当中。但是从考试的角度，我们并不会重点考察债的履行，而重点考的其实就是合同的履行。合同的履行这个专题是客观题和主观题都会考察到的，所以旁边拿笔写上**主观题**。

这个专题当中最难的问题就是有很多的“第三人”——向第三人履行、由第三人履行、第三人代为履行。每一个第三人在不同的语境之下意思不一样，解决的问题也不一样。很多同学容易混淆，所以要一个一个地识别它的本质区别。


### 第二段：电子合同的履行（网购）

先来看第一个，**电子合同的履行**（网购）【补充：民法典第491条、第512条】。

网购和我们前面讲过的要约与承诺没什么区别，唯一的区别是正常交易在线下，网购在线上通过互联网进行。

**成立时间**：消费者选择商品或服务并**提交订单成功时**，合同成立（承诺）。

**合同形式**：网购是**书面形式**的合同。书面形式不限于一张纸，包括能够有形地表现所载内容、并可以随时调取查用的数据电文。

**履行时间**（看标的物）：
1. **商品**：**签收时间**为交付时间（不是拆快递的时间）【补充：民法典第512条第1款】
2. **服务**：**凭证中载明的时间**为准；实际提供服务时间与凭证不一致的，以**实际提供服务的时间**为准【补充：民法典第512条第2款】
3. **在线传输**（电子书等）：**进入对方指定的特定系统且能够检索识别时**为交付时间【补充：民法典第512条第3款】

> **例题（网购石斛案）** ：孟某在天猫商城从A公司购买石斛风斗11件，单价390元，促销价138元，实际付款1518元。收货人及地址为孟先生，快递方式为圆通。
>
> **问**：下列哪些说法正确？
>
> A. 孟某与A公司所订合同系采用书面形式——**正确**（网购为书面形式）
>
> B. A公司发布的商品信息构成要约，孟某提交订单构成承诺——**正确**
>
> C. 孟某提交订单成功时合同成立——**正确**
>
> D. 孟某签收时间为A公司交付标的物的时间——**正确**（商品交付以签收时间为准）
>
> **答案：A、B、C、D**


### 第三段：涉他合同的履行（一）——向第三人履行的合同（核心考点）

第二个问题，**涉他合同的履行**，这是基本上每年必考事项。

涉他合同涉及第三人。根据合同相对性的不同，合同分为**束己合同**和**涉他合同**。

**束己合同**：租赁合同——大房东与二房东有合同，二房东与次承租人有合同，但大房东与次承租人没有合同关系。打死我也不能追究次承租人的违约责任，因为没合同，无约可为。

**涉他合同**又分为两种：
1. **向第三人履行的合同**（利他合同）——给第三人创设**权利**（民法典第522条）
2. **由第三人履行的合同**——给第三人创设**义务**（民法典第523条）

考试重点肯定是**向第三人履行的合同**。

#### （一）不真正利他合同 vs 真正利他合同

**判断标准**：看合同约定中有没有 **“直接请求”** 四个字。

| 类型 | 有无“直接请求” | 第三人的权利 | 法典依据 |
| :--- | :--- | :--- | :--- |
| **不真正利他合同** | ❌ 无 | **2个权利**：受领权、拒绝受领权（要还是不要） | 第522条第1款 |
| **真正利他合同** | ✅ 有 | **4个权利**：受领权、拒绝受领权 + **独立的履行请求权** + **独立的违约责任请求权** | 第522条第2款 |

> **买花例题（不真正利他合同）** ：孟某与美心花店签订订花合同，约定每天给女生宿舍的刘某送一束百合花。合同中没有“刘某可以直接请求”的约定。
>
> **分析**：合同当事人是孟某和花店，刘某是第三人。刘某只有2个权利——要还是不要。如果花店不送花，只有孟某能追究花店违约责任，刘某不能。如果孟某不给钱，花店只能追究孟某违约责任，不能追究刘某。

> **买花例题（真正利他合同）** ：孟某与花店约定：“因孟某即将离开北京，**刘某可以直接请求**花店向其交付百合花。”
>
> **分析**：有“直接请求”四个字 → 真正利他合同。刘某有4个权利：要、不要、独立的履行请求权、独立的违约责任请求权。但刘某**无权撤销或解除**孟某与花店的合同（第三人不是合同当事人，不能决定合同生死）。

**抗辩权的移转**：在真正利他合同中，债务人对债权人的抗辩，可以向第三人主张（基于债的同一性原理）。

#### （二）新增考点：离婚协议中给子女财产（民法典婚姻家庭编司法解释二第20条）——从未考过

> **模型**：张三和李四离婚，在离婚协议中约定将婚后共同共有的房屋给儿子小张。小张就是合同以外的第三人。
>
> **判断**：看离婚协议中有没有“子女可以**直接主张权利**”的约定。
>
> - **无“直接主张权利”** → 不真正利他合同 → 小张只有2个权利（要还是不要）→ 如果张三不配合过户，由**李四**去告张三
>
> - **有“直接主张权利”** → 真正利他合同 → 小张有4个权利 → 如果张三不配合过户，**小张**可以直接告张三（独立的履行请求权/违约责任请求权）


### 第四段：涉他合同的履行（二）——由第三人履行的合同

**由第三人履行的合同**（民法典第523条）：当事人约定由第三人向债权人履行债务。第三人不履行债务或者履行债务不符合约定的，**债务人应当向债权人承担违约责任**。

这个第三人**只有义务，没有权利**。

> **例题（由第三人履行）** ：孟某请客吃饭，跟餐馆约定“一会儿马某来结账”。
>
> **分析**：如果马某不来结账，餐馆只能找孟某要钱——因为合同当事人是孟某和餐馆，马某是第三人。这就是合同相对性原理。

> **例题（连环买卖）** ：孟某将手表卖给马某（第一个合同），马某将手表转卖给曹某（第二个合同），转卖合同中约定“由孟某负责交付手表给曹某”。
>
> **分析**：
> - 从马某和曹某的**转卖合同**角度看——**由第三人履行的合同**（孟某是第三人，只有义务没有权利）
> - 从孟某和马某的**买卖合同**角度看——**向第三人履行的合同**（曹某是第三人，只有权利没有义务）
> - 如果孟某未向曹某交付手表：曹某只能追究**马某**的违约责任（合同相对性），马某再追究孟某的违约责任

**设他合同 vs 指示交付的区别**（物债两分区分原则）：
- **设他合同**：标的物的所有人和占有人**未发生分离**，只是债权债务关系 → 追究违约责任
- **指示交付**：标的物的所有人和占有人**已发生分离**（自己所有、他人占有），发生物权变动


### 第五段：第三人代为履行（核心考点★★★）

**第三人代为履行**（民法典第524条）：第三人在**无合同约定**的情况下**自愿**代债务人履行债务。

**与“由第三人履行”的最大区别**：由第三人履行是**合同约定**的；第三人代为履行是**无约定**、第三人**主动**的。

#### （一）构成要件

1. **无约定**（合同未约定第三人具有履行义务）——与“由第三人履行”区分的核心
2. **债务人不履行债务**
3. **第三人对履行该债务具有合法利益**（核心考点）
4. **根据债务性质、合同约定或法律规定，未明确将第三人代为履行排除在外**

**具有合法利益的第三人**（民法典第524条；合同编通则司法解释第30条）：

| 类型 | 示例 | 说明 |
| :--- | :--- | :--- |
| **次承租人** | 转租中，次承租人代付租金 | 防止大房东解除租赁合同导致次承租人被赶走 |
| **母子公司** | 母公司为子公司代偿债务 | 关联关系，子公司债务可能影响母公司信用 |
| **抵押物买受人** | 买受人代偿抵押债务（抵除权） | 消灭抵押权以取得抵押物所有权 |
| **近亲属** | 配偶、父母、子女等代偿 | 情感利益 |

**不具有合法利益的第三人**（不能代为履行）：竞争对手故意代偿以破坏对方财务计划或商业安排。

#### （二）法律效果——法定债权转让

第三人代为履行后，**债权人对债务人的债权法定转让给第三人**。第三人取得原债权人的地位，可以向债务人追偿。

> **例题（次承租人代为履行）** ：孟某将房屋出租给马某，马某转租给曹某。马某欠孟某两个月租金及违约金共计5万元，孟某欲解除租赁合同。曹某（次承租人）为避免被赶走，代马某支付了5万元。
>
> **问一**：曹某是否有权代为履行？
>
> **答**：**有权**。曹某是次承租人，具有合法利益。
>
> **问二**：孟某能否拒绝曹某代为履行？
>
> **答**：**不能**。曹某具有合法利益，孟某无拒绝权。
>
> **问三**：曹某代付后，对马某是否有追偿权？
>
> **答**：**有**。法定债权转让，曹某取得原债权人孟某的地位，可以向马某追偿5万元。曹某可以抵充应当向马某支付的租金。


### 第六段：双务合同履行中的抗辩权（常考点）

双务合同（典型：买卖合同）中，双方互为权利人和义务人。三大抗辩权都是**暂时性抗辩权**（延期抗辩权）——一旦对方履行，抗辩权消灭。

#### （一）三大抗辩权的区分

| 抗辩权 | 适用情形 | 行使主体 | 法典依据 |
| :--- | :--- | :--- | :--- |
| **同时履行抗辩权** | 无先后履行顺序 / 约定同时履行（一手交钱一手交货） | 双方均可 | 第525条 |
| **先履行抗辩权**（顺序履行抗辩权） | 有先后履行顺序，**后履行一方**先请求对方履行 | **后履行一方** | 第526条 |
| **不安抗辩权** | 有先后履行顺序，**先履行一方**有确切证据证明对方可能无法履行 | **先履行一方** | 第527条 |

#### （二）三大抗辩权详解

**1. 同时履行抗辩权**（民法典第525条）

适用三种情形：
- 合同约定同时履行（“一手交钱，一手交货”）
- 合同未约定先后履行顺序 → 推定同时履行
- 合同无效/被撤销后的恢复原状（民法典第157条）——同时返还

**2. 先履行抗辩权**（民法典第526条）

后履行一方对先履行一方的抗辩。先履行一方未履行或履行不符合约定时，后履行一方有权拒绝其履行请求。

**3. 不安抗辩权**（民法典第527条、第528条）

先履行一方有**确切证据**证明对方存在以下情形时，可以中止履行：
- 经营状况严重恶化
- 转移财产、抽逃资金，以逃避债务
- 丧失商业信誉
- 有丧失或可能丧失履行债务能力的其他情形

**程序**：中止履行 → **应当及时通知对方**（诚信原则）→ 对方在合理期限内未恢复履行能力且未提供适当担保的，视为以自己的行为表明不履行主要债务 → 中止履行一方可以**解除合同**并请求**承担违约责任**

**抗辩的范围**：全部不履行 → 全部抗辩；部分不履行 → 部分抗辩（**相应抗辩**）。

> **提示**：法考共考五个抗辩权——①诉讼时效抗辩权；②同时履行抗辩权；③先履行抗辩权；④不安抗辩权；⑤一般保证人的先诉抗辩权。赠与合同中的穷困抗辩权了解即可。


### 第七段：情势变更（民法典第533条，主观题重点★★★）

**情势变更**：合同成立后，合同的基础条件发生了当事人在订立合同时**无法预见的、不属于商业风险的重大变化**，继续履行合同对当事人一方**明显不公平**的，受不利影响的当事人可以与对方重新协商；合理期限内协商不成的，可以请求人民法院或仲裁机构**变更或解除合同**。【补充：民法典第533条】

**情势变更 vs 显失公平**：
- **显失公平**：合同**订立时**就明显不公平 → 可撤销（民法典第151条）
- **情势变更**：合同**履行过程中**发生重大变化导致不公平 → 可变更或解除

**情势变更 vs 商业风险**：
- **商业风险**：价格**正常**波动（如菜价涨跌、房价涨跌、股票涨跌）→ 风险自担
- **情势变更**：价格**异常**波动（如原材料价格暴涨数倍）→ 可主张情势变更

**情势变更的处理**（民法典第533条）：
1. 先**协商**（前置程序）
2. 协商不成 → 请求法院或仲裁机构**变更或解除合同**
3. 当事人**均主张变更**合同的 → 法院**不得解除**合同（鼓励交易原则）
4. 当事人**约定排除**情势变更适用的 → **该约定无效**（司法强制干预契约自由）

**情势变更的典型情形**：国家政策调整、市场价格异常波动或涨跌。

**不可抗力 vs 情势变更**：
- **不可抗力**（民法典第180条）：不能预见、不能避免且不能克服的客观情况 → 免责事由
- **情势变更**（民法典第533条）：不能预见、不属于商业风险的重大变化 → 变更或解除合同
- 二者可以重合（如新冠疫情，立法机关认为属不可抗力，最高人民法院认为属情势变更）

> **例题（2022年真题）** ：甲公司与乙公司签订长期供货合同，后因地震导致原材料成本和运输费用均上涨，若继续履行合同甲公司将遭受重大损失。甲公司与乙公司协商变更合同被拒。
>
> **问**：下列哪一说法正确？
>
> A. 合同因重大误解可撤销——**错**（重大误解是合同订立时的问题）
>
> B. 合同因不可抗力可撤销——**错**（不可抗力不是可撤销事由）
>
> C. 合同因显失公平可撤销——**错**（显失公平是合同订立时的问题）
>
> D. 合同因情势变更可变更——**正确**


### 第八段：本专题总结

翻回二百零七页，合同的履行这个专题五个问题，除了电子合同履行以外，剩下的几个都是重点：

1. **涉他合同的履行——向第三人履行的合同**（每年必考）：记住“给女朋友送花”的模型。第三人到底是2个权利还是4个权利？看合同中有没有 **“直接请求”** 四个字。今年重点关注**离婚协议给子女财产**的新增考点。

2. **第三人代为履行**：本质是**无约定、主动履行、合法利益、法定债权转让**。与“向第三人履行”“由第三人履行”最大的区别是**没有约定**。

3. **双务合同履行抗辩权**：三大抗辩权都是**暂时性抗辩权**。考的可能性最大的是**同时履行抗辩权**。主观题也可能考察。

4. **情势变更**：合同履行中的公平问题。**两个不**——不能预见、不属于商业风险。
