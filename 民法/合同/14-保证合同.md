

## 1-1部分：对照《民法典》保证合同提取备考提纲


### 一、保证合同概述（民法典第681条）

**保证合同**：为保障债权的实现，保证人和债权人约定，当债务人不履行到期债务或者发生当事人约定的情形时，保证人履行债务或者承担责任的合同。


### 二、保证合同的性质

| 性质 | 说明 |
| :--- | :--- |
| **单务合同** | 保证人只有义务无权利，债权人只有权利无义务 |
| **无偿合同** | 保证人和债权人之间无对价 |
| **要式合同** | 必须采用**书面形式**（民法典第685条） |


### 三、保证人的限制（民法典第683条）

**不得为保证人**：
1. **机关法人**（例外不考）
2. **以公益为目的的非营利法人、非法人组织**（公立的学校、医院、养老院等）


### 四、保证方式的四种书面形式（民法典第685条）

1. 单独的书面保证合同
2. 主合同中的保证条款
3. 主合同中无保证条款，第三人以保证人身份签字
4. 第三人**单方以书面形式**向债权人作出保证，债权人**接收且未提出异议**


### 五、一般保证（民法典第687条）

**识别**：当事人约定“债务人**不能履行**债务时，由保证人承担保证责任”。

**先诉抗辩权**：一般保证人有权要求先就债务人财产强制执行，在执行不能前**拒绝**承担保证责任。

**丧失先诉抗辩权的情形**（债务人客观上没钱了）：
1. 债务人下落不明，且无财产可供执行
2. 法院已受理债务人破产案件
3. 债权人有证据证明债务人财产不足以履行全部债务或丧失履行能力
4. 保证人书面放弃先诉抗辩权


### 六、连带责任保证（民法典第688条）

**识别**：当事人约定保证人和债务人对债务承担**连带责任**。

**特征**：无顺序限制，债权人可任意选择债务人或保证人承担责任，保证人**无先诉抗辩权**。


### 七、保证方式的推定（民法典第686条第2款）

保证方式**没有约定或约定不明确的**，按照**一般保证**承担保证责任。

## 第1-2部分：对照《民法典》保证合同提取备考提纲


### 一、保证范围（民法典第691条）

保证的范围包括**主债权及其利息、违约金、损害赔偿金和实现债权的费用**。当事人另有约定的，按照其约定。


### 二、变更的从属性（民法典第695条）

| 变更类型 | 是否加重保证人责任 | 是否需要保证人书面同意 |
| :--- | :--- | :--- |
| **增加主债权数额** | ✅ 加重 | **需要**（未经同意的，对加重部分不承担责任） |
| **减少主债权数额** | ❌ 减轻 | **不需要**（对变更后的债务承担责任） |
| **延长履行期限** | ✅ 加重 | **需要**（未经同意的，对延长部分不承担责任） |
| **缩短履行期限** | ❌ 减轻 | **不需要** |


### 三、债权转让的从属性（民法典第696条）

- **原则**：债权人转让全部或者部分债权，**通知**保证人即可，保证人在原保证范围内继续承担保证责任
- **例外**：保证人与债权人约定**禁止债权转让**的，未经保证人书面同意，保证人**不再承担保证责任**


### 四、债务转让的从属性（民法典第697条）

**免责的债务承担**（债务人退出）：
- **双重同意**：债权人同意（不要式）+ 保证人**书面同意**（要式）
- 未经保证人书面同意，保证人对**转移的债务部分不再承担保证责任**

**并存的债务承担**（债务加入）：
- 第三人加入债务，保证人的责任**不受影响**（无需保证人同意）


### 五、保证期间（民法典第692条）

> **第692条**：保证期间是确定保证人承担保证责任的期间，**不发生中止、中断和延长**。债权人与保证人可以约定保证期间；没有约定或者约定不明确的，保证期间为主债务履行期限届满之日起**六个月**。

**性质**：**除斥期间**（不变期间）

**起算**：主债务履行期限**届满之日**

**视为约定不明**：约定“保证人承担保证责任直至本息还清为止” → 视为约定不明 → 仍为**六个月**


### 六、保证期间届满的法律后果（民法典第693条）

| 保证方式 | 债权人应在保证期间内 | 未行权的后果 |
| :--- | :--- | :--- |
| **一般保证** | 对债务人**提起诉讼或者申请仲裁** | 保证人**不再承担保证责任** |
| **连带责任保证** | **请求**保证人承担保证责任 | 保证人**不再承担保证责任** |


### 七、保证债务诉讼时效（民法典第694条）

| 保证方式 | 诉讼时效起算点 |
| :--- | :--- |
| **一般保证** | 从保证人**拒绝承担保证责任的权利消灭之日**（即丧失先诉抗辩权之日）起开始计算 |
| **连带责任保证** | 从债权人**请求保证人承担保证责任之日**起开始计算 |

## 第1-3部分：对照《民法典》保证合同与共同担保提取备考提纲


### 一、保证人的权利（民法典第700条、第701条、第702条）

| 权利 | 法典依据 | 说明 |
| :--- | :--- | :--- |
| **追偿权** | 第700条 | 保证人承担保证责任后，有权向债务人追偿 |
| **法定代位权** | 第700条 | 保证人享有债权人对债务人的权利（**新增，从未考过**） |
| **抗辩权** | 第701条 | 保证人享有主债务人的抗辩权（包括诉讼时效抗辩权） |
| **抵销权/撤销权** | 第702条 | 债务人享有的抵销权/撤销权，保证人也可以主张 |


### 二、共同保证

| 类型 | 认定标准 | 外部关系 | 内部关系 |
| :--- | :--- | :--- | :--- |
| **按份共同保证** | 保证人与债权人**约定了份额** | 按保证方式（一般/连带）确定 | 按约定份额承担，互不追偿 |
| **连带共同保证** | 保证人与债权人**未约定份额** | 按保证方式（一般/连带）确定 | 各保证人连带，可互相追偿 |

**核心区分**：
- **连带责任保证** = 保证人与**债务人**连带（有/无先诉抗辩权）
- **连带共同保证** = 保证人与**保证人**连带（各保证人之间）


### 三、共同物保（民法典担保制度解释第45条）

| 情形 | 起诉时 | 执行时 |
| :--- | :--- | :--- |
| **债务人提供物保 + 第三人物保** | 应当将债务人和担保人作为**共同被告** | **应当先执行**债务人物保（类推民法典第392条） |
| **均为第三人物保** | 应当将债务人和担保人作为**共同被告** | **无顺序限制**，债权人有选择权 |


### 四、混合担保（民法典第392条）

> **第392条**：被担保的债权既有物的担保又有人的担保的，债务人不履行到期债务或者发生当事人约定的实现担保物权的情形，债权人应当按照约定实现债权；没有约定或者约定不明确的，**债务人自己提供物的担保的，债权人应当先就该物的担保实现债权**；第三人提供物的担保的，债权人可以就物的担保实现债权，也可以请求保证人承担保证责任。

**顺序规则**：有约从约 → 债务人物保优先 → 第三人物保/人保随意选择

## 第1-4部分：对照《民法典》混合担保、共同担保、反担保提取备考提纲


### 一、混合担保（民法典第392条）

> **第392条**：被担保的债权既有物的担保又有人的担保的，债务人不履行到期债务或者发生当事人约定的实现担保物权的情形，债权人应当按照约定实现债权；没有约定或者约定不明确的，**债务人自己提供物的担保的，债权人应当先就该物的担保实现债权**；第三人提供物的担保的，债权人可以就物的担保实现债权，也可以请求保证人承担保证责任。

**核心规则**：
- **有约从约**
- **无约定** → 债务人物保**优先**（应当先执行）
- 第三人物保与保证**并列**（无顺序限制）


### 二、共同担保人之间的追偿权（担保制度解释第13条）

| 情形 | 有无追偿权 | 理由 |
| :--- | :--- | :--- |
| **互相不知对方存在**（背对背） | ❌ **无** | 无意思联络 |
| **按份共同担保**（约定了份额） | ❌ **无** | 有约从约，各自承担 |
| **担保人之间约定了追偿** | ✅ **有** | 意思自治 |
| **连带共同担保**（未约定份额） | ✅ **有** | 各担保人连带 |
| **在同一份合同上签章**（面对面） | ✅ **有** | 互相知道对方存在 |


### 三、反担保（民法典第689条；担保制度解释第19条）

**反担保**：为担保人将来承担担保责任后向债务人追偿的实现而提供的担保。

| 要点 | 规则 |
| :--- | :--- |
| **反担保人** | 债务人 或 第三人 |
| **债务人提供反担保** | **只能提供物保**（不能自己给自己做保证） |
| **第三人提供反担保** | 可以提供**物保或人保** |
| **反担保的独立性** | 本担保合同无效，**不影响**反担保合同的效力 |

| **保证债务的诉讼时效** | 客观题 | 常考 | 考查一般保证和连带保证诉讼时效起算的区别 | 一般保证：从保证人**拒绝承担保证责任的权利消灭之日**起算；连带保证：从债权人**请求保证人承担保证责任之日**起算 |

> **老师原话总结**：
> - “保证合同是**单务、无偿、要式合同**——保证人只有义务无权利，必须书面。”
> - “学好担保，**两手抓，两手硬**——一手抓标的物，一手抓顺序。”
> - “公益法人（公立学校、医院）**不得做保证人**——保证合同无效，但承担缔约过失责任。”
> - “保证方式约定不明——推定为**一般保证**（民法典第686条）。”
> - “一般保证——有**先诉抗辩权**（顺序利益保护）——先债务人，后保证人。”
> - “连带责任保证——**无顺序**，债权人可任意选择。”
> - “债务加入 vs 连带责任保证——**债务加入独立于主合同**（主合同无效仍有效），风险更大；**连带责任保证从属于主合同**（主合同无效则保证合同无效），风险更小。”
> - “识别不清时，应当认定为**保证**——对第三人更有利。”

> - “保证期间——**有约从约，无约六个月**，性质为**除斥期间**（不变期间）。”
> - “保证人受**双重时间保护**：第一层是保证期间，第二层是保证债务诉讼时效。”

| **保证人的诉讼时效抗辩权** | 客观题 | 常考 | 考查保证人能否主张主债权的时效抗辩 | 主债权已过诉讼时效，保证人仍可主张**诉讼时效抗辩权** |
| **按份共同保证的识别** | 客观题 | 常考 | 考查“按份”是谁与谁约定的 | 保证人与债权人约定份额——**按份共同保证**；仅保证人之间约定——**不构成按份共同保证** |
| **连带责任保证 vs 连带共同保证** | 客观题 | **易混淆** | 考查二者的区别 | 连带责任保证=保证人与**债务人**连带；连带共同保证=保证人与**保证人**连带 |
| **一般保证+共同保证（起诉限制）** | **✅ 主观题** | 常考 | 考查一般保证人的先诉抗辩权在共同保证中的应用 | 两个一般保证人，债权人能否单独先告保证人？**不能**（须先诉债务人） |
| **共同物保——债务人物保优先** | **✅ 主观题** | 常考 | 考查债务人自己提供物保时的执行顺序 | 债务人物保 + 第三人物保并存 → **应当先执行**债务人物保 |
| **共同物保——均为第三人物保** | 客观题 | 偶有考查 | 考查无顺序限制时的债权人选择权 | 均为第三人物保 → 债权人**可任意选择**执行哪一个 |
| **共同被告规则（担保制度解释第45条）** | 客观题 | 常考 | 考查诉讼方式实现担保物权的当事人 | 诉讼方式实现担保物权，应当将债务人和担保人作为**共同被告** |

> **老师原话总结**：
> - “保证人的权利——债务人有的权利，保证人都有（追偿权、抗辩权、抵销权、撤销权）。”
> - “法定代位权——**新增，从未考过**！本质是债权的法定转让。保证人承担后享有债权人对债务人的一切权利（包括担保物权）。”
> - “法定代位权行使**不得损害债权人利益**——原债权人优先于担保人受偿。”
> - “按份共同保证——‘按份’是**保证人与债权人**约定的份额，不是保证人之间自己约定的！”
> - “连带责任保证 vs 连带共同保证——完全不同的概念！前者是保证人与债务人连带，后者是保证人与保证人连带。”
> - “共同物保——找**债务人的物保**！债务人提供物保的，应当先执行。”
> - “共同担保的核心——**先看标的物，后看顺序**。找特殊的人（一般保证人）、找特殊的物（债务人的物保）、找特殊的关系（人保+物保）。”

### 第一段：保证合同概述

好，接下来咱们来看专题二十四，保证合同。这个合同是客观题和主观题都会考到的，在旁边拿笔写上**主观题**。

保证合同是担保家族当中的**人保**。我们前面讲担保制度，典型担保项下，事前的狭义担保分成了**物保、人保、金钱保**。保证就是人保。

保证：以人的信用或信誉提供的担保，保证人的**全部财产**均属于责任财产。保证合同是为保障债权的实现，保证人和债权人约定，当债务人不履行到期债务或发生当事人约定的情形时，保证人履行债务或承担责任的合同（民法典第681条）。

**担保 vs 保证**：担保是大圈，里面有抵押、有质押、有保证等等。保证合同是担保合同向下的一种分支。


### 第二段：保证合同的性质——单务、无偿、要式

#### （一）单务合同

保证合同是典型的**单务合同**。我找你借一万块钱，李家提供保证担保。你和李家签的保证合同——李家作为保证人，**只有义务没有权利**；你作为债权人，**只有权利没有义务**。

#### （二）无偿合同

保证合同是典型的**无偿合同**。债权人和保证人之间不需要给付对价。

> **注意**：实践中担保公司收费，那是担保公司和**债务人**之间的**委托合同**（有偿），不是保证合同本身有偿。保证合同的双方当事人是**债权人和保证人**，不是债务人和保证人。

#### （三）要式合同

保证合同必须是**书面形式**（民法典第685条）。因为保证人责任太重（无限责任），要让人慎重考虑。

**保证合同的四种书面形式**（民法典第685条）：
1. 单独的书面保证合同
2. 主合同中的保证条款
3. 主合同中无保证条款，但第三人以保证人身份签字
4. **第三人单方以书面形式向债权人作出保证，债权人接收且未提出异议**（考最多）


### 第三段：保证合同当事人与责任性质

> **合同当事人判断例题（基础模型）** ：孟某向马某借款2000万元，二者签订借款合同。曹某为该笔借款提供保证担保。

**第一问：借款合同的当事人是谁？**
答：**孟某和马某**（这是主法律关系）。

**第二问：保证合同的当事人是谁？**
答：**马某和曹某**（保证人和债权人）。

> **核心要点**：保证合同是**保证人和债权人**签的，而不是和债务人签的。只有理解了这个，才不会答错题目。

**第三问：保证人曹某以何种标的物提供担保？**
答：只要曹某没说七大标的物（房地车设备股权债权票据），就默认以**全部财产**做担保。全部财产 → 保证。

**第四问：保证人曹某承担何种责任？**
答：**无限责任**。整个担保家族中，就保证人一个人承担无限责任。给别人做保证风险最大——债务人还不上钱，债权人可以直接执行保证人的全部财产。

> 核心口诀：**学好担保，两手抓，两手硬——一手抓标的物，一手抓顺序。**


### 第四段：保证人的限制（民法典第683条）

保证人须以**全部财产**承担无限责任，因此如下法人**不得为保证人**：

1. **机关法人**（政府、法院、检察院等）——原则不能，例外不考
2. **以公益为目的的非营利法人、非法人组织**（公立的幼儿园、学校、医院、养老院等）——**重点**

> **公益法人保证例题**：孟某向马某借款2万元，**中关村第三小学**（公益小学）为该笔债务提供保证担保。
>
> **分析**：中关村第三小学属于事业单位法人，系以公益为目的的非营利法人，**不得做保证人**。保证合同**无效**（民法典第683条）。
>
> **注意**：保证合同无效不是没有法律后果——根据民法典第157条，承担**缔约过失责任**。


### 第五段：保证合同的四种书面形式（民法典第685条）

> **第四种形式例题（考最多）** ：孟某向马某借款200万元用于购买婚房，曹某**单方**向马某出具了一份**担保承诺函**，注明：“如孟某到期不能还款，本人承担**全部责任**。”
>
> **分析**：曹某单方以书面形式向债权人作出保证，马某接收且未提出异议，保证合同**成立**。曹某未指定七大标的物（房地车设备股权债权票据）中的任何一个，故应认定为以**全部财产**提供保证，即**保证合同**。
>
> **结论**：保证合同成立（民法典第685条第2款）。


### 第六段：保证方式——一般保证 vs 连带责任保证（核心考点★★★★★）

保证方式就是承担责任**有没有顺序**的问题。

#### （一）一般保证（民法典第687条）

**含义**：当事人在保证合同中约定，债务人**不能履行**债务时，由保证人承担保证责任。

**核心特征**：有**先诉抗辩权**（顺序利益保护）——先债务人，后保证人。

**识别关键词**：“债务人不能履行债务时”、“债务人不能还款时”、“一般保证”等字样。

**先诉抗辩权**（民法典第687条第1款）：一般保证的保证人在主合同纠纷未经审判或者仲裁，并就债务人财产依法强制执行仍不能履行前，有权**拒绝**向债权人承担保证责任。

| 层面 | 含义 |
| :--- | :--- |
| **程序上** | 债权人**不得单独先起诉或仲裁**一般保证人（不是不能告，是不得**单独先**告） |
| **实体上** | 法院**应当先执行**债务人财产，执行完了还不够才能执行一般保证人财产 |

> **一般保证先诉抗辩权模型**：孟某向白某借款100万，白某是债权人，孟某是债务人，李家做一般保证人（出具函件：“如到期孟某**不能**还钱，本人承担保证责任”）。
>
> **到期孟某不还款，白某面临的问题**：
>
> **起诉时（程序上）** ：
> - 方案一：**单独告孟某**（合同相对性，肯定可以）
> - 方案二：将孟某和**李家作为共同被告**一起告（也可以，但不能**单独先**告李家）
>
> **执行时（实体上）** ：
> - 法院**应当先执行**孟某的财产
> - 孟某的财产全部执行光了还不够，才能执行李家的财产
> - 李家承担的是**第二顺序责任**（补充责任）

**先诉抗辩权的丧失**（民法典第687条第2款）——债务人客观上没钱了：

1. 债务人下落不明，且无财产可供执行
2. 人民法院已经受理债务人破产案件
3. 债权人有证据证明债务人的财产不足以履行全部债务或者丧失履行债务能力
4. 保证人书面放弃先诉抗辩权

> **丧失先诉抗辩权例题**：孟某向白某借款100万，李家做一般保证人。后孟某被法院受理破产案件。
>
> **问**：李家还能否主张先诉抗辩权？
>
> **答**：**不能**。法院已受理债务人破产案件 → 一般保证人丧失先诉抗辩权，债权人可直接执行李家的财产。

#### （二）连带责任保证（民法典第688条）

**含义**：当事人在保证合同中约定保证人和债务人对债务承担**连带责任**。

**核心特征**：**无顺序**，**无先诉抗辩权**，债权人可以**任意选择**债务人或者保证人承担责任。

| 层面 | 含义 |
| :--- | :--- |
| **程序上** | 可以**单独告保证人**，也可以告债务人，也可以共同告 |
| **实体上** | 法官可以**任意执行**债务人和保证人的财产 |

> **连带责任保证模型**：孟某向白某借款100万，李家做连带责任保证人（保证合同中明确约定“承担连带责任”）。
>
> **到期孟某不还款，白某面临的问题**：
>
> **起诉时（程序上）** ：三种方案——
> - 方案一：**单独告孟某**
> - 方案二：**单独告李家**
> - 方案三：将孟某和李家**作为共同被告**
>
> **实体上**：李家**无先诉抗辩权**，法院可以直接执行李家的财产。


### 第七段：保证方式的推定（民法典第686条第2款，核心考点★★★★★）

当事人在保证合同中对保证方式**没有约定或约定不明确的**，按照**一般保证**承担保证责任。

> **价值理念变化**：民法典之前是债权人保护主义；民法典之后是**对保证人好一点**（毕竟是拿全部财产承担无限责任）。


### 第八段：保证期间与保证债务诉讼时效

【补充：讲稿在本部分音频中未展开保证期间与诉讼时效的具体规则，此段为补充内容。后续音频中将详细讲授，届时补充。】

**民法典第692条**：保证期间是确定保证人承担保证责任的期间，不发生中止、中断和延长。债权人与保证人可以约定保证期间；没有约定或者约定不明确的，保证期间为主债务履行期限届满之日起**六个月**。

**民法典第693条**：
- 一般保证的债权人未在保证期间对债务人提起诉讼或者申请仲裁的，保证人**不再承担保证责任**
- 连带责任保证的债权人未在保证期间请求保证人承担保证责任的，保证人**不再承担保证责任**

**民法典第694条**：
- 一般保证的债权人在保证期间届满前对债务人提起诉讼或者申请仲裁的，从保证人**拒绝承担保证责任的权利消灭之日**起，开始计算保证债务的诉讼时效
- 连带责任保证的债权人在保证期间届满前请求保证人承担保证责任的，从债权人**请求保证人承担保证责任之日**起，开始计算保证债务的诉讼时效


### 第九段：债务加入 vs 连带责任保证 vs 第三人代为履行（对比★★★★★）

三者都是第三人帮助债务人解决债务困境的方式，但规则完全不同。

| 对比项 | 第三人代为履行 | 债务加入（并存的债务承担） | 连带责任保证 |
| :--- | :--- | :--- | :--- |
| **是否有约定** | ❌ **无约定**（主动履行） | ✅ 有约定（函件/协议） | ✅ 有约定（保证合同/函件） |
| **是否履行** | ✅ **实际履行**（还钱了） | ❌ 未履行（只给承诺） | ❌ 未履行（只给承诺） |
| **第三人是否成为当事人** | ❌ 否（不加入法律关系） | ✅ **是**（成为连带债务人） | ❌ 否（仅是担保人） |
| **法律效果** | **法定债权转让**（追偿） | 承担**连带责任**（追偿） | 承担**连带责任**（追偿） |
| **独立性** | — | **独立**于主合同 | **从属于**主合同 |

**核心区别**：
- **第三人代为履行**：无约定，实际还钱，不加入法律关系
- **债务加入**：有约定，有独立性（主合同无效，债务加入仍可能有效）
- **连带责任保证**：有约定，有从属性（主合同无效，保证合同无效）

> **母公司帮子公司脱困三方案（综合对比例题）** ：甲公司（全资子公司）欠工商银行2000万，到期可能还不了。乙公司（母公司）作为集团公司，需要帮子公司脱困，否则影响集团信用评级。
>
> **方案一（第三人代为履行）** ：母公司乙公司**直接替子公司还了2000万**。
>
> **法律定性**：**第三人代为履行**（民法典第524条）——无约定，主动履行，合法利益（母子公司关联关系）→ **法定债权转让**，乙公司取得对甲公司的**无条件追偿权**。
>
> **方案二（债务加入）** ：乙公司向工商银行发函：“如甲公司到期不还款，**本公司也还**。”
>
> **法律定性**：**债务加入**（民法典第552条）——有约定，未实际履行，**独立于主合同**。乙公司在加入范围内承担**连带责任**，取得对甲公司的**追偿权**。
>
> **方案三（连带责任保证）** ：乙公司向工商银行发函：“如甲公司到期不还款，**本公司无条件承担责任**。”
>
> **法律定性**：**连带责任保证**（民法典第688条）——有约定，未实际履行，**从属于主合同**。乙公司承担连带责任，取得对甲公司的**追偿权**（民法典第700条，法定代位权）。
>
> **深度对比——主合同无效时的区别（核心考点）** ：
>
> 假设甲公司借款是为了**放高利贷**（高利转贷），主合同无效，且乙公司不知情。
>
> - **债务加入**：因具有**独立性**，主合同无效不影响债务加入的效力 → 乙公司**仍要承担责任**2000万
> - **连带责任保证**：因具有**从属性**，主合同无效则保证合同无效 → 乙公司**最多承担缔约过失责任**（无过错则责任为零）
>
> **风险结论**：债务加入责任更重；连带责任保证因有从属性，对保证人更有利。
>
> **识别不清时**：应当认定为**保证**（对第三人更有利）。


### 第十段：六大抗辩权回顾

讲稿在讲解一般保证先诉抗辩权时，提到了整个民法中的**六大抗辩权**：

| 序号 | 抗辩权 | 类型 | 法典依据 |
| :--- | :--- | :--- | :--- |
| 1 | **诉讼时效抗辩权** | **永久性抗辩权** | 民法典第192条 |
| 2 | **同时履行抗辩权** | 暂时性抗辩权 | 民法典第525条 |
| 3 | **先履行抗辩权** | 暂时性抗辩权 | 民法典第526条 |
| 4 | **不安抗辩权** | 暂时性抗辩权 | 民法典第527条 |
| 5 | **一般保证人的先诉抗辩权** | 暂时性抗辩权 | 民法典第687条 |
| 6 | **赠与人的穷困抗辩权** | 暂时性抗辩权 | 民法典第666条 |

> **核心区别**：诉讼时效抗辩权是**永久性抗辩权**（权利本身消灭），其他五个都是**暂时性抗辩权**（义务暂时可以拒绝履行，但权利本身不消灭）。

### 第一段：保证范围——责任大小

第三个问题，叫做保证范围。保证范围，说的就是责任大小的问题。

保证人要拿他们家的全部财产来承担这个责任，那在多大的范围内承担责任呢？有约从约，无约包括全部——主债权、利息、违约金、损害赔偿金【修正：原文“索赔金”】、实现债权的费用，通通都得担保。

> **保证范围例题**：甲向乙借款五万元，乙要求甲提供担保。甲分别找到丙、丁、戊、己，各自做出以下表示，其中哪些构成保证？（判断标准：看标的物——只要没说七大标的物——房地车设备股权债权票据，就是保证）
>
> A. 丙在甲向乙出具的借据上签署“保证人：丙”——**构成保证**（以保证人身份签字）
>
> B. 丁向乙出具称：“如甲到期不向乙还款，本人愿代还**三万元**。”——**构成保证**（三万是保证范围，不是特定财产）
>
> C. 戊向乙出具称：“如甲到期不向乙还款，由本人负责。”——**构成保证**（无特定财产，默认全部财产）
>
> D. 己向乙出具称：“如甲到期不向乙还款，由本人以**某处私房**抵债。”——**不构成保证**（私房是不动产，只能做抵押担保）
>
> **答案：A、B、C**


### 第二段：保证的从属性——变更的从属性

保证有六大从属性：发生、变更、转让、消灭、效力、范围。这里重点讲**变更和转让**。

核心原则：看债权人和债务人之间的约定有没有**加重**保证人的责任。加重了 → 需保证人**书面同意**；减轻了 → 无需书面同意。

#### （一）数额变更

- **增加**（加重责任）→ 需保证人书面同意；未经同意的，对**加重的部分**不承担保证责任
- **减少**（减轻责任）→ 无需同意，保证人对**变更后的债务**承担保证责任

> **数额增加例题**：孟某向马某借款2000万，借款期限一年，曹某提供保证担保。后孟某和马某协商将借款数额增加至2250万（增加250万），曹某未知可否。
>
> **问**：曹某如何承担保证责任？
>
> **答**：对增加的250万**不承担保证责任**，但对原来的2000万仍承担保证责任。

#### （二）履行期限变更

- **延长**（加重责任）→ 需保证人书面同意；未经同意的，对**延长后增加的部分**不承担保证责任
- **缩短**（减轻责任）→ 无需同意，保证人对缩短后的债务承担保证责任

> **期限延长+保证期间覆盖例题（时间轴模型）** ：2024年3月2日，孟某向马某借款11万元，借款期限一年（2024.3.2—2025.3.2）。曹某以保证人身份在借款合同中签字，保证期间约定为一年（2025.3.2—2026.3.2）。马某要求从11万中预先扣除1万元利息（砍头息），实际借款数额为10万元。后债权人与债务人将借款期间延长为两年（2024.3.2—2026.3.2），但保证期间纹丝不动仍为一年。
>
> **分析**：新的借款期间（2026.3.2届满）**全程覆盖**了保证期间（2026.3.2届满）。债权人最早可找债务人要债的日期是2026年3月3日（未到期的债权视为无债权），但此时曹某的保证期间已于前一日届满。
>
> **结论**：保证人曹某**不需要承担保证责任**（延长期限全部覆盖保证期间）。【补充：讲稿提到这是早些年考过的，法考以来未考过】


### 第三段：保证的从属性——转让的从属性

#### （一）债权转让的从属性

债权转让对保证人影响不大（债务人没变），原则上仅需**双重通知**（通知债务人 + 通知保证人）。

**例外**：保证人与债权人约定“禁止债权转让”或“仅对特定债权人承担保证责任”的，未经保证人书面同意转让债权的，保证人**不再承担保证责任**。

> **债权转让例题**：2025年4月2日，孟某向徐某借款850万，马某为连带责任保证人。4月21日，徐某将该850万债权转让给曹某。4月25日，孟某和马某接到债权转让通知。
>
> **问一（原则）** ：保证合同无特别约定，马某如何承担保证责任？
>
> **答**：在原保证范围内**继续承担保证责任**。
>
> **问二（例外）** ：保证合同明确约定“禁止债权转让”，马某如何承担保证责任？
>
> **答**：**不再承担保证责任**（全部转让出去了）。

#### （二）债务转让的从属性——免责的债务转让（核心重点★★★★★）

债务人要退出（免责的债务承担），对债权人和保证人影响都巨大。需**双重同意**：
- **债权人同意**（不要式的：书面、口头、行为均可）
- **保证人书面同意**（要式的：必须书面）

> **双重同意例题**：2025年3月2日，孟某向徐某借款850万，马某为连带责任保证人。4月21日，孟某将500万债务转让给曹某（免责的债务承担）。
>
> **问一**：如债权人徐某**不同意**债务转让，马某如何承担责任？
>
> **答**：马某**继续承担全部保证责任**（债务未转让，孟某仍是债务人）。
>
> **问二**：如徐某**同意**债务转让，但马某**未书面同意**，马某如何承担责任？
>
> **答**：马某**仅对剩余350万元**承担保证责任（850万-500万=350万）。转多少免多少——未经保证人书面同意，保证人在500万范围内免责。如果全部转让，他就全部免责。
>
> **问三**：如徐某同意，马某亦书面同意，马某如何承担责任？
>
> **答**：继续对**850万**承担保证责任（意思自治）。

#### （三）债务转让的从属性——并存的债务转让（债务加入）

债务加入（债务人不退出，第三人加入）对保证人是**好事**（多了一个偿债的人），**无需**保证人书面同意。

> **原理**：第三人加入债务，原债务人还是当然的债务人，保证人的权益不仅没有受到侵害，反而因第三人加入债务有可能增加履行财产，最终使保证责任减轻。


### 第四段：保证期间（核心考点★★★★★）

**保证期间**：确定保证人承担保证责任的期间。核心功能：**保护保证人** + 督促债权人及时行权。

**性质**：**除斥期间**（不变期间，不发生中止、中断、延长）。

#### （一）保证期间的确定

- **有约从约**（当事人可以约定，如约定一年、两年、三年均可）
- **无约六个月**（民法典第692条）
- **约定“直至本息还清为止”** → **视为约定不明** → 仍按**六个月**处理

**起算点**：自主债务**履行期限届满之日**起计算。

#### （二）保证期间届满的法律后果

| 保证方式 | 债权人应在保证期间内 | 未行权的后果 |
| :--- | :--- | :--- |
| **一般保证** | 对债务人**提起诉讼或申请仲裁**（至少告债务人） | 一般保证人**免责** |
| **连带责任保证** | **请求**保证人承担保证责任（或诉或裁或通知） | 连带保证人**免责** |


### 第五段：保证期间与保证债务诉讼时效的关系（难点★★★★★）

保证人受**双重时间保护**：第一层是保证期间，第二层是保证债务诉讼时效（三年）。

> **例一（保证期间届满——保证人免责）** ：2023年2月1日，孟某向马某借款2000万，借款期限一年（2023.2.1—2024.2.1）。曹某为**一般保证人**，未约定保证期间，法定保证期间为六个月（2024.2.1—2024.8.1）。到期孟某未还款，马某于**2024年9月1日**起诉孟某。
>
> **分析**：马某起诉时（2024.9.1）已在保证期间（2024.8.1届满）之外。债权人在保证期间内**未依法行权**（一般保证须至少起诉债务人）。
>
> **结论**：一般保证人曹某**免责**。

> **例二（保证期间内依法行权→保证期间消灭→转诉讼时效）** ：2023年2月1日，孟某向马某借款2000万，借款期限一年。曹某为**一般保证人**，未约定保证期间，法定保证期间为六个月（2024.2.1—2024.8.1）。到期孟某未还款，马某于**2024年3月1日**起诉孟某和曹某（在保证期间内依法行权）。
>
> **第一步**：2024.3.1，马某在保证期间内起诉 → 保证期间**消灭**（功成身退）
>
> **第二步**：4月1日法院判决马某胜诉，执行顺序为先孟某后曹某（一般保证先诉抗辩权）
>
> **第三步**：经法院强制执行，6月1日确认孟某无财产可供执行（**一般保证人丧失先诉抗辩权**，即“拒绝权消灭”）
>
> **第四步**：自6月2日起，开始计算**保证债务三年诉讼时效**（2024.6.2—2027.6.1）
>
> **结论**：保证期间保护了保证人一次（债权人必须在保证期间内行权）；若保证期间内行权了，还有三年诉讼时效保护第二次（债权人未在三年内起诉，保证人可提诉讼时效抗辩）。

**核心规则**：债权人在保证期间内依法行权 → **保证期间消灭** → 开始计算保证债务诉讼时效。

### 第一段：保证人的权利概述

好，接下来咱们来看第六个问题，叫做保证人的权利。

保证人的权利一共包括四种：第一个叫**追偿权**，第二个叫**抗辩权**，第三个叫**抵销权**，第四个叫**撤销权**。其实上就是债务人有的权利，保证人他都有。从考试的角度来看，重点是前两个，一个是追偿权——特别是民法典第七百条新增了一个制度叫**法定代位权**，还从来没考过；第二个是抗辩权——特别是诉讼时效抗辩权。


### 第二段：追偿权（民法典第700条）

二百九十六页，第一个叫追偿权。啥叫追偿权呢？我现在找你借一万块钱，李家来做保证担保，到期我不能还款，于是你去找李家追究保证责任。李家承担完保证责任后，他不是最终的背锅侠，因为钱是我借的、我花的，所以李家承担完后可以来找我追偿——这叫追偿权。

**追偿权的条件**（重点四和五）：

| 条件 | 说明 |
| :--- | :--- |
| **1. 追偿范围** | 只能在**承担保证责任的范围内**追偿（承担多少追多少） |
| **2. 履行行为必须没有过错** | 如有过错，对有过错的部分不能追偿 |

> **追偿权例题（历年真题）** ：张某从甲银行分支机构乙支行借款20万元，李某提供保证担保。李某与甲银行有特别约定：“如保证人不履行保证责任，债权人有权直接从保证人在甲银行及其支行处开立的任何账户内扣收。”到期张某、李某均未还款，甲银行直接从李某在甲银行下属丙支行账户内扣划了18万元存款，用于偿还张某的借款。
>
> **问一**：该“直接扣收”约定是否有效？
>
> **答**：**有效**。这不是流押条款（那是物保的事），保证合同中没有流押条款的问题。该约定明确表明李某是**连带责任保证人**。
>
> **问二**：李某承担18万保证责任后，能否向张某追偿？
>
> **答**：**可以**。承担多少追偿多少，在**18万范围内**追偿。
>
> **问三**：乙支行（分支机构）应以谁的名义行使追偿权？
>
> **答**：以**自己的名义**。总行和支行是一家银行，但支行作为非法人组织，可以以自己的名义起诉应诉（有程序无实体）。
>
> **答案：D项正确**


### 第三段：法定代位权（新增，从未考过★★★★★）

二百九十七页，第二个是新增的，还从来没考过。旁边写两个字：**新增**。

**法定代位权**的本质就是**债权的法定转让**（民法典第700条）。保证人承担保证责任后，除当事人另有约定外，有权在其承担保证责任的范围内向债务人追偿，**享有债权人对债务人的权利**。

简单说：保证人替主债务人还了钱，就相当于债权从原债权人转移到了保证人身上，保证人取得了原债权人对债务人的一切权利（包括担保物权）。

> **法定代位权例题**：孟某向王某借款2000万，孟某以自己价值1000万的房屋提供抵押担保并登记。曹某（一般保证人）单方向王某出具担保函。徐某以自己价值500万的宾利轿车提供抵押担保并登记。王某以自己持有的价值200万的海马公司20%股权提供质押担保并登记。到期孟某不能清偿债务。
>
> **问一**：如曹某主动向王某承担保证责任，是否有权请求拍卖孟某的房屋优先受偿？
>
> **答**：**有权**。曹某承担保证责任后，取得法定代位权，享有原债权人对债务人孟某的一切权利（包括抵押权）。
>
> **问二**：徐某（物保人）承担担保责任后，是否有权请求拍卖孟某的房屋优先受偿？
>
> **答**：**有权**。民法典担保制度解释第18条将保证人的法定代位权扩充到了物保人。

> **法定代位权例外（但书）** ——“不得损害债权人利益”：孟某向王某借款2000万，孟某以价值1000万的房屋提供抵押担保并登记。曹某以价值500万的宾利轿车提供抵押担保并登记。到期孟某不能清偿债务，曹某向王某偿还500万。经查，孟某除价值1000万的房屋外无其他责任财产。
>
> **问一**：王某（原债权人）是否有权就剩余1500万拍卖孟某的房屋优先受偿？
>
> **答**：**有权**。抵押权仍存在，主债权尚未清偿完毕，担保物一个都不能少（不可分性）。
>
> **问二**：曹某是否有权就500万拍卖孟某的房屋优先受偿？
>
> **答**：**有权**（法定代位权）。
>
> **问三**：王某和曹某就房屋拍卖后的1000万如何分配？
>
> **答**：**1000万全部归王某**。担保人行使法定代位权**不得损害债权人利益**。债权人优先于担保人受偿。


### 第四段：抗辩权、抵销权、撤销权

二百九十八页第三个，抗辩权。重点就是**诉讼时效抗辩权**。债务人有的抗辩权，保证人都有（保证具有从属性）。

> **例题**：主债权已过诉讼时效，保证人能否主张诉讼时效抗辩权？
>
> **答**：**能**。即使债务人放弃，保证人仍可主张。

同样道理，债务人享有的**抵销权**和**撤销权**，保证人也都享有。总之就一句话：债务人有的权利（抗辩权、抵销权、撤销权），保证人都有。


### 第五段：共同担保概述

二百九十九页，共同担保——担保人为**二人以上**。

- **共同保证**：都是保证人（人保）
- **共同物保**：都拿特定财产（物保）
- **混合担保**：既有人的担保，也有物的担保（**考试重点**）

学习共同担保，抓住两脉：**先看标的物，后看顺序**。


### 第六段：共同保证——按份共同保证 vs 连带共同保证

共同保证是指两个或两个以上的保证人为同一债权提供担保。

**区分标准**：看债权人与保证人之间**有没有约定份额**。

| 类型 | 认定标准 | 说明 |
| :--- | :--- | :--- |
| **按份共同保证** | 保证人与债权人**约定了**各自承担的份额 | 债权人不能要求保证人承担超出约定份额的责任 |
| **连带共同保证** | 保证人与债权人**未约定**份额 | 推定各保证人承担连带责任 |

> **重要区分（常见混淆点）** ：
> - **连带责任保证** = 保证人与**债务人**之间连带（无先诉抗辩权）
> - **连带共同保证** = 保证人与**保证人**之间连带（各保证人之间份额连带）
>
> **二者完全不同！**

#### （一）类型一：按份共同保证（外部一般保证 + 内部按份）

> **模型**：孟某向徐某借款1000万，马某、曹某与徐某签订保证合同，约定“如孟某**不能**还款，马曹承担保证责任”（**一般保证**）。马某承担40%，曹某承担60%（**与债权人约定了份额——按份共同保证**）。
>
> **起诉时**：马曹均是一般保证人，享有先诉抗辩权。方案四种：①单独告孟；②告孟+马；③告孟+曹；④告孟+马+曹。**不能单独先告马或曹**。
>
> **执行时**：法院应当判决**先执行孟某财产**，不足部分再执行马曹财产，且马曹按40%和60%份额分别承担。

#### （二）类型二：连带共同保证（外部一般保证 + 内部连带）

> **模型**：孟某向徐某借款1000万，马曹与徐某签订保证合同，约定“如孟某**不能**还款，马曹承担保证责任”（**一般保证**）。但**未约定份额**（**连带共同保证**）。
>
> **起诉时**：同类型一（均系一般保证人），方案四种。
>
> **执行时**：同类型一，应先执行孟某财产。
>
> **区别**：类型一和类型二的区别不在起诉时，也不在执行时，而在**互相追偿权**上——类型一按份，互不追偿；类型二连带，可以互相追偿。

#### （三）类型三：按份共同保证（外部连带责任保证 + 内部按份）

> **模型**：孟某向徐某借款1000万，马曹与徐某签订保证合同，约定“马曹提供**连带责任保证**”（**无先诉抗辩权**）。马某承担40%，曹某承担60%（**与债权人约定了份额——按份共同保证**）。
>
> **起诉时**：方案七种（排列组合，爱告谁告谁）。
>
> **执行时**：法院可以任意执行孟马曹的财产。马某责任上限400万，曹某上限600万。

#### （四）类型四：连带共同保证（外部连带责任保证 + 内部连带）

> **模型**：孟某向徐某借款1000万，马曹与徐某签订保证合同，约定“马曹提供**连带责任保证**”（**无先诉抗辩权**）。**未约定份额**（**连带共同保证**）。
>
> **起诉时**：方案七种，爱告谁告谁。
>
> **执行时**：法院可以任意执行全部财产，无份额限制（但各自以全部财产承担无限责任）。

**类型三与类型四的核心区别**：看债权人与保证人有没有约定份额。有约定→按份共同保证；没约定→连带共同保证。


### 第七段：共同物保

共同物保是担保人拿**特定财产**来做担保。

**核心问题**：看这个物保的物是**债务人提供的**还是**第三人提供的**。

#### （一）债务人提供的物保 + 第三人提供的物保并存

> **模型**：众森公司向工商银行贷款2亿，众森公司以**自己的办公楼**提供抵押担保并登记（**债务人物保**），海马公司以厂房提供抵押担保并登记（**第三人物保**）。
>
> **起诉时**：债权人通过诉讼方式实现担保物权，**应当**将债务人和担保人作为**共同被告**（民法典担保制度解释第45条第3款）。
>
> **执行时**：**应当先执行债务人物保**（办公楼），不足部分再执行其他担保财产（类推适用民法典第392条，基于利益承受理论）。

#### （二）均为第三人提供的物保

> **模型**：众森公司向工商银行贷款2亿，嘉英公司以机械设备提供抵押并登记，海马公司以厂房提供抵押并登记（**均为第三人物保**）。
>
> **起诉时**：应当以众森公司、嘉英公司、海马公司为**共同被告**。
>
> **执行时**：**无顺序限制**，债权人有选择权（物保是有限责任，对第三人物保的限制不需要特别保护）。


### 第八段：共同担保的规律总结

| 担保类型 | 特殊要素 | 核心规则 |
| :--- | :--- | :--- |
| **共同保证** | 找**特殊的人**（一般保证人） | 一般保证人享有先诉抗辩权，限制债权人的行权顺序 |
| **共同物保** | 找**特殊的物**（债务人的物保） | 债务人物保应先执行（利益承受理论） |
| **混合担保** | 找**特殊的关系**（人保+物保并存） | 债务人物保→人保→第三人物保（有顺序限制，民法典第392条） |

> **核心口诀**：共同担保，先看标的物（是人是物），后看顺序（找特殊的人、特殊的物、特殊的关系）。

### 第一段：混合担保——债务人物保与保证并存

好，接下来咱们来看第三个，叫做混合担保，这个是考试的重点，也是民法典第三百九十二条标准的条文的应用。在旁边写上**重点**。

核心问题依然是**看那个债务人有没有提供物的担保**。只要发现债务人自己提供了物的担保，在没有特别约定的情况下，**应当先执行**他的财产——这就是执行时有顺序的考点。

> **混合担保模型（债务人物保+保证）** ：众森公司向工商银行贷款2亿元，以**自己的办公楼**为该笔债务提供抵押担保并办理登记（**债务人物保**），海马公司提供**连带责任保证**（人保）。众森公司到期无力还款。
>
> **第一，有约从约**：考试一般没有特别约定。但实践中有约定的话，从约定。（从债权人角度看，直接扣划保证人账上的钱可能只需一秒，拍卖办公楼成本高、周期长——进可攻退可守。）
>
> **第二，无特别约定时**：债权人**应当先**就债务人提供的物保实现债权（民法典第392条）。这背后的法理与逻辑是：
> - **公平**：债务人自己提供了物保，如果债权人转而先请求保证人承担责任，对保证人不公平
> - **效率/成本**：如果先请求保证人，保证人承担后还要向债务人追偿，增加司法成本
>
> **结论**：执行时有顺序限制——**应当先执行债务人物保**。


### 第二段：混合担保——第三人物保与保证并存

第三人提供的物保和保证并存，就没什么特色了。

> **模型（第三人物保+保证）** ：众森公司向工商银行贷款2亿，嘉英公司以大型机器设备提供抵押担保并登记（**第三人物保**），海马公司提供连带责任保证（人保）。众森公司到期无力还款。
>
> **分析**：物的担保不是债务人自己提供的，嘉英公司不是最终责任人，海马公司也不是。此时债权人**没有顺序限制**，有极大的选择权——可以任意选择实现物保或请求保证人承担责任。


### 第三段：共同担保的规律总结

共同保证、共同物保和混合担保，核心就是**万军丛中去找例外**。

| 担保类型 | 找什么 | 核心规则 |
| :--- | :--- | :--- |
| **共同保证** | 找**特殊的人**（一般保证人） | 一般保证人享有先诉抗辩权，限制债权人行权顺序 |
| **共同物保** | 找**特殊的物**（债务人的物保） | 债务人物保应当先执行（利益承受理论） |
| **混合担保** | 找**特殊的关系**（人保+物保并存） | 债务人物保 → 人保/第三人物保（有顺序限制） |


### 第四段：共同担保人之间的内部追偿权（主观题重点★★★★★）

三百零五页最后一个问题——共同担保人之间的内部追偿权，旁边写**主观题**。

**最高人民法院不希望担保人之间互相追偿**（否则会出现循环追偿，浪费司法资源）。

#### （一）原则：互相不知对方存在 → 无追偿权

> **模型**：两个保证人背对背做担保，互相不知道对方存在。他们都没有心理准备，都以为自己是唯一的保证人。
>
> **结论**：**无追偿权**。任何一个担保人承担保证责任后，只能找**债务人**追偿（民法典担保制度解释第13条）。

#### （二）按份共同担保 → 无追偿权

> **模型**：马峰和李家与债权人三方一起签订合同，约定李家承担20%，马峰承担80%。
>
> **结论**：**有约从约，自己承担自己的责任**，互相不能找对方追偿（否则会循环追偿，浪费司法资源）。

#### （三）例外：三种情况下有追偿权

| 例外情形 | 说明 |
| :--- | :--- |
| **1. 担保人之间约定了追偿** | 意思自治，有约从约 |
| **2. 连带共同担保** | 各担保人之间连带，可以互相追偿 |
| **3. 在同一份合同书上签章按指印** | 面对面做担保，互相知道对方存在 |

> **共同担保人追偿权例题（历年真题）** ：甲公司向乙公司借款1000万，丙公司在借款协议保证人栏下**盖章**（保证人），但未载明保证方式。丁以**自有房屋**为该债务设立担保并办理抵押登记（物保人）。题目未说明在同一份合同上签章（即互相不知对方存在）。
>
> **问**：下列说法正确的是？
>
> A. 丙公司应按一般保证承担保证责任——**正确**（民法典第686条第2款，未约定保证方式→一般保证）
>
> B. 丁承担责任后，有权向丙追偿——**错**（互相不知对方存在，无追偿权）
>
> C. 丁承担责任后，有权向甲公司追偿——**正确**（甲公司是最终责任人）
>
> D. 丙公司承担保证责任后，有权向甲公司追偿——**正确**（最终责任人）
>
> **答案：A、C、D**


### 第五段：物保与人保的对比

| 对比项 | 物的担保 | 人的担保（保证） |
| :--- | :--- | :--- |
| **主体** | 可能是债务人自己，也可能是第三人 | **只能是债务人以外的第三人**（自己不能给自己做保证） |
| **客体** | **特定财产**（七大标的物） | **全部财产**（信用/信誉） |
| **责任性质** | **有限责任**（最多标的物没了） | **无限责任**（全部财产都可能被抄家） |
| **优势** | **优先受偿权**（特定财产） | 全部财产都可被执行（**尽可攻退可守**） |
| **劣势** | 仅能就特定财产优先受偿 | 任何财产都不能优先受偿（普通债权） |
| **期间保护** | 抵押权适用三年诉讼时效 | 保证期间（六个月）+ 保证债务诉讼时效（三年） |


### 第六段：反担保

反担保，最大的特点就是**独立性**。

反担保 = 担保人为自己未来承担担保责任后向债务人追偿的实现提供担保。简单说：本担保是担保主债权，反担保是**担保追偿权**。

#### （一）反担保与本担保的关系

> **反担保基本模型**：北京的孟某向江西的刘某借款2000万，孟某欲以山西的一套房产抵押担保，刘某不同意（去山西办理登记和实现担保物权成本太高）。孟某遂找到山西的马某做**保证人**（本担保合同）。马某担心承担保证责任后无法向孟某实现追偿权，于是请求孟某拿山西的房产给他提供**反担保**。
>
> **本担保**：马某与刘某签订的保证合同（担保主债权）
>
> **反担保**：孟某或第三人向马某提供的担保（担保追偿权）

#### （二）反担保的方式

| 反担保人 | 可提供的方式 |
| :--- | :--- |
| **债务人自己** | **只能提供物保**（不能自己给自己做保证） |
| **第三人** | 可以提供**物保或人保**（均可） |

#### （三）反担保的独立性（核心考点★★★★★）

反担保**独立于本担保**（民法典第689条、担保制度解释第19条）。

> **反担保独立性例题**：甲公司向乙公司借款2000万，**北京妇产医院**（公立医院，公益法人）提供保证担保（本担保合同）。因违反民法典第683条第2款（公益法人不得为保证人），**本担保合同无效**。丙公司为北京妇产医院提供反担保。
>
> **三层法律关系**：
>
> **第一层（主合同）** ：甲公司与乙公司的借款合同——**合法有效**
>
> **第二层（本担保合同）** ：北京妇产医院与乙公司的保证合同——**无效**（公益法人不得为保证人）
>
> **第三层（反担保合同）** ：北京妇产医院与丙公司的反担保合同——**有效**（反担保独立于本担保）
>
> **结论**：本担保合同无效，不影响反担保合同的效力。丙公司承担反担保责任后，最终可以向甲公司追偿（实践中最可能是丙公司是甲公司的全资子公司）。


### 第七段：保证合同专题总结

翻回二百八十六页，保证合同是每年绝对必考事项（客观和主观基本每年都考），因为它是担保家族中**唯一承担无限责任**的担保。

| 考点 | 核心内容 | 考频 |
| :--- | :--- | :--- |
| **基本原理** | 公益法人不得为保证人（民法典第683条） | 常考 |
| **保证合同订立方式** | 四种书面形式，重点：单方出具担保函 | 常考 |
| **保证方式** | 一般保证 vs 连带责任保证（责任顺序问题） | **绝对必考** |
| **保证范围** | 责任大小问题（有约从约，无约包括全部） | 考得较少 |
| **保证的从属性** | 重点：免责的债务承担时加重责任需书面同意 | 考得较少 |
| **保证期间** | 有约从约，无约六个月；“直至本息还清”视为约定不明→六个月 | 偶有考查 |
| **保证人的权利** | **法定代位权**（新增，从未考过） | **重点关注** |
| **共同担保** | 找特殊的人（一般保证人）、找特殊的物（债务人物保）、找特殊的关系（连带共同担保） | **每年必考（主观题）** |
| **反担保** | **独立性**（本担保无效不影响反担保效力） | 考得较少 |
