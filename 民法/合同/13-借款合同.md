
## 第1部分：对照《民法典》借款合同提取备考提纲


### 一、借款合同的分类

| 类型 | 特征 | 法典依据 |
| :--- | :--- | :--- |
| **商业借贷** | 有金融机构（银行、小贷公司等） | 民法典第667-680条 |
| **民间借贷** | 无金融机构（企业之间、企业与自然人、自然人与自然人之间） | 民间借贷司法解释 |


### 二、贷款人的权利（民法典第673条）

借款人未按约定用途使用借款的，贷款人可以：
- **停止发放借款**
- **提前收回借款**
- **解除合同**（根本违约）


### 三、贷款人的义务——禁止砍头息（民法典第670条）

借款的利息不得预先在本金中扣除。利息预先扣除的，按**实际借款数额**返还借款并计算利息。


### 四、自然人之间的借款合同（民法典第679条、第675条）

| 要点 | 规则 |
| :--- | :--- |
| **合同性质** | **实践性合同**（自**提供借款时**成立） |
| **利息约定** | 未约定→视为**无利息**；约定→不得超过**LPR四倍**（超过部分无效） |
| **还款期限** | 未约定→借款人可**随时返还**；贷款人可**催告**在合理期限内返还 |
| **逾期利息** | 逾期后应加算逾期利息 |


### 五、让与担保（非典型担保）

**定义**：债务人或第三人将标的物所有权**形式上转移**至债权人名下，以担保债务履行的非典型担保。

**本质**：**名为让与，实为担保**（一交易两合同，名实不符）。

**售后回购**：先卖后买，差价是利息。

**法律结构**：
- 名为让与（买卖合同）→ **双方通谋虚假意思表示，无效**（民法典第146条）
- 实为担保（隐藏行为）→ **有效**，类推适用典型担保规则（看标的物决定是抵押还是质押）

**先让与担保 vs 后让与担保**：

| 类型 | 要件 | 法律效果 |
| :--- | :--- | :--- |
| **先让与担保** | 不动产已登记 / 动产已交付 | 债权人享有**优先受偿权** |
| **后让与担保** | 未登记 / 未交付 | 债权人仅为**普通债权人** |

### 第一段：借款合同概述

好，接下来咱们来看专题二十三，借款合同。这个合同是客观题和主观题都会考查到的，在旁边拿笔写上**主观题**。

借款合同和前面的买卖合同、赠与合同的相同点就是它也能移转标的物的所有权，只是说这个标的物比较特殊，是**货币**（一般等价物）。

借款合同根据主体的不同，分成了两类：

| 类型 | 特征 | 典型主体 |
| :--- | :--- | :--- |
| **商业借贷** | **有金融机构** | 银行、小额贷款公司、保险公司、证券公司、信托公司 |
| **民间借贷** | **没有金融机构** | ①企业与企业之间；②企业与自然人之间；③自然人与自然人之间（考最多） |

> **重点提示**：很多同学错误地认为“民间借贷就是自然人与自然人之间的借贷”——不对！自然人与自然人之间的借贷**只是民间借贷的一种**（下位概念），民间借贷是上位概念。


### 第二段：借款合同的基本原理——贷款人的权利和义务

**借款合同**：借款人向贷款人借款，到期返还借款并支付利息的合同（民法典第667条）。

#### （一）贷款人的权利——监督借款用途（民法典第673条）

借款人未按照约定的借款用途使用借款的，贷款人可以：
- **停止发放借款**
- **提前收回借款**
- **解除合同**

本质上是根本违约 → 法定解除权的应用。

> **例题**：众森公司与工商银行签订借款合同，约定工行提供2000万贷款，分两批发放，众森公司应按约定用途使用贷款。第一批贷款发放后，众森公司未按约定使用贷款。
>
> **问**：工商银行有何权利？
>
> **答**：有权**解除贷款合同**（众森公司构成根本违约）。

#### （二）贷款人的义务——禁止“砍头息”（民法典第670条）

借款的利息不得预先在本金中扣除。利息预先在本金中扣除的，应当按照**实际借款数额**返还借款并计算利息。

> **砍头息例题**：孟某与马某约定，孟某向马某借款11万元，借款期限一年，马某要求从11万中预先扣除1万元利息，孟某应允。马某实际交付10万元。
>
> **问**：借款数额是多少？
>
> **答**：**10万元**（不是11万）。利息预先扣除的，按实际借款数额计算。


### 第三段：自然人之间的借款合同（实践性合同，核心考点）

自然人之间的借款合同具有**互助性**——你借给钱是帮助我，法律不能强迫每个人道德高尚。

#### （一）性质——实践性合同（民法典第679条）

自然人之间的借款合同，自**提供借款时成立**（不是合同签订时）。

#### （二）利息规则

| 情形 | 规则 |
| :--- | :--- |
| **未约定利息** | **视为无利息**（法律拟制） |
| **约定利息** | 可以约定，但不得超过**LPR的四倍**（目前LPR约3%，四倍约12%） |
| **超过LPR四倍** | 超过部分**无效**，构成**不当得利**，应返还 |

#### （三）还款期限（民法典第675条）

- **约定了还款期限**：按期偿还
- **未约定还款期限**：借款人可以**随时返还**；贷款人可以**催告**借款人在合理期限内返还

#### （四）逾期利息

- 借款期限内：按约定（未约定则无利息）
- 逾期后：应加算**逾期利息**（从逾期之日起计算）

> **自然人借贷模型**：孟某和马某书面约定，孟某向马某借款10万元，未约定利息，也未约定还款期限。
>
> **问一**：借款合同何时成立？
>
> **答**：**提供借款时成立**（实践性合同）。
>
> **问二**：马某是否有权随时请求孟某返还借款？
>
> **答**：**有权**（无期限则随时可要）。
>
> **问三**：马某能否请求按银行同期利率支付利息？
>
> **答**：**不能**（未约定利息视为无利息）。
>
> **问四**：马某催告后孟某仍不还款，马某能否主张逾期利息？
>
> **答**：**有权**（逾期后应加算逾期利息）。


### 第四段：让与担保合同（非典型担保之二，核心难点★★★★★）

让与担保是**第二个非典型担保合同**（民法典第388条“其他具备担保功能的合同”之第二）。

**定义**：债务人或第三人为担保债务的履行，将标的物的所有权**形式上转移**至债权人名下。债务清偿后标的物返还；债务不获清偿时，债权人得就该标的物受偿或优先受偿的一种非典型担保。

**本质**：**名为让与，实为担保**（**一交易两合同，名实不符**）。

#### （一）为什么会出现让与担保？——典型担保的局限性

典型担保（抵押权）存在局限：房屋抵押后，抵押人（债务人）仍可将房屋出售（带押过户），房屋可能因买受人行为（凶宅、黄赌毒）而大幅贬值；且同一房屋可设立多个抵押权，若登记顺序被操作，第一顺位抵押权人可能无法受偿。

**让与担保的解决方式**：不签抵押合同，改签**买卖合同**，办理**过户登记**。登记簿上显示债权人是**所有权人**（名义上），但实际上是担保权人。

**售后回购**——让与担保最典型的表现形式：**先卖后买，差价是利息**。

> **售后回购例题**：我把戒指1万元卖给你并交付，1年后我再以1.2万元买回来。
>
> **分析**：一卖一买差价2000元——就是**利息**。名义上是买卖，实则是借款担保（1万元借款，1年后还本付息1.2万）。

#### （二）让与担保的法律结构

1. **主合同**：**民间借贷合同**
2. **从合同（让与担保协议）**：无名合同，包含两层意思表示
   - **名为让与**（买卖合同）→ **双方通谋虚假意思表示，无效**（不发生所有权变动）
   - **实为担保**（隐藏行为）→ **有效**，类推适用典型担保规则

#### （三）先让与担保 vs 后让与担保

| 类型 | 是否完成公示 | 法律效果 |
| :--- | :--- | :--- |
| **先让与担保** | 不动产已**登记** / 动产已**交付** | 债权人享有**优先受偿权**（类推适用担保物权） |
| **后让与担保** | 未登记 / 未交付 | 债权人仅为**普通债权人**（可拍卖受偿，但**无优先受偿权**） |

#### （四）留押条款（流押条款）vs 让与担保

| 对比项 | 让与担保协议 | 流押条款 |
| :--- | :--- | :--- |
| **时间** | 履行期限**届满前** | 履行期限**届满前** |
| **形态** | 协议（整个合同） | 条款（合同中的一句话） |
| **内容** | 以标的物担保债权 | 约定“债务不履行时，标的物**直接归债权人所有**” |
| **效力** | 协议整体**有效**（名为让与无效，实为担保有效） | **条款无效**（剥夺市场定价权） |

> **让与担保经典例题**：自然人甲与乙签订了年利率30%、为期一年、1000万元的借款合同（民间借贷，利率超过LPR四倍）。双方又签订了房屋买卖合同，约定甲将房屋卖给乙，甲需在一年内以该房款分六期**回购**房屋（**先卖后买，差价是利息，典型让与担保**）。约定“如果甲不回购，乙有权**直接取得房屋所有权**”（**流押条款，无效**）。甲未按约定回购，也未将房屋过户给乙（**后让与担保**）。
>
> **问**：下列说法正确的是？
>
> A. 甲乙之间是借贷合同关系，不是房屋买卖合同关系——**正确**（双方通谋虚假意思表示，买卖合同无效）
>
> B. 应在不超过LPR四倍以内承认借款利息——**正确**（超过部分无效）
>
> C. 乙不能获得房屋所有权——**正确**（流押条款无效）
>
> D. 因甲未按约偿还借款，应承担违约责任——**正确**（违约合法有效的民间借贷合同）
>
> **答案：A、B、C、D**


### 第五段：让与担保 vs 流押条款 vs 清偿型以物抵债

| 对比项 | 让与担保 | 流押条款 | 清偿型以物抵债 |
| :--- | :--- | :--- | :--- |
| **签订时间** | 履行期限**届满前** | 履行期限**届满前** | 履行期限**届满后** |
| **内容** | 以标的物担保债权 | 债务不履行时直接归债权人 | 以他种给付代替原给付 |
| **法律效果** | 名为让与无效，实为担保有效 | **条款无效** | **有效**（诚信表现） |


### 第六段：本专题总结

翻回二百八十一页：

1. **主体**：先看主体——区分商业借贷（有金融机构）和民间借贷（无金融机构）
2. **内容**：
   - 贷款人的权利（监督用途）和义务（禁止砍头息）
   - 自然人借贷：**实践性合同**；未约定利息→视为无利息；利息≤LPR四倍
3. **让与担保**（核心难点）：**名为让与，实为担保**；**先卖后买，差价是利息**；先让与担保（有优先受偿权）vs 后让与担保（普通债权人）
