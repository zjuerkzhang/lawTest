

## 第1部分：对照《民法典》赠与合同提取备考提纲


### 一、赠与合同的定义与性质（民法典第657条）

> **第657条**：赠与合同是赠与人将自己的财产无偿给予受赠人，受赠人表示接受赠与的合同。

**四大属性**：

| 属性 | 说明 |
| :--- | :--- |
| **双方民事法律行为** | 需赠与人送 + 受赠人接受（两个意思表示合意） |
| **单务合同** | 赠与人只有义务无权利；受赠人只有权利无义务 |
| **无偿合同** | 受赠人无须给付对价 |
| **诺成合同** | 意思表示合意即成立，不是实践性合同 |


### 二、瑕疵担保责任（民法典第662条）

| 规则 | 内容 |
| :--- | :--- |
| **原则** | 赠与财产有瑕疵，赠与人**不承担责任** |
| **例外一** | **附义务的赠与**，在所附义务范围内承担责任 |
| **例外二** | **故意不告知瑕疵或保证无瑕疵**，承担违约责任 |


### 三、任意撤销权（民法典第658条）

> **第658条**：赠与人在赠与财产的权利转移之前可以撤销赠与。经过公证的赠与合同或者依法不得撤销的具有救灾、扶贫、助残等公益、道德义务性质的赠与合同，不适用前款规定。

**行权时间**：**物权变动前**（动产交付前、不动产过户登记前）。

**例外（不可撤销）** ：
1. **公证的赠与**
2. **公益性质的赠与**（救灾、扶贫、助残及科教文卫体新闻等）
3. **道德义务性质的赠与**（直系血亲之间、养子女对生父母、报答救命之恩等）


### 四、法定撤销权（民法典第663条、第664条）

> **第663条**：受赠人有下列情形之一的，赠与人可以撤销赠与：
> 1. 严重侵害赠与人或者赠与人近亲属的合法权益
> 2. 对赠与人有扶养义务而不履行
> 3. 不履行赠与合同约定的义务
>
> 赠与人的撤销权，自知道或者应当知道撤销事由之日起**一年内**行使。

> **第664条**：因受赠人的违法行为致使赠与人死亡或者丧失民事行为能力的，赠与人的继承人或者法定代理人可以撤销赠与。撤销权自知道或者应当知道撤销事由之日起**六个月内**行使。

**行权时间**：**物权变动后**。

**除斥期间**：赠与人本人**1年**；继承人或法定代理人**6个月**。


### 五、穷困抗辩权（民法典第666条）

> **第666条**：赠与人的经济状况显著恶化，严重影响其生产经营或者家庭生活的，可以**不再履行**赠与义务。

**性质**：**暂时性抗辩权**（经济状况恢复后可以继续履行）

### 第一段：赠与合同概述

好，接下来咱们来看专题二十二，赠与合同。这个合同是客观题和主观题都会考查到的，在旁边拿笔写上**主观题**。

赠与合同的考点一共就**四大属性、三大权利**。

先来看赠与合同的基本原理和含义。赠与合同是指赠与人无偿将自己的财产转移给受赠人，**受赠人表示接受**的合同。这一定义的重点在后段——受赠人表示接受。你赠与人送，受赠人得要，它需要两个意思表示合意，合同才能成立。于是，赠与属于**双方民事法律行为**。

但是，赠与人把自己的财产转移给受赠人，受赠人表示接受以后，**受赠人只有权利没有义务，赠与人只有义务没有权利**，不存在对待给付义务关系。因此，赠与合同是最典型的**单务合同**。

而第三，赠与属于典型的**无偿合同**，和买卖合同最大的区别就在这儿了。

第四，赠与属于**诺成性合同**。整个民商法当中实践性合同一共就四个（定金、自然人借贷、保管、借用），赠与当然是诺成性合同——我送你要，合同就成立了。【补充：民法典第657条、第658条】

**赠与合同的四个属性**：**双方民事法律行为、单务、无偿、诺成合同**。


### 第二段：瑕疵担保责任（违约责任）

因为单务、因为无偿，对赠与人的要求不能太高。对于违约责任，赠与人原则上承担**过错责任**——没有过错，不用承担责任。

**原则**：赠与的财产有瑕疵的，赠与人**不承担责任**（民法典第662条第1款）。

**两个例外**（民法典第662条第2款）：

1. **附义务的赠与**：在所附义务范围内承担责任。
2. **故意不告知瑕疵或保证无瑕疵**：要承担违约责任。

> **例题**：老孟把手机送给别人，明知手机充电会漏电，却对受赠人说“没什么问题”。受赠人充电被电伤——老孟应承担**违约责任**（故意不告知瑕疵）。


### 第三段：任意撤销权（民法典第658条）

**行权时间点**：**物权变动前**（动产交付前、不动产过户登记前）。

**一个原则，三个例外**：原则可以随便撤，但三种情况不能撤（基于**公序良俗**）：

| 例外情形 | 说明 |
| :--- | :--- |
| **公证的赠与** | 仪式感满满，已经慎重考虑 |
| **公益性质的赠与** | 救灾、扶贫、助残及科学、教育、文化、卫生、体育、新闻等 |
| **道德义务性质的赠与** | 直系血亲之间、养子女对生父母、报答救命之恩等 |

> **公益赠与例题**：某员工在公司年会中奖，公司资助其子女**教育费用**，如员工离职则资助失效。
>
> **分析**：“如员工离职”是发生与否不确定的事实，影响合同失效——**附解除条件的赠与合同**。
>
> A. 甲公司与魏某成立附条件赠与——**正确**（附解除条件）
>
> B. 甲公司与魏某成立附义务赠与——**错**（附义务是限制用途，不是影响生效/失效）
>
> C. 如魏某次年离职，条件成就，赠与合同失效，甲公司无给付义务——**正确**
>
> D. 如魏某次年未离职，甲公司在给付前可撤销——**错**（公益性质的赠与，不准行使任意撤销权）


### 第四段：法定撤销权（民法典第663条）

**行权时间点**：**物权变动后**（已经交付或过户之后）。

**一个原则，三个例外**：原则上要不回来了，除非受赠人**忘恩负义**。

| 例外情形 | 说明 |
| :--- | :--- |
| **严重侵害赠与人或赠与人近亲属的合法权益** | “严重”需达到重伤或死亡 |
| **对赠与人有扶养义务而不履行** | 包括夫妻之间的扶养、子女对父母的赡养等 |
| **不履行赠与合同约定的义务** | 附义务的赠与，受赠人不履行义务 |

> **严重侵害例题**：甲表示将赠与乙5000元，已实际交付乙2000元。后乙在与甲之子丙的纠纷中，将丙殴成**重伤**。甲可以撤销对乙的赠与——**可以**（严重侵害赠与人近亲属）。

**行权主体与除斥期间**（民法典第663条第2款、第664条）：
- **赠与人本人**行使：自知道或应当知道撤销事由之日起**1年**
- **赠与人的继承人或法定代理人**行使：自知道或应当知道撤销事由之日起**6个月**


### 第五段：附条件 vs 附义务的赠与

| 对比项 | 附条件的赠与 | 附义务的赠与 |
| :--- | :--- | :--- |
| **判断标准** | 是否影响合同的**生效或失效** | 是否限制标的物的**用途** |
| **关键词** | “如果……”、“如……”（影响生效/失效） | “专款专用”、“专门用于”、“只能自用” |
| **法律效果** | 条件不成就 → 合同不生效/失效 | 不履行义务 → 赠与人可**法定撤销** |


### 第六段：穷困抗辩权（民法典第666条）

**含义**：赠与人的经济状况**显著恶化**，严重影响其生产经营或者家庭生活的，**可以不再履行**赠与义务。

**理论基础**：**法不强人所难**——赠与人自己都过不下去了，不能强迫他必须做好人。

**特点**：**暂时性抗辩权**——经济状况恢复后，可以继续履行。

> **穷困抗辩权例题**：甲公司与乙学校（公益）签订捐赠1000万元的协议，约定专门用于购买书籍（公益性质+附义务）。甲公司已支付300万后**资金链断裂**，乙学校请求继续支付剩余700万。
>
> **分析**：公益性质的赠与不准行使任意撤销权，但可以行使**穷困抗辩权**。
>
> A. 甲公司可申请撤销尚未支付的700万——**错**（公益赠与不准任意撤销）
>
> B. 乙学校无权请求甲公司继续支付——**错**（乙学校有权请求，但甲有抗辩权）
>
> C. 甲公司可请求返还已支付的300万——**错**（没有法定撤销事由）
>
> D. 甲公司可以不再支付剩余的700万——**正确**（穷困抗辩权）


### 第七段：本专题总结

赠与合同 **“四大属性、三大权利”** ：

| 类别 | 内容 |
| :--- | :--- |
| **四大属性** | 双方民事法律行为、单务、无偿、诺成合同 |
| **三大权利** | 任意撤销权、法定撤销权、穷困抗辩权 |

**撤销权对比**：

| 对比项 | 任意撤销权 | 法定撤销权 |
| :--- | :--- | :--- |
| **行权时间** | **物权变动前** | **物权变动后** |
| **原则** | 可**任意**撤销 | 原则上**不可**撤销 |
| **例外** | 三种不能撤（公证/公益/道德义务） | 三种可以撤（严重侵害/不履行扶养义务/不履行约定义务） |
| **除斥期间** | 无（合同未履行） | 本人1年；继承人/法定代理人6个月 |
