## 第1-1部分：对照《民法典》违约责任提取备考提纲


### 一、违约责任概述（民法典第577条）

**违约责任**：当事人一方不履行合同义务或者履行合同义务不符合约定的，应当承担继续履行、采取补救措施或者赔偿损失等违约责任。


### 二、双方违约与与有过失（民法典第592条）

| 制度 | 法典依据 | 含义 |
| :--- | :--- | :--- |
| **双方违约** | 第592条第1款 | 双方都违约，各自承担**相应的**违约责任 |
| **与有过失**（过失相抵/混合过错） | 第592条第2款 | 非违约方对损害发生也有过错的，**减轻**违约方的责任 |


### 三、违约责任的归责原则

| 归责原则 | 含义 | 适用合同类型 |
| :--- | :--- | :--- |
| **无过错责任原则**（原则） | **不考虑过错**，没有过错也要承担违约责任 | 绝大多数合同 |
| **过错责任原则**（例外） | **考虑过错**，有过错才承担违约责任 | 赠与合同、委托合同、保管合同、仓储合同、客运合同（承运人对旅客自带物品） |


### 四、违约责任的构成要件

| 要件 | 说明 | 法典依据 |
| :--- | :--- | :--- |
| **1. 存在合法有效的合同** | 有约可违。合同无效/被撤销/不成立→不能追究违约责任 | — |
| **2. 存在违约行为** | 预期违约 + 实际违约（不履行/不适当履行） | 第578条、第577条 |
| **3. 不存在法定或约定的免责事由** | 法定免责事由——**不可抗力**（三个不：不能预见、不能避免、不能克服） | 第180条、第590条 |


### 五、违约行为的分类体系

```
违约行为
├── 预期违约（履行期届满前）
│   ├── 明示毁约
│   └── 默示毁约（如迟迟未开工）
│
└── 实际违约（履行期届满后）
    ├── 不履行（根本违约）
    │   ├── 拒绝履行（主观）
    │   └── 履行不能（客观）
    ├── 不适当履行
    │   ├── 瑕疵履行（质量问题）→ 单纯违约
    │   ├── 加害给付 → 既违约又侵权→**择一选择**（第186条）
    │   ├── 部分履行（少履行）
    │   └── 迟延履行 → 需履行**法定催告义务**（第563条第3项）
    └── 债权人迟延受理 → **提存**（第570条）
```


### 六、法定催告义务（民法典第563条第3项）

迟延履行主要债务，非违约方必须**催告**，在合理期限内仍未履行的，才能**解除合同**并追究违约责任。不能一迟延就解除，要给违约方改过自新的机会。


### 七、不可抗力（民法典第180条、第590条）

**不可抗力**：不能预见、不能避免且不能克服的客观情况。

**法律效果**：
- 因不可抗力不能履行合同的，根据影响**部分或全部免除**责任
- 应当**及时通知**对方（诚信原则）
- 债务人**迟延履行后**发生不可抗力的，**不免除**其违约责任

## 第1-2部分：对照《民法典》违约责任五种形式提取备考提纲


### 一、继续履行（民法典第577条、第579-581条）

| 债务类型 | 能否继续履行 | 法典依据 |
| :--- | :--- | :--- |
| **金钱之债** | ✅ **必须继续履行**（金钱之债永不灭） | 第579条 |
| **非金钱债务**——原则上可以继续履行 | 原则：✅ | 第580条第1款 |
| **非金钱债务**——例外（不能继续履行） | ①法律上或事实上不能履行（**特定之债**毁损灭失或一物数卖）；②债务标的不适于强制履行（**劳务之债**）；③履行费用过高；④债权人在合理期限内未要求履行 | 第580条第1款 |

**合同僵局**（民法典第580条第2款，新增）：
- 违约方起诉解除合同的要件：①**非恶意**；②继续履行**显失公平**；③守约方**权利滥用**
- 解除合同**不影响违约责任的承担**（解除权+违约责任）


### 二、采取补救措施（民法典第582条）

五种方式：**修理、更换、重作、退货、减价**。重点掌握**减价**。


### 三、损害赔偿金（第一金，民法典第584条）

#### （一）可得利益损失

| 类型 | 典型场景 |
| :--- | :--- |
| **生产利润损失** | 生产设备迟延交付导致停工 |
| **经营利润损失** | 租赁经营合同，迟延交房导致晚开业 |
| **转售利润损失** | 连环买卖，中间商赚差价 |

#### （二）三个限制规则（减法）

| 规则 | 含义 | 法典依据 |
| :--- | :--- | :--- |
| **可预见性规则** | 间接损失不赔——违约方订立合同时无法预见的损失不赔 | 第584条但书 |
| **减损规则** | 扩大损失不赔——守约方未采取合理措施防止损失扩大的，扩大部分不赔 | 第591条 |
| **损益相抵规则** | 因祸得福要扣减——守约方因同一违约行为获有利益的，从损失赔偿额中扣除 | 合同编通则司法解释第63条（**新增，从未考过**） |

## 第1-3部分：对照《民法典》违约责任五种形式提取备考提纲


### 一、违约金（民法典第585条）

**性质**：当事人约定的违约赔偿方式。

**司法调整**（第585条第2款）：
- 约定违约金**低于**实际损失 → 可请求**增加**
- 约定违约金**过分高于**实际损失 → 可请求**适当减少**
- 判断标准：实际损失 × **130%**

**法院权限**：可以**主动释明**（提示违约金可能过高），但不能**主动调整**（须当事人请求）。

**惩罚性违约金**（新增，从未考过）：
- 判断：合同明确写明“惩罚性违约金”或有两个违约金数额（数大者为惩罚性）
- **原则上不调整**（恶意违约的惩罚）
- 例外：可能导致违约方破产、影响社会稳定的，可适当调整


### 二、定金（民法典第586-588条）

#### （一）定金合同的性质（第586条）

| 性质 | 规则 |
| :--- | :--- |
| **要式合同** | 必须采用**书面形式** |
| **实践性合同** | 自**实际交付定金时成立** |
| **数额限制** | 不得超过主合同标的额的**20%**，超过部分**不产生定金效力**（按不当得利处理） |

#### （二）定金法则（第587条）

根本违约才能适用定金罚则——**给付方**根本违约，**无权请求返还定金**；**收受方**根本违约，应当**双倍返还定金**。

#### （三）违约金与定金的关系（第588条）

当事人既约定违约金，又约定定金的，一方违约时，对方可以选择适用违约金或者定金条款——**择一选择，不得并用**。

| **迟延履行——法定催告义务** | **✅ 主观题** | 常考 | 给出“迟延履行但未催告”的场景，问能否解除合同 | 承租人迟延交租，出租人未催告直接要求搬离。问：能否解除？（不能，须先催告） |

| **迟延履行后不可抗力不免责** | 客观题 | 常考 | 给出“迟延履行后发生不可抗力”的场景，问是否免责 | 开发商本应5月1日交房，6月1日才交，6月15日地震房屋毁损。问：开发商是否免责？（不免责） |

> **老师原话总结**：
> - “违约责任三大要件——**存在合法有效的合同 + 存在违约行为 + 不存在法定或约定的免责事由**。”
> - “违约责任的归责原则——**无过错责任原则是原则**，不考虑过错。例外（赠与、委托等五个合同）是**过错责任**。”
> - “违约责任的相对性——**谁签合同，谁来承担违约责任**。合同以外的第三人不需要承担。”
> - “加害给付——**既违约又侵权，择一选择**（民法典第186条）。”
> - “迟延履行——必须履行**法定催告义务**，不能一迟延就解除合同。”
> - “不可抗力——**三个不**：不能预见、不能避免、不能克服。**迟延履行后发生的不可抗力不免责**。”
> - “民事责任三大责任——**缔约过失责任（信赖利益）、违约责任（履行利益）、侵权责任（固有利益）**。”

> - “定金法则——**根本违约**方可适用（致使不能实现合同目的）。”
> - “违约金和定金——**只能择一选择，不能并用**（禁止重复评价）。”

### 第一段：违约责任概述

好，接下来咱们来看专题十九，违约责任。这个专题客观题和主观题都会考察，所以在旁边拿笔写上**主观题**。

这个专题的核心考点就两个：一个是违约责任的**三大要件**，一个是违约责任的**五种形式**。历届同学重点难点往往总是错的，其实是违约责任的五种形式，后面咱们会专门展开讲述。

二百四十八页下方先来看违约责任的基本原理。所谓的违约责任，是指当事人**不履行或不适当履行**合同义务而应承担的民事责任。它属于民事责任三大责任之一——**缔约过失责任、违约责任、侵权责任**。

【补充】缔约过失责任前面已经讲过了，侵权责任后边第七编会讲。


### 第二段：双方违约与与有过失（民法典新增）

违约责任的特征主要有两个：

**第一，双方违约**（民法典第592条第1款）：合同当事人双方都违反了各自的义务，一个巴掌拍不响，双方应各自承担相应的违约责任。做题时务必看清楚是否双方当事人都有违约行为。

**第二，与有过失**（民法典第592条第2款）：又称**过失相抵、混合过错**。虽然非违约方没有违约，但对其自身损害的发生也有过错，应当相应减轻违约方的违约责任。

> **与有过失例题**：某同学拿到学校发的借记卡后，把密码写在卡背面用胶条粘住。后来卡丢了被人捡到异地盗刷1万元。同学将银行告到法院要求返还1万元。
>
> **分析**：银行虽然违约了，但同学自己也有过错（把密码写在卡背面）。最终法院判决银行返还8000元，同学自己承担2000元损失——**与有过失**，相应减轻银行的违约责任。


### 第三段：违约责任的相对性原理（核心考点）

**违约责任的相对性原理**——就是**合同相对性原理**在违约责任问题上的应用：**谁签合同，谁来承担违约责任**。合同以外的第三人不需要承担违约责任，合同当事人也不能追究合同以外的第三人的违约责任。

> **违约责任例题**：我把戒指卖给你，咱俩约定明天交戒。结果当晚马峰把戒指抢走扔到秦淮河里了。第二天我无法交付戒指。
>
> **分析**：你只能追究我的**违约责任**（咱俩有合同）；你不能追究马峰的违约责任（你跟马峰没合同）。我的损失由我追究马峰的**侵权责任**。

> **经典真题（连环交易）** ：甲公司告知乙公司为回馈老客户，近期会赠送礼品给乙（甲乙之间是**赠与合同**）。邻居丙购买了一台电脑，供货商丁的配送员送货时看错门牌号码，将本该给丙的电脑送至乙家（丙丁之间是**买卖合同**）。乙的家人误以为是甲公司赠送的礼品而接收（**善意不当得利**）。当晚家中失火导致电脑被毁（风险负担，善意不当得利人仅返还现存利益，电脑被毁则无需赔偿）。
>
> **问**：下列说法正确的是？
>
> A. 乙对丙不负赔偿义务——**正确**（善意不当得利人不承担风险）
>
> B. 丙有权要求丁交付相同型号相同配置的电脑——**正确**（丙追究丁的违约责任）
>
> C. 丙有权要求甲公司向自己交付乙的礼品——**错**（合同相对性，丙与甲没合同）
>
> D. 乙对丁承担连带责任——**错**（乙无需承担责任）
>
> **答案：A、B**


### 第四段：违约责任的归责原则

**归责原则**就是请求行为人承担责任的理由——你凭什么让他承担违约责任？

**原则上**：违约责任采用**无过错责任原则**（又称**严格责任原则**）——不考虑过错，没有过错也要承担违约责任。

> **无过错责任例题**：孟某将手表卖给王某，约定第二天交表。当晚邻居将手表偷走扔进长江。第二天王某要追究孟某的违约责任，孟某说“我没过错，是邻居偷的，你应该追究邻居”。
>
> **分析**：孟某**仍要承担违约责任**（无过错责任原则）。孟某承担后可以追究邻居的侵权责任。

**例外**（过错责任原则）：下列五个合同采用过错责任——**赠与合同、委托合同、保管合同、仓储合同、客运合同**（承运人对旅客自带物品毁损）。考试重点掌握**赠与合同**和**委托合同**。

为什么这五个合同是例外？因为它们要么是**无偿的**（赠与），要么是**单务的**（赠与人只有义务没有权利），对行为人的要求不能太高。

> **赠与合同**：白拿东西，只要赠与人没有故意不告知瑕疵或保证无瑕疵，就不用承担违约责任。
>
> **委托合同**：律师事务所与当事人签的委托合同，只要律师在提供法律服务过程中**没有过错**，就不用承担赔偿责任。有过错才赔（如上海“代书遗嘱”案，因律师事务所只派一名律师导致遗嘱无效，最终赔偿800多万）。


### 第五段：违约责任的构成要件（核心考点★★★★★）

**三大要件，同时具备，缺一不可**（民法典第577条）：

**要件一：存在合法有效的合同**（有约可违）——合同被撤销、被宣告无效、不成立、确定不发生效力的，只能追究**缔约过失责任**，不能追究违约责任。

**要件二：存在违约行为**（重点★★★）——根据是否到达履行期，分为**预期违约**和**实际违约**。

#### （一）违约行为的分类体系

| 分类 | 具体类型 | 说明 |
| :--- | :--- | :--- |
| **预期违约**（履行期届满前） | 明示毁约 / 默示毁约 | 如开发商迟迟未开工，按期交房无望——非违约方可以**解除合同同时追究违约责任** |
| **实际违约**（履行期届满后） | **不履行**（根本违约） | ①拒绝履行（主观上拒不履行）；②履行不能（客观上无法履行）——可解除合同+追究违约责任 |
| | **不适当履行** | ①**瑕疵履行**（质量问题）——单纯违约；②**加害给付**（既违约又侵权→**择一选择**，民法典第186条）；③部分履行（少履行；多履行不构成违约）；④迟延履行（需履行**法定催告义务**，民法典第563条第3项） |

**加害给付**：产品质量缺陷既造成合同标的物本身的损害，又造成人身或合同标的物以外的财产损害——既违约又侵权，**一行为两责任，择一选择**（民法典第186条）。

> **加害给付例题**：买电视机，电视机不显示图像 → **瑕疵履行**（单纯违约）；买电视机，电视机**爆炸**炸坏了墙 → **加害给付**（既违约又侵权），可选择主张违约责任或侵权责任。

**迟延履行的法定催告义务**（民法典第563条第3项）：一方迟延履行主要债务，经催告后在合理期限内仍未履行的，非违约方才能单方解除合同。不能一迟延就解除——要给人改过自新的机会。

**要件三：不存在法定或约定的免责事由**

**法定免责事由**：**不可抗力**（民法典第180条第2款）——三个不：**不能预见、不能避免、不能克服**的客观情况（洪水、海啸、地震、泥石流、雷击、失火等）。

不可抗力的法律效果（民法典第590条）：
- 根据不可抗力的影响，**部分或全部免除**责任
- 当事人应当**及时通知**对方（诚信原则/附随义务）
- **提前履行后发生不可抗力的，不免除违约责任**——本来就应该在不可抗力发生前履行。

> **不可抗力例题**：双方约定2025年5月1日交房，4月15日发生地震，房屋毁损——**因不可抗力致使不能实现合同目的**，双方均可解除合同，但**谁也不能追究谁的违约责任**（存在法定免责事由）。如果开发商6月1日才交房，6月15日发生地震——**不免责**（本来就应该在地震前交房，已构成违约）。


### 第六段：三大民事责任的对比总结

二百五十一页的表格是必考点——**缔约过失责任、违约责任、侵权责任**的对比：

| 对比项 | 缔约过失责任 | 违约责任 | 侵权责任 |
| :--- | :--- | :--- | :--- |
| **构成要件** | 缔约、过失、责任、因果（4要件） | 合法有效合同 + 违约行为 + 无免责事由（3要件） | 过错归责：4要件（侵害行为+损害后果+因果关系+过错）；无过错归责：3要件（不考虑过错） |
| **赔偿范围** | **信赖利益损失**（订约费用+订约机会） | **履行利益损失**（可得利益） | **固有利益损失**（恢复原状） |
| **是否有合同** | 有（缔约过程中） | 有（合同关系中） | 一般无 |

**加害给付**（产品责任）最容易出现既违约又侵权——择一选择。

### 第一段：违约责任五种形式概述

好，接下来咱们来看第二个问题，叫做违约责任的五种形式。这个部分是历届同学都容易忽视，错误率非常高的。二六届的同学一定要引以为戒，要重点学习。

违约责任一旦符合了前面的三大构成要件，守约方具体要追究对方的何种形式的违约责任呢？五种方式：**第一，继续履行；第二，采取补救措施；接下来就是三金——赔偿金、违约金和定金**。


### 第二段：继续履行（民法典第577条，每年必考）

先来看第一个，几乎每年必考事项——**继续履行**。它只是一个选项，不是一个题目。

**继续履行**又称实际履行、强制实际履行：守约方有权请求法院或仲裁机构强制违约方按照合同约定的给付内容履行义务，而不得以支付违约金或赔偿损失代替履行。

> **继续履行例题**：开发商把房屋卖给老百姓，签合同时房价100万，两年后交房时涨到300万。开发商说“我退本金加利息，再额外赔50万，房子不交了”。老百姓说“我要房子”。咋办？
>
> **答**：**继续履行**。老百姓买房的目的是取得房屋所有权，不是为了要赔偿金。

#### （一）金钱之债（货币之债）——原则上必须继续履行

**金钱之债永不灭**（民法典第579条）——欠债还钱，天经地义，原则上不存在履行不能的问题。

#### （二）非金钱债务的继续履行（民法典第580条，绝对重点）

**不能请求继续履行的例外**：

**第一种例外：法律上或事实上不能履行**（民法典第580条第1款第1项）——就是**特定之债**。两个场景：
- **事实上不能履行**：标的物已经毁损灭失了。比如戒指在交付前被第三人抢走扔河里了，无法继续履行。
- **法律上不能履行**：标的物已经发生了物权变动（一物数卖，已经交付给了第三人）。比如戒指卖给了三个人，已经交付给了白浪涛，白浪涛取得了所有权——对其他人已经“法律上不能履行”。

**特定之债**：不动产（房屋）一定是特定之债；二手动产（二手戒指、二手手机）也一定是特定之债。

**第二种例外：债务标的不适于强制履行**（民法典第580条第1款第2项）——就是**劳务之债**（有人身依附性）。比如讲课、演出、家教、保姆干活等，不能直接强制人身，但可以**间接强制**——找第三人替代履行，费用由违约方负担（民法典第581条）。

> **劳务之债例题**：家里保姆临时有事不来，雇主不能强制保姆必须来干活。可以临时从家政公司另请保姆，产生的额外费用从保姆工资里扣——**间接强制**。

#### （三）合同僵局——违约方起诉解除合同（民法典第580条第2款，新增，从未考过★★★★★）

**什么叫合同僵局？** 持续性定期合同（如租赁合同、培训合同等）中，违约方不是恶意违约，继续履行对违约方显失公平，守约方拒绝解除合同构成权利滥用——此时允许违约方向法院起诉解除合同，但不影响违约责任的承担。

> **合同僵局例题（租赁合同）** ：老孟租了左宁的门面房开奶茶店，租期20年。结果两个月只卖出去一杯奶茶，实在做不下去了，想解除合同。左宁说房子没问题，你必须继续卖奶茶。老孟陷入僵局。
>
> **分析**：允许老孟（违约方）起诉解除合同，但要对左宁承担违约责任（赔偿损失）。
>
> **三个要件**：①违约方**非恶意**；②继续履行对违约方**显失公平**；③守约方**权利滥用**（拒绝解除合同违反诚信原则）。

**合同僵局 vs 情势变更 vs 显失公平（民法典中公平原则的三个应用）** ：

| 制度 | 适用阶段 | 规则 |
| :--- | :--- | :--- |
| **显失公平**（可撤销） | **订立**合同时 | 受损方撤销权（或诉或裁） |
| **情势变更**（民法典第533条） | **履行**过程中（客观原因） | 协商前置；协商不成可请求变更或解除 |
| **合同僵局**（民法典第580条第2款） | **履行**过程中（主观原因，违约方原因） | 违约方起诉解除，不影响违约责任 |


### 第三段：采取补救措施（民法典第582条）

**五种具体方式**：**修理、更换、重作、退货、减价**。重点是**减价**。

> **采取补救措施例题（2023年真题）** ：王某去李某手机店购买新款手机。手机用了一个月后发生**爆炸**，致其受伤（**加害给付**——既违约又侵权，择一选择）。经查，该手机有明显使用痕迹，是**二手手机**冒充新款手机（**欺诈**）。
>
> **问**：下列说法正确的是？
>
> A. 王某可以以显失公平为由撤销合同——**错**（理由应为**欺诈**）
>
> B. 王某可以主张侵权责任——**正确**（加害给付，择一选择）
>
> C. 王某可以主张违约责任——**正确**（加害给付，择一选择）
>
> D. 王某可以继续保留手机，但请求减少手机价格——**正确**（**减价**）
>
> **答案：B、C、D**


### 第四段：损害赔偿金（第一金）

**损害赔偿金**：违约方以支付金钱的方式弥补受害方因违约行为而遭受损失的责任形式。核心问题：**赔多少钱？怎么算？**

#### （一）可得利益损失的计算（民法典第584条）

可得利益是指合同履行后可以获得的利益。三种计算方法：

| 类型 | 典型场景 | 计算方式 |
| :--- | :--- | :--- |
| **生产利润损失** | 买设备做生产，因卖方迟延交货导致停工 | 停产期间的预期利润 |
| **经营利润损失** | 租门面经营，因出租人迟延交房导致晚开业 | 晚开业期间的预期利润 |
| **转售利润损失** | 中间商赚差价（连环买卖） | 转售价与进价的差额 |

> **转售利润例题（2019年真题）** ：超市向农民采购老母鸡，采购价100元/只，零售价250元/只（差价150元）。合同约定每季度供应20只。第二季度超市只采购到10只。
>
> **问**：超市可就少交付的10只鸡主张多少损害赔偿？
>
> **答**：10只 × 150元 = **1500元**（转售利润损失）。

#### （二）损害赔偿金的三个限制规则（减法！民法典第584条、第591条、合同编通则司法解释第63条）

考试重点不是做加法，而是做减法！

**第一，可预见性规则**（民法典第584条但书）：赔偿额不得超过违约方在**订立合同时**预见到或应当预见到的因违约可能造成的损失。**间接损失不赔**。

> **可预见性例题**：老孟买电脑是为了给众和写讲义，电脑坏了导致迟延交付讲义，被众和索赔10万。老孟要求卖电脑的公司赔这10万。
>
> **分析**：卖电脑的公司卖电脑时不可能预见到老孟买电脑是给众和写讲义的——**间接损失不赔**，只能赔电脑本身的损失。

**第二，减损规则**（民法典第591条）：守约方应当采取适当措施防止损失扩大。**扩大损失不赔**（诚信原则）。

> **减损例题**：房屋漏雨，出租人已打钱让承租人修房，承租人无动于衷，导致财物被泡坏。出租人要不要赔被泡坏的财物？
>
> **答**：**不赔**。守约方应当采取合理措施减损（放水桶、找人修房）——扩大损失不赔。

**第三，损益相抵规则**（合同编通则司法解释第63条，**新增，从未考过★★★★★**）：守约方因同一违约行为而获有利益的，应当从损失赔偿额中**扣除该部分利益**——**禁止因祸得福**。

> **损益相抵例题（国产牛肉案）** ：合同约定买卖进口牛肉，违约金10万。卖方交了国产牛肉（违约），恰逢国外发生疯牛病，进口牛肉大跌，国产牛肉大涨。买方把国产牛肉卖了8万。
>
> **问**：卖方应赔多少？
>
> **答**：10万 - 8万 = **2万**（损益相抵）。

> **损益相抵例题（买股票案）** ：马峰委托老孟买中石油股票（100万），约定买错造成损失赔10万。老孟摔了一跤买成了中石化，但中石化反而涨了，股票值8万。
>
> **问**：老孟应赔多少？
>
> **答**：10万 - 8万 = **2万**（因祸得福，损益相抵）。

#### （三）继续履行与损害赔偿金可以并用

继续履行后还有其他损失的，仍然可以请求赔偿损失——二者不矛盾、不冲突，可以并用。


### 第五段：本小节总结

**继续履行**：金钱之债永不灭；非金钱债务中，特定之债（毁损灭失或一物数卖）和劳务之债（人身依附性）不能继续履行。**合同僵局**（违约方起诉解除）是新增考点，从未考过。

**采取补救措施**：修理、更换、重作、退货、减价——重点是**减价**。

**损害赔偿金**：
- 可得利益损失 = 生产利润损失 + 经营利润损失 + 转售利润损失
- 三个限制规则（**减法**）：可预见性规则（间接损失不赔）、减损规则（扩大损失不赔）、损益相抵规则（因祸得福要扣减，**新增**）

### 第一段：违约金——性质与司法调整

好，接下来我们来看违约金。违约金的考点，书的二百五十五页下方，重点就两个：一个叫**性质**，一个叫**司法调整**。这里边唯有一个没有考过的考点就是**惩罚性违约金**，它是个新增制度。

翻过来书的二百五十六页，先来看啥叫违约金。违约金是指依据法律规定或当事人约定，一方不履行或不适当履行合同时，应向对方支付的一定数额的金钱。它跟前面的损害赔偿金一样都是赔钱，这是相同点。不同点就是，违约金是当事人自己约定的，并且在合同当中会明示“违约金”三个字。

约定的方式主要就两种：第一种是约定一个具体的数额；第二种是约定一个计算方法（如“每迟延一天，支付合同总价款的万分之一”）。

#### （一）司法调整——违约金过高或过低（民法典第585条第2款）

当事人约定的违约金过分高于造成的损失的，人民法院或者仲裁机构可以根据当事人的请求予以**适当减少**。约定的违约金低于造成的损失的，人民法院或者仲裁机构可以根据当事人的请求予以**增加**。

**调整规则**（以实际损失100万为例）：

| 情形 | 约定违约金 | 处理 |
| :--- | :--- | :--- |
| **低于实际损失** | 80万 | **上调**至100万（填补实际损失） |
| **高于但不过分** | 120万（≤130万） | **不调整**，适用120万 |
| **过分高于** | 150万（>130万） | **下调**至100-130万之间（法官自由裁量） |

> **判断标准**：实际损失 × 130% = 分界线。超过部分视为“过分高于”。

**法院可以主动释明，但不能主动调整**——法官可以向违约方提示违约金可能过高，但不能主动调整，必须等当事人提出调整请求。

> **违约金计算例题（历年真题）** ：甲乙签订一份买卖合同，约定违约方应向对方支付**18万元**违约金。后甲违约给乙造成损失**15万元**。
>
> **分析**：15万 × 130% = 19.5万。约定18万 < 19.5万，虽高于实际损失但**并不过分**，直接适用18万。
>
> **问**：下列哪一表述正确？
>
> **答**：甲应向乙支付违约金18万元，不再支付其他费用或者赔偿损失。

#### （二）惩罚性违约金（新增，从未考过★★★★★）

**判断标准**：①合同中明确写明“惩罚性违约金”；②合同中有两个违约金数额（一个数小、一个数大），数大的是惩罚性违约金。

> **惩罚性违约金例题（最高人民法院判例）** ：甲乙签订钢材买卖合同，约定一方违约支付违约金100万（赔偿性）。甲第一次违约后，乙起诉并查封甲账户。甲请求和解：乙撤诉、解除查封，再给三个月宽限期，若三个月后仍不交付，甲愿支付**500万**违约金（惩罚性）。乙撤诉并解除查封。三个月后甲仍不履行。
>
> **原则**：**原则上不调整**——甲恶意违约（利用宽限期转移财产），五百万违约金照赔。
>
> **例外**：如果甲公司赔500万就会破产，影响500个员工和500个家庭，法院可基于**社会效益**适当调整（从未考过）。

**迟延履行的违约金与继续履行可以并用**——买房合同约定迟延交房违约金，交房延期了，可以同时要求支付迟延违约金和继续履行交房义务。


### 第二段：定金（民法典第586-588条）

定金是**金钱担保**（事前的担保包括物保、人保、金钱保）。

#### （一）定金的识别

**两种识别方式**（满足其一即可）：
1. **形式上**：合同明确写明“**定金**”（宝盖头的“定”）
2. **实质上**：合同内容约定的是**定金法则**——给付定金一方根本违约不履行，无权请求返还定金；收受定金一方根本违约不履行，应当**双倍返还**定金

> **定金合同的性质**（民法典第586条）：
> - **要式合同**：必须**书面**形式
> - **实践性合同**：自**实际交付定金时成立**（“定金合同自实际交付定金时成立”——民法典第586条第1款）

#### （二）定金数额的限制（民法典第586条第2款）

定金的数额由当事人约定；但是，不得超过主合同标的额的**百分之二十**，超过部分**不产生定金的效力**，超过部分按**不当得利**处理。

#### （三）定金的类型（学理分类，民法典合同编通则司法解释）

| 类型 | 含义 | 考试重点 |
| :--- | :--- | :--- |
| **立约定金** | 为保证正式缔约而交付的定金（结合预约合同） | ★★★ |
| **成约定金** | 以交付定金作为主合同成立或生效要件 | ★ |
| **违约定金** | 为担保合同履行而交付的定金（**无明确约定时推定为此种**） | ★★★★★ |
| **解约定金** | 以丧失定金为代价而解除合同 | ★ |

#### （四）定金法则（民法典第587条）

**给付定金的一方**不履行债务或者履行债务不符合约定，致使**不能实现合同目的**的，**无权请求返还定金**。

**收受定金的一方**不履行债务或者履行债务不符合约定，致使**不能实现合同目的**的，应当**双倍返还定金**。

**适用条件**：必须是**根本违约**（致使不能实现合同目的），轻微违约不能适用定金罚则。

> **定金例题（司考真题）** ：甲乙约定，甲将100吨汽油卖给乙，合同签订后3天交货，交货后10天内付货款。为担保，约定乙应向甲支付10万元**定金**，“合同在支付定金时生效”。合同订立后，乙未交付定金，甲按期向乙交付了货物，乙接受了货物。
>
> **分析**：定金合同是实践性合同，未交付定金，**定金合同不成立**。但主合同（汽油买卖合同）因甲已交付货物、乙已接受——**履行抵销规则**，主合同**成立并生效**（民法典第490条）。
>
> **问**：下列哪一说法正确？
>
> A. 甲可请求乙支付定金——**错**（定金合同未成立，不能请求履行）
>
> B. 乙未支付定金不影响买卖合同的效力——**正确**（主合同已通过履行抵销成立生效）
>
> C. 甲交付汽油时定金合同生效——**错**（定金合同与主合同标的物不同，不能串台）
>
> D. 甲无权请求乙支付价款——**错**（主合同已成立生效，有权请求支付价款）
>
> **答案：B**

#### （五）违约金与定金的关系（民法典第588条）

**违约金和定金不得并用，只能择一选择**——禁止对同一个违约行为进行重复评价。

> **例题**：买房合同100万，约定定金20万（已交），另约定违约金50万。买方根本违约不买了。
>
> **分析**：一个违约行为，只能选择一种救济方式：
> - 选择**定金**：20万不退还（定金罚则）
> - 选择**违约金**：主张50万违约金（但可能需要司法调整）
>
> **注意**：选择权在守约方，选择对自己最有利的即可。


### 第三段：违约责任五种形式总结

翻回二百五十二页上方，违约责任的五种形式：

| 形式 | 核心要点 |
| :--- | :--- |
| **继续履行** | 金钱之债永不灭；特定之债和劳务之债不能继续履行；合同僵局（违约方起诉解除）是新增 |
| **采取补救措施** | 修理、更换、重作、退货、**减价**（重点） |
| **赔偿金（损害赔偿金）** | 可得利益损失（生产/经营/转售）；三个限制规则（可预见性、减损、损益相抵） |
| **违约金** | 赔偿性/惩罚性；司法调整（过高或过低）；法院可主动释明但不可主动调整 |
| **定金** | 实践性合同；不超过主合同标的额的20%；定金法则（根本违约）；与违约金择一选择 |
