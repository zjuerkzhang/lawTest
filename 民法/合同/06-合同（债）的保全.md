# 合同（债）的保全

## 第1部分：对照《民法典》合同保全相关条款提取备考提纲


### 一、合同的保全概述

**概念**：为防止债务人的财产不当减少而害及债权人的债权，法律赋予债权人以**代位权**和**撤销权**，通过诉讼方式保全债权的制度。

**两种权利**：
1. **债权代位权**（民法典第535条）——**消极诈害债权**（债务人怠于行权）
2. **债权撤销权**（民法典第538条、第539条）——**积极诈害债权**（债务人转移财产）

**性质**：**事后债的保全**，本质上属于担保制度，**突破合同相对性原理**，必须通过**诉讼**方式行使。


### 二、债权代位权诉讼（民法典第535条）

#### （一）构成要件

| 要件 | 内容 | 法典依据 |
| :--- | :--- | :--- |
| **1. 两个债权均合法且到期** | 债权人对债务人的债权 + 债务人对相对人的债权（原则） | 第535条第1款 |
| **2. 债务人怠于行使权利** | 债务人未以**诉讼或仲裁**方式向相对人主张权利 | 合同编通则司法解释第33条 |
| **3. 因果关系** | 债务人怠于行权**害及**债权人的债权 | 第535条第1款 |
| **4. 非人身专属性** | 次债权不属于专属于债务人自身的权利（如**三费**） | 第535条第1款但书 |

#### （二）诉讼当事人

| 角色 | 当事人 | 说明 |
| :--- | :--- | :--- |
| **原告** | **债权人** | 以**自己的名义**起诉 |
| **被告** | **相对人**（次债务人） | — |
| **第三人** | **债务人** | **应当**追加为**无独立请求权的第三人** |

#### （三）相对人的抗辩

- **实体抗辩**：相对人**可以**援用对债务人的抗辩对抗债权人（如诉讼时效抗辩权）
- **程序抗辩**：不得以仲裁条款、协议管辖条款为由进行抗辩

#### （四）法律效果（第537条）

- 相对人**直接向债权人履行**债务（**直接受偿规则**）
- 履行后，债权人与债务人、债务人与相对人之间的**相应债权债务关系消灭**
- 代位权诉讼的提起导致**诉讼时效中断**，两个债均中断
- 必要费用由**债务人**负担


### 三、债权撤销权诉讼（民法典第538条、第539条）

#### （一）无偿诈害债权行为（第538条）——三个要件

1. **合法有效的债权**（**不要求到期**）
2. **诈害债权的行为**（如**赠与**等无偿行为）
3. **因果关系**（诈害行为损害债权人的债权）

#### （二）有偿诈害债权行为（第539条）——四个要件

1. **合法有效的债权**（**不要求到期**）
2. **诈害债权的行为**（以**明显不合理的低价**转让财产等）
3. **相对人知情**（≠恶意串通）
4. **因果关系**

**价格判断**：转让价格低于市场价**70%**的，视为“明显不合理的低价”（合同编通则司法解释第42条）。

**亲属关系/关联关系例外**：债务人与相对人存在**亲属关系**或**关联关系**的，**不受70%限制**（合同编通则司法解释第42条第2款）。

#### （三）诉讼当事人

- **原告**：**债权人**
- **被告**：**债务人和相对人为共同被告**

#### （四）法律效果

- 一经撤销，**自始无效**，财产**返还给债务人**（**入库规则**）
- 必要费用（含合理的**律师代理费**）由**债务人**负担（第540条）
- 撤销权受**除斥期间**限制：**主观1年，客观5年**（第541条）


### 四、代位权 vs 撤销权对比

| 对比项 | 代位权诉讼 | 撤销权诉讼 |
| :--- | :--- | :--- |
| **行为性质** | **消极诈害债权**（怠于行权） | **积极诈害债权**（转移财产） |
| **主观恶性** | 较轻 | 较重 |
| **债权是否到期** | 原则要求到期 | **不要求到期** |
| **被告** | 相对人 | 债务人和相对人（**共同被告**） |
| **法律效果** | **直接受偿** | **入库规则** |
| **必要费用** | 由债务人负担 | 由债务人负担（含**合理的律师代理费**） |
| **法典依据** | 第535-537条 | 第538-542条 |

### 第一段：合同的保全概述

好，接下来咱们来看专题十六，叫做合同的保全，又叫债的保全。这个专题是每年绝对必考事项，客观题和主观题都会考察。在旁边拿笔写上**主观题**。

这个专题为什么重要？它的本质就是**担保**。担保分成典型担保和非典型担保，典型担保分为事前和事后，而本专题就是**事后债的保全**。它包括两个权利：一个叫**债权代位权诉讼**，一个叫**债权撤销权诉讼**。

它要解决的问题就是：债务人逃避债务，债权人如何保护自己的合法权益。它重要的原因在于：第一，它在民法实体法意义上**突破了合同相对性原理**；第二，它在行权方式上**只能通过诉讼的方式行使**，所以叫债权代位权诉讼、债权撤销权诉讼。它天然就和民事诉讼法结合在一起，所以属于客观题和主观题都会考察的内容。


### 第二段：债权代位权诉讼——基本模型

先来看第一个，**债权代位权诉讼**。它解决的就是**消极诈害债权**的问题，背后的法理与逻辑就是**三角债关系**。

模型关系：A公司欠B公司100万，B公司欠C公司50万。
- **本质**：基于合同相对性原理，A和B有一个合同关系（A欠B钱，B可告A），B和C有一个合同关系（B欠C钱，C可告B）。但在**极度例外**的情况下，法律允许C直接去告A，虽然A和C没合同——这就是**债权代位权诉讼**。
- **标签**：C = **债权人**，B = **债务人**，A = **相对人**（次债务人）

【补充】讲稿原文“代于行权”应为“**怠于行权**”，下同修正。

**含义**（民法典第535条）：当债务人（B）**怠于行使**其对相对人（A）的权利，害及债权人（C）的债权时，债权人（C）为保全其债权，可以**以自己的名义**代为行使债务人对相对人的权利提起诉讼。


### 第三段：债权代位权诉讼——构成要件

**四个要件，须同时具备**：

**要件一：两个债权均合法且到期**（原则）

【补充】这里做**扩大解释**，债权既可能是约定之债，也可能是法定之债，也可能是自然之债。自然之债也可以去主张，只是看债务人提不提诉讼时效抗辩权。

**要件二：债务人怠于行使权利**（核心考点）

如何理解“怠于行使”？法律要求债务人的行权方式必须是**法定的**——必须采取**诉讼或仲裁**的方式主张权利。否则，即便债务人天天打电话、发催收通知书、跑到对方家里去要债，都构成**怠于行权**。

**为什么？** 因为如果允许打电话、发短信等方式就认定为不怠于行权，A和B可能恶意串通演戏（B每天给A发短信催收，A配合保留证据，B免A十万债务）。但诉讼或仲裁的进程由法院或仲裁机构主导，A和B控制不了，演戏成本太高。

**结论**：**或诉或裁**。只要债务人未采用诉讼或仲裁方式主张权利，均构成怠于行权。

**要件三：因果关系**（近年来重点）

债务人怠于行使权利的行为**已经危害到债权人的债权**。如果债务人有足够财产清偿债务，债权人就不能突破合同相对性去告相对人。

> **自然例**：老孟欠你1万，马峰欠老孟5万。但老孟有5套别墅、2000万存款，你1万债权轻松能还，你就不能去告马峰。

**要件四：次债权非人身专属性**

人身专属性 = **救命钱**（生存权），最典型的就是**三费**（抚养费、赡养费、扶养费），这类债权**不得代位**。【补充】三费还不受诉讼时效期间限制。


### 第四段：债权代位权诉讼——诉讼当事人与管辖

**原告**：**债权人（C）** ，以**自己的名义**起诉。
**被告**：**相对人（A）** 。
**第三人**：**债务人（B）** ，应当追加为**无独立请求权的第三人**（无独三）——必须把他弄来，因为两个债的情况只有B最清楚。

**诉讼请求金额**：以**自己的到期债权为限**（C只能告50万，不能告100万）。

**管辖法院**：被告（A）所在地人民法院（原告就被告）。


### 第五段：债权代位权诉讼——相对人的抗辩

**实体抗辩**（可援用）：相对人（A）**可援用对债务人（B）的抗辩对抗债权人（C）**。最重要的是**诉讼时效抗辩权**——代位权诉讼不得使相对人的处境恶化。原来B找A要债时A可以提诉讼时效抗辩，现在换了个C，A照样可以提。

**程序抗辩**（不予支持）：最重要的是**仲裁条款的抗辩**和**协议管辖条款的抗辩**。为什么？因为仲裁条款和协议管辖条款容易被A和B伪造（基于合同相对性 + 可伪造性），容易出现A和B恶意串通损害C利益的情况，所以不准提这样的程序抗辩。


### 第六段：债权代位权诉讼——法律效果

**直接受偿规则**（民法典第535条）：
- 法院认定代位权成立的，由相对人（A）**直接向债权人（C）履行债务**。
- 这就是“**早起的鸟儿有虫吃**”——谁积极提起代位权诉讼，谁就能拿到钱。即使B还有其他债权人D、E，只要他们没提代位权诉讼，就保护C。

**费用承担**：债权人行使代位权的**必要费用**（差旅费、保全费等）由**债务人（B）** 负担——因为B怠于行权有过错。

**诉讼时效中断**：代位权诉讼的提起会导致**两个债（A—B和B—C）的诉讼时效均中断而重新起算**。

> **例题（2024法考类似题）** ：甲公司为乙医院提供医疗设备，乙医院欠甲公司设备费用300万元，满一年。丙公司欠乙医院医疗服务费500万元，逾期支付满两年，医院一直未催促丙公司支付服务费。甲公司欲向丙公司行使代位权。
>
> A. 甲公司代位权诉讼的提起，会导致乙医院对丙公司的债权诉讼时效中止——**错**（是**中断**，不是中止）
>
> B. 甲公司代位权诉讼的提起，会导致甲公司对乙医院的债权诉讼时效中止——**错**（是**中断**）
>
> C. 甲公司的代位权以多少钱为限——**300万元**（以自己的到期债权为限）
>
> D. 甲公司的代位权以500万元为限——**错**
>
> **答案：C**


### 第七段：债权撤销权诉讼——基本概念

债权撤销权诉讼解决的是**积极诈害债权**（转移财产）的行为。它的本质就是**不应减少而减少**——债务人为了逃避债务，把自己的财产转移出去。

**两种诈害债权行为**（民法典第538条、第539条）：

| 类型 | 典型行为 | 要件数量 |
| :--- | :--- | :--- |
| **无偿诈害债权行为** | 赠与（最常见） | **3个要件** |
| **有偿诈害债权行为** | 明显不合理的低价转让财产 | **4个要件**（多了“相对人知情”） |

**重要区别**：撤销权诉讼**不要求债权到期**！如果等债权到期再行使，债务人早就把财产转移完了。


### 第八段：无偿诈害债权行为——三个要件

**要件一：合法有效的债权**（**不要求到期**）

**要件二：诈害债权的行为**（重点：**赠与**——直接把自己的财产送出去）

**要件三：因果关系**（诈害行为损害了债权人的债权）

> **例题一（入门级）** ：甲公司欠乙公司货款500万元。甲公司向丙公司赠送一套价值50万元的机器设备，尚未履行。
>
> **问一**：债权人乙公司可否向法院提起诉讼撤销甲公司的赠与行为？
>
> **答**：**可以**。无偿转让财产，符合三个要件（合法债权、诈害行为、因果关系），可以撤销。
>
> **问二**：赠与人甲公司可否撤销对丙公司的赠与？
>
> **答**：**可以**。赠与合同尚未履行，甲公司享有**赠与人的任意撤销权**。
>
> **结论**：**钩钩**（两个都能撤，但权利性质不同——乙行使的是债权撤销权，甲行使的是赠与人的任意撤销权）

> **例题二（公益赠与）** ：甲公司欠乙公司货款500万元。甲公司向**戊希望学校**捐赠价值100万元的电脑，尚未履行。
>
> **问一**：债权人乙公司可否撤销甲公司的捐赠行为？
>
> **答**：**可以**。公益性质的赠与只限制**赠与人**，不能限制赠与人的**债权人**。否则希望小学会沦为逃债工具。
>
> **问二**：赠与人甲公司可否撤销对戊希望学校的赠与？
>
> **答**：**不能**。这是**公益性质的赠与**，赠与人**不享有任意撤销权**（民法典第658条）。
>
> **结论**：**钩叉**

> **例题三（时间因素——因果关系）** ：甲公司欠乙公司货款500万元（**2024年6月1日**欠的）。甲公司于**2024年3月1日**向丁基金会捐赠50万元现金（在债权成立**之前**），尚未履行。
>
> **问**：债权人乙公司可否撤销甲公司的捐赠行为？
>
> **答**：**不可以**。**债权成立在前，诈害行为在后**才有因果关系。行为发生在债权成立之前，不存在因果关系。
>
> **注意**：不要和《企业破产法》的破产撤销权混淆——那可以往前追溯，但这里是普通民事制度，**不能向前溯及**。
>
> **结论**：**叉叉**

**三题总结**：例题一钩钩，例题二钩叉，例题三叉叉——一个要素变化，答案完全不同。


### 第九段：无偿诈害债权行为——新增考点：假离婚真逃债

【补充】民法典婚姻家庭编司法解释二第3条（**从未考过**）

实践当中夫妻双方可能假离婚真逃债，离婚协议中出现**净身出户条款**。此时债权人可以撤销的只是**财产分割及债务履行条款**，而不是整个离婚协议。

**离婚协议三项内容**：
1. 自愿解除婚姻关系（人身关系条款）——不能撤销
2. 子女抚养（人身关系条款）——不能撤销
3. 财产分割及债务履行（财产关系条款）——**可能被撤销**

**参照适用**民法典第538条或第539条，综合考虑夫妻共同财产整体分割及履行情况、子女抚养费负担、离婚过错等因素。

> **例题（经典真题）** ：甲欠乙30万元到期未还。甲与丙结婚数日后即办理离婚，离婚协议书中约定将甲婚前的一处住房赠与**知悉**甲欠乙债务的丙，并办理了所有权变更登记。丙有稳定工作和汽车等财产可供还债。
>
> **问**：下列说法正确的是？
>
> A. 离婚协议书因恶意串通损害第三人利益而无效——**错**（**知悉 ≠ 恶意串通**，且题目表明丙有财产可供还债，没有损害债权）
>
> B. 如甲证明自己有稳定工资收入及汽车等财产可供还债，法院应驳回乙的诉讼请求——**正确**（不存在因果关系）
>
> C. 如乙仅以甲为被告，法院应当追加丙为被告——**正确**（债权撤销权诉讼是**共同被告**）
>
> D. 如法院认定乙的撤销权成立，应一并支持乙提出的由甲承担律师代理费的请求——**正确**（撤销权诉讼可主张合理的律师代理费）
>
> **答案：B、C、D**


### 第十段：有偿诈害债权行为——四个要件

**四个要件**：

1. **合法有效的债权**（**不要求到期**）
2. **诈害债权的行为**（重点：**明显不合理的低价转让财产**）
3. **相对人知情**（主观要件，≠恶意串通）
4. **因果关系**

**价格判断标准**（客观要件）：**上下浮动30%**。转让价格达不到交易时交易地的指导价或市场交易价**70%**的，一般可视为“明显不合理的低价”（民法典合同编通则司法解释第42条）。

**知情 ≠ 恶意串通**：知情只能说明相对人**非善意**，可以撤销；但不能因此认定合同无效。除非题目明确说“能够证明构成恶意串通”，才能认定无效（民法典第154条）。


### 第十一段：有偿诈害债权行为——新增考点：亲属关系/关联关系

【补充】民法典合同编通则司法解释第42条第2款（**从未考过**）

**规则**：债务人与相对人存在**亲属关系**或**关联关系**的，**不受70%和30%的限制**——因为亲属/关联方协助逃债的可能性极大，存在高度盖然性。

> **亲属关系范围**（民法典第1045条）：亲属包括**配偶、血亲和姻亲**。近亲属（8类）：配偶、父母、子女、兄弟姐妹、祖父母、外祖父母、孙子女、外孙子女。此外，舅舅、姑姑、姨妈、叔叔、伯伯等是**血亲**；公公婆婆、岳父岳母、小舅子、小姨子等是**姻亲**——这些人帮着逃债的可能性极大。

**关联关系**：主要指**母子公司**关系——母公司帮子公司逃债的可能性极大。

> **对比例题**：
>
> **左侧（无亲属关系）** ：孟某欠马某1万。孟某将价值1万的手表以**5000元**卖给白某。白某不知情（推定善意）。
>
> **能否撤销？** **不能**。价格低于70%（7000元）符合客观要件，但白某**不知情**，不满足主观要件。
>
> **右侧（有亲属关系）** ：孟某欠马某1万。孟某将价值1万的手表以**9000元**卖给**小舅子**白某。白某**知情**。
>
> **能否撤销？** **可以**。虽然9000元不低于70%（7000元），但因为白某是**亲属关系**，不受70%限制，且白某知情，可以撤销。


### 第十二段：有偿诈害债权行为——价款以实际支付为准

这里的“价款”指的是**实际支付的价款**，而不是约定的价款。如果约定的价款是合理的，但实际支付的钱少得很，同样可以撤销。

> **例题（经典真题）** ：杜某拖欠谢某100万。杜某称已把市价120万的房屋作价**90万**卖给了赖某，房屋钥匙已交但产权尚未过户。
>
> **分析**：120万×70% = 84万，90万 > 84万，**价格合理**。
>
> A. 谢某可请求撤销杜赖的买卖合同——**错**（价格合理，客观要件不满足）
>
> B. 因房屋尚未过户，杜赖的买卖合同无效——**错**（债务人卖自己的房，有权处分，合同合法有效）
>
> C. 如谢某能举证杜某赖某构成**恶意串通**，则杜赖的买卖合同无效——**正确**（恶意串通损害他人合法权益的合同无效，民法典第154条）
>
> D. 因房屋尚未过户，房屋仍属杜某所有，谢某有权**直接取得**房屋所有权以实现其债权——**错**（直接取得构成不当得利，房屋值120万，债权只有100万）
>
> **答案：A、B、D**


### 第十三段：债权撤销权诉讼——程序问题

**原告**：债权人（B）
**被告**：**债务人和相对人为共同被告**（A和C）
**管辖**：被告所在地人民法院

**为什么是共同被告？** 诈害债权的行为通常是合同行为（赠与合同、买卖合同），是双方民事法律行为，本质上是A和C共同损害B的利益（理论上可理解为共同侵权）。而代位权诉讼中，B只是消极怠于行权，所以B是无独三。

**撤销范围**：以**自己的债权为限**。如果诈害行为的标的可分（如现金），撤销相应的部分即可；如果标的不可分（如一套别墅），全部撤销（民法典合同编通则司法解释第45条）。


### 第十四段：债权撤销权诉讼——法律效果

**入库规则**：一经撤销，自始无效，**返还给债务人**（A），而非直接返还给债权人（B）。

**与代位权诉讼的对比**：

| 对比项 | 代位权诉讼 | 撤销权诉讼 |
| :--- | :--- | :--- |
| **行为性质** | 消极诈害债权（怠于行权） | 积极诈害债权（转移财产） |
| **法律效果** | **直接受偿**（早起的鸟儿有虫吃） | **入库规则**（返还债务人） |
| **当事人结构** | 原告C、被告A、第三人B | 原告B、被告A和C（共同被告） |
| **必要费用** | 由债务人负担 | 由债务人负担（含合理的律师代理费） |

撤销权诉讼实践当中用得相对较少，因为它**没有激励机制**——债权人耗时耗力提诉讼，胜诉后财产返还给债务人，其他债权人D、E也能来分钱（搭便车）。

**合理的律师代理费**：整个民商法当中，法律明文规定支持合理的律师代理费的制度，**就是债权撤销权诉讼**。

**除斥期间**：**主观1年，客观5年**（民法典第541条）。考试一般不考时间计算，重点考实体要件分析。


### 第十五段：本专题总结

翻回二百一十九页，合同的保全是**每年绝对必考事项**。

| 对比项 | 债权代位权诉讼 | 债权撤销权诉讼 |
| :--- | :--- | :--- |
| **本质** | **应增加而不增加**（债务人怠于行权） | **不应减少而减少**（债务人转移财产） |
| **主观恶性** | 较轻（懒） | 较重（积极逃债，不诚信） |
| **债权是否到期** | 原则上要求到期 | **不要求到期** |
| **构成要件** | 4个 | 无偿3个 / 有偿4个 |
| **行使范围** | 以债权人的债权为限 | 以债权人的债权为限 |
| **必要费用** | 由债务人负担 | 由债务人负担（含合理的律师代理费） |
| **法律效果** | **直接清偿**（谁起诉谁受益） | **入库规则**（返还债务人） |

> **老师原话总结**：
> - “代位权诉讼四个要件：**合法到期、怠于行权、因果关系、非人身专属性**。重点是**怠于行权**和**因果关系**。”
> - “撤销权诉讼——**不要求到期**！如果等到期再行使，财产早转移了。”
> - “有偿诈害债权行为——**知情 ≠ 恶意串通**。知情只能撤销，不能认定无效。除非题目明确说构成恶意串通。”
> - “**亲属关系/关联关系**不受70%限制——新增考点，从未考过。”
> - “撤销权诉讼——**共同被告**，**入库规则**，**律师代理费**。”
> - “**知悉不等于恶意串通！** 现代社会资讯发达，知悉的事情太多了，不能一知悉就撤。”
