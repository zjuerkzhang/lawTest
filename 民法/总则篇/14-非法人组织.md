
## 第1部分：对照《民法典》非法人组织提取备考提纲


### 一、非法人组织的定义（民法典第102条）

> **第102条**：非法人组织是**不具有法人资格**，但是能够**依法以自己的名义从事民事活动**的组织。

非法人组织包括：**个人独资企业、合伙企业、不具有法人资格的专业服务机构**等。


### 二、非法人组织 vs 法人的核心区别

| 对比项 | 法人（民法典第57条） | 非法人组织（民法典第102条） |
| :--- | :--- | :--- |
| **独立性** | **绝对独立**（三独俱全） | **相对独立**（有程序无实体） |
| **财产独立** | ✅ 是 | ❌ 否（财产不独立） |
| **人格独立** | ✅ 是（以自己的名义） | ✅ 是（以自己的名义） |
| **责任独立** | ✅ 是（自己行为自己责任） | ❌ 否（责任不独立） |
| **投资人责任** | **有限责任**（以出资额为限） | **补充的无限责任**（先组织后投资人） |
| **典型类型** | 有限责任公司、股份有限公司 | 个人独资企业、合伙企业 |


### 三、非法人组织的责任承担（民法典第104条）

> **第104条**：非法人组织的财产不足以清偿债务的，其**出资人或者设立人承担无限责任**。法律另有规定的，依照其规定。

**核心规则**：

| 规则 | 含义 |
| :--- | :--- |
| **补充责任** | **有顺序**——先组织后投资人 |
| **无限责任** | 投资人以**全部财产**承担责任 |
| **特别法优先** | 《个人独资企业法》《合伙企业法》有特别规定的，按特别规定来 |


### 四、非法人组织的程序地位（民法典第105条）

> **第105条**：非法人组织可以确定一人或者数人代表该组织从事民事活动。

非法人组织可以**以自己的名义**从事民事活动，在民事诉讼中具有**适格当事人**地位（可以做原告或被告）。


### 五、非法人组织的类型（民法典第102条第2款）

| 类型 | 说明 |
| :--- | :--- |
| **个人独资企业** | 一个自然人投资设立 |
| **合伙企业** | 普通合伙企业、有限合伙企业 |
| **不具备法人资格的专业服务机构** | 如律师事务所（特殊的普通合伙企业） |

```
非法人组织（民法典第102-108条）
│
├── 一、定义（第102条）
│   └── 不具有法人资格 + 能够以自己的名义从事民事活动
│

│

│

```

### 第一段：非法人组织概述

好，接下来咱们来看第三节叫非法人组织。那么这一节呢，在咱们法考范围内考得就比较少了。我们学习它的目的，就是为未来大家商法的学习做一个准备。在咱们民法这个部分，重点给大家介绍一下非法人组织和前面第二节法人组织到底有什么样的区别就可以了。

**非法人组织的含义**（民法典第102条）：非法人组织是指**不具有法人资格**，但是能够**依法以自己的名义从事民事活动**的组织。


### 第二段：法人组织 vs 非法人组织——独立性的区别

大家请看含义，非法人组织是指不具有法人资格，但是能够依法以自己的名义从事民事活动的组织。那这一点你还记不记得我们前面学过的法人的概念？书的三十三页，民法典第五十七条说法人是指具有民事权利能力和民事行为能力，依法独立享有民事权利和承担民事义务的组织。

他们俩的相同点都是**组织**。法人组织简称法人，第三节叫非法人组织。一个叫法人，一个叫非法人——就是看有没有**独立性**。

法人具有**绝对的独立性**——**三独**：财产独立、人格独立、责任独立。

非法人组织的核心点是**有程序而无实体**，仅具有**相对独立性**。那这个相对独立是啥意思呢？就是**有程序地位，无实体责任**。

**有程序地位**包括两点：
1. 能不能以自己的名义对外签合同？**可以**（人格独立）
2. 在程序上能不能单独告它？有没有民事诉讼中的适格原被告地位？**也可以**——比如个人独资企业可以当原告、也可以当被告

**无实体**的意思是**责任不独立**。责任不独立啥意思？就是**投资人要承担责任**。


### 第三段：责任承担——先组织后投资人（补充责任）

【补充】非法人组织与法人最核心的区别在于责任承担方式不同。

> **个人独资企业例题**：老王拿十万块钱开一个个人独资企业（满意水果店），卖水果给消费者，消费者吃了拉肚子，产生医药费一百万。个人独资企业账上只有十万块，损害却有一百万。
>
> **怎么承担责任？有顺序——先组织后投资人。**
>
> 第一步：先把个人独资企业账上的十万块钱拿出去承担责任，一百万减十万，还剩九十万。
>
> 第二步：剩下的九十万追到老王家里去了，老王得把自己家里的钱再拿九十万出来继续赔。
>
> **结论：投资人的责任是补充的、无限的——先组织后投资人。**

> **对比：开公司（法人组织）**
>
> 同样拿十万块钱开公司卖水果，同样产生一百万损害赔偿。公司账上十万块执行走，剩下九十万咋办？**就没办法了**。公司破产，跟老王家里没关系。王老板的责任是**有限责任**——仅以其出资为限。


### 第四段：非法人组织 vs 法人的利弊分析

【补充】开公司（法人）和开个人独资企业（非法人组织）各有利弊。

| 对比项 | 法人（公司） | 非法人组织（个人独资企业） |
| :--- | :--- | :--- |
| **投资人责任** | **有限责任**（以出资额为限） | **补充的无限责任**（先组织后投资人） |
| **税收** | 较高（企业所得税约25%） | 较低（约5%） |
| **监管** | 严格，需做商事账簿 | 相对宽松 |
| **举证责任** | 较重（一人公司需证明财产独立） | 较轻 |

所以大家开公司是为了有限责任的保护，但税收比较高、监管比较严格。非法人组织虽然投资人要承担补充的无限责任，但税收比较轻。

【注释】法人人格否认制度（公司人格否认）：如果股东不能证明公司财产独立于自己的财产，就要对公司债务承担**连带责任**。这是公司法的内容，由商法老师讲授。


### 第五段：非法人组织的法定类型（民法典第102条）

非法人组织包括：
1. **个人独资企业**
2. **合伙企业**
3. **不具备法人资格的专业服务机构**等


### 第六段：核心规则——非法人组织责任承担（民法典第104条）

**民法典第104条**：非法人组织的财产不足以清偿债务的，其**出资人或者设立人承担无限责任**。法律另有规定的，依照其规定。

**核心要点**：
- **补充责任**——有顺序，先组织后投资人
- **无限责任**——投资人以**全部财产**承担责任
- 特别法优于一般法——如果《个人独资企业法》《合伙企业法》有特别规定，按特别规定来


### 第七段：例题——普通合伙企业

> **例题**：甲企业是由自然人安某与乙企业（个人独资企业）各出资50%设立的**普通合伙企业**。欠丙企业货款50万元，由于经营不善，甲企业全部资产仅剩20万元。现所欠货款到期，相关各方因货款清偿发生纠纷。
>
> **分析**：
> - 第一步：先找组织——甲企业账上20万，50万 - 20万 = 30万
> - 第二步：后找投资人——剩下的30万由安某和乙企业（两个投资人）承担**连带责任**
>
> **问**：下列哪一表述是正确的？
>
> A. 丙企业只能要求安某与乙企业各自承担15万元——**错**（二人承担**连带责任**，不是各自承担）
>
> B. 丙企业只能要求甲企业承担清偿责任——**错**（非法人组织责任不独立，财产不够时要追到投资人）
>
> C. 欠款应先以甲企业的财产偿还，不足部分由安某与乙企业承担**无限连带责任**——**正确**
>
> D. 乙企业对丙企业的应偿债务，乙企业投资人不承担责任——**错**（非法人组织的投资人要承担责任）


### 第八段：民事法律关系——三大主体总结

【补充】讲稿在此部分对整个民事法律关系（自然人、法人、非法人组织）做了全面回顾，是对前面多个专题的总结。

民事法律关系客观题考三分，重点考**三大主体**：**自然人、法人组织、非法人组织**。

#### 自然人（五个问题）

1. **民事权利能力**：始于出生，终于死亡
   - 生前：**胎儿利益保护**——一个原则三个例外（遗产继承、接受赠与、侵权行为）——纯获法律上利益，只有权利没有义务
   - 死后：**死者人格利益保护**——七项人格利益、八类近亲属、两个顺位

2. **民事行为能力**：三类——无民事行为能力（5岁）、限制民事行为能力（15岁）、完全民事行为能力（25岁）
   - 无民事行为能力人独立实施的民事法律行为**一律无效**
   - 限制民事行为能力人：看是否与其意思能力相适应（生活相关程度 + 价款数额大小）

3. **监护**：
   - 法定监护（五个顺序）
   - 遗嘱指定监护（父母死亡前通过遗嘱指定，无顺序人数限制）
   - 协议监护（开家庭会议，无顺序人数限制）
   - 指定监护（居委会、村委会、民政部门、人民法院，法院具有终局效力）
   - 成年人意定监护（成年完全人通过书面委托合同自己给自己找监护人）
   - 临时委托监护（临时找人帮忙照顾被监护人）

4. **宣告失踪**：下落不明满2年 → 利害关系人申请 → 基层法院受理 → 公告期3个月 → 指定财产代管人

5. **宣告死亡**：四二零幺幺三——一般情形满4年，意外事件满2年；公告期一般1年，意外事件且证明无生存可能公告期3个月

#### 法人组织（五个问题）

1. 民事权利能力和民事行为能力
2. 法人的法定分类（营利法人、非营利法人、特别法人）
3. 法人的学理分类（公法人/私法人 → 社团法人/财团法人）
4. 基金会（捐助法人）
5. **法定代表人**（核心考点）——职务行为由法人承受，非职务行为个人承担；代表权的限制（外部限制+内部限制）；表见代表（民法典第504条）

#### 非法人组织

**有程序，无实体，相对独立**。
- 有程序地位：可以自己的名义签合同，可以做适格原被告
- 无实体责任：责任不独立，投资人承担**补充的无限责任**
- 核心规则：**先组织后投资人**（民法典第104条）
- 类型：个人独资企业、合伙企业、不具备法人资格的专业服务机构等
