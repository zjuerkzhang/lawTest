

## 第1部分：对照《民法典》诉讼时效提取备考提纲


### 一、诉讼时效概述

> **第188条**：向人民法院请求保护民事权利的诉讼时效期间为**三年**。法律另有规定的，依照其规定。诉讼时效期间自权利人**知道或者应当知道**权利受到损害以及义务人之日起计算。自权利受到损害之日起超过**二十年**的，人民法院不予保护，有特殊情况的，人民法院可以根据权利人的申请决定延长。

**诉讼时效的性质**：
- **强制性规定**（第197条）：诉讼时效期间、计算方法以及中止、中断的事由由法律规定，当事人约定无效。当事人对诉讼时效利益的预先放弃无效。

**诉讼时效的法理**：惩罚“睡眠在权利上的人”，保护债务人利益。


### 二、诉讼时效的适用范围

#### （一）债权请求权——一个原则五个例外

| 类型 | 是否适用诉讼时效 | 理由 |
| :--- | :--- | :--- |
| **原则：债权请求权** | ✅ **适用**（三年） | 一般要钱的权利，都受时效限制 |
| **例外一：支付存款本金及利息请求权** | ❌ 不适用 | 权利人未怠于行权（吃利息） |
| **例外二：兑付国债等债券本息请求权** | ❌ 不适用 | 权利人未怠于行权 |
| **例外三：基于投资关系产生的缴付出资请求权** | ❌ 不适用 | 保护公司及债权人利益（股东出资义务不能因时效免除） |
| **例外四：国有资产受到侵犯的请求权** | ❌ 不适用 | 防止国有资产流失（不考） |
| **例外五：抚养费、赡养费、扶养费请求权** | ❌ 不适用 | 保护弱势群体生存权 |

**特别提示**：
- **公共维修资金请求权** → ❌ 不适用（类推股东出资规则）
- **物业费请求权** → ✅ **适用**三年
- **劳动者工资报酬请求权** → ✅ **适用**三年

#### （二）物权请求权——一个原则两个例外

| 类型 | 是否适用诉讼时效 | 理由 |
| :--- | :--- | :--- |
| **原则：物权请求权** | ❌ 不适用 | — |
| **例外一：未登记的动产物权返还请求权（普通动产）** | ✅ **适用**（三年） | 无登记制度，举证困难 |
| **例外二：抵押权** | ✅ **适用**（三年） | 标的物不在债权人手里 |

**核心判断标准**：看标的物有无**登记制度**。有登记制度（不动产、特殊动产）→不适用时效；无登记制度（普通动产）→适用时效。

#### （三）人格权请求权（民法典第995条）

停止侵害、排除妨碍、消除危险、消除影响、恢复名誉、赔礼道歉 → ❌ **不适用**诉讼时效（侵权行为持续存在）


### 三、诉讼时效经过的法律效果——抗辩权发生说（民法典第192条、第193条）

> **第192条**：诉讼时效期间届满的，义务人可以提出**不履行义务的抗辩**。诉讼时效期间届满后，义务人同意履行的，不得以诉讼时效期间届满为由抗辩；义务人已经自愿履行的，不得请求返还。

> **第193条**：人民法院**不得主动适用**诉讼时效的规定。

| 要点 | 规则 |
| :--- | :--- |
| **抗辩权发生说** | 时效届满后，**实体权利、诉权都不消灭**，义务人取得**拒绝履行的抗辩权** |
| **法院不得主动适用** | 法官不能主动提示债务人提抗辩 |
| **起诉权不受影响** | 法院**应当受理**时效届满的案件 |
| **债务人提抗辩** | 法院判决驳回原告诉讼请求 |
| **债务人不提抗辩** | 法院判决债权人胜诉 |
| **二审提抗辩** | 原则上不允许，**除非有新证据** |
| **债务人自愿履行** | 不得反悔，**不构成不当得利** |


### 四、诉讼时效期间届满后的债务履行

- **同意履行（明示）** ：同意履行多少，就在多少范围内形成**单方允诺之债**，诉讼时效**重新起算**
- **自愿履行（默示）** ：视为放弃诉讼时效效力，**不得反悔**，债权受领给付的**不构成不当得利**


### 五、诉讼时效的种类（民法典第188条）

| 种类 | 期间 | 起算点 |
| :--- | :--- | :--- |
| **普通诉讼时效** | **三年** | 知道或应当知道权利受损及义务人之日 |
| **特殊诉讼时效** | 法律另有规定 | 依特别法 |
| **最长诉讼时效** | **二十年** | **自权利受到损害之日起**（客观标准） |


### 六、诉讼时效的起算

#### （一）合同之债（三种情况）

| 情形 | 起算点 |
| :--- | :--- |
| **未约定还款期限** | 宽限期届满之日 或 债务人明确拒绝之日 |
| **约定一个确定履行期限** | 履行期限**届满之日** |
| **分期履行** | 从**最后一期**履行期限届满之日起 |

#### （二）侵权之债

- **第三人侵权**：法定代理人**知道或应当知道**之日起
- **监护人侵权——性侵害**（第191条）：从**年满十八周岁**之日起
- **监护人侵权——性侵害以外的**：从**法定代理终止**之日起
- **例外补漏**（总则编司法解释第37条）：从**知道或应当知道**之日起


### 七、诉讼时效的中止（民法典第194条）

**中止 = 暂停**（最后六个月内因客观障碍不能行使请求权）

| 要件 | 内容 |
| :--- | :--- |
| **时间** | 诉讼时效期间的**最后六个月内** |
| **事由** | **客观障碍**（不可抗力等），权利人**欲行权而不能** |

**法律效果**：中止事由消除后，诉讼时效期间**再计算六个月**。


### 八、诉讼时效的中断（民法典第195条）

**中断 = 清零重来**

**中断事由**：
1. **权利人向义务人提出履行请求**
2. **义务人同意履行义务**
3. **权利人提起诉讼或者申请仲裁**
4. **与提起诉讼或者申请仲裁具有同等效力的其他情形**

**重点**：
- 权利人**提起诉讼**，从**提交起诉状或口头起诉之日**起中断（不要求法院受理）
- 可以**多次中断**
- 但最长不得超过**二十年**（总则编司法解释第38条第1款）

### 第一段：诉讼时效的概念——时间轴模型

好，接下来咱们来看专题五，叫做诉讼时效。诉讼时效就是一定时间的经过产生一定的法律效果。一个权利长时间不行使，到底会发生什么样的情况呢？

举个例子，我现在找你借一万块钱，咱俩约定一年还。一年期届满以后，你三年研究生都毕业了，你都没找我要。那么这个时候法律是保护你还是保护我呢？

看一个时间轴：2021年1月1号，我找你借钱一万块钱，咱俩约定一年还，就是2022年1月1号。这一年叫做**借款期间**。这一年内，你不能找我要债，我也不需要还钱——未到期的债权视为无债权。

从2022年1月2号以后，到期了，你就可以找我要债了。然后往后延三年，也就是**普通的诉讼时效期间**（三年），到了2025年1月1号。这三年我们管它叫诉讼时效期间。

在这绿色的三年诉讼时效期间内，法律肯定要优先保护你权利人债权。这三年内你随时随地都可以找我要债，你只要找我要债，我就得还钱，不还钱你把我告法院去，法院肯定支持你。

但问题是你太懒了，你怠于行权这三年，啥也没干——既不给我打电话，也不发短信，也不发邮件，也不去找村长解决纠纷，也不去镇政府的调解委员会。结果麻烦了。

从2025年1月2号以后，诉讼时效期间届满以后，法律前面已经优先保护你债权了三年，你都怠于行权。这个时候，我欠的一万块钱，我们给它起了一个专业术语叫**自然之债**——债权本身依然存在，但是**丧失了人民法院的强制执行力**。

到了蓝色的部分，我们开始**保护债务人利益**。为什么要保护债务人？因为债权人怠于行权，法律要惩罚“睡眠在权利上的人”。

所以这个专题背后的法理与逻辑就是：因为权利人怠于行权，因此我们要去保护债务人的利益。


### 第二段：诉讼时效的体系与法定性

诉讼时效就是时间的经过，它本质上是一个**自然事件**问题。立法背后为什么要保护债务人？因为债权人怠于行权，惩罚睡眠在权利上的人。

在借款合同订立之时，债权人是处于优势地位的。既然你处于优势地位，你完全可能在订立借款合同的时候，排除法律对于债务人的保护措施（诉讼时效）。但法律不允许你排除——**诉讼时效一律法定**。

诉讼时效一共包括五个问题：基本原理、种类、起算、中止、中断，一律都是法律规定。

**诉讼时效的性质是强制性规定**（民法典第197条）：诉讼时效的期间、计算方法以及中止、中断的适用由法律规定，当事人约定无效。当事人对诉讼时效利益的预先放弃无效。违反法律强制性规定的约定无效。


### 第三段：诉讼时效的适用范围（一）——债权请求权（一个原则五个例外）

哪些权利的形式要适用诉讼时效期间的限制？答案是**请求权**。所谓的请求权，是指特定权利人请求特定义务人为或不为特定行为的权利——权利人和义务人均特定。

从考试角度来看，一共就这几种。第一个叫**债权请求权**，用我的话来说就是“要钱不要物”。

债权请求权要钱的，原则上都得抓紧时间要钱，三年诉讼时效期间，原则上会受限制，但**它有例外**。

**一个原则五个例外**：

**例外一：支付存款本金及利息请求权**（旁边写“银行”）。老孟把一万块钱存到银行里去，三年定期理财到期后不去取，不是因为怠于行使，是因为想吃利息。所以权利人有怠于行使吗？没有，所以**不适用**诉讼时效期间限制。

**例外二：兑付国债、金融债券以及向不特定对象发行的企业债券本息请求权**。跟上面一样，债权人没有怠于行使权利，**不适用**诉讼时效。

**例外三：基于投资关系产生的缴付出资请求权**（旁边写“股东”）。股东出资义务不能因诉讼时效而免除。为什么？因为股东缴付出资才能保护公司利益，保护公司外部的债权人利益。股东绝对不能对于出资问题提诉讼时效抗辩权。

**类推适用**：业主大会请求业主缴付**公共维修资金或维修基金**的权利，可以类推适用投资关系的规则，也**不适用**诉讼时效限制。

**注意区分**：**物业费请求权**属于普通的债权请求权，适用三年诉讼时效期间。

**例外四：国有资产受到侵犯的请求权**。为防止国有资产流失，原则上不适用诉讼时效期间限制。考试基本不考。

**例外五：三费——抚养费、赡养费、扶养费**（既是民法重点，也是民诉重点）。这是为了保护弱势群体的生存权。三费**不适用**诉讼时效期间限制。

**注意**：**劳动者的工资报酬请求权适用诉讼时效期间限制**（劳动者有劳动仲裁等完善救济途径，不能躺在权利上睡觉）。【补充：民法典第188条】


### 第四段：诉讼时效的适用范围（二）——物权请求权

**物权请求权**，用我的话来说就是“要物不要钱”。物权请求权是否适用诉讼时效期间限制？不可一概而论。**一个原则，两个例外**。

原则：物权请求权原则上**不受**诉讼时效期间限制。但有两个例外。

背后的法律与逻辑就两个字：**登记**。看标的物有没有登记制度。

**例外一：未登记的动产物权（普通动产）返还请求权**。普通动产（书、手表、水杯、包包、衣服）没有登记制度，存在举证困难，一旦受到侵犯要及时行权——**适用三年诉讼时效期间**。

用八个字概括：**“普通动产，返还原物”**。

**例外二：抵押权**（民法典第419条）。抵押权为什么受诉讼时效限制？因为**标的物不在债权人手里**（在抵押人手里），所以要催促债权人及时行权。

只要看到抵押权，**受三年诉讼时效期间限制**。


### 第五段：诉讼时效的适用范围（三）——人格权请求权与其他

**人格权请求权**（停止侵害、排除妨碍、消除危险、消除影响、恢复名誉、赔礼道歉）**不适用**诉讼时效期间限制（民法典第995条）。背后的法理逻辑是侵权行为一直在持续。

**离婚后的财产分割请求权**（再次分割夫妻共同财产）：什么时候发现什么时候分，**适用三年诉讼时效期间**，起算点是**自发现之日起**起算。


### 第六段：诉讼时效经过的法律效果——抗辩权发生说

三年诉讼时效期间内权利人怠于行权，啥也没干，那产生了什么法律效果？

> **模型**：2021年10月10号，孟某向马某借款2000万，约定借款期限一年。
>
> **第一问**：2021年10月10号到2022年10月10号借款期内，债权人可否请求还款？**不行**（未到期的债权视为无债权）。
>
> **第二问**：2022年10月11号到2025年10月10号诉讼时效期间内，债权人可否请求还款？**可以**。
>
> **第三问**：2025年10月16号以后（诉讼时效期间届满后），债权人马某向法院起诉孟某还款。这时的债权已经变成了**自然之债**——债权本身依然存在，但丧失了人民法院的强制执行力。

**什么叫做抗辩权发生说？**

1. **不影响起诉**——权利人去人民法院起诉，人民法院**应当受理**，不得裁定不予受理（因为债权本身依然存在）。

2. **法院不得主动适用**（民法典第193条）——法官在审理过程中，**不能主动**向债务人提示“这个债过了诉讼时效了，你要不要提抗辩权”。法院必须被动司法，不得主动释明，也不能主动使用。否则法院教债务人不还钱，有违诚信原则。

3. **债务人提了抗辩权**——法院判决驳回债权人的诉讼请求。**债务人不提**——法院判决债权人胜诉。债务人行使还是不行使，完全由他自主决定。

4. **二审期间**——原则上不可以提诉讼时效抗辩权，**除非有新证据**（如一审没找到的借条二审找到了，或证人二审出现了）。

**时效抗辩权提出的时间**：原则上应当在一审期间提出。


### 第七段：诉讼时效期间届满后的债务履行——同意履行与自愿履行

**事后放弃诉讼时效效力**：诉讼时效期间届满后，债务人可能放弃诉讼时效抗辩权。

- **明示放弃（同意履行）** ：债务人同意履行多少，就在多少范围内形成**单方允诺之债**，诉讼时效重新起算。

> **例题**：甲公司向乙公司催讨一笔已过诉讼时效期间的10万元货款。乙公司书面答复称：“该债务已过诉讼时效，本公司本无义务偿还，但鉴于双方长期合作关系，**可偿还3万块**。”
>
> **分析**：3万块变成**单方允诺之债**，诉讼时效重新计算三年；剩下的7万仍然过时效。

- **默示放弃（自愿履行）** ：债务人直接把钱还了。还了以后能不能以“不当得利”为由要求返还？**不能**——债权本身自始至终存在，债务人自愿履行视为放弃诉讼时效效力，不得反悔，债权受领给付的**不构成不当得利**。

> **核心**：不管在哪个时间段（借款期间、时效期间内、时效届满后），债务人主动还款，债权人都不会构成不当得利——债务本身始终存在。


### 第八段：诉讼时效的种类

诉讼时效一共分三种：**普通、特殊和最长**（民法典第188条）。

- **普通**：三年
- **特殊**：法律另有规定的，依照其规定（如国际货物买卖合同四年）
- **最长**：**二十年**（自权利受到损害之日起计算）

**最长的诉讼时效期间二十年**要解决的极端情况——偷换人生二十八年、拐卖儿童等。如果有特殊情况，法院可以根据权利人的申请**决定延长**。


### 第九段：诉讼时效的起算

正常的起算原则是**主观起算标准**：自**知道或应当知道**权利受到损害的事实以及**义务人**之日起起算（民法典第188条第2款）。

**要同时知道“事儿”和“人”** ，两个都知道才能起算。

#### （一）合同之债——三种情况

| 情形 | 起算点 |
| :--- | :--- |
| **未约定还款期限** | 债权人第一次主张权利时，给予宽限期届满之日起起算；或债务人明确拒绝之日起起算 |
| **约定一个确定履行期限** | 履行期限届满之日起起算 |
| **分期履行**（如房贷） | **从最后一期履行期限届满之日起起算**（银行常考点） |

#### （二）侵权之债

- **第三人侵权**：从法定代理人知道或应当知道之日起起算
- **监护人侵权——性侵害**（民法典第191条）：从**年满十八周岁之日起**起算（十八到二十一）
- **监护人侵权——性侵害以外的**：从**法定代理终止之日起**起算

**例外补漏**（民法典总则编司法解释第37条）：如果超出上述时间段，从相应的民事主体**知道或应当知道**之日起起算——不能让侵权人免责。


### 第十段：诉讼时效的中止

**中止 = 暂停**（按下了pause键）。

**两个条件**（民法典第194条）：
1. **时间**：诉讼时效期间的**最后六个月内**
2. **事由**：发生了**客观障碍**（不可抗力等），导致权利人**欲行权而不能**

**法律效果**：中止事由消除后，诉讼时效期间**再计算六个月**——不管前面还剩几个月，再给你加六个月（六六大顺）。

> **例题**：2021年10月10号借款，期限一年，诉讼时效期间为2022年10月11号至2025年10月10号。2025年5月10号（最后六个月内），债务人所在地发生地震，导致债权人无法行使请求权。地震于2025年10月10号重建完毕。
>
> **分析**：诉讼时效**中止**（最后六个月 + 客观障碍），中止事由消除后**再计算六个月**至2026年5月10号。

**核心**：如果地震发生在最后六个月**以前**，只要持续到最后六个月内，就中止；如果结束时间还没到最后六个月，就不影响。


### 第十一段：诉讼时效的中断

**中断 = 清零重来**（民法典第195条）。

在三年诉讼时效期间内，只要权利人**积极行权**或义务人**同意履行**，诉讼时效就**中断，重新起算三年**。

**权利人行权的方式**（任何方式都行）：
- 起诉、申请仲裁
- 向债务人主张权利（打电话、发微信、发邮件、上门要债）
- 找调解委员会调解
- 向公安机关报案
- 发布公告（债务人下落不明的，在国家级或省级媒体上发布公告）

**义务人同意履行的方式**：
- 同意分期还款、先还一部分、提供担保、请求延期还款、制定清偿计划等

**重点**：权利人**提起诉讼**，从**提交起诉状或口头起诉之日**起中断，不一定法院要受理。

**法律效果**：
- 诉讼时效中断后**重新起算三年**
- 可以**多次中断**
- 但最长不得超过**二十年**（民法典总则编司法解释第38条第1款）

**总结**：诉讼时效没那么容易经过，除非权利人真的是太懒了，完全无动于衷。


### 第十二段：总则编总结

翻回七十五页，诉讼时效的本质：一定时间的经过产生一定的法律效果——**权利减损**，因为债权人怠于行权，法律惩罚“睡眠在权利上的人”，给债务人赋予了一个诉讼时效抗辩权（挡箭牌）。这个挡箭牌用不用，那是债务人的事。

诉讼时效**一律法定**，防止债权人在订立合同时处于强势地位时剥夺法律对债务人的保护措施。一旦改变法律对债务人的保护措施，都会被认定为无效。

**总则编总结**：两个大的制度，六个小的制度。
- **民事法律关系**（3分）→ 三大主体：自然人、法人、非法人组织
- **民事法律事实**（5分）→ 三个制度：民事法律行为、代理、**诉讼时效**
