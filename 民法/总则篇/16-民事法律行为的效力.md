

## 第1部分：对照《民法典》民事法律行为效力提取备考提纲


### 一、民事法律行为的效力状态

| 效力状态 | 特点 | 典型情形 |
| :--- | :--- | :--- |
| **有效** | 发生当事人意欲的法律效果 | 符合民法典第143条 |
| **无效** | 不发生当事人意欲的法律效果（发生法律意欲的法律效果） | 民法典第144、146、153、154条 |
| **效力待定** | 悬而未决，待权利人追认或拒绝 | 限制民事行为能力人超越能力范围订立的合同（第145条） |
| **可撤销** | 未被撤销前有效，撤销后自始无效 | 重大误解、欺诈、胁迫、显失公平（第147-151条） |


### 二、有效的民事法律行为（民法典第143条）

**四个要件（强公主义）** ：
1. **行为人具有相应的民事行为能力**（主）
2. **意思表示真实**（义）——最重要
3. **不违反法律、行政法规的强制性规定**（强）
4. **不违背公序良俗**（公）


### 三、无效的民事法律行为

#### （一）五种情形（民法典第144、146、153、154条）

| 情形 | 法典依据 | 要点 |
| :--- | :--- | :--- |
| 违反法律、行政法规的**强制性规定** | 第153条第1款 | **违法不一定无效**（婚姻家庭编解释二第15条） |
| 违背**公序良俗** | 第153条第2款 | 善良风俗=道德问题 |
| 无民事行为能力人独立实施 | 第144条 | 考的很少 |
| 双方通谋虚假意思表示 | 第146条 | **主观题重点**，明实不符 |
| 恶意串通损害他人合法权益 | 第154条 | 考的很少，**知情≠恶意串通** |

#### （二）违背公序良俗的典型情形

1. 强迫结婚、限制生育
2. 买卖人体器官（第1007条）
3. 随意起名（北雁云依案，第1015条）
4. 代孕合同
5. **第三者赠与**（婚姻家庭编解释二第7条，**重点**）
6. 危房出租

#### （三）双方通谋虚假意思表示（民法典第146条）

- 阴阳合同：虚假意思表示的部分**无效**，隐藏行为的效力**具体问题具体分析**
- **法律上不承认假离婚**（婚姻家庭编解释二第2条，新增）

#### （四）部分无效（民法典第156条）

民事法律行为部分无效，不影响其他部分的效力。争议解决条款具有**独立性**（第507条）。

#### （五）无效的法律后果（民法典第157条）

1. **恢复原状**
2. **缔约过失责任**


### 四、效力待定的民事法律行为（民法典第145条）

**主要考**：限制民事行为能力人**独立实施**的、与其**意思能力不相适应**的**双方民事法律行为（合同行为）**。

**各方权利**：
- **法定代理人**：追认权（形成权）、拒绝权（形成权）
- **法定代理人收到催告后30日内不表态** → **视为拒绝追认**
- **善意相对人**：撤销权（形成权）——须在法定代理人追认前行使


### 五、可撤销的民事法律行为（民法典第147-151条）

#### （一）四种情形

| 类型 | 本质 | 关键判断 |
| :--- | :--- | :--- |
| **重大误解**（第147条） | 内因（自身原因），主给付义务认识错误 | 从给付义务认识错误不构成重大误解 |
| **欺诈**（第148条） | 外因（对方原因），“知假卖假” | 第三人欺诈一分为二（相对人知情与否） |
| **胁迫**（第150条） | 不法性，恐惧/害怕心理 | 第三人胁迫不问相对人是否知情 |
| **显失公平**（第151条） | **利用**危困状态/缺乏判断能力 | **看主观不看客观** |

#### （二）重大误解的要点

- 重大误解 = **重大** + **误解**（民法典总则编司法解释第19条）
- **从给付义务**（配套设施、单证资料）认识错误 → **不构成**重大误解
- 赌石、古玩等特殊标的物 → **不构成**重大误解
- **第三人转达错误** → 可参照适用重大误解（总则编司法解释第20条，**新增，从未考过**）

#### （三）欺诈的要点（民法典第148、149条）

- **知假卖假** = 欺诈；**不知假卖假** = 重大误解
- 第三人欺诈：相对人**知情** → 可撤销；相对人**不知情** → 合同**有效**

#### （四）胁迫的要点（民法典第150条）

- 具有**不法性**，谁实施的都可以撤销
- 第三人胁迫**不问**相对人是否知情

#### （五）显失公平的要点（民法典第151条）

- **看主观**（是否利用），**不看客观**（不看数字大小）
- 假一罚十是**单方允诺之债**，不是显失公平

#### （六）撤销权的行使（民法典第152条）

- **行权方式**：**或诉或裁**（不能通知）
- **除斥期间**：主观**1年** + 客观**5年**，先到为准
- **法律效果**：撤销 → 自始无效 → 恢复原状 + 缔约过失责任；不行使 → 有效 → 违约责任

### 第一段：民事法律行为效力概述

好，接下来咱们来看第二节叫民事法律行为的效力。这一节是绝对重点，每年绝对必考事项，并且主观题也会考查，所以在这一节旁边写上**主观题**。

民事法律行为的效力，其实就是对民事法律行为进行的**价值判断**。民法典总则编中的民事法律行为的效力，专指的其实就是**合同的效力**。在民事法律行为旁边可以写两个字“合同”，按照合同的效力来学习。

**合同的成立 vs 合同的效力**：完全不是一回事——一个是**事实判断问题**，一个是**价值判断问题**。

> **例题**：我把表卖给你，十二万要不要？你说不要。合同**没成立**，不需要去看效力。如果我卖毒品给你，合同**成立了**，但效力如何？**无效**。

法律要画个圈圈。民法讲究意思自治、自愿原则，但不是绝对的自由。自愿的意思是在法律规定的范围内可以自愿，是相对的自由。去买本民法典可以，卖毒品不可以。法律会进行否定性的价值评价。


### 第二段：民事法律行为的四种效力状态

民事法律行为的效力一共有四种：**有效、无效、待定、撤销**。

| 权利类型 | 效力状态 |
| :--- | :--- |
| **合同** | 四种均有 |
| **婚姻行为** | 三种（有效婚、无效婚、撤销婚，没有待定） |
| **遗嘱/遗赠（单方民事法律行为）** | 两种（有效和无效，非黑即白） |


### 第三段：有效的民事法律行为

**有效的民事法律行为**：能够发生当事人**意欲发生、主动追求或内心期待的法律效果**。说白了就是**求仁得仁**，法律认可你。

要构成有效，得符合**民法典第一百四十三条**最基本的有效要件——**强公主义**（谐音记忆）：
- **强** = 不违反法律、行政法规的**强制性规定**
- **公** = 不违背**公序良俗**
- **主** = 主体具有**相应民事行为能力**
- **义** = **意思表示真实**（最重要的是这个，旁边写上“自愿原则”）

> **注意**：意思表示不真实**不一定**就是无效的，也可能是**可撤销**的（比如欺诈、胁迫）。做题时看到“意思表示不真实因此无效”这种表述，肯定是错的。

> **例题**：孟某与甲教育培训机构签订课外辅导协议，约定交费5万，若高考分数达不到二本线全额退费。该协议不违法、不背俗，主体合格，意思真实——**合同合法有效**。若未达到目标，退费承担违约责任。


### 第四段：无效的民事法律行为——含义

**无效**的意思：**不发生**当事人意欲发生、主动追求或内心期待的法律效果（法律不让你这么干）。但**会发生法律意欲发生的法律后果**——**恢复原状、缔约过失责任**（民法典第157条）。

无效是效力瑕疵中最严重的一种，**慎之又慎，绝对例外**。百分之九十五的合同都不会是无效的。不要动不动就选无效。

**无效的五种情形（强公主义恶）：**

| 情形 | 法典依据 | 讲稿重点 |
| :--- | :--- | :--- |
| **1. 违反法律、行政法规的强制性规定** | 第153条第1款 | 买卖毒品、枪支弹药无效。但**违法不一定无效**！ |
| **2. 违背公序良俗** | 第153条第2款 | 重点考善良风俗——道德问题 |
| **3. 无民事行为能力人独立实施的民事法律行为** | 第144条 | 考的非常少 |
| **4. 双方通谋虚假意思表示** | 第146条 | **主观题必考**，明实不符 |
| **5. 恶意串通损害他人合法权益** | 第154条 | 考的非常少，慎选 |


### 第五段：违法不一定无效（民法典婚姻家庭编司法解释二第15条）

旁边写上“**违法不一定无效**”。据不完全统计，中国民商事法律有三千八百多件，老百姓动不动就会“犯法”，但法官绝不能认为违法就一律无效。

> **借名买房案**：夫妻用共同财产买房登记在未成年子女名下，后来把房卖了，房价涨了反悔，以“违反民法典第35条（监护人不得处分被监护人财产）”为由主张合同无效。
>
> **法院裁判**：**不予支持**。如果支持，就破坏了诚信原则和交易安全。这恰恰印证了“违法不一定无效”的理念。


### 第六段：违背公序良俗（民法典第153条第2款）

**公序良俗**包括**公序**（公共秩序、国家安全、社会公共利益）和**良俗**（善良风俗、社会公德、家庭伦理、人格尊严）。重点考**善良风俗**，本质是**道德问题**，用一般理性人标准判断“缺不缺德”。

**违背善良风俗的典型情形：**

1. **结婚自由**：强迫、包办买卖婚姻无效。例题：约定付保证金强迫结婚，否则不得返还——**无效**。
2. **生育自由**：离婚协议约定女方再婚后不得再生育——**无效**。
3. **买卖人体器官**：买卖人体细胞、组织、器官、遗体——**无效**（民法典第1007条）。
4. **姓名权行使**：随意起名违背文化传统——公安机关不予登记（民法典第1015条，北雁云依案）。
5. **代孕**：代孕合同在境内因将人物化——**无效**。
6. **第三者赠与**（民法典婚姻家庭编司法解释二第7条，**重点**）：夫妻一方为重婚、与他人同居等目的，将**夫妻共同财产**赠与他人（房车钱），另一方主张该赠与违背公序良俗无效的，法院**应予支持**。无效后果：恢复原状，**全部返还**。
7. **危房出租**：存在严重结构隐患的危房出租用于酒店经营，危及不特定公众人身财产安全——**违背公序良俗，租赁合同无效**。


### 第七段：双方通谋虚假意思表示（民法典第146条，主观题重点★★★）

**双方通谋虚假意思表示无效**——明修栈道暗度陈仓，明实不符，一交易两合同（主要是税收问题）。

> **房屋买卖阴阳合同**：甲和乙签订两份房屋买卖合同——第一份500万（真实，私下履行），第二份200万（给税务机关看，备案用）。
>
> **分析**：第二份合同的200万价款条款是**双方通谋虚假意思表示，无效**；第一份合同（隐藏行为）效力如何？**具体问题具体分析**——若无其他无效事由则有效，若是凶宅则可能构成欺诈可撤销。
>
> **答案**：第一份合同有效，第二份合同**部分无效**。
>
> **注意**：过户备案不影响合同效力——“违法不一定无效”又来了。

**民法典婚姻家庭编司法解释二第2条**（新增，从未考过）：**法律上不承认假离婚**。夫妻登记离婚后，一方以双方意思表示虚假为由请求确认离婚无效的，法院**不予支持**。


### 第八段：恶意串通（民法典第154条）

**恶意串通损害他人合法权益的民事法律行为无效**。

**三个要件**：互相勾结、谋取私利、损害他人合法权益。

**注意**：**知情 ≠ 恶意串通**。知情只能说明“非善意”，不等于“恶意串通”。不要一看到“知情”就选无效。

> **夫妻股权转让案**（民法典婚姻家庭编司法解释二第9条，新增，从未考过）：夫妻一方转让用夫妻共同财产出资但登记在自己名下的股权，另一方以未经其同意侵害夫妻共同财产利益为由请求确认股权转让合同无效的，法院**不予支持**。但有证据证明转让人与受让人**恶意串通**损害另一方合法权益的除外。
>
> **原则**：**看出资不看登记**。买受人只需查登记簿，不需要调查出卖人有没有老婆——商事制度效率优先。


### 第九段：部分无效（民法典第156条）

不要把整个合同认定为无效！争议解决条款具有**独立性**（民法典第507条），合同死了它还活着。

> **例题**：王某将孟某打成重伤（刑事犯罪，公法问题）。双方签协议：王某赔偿5万，孟某不得告发王某。前段（赔偿5万）**有效**，后段（不得告发）**无效**——部分无效。


### 第十段：无效的法律后果（民法典第157条，绝对重点）

**民事法律行为无效、被撤销、确定不发生效力后**：
1. **恢复原状**——行为人因该行为取得的财产应当予以返还
2. **缔约过失责任**——有过错的一方赔偿对方损失（各方都有过错的，各自承担相应责任）

**核心**：无效不是不发生任何法律后果——只是不发生当事人意欲发生的法律效果，要发生**法律意欲发生的法律后果**。


### 第十一段：效力待定的民事法律行为（主要是主体问题）

**效力待定**：民事法律行为虽已成立，但**是否生效尚不确定**，有待享有形成权的第三人作出**追认或拒绝**的意思表示来使效力得以确定——是一个**悬而未决**的状态。

**主要考**：**限制民事行为能力人独立实施的、与其意思能力不相适应的双方民事法律行为（合同行为）**（民法典第145条）。

> **例题**：孟某十七周岁（高三学生），着便装以10万元从君泰百货购买钻戒一枚。合同效力如何？
>
> **答**：**效力待定**——限制民事行为能力人、独立实施、不相适应、双方民事法律行为（合同行为）。

**各方权利**：
- **法定代理人（父母）** ：**追认权**（形成权）→ 追认则**自始有效**；**拒绝权**（形成权）→ 拒绝则**自始无效**
- **法定代理人收到催告后30日内不作表示** → **视为拒绝追认**（法律拟制）
- **善意相对人（君泰百货）** ：**撤销权**（形成权）→ 撤销则**自始无效**（相对人须为善意）

**催告权不属于形成权**（只是意思通知行为，不产生法律效果）。


### 第十二段：可撤销的民事法律行为（每年必考★★★★★）

**四种情形**：**重大误解、欺诈、胁迫、显失公平**。考试经常A、B、C、D各对应一个。

**共同特点**：只影响当事人自己的利益，充分尊重意思自治——可以撤也可以不撤。

#### （一）重大误解

**本质**：**自身原因**对**主给付义务**认识错误（内因）。

**重大误解 ≠ 一般误解**：必须对**主给付义务**认识错误。对**从给付义务**（配套设施、单证资料）认识错误，**不构成重大误解**。

**误解内容法定**（四种）：人、物、关系、信息传输工具。
- **对人的误解**：认错交易对象（双胞胎），可撤销（但**对行为能力的误解不行**——看到未成年人就是效力待定）
- **对标的物的误解**：房屋面积误差（3%以内不构成重大误解）；赌石、古玩等特殊标的物**不构成重大误解**（愿赌服输，买定离手）
- **对行为性质的误解**：误把有偿当无偿（如宾馆误以为饮料免费）
- **第三人转达错误**（民法典总则编司法解释第20条，**新增，从未考过**）：第三人转达错误可参照适用重大误解规则（如网购平台价格显示错误，商家可撤销）

> **例题（陈老伯案）** ：陈老伯听销售经理介绍“周边有轨道交通”就买了房，后发现是市域铁路不是地铁，出行成本较高。
>
> **问**：陈老伯可否基于重大误解撤销合同？
>
> **答**：**不可以**。轨道交通属于**从给付义务**（配套设施），不构成重大误解。也不构成欺诈（销售没说“这是地铁”），也不构成显失公平（没有利用危困状态）。

> **例题（2025年真题）** ：甲从沈某手里花5000元买了个奢侈品包，后发现是赝品。沈某**不知情**，很吃惊，出具了从王某处以6000元购得的交易记录。
>
> **问**：下列哪一说法正确？
>
> A. 依据重大误解撤销合同——**正确**（不知假卖假，双方均无过错）
>
> B. 依据欺诈撤销合同——**错**（欺诈须“知假卖假”）
>
> C. 可以申请解除合同——**错**（合同已履行完毕，不存在解除问题）
>
> D. 依据显失公平解除合同——**错**

#### （二）欺诈

**本质**：**非自身原因**陷入认识错误（外因）——“知假卖假”构成欺诈。

**两种行为**：
- **积极欺诈**：故意告知虚假情况（冒充、伪造）
- **消极欺诈**：负有告知义务的人故意隐瞒真实情况（隐瞒凶宅、事故车、泡水车）

**欺诈 vs 重大误解**：
| 对比项 | 重大误解 | 欺诈 |
| :--- | :--- | :--- |
| 原因 | **内因**（自身原因） | **外因**（对方原因） |
| 对象 | 须对**主给付义务**认识错 | 主给付/从给付均可 |
| 过错 | 双方均无过错 | 欺诈方有过错 |
| 撤销权主体 | **双方**均可 | 仅**受欺诈方** |

**第三人欺诈**（民法典第149条，**主观题重点**）：合同效力**一分为二**——相对人**知情**（非善意）→ 可撤销；相对人**不知情**（善意）→ 不可撤销，合同**有效**。

> **第三人欺诈例题**：甲（托儿）骗乙，乙与齐某签合同买石雕。齐某在场知甲骗乙但不做反对表示。乙能否撤销？
>
> **答**：**能**。第三人欺诈，相对人**知情**，非善意，可撤销。

#### （三）胁迫

**本质**：使对方陷入**恐惧或害怕**的心理状态（**具有不法性**）。谁实施的都可以撤销——**第三人胁迫与第三人欺诈不同，不需要看相对人是否知情**（民法典第150条）。

**判断标准**：以**特定受害人**而非一般人在当时情况下是否感到恐惧为标准——人跟人不一样（有人不怕蛇，有人怕）。

**不正当的预告危害**：手段不法、目的不法或二者均不法。正当行使权利不构成胁迫（“咱们法院见”不构成胁迫）。

#### （四）显失公平

**本质**：**利用**对方处于**危困状态**或**缺乏判断能力**，致使合同成立时显失公平——**看主观不看客观**！

**两种利用情形**：
1. **危困状态**：家人患病急需用钱、急需筹钱等
2. **缺乏判断能力**：大学生、老年人等特定人群交易经验不足

**注意**：不要只看数字大小来判断显失公平！

> **假一罚十**：卖假烟的老板说“假一罚十”，80元的烟要赔800元——不是显失公平，是**单方允诺之债**（优势地位一方给自己做出不利承诺）。

> **醉酒签合同**：
>
> - **别人灌醉** → 利用缺乏判断能力 → **显失公平**，可撤销
> - **自己喝醉**（司机说80，自己输成8080）→ **重大误解**，可撤销（自身原因，主给付义务认识错误）
> - **自己喝醉被利用**（司机明知你喝醉，让你输成8080）→ **显失公平**，可撤销（利用缺乏判断能力）

> **例题（醉酒打车案）** ：李某醉酒打车，车费80元，误将80元输成8080元，司机收到后未提醒。
>
> A. 运输合同因显失公平可撤销——**错**（运输合同本身合法有效）
>
> B. 多付的8000元视为李某对司机的赠与——**错**（赠与是双方民事法律行为）
>
> C. 醉酒不影响运输合同的效力——**正确**
>
> D. 李某有权请求司机返还8000元不当得利——**正确**
>
> **答案：C、D**


### 第十三段：撤销权的行使

**行权主体**：
- **重大误解**：**双方**均享有撤销权（双方均无过错）
- **欺诈**：仅**受欺诈方**
- **胁迫**：仅**受胁迫方**
- **显失公平**：仅**受损害方**

**行权方式**：**或诉或裁**（只能通过法院或仲裁机构行使，**不能通知**）——因为需要专业人员价值判断。

**除斥期间（双重）** ：有权不用，过期作废。
- **主观**：自知道或应当知道撤销事由之日起**1年**
- **客观**：自民事法律行为发生之日起**5年**
- **哪个先到按哪个**——只要有一个到期，撤销权即消灭

**法律效果**：
- **行使撤销权** → 合同**自始无效** → 恢复原状 + 缔约过失责任
- **不行使或过除斥期间** → 合同**有效** → 可追究违约责任

**可撤销 vs 效力待定**：可撤销在没有被撤销之前是**合法有效**的；效力待定在被追认前是**悬而未决**。
