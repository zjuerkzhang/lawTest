

## 第1部分：对照《民法典》民事法律行为基本原理提取备考提纲


### 一、民事法律事实（民法典第133条）

> **第133条**：民事法律行为是民事主体通过**意思表示**设立、变更、终止民事法律关系的行为。

**民事法律事实的分类**：

| 类型 | 与人的意志关系 | 典型例子 |
| :--- | :--- | :--- |
| **行为** | 与人的意思有关（有意识支配） | 签合同、打人、创作 |
| **事件** | 与人的意志无关 | 时间的经过（诉讼时效）、自然灾害 |


### 二、民事法律行为 vs 事实行为

| 对比项 | 民事法律行为 | 事实行为 |
| :--- | :--- | :--- |
| **法律效果由谁决定** | **当事人的意思表示** | **法律规定** |
| **是否需要行为人具有民事行为能力** | **需要**（与年龄有关） | **不需要**（与年龄无关） |
| **判断标准** | 不同年龄法律效果不同 | 谁干的都一样 |
| **典型例子** | 合同、婚姻、遗嘱 | 侵权、创作（著作权取得）、无因管理 |


### 三、民事法律行为的法定分类（民法典第134条）

> **第134条**：民事法律行为可以基于**单方**、**双方**或者**多方**的意思表示一致成立。法人、非法人组织依照法律或者章程规定的议事方式和表决程序作出决议的，该决议行为成立。

| 类型 | 成立要件 | 典型例子 |
| :--- | :--- | :--- |
| **单方民事法律行为** | **一个意思表示**（一个人说了算） | 动产所有权的抛弃、遗嘱、遗赠 |
| **双方民事法律行为** | **两个意思表示合意** | 合同（买卖、赠与） |
| **决议行为** | 多方按程序表决 | 股东会决议、董事会决议 |


### 四、民事法律行为的学理分类

#### （一）财产行为 vs 身份行为

- **财产行为**：遗嘱、遗赠、合同等（涉及财产关系）
- **身份行为**：结婚、离婚、收养等（涉及人身关系）

#### （二）负担行为 vs 处分行为

| 对比项 | 负担行为 | 处分行为 |
| :--- | :--- | :--- |
| **核心效力** | **债权效力**（发生债权债务关系） | **物权效力**（发生物权变动） |
| **典型例子** | 租赁合同 | 设立抵押权、质押权 |
| **法考应用** | 主观题可写“负担行为”得分 | 配合**物债两分区分原则**（民法典第215条） |


## 第2部分：近年考点与2026年预测

| 知识点 | 是否涉及主观题 | 近年考情 | 2026年可能考试方式 | 举例 |
| :--- | :--- | :--- | :--- | :--- |
| **民事法律行为 vs 事实行为的区分** | 客观题 | **每年必考** | 给出具体行为，问属于民事法律行为还是事实行为 | 十三岁中学生买可乐——民事法律行为（合同）；打人致伤——事实行为（侵权） |
| **事实行为的典型类型** | 客观题 | 常考 | 考查哪些行为属于事实行为 | 侵权行为、创作行为、无因管理——都属于事实行为（与年龄无关） |
| **单方民事法律行为的类型** | 客观题 | 常考 | 考查哪些属于单方民事法律行为 | 动产所有权的抛弃、遗嘱、遗赠——属于单方行为（一个人说了算） |
| **赠与合同是双方行为（易错点）** | 客观题 | **每年必考** | 考查赠与合同的性质 | 甲对乙说“这瓶水送你了”，乙说“不要”——赠与合同是否成立？（不成立，赠与是双方行为，需受赠人表示接受） |
| **抛弃 vs 赠与的区分** | 客观题 | **2019年考过** | 给出“扔掉瓶子被他人捡走”的场景，问是抛弃还是赠与 | 甲喝完水把瓶子放在球场走了，乙捡走——甲的行为是**抛弃**（单方行为），不是赠与 |
| **无主物 vs 遗失物** | 客观题 | 常考 | 考查抛弃后的法律后果 | 甲抛弃的可乐瓶成为**无主动产**，人人可得先占取得所有权 |
| **负担行为 vs 处分行为** | **✅ 主观题** | 偶有考查 | 考查某一行为是负担行为还是处分行为 | 房屋租赁合同——负担行为（不发生物权变动）；房屋抵押合同——处分行为（可能发生物权变动） |
| **特殊体质不免责** | 客观题 | 常考（结合侵权责任） | 给出“受害人特殊体质”的场景，问能否减轻责任 | 甲殴打乙，乙骨质疏松医药费高，甲能否以“只想打两百块的”为由减轻责任？（不能，侵权是事实行为，法律效果不考虑主观） |

> **老师原话总结**：
> - “民事法律行为和事实行为的区别——**就看跟年龄有没有关系**。跟年龄有关系→民事法律行为；跟年龄没关系→事实行为。”
> - “事实行为重点掌握三个：**侵权行为、创作行为、无因管理**。”
> - “单方民事法律行为重点掌握三个：**动产所有权的抛弃、遗嘱、遗赠**——一个人说了算。”
> - “赠与合同是**双方民事法律行为**——你送人家，人家得要。送一坨屎人家不要，合同不成立。”
> - “负担行为 = **债权效力**（租赁合同）；处分行为 = **物权效力**（抵押合同）。为物权编学习打基础。”
> - “特殊体质不免责——打人赔钱只看四个要件，不看你想赔多少。”

### 第一段：民事法律事实概述

好，接下来咱们来看专题三，叫民事法律行为。从专题三开始，咱们就正式的进入到了民法典总则编的第二个核心制度，叫**民事法律事实**。它在客观题考试当中占**五分**左右。它一共向下三个问题，咱们分成了三个专题，分别是**专题三民事法律行为**、**专题四代理**和**专题五诉讼时效**。

什么叫民事法律事实呢？民事法律事实是**能够引起民事法律关系产生、变更和消灭的客观现象**。

根据事实与人的意志有没有关系，分成了两类：一类叫**行为**（与人的意思有关，身体有意识支配的动和静），一类叫**事件**（如时间的经过）。

考试的重点必然是**行为**。但是，时间的经过也能产生法律关系的变化（如诉讼时效），等到专题五再讲。


### 第二段：民事法律行为 vs 事实行为（核心区别）

第一节是民事法律行为的基本原理。考试往往就考一个选择题——给一个未成年人签合同、侵权、打人，然后问咋办。

民事法律行为和事实行为都叫行为，它俩有啥区别呢？

**判断标准就一句话：看这个事儿跟年龄有没有关系。**

- **跟年龄有关系**（不同年龄法律效果不同）→ **民事法律行为**
- **跟年龄没关系**（谁干的都一样）→ **事实行为**

> **签合同**：五岁能不能签？不能。十五岁能不能签？不一定。二十五岁能不能签？可以。→ **跟年龄有关系** → 民事法律行为
>
> **结婚**：五岁不能，十五岁不能，二十五岁可以。→ 民事法律行为
>
> **订立遗嘱**：五岁不能，十五岁不能，二十五岁可以。→ 民事法律行为
>
> **侵权行为**：五岁打人要赔，十五岁打人要赔，二十五岁打人要赔。→ **跟年龄没关系** → 事实行为
>
> **创作行为**：五岁画画能取得著作权，十五岁也能，二十五岁也能。→ **跟年龄没关系** → 事实行为
>
> **无因管理（学雷锋）**：五岁可以做雷锋，十五岁也可以，二十五岁也可以。→ **跟年龄没关系** → 事实行为


### 第三段：事实行为——考试重点

**事实行为的含义**：行为人不具有设立、变更、终止民事法律关系的意图，但根据**法律规定**能引起既定的民事法律后果的行为。

说白了，它跟当事人的意思没关系，几岁的都一样，按照法律规定来产生法律后果。

> **例题（打人赔钱）** ：孟某失恋心情不爽想打人，因自己口袋里只有两百块钱，只想打马某两百块钱的。结果孟某把马某暴揍一顿，马某天生骨质疏松，医药费两千。
>
> **问**：是赔两百还是赔两千？
>
> **答**：**赔两千**。侵权行为是事实行为，法律效果是法律规定的，不考虑行为人主观上怎么想。只要符合一般侵权的四个要件（侵害行为、损害后果、因果关系、主观过错），就照赔。
>
> **注意**：**特殊体质不免责**——比如骨质疏松，不能减轻责任。

**民事法律行为 vs 事实行为对比**：

| 对比项 | 民事法律行为 | 事实行为 |
| :--- | :--- | :--- |
| **法律效果由谁决定** | 当事人的意思表示 | 法律规定 |
| **是否需要行为人具有民事行为能力** | **需要**（跟年龄有关） | **不需要**（跟年龄无关） |
| **典型例子** | 合同、婚姻、遗嘱 | 侵权、创作、无因管理 |


### 第四段：民事法律行为的法定分类

民事法律行为根据民法典的规定，分为三类：

#### （一）单方民事法律行为

**一个人说了算**——只要有一个意思表示就行了。

考试重点掌握**三个**：
1. **动产所有权的抛弃**——不要了，一扔就丧失所有权，变成无主动产
2. **遗嘱**——对象是**法定继承人**
3. **遗赠**——对象是**法定继承人以外的人**【补充：民法典第1133条】

#### （二）双方民事法律行为

**两个人意思表示合意**——我卖你买、我送你要。

最典型的是**合同**，包括买卖合同、赠与合同等。特别注意：**赠与合同是双方民事法律行为**——你送人家，人家得要。送一坨屎，人家不要，合同不成立。

> **易错点**：很多同学以为赠与是一个人说了算——**错**！赠与人送，受赠人表示接受，这才成立赠与合同。

#### （三）决议行为

了解即可，未来公司法老师会讲（股东会决议、董事会决议）。


### 第五段：单方行为 vs 双方行为——例题

> **例题（2019年法考真题）** ：中学生甲十三周岁，每天下午去篮球场打篮球，顺带买一瓶可乐饮用（第一个行为：买可乐，三块五，十三岁孩子能懂，合同合法有效）。打完球后，将装有半瓶可乐的瓶子放在球场上，拿着书包和同学一起离开了球场（第二个行为：不要了，**动产所有权的抛弃，单方民事法律行为**）。乙（拾荒者）随后捡走了可乐瓶。
>
> **问**：下列哪一说法正确？
>
> A. 甲的行为是赠与——**错**（赠与是双方民事法律行为，一方送一方要，这里没有“要”的意思表示）
>
> B. 甲的行为是抛弃——**正确**（单方民事法律行为）
>
> C. 甲的行为不需要意思表示——**错**（动产所有权的抛弃需要意思表示）
>
> D. 可乐瓶属于遗失物——**错**（可乐瓶属于**无主物**，不是遗失物）

**本题考察要点**：区分赠与（双方行为）和抛弃（单方行为）；区分无主物和遗失物。


### 第六段：民事法律行为的学理分类（了解）

#### （一）财产行为 vs 身份行为

根据效果不同：
- **财产行为**：订立遗嘱、遗赠等
- **身份行为**：结婚、离婚、收养等

#### （二）负担行为 vs 处分行为（重点理解，为物权编做准备）

| 对比项 | 负担行为 | 处分行为 |
| :--- | :--- | :--- |
| **核心效力** | **债权效力**（只发生债权债务关系） | **物权效力**（发生物权变动） |
| **典型例子** | **租赁合同**（租房子只发生债权债务，不发生物权变动） | **设立抵押权、质押权**（可能发生物权变动） |
| **法考意义** | 主观题写“负担行为”可得分 | 配合**物债两分区分原则**（民法典第215条） |

> **租赁合同**：我把房租给你三年，你给租金，但房屋所有权始终是我的——只发生债权债务关系，不发生物权变动 → **负担行为**
>
> **设立抵押权**：我把房屋抵押给银行，到期不还钱，银行拍卖房屋，房屋所有权可能从我这转移到买受人那里 → **处分行为**


### 第七段：本专题总结

1. **民事法律事实** → 引起民事法律关系变化
2. **行为 vs 事件** → 重点是行为
3. **民事法律行为 vs 事实行为** → 看是否与年龄有关（核心考点）
4. **法定分类** → 单方（抛弃、遗嘱、遗赠）、双方（合同，尤其是赠与合同）、决议
5. **学理分类** → 重点是**负担行为**（债权效力）和**处分行为**（物权效力），为物权编学习打基础
