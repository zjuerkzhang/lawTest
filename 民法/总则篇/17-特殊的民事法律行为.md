

## 第1部分：对照《民法典》附条件与附期限民事法律行为提取备考提纲


### 一、附条件的民事法律行为（民法典第158条、第159条）

#### （一）概念（第158条）

民事法律行为可以附条件，但是根据其性质不得附条件的除外。附生效条件的民事法律行为，自条件成就时生效。附解除条件的民事法律行为，自条件成就时失效。

#### （二）条件的特征

| 特征 | 说明 |
| :--- | :--- |
| **表人选定的** | 当事人自己约定，不是法律规定的 |
| **将来发生的** | 不能是已经发生的事实 |
| **发生与否不确定（或然性）** | 与期限的核心区别 |
| **合法事实** | 违法事实不能作为条件 |

#### （三）条件的分类

| 分类标准 | 类型 | 含义 |
| :--- | :--- | :--- |
| **对效力的影响** | 生效条件（停止条件/延缓条件） | 条件成就→法律行为**生效** |
| | 解除条件（失效条件） | 条件成就→法律行为**失效** |
| **事实的发生方式** | 积极条件 | 以某种事实的**发生**为条件 |
| | 消极条件 | 以某种事实的**不发生**为条件 |

#### （四）条件成就的拟制效力（第159条）

> **第159条**：当事人为自己的利益不正当地**阻止**条件成就的，视为条件**已成就**；不正当地**促进**条件成就的，视为条件**不成就**。

| 行为 | 法律效果 |
| :--- | :--- |
| 不正当**阻止**条件成就 | **视为条件已成就** |
| 不正当**促进**条件成就 | **视为条件不成就** |


### 二、附期限的民事法律行为（民法典第160条）

民事法律行为可以附期限，但是根据其性质不得附期限的除外。附生效期限的民事法律行为，自期限届至时生效。附终止期限的民事法律行为，自期限届满时失效。


### 三、附条件 vs 附期限的核心区别

| 对比项 | 附条件 | 附期限 |
| :--- | :--- | :--- |
| **所附事实** | 发生**不确定**（或然性） | 发生**确定**（必然性） |
| **典型词** | “如果……”、“如……” | “当……时”、“到……时” |
| **示例** | “如果通过法考，就买辆车” | “我死了，表就送你” |

### 第一段：特殊的民事法律行为概述

那么接下来我们来看第三节，第三节特殊的民事法律行为，说白了就是**附条件和附期限的民事法律行为**。

这个是啥意思呢？就是当事人基于自己的特殊需求，不希望这个合同马上就发生法律效力，希望等待一段时间，达到一定的条件或者时间，才会让这个合同发生约束力。

> **典型案例**：你爸妈特别希望你一次性通过法考，对你说：“今年如果你通过了法考，就给你买套房或买辆车。”双方白纸黑字签了个合同——这是一个**附生效条件的赠与合同**。合同已经成立了，但没有发生效力，因为成绩还没出来。


### 第二段：附条件的民事法律行为——定义与特征

**附条件的民事法律行为**：双方当事人在民事法律行为中特别规定一定的条件，以条件是否成就来决定民事法律行为生效还是失效（或终止）。

**四个特征**（民法典第158条）：
1. **表人选定的**——不是法律规定的，是当事人自己选的
2. **将来发生的**
3. **发生与否不确定，具有或然性**（**条件 vs 期限的最大区别**）【修正：原文“豁然性”“霍人性”】
4. 必须是**合法事实**——如果是违法事实（如保持不正当男女关系），合同直接无效

**重点掌握**：发生与否不确定 + 合法事实


### 第三段：条件的分类

#### （一）生效条件 vs 解除条件

| 类型 | 作用 | 典型例子 |
| :--- | :--- | :--- |
| **附生效条件**（附延缓条件/停止条件） | 影响合同**何时生效/履行** | “如果过了法考，就给你买辆车”——赠与合同，条件成就时生效 |
| **附解除条件**（附失效条件） | 影响合同**何时失效/终止** | “如果员工离职，则资助失效”——租赁合同/资助合同，条件成就时失效 |

> **附解除条件例题**：中森公司员工在公司年会抽奖中中奖，依活动规则，公司资助中奖员工子女次年的教育费用，**如员工离职则资助失效**。
>
> **分析**：“如员工离职”是发生与否不确定的事实→**附解除条件**（失效）。

#### （二）积极条件 vs 消极条件

考得很少，了解即可。
- **积极条件**：以某种事实的**发生**为条件（“如果考上法考”）
- **消极条件**：以某种事实的**不发生**为条件（“如果不离职”）


### 第四段：条件成就的拟制效力（民法典第159条，核心考点★★★）

**民法典第159条**：当事人为自己的利益不正当地**阻止**条件成就的，**视为条件已成就**；不正当地**促进**条件成就的，**视为条件不成就**。

> **例题（2020年真题）** ：廖某与石某约定：“**如**将来廖某不在甲公司上班，则廖某将其在甲公司附近的一套自有公寓卖给石某。”（附生效条件的房屋买卖合同，生效条件=廖某离职）
>
> 石某为尽快购得该房屋，**贿赂**甲公司经理，让其**辞退**了廖某。
>
> **分析**：石某为了自己的利益不正当的**促进**条件成就→**视为条件不成就**。
>
> **答案**：应视为条件不成就（欲速则不达）。

> **喜鹊案例（理解拟制效力）** ：
> - **买受人阻止条件成就**：约定“明天上午九点屋顶上有喜鹊就买房”，买受人八点半拿弹弓把喜鹊全打死了→不正当**阻止**条件成就→**视为条件已成就**→必须买房
> - **出卖人促进条件成就**：约定“明天上午九点屋顶上有喜鹊就卖房”，出卖人买了一只喜鹊用胶水粘在屋顶上→不正当**促进**条件成就→**视为条件不成就**→不用卖房


### 第五段：附期限的民事法律行为

**附期限的民事法律行为**：以将来确定发生的事实作为决定法律行为效力的依据（民法典第160条）。

**核心区别：条件 vs 期限**

| 对比项 | 附条件 | 附期限 |
| :--- | :--- | :--- |
| **所附事实** | 发生**不确定**（或然） | 发生**确定**（必然） |
| **判断方法** | “如”后面的事**不一定**发生 | “如”后面的事**一定**会发生 |

> **判断例题**：
>
> **例1**：“当我活到一百岁的时候，把表送给你。”→**附条件**（活到一百岁不确定）
>
> **例2**：“我死了，手机就送你。”→**附期限**（人的死亡是必然的）
>
> **例3**：“下次北京下雨，杯子送你了。”→**附期限**（北京下次一定会下雨）


### 第六段：本专题总结

**专题三（民事法律行为）总结**：

1. **民事法律行为 vs 事实行为**：看跟年龄有没有关系。有关系→民事法律行为；没关系→事实行为。
2. **单方民事法律行为**：动产所有权的抛弃、遗嘱、遗赠（一个人说了算）
3. **双方民事法律行为**：合同（特别是赠与合同）——需要双方意思表示合意
4. **效力状态**：有效、无效、待定、撤销
5. **附条件的民事法律行为**：平均每2-3年考一次。看到“如”后面那句话，判断是否发生不确定——不确定的就是**附条件**。瞎蒙也蒙附条件。
