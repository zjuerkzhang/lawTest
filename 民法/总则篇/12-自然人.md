
## 第1部分：对照《民法典》自然人提取备考提纲


### 一、民事权利能力（民法典第13-16条）

| 阶段 | 规则 | 法典依据 |
| :--- | :--- | :--- |
| **活着** | 民事权利能力始于出生，终于死亡 | 第13条 |
| **生前（胎儿）** | 原则上不是人；例外（遗产继承、接受赠与、侵权行为）**视为**具有民事权利能力 | 第16条 |
| **死后（死者）** | 死者人格利益保护——保护**近亲属**的情感利益 | 第994条 |


### 二、民事行为能力（民法典第17-24条）

| 类型 | 年龄 | 行为效力 |
| :--- | :--- | :--- |
| **无民事行为能力人** | 不满8周岁 | 独立实施的行为**一律无效**（第144条） |
| **限制民事行为能力人** | 8周岁以上不满18周岁 | 相适应的**有效**；不相适应的**效力待定**（第145条） |
| **完全民事行为能力人** | 18周岁以上 | 可独立实施法律行为 |
| **视为完全民事行为能力人** | 16周岁以上+劳动收入为主要生活来源 | **视为**完全民事行为能力人（第18条第2款） |


### 三、监护制度

| 监护类型 | 法典依据 | 核心规则 |
| :--- | :--- | :--- |
| **法定监护** | 第27条 | 五个顺序限制；父母是当然监护人；离婚不影响监护 |
| **遗嘱指定监护** | 第29条 | 父母通过遗嘱指定；无顺序人数限制；优先于法定监护 |
| **协议监护** | 第30条 | 开家庭会议；无顺序人数限制；父母不得免责 |
| **指定监护** | 第31条 | 四个单位介入；法院终局效力；择优指定；不可拒绝 |
| **成年人意定监护** | 第33条 | 成年完人书面自己找；优先于法定监护 |
| **临时委托监护** | 第1189条 | 找人临时照顾；委托人仍是监护人 |


### 四、宣告失踪（民法典第40-45条）

| 要点 | 规则 |
| :--- | :--- |
| **条件** | 下落不明满2年 |
| **法律效果** | 不影响人身关系+财产关系 → 设财产代管人 |
| **财产代管人** | 以自己名义起诉应诉；无处分权；故意或重大过失才赔偿 |


### 五、宣告死亡（民法典第46-53条）

| 要点 | 规则 |
| :--- | :--- |
| **条件** | 一般4年 / 意外2年 / 意外+证明不可能生还=随时 |
| **法律效果** | 妻离子散、家破人亡（影响人身+财产关系） |
| **撤销后的婚姻** | 原则上自动恢复；例外（再婚、书面声明不愿恢复）不恢复 |
| **撤销后的财产** | 能返还的返还；不能返还的**适当补偿**；恶意宣告死亡要**赔偿** |

### 第一段：自然人概述

好，接下来咱们来看专题二，叫做民事主体。咱们前面专题一就给大家介绍过，咱们的民事主体是**三元论**——自然人、法人和非法人组织。咱们这个专题也会用三节分别全面细致地讲授这三大主体。

先来看第一节，叫自然人。它是客观题每年必考事项，又简单，分又多。自然人向下一共就**五个问题**：
1. **民事权利能力**
2. **民事行为能力**
3. **监护**
4. **宣告失踪**
5. **宣告死亡**

就一个巴掌五个问题。

### 第二段：民事权利能力——做人的资格

先来看第一个问题，叫自然人的民事权利能力。在它旁边写上四个字——**做人的资格**。它要回答的第一个问题就是：**你是法律意义上的人吗？**

民法典第十三条明文规定：自然人的民事权利能力**始于出生，终于死亡**。出生和死亡两个节点，把人的一生分成了三段：**生前、活着和死后**。活着是原则，考试考两端例外——**生前（胎儿）** 和**死后（死者）**。


### 第三段：胎儿利益保护（民法典第16条）

**一个原则，三个例外**：原则上胎儿不是法律意义上的人，不具有民事权利能力。但在三种例外情况下，**视为**具有民事权利能力。

**三种例外情况**：

1. **遗产继承**（胎儿是**继承人**）
2. **接受赠与**（胎儿是**受赠人**）
3. **侵权行为**（胎儿是**受害人**）

**核心特点**：纯获法律上利益——**只有权利，没有义务**。法律拟制的立法技术。

> **胎儿应留份例题**：熊某与杨某结婚后，杨某与前夫所生之子小强由二人共同抚养。熊某死亡，未立遗嘱。杨某怀有一对龙凤胎（于熊某死后生产），男婴娩出时为死体，女婴娩出时为活体（但旋即死亡）。
>
> **问一**：熊某第一顺序法定继承人有几个？
>
> **答**：**四个**——配偶杨某、有抚养关系的继子小强、男婴（预留一份）、女婴（预留一份）。
>
> **问二**：为男婴保留的遗产份额如何处理？
>
> **答**：男婴娩出时为死体，民事权利能力**自始不存在**。其保留的份额按熊某的遗产重新分割——由杨某、小强、女婴继承。
>
> **问三**：为女婴保留的遗产份额如何处理？
>
> **答**：女婴活体出生，已取得遗产份额，活了一秒后死亡，其份额按**女婴的遗产**处理，由其唯一第一顺序法定继承人**母亲杨某**继承。

> **胎儿接受赠与例题**：郭某死亡，其妻甲怀孕两个月。郭某父亲与甲签订协议：如把孩子顺利生下来，就送十根金条给孩子。当日已将八根金条交给甲。孩子顺利出生后，甲不同意由郭某父亲抚养孩子，郭某父亲拒绝交付剩余两根金条，并要求甲退回八根金条。
>
> **问一**：胎儿能否接受赠与？
>
> **答**：**能**。接受赠与是纯获法律上利益（只有权利没有义务）。
>
> **问二**：赠与合同的当事人是谁？
>
> **答**：郭某父亲（赠与人）和胎儿（受赠人）。由胎儿母亲甲作为法定代理人代为表示接受。
>
> **问三**：郭某父亲是否有权不交付剩余两根金条？是否有权要求退回八根金条？
>
> **答**：**无权**。合同有效，必须履行。这不是戏谑行为（签了协议，不是口头说大话）。
>
> **问四**：如胎儿娩出时为死体，如何处理？
>
> **答**：民事权利能力自始不存在，已交付的财产构成**不当得利**，应当返还。

> **胎儿侵权损害赔偿例题**（孕妇医疗损害案）：孕妇A前往甲医院就诊，医生因重大过失开错药，造成A身体受损，孩子B娩出后亦有残疾。
>
> **分析**：
> - **甲医院与孕妇A**：医疗服务合同关系，既违约又侵权——**择一选择**（民法典第186条）
> - **甲医院与医生**：劳动合同关系，甲医院替医生承担侵权责任（**替代责任**）
> - **甲医院与孩子B**：侵权损害赔偿关系（只有侵权，没有合同）
>
> **问**：下列说法正确的是？
>
> A. 仅A可要求医院赔偿——**错**
>
> B. 仅B可要求医院赔偿——**错**
>
> C. A和B均可要求医院承担损害赔偿责任——**正确**（A可二选一，B只有侵权）
>
> D. 医生承担侵权责任——**错**（医院承担替代责任）


### 第四段：死者人格利益保护（民法典第994条）

死者人格利益保护保护的不是死者，保护的是**八类近亲属的情感利益**。

**三个数字：七八二**

- **七**：七项人格利益——**姓名、肖像、名誉、荣誉、隐私、遗体、遗骨**。考试重点：**名誉**和**遗骨**。
- **八**：八类近亲属——**配偶、父母、子女、兄弟姐妹、祖父母、外祖父母、孙子女、外孙子女**。
- **二**：两个顺序——**第一顺序：配偶、父母、子女**；第二顺序：其他五类近亲属。有第一顺序的，第二顺序的不能上场。

> **死者人格利益保护例题**：欣欣美容医院为欢欢实施隆鼻手术过程中因未严格消毒导致欢欢面部感染留下疤痕。欢欢诉诸法院。欢欢因急性病发作猝死。网络名人杨洋在其微博上杜撰“欢欢吸毒过量致死”。
>
> **问一**：杨洋侵犯了欢欢的何种人格利益？
>
> **答**：**名誉**（一字之差——是“利益”不是“权利”）。
>
> **问二**：欢欢的舅舅能否提起侵权之诉？
>
> **答**：**不能**。舅舅**不是**近亲属。
>
> **问三**：欢欢的姐姐能否提起侵权之诉（在欢欢配偶、父母、子女均存在的情况下）？
>
> **答**：**不能**。姐姐是第二顺序，有第一顺序时第二顺序不能上场。
>
> **问四**：欢欢的配偶、父母、子女意见不一致，怎么办？
>
> **答**：谁想去谁去，**意思自治**。
>
> **问五**：欢欢的母亲起诉，应以谁的名义？
>
> **答**：**自己的名义**（不是以死者的名义）。
>
> **问六**：法院判决赔偿金10万给母亲，配偶和子女能否主张遗产分配？
>
> **答**：**不能**。这是**死后**产生的赔偿金，不是“死亡时遗留的个人合法财产”，不属于遗产。谁起诉归谁。


### 第五段：民事行为能力——独立做事的资格

民事权利能力要回答“你是人吗”，民事行为能力要回答“**你是什么人**”——无民事行为能力、限制民事行为能力还是完全民事行为能力？重点解决**独立做事的资格**。

**影响民事行为能力的因素**：年龄 + 精神健康状况。考试重点考年龄。

#### （一）三类行为能力人（民法典第18-22条）

| 类型 | 年龄标准 | 行为效力 |
| :--- | :--- | :--- |
| **无民事行为能力人** | 不满**八周岁** | 独立实施的民事法律行为**一律无效** |
| **限制民事行为能力人** | 八周岁以上不满**十八周岁** | 与其意思能力相适应的行为**有效**；不相适应的**效力待定** |
| **完全民事行为能力人** | 十八周岁以上 | 可以独立实施民事法律行为 |

**特别注意**：民法典第18条第2款（**八幺六**）——十六周岁以上的未成年人，以**自己的劳动收入**为主要生活来源的，**视为**完全民事行为能力人。两个条件缺一不可：①年龄≥16周岁；②劳动收入为主要生活来源。

**就低不就高原则**：年龄标准与精神健康状况并存时，以**就低不就高**判断行为能力（保护弱者）。

#### （二）无民事行为能力人独立实施的行为一律无效（民法典第144条）

无民事行为能力人（不满8周岁）独立实施的民事法律行为**一律无效**。吃穿用都由家长决定。

**为什么胎儿接受赠与有效，但无民事行为能力人独立接受赠与无效？** 这是两个不同层面的问题：
- **胎儿利益保护**（民事权利能力问题）：胎儿纯获法律上利益（只有权利没有义务），由妈妈代理，不损害胎儿利益
- **无民事行为能力人独立签合同**（民事行为能力问题）：五岁孩子独立决定可能损害其利益（如抽烟、放爆竹），必须由家长代理

#### （三）限制民事行为能力人独立实施的民事法律行为的效力（民法典第145条，每年必考）

**判断标准**：与本人生活关联程度 + 价款或报酬数额大小（一般理性人标准）。

**买卖合同**：买矿泉水（5元）→ **有效**；买钻戒（2万元）→ **效力待定**（家长决定）。

**赠与合同**：需区分赠与人身份和是否附义务。

| 情形 | 效力 |
| :--- | :--- |
| 15岁学生赠5元矿泉水 | **有效**（与意思能力相适应） |
| 15岁学生赠2万元钻戒 | **效力待定**（价款过高，不相适应） |
| 15岁学生受赠1万元压岁钱（无附义务） | **有效**（纯获法律上利益） |
| 15岁学生受赠1万元（附义务：专款专用） | **效力待定**（不是纯获利益） |

> **例题（肖特案，2017年真题）** ：肖特有音乐天赋，16周岁便不再上学，以演出收入为主要生活来源。
>
> **问**：下列哪一说法正确？
>
> A. 肖特尚不具备完全的民事行为能力——**错**（16周岁+劳动收入=**视为**完全民事行为能力人）
>
> B. 7周岁时受赠口琴的行为无效，应由其法定代理人代理实施——**正确**（无民事行为能力人独立行为一律无效）
>
> C. 9周岁时受赠钢琴的行为无效——**错**（9周岁限制民事行为能力人，纯获利益，有效）
>
> D. 15周岁时受赠名贵小提琴的行为无效——**错**（纯获利益，有效）
>
> **答案：B**


### 第六段：监护制度——找家长

监护的概念：为无民事行为能力人和限制民事行为能力人**找家长**。

监护制度的体系：**以家庭监护为基础，以社会监护为补充，以国家监护为兜底**的三位一体监护制度。

**监护人确定（选任）**——考试的绝对重点★★★★★

#### （一）法定监护（民法典第27条）

法律直接给孩子找监护人，**有五个顺序限制**（为防止推卸责任）：
1. **父母**（第一顺序，当然的监护人）
2. 祖父母、外祖父母
3. 兄、姐
4. 其他愿意担任监护人的个人或组织（须本人愿意 + 有关单位同意）
5. 民政部门或具备条件的居委会/村委会（公职监护人/国家监护）

**核心**：只要父母还健在且还有监护能力，**其他任何人不得插手孩子的监护事宜**。父母是未成年人的第一顺序的法定的首要的当然的监护人。

**父母的扩大解释**：不仅包括生父母，还包括**养父母**（拟制血亲），以及有抚养关系的继父母（考得少）。

**离婚不影响监护**（民法典第1084条）：父母离婚**不影响监护**，也不影响侵权责任，**仅影响抚养**。谁直接抚养、谁间接抚养（给抚养费、有探望权）的问题。

> **例题**：陆某因诈骗罪被判入狱，和妻子协议离婚，约定12岁的儿子由陆某抚养（实际由陆某父母抚养）。
>
> **问**：下列说法正确的是？
>
> **答**：**夫妻二人均为监护人**。离婚不影响监护。

**父母一方死亡**：另一方为监护人。第二顺序（祖父母、外祖父母）不能“争当”监护人——有前一个顺序的，后一个顺序不能上场。

#### （二）遗嘱指定监护（民法典第29条）

父母死亡前通过遗嘱给孩子找监护人。

**核心规则**：
- **主体**：只能是**父母**（其他人不能通过遗嘱指定监护人）
- **无顺序人数限制**：爱指定谁指定谁（爸爸的哥们、妈妈的闺蜜、隔壁老王）
- **被指定人可以拒绝**（说NO）
- **优先性**：被指定人愿意担任的，**优先于**法定监护（尊重父母的选择权）

**法定监护是兜底**——如果遗嘱指定的人拒绝，由法定监护兜底。

#### （三）协议监护（民法典第30条）

依法具有监护资格的人之间通过协议确定监护人——**开家庭会议**给孩子找监护人。

**核心规则**：
- **主体**：必须是**具有监护资格的人**（家庭成员）
- **父母不得通过协议免除监护职责**（免责无效）
- **无顺序人数限制**：可以打破法定监护的顺序，可以选单独监护人也可以选共同监护人（随便拼）
- **前提**：父母双亡或均丧失监护能力

#### （四）指定监护（民法典第31条）

协议监护不成（都想当或都不想当）时，由四个单位介入给孩子找监护人。

**四个单位**：**居委会、村委会、民政部门、人民法院**。其中**人民法院具有终局效力**。

**重大制度修改（从未考过）** ：指定监护由**前置程序改为并列关系**——当事人可以直接申请法院司法指定，不需要先经过居委会/村委会/民政部门的行政指定。

**前提**：父母双亡或均丧失监护能力 + 协议监护不成。

**法院指定原则**：**择优指定原则**，无顺序人数限制，但限定在“具有监护资格的人”（家庭成员）范围内。

**临时监护人制度（新增，从未考过）** ：法院指定监护人之前可能有一个空档期，此时由**四类组织**（居委会、村委会、有关组织、民政部门）担任临时监护人，**不能由自然人担任**。

**法院指定不可拒绝**——被指定人必须担任。

#### （五）遗嘱指定监护 vs 指定监护的区别

| 对比项 | 遗嘱指定监护 | 指定监护 |
| :--- | :--- | :--- |
| **指定主体** | **孩子的父母** | **四个单位**（居委会、村委会、民政部门、法院） |
| **被指定人范围** | **无限制**（爱指定谁指定谁） | 限于**具有监护资格的人**（家庭成员） |
| **被指定人能否拒绝** | **可以拒绝** | **法院指定不可拒绝** |

#### （六）成年人意定监护（民法典第33条）

**成年完人**通过**书面委托合同**自己给自己找监护人。

**核心规则**：
- 主体：必须是**成年完人**（在自己还清醒的时候）
- 形式：**书面**（证据 + 慎重）
- 无顺序人数限制
- **优先于法定监护**（尊重老年人意思自治）
- 本质是委托合同，双方均有**任意解除权**（在生效前可以随意反悔）

#### （七）临时委托监护（民法典第1189条）

通过委托合同找人临时照顾被监护人——解决**留守儿童**问题。

**核心规则**：
- **无顺序人数限制**
- **不导致监护职责的丧失，委托人（父母）依然是监护人**
- 受托人**有过错的承担相应责任**

> **临时委托监护 vs 协议监护**：
>
> | 对比项 | 协议监护 | 临时委托监护 |
> | :--- | :--- | :--- |
> | **目的** | 确定监护人（找家长） | 找人临时照顾被监护人（找人干活） |
> | **主体** | 限于具有监护资格的人（家庭成员） | 不受监护资格限制（爱选谁选谁） |
> | **法律效果** | 监护人承担侵权责任 | 委托人（父母）仍是监护人，受托人有过错才承担相应责任 |


### 第七段：监护人的职责（民法典第34-35条）

**监护是上位概念**，包括**抚养、教育、保护**三个下位概念。

**学校 vs 监护人**：学校仅有教育、管理和保护的职责（没有抚养义务）。未成年人在校期间，**监护职责并未转移给学校**——孩子在学校打人，侵权责任仍由父母承担。

**处分被监护人财产的限制**（民法典第35条）：监护人除为维护被监护人利益外，**不得处分被监护人的财产**——“非处分不可的程度”。判断标准：与孩子生活相关联程度。给孩子买文具、治病可以；给孩子炒股、买车不行。

> **例题**：甲8周岁，多次在国际钢琴大赛中获奖并获得大量奖金。甲的父母乙丙为了甲的利益，将奖金全部购买了股票（恰逢股市暴跌，损失过半）。
>
> **问**：乙丙作为监护人，是否要承担责任？
>
> **答**：**要承担赔偿责任**。购买股票不属于“非处分不可的程度”，与8周岁孩子生活相关程度极弱。


### 第八段：监护人资格的撤销（民法典第36-37条）

**适用情形**：**严重侵权**（积极侵权：性侵害、出卖、遗弃、虐待、暴力伤害；消极侵权：怠于行权导致被监护人危困）。

**申请主体**：有关个人或组织均可，重点——**民政部门必须申请**。

**撤销机关**：**法院**（适用特别程序）。

**法律后果**：监护职责被撤销，但**抚养、教育、保护的义务继续履行**——不能因祸得福免责。

> **例题**：孙某依法收养了孤儿小丽（11周岁）为养女，后多次对小丽实施性侵害。法院判决孙某构成强奸罪，有期徒刑三年。
>
> **问**：下列说法错误的是？
>
> A. W县民政部门可以直接撤销孙某的监护人资格——**错**（民政部门只能申请，撤销权在法院）
>
> B. 孙某被撤销监护人资格后，可以不再继续给付抚养费——**错**（继续履行义务）
>
> C. 孙某出狱后如确有悔改表现，经申请，法院可以恢复其监护人资格——**错**（故意犯罪，永久丧失监护人资格，一次不中百次不用）
>
> D. 小丽对孙某的损害赔偿请求权的诉讼时效期间自法定代理终止之日起计算——**错**（性侵害侵权自年满18周岁之日起算）


### 第九段：监护人资格的恢复（民法典第38条）

**适用情形**：确有悔改表现 + 尊重被监护人真实意愿。

**例外（不得恢复）** ：**故意犯罪**——一次不中，百次不用，永久丧失监护人资格。

**申请主体**：只有**父母或子女**（父母子女之间才能恢复）。


### 第十段：监护关系的终止（民法典第39条）

- **绝对终止**：被监护人取得或恢复完全民事行为能力 / 被监护人或监护人死亡
- **相对终止**：监护人丧失监护能力（可能恢复）

**父母离婚不会导致监护关系终止**——重申：离婚不影响监护。


### 第十一段：宣告失踪——财产代管制度

**法律效果**：既不影响人身关系，也不影响财产关系。目的是为失踪人找一个**财产代管人**。

**条件**：下落不明持续满**2年**（民法典第40条）。

**程序**：利害关系人申请 → 基层人民法院受理 → 公告期**3个月** → 宣告失踪 → 指定财产代管人。

**利害关系人**（宣告失踪）：
- **当然的利害关系人**：配偶、父母、子女、履行了赡养义务的丧儿媳/丧女婿
- **财产性利害关系人**（债权人、债务人、合伙人）：原则上可以申请，例外——不申请宣告失踪不影响其权利行使的除外（如夫妻共同债务中，可直接告另一方）

**财产代管人**（民法典第42-44条）：
- 适格代管人：配偶、父母、成年子女、其他愿意担任的人
- 财产代管人**没有处分失踪人财产的权利**——处分属于**无权处分**
- 财产代管人**以自己的名义**起诉应诉
- 无偿代管，只有**故意或重大过失**才承担赔偿责任（民法典第43条）
- 变更财产代管人：利害关系人申请（被动罢免）或代管人主动辞职

> **宣告失踪例题（2022年真题）** ：袁某和祖某系夫妻，后二人吵架，袁某离家出走，两年后祖某申请法院宣告袁某失踪。
>
> **问**：下列哪一说法正确？
>
> A. 袁某被宣告失踪后丧失民事行为能力——**错**（宣告失踪不影响人身关系）
>
> B. 袁某被宣告失踪后，祖某可以继承袁某的财产——**错**（宣告失踪不影响财产关系）
>
> C. 袁某父母不同意，可以申请撤销失踪宣告——**错**（撤销需“重新出现”）
>
> D. 祖某提出离婚诉讼的法院应当准许——**正确**（宣告失踪是离婚的法定情形之一）

**撤销**：重新出现 → 本人或利害关系人申请 → 基层法院撤销。


### 第十二段：宣告死亡（法律推定死亡）

**与宣告失踪的区别**：宣告死亡既影响人身关系，也影响财产关系——**“妻离子散，家破人亡”**。

#### （一）宣告死亡的条件（民法典第46条）

| 情形 | 期间 | 公告期 |
| :--- | :--- | :--- |
| **一般情形** | 下落不明满**4年** | **1年** |
| **意外事件** | 下落不明满**2年** | **1年** |
| **意外事件且证明不可能生还** | **随时**（无时间限制） | **3个月** |

**口诀：四二零幺幺三**

#### （二）利害关系人（宣告死亡 vs 宣告失踪）

| 对比项 | 宣告失踪 | 宣告死亡 |
| :--- | :--- | :--- |
| **当然的利害关系人** | 配偶、父母、子女、丧儿媳/女婿 | 配偶、父母、子女、丧儿媳/女婿 |
| **财产性利害关系人** | **原则上可以**（例外：共同债务中可直接告另一方） | **原则上不可以**（例外：不申请宣告死亡不能保护其合法权益的除外——如人寿保险中的保险公司） |
| **代位继承人** | 可以 | **有条件**（如遗嘱给孙子，儿子不去宣告死亡会损害孙子利益的，可以） |

#### （三）死亡日期的确定（民法典第48条）

- **原则**：**判决作出之日**视为死亡日期
- **例外**：意外事件，**意外事件发生之日**视为死亡日期（未考过）

#### （四）宣告死亡的法律效果——“妻离子散，家破人亡”（民法典第51条）

- **妻离**：与配偶的婚姻关系**自动消除**
- **子散**：配偶**单方有权送养**未成年子女（限制：①符合送养条件；②死亡一方的父母有**优先抚养权**）
- **家破**：个人合法财产按遗产开始继承（有配偶的，**先析产**再继承）
- **人亡**：推定自然死亡（可推翻）

#### （五）死亡宣告的撤销——王者归来（民法典第50-53条）

**撤销条件**：**重新出现**（本人或利害关系人申请）。

**撤销后的法律效果——不一定★★★★★**：

**婚姻关系**（民法典第51条）：是否自动恢复？**不一定**。
- **原则**：婚姻关系**自动恢复**
- **例外**（不自动恢复）：①配偶**再婚**（包括再婚后又离婚、再婚后配偶死亡）；②配偶向婚姻登记机关**书面声明**不愿意恢复（未考过）

**子女收养关系**（民法典第52条）：不得以未经本人同意为由主张收养行为无效（原则）；例外：收养人同意 + 被收养人同意（8周岁以上需征得其同意）。

**财产关系**（民法典第53条）：
- **能返还的返还**；不能返还的**适当补偿**
- **恶意宣告死亡** → 不仅要返还财产，还要**赔偿损失**（侵权责任，未考过）

**被宣告死亡期间实施的民事法律行为**（民法典第49条）：“不影响” = 具体问题具体分析，跟没死一样判断——可能有效、无效、可撤销、效力待定，**不一定**。

#### （六）宣告失踪与宣告死亡的关系（民法典第47条，从未考过）

同一自然人，有的利害关系人申请宣告失踪，有的申请宣告死亡，符合宣告死亡条件的，人民法院**应当宣告死亡**（定分止争，一次性解决纠纷）。


### 第十三段：自然人部分总结

五个问题，一个巴掌：
1. 民事权利能力——你是人吗？（生前胎儿保护 + 死后人格利益保护）
2. 民事行为能力——你是什么人？（三类行为能力人）
3. 监护——你的家长呢？（找家长：法定、遗嘱指定、协议、指定、成年意定、临时委托）
4. 宣告失踪——找财产代管人（不影响人身关系 + 财产关系）
5. 宣告死亡——妻离子散、家破人亡（影响人身关系 + 财产关系；撤销后不一定）
