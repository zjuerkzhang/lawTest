# 民事法律行为

民事法律事实：能够导致民事法律关系产生，变更，消灭
- 行为：与人的意识有关
- 事件

## 民事法律行为

### 基本原理
考题：未成年人

注意区分民事法律行为和事实行为的区别：如果跟年龄有关系，就是民事法律行为，比如签订合同，结婚等；如果无关，就是事实行为，比如侵权行为（打架致人受伤），创作行为。

### 法定分类
- 单方民事法律行为：一个人说了算，完成时即成立
    - 考点：
        - 动产所有权的抛弃
        - 遗嘱：对象为法定继承人
        - 遗赠：对象为法定继承人之外的人
- 双方民事法律行为：**意思表示**合意的结果
    - 考点：合同
        - 买卖合同
        - 赠予合同：受赠人需要表示接受才能生效
- 决议行为：公司法中股东会决议

### 学理分类
- 财产行为：物债两分
    - 负担行为：发生债权债务，房屋出租
    - 处分行为：物权效力
- 身份行为


### 效力（主观题）
总则编中民事法律行为的效力专指合同的效力

#### 合同效力
分两个阶段
- 成立：事实判断问题，解决有没有合同关系
- 效力：价值判断问题，解决经不经得起法律的考量

合同涉及到自愿和法律限制问题
- *个人理解自愿保证了成立，法律限制确定其效力*

#### 民事法律行为效力形态（2018年之后，主要考合同的效力形态）
- 有效
    - 能够发生当事人意欲发生、主动追求或内心期待的法律效果
    - 强公主义
        - 强：不违反法律行政法规的强制性规则 -> 合法合规
        - 公：不违背公序良俗
        - 主：主体具有相应的民事行为能力
        - 义：意思表示真实
            - 意思表示不真实的不一定就是无效，也可能是可撤销的（比如欺诈，胁迫）
- 无效
    - 不发生当事人意欲发生、主动追求或内心期待的法律效果
    - 会发生法律意欲发生的法律后果：恢复原状、缔约过失责任
    - 五种法定情形：
        - 强：违法，买卖毒品
        - 公：背俗
            - 强迫结婚约定
            - 限制生育约定
            - 买卖器官
            - 代孕
            - 第三者赠予
            - 危房出租，影响其三人利益，违反公共秩序
        - 主：主体不具备相应民事法律行为能力
            - 无民事行为能力人无效
        - 义（主观题，民法典146条）
            - 违反自愿原则，名实不符
            - 双方通谋意思虚假表示
                - 主要税收问题：明星偷税，房屋交易，一交易两合同（阴阳合同）
        - 恶：恶意串通损害他人合法权益
            - 三个条件：
                - 互相勾结（知情不等于恶意传统，很难证明）
                - 谋取私利
                - 损害他人合法权益
            - 例外例子：
                - 夫妻股权转让：夫妻共同出资，登记在一个人名下，股权转让原则有效。除非转让人和受让人串通，导致损害夫妻关系另一方
    - 部分无效
        - 争议解决条款有效
        - 例子：够上刑法的故意伤害中，私了协议中民事部分有效，涉及刑事部分无效
    - 无效的民事法律行为（民法典157条）
        - 恢复原状
        - 缔约过失责任
- 待定（主体的问题）
    - 行为能力欠缺（民法典145条）
        - 原则：未成年人的利益保护要优先于交易安全的保护
        - 四要素
            - 限制民事行为能力人
            - 独立实施
            - 与其意思表示能力不相适应
            - 双方民事行为（合同行为）
        - 法定监护人/代理人的追认权和拒绝权，都是形成权（待定 -> 自始有效/无效）
            - 明示追认
            - 明示拒绝
            - 默示拒绝（30天【除斥期间】不表态）
        - 善意相对人的催告权和撤销权（合同相对方）
            - 催告权不是形成权，撤销权是形成权
            - 善意表示对方不知情主体是限制行为能力人
            - 非善意没有撤销权
    - 代表权和代理权的欠缺
- 撤销
    - 可以撤销，也可以不撤销
    - 四种情况：
        - 重大误解
            - 本质：自身原因，对主给付义务认识错误
            - 双方都可以要求撤销
            - **重大**
                - 主给付义务（对应**重大**）：买房
                    - 房屋所有权
                    - 房屋质量
                - 从给付义务：买房
                    - 配套设施
                    - 单证资料
                        - 发票
                        - 使用说明书
                - 附随义务
                    - 通知，协助，保密
                - 不真正义务
            - **误解**
                - 当事人
                    - 对对方当事人认识错误，双胞胎
                - 标的物物
                    - 标的物：期房的实际面积和合同面积不同，不算误解
                    - 赌石、古玩等特殊标的物，不算误解
                - 法律行为性质（关系）
                    - 误把有偿当无偿，把买卖合同当赠予合同
                - 工具：信息传递
                    - 第三人转达错误：网购
        - 欺诈
            - 故意告知虚假情况，或负有告知义务的人故意隐瞒，致使当事人基于错误认识做出违背内心真意的意思表示。
            - 受欺诈方可以要求撤销
            - 本质：非自身原因，认识错误
            - 要件：
                - 故意
                - 欺诈行为
                    - 积极欺诈
                        - 对应故意告知虚假情况
                    - 消极欺诈
                        - 负有告知义务的人故意隐瞒，凶宅
                - 认识错误
                - 违背内心真意的意思表示
            - 第三人欺诈（主观题，149条）
                - 对方知道或应当知道，非善意
        - 胁迫
            - 对象包括自然人、法人和非法人组织
            - 受胁迫方可以单方要求撤销
            - 具有不合法性
            - 胁迫的核心要点
                - 不正当的预告危害，刑法的说法是以恶害相通告，手段或目的不合法
                - 陷入恐惧和害怕的心理状态，应当看受胁迫人的心理状态
                - 第三人胁迫：具有不法性，谁实施都能撤销
        - 显失公平
            - 成立时
            - **利用**对方处于**危困状态**：家人患病急需用钱
            - **缺乏判断能力**：特定人群、大一大学生、老年人
                - 醉酒
                    - 灌醉
                    - 自醉
                        - 被**利用** -> 显示公平
                        - 没有被**利用** -> 重大误解
            - 假一罚十构成单方允诺之债，不得基于显失公平而实施撤销权
    - 撤销权的行使
        - 行权主体
            - 只有重大误解，双方可以撤销，其他都是单方撤销
        - 行权方式（形成诉权）
            - 需要专业人员价值判断
            - 或诉（起诉），或裁（仲裁）
            - 双重（主观，客观）除斥之间，权利不用，过期作为
                - 主观：当事人主观知道起算，1年有效
                - 客观：民事法律行为（合同关系）成立起算，5年有效
                - 哪个先到期算哪个
            - 法律后果
                - 撤销前，合法有效
                - 撤销权，并追究缔约过失责任

**婚姻关系没有待定的效力状况**

**遗嘱，遗赠，动产所有权的抛弃只有有效，无效效力状况，没有待定和撤销**（单方面法律行为，完成即生效）

**知假卖假**构成欺诈，**不知假卖假**构成重大误解

### 特殊的民事法律行为
**附条件**和**附期限**的民事法律行为

#### 特征
- 表意人选定
- 将来发生
- 发生与否不确定，或然性
    - 附条件不确定
    - 附期限确定
- 合法事实
    - 包养情人中的赠予合同，包养不合法

#### 分类
- **生效条件**（民法典158）
    - 不诚信，为了自己的利益，阻止条件的成就，视为条件已成就（拟制效力）（民法典159）
    - 例子：附生效条件的赠予合同
- **解除条件**
    - 例子：附解除条件的租赁合同
- 积极条件
- 消极条件


## 代理（代替他人处理事务）
### 种类
- 代理他人打官司
- 代理他人去签合同

事实行为无法代理，代理的对象是民事法律行为，不是所有的民事法律行为都可以被代理
- 可代理的民事法律行为
    - 合同签订
    - 诉讼离婚
- 不可代理的民事法律行为（以民政部门登记为准的话，不能代理）
    - 结婚
    - 协议离婚
    - 收养行为
    - 遗嘱、遗赠（可以代书）

### 委托和代理
- 代理
    - 三方关系
        - 被代理人/本人
        - 代理人
        - 相对人/第三人
- 委托
    - 两方
        - 委托人
        - 受托人
- 有委托不一定有代理：委托监护
- 有代理不一定有委托：法定代理人/监护人

### 代理权滥用

#### 前提
有权代理

#### 类型
##### 自己代理
- 代理人和相对人为同一个人
- 合同相对无效
- 道德风险

##### 双方代理
- 代理人代理双方
- 合同相对无效

##### **恶意代理**
- 代理人吃里爬外，拿回扣
- 合同效力待定，让被代理人自己做决定
    - 追认 -> 合同自始有效
    - 不追认 -> 合同自始无效，被代理人有损失，则代理人和相对人承担连带责任

```
**相对无效和效力待定**
- 相对无效：默认无效。无效是常态，有效是例外
- 效力待定：不确定/悬而未决，依赖被代理人的选择
```

##### 复代理（转委托，第二次代理）
本代理（一次代理）

为被代理人的利益，在紧急情况下，可以先斩后奏，找复代理

##### 职务代理
公司对外有合同签约权的只有采购部和销售部，必需授权委托书

- 被代理人是法人或非法人组织
- 代理人是员工
- 相对人是客户

职务代理是**有权代理**，其权力来自于本职工作，非本职工作无权代理

**考点**：法人或非法人组织，对执行其工作任务的人员的职权范围的限制，不得对抗善意相对人。内涵：保护交易秩序，交易安全。

```
自然人代理，属于民事制度，强调公平性，一事一授权
商事代理强调效率，概括授权
```

##### 狭义的无权代理（民法典171条）
员工超越本质工作的代理为无权代理

主要出发点是保护被代理人

- 行为人类型
    - 没有代理权
    - 超越代理权，标的物
    - 代理权终止以后

- 被代理人有追认权和拒绝权
    - 追认
    - 拒绝追认（未做表示，视同拒绝追认）

- 善意相对人的催告权和撤销权（前提：相对人不知道行为人无权代理）
    - 追认前，相对人有权撤销
    - 被代理人追认后，执行合同
    - 被代理人拒绝追认后，善意相对人有权要求行为人履行义务或赔偿

##### 表见代理（民法典172条）

出发点是保护善意相对人

表见代理也是无权代理，相对人有理由相信

和无权代理的区别在于相对人是否**有理由**相信行为人有代理权
- 主观上相对人要善意，且无过失
    - 善意是推定的，一般都是
    - 无过失表示相对人要形式审查行为人的权利外观
        - 权利外观四择一：
            - 授权委托书，五要素
                - 代理人姓名或名称
                - 事项
                - 权限
                - 期限
                - 被代理人的签章
            - 介绍信：包括盖章
            - 交易习惯：考题包含“经常”，“往常”，“常常”，“通常”等词汇
            - 签约地在被代理人营业或办公场所

个人思考：被代理人和行为人串通

##### 真人假章，有人无章，有章无人
- 真人假章
    - 看人不看章，合同有效
- 有人无章
    - 看人不看章，合同有效
    - 合同约定必需盖章除外，有约从约
- 有章无人（结合民诉法）
    - 相对人（原告）能证明法定代表人或工作人员订立，则合同有效

```
表见代表和表见代理
表见代表：看起来像法定代表人
表见代理：看起来像代理人
合同效力合法有效，责任承担都由被代理人（公司）承担
相对人都要善意且无过失，都有审查义务
```


## 诉讼时效

以借款为例，在2021.1.1签订借款合同，在2022.1.1还款，时间划分如下：
- 2021.1.1~2022.1.1: 借款期间
    - 无债权期
    - 债务人无权要求还款
- 2022.1.1~2025.1.1: 诉讼时效期间（一般3年）
    - 保护债权人利益
- 2025.1.1~：诉讼时效期间届满后
    - 自然之债
    - 保护债务人利益
    - 惩罚权利人怠于行权

### 基本原理
#### 性质
强制性规定，完全依照法律规定

#### 适用范围
诉讼时效约束的是请求权

- 债权请求权：要钱不要物
    - 权利人没有怠于行权，就不能使用诉讼时效
        - 存款人要求银行还本付息
        - 债权人要求国债，企业债还本付息
    - 保护股东，企业，群体利益，不适用诉讼时效
        - 要求股东认缴出资（目的：保护股东和企业利益）
        - 国有资产，业主大会要求缴付公共维修资金（目的：保护广大业主利益）
    - 保护弱势群体，不适用诉讼时效
        - 三费：抚养费，赡养费，扶养费（保护弱势群体）
    - *工资债权适用诉讼时效*，有劳动法保护员工利益
- 物权请求权
    - 关键看是否有**登记**
    - 不动产不适用诉讼时效
    - 特殊动产不适用诉讼时效：汽车
    - 未登记普通动产适用诉讼时效
- 人格权请求权
    - 人格权受到侵害后，请求停止侵害、排除妨碍、消除危险，特别是消除影响、恢复名誉、赔礼道歉
    - 不适用诉讼时效，因为侵害一直在持续
- 离婚后的财产分割请求权
    - 不适用诉讼时效

```
请求权是权利人对义务人为或不为特定行为的权利
请求权的权利人和义务人都是特定的
```

#### 法律效果
##### 抗辩权发生说

过了诉讼时效，债权依然有效，成为自然之债，债务人可以提出抗辩，债权人丧失人民法院的强制执行力。
- 债权人依然可以起诉
- 法院应当受理
    - 法院不得提示债务人诉讼时效已过
- 债务人提出诉讼时效已过，法院驳回债权人诉讼请求
- 如果债务人一审未提抗辩权，二审时通常不支持，除非有新的证据

放弃抗辩权与同意履行
- 明示放弃
    - 部分同意履行，那么部分新债（单方承诺之债）发生，时效重新计算
- 默认放弃
    - 债务人直接还钱，但是不能反悔


### 种类
- 普通，3年
- 特殊
- 最长，20年
    - 医院抱错孩子不适用最长20年，法院可以延长时效
    - 拐卖儿童

### 起算
#### 合同之债
从主观起算，应当知道人和事
- 无约定还款期限
    - 催要时，如果债务人明确拒绝还款，以拒绝为起算点
    - 催要时，如果债务人未明确拒绝还款，以债权人要求的最后期限起算
- 约定确定还款期限
    - 按约定还款日起算
- 分期履行：房贷，分期还款，多个还款期限
    - 从最后一期为诉讼时效起算

#### 侵权之债
侧重于被监护人的利益被侵害
- 第三方实施侵害
    - 以监护人知道侵害发生为起算
- 监护人实施：起算复杂，有隐蔽性
    - 性侵害
        - 18岁-21岁，被监护人18岁作为起算
        - 如果超出上述时间，从当事人知道或应当知道侵权事件起算
    - 财产侵害
        - 从法定代理关系终止时起算
        - 如果超出上述时间，从当事人知道或应当知道侵权事件起算

### 中止（暂停）
最后六个月内，发生了特殊事件，暂停。特殊事件恢复之后，再算6个月
- 事件最后6个月内
- 客观条件，不可抗力

权利人欲行权而不能，一般由于客观原因，或不可抗力

### 中断
中断一次，加3年

中断触发条件
- 权利人积极行权
- 义务人同意履行

诉讼时长可以多次中断，但是总时长不能超过20年
