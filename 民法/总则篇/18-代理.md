

## 第1部分：对照《民法典》代理制度提取备考提纲


### 一、代理的适用范围（民法典第161条、第162条）

| 可以代理 | 不能代理 |
| :--- | :--- |
| 合同行为 | 事实行为（侵权等） |
| 诉讼（诉讼离婚可以代理） | 结婚、协议离婚、收养（必须亲自） |
| — | 遗嘱、遗赠（不能代理，但可以代书） |


### 二、委托与代理的关系（民法典第165条）

| 对比项 | 委托合同 | 代理 |
| :--- | :--- | :--- |
| 人数 | 2人 | 3人 |
| 关系 | 委托人与受托人 | 被代理人、代理人、相对人 |
| 代理权来源 | — | **授权委托书**（不是委托合同！） |

**授权委托书五个事项**（民法典第165条）：
1. 代理人姓名/名称
2. 代理事项
3. 权限
4. 期限
5. 被代理人签章


### 三、代理权滥用（民法典第164条、第168条）

| 类型 | 含义 | 法律效果 | 法典依据 |
| :--- | :--- | :--- | :--- |
| **自己代理** | 代理人以被代理人名义与自己实施民事法律行为 | **相对无效** | 第168条第1款 |
| **双方代理** | 同一代理人同时代理双方当事人 | **相对无效** | 第168条第2款 |
| **恶意代理** | 代理人与相对人恶意串通损害被代理人利益 | **效力待定**，被代理人不追认时代理人与相对人承担**连带责任** | 第164条 |

**恶意代理 vs 恶意串通（第154条）** ：
- 恶意代理（第164条）：损害**合同当事人**利益 → **效力待定**
- 恶意串通（第154条）：损害**合同以外第三人**利益（有外部性）→ **无效**


### 四、复代理（转委托，民法典第169条）

- 转委托原则上需要**被代理人同意**
- **紧急情况**下为被代理人利益，可**先斩后奏**
- 复代理人以**被代理人名义**实施法律行为，法律后果由**被代理人**承受
- 复代理人对被代理人造成损失的，代理人（转委托人）与复代理人可能承担**连带责任**（看各自过错）


### 五、职务代理（民法典第170条，主观题★★★）

**特征**：
- 被代理人：**法人或非法人组织**
- 代理人：**执行工作任务的人员**（员工）
- 代理事项：**本职工作范围内**

**核心规则**（第170条第2款）：法人或非法人组织对执行其工作任务的人员职权范围的限制，**不得对抗善意相对人**。

**职务代理一定是有权代理！** 在本职工作范围内即为有效。


### 六、无权代理（民法典第171条）

**三种情形**：
1. 没有代理权
2. 超越代理权
3. 代理权终止后

**法律效果**：**效力待定**
- 被代理人**追认** → 自始有效
- 被代理人**不追认** → 对被代理人不发生效力
- 善意相对人可请求**行为人**履行债务或赔偿损失（择一选择）


### 七、表见代理（民法典第172条）

**表见代理 = 无权代理 + 相对人有理由相信行为人有代理权**

**构成要件**：
- 主观：相对人**善意且无过失**
- 客观：有**权利外观**（授权委托书、介绍信、交易习惯、签约地/营业地）

**法律效果**：代理行为**有效**，对被代理人发生效力


### 八、表见代表（民法典第504条）vs 表见代理

| 对比项 | 表见代表 | 表见代理 |
| :--- | :--- | :--- |
| **行为人身份** | **法定代表人** | 非法定代表人（员工） |
| **是否需要授权委托书** | ❌ 不需要 | ✅ 需要 |
| **法律效果** | 有效，由法人承受 | 有效，由被代理人承受 |

**一字之差，差之千里——看身份！**


### 九、异常人章关系（合同编通则司法解释第22条）

**看人不看章**原则：

| 情形 | 规则 |
| :--- | :--- |
| **真人假章**（人真章假） | 合同**有效**（已考过） |
| **有人无章**（人签名无盖章） | 合同**有效**（除非约定必须盖章）（未考过） |
| **有章无人**（盖章无人签名） | 相对人**举证**证明在权限内订立 → **有效**（未考过，结合民诉） |

### 第一段：代理的适用范围

好，接下来咱们来看专题四，叫做代理制度。

代理的本质就是**代替他人处理事务**。从考试角度来看，就两件事：第一，代理他人去打官司；第二，代理他人去签合同。近些年来主要考的是**代理他人去签合同**。

**哪些事务可以代理，哪些必须亲自实施？**

代理的适用范围主要是**民事法律行为**。

- **事实行为（如侵权行为）不能代理**——你不能让别人替你去打人，否则构成共同侵权承担连带责任。
- **婚姻行为**——结婚必须亲自前往民政局，**不能代理**；协议离婚必须亲自前往民政局，**不能代理**；诉讼离婚**可以代理**。
- **收养行为**——必须亲自去民政部门办理收养登记，**不能代理**。
- **遗嘱、遗赠**——**不能代理**，但**可以代书**（别人不能替你写遗嘱，但可以帮你写字，你口述即可）。

【补充】法考以来，代理的适用范围还没考过。记住：能替你干的事就能代理，不能替你干的就不能代理。


### 第二段：委托与代理的关系

实践当中经常说“委托代理”，貌似他俩是一个事儿。其实很简单：**看人数**。

- **委托**：**两个人**——委托人和受托人，是合同关系。
- **代理**：**三个人**——**被代理人（本人）、代理人、相对人（第三人）** 。

| 关系类型 | 人数 | 关系 | 典型例子 |
| :--- | :--- | :--- | :--- |
| **委托合同** | 2人 | 委托人与受托人 | 委托邻居帮忙带孩子 |
| **代理** | 3人 | 被代理人、代理人、相对人 | 委托别人帮忙买茶叶并签约 |

> **判断方法**：看题目中有几个人。2个人→委托合同；3个人→代理。

**委托 vs 代理的辩证关系**：
- **有委托不一定有代理**——如委托监护（委托邻居带孩子，但没有代理关系）
- **有代理不一定有委托**——如法定代理（父母基于法律规定自然有代理权，不需要委托合同）


### 第三段：代理权滥用（有权代理）——核心考点

代理权滥用的前提是**有权代理**（代理人确实有代理权，只是滥用了）。

三种类型：

| 类型 | 含义 | 法律效果 |
| :--- | :--- | :--- |
| **自己代理** | 代理人以被代理人名义**与自己**实施民事法律行为（一身二职） | **相对无效**（原则上无效，被代理人追认则有效） |
| **双方代理** | 同一代理人**同时代理双方**当事人（一手托两家，一仆二主） | **相对无效**（原则上无效，双方均追认则有效） |
| **恶意代理** | 代理人与相对人**恶意串通**损害被代理人利益（吃里扒外、拿回扣） | **效力待定**（被代理人追认则有效，不追认则自始无效） |

#### （一）自己代理（民法典第168条第1款）

> **例题**：我委托你去帮我买茶叶，结果你把你们家自己种的茶叶卖给了我。
>
> **分析**：你既是代理人又是出卖人（相对人），一身二职。你跟我（被代理人）有利益冲突——我想越便宜越好，你想越贵越好。你容易诱发道德风险。
>
> **法律效果**：**相对无效**。被代理人（我）同意追认→合同有效；不同意或不追认→合同无效。

#### （二）双方代理（民法典第168条第2款）

> **例题**：我委托你帮我买茶叶，李家委托你帮他卖茶叶。你左手代理我，右手代理李家，自己跟自己签合同。
>
> **分析**：你一手托两家，一仆二主。我跟李家有利益冲突——我是买方想便宜，李家是卖方想贵。
>
> **法律效果**：**相对无效**。我与李家均同意→有效；否则无效。

#### （三）恶意代理（民法典第164条）——考试重点★★★★★

> **例题**：潘某委托仲某找买家卖红木。刘某出价600万，辛某出价400万。辛某找到仲某表示愿意给50万回扣，出价350万购入。仲某同意，并告知潘某“市场状况欠佳，只能卖这个价”，与辛某签订了合同。
>
> **分析**：仲某吃里扒外、私拿回扣——**恶意代理**。
>
> **法律效果**：
> - 合同**效力待定**（被代理人潘某选择追认或不追认）
> - 如果潘某**追认**→合同自始有效
> - 如果潘某**不追认**→合同自始无效，损失由代理人和相对人承担**连带责任**

> **注意**：恶意代理（民法典第164条）与恶意串通（民法典第154条）的区别：
> - **恶意代理**（164条）：损害的是**合同当事人**（被代理人）的利益 → **效力待定**
> - **恶意串通**（154条）：损害的是**合同以外的第三人**利益（有外部性） → **无效**


### 第四段：复代理（转委托）——民法典第169条

复代理就是**两次代理**（本代理 + 复代理）。

> **例题**：孟某委托马某去杭州买龙井茶。马某到杭州后水土不服生病，无法亲自处理，为了孟某的利益，**以自己的名义**转委托给曹某。曹某以孟某名义与茶农徐某签订合同（但买成了**一级茶**而非**特级茶**，且曹某胁迫徐某低价卖茶，造成徐某损失2万元）。
>
> **人物关系**：孟某（被代理人）← 本代理 → 马某（代理人）← 复代理 → 曹某（复代理人）→ 徐某（相对人）
>
> **分析**：
> - 马某**有权**转委托（紧急情况下为被代理人利益，可先斩后奏）
> - 曹某以**孟某的名义**签合同 → 合同当事人是**孟某和徐某**
> - 孟某对外承担完责任后，可向马某和曹某**追偿**（二人有过错，承担连带责任）


### 第五段：职务代理（民法典第170条，主观题★★★）

**职务代理**又叫商事代理。被代理人是**法人或非法人组织**（公司）。

**特征**：
1. **被代理人是公司**（商事代理，概括授权，不是一事一授权）
2. **代理人是员工**（执行工作任务的人员）
3. **代理事项必须是本职工作范围内的事项**（**职务代理一定是有权代理**！）

**内部权限限制不得对抗善意相对人**（民法典第170条第2款）——公司内部对员工职权的限制，客户不知道，不能用来对抗善意的客户。

> **例题**：众森公司市场部内部规定：总监权限2000万，副总监1000万，普通员工100万。新录用的员工小孟签了一个**500万**的合同（超过100万权限）。
>
> **问**：该合同是否有效？
>
> **答**：**有效**。内部权限限制**不得对抗善意相对人**（客户不知道内部规定）。公司承担后果后，可依据劳动法处罚员工。


### 第六段：狭义无权代理 vs 表见代理（核心考点★★★★★）

无权代理的三种情况（民法典第171条）：
1. **没有代理权**（根本不是员工或从未获得授权）
2. **超越代理权**（让买茶叶，结果买苹果——**标的物变了**！）
3. **代理权终止后**（被开除了、离职了还签合同）

**无权代理 vs 表见代理的本质区别**：

| 对比项 | 狭义无权代理 | 表见代理 |
| :--- | :--- | :--- |
| **本质** | 无权代理 | 无权代理（表见代理也是无权代理） |
| **保护对象** | **保护被代理人** | **保护相对人** |
| **相对人是否有理由相信** | ❌ 没有 | ✅ **有** |
| **法律效果** | **效力待定**（被代理人追认才有效） | **有效**（对被代理人发生效力） |
| **相对人要求** | 善意即可 | **善意+无过失**（要求更高） |

#### （一）狭义无权代理（民法典第171条）

> **肖经理卖烟案**：肖某系甲公司经理，被开除后伪造公章，以甲公司名义与不知情的乙公司签订烟草买卖合同（价款50万）。乙公司得知后被询问甲公司是否认可，甲公司通知乙公司**不承认**该合同。
>
> **分析**：
> - 合同**成立**（意思表示合意）
> - 效力**待定**（被代理人甲公司决定是否追认）
> - 甲公司不追认 → 对甲公司**不发生效力**
> - 乙公司（善意相对人）可请求**肖某**：①履行合同（要烟）或②赔偿损失（要钱）——**择一选择**

#### （二）表见代理（民法典第172条）

**表见代理 = 无权代理 + 相对人有理由相信行为人有代理权**

**表见代理的构成要件**：
- 主观上：相对人**善意且无过失**
- 客观上：有**权利外观**（四选一）：
  1. **授权委托书**（有公司签章）——最重要的！
  2. **介绍信**（有公司公章）
  3. **交易习惯**（题目中必须有“经常、往常、通常、常常、长期”等词）
  4. **签约地/营业地**（在公司的办公场所签约）

**授权委托书五个必备事项**（民法典第165条）：
1. 代理人的姓名或名称
2. 代理事项
3. 权限
4. 期限
5. 被代理人的签章

> **注意**：表见代理是**例外**，无权代理是**常态**。客观题一般选无权代理，主观题才可能考表见代理。


### 第七段：四个代理制度的总结

| 制度 | 性质 | 行为人身份 | 法律效果 |
| :--- | :--- | :--- | :--- |
| **恶意代理** | **有权代理** | 现员工（吃里扒外） | 效力待定（被代理人决定） |
| **职务代理** | **有权代理** | 现员工（本职工作） | **有效** |
| **狭义无权代理** | **无权代理** | 前员工（无理由相信） | 效力待定 |
| **表见代理** | **无权代理** | 前员工（有理由相信） | **有效**（相对人善意+无过失） |

**表见代表 vs 表见代理**：一字之差，差之千里——看**身份**！是**法定代表人**→表见代表；是**非法定代表人**→表见代理。法律效果相同：合同**有效**，由公司承担。


### 第八段：异常人章关系（民法典合同编通则司法解释第22条）

实践当中商人们钻法律空子：盖假章、不盖章、有章无人、有人无章——法院采**看人不看章**原则。

| 情形 | 规则 | 考情 |
| :--- | :--- | :--- |
| **真人假章**（人真章假） | 合同**成立有效**（看人不看章） | **2024年考过** |
| **有人无章**（人签名了，没盖章） | 合同**成立有效**（除非约定必须盖章才成立） | **从未考过** |
| **有章无人**（盖了真章，没人签名） | 相对人证明是在权限内订立的→有效 | 结合民诉（谁主张谁举证） |

> **真人假章例题**：甲公司与乙公司订立钢材购销合同，双方法定代表人刘某和谢某分别代表各自公司签字盖章。后乙公司发现所盖甲公司公章系刘某**伪造**。
>
> **问**：合同效力如何？
>
> **答**：**合同有效**。看人不看章——人是真的，章是假的没关系。

> **有章无人**（从未考过）：合同只盖了公司真章，没有人签名。相对人**举证证明**签约人是在权限范围内订立的，法院应认定合同有效。


### 第九段：本专题总结

代理就是**代替他人签合同**。

逻辑链条：
1. **代理的适用范围** → 民事法律行为（合同、诉讼）
2. **代理权的来源** → 授权委托书（不是委托合同！）
3. **代理权滥用**（有权代理） → 自己代理、双方代理、恶意代理（重点）
4. **复代理**（转委托） → 紧急情况下可为被代理人利益转委托
5. **职务代理**（商事代理） → 本职工作范围内 → **有权代理，有效**
6. **无权代理** → 超越本职工作 → 效力待定（保护被代理人）
7. **表见代理** → 无权代理 + 相对人有理由相信 → **有效**（保护相对人）
8. **异常人章关系** → 看人不看章
