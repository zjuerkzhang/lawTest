## 第1部分：对照《民法典》民事权利提取备考提纲


### 一、民事权利的法定分类（民法典内容分类）

| 类型 | 包含内容 |
| :--- | :--- |
| **财产权** | 物权、债权 |
| **人身权** | 人格权、身份权 |
| **综合性权利** | 知识产权、继承权、股权 |


### 二、民事权利的学理分类（按作用分类）

#### （一）支配权（对世权/绝对权）

- **含义**：对权利客体直接支配并排除他人干涉
- **特征**：权利人特定，义务人不特定
- **典型**：物权、人身权、知识产权

#### （二）形成权（变动权）

- **含义**：一单方意思表示使法律关系产生/变更/消灭（一个人说了算）
- **产生**：可法定，可约定
- **行使**：可明示，可默示（“视为”）
- **分类**：简单形成权（通知即可，如解除权）；形成诉权（或诉或裁，如撤销权）
- **期间**：受**除斥期间**限制
- **典型**：撤销权、解除权、追认权、拒绝权

#### （三）请求权

- **含义**：特定权利人请求特定义务人为或不为特定行为
- **特征**：相对性（权利人和义务人均特定）；非公示性
- **典型**：债权请求权、物权请求权

#### （四）抗辩权

- **含义**：对抗请求权的防御性权利
- **分类**：一时性抗辩权（暂时拒绝）vs 永久性抗辩权（永远拒绝）
- **典型**：诉讼时效抗辩权（永久）、同时履行抗辩权（一时）、先诉抗辩权（一时）


### 三、物权 vs 债权对比（财产权两大支柱）

| 对比项 | 物权 | 债权 |
| :--- | :--- | :--- |
| **定义** | 对特定物直接支配并排除他人干涉 | 特定人请求特定人为或不为特定行为 |
| **效力范围** | 对世权/绝对权 | 相对权/对人权 |
| **客体** | 物（例外：权利） | 行为 |
| **权利性质** | 支配权 | 请求权 |
| **公示性** | 需要（登记/交付） | 非公示性 |


### 四、诉讼时效 vs 除斥期间

| 对比项 | 诉讼时效 | 除斥期间 |
| :--- | :--- | :--- |
| **适用权利** | 请求权 | 形成权 |
| **法律效果** | 债务人取得抗辩权 | 实体权利消灭 |
| **可变性** | 可变（中止、中断） | 不可变 |
| **法院主动适用** | 不能 | 可以审查 |
| **产生依据** | 一律法定 | 可法定、可约定 |

> - “支配权的特征——**权利人特定，义务人不特定**（对世权/绝对权）。”

> - “权利的限制——**权利不是绝对的，要受国家法律、社会公共利益、他人权利、诚信原则和时间的限制**。”

### 第一段：民事权利法定分类概述

好，最后咱们来看专题四十九，民事权利。这个专题客观题中考，主观题不会单独考。民事权利重点考察**法定分类、学理分类**以及**民事权利的行使和限制**。

先来看四百七十四页下方民事权利的法定分类。法定分类就是民法典直接采用的分类方式。

民法是一个**私法**、**实体法**、**权利法**、**市民社会的法**、**市场经济的法**。民事权利是指民事主体为实现某种利益而依法为或不为某种行为的自由。

民事权利按照**内容的不同**，分成了三类：**财产权、人身权和综合性权利**。


### 第二段：民事权利的学理分类——支配权

四百七十五页，第二是民事权利的学理分类。学理上按照**民事权利的作用**的不同，分成了四类：**支配权、形成权、请求权和抗辩权**。最为专业拗口的就是**形成权**（撤销权和解除权）。

**支配权**（又称管理权）：对权利的客体直接支配并排除他人干涉的权利。

典型的支配权就三个：
1. **物权**（面前的物可以支配）
2. **人身权**（自己的人身可以支配）
3. **知识产权**（自己的智力成果可以支配）

支配权的特征：**对世性**（权利人特定，义务人不特定）。物权、人身权、知识产权都属于**对世权**、**绝对权**。


### 第三段：形成权（核心★★★★★）

**形成权**（又称变动权、能为权）：权利人**一单方意思表示**而使民事法律关系**产生、变更或消灭**的权利。

**通俗而言：一个人说了算。**

#### （一）形成权的产生

| 产生方式 | 典型例子 |
| :--- | :--- |
| **当事人约定** | **合同解除权**（如买卖合同约定解除条件） |
| **法律规定** | **可撤销民事法律行为的撤销权**（重大误解、欺诈、胁迫、显失公平，民法典第147-152条） |

> **形成权例题（约定）** ：孟某将手表卖给马某，约定如孟某不履行合同，马某有权解除合同并追究违约责任。马某享有的解除权属于典型的**形成权**（基于买卖合同约定产生）。
>
> **形成权例题（法定）** ：孟某将欧米茄手表冒充江诗丹顿卖给马某，因欺诈行为，法律赋予马某**撤销权**，该撤销权属于典型的形成权（基于法律规定产生）。

#### （二）形成权的行使方式

- **明示** 或 **默示**。默示的重点就是“**视为**”二字（法律拟制，不能用证据推翻）。

> **默示行使形成权例题**：
> - 限制民事行为能力人（15岁）花10万买钢琴，合同效力待定。法定代理人收到催告后30日不作表示——**视为拒绝追认**（民法典第145条）。
> - 无权代理合同，被代理人收到催告后30日不作表示——**视为拒绝追认**（民法典第171条）。
> - 受遗赠人60日内不作表示——**视为放弃受遗赠**。

#### （三）简单形成权 vs 形成诉权

| 类型 | 行权方式 | 是否需要专业人员判断 | 典型例子 |
| :--- | :--- | :--- | :--- |
| **简单形成权** | **或诉或裁或通知**（通知即可） | 不需要（事实判断） | **合同解除权** |
| **形成诉权** | **或诉或裁**（必须诉讼/仲裁，不能通知） | 需要（价值判断） | **可撤销民事法律行为的撤销权** |

> **形成诉权例题**：重大误解、欺诈、胁迫、显失公平——都得去法院或仲裁机构，**不能通知**（张大爷理解的欺诈和法官理解的欺诈不一样）。

#### （四）形成权的除斥期间

形成权受**除斥期间**限制——**有权不用，过期作废**（直接剥夺实体权利）。

**除斥期间可以法定，也可以约定**（与诉讼时效“一律法定”不同）。

> **可约定除斥期间**：合同解除权的行使期间可由当事人约定（如买卖合同约定“违约后三个月内可解除”）；保证期间也可由当事人约定。


### 第四段：请求权

**请求权**：特定的权利人请求特定的义务人为或不为特定行为的权利。说的就是**债权**。

**特征**：
1. **相对性**（权利人和义务人均特定）——又称**相对权**、**对人权**
2. **非公示性**（不需要公开，具有可伪造性）

**请求权的发生**：
- **基础权利受侵害发生**（侵权之债）
- **基础权利不受侵害发生**（合同之债）


### 第五段：物权 vs 债权（财产权两大支柱对比★★★★★）

| 对比项 | 物权 | 债权 |
| :--- | :--- | :--- |
| **本质** | 静态财产权（调整归属和利用） | 动态财产权（调整流转） |
| **含义** | 对特定物直接支配并排除他人干涉 | 特定人请求特定人为或不为特定行为 |
| **效力范围** | 权利人特定，义务人不特定（**对世权/绝对权**） | 权利人和义务人均特定（**相对权/对人权**） |
| **客体** | 特定化的物（例外：**权利**，如权利质权） | **行为**（积极作为 + 消极不作为，如竞业禁止） |
| **权利性质** | **支配权** | **请求权** |
| **效力** | 追及效力、排他效力、优先效力 | 相容性、平等性、相对性 |
| **是否需公示** | **原则需要**（登记/交付） | **非公示性**（可伪造） |
| **立法模式** | **物权法定**（但留置权可约定排除） | **债权自由**（但受公序良俗限制） |
| **全能** | 占有、使用、收益、处分 | 请求力、执行力、处分力等 |
| **特点** | 具有民族性（各国不同） | 具有国际性（国际贸易规则通用） |


### 第六段：抗辩权

**抗辩权**：一时或永久对抗对方请求权的权利（**请求权是矛，抗辩权是盾**）。

| 类型 | 含义 | 典型例子 |
| :--- | :--- | :--- |
| **一时性抗辩权**（延期/延缓抗辩权） | 暂时拒绝，对方履行后仍需履行 | 同时履行抗辩权、先履行抗辩权、不安抗辩权、一般保证人先诉抗辩权、赠与人的穷困抗辩权 |
| **永久性抗辩权** | 永远拒绝对方请求权 | **诉讼时效抗辩权** |

**核心特征**：抗辩权行使与否**完全由当事人决定**，法院不得主动适用（民法典第193条）。


### 第七段：民事权利的行使与限制

四百七十九页，最后来看民事权利的行使和限制。

**行使原则**：
1. **自愿**——但**自己行为自己责任**（签了合同就得认）
2. **权利与义务相统一**（整体意义上；个别合同如赠与合同是例外）

**行使方法**：
- **事实行为**（消费掉旺仔牛奶，无需意思表示）
- **民事法律行为**（把旺仔牛奶送给你，需要意思表示）

**权利的限制**（五种方式）：

| 限制类型 | 举例 |
| :--- | :--- |
| **国家法律的限制** | 土地所有权不得转让；宅基地使用权不得抵押 |
| **社会公共利益的限制** | 政府征收（为了公共利益） |
| **他人权利的限制** | 相邻关系（不能建五米高墙让邻居如坐牢） |
| **诚信原则（不得权利滥用）** | 以损害他人为主要目的行使权利 |
| **时间期间的限制** | 诉讼时效 vs 除斥期间 |


### 第八段：诉讼时效 vs 除斥期间（对比★★★★★）

| 对比项 | 诉讼时效 | 除斥期间 |
| :--- | :--- | :--- |
| **适用权利** | **请求权**（债权请求权、物权请求权的例外） | **形成权**（撤销权、解除权） |
| **期间性质** | 原则**可变**（可中止、中断） | **不变**（固定期间） |
| **法律效果** | 债务人取得**永久性抗辩权**（权利不消灭） | **实体权利消灭**（有权不用过期作废） |
| **是否可变期间** | 除最长20年外，均**可变** | **不可变** |
| **法院能否主动适用** | **不能**（民法典第193条） | **可以主动审查**（如保证期间是否届满） |
| **起算点** | 原则**主观**（知道或应当知道权利受损及义务人之日）；最长20年**客观**（权利受损之日） | 除法律规定外，原则上自权利人**知道或应当知道权利产生之日**起算 |
| **产生依据** | **一律法定**（不可约定） | **可以法定，也可以约定**（如保证期间、解除权行使期间） |


### 第九段：民法学体系框架图（全书总结）

翻到目录后边的《民法学体系框架图》：

**民法本质** = **民事权益** = 权利 + 利益（胎儿利益、死者人格利益）

**权利按内容分类**：

| 类别 | 内容 |
| :--- | :--- |
| **财产权** | 物权（自物权/他物权/准物权）+ 债权（约定之债/法定之债/自然之债） |
| **人身权** | 人格权（一般人格权+具体人格权）+ 身份权（配偶权/亲子权/亲属权） |
| **综合性权利** | 知识产权、继承权、股权（兼具人身性和财产性） |

**债权发生原因**：
- **约定之债**：单方允诺之债 + 合同之债
- **法定之债**：缔约过失之债 + 无因管理之债 + 不当得利之债 + 侵权之债
- **自然之债**：已过诉讼时效之债、赌债、彩礼、限定继承超出遗产范围之债

**人身权**：
- **一般人格权**：人身自由权 + 人格尊严权（祭奠权属于人格尊严权）
- **具体人格权**：生命权、身体权、健康权、姓名权、肖像权、名誉权、荣誉权、隐私权
- **身份权**：配偶权、亲子权、亲属权
