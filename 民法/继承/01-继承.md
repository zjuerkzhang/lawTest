
## 第1-1部分：对照《民法典》继承编提取备考提纲


### 一、继承权概述（民法典第1121条、第1122条）

| 概念 | 内容 |
| :--- | :--- |
| **继承人** | 只能是**自然人** |
| **被继承人** | 只能是**自然人** |
| **受遗赠人** | 可以是**法人、非法人组织** |
| **继承开始时间** | 被继承人**死亡**时（包括宣告死亡） |


### 二、死亡时间的推定（民法典第1121条第2款）

相互有继承关系的数人在同一事件中死亡，难以确定死亡时间的：
1. **推定没有其他继承人的人先死亡**
2. 都有其他继承人的，**推定长辈先死亡**（晚辈后死亡）
3. 辈份相同的，**推定同时死亡，相互不发生继承**

**核心原理：防止遗产归公**。


### 三、遗产的范围（民法典第1122条）

遗产是自然人死亡时遗留的**个人合法财产**。

**不属于遗产的情形**：
- **死亡赔偿金**（死亡后才产生，非死亡时遗留）
- **死者人格利益保护的精神损害赔偿金**（保护近亲属，非遗产）
- **著作权中的人身性权利**（署名权等，不能继承）
- **夫妻共同财产中属于配偶的部分**（须先析产）

**可以继承的情形**：
- **股权**（股东资格可以继承，章程另有规定的除外；股东不受行为能力限制）


### 四、继承权的放弃（民法典第1124条）

| 要点 | 规则 |
| :--- | :--- |
| **时间** | 继承开始后、遗产分割前 |
| **死亡前放弃** | ❌ **无效** |
| **遗产分割后放弃** | ❌ 放弃的是**所有权**，不是继承权 |
| **方式** | **书面形式**（明示，不能默示） |
| **反悔（分割前）** | 由**法院**决定是否承认 |
| **反悔（分割后）** | ❌ **不予承认** |
| **放弃无效的情形** | 导致放弃方不能履行**法定扶养义务**的，放弃行为**无效**（民法典婚姻家庭编司法解释二第11条，**新增，从未考过**） |


### 五、继承权的丧失（民法典第1125条）

#### （一）绝对丧失（不可宽恕）

1. **故意杀害被继承人**（不论既遂未遂、不论目的）
2. **为争夺遗产而杀害其他继承人**（须有“为争夺遗产”的目的）

#### （二）相对丧失（可宽恕）

1. 遗弃被继承人或虐待被继承人情节严重
2. 伪造、篡改、隐匿或销毁遗嘱情节严重
3. 以欺诈、胁迫手段迫使或妨害被继承人设立、变更或撤回遗嘱情节严重

**恢复条件**：被继承人表示**宽恕**，或者事后在遗嘱中将其列为继承人。

## 第1-2部分：对照《民法典》继承编提取备考提纲


### 一、法定继承（民法典第1127条、第1129条、第1152条）

#### （一）法定继承的适用情形（第1154条）

重点是第3项：**遗嘱继承人、受遗赠人先于遗嘱人死亡或终止** → 遗嘱不生效 → 按法定继承。

#### （二）继承顺序

| 顺序 | 继承人 | 法典依据 |
| :--- | :--- | :--- |
| **第一顺序** | 配偶、父母、子女（子女包括：婚生、非婚生、养子女、有抚养关系的继子女、胎儿） | 第1127条 |
| **第一顺序拟制** | ①丧偶儿媳对公婆/丧偶女婿对岳父母**尽了主要赡养义务**；②**代位继承**（孙子女外孙子女） | 第1129条、第1152条 |
| **第二顺序** | 兄弟姐妹、祖父母、外祖父母 | 第1127条 |


### 三、遗产酌给制度（民法典第1131条）

继承人以外的**依靠被继承人扶养的人**，或者**对被继承人扶养较多的人**，可以分给适当的遗产。

**三种常考情形**：
1. 孤儿或弃婴（未办收养登记）
2. 侄子侄女/外甥外甥女对无儿无女的伯伯叔叔姑姑舅舅尽了赡养义务
3. 养子女对生父母尽了赡养义务


### 四、遗嘱继承（民法典第1133-1144条）

#### （一）遗嘱形式（第1134-1139条）

| 形式 | 核心要件 |
| :--- | :--- |
| 公证遗嘱 | 公证机构办理 |
| 自书遗嘱 | **写、签、注**（亲笔书写+签名+年月日） |
| 代书遗嘱 | 两个以上合格见证人+一人代书+签名+年月日 |
| 打印遗嘱 | 两个以上合格见证人+**每一页签名**+年月日（**新增，从未考过**） |
| 录音录像遗嘱 | 两个以上合格见证人+**见证人和遗嘱人出镜**+姓名肖像+年月日 |
| 口头遗嘱 | **危急情况**+两个以上合格见证人；情况消除后可书面则口头**无效** |

#### （二）遗嘱的撤回（第1142条）

- 可以撤回、变更
- 以**行为**撤回：与遗嘱内容相反的行为 → 视为撤回
- 数份遗嘱内容抵触 → **以最后的遗嘱为准**

#### （三）遗嘱无效的情形（第1143条、第1144条）

- 无/限制民事行为能力人所立遗嘱**无效**
- 受欺诈、胁迫所立遗嘱**无效**
- 伪造的遗嘱**无效**
- 篡改的**篡改内容无效**
- 处分他人财产的**该部分无效**
- **未保留必要份额**的**该部分无效**（以**死亡时**为准判断是否有“双缺”继承人）


### 五、欺诈胁迫在不同场景中的效力对比（讲稿总结）

| 行为类型 | 欺诈 | 胁迫 |
| :--- | :--- | :--- |
| 合同 | **可撤销**（第三人欺诈一分为二） | 可撤销 |
| 婚姻 | 原则有效；**重大疾病婚前未告知**→可撤销 | 可撤销 |
| 遗嘱 | **无效** | **无效** |

## 第1-3部分：对照《民法典》继承编提取备考提纲


### 一、代位继承（民法典第1128条）

| 类型 | 条件 | 代位继承人 |
| :--- | :--- | :--- |
| **传统型** | 被继承人的子女**先于**被继承人死亡 | 被继承人子女的**直系晚辈血亲**（孙子女外孙子女） |
| **新型** | 被继承人的兄弟姐妹**先于**被继承人死亡 | 被继承人兄弟姐妹的**子女**（侄子侄女外甥外甥女） |

**关键限制**：代位继承只能发生在**法定继承**中。

**血亲的理解**：包括自然血亲（生子女）和拟制血亲（**养子女**），不包括有抚养关系的**继子女**。

**立法目的**：防止遗产归公（国不与民争利）。


### 二、转继承（民法典第1152条）

**条件**：继承开始后，**继承人于遗产分割前死亡**，并没有放弃继承的，该继承人应当继承的遗产**转给其继承人**。

**本质**：**两次继承**（被继承人→继承人→转继承人）。

**通俗**：**死的快**（继承开始后、遗产分割前死亡）→ 转继承。


### 三、代位继承 vs 转继承对比

| 对比项 | 代位继承 | 转继承 |
| :--- | :--- | :--- |
| **死亡时间** | 继承人**先于**被继承人死亡（**死的早**） | 继承开始后、遗产分割前死亡（**死的快**） |
| **继承次数** | **一次继承** | **两次继承** |
| **适用范围** | **只能**法定继承 | **无限制**（法定/遗嘱均可） |


### 四、遗产管理人（民法典第1145-1149条）

| 情形 | 遗产管理人 |
| :--- | :--- |
| 有遗嘱执行人 | 遗嘱执行人 |
| 无遗嘱执行人 | 继承人**推选** |
| 推选不出 | 全体继承人**共同担任** |
| 无继承人/继承人均放弃 | 市民→**民政部门**；村民→**村民委员会** |

**责任**：故意或重大过失造成损失 → 赔偿（遗产管理人可有偿可无偿，原则无偿）。


### 五、遗嘱 vs 遗赠（民法典第1133条、第1124条）

| 对比项 | 遗嘱 | 遗赠 |
| :--- | :--- | :--- |
| **对象** | **法定继承人** | **法定继承人以外的人** |
| **沉默后果** | **视为接受** | **视为放弃**（60天内不作表示） |
| **性质** | 单方民事法律行为 | 单方民事法律行为 |


### 六、遗赠扶养协议（民法典第1158条）

**性质**：**唯一双方民事法律行为**（协议=合同）。

**本质**：**生养死葬**。

**效力位阶**：**遗赠扶养协议 > 遗嘱/遗赠 > 法定继承**。

**主体限制**：遗赠人与抚养人之间**不得存在法定的扶养义务关系**。


### 七、限定继承与债务清偿（民法典第1161条、第1162条、第1163条）

**限定继承**：继承人仅在遗产继承范围内承担被继承人的债务，超出部分为**自然之债**。

**清偿顺序**：
1. **法定继承人**（先用法定继承部分偿还）
2. **遗嘱继承人和受遗赠人**（按比例）
3. **遗赠扶养协议的抚养人**（不用还——已履行生养死葬义务）

> - “继承人和被继承人都只能是**自然人**，法人只能是受遗赠人。”

> - “死亡赔偿金、死者人格利益精神损害赔偿金——**都不是遗产**。”
> - “股权——**可以继承**，股东不受行为能力限制。”
> - “放弃继承权——**死亡前无效，遗产分割后放弃的是所有权**，必须是**书面形式**。”
> - “绝对丧失——**就两个点**：故意杀害被继承人；为争夺遗产杀害其他继承人。”
> - “绝对丧失和相对丧失的区别——**杀人不可宽恕，其他可宽恕**。”

| **遗嘱 vs 遗赠——沉默后果** | 客观题 | 常考 | 考查沉默是视为接受还是放弃 | 爷爷遗嘱给孙子（孙子是遗赠受领人）→60天沉默=**放弃** |
| **遗赠扶养协议——主体限制** | 客观题 | **2025年考过** | 考查谁不能做遗赠扶养协议的抚养人 | 孙子女对爷爷奶奶有法定赡养义务时→不能签遗赠扶养协议 |
| **遗赠扶养协议——效力位阶** | 客观题 | 常考 | 考查遗赠扶养协议 vs 遗嘱的效力优先顺序 | 遗赠扶养协议 **>** 遗嘱/遗赠 **>** 法定继承 |
| **遗产管理人——确定** | 客观题 | 已考过 | 考查无人继承时谁当遗产管理人 | 市民无人继承→**民政部门**；村民→**村民委员会** |
| **限定继承——债务清偿顺序** | 客观题 | 经典考点 | 给出“法定继承+遗嘱继承+债务”的计算场景 | 法定继承人先还→遗嘱继承人按比例还→不足部分为自然之债 |
| **遗赠扶养协议的抚养人——不用还债** | 客观题 | 常考 | 考查抚养人是否承担被继承人的债务 | 遗赠扶养协议的抚养人**不用**替被继承人还债（已履行生养死葬义务） |

> **老师原话总结**：
> - “代位继承和转继承——**死的早（代位）vs 死的快（转继承）**。代位继承是一次继承，转继承是两次继承。”
> - “代位继承**只能发生在法定继承中**；转继承无限制。”
> - “遗嘱 vs 遗赠——**沉默后果不同**：遗嘱继承人沉默=**接受**；受遗赠人沉默=**放弃**。”
> - “遗赠扶养协议——**唯一双方民事法律行为，效力最优，生养死葬**。”
> - “限定继承——**遗产范围内还债，超出部分自然之债**。法定继承人先还，遗嘱继承人按比例还，遗赠扶养协议的抚养人**不用还**。”

### 第一段：继承编概述

好，接下来咱们来看第七部分，叫做继承编。继承这个部分基本上就是在客观题中进行考察，主观题一般就不涉及到继承编了。客观题当中每年几乎都是固定的，一道两分的多选题，咱们把它搞定就行了。

反过来我们来看书的四百零九页。专题四十三继承，先来看它的基本原理。啥叫继承啊？继承是指自然人死亡以后，由法律规定的一定范围内的人或遗嘱指定人依法取得死者遗留的个人合法财产的法律制度（民法典第1121条）。

这里边必须要强调一个点：**继承人和被继承人都只能是自然人**。法人、非法人组织最多只能作为**受遗赠人**。比如说老孟说：“在我死后，我的欧米茄手表归中国政法大学。”中国政法大学不能被定为继承人，它叫受遗赠人。


### 第二段：继承发生的时间——死亡推定（核心考点）

第一个列入到考试考点的，就是四百零九页下方的第二个问题——**时间**。继承在什么时候才会发生？就是被继承人死亡的时候。

在法律上，“死亡”应当做扩大解释，一方面可能是自然死亡、生理死亡，另一方面考试往往会考**宣告死亡**（民法典第1121条第2款）。宣告死亡以后，和自然死亡一样，“妻离子散，家破人亡”——其中的“家破”，就是个人合法财产开始发生继承。

**比较复杂的问题**：在同一事故中，相互有继承关系的几个人都死了，又无法确定谁先死谁后死，法律就要进行推定。

翻过来四百一十页，就三种情况：

**第一种情况：事故以外都没有其他继承人**

比如张三、李四夫妻带着唯一的孩子小张，三口之家在这个世界上就他们三个人，结果出去玩的时候全死了。事故以外每一个人都没有其他继承人，死亡先后顺序无所谓，因为最终的结局是一样的——这个财产如果是市民的归国家用于公益事业，如果是村民的归集体。这就是**无人继承又无人受遗赠的遗产**。

**第二种情况：事故以外都有其他继承人**

比如张三、李四夫妻带着孩子小张去旅游，张三的父母和李四的父母都健在。那都有继承人的话，按照**长辈先死，晚辈后死；同辈同时死，互相不继承**的规则推定。

那为什么张三和李四之间同辈同时死，互相不继承呢？最终不都是归小张吗？为了简化法律关系，同辈同时死，互相不继承。

**第三种情况（重点，法考以来从未考过）** ：事故以外有的有其他继承人，有的没有其他继承人。

比如张三、李四夫妻生一子小张，张三的哥哥活着，李四的姐姐活着（张三和李四有继承人），但小张没有其他继承人。

**规则**：**推定没有其他继承人的人先死亡**——即小张先死，然后张三、李四再死。张三、李四死亡后，由他们的继承人（张三的哥哥、李四的姐姐）继承。

为什么这样推定？如果按正常情况推定张三、李四先死，小张再死，小张没有继承人，最终遗产就归公了。所以反着推定，没有其他继承人先死亡，**防止遗产归公**。

> **死亡推定总结**：
> - 情况一：都没有其他继承人 → 谁先死无所谓（最终归公）
> - 情况二：都有其他继承人 → 长辈先死、晚辈后死，同辈同时死、互不继承
> - 情况三（重点）：有的有其他继承人、有的没有 → **推定没有其他继承人先死亡**（防止遗产归公）


### 第三段：遗产的范围（核心考点）

遗产是自然人死亡时遗留的**个人合法财产**（民法典第1122条）。几个要点：

**1. 自然人**：被继承人和继承人都只能是自然人。

**2. 死亡时（重点）** ：死了以后才有的，不叫遗产。
- **死亡赔偿金**：张三开车把李四撞死，赔给李四的死亡赔偿金一百万——**不是遗产**（李四死之前没有这一百万）。虽然参照遗产处置，但性质上不是遗产（民法典第1122条）。
- **死者人格利益保护的精神损害赔偿金**：张三死后，李四造谣，张三的近亲属起诉获得精神损害赔偿金十万——**不是遗产**（保护的是近亲属的情感利益，谁起诉归谁，不能按遗产处置）。

**3. 个人**：如果死者有配偶，要先**析产**。张三李四是夫妻，张三死了，张三家的全部财产不是都算遗产——得先把一半分给李四，剩下的一半才是张三的遗产（民法典第1153条）。

> **析产例题**：张三和李四夫妻共同财产1000万，张三死亡。遗产是多少？
>
> **答**：500万（先分出500万给李四，剩余500万为张三的遗产）。

**4. 合法**：赌债不能当遗产。

**5. 财产（结合知识产权和公司法，重要）** ：
- **著作权中的署名权等人身性权利**——**不能继承**（民法典第1122条）。
- **股权**——股东资格**可以继承**（公司章程另有规定的除外）。股东不受行为能力的限制，无民事行为能力人、限制民事行为能力人也可以当股东。


### 第四段：继承权的放弃

四百一十一页上方，**继承权的放弃**是指继承人在继承开始后、遗产处理前，以**明示的方式**作出的拒绝接受被继承遗产的意思表示（民法典第1124条）。

#### （一）放弃的时间

分为三个阶段（**两限三区**）：

| 阶段 | 放弃行为效力 |
| :--- | :--- |
| **死亡前** | ❌ **无效**（死亡前你还真不一定有继承权） |
| **继承开始后、遗产分割前** | ✅ **可以放弃**（向遗产管理人、其他继承人或法院作出，须书面形式） |
| **遗产分割后** | ❌ 放弃的不再是继承权，而是**遗产的所有权**（已分完了） |

#### （二）放弃的方式

必须采用**书面形式**（两个法理逻辑：①要慎重；②要证据）。

#### （三）放弃的效力

- **有溯及力**：溯及到继承开始时。
- **导致不能履行法定义务的放弃无效**（民法典婚姻家庭编司法解释二第11条，**新增，从未考过**）：如果放弃继承权导致放弃方不能履行法定扶养义务（如对配偶的抚养义务），该放弃行为无效。

> **放弃继承权无效例题**：张三和李四夫妻正在闹离婚，张三的父亲去世，张三放弃继承父亲遗产，导致无法履行对李四的抚养义务（李四缺乏劳动能力、无生活来源）。这个放弃行为**无效**。

#### （四）反悔

- **遗产分割前反悔**：由**法院**决定是否承认。
- **遗产分割后反悔**：**不予承认**（已是财产所有权处置问题）。


### 第五段：继承权的丧失（民法典第1125条）

继承权的丧失是**被动**的——因实施**违法行为**被依法剥夺。

#### （一）绝对丧失（不可宽恕，重点★★★）

| 行为 | 法典依据 | 说明 |
| :--- | :--- | :--- |
| **故意杀害被继承人** | 第1125条第1款第1项 | 不论既遂未遂，不论目的，**绝对丧失** |
| **为争夺遗产而杀害其他继承人** | 第1125条第1款第2项 | **必须是为了争夺遗产**（目的要件） |

**区别**：
- 杀被继承人：不论什么目的，只要故意杀害，就绝对丧失。
- 杀其他继承人：必须**为了争夺遗产**。如果因为其他矛盾杀害，不丧失继承权。

#### （二）相对丧失（可宽恕）

| 行为 | 法典依据 | 说明 |
| :--- | :--- | :--- |
| 遗弃被继承人或虐待被继承人情节严重 | 第1125条第1款第3项 | 被继承人可以宽恕 |
| 伪造、篡改、隐匿或销毁遗嘱情节严重 | 第1125条第1款第4项 | 被继承人可以宽恕 |
| 以欺诈、胁迫手段迫使或妨害被继承人设立、变更或撤回遗嘱情节严重 | 第1125条第1款第5项 | 被继承人可以宽恕 |

**相对丧失的恢复**：被继承人表示宽恕或者事后在遗嘱中将其列为继承人的，不丧失继承权（民法典第1125条第2款）。

> **老师原话**：“考试重点考绝对丧失——杀人了。考绝对丧失就两个点：故意杀害被继承人；为争夺遗产杀害其他继承人。”


### 第六段：本部分总结

继承编基本原理部分：
1. **继承发生的时间**：死亡（自然死亡/宣告死亡）→ 死亡推定（三种情况）
2. **遗产的范围**：死亡时遗留的个人合法财产（死亡赔偿金不是遗产；精神损害赔偿金不是遗产；要先析产）
3. **继承权的放弃**：死亡前无效；遗产分割前可放弃（书面）；遗产分割后放弃的是所有权
4. **继承权的丧失**：绝对丧失（杀人）不可宽恕；相对丧失（其他）可宽恕

### 第一段：法定继承概述

好，接下来咱们来看第三个问题，继承方式之法定继承。

继承方式就两种：一种叫法定继承，一种叫遗嘱继承。先来看啥叫法定继承。书的四百一十二页的下方。法定继承是指在没有遗赠扶养协议和遗嘱（或遗赠扶养协议和遗嘱无效）的情况下，继承人依据法律确定的继承人范围、继承顺序以及遗产分配的原则，取得被继承人遗产的继承方式。它的本质就是**依据法律**。【修正：原文“遗赠抚养协议”应为“遗赠扶养协议”】

四百一十三页上方，法定继承的情形一共五种，重点是**第三种**：

| 序号 | 情形 | 说明 |
| :--- | :--- | :--- |
| 1 | 遗嘱继承人放弃继承或受遗赠人放弃受遗赠 | 主动放弃 |
| 2 | 遗嘱继承人丧失继承权或受遗赠人丧失受遗赠权 | 被动丧失 |
| **3（重点）** | **遗嘱继承人、受遗赠人先于遗嘱人死亡或终止** | 写谁谁先死 → 遗嘱不生效 → 按法定继承 |
| 4 | 遗嘱无效部分所涉及的遗产 | 遗嘱部分无效 → 该部分按法定继承 |
| 5 | 遗嘱未处分的遗产 | 没写到的部分按法定继承 |

**第三种情形例题**：张三写了一个遗嘱，说待我死后，我的全部个人遗产归我的大儿子。结果大儿子先于张三死亡。那怎么办？只能按**法定继承**处理（遗嘱没写给二儿子，也没写给其他人）。

> **老师原话**：“写谁谁先死，他爸爸张三写的不是遗嘱，写的是个死亡笔记、死亡通知书。”


### 第二段：法定继承的顺序

考试重点看**第一顺序**（民法典第1127条）。

**第一顺序法定继承人（三个）** ：**配偶、父母、子女**。

**“子女”的扩大解释**（五类）：婚生子女、非婚生子女、养子女、有抚养关系的继子女、**胎儿**。

> **注意**：这五类都叫“子女”，都得分遗产。一人一份，原则均等。

**第一顺序的拟制继承人（考试重点）** ：

1. **丧偶儿媳对公婆、丧偶女婿对岳父母，尽了主要赡养义务的**——无论是否再婚，都可以作为**第一顺序继承人**（民法典第1129条）。
> 张三死了，李四（儿媳）本无义务赡养张三的爸妈，但她尽了这个义务——就是中国好儿媳。等张三爸妈去世时，李四可以作为第一顺序继承人继承。

2. **孙子女、外孙子女的代位继承权**——爸爸妈妈先于爷爷奶奶、姥姥姥爷死亡，孙子女外孙子女代位继承（后面会重点讲代位继承）。

**第二顺序**（了解）：**兄弟姐妹、祖父母、外祖父母**。有第一顺序的，第二顺序不上场。


### 第三段：遗产分配规则

原则：均等——一人一份，按人头点。

**例外一：应当多分**

生活有特殊困难且缺乏劳动能力的继承人（**困且乏，两个条件同时满足**）——**应当予以照顾**（应当多分）。

**例外二：可以多分**

对被继承人尽了**主要扶养义务**或与被继承人**共同生活**的继承人——**可以多分**（需要法官裁量判断）。

**例外三：应当不分或少分**

有扶养能力和有扶养条件的继承人，不尽扶养义务的——**应当不分或少分**（“丧尽天良”型）。

**例外四：归公**

无人继承又无人受遗赠的遗产——市民归国家用于公益事业，村民归集体所有（民法典第1160条）。


### 第四段：遗产酌给制度（可适当分得遗产，核心考点★★★）

这部分是考试重点。**三类人可以适当分得遗产**（民法典第1131条）：

| 序号 | 情形 | 举例 |
| :--- | :--- | :--- |
| **1** | **继承人以外的、依靠被继承人扶养的人**（如孤儿或弃婴，但未办理收养登记手续） | 张大爷在马路边捡了个胖娃娃，抱回家养了五年，但没办收养登记。张大爷死了，这孩子不能当法定继承人，但可以**适当分得遗产** |
| **2** | **继承人以外的、对被继承人扶养较多的人** | 侄子侄女无儿无女的大伯（没写遗嘱），适当分得遗产 |
| **3** | **养子女对生父母尽了赡养义务的** | 小梦被收养后，发现亲生父母在捡垃圾，每月给生活费。生父母去世后，小梦可**适当分得生父母的遗产** |

> **专业术语**：这叫**遗产酌给制度**——酌情给付一部分。

> **注意**：这里说的“继承人”是指民法典第1127条规定的六个法定继承人（第一顺序三人+第二顺序三人）。侄子侄女、外甥外甥女、养子女对生父母——都不是法定继承人。


### 第五段：双重继承（再婚重组家庭）

**有抚养关系的继子女可以双重继承**——既可以继承生父母的遗产，也可以继承继父母的遗产（民法典第1072条、第1127条）。

> **双重继承例题**：张三和李四离婚，女儿小小张10周岁跟张三生活，后张三与王五结婚。小张（20周岁）与王五**无抚养关系**，互相无继承权；小小张（10周岁）与王五**有抚养关系**，可以双重继承——既能继承生母李四的遗产，也能继承继母王五的遗产。


### 第六段：法定继承部分总结

法定继承的核心：
1. **适用情形**：重点是“遗嘱继承人先于遗嘱人死亡”——写谁谁先死，遗嘱不生效，倒回去法定继承。
2. **继承顺序**：重点是第一顺序（配偶、父母、子女）+ 第一顺序的拟制继承人（丧偶儿媳/女婿尽了主要赡养义务；孙子女外孙子女代位继承）。
3. **遗产分配**：均等原则 + 例外（应当多分、可以多分、应当不分或少分、归公）。
4. **遗产酌给制度**：三种人可以适当分得遗产。
5. **双重继承**：有抚养关系的继子女可以双重继承。


### 第七段：遗嘱继承概述

四百一十五页来看第四个问题，继承方式之遗嘱继承。遗嘱继承显然是当事人**意思自治**的结果。

**适用的条件**（民法典第1133条）：
1. 没有遗赠扶养协议（如果有，遗赠扶养协议效力优先，排第一位）
2. 立有**合法有效**的遗嘱
3. 继承人既没有主动放弃、也没有被动丧失，也没有先于被继承人死亡


### 第八段：遗嘱的六种形式（核心考点★★★★★）

遗嘱是**死因行为**——死了才能生效。死无对证，形式要件必须完美，否则无法证明是遗嘱人的真实意思表示。

民法典规定了六种遗嘱形式（民法典第1134-1139条）：

| 遗嘱形式 | 核心要件 | 法典依据 |
| :--- | :--- | :--- |
| **公证遗嘱** | 由公证机构办理 | 第1139条 |
| **自书遗嘱** | **写、签、注**：亲笔书写 + 签名 + 注明年月日 | 第1134条 |
| **录音录像遗嘱** | 两个以上合格见证人在场见证 + **遗嘱人和见证人**在录音录像中记录**姓名、肖像及年月日**（见证人须出镜） | 第1137条 |
| **口头遗嘱** | **危急情况下** + 两个以上合格见证人在场见证；危急情况消除后能用其他方式的，口头遗嘱**无效** | 第1138条 |
| **代书遗嘱** | 两个以上合格见证人在场见证 + 一人代书 + 遗嘱人、代书人、其他见证人**签名并注明年月日** | 第1135条 |
| **打印遗嘱** | 两个以上合格见证人在场见证 + **遗嘱人和见证人应当在每一页签名** + 注明年月日（**新增，从未考过**） | 第1136条 |

**合格见证人的排除**（民法典第1140条）：无民事行为能力人、限制民事行为能力人；继承人、受遗赠人；与继承人、受遗赠人有利害关系的人——都不能当见证人。见证人一旦不合格，遗嘱无效。

> **录音录像遗嘱例题**：张三在录音录像遗嘱中说了全部内容，但见证人没有出镜——**无效**。见证人出镜了，但两个见证人说的日期不一致（一个说10号、一个说20号）——**无效**。只有**一个**见证人——**无效**。

> **口头遗嘱例题**：张三出车祸，在危急情况下口头立遗嘱（两个见证人在场），被送到医院后抢救活了，且能够以书面或录音录像形式立遗嘱——口头遗嘱**无效**。

> **代书遗嘱例题**（2025年考过）：只有**一个律师**代书，没有两个以上合格见证人——**无效**。

> **打印遗嘱例题**（新增，从未考过）：打印遗嘱只有最后一页签名——**无效**（须每一页签名）。【修正：原文“每一页签名”强调，骑缝签名亦可】


### 第九段：自书遗嘱例题

> **例题**：甲于2024年1月1日亲笔书写一份遗嘱，写明其全部遗产由乙继承，并签名和注明年月日（自书遗嘱**有效**）。同年3月2日，甲请张律师代书一份遗嘱，写明其全部遗产由丙继承（仅一名律师代书，无两个以上合格见证人——**无效**）。同年5月3日，甲因病被送至医院急救，口头遗嘱一份，内容是其全部遗产由丁继承，赵医生和李护士见证（危急情况，有效）。甲病**好转**后出院修养，未立新遗嘱。如甲死亡，应以哪份遗嘱为准？
>
> **答**：以**自书遗嘱**为准（2024年1月1日那份）。代书遗嘱无效，口头遗嘱因危急情况消除而失效。


### 第十段：遗嘱的撤回与变更

遗嘱人**可以撤回、变更**自己所立的遗嘱（民法典第1142条）。

- **以行为撤回**：立遗嘱后，遗嘱人实施与遗嘱内容相反的民事法律行为的，视为对遗嘱相关内容的**撤回**。
- **数份遗嘱内容抵触**：**以最后的遗嘱为准**（民法典第1142条第3款）。

> **注意**：数份遗嘱内容相抵触时，**每一份遗嘱都有效**，只是按照最后一份来执行。不能说是“前一份无效”——无效是绝对例外，不能乱用。


### 第十一段：遗嘱无效的情形（民法典第1143条、第1144条）

遗嘱只有两种效力状态：**有效或无效**（没有效力待定，也没有可撤销，因为死因行为——死了才能生效，死人没法表态）。

**无效情形**：
1. 无民事行为能力人、限制民事行为能力人所立的遗嘱**无效**
2. 受欺诈、胁迫所立的遗嘱**无效**
3. 伪造的遗嘱**无效**
4. 被篡改的遗嘱**被篡改的内容无效**
5. 遗嘱处分了国家、集体或他人财产的——**该部分无效**（不是整个遗嘱无效）
6. **未对缺乏劳动能力又没有生活来源的继承人保留必要遗产份额的**——**该部分无效**

**第六种情形的关键**：以**死亡时**为准判断（不是以写遗嘱时为准）。

> **必要份额例题**：张三写遗嘱“房子归母亲叶某”。两年后张三死亡，死亡时有儿子12周岁、女儿6周岁（两个未成年子女）——遗嘱**部分无效**，奶奶不能全部拿走，须给未成年子女保留必要份额。
>
> 如果二十二年后张三死亡，儿子32周岁、女儿26周岁（均已成年）——遗嘱**全部有效**，奶奶可以全部拿走。


### 第十二段：附义务的遗嘱（民法典第1144条）

遗嘱继承或遗赠**附有义务**的，继承人或受遗赠人应当履行义务。没有正当理由不履行的，经利害关系人或者有关组织请求，法院可以取消其接受附义务部分遗产的权利。

> **附义务遗嘱例题**：老王写遗嘱“遗产全部用于设立中国政法大学孟宪贵奖学金”。儿子拿了钱却不设立奖学金，女儿可以请求法院取消儿子的继承权，由女儿来完成父亲的遗愿。


### 第十三段：欺诈胁迫在不同场景下的效力对比（讲稿总结）

讲稿在本部分对欺诈胁迫在不同民事法律行为中的效力做了系统性总结（**重点，从未考过**）：

| 民事法律行为类型 | 欺诈的效力 | 胁迫的效力 | 原因 |
| :--- | :--- | :--- | :--- |
| **合同** | **可撤销**（第三人欺诈：一分为二） | 可撤销 | 欺诈不具不法性 |
| **婚姻** | 原则**合法有效**（可离婚）；重大疾病婚前未告知→**可撤销** | 可撤销 | 婚姻自由/配偶选择权 |
| **遗嘱/遗赠** | **无效** | **无效** | 死因行为，死无对证 |

> **注意**：同样的欺诈行为，在合同、婚姻、遗嘱中的效力完全不同——因为民事法律行为的性质不同。

### 第一段：代位继承

好，接下来咱们继续来看第五个问题：**代位继承和转继承**，这是继承编当中最为专业的两个术语，需要我们严格把它区分开。

首先呢，我们先来看什么叫代位继承。书的四百一十七页下方的表格，请看含义。民法典第一千一百二十八条，代位继承包括两种：

**第一种：传统型代位继承**

被继承人的子女**先于**被继承人死亡的，由被继承人的子女的**直系晚辈血亲**代位继承。简单来说就是：**爸爸妈妈先于爷爷奶奶姥姥姥爷死亡，孙子女外孙子女代位继承**。这就是我们前面看到的第一顺序的拟制继承人——孙子女外孙子女。

> **传统型代位继承例题**：2024年10月10日，爸爸小甲50周岁因车祸身亡。10月20日，爷爷甲90周岁因心脏病死亡。此时孙子小小甲10周岁，即可**代爸爸小甲之位**继承爷爷甲的遗产。

**关键点：血亲（重点）**

传统型代位继承强调的是**血亲**。法律意义上的血亲包括**自然血亲**（生子女）和**拟制血亲**（养子女），**不包括**有抚养关系的继父母子女关系。

> **再婚重组家庭例题（家有儿女）** ：刘梅先于姥姥死亡，刘星（刘梅的儿子）可以代位继承姥姥的遗产。那夏雨和夏雪（刘梅的继子女）能不能代位继承姥姥的遗产呢？
>
> **答**：**不一定**。原则上不可以（夏雨夏雪不是姥姥的血亲）。但例外——如果刘梅把夏雨夏雪**收养**了（办理了收养登记），就变成了拟制血亲，可以代位继承。

**第二种：新型代位继承**

被继承人的兄弟姐妹**先于**被继承人死亡的，由被继承人的兄弟姐妹的**子女**代位继承。简单来说就是：**侄子侄女、外甥外甥女有权代位继承伯伯、叔叔、姑姑、舅舅、姨妈的遗产**。

> **新型代位继承例题**：甲和乙是兄弟（互为第二顺序法定继承），父母早故。甲一生未娶妻生子。乙和丙结婚后生子丁（丁是甲的侄子）。乙先于甲死亡，甲死亡后留有两千万遗产。此时侄子丁即可**代位继承**伯伯甲的遗产。

**立法目的**：防止遗产归国家或集体（国不与民争利）。【补充：民法典第1128条第2款】


### 第二段：代位继承 vs 转继承（核心区别）

代位继承和转继承到底有啥区别？就是看**谁先死谁后死**的问题。

> **关键例题**：2025年5月10日，爷爷甲85周岁，亲笔书写一份有效的自书遗嘱：“待我死后，自己的全部遗产两千万归长子乙所有，次子丙不能继承。”（写谁谁先死——这是死亡笔记、死亡通知书）
>
> 6月25日，**长子乙出车祸身亡**（乙先死）。
>
> 7月2日，**甲得知噩耗后伤心欲绝，突发脑溢血死亡**（甲后死）。
>
> 在甲办理丧事期间，**次子丙伤心过度死亡**（丙后死，但死太快了）。
>
> 甲未重新写遗嘱。
>
> **丁（乙的儿子）怎么继承？**
>
> 爸爸乙先于爷爷甲死亡 → **代位继承**。丁是爷爷甲的第一顺序拟制继承人（孙子女代位继承）。丁拿到1000万。
>
> **戊（丙的儿子）怎么继承？**
>
> 甲死亡时，丙作为法定继承人本应拿到1000万，但**继承开始后、遗产分割前**丙死亡（死太快了）→ **转继承**。丙的1000万转给戊继承。戊拿到1000万。
>
> **最终：丁和戊各拿1000万。**

**代位继承 vs 转继承的区别**：

| 对比项 | 代位继承 | 转继承 |
| :--- | :--- | :--- |
| **通俗理解** | **死的早**（爸爸先死） | **死的快**（继承开始后、遗产分割前死亡） |
| **继承次数** | **一次继承** | **两次继承**（被继承人→继承人→转继承人） |
| **适用范围** | **只能发生在法定继承中**（遗嘱继承无代位继承） | **无限制**（法定继承、遗嘱继承均可） |

> **老师原话**：“长孙是爸爸死太早，于是代位继承；次孙是爸爸死太快，于是叫转继承。”


### 第三段：遗产管理人（民法典新增制度，已考过）

**遗产管理人是谁？**

- 有遗嘱执行人的 → 遗嘱执行人是遗产管理人
- 没有的 → 继承人**推选**遗产管理人
- 推选不出的 → **全体继承人共同担任**
- 全体继承人都放弃的 → 市民归**民政部门**管理；村民归**村民委员会**管理

**职责**：清理遗产、制作遗产清单、处理债权债务、分割遗产等（类似财产代管人、破产管理人）。

**责任**：遗产管理人**可有偿可无偿**（原则上无偿）。无偿情况下，**故意或重大过失**造成损失的才赔偿。


### 第四段：遗嘱 vs 遗赠（核心区别——沉默的后果不同）

**遗嘱和遗赠的性质相同**：都是**单方民事法律行为**（一个人说了算）。

**区别：对象不同**

| | 遗嘱 | 遗赠 |
| :--- | :--- | :--- |
| **对象** | **法定继承人**（配偶、父母、子女、兄弟姐妹、祖父母、外祖父母） | **法定继承人以外的人**（国家、集体、组织、其他个人） |
| **沉默的后果** | **视为接受**（国不与民争利，尽量少归公） | **视为放弃**（60天内不作表示 → 放弃） |

> **祖孙关系例题（易错点）** ：
>
> **例一**：孙子小甲自书遗嘱：“待自己死后，爷爷甲继承其全部遗产。”小甲死后，爷爷甲未作表示——**视为接受**（爷爷是孙子的第二顺序法定继承人，是遗嘱）。
>
> **例二**：爷爷甲自书遗嘱：“待自己死后，孙子小甲继承其全部遗产。”甲死后，孙子小甲未作表示——**视为放弃**（孙子不是爷爷的法定继承人，是遗赠）。


### 第五段：遗赠 vs 赠与

| 对比项 | 遗赠 | 赠与 |
| :--- | :--- | :--- |
| **相同点** | 都是**无偿**行为 | 都是**无偿**行为 |
| **性质** | **死因行为**（死后生效） | **生前行为**（生前生效） |
| **意思表示数量** | **单方民事法律行为**（一个人说了算） | **双方民事法律行为**（合同，需合意） |
| **形式** | **要式**（必须书面） | **不要式**（可以口头） |


### 第六段：遗赠扶养协议（继承编唯一双方民事法律行为）

**概念**：自然人可以与继承人以外的组织或个人签订遗赠扶养协议。按照协议，该组织或个人承担该自然人的**生养死葬**义务，享有受遗赠的权利（民法典第1158条）。

**性质**：**唯一一个双方民事法律行为**（协议=合同）。

**本质**：**生养死葬**（四个字）。

**双重属性**：抚养人的抚养义务**生前**生效；遗赠人的财产**死后**转移。

**主体限制**：遗赠人与抚养人之间**不能存在法定的扶养义务关系**。

> **遗赠扶养协议例题**：梁某已80多岁，老伴和子女都已过世（子女过世 → 孙子女外孙子女有法定赡养义务），年老体弱，欲立遗赠抚养协议，死后将三间房屋送给照顾自己的人。梁某的外孙子女不可以作为遗赠扶养协议的抚养人，因为二者之间存在法定的赡养义务关系。

**效力位阶**：**遗赠扶养协议 > 遗嘱/遗赠 > 法定继承**。

为什么遗赠扶养协议效力最优？因为子女不履行赡养义务，老人只能找外人（村委会等）来生养死葬。别人履行了义务，当然应该得到遗产。

**解除**：抚养人不履行生养死葬义务的，遗赠人可以解除协议。


### 第七段：限定继承与债务清偿（民法典第1161条）

**限定继承**：继承人**仅在遗产继承范围内**承担被继承人的债务，超出的部分**自然之债**（愿意还就还，不愿意还就不还）。

**清偿顺序**：

1. **法定继承人**先还
2. 不够的 → **遗嘱继承人**和**受遗赠人**按比例还
3. **遗赠扶养协议的抚养人**不用还（因为已经履行了生养死葬义务）

> **经典例题（债务清偿）** ：何某死后留下一间价值6万元的房屋和4万元现金。何某立有遗嘱：4万元现金由四个子女平分（遗嘱继承，每人1万）。房屋未作处理（按法定继承）。何某女儿主动放弃对房屋的继承权，三个儿子将房屋变卖，每人分得2万（法定继承共6万）。
>
> 现债权人主张何某生前借款12万，有借据为证。
>
> **第一步**：法定继承人先还 → 6万（房屋变卖款）
>
> **第二步**：12万 - 6万 = 6万，遗嘱继承人上场 → 4万（现金，四个子女每人1万）
>
> **第三步**：6万 - 4万 = 2万，还有2万未清偿 → 变为**自然之债**，四个子女愿意替父还就还，不愿意就不还。
>
> **结论**：D项“剩余两万元四人可以不予清偿”**正确**。


### 第八段：继承编专题总结

翻回四百零九页，继承部分重点：

1. **继承方式**：法定继承 vs 遗嘱继承（最核心）
2. **代位继承 vs 转继承**（最专业术语）
3. **遗嘱 vs 遗赠**（沉默后果不同）
4. **遗赠扶养协议**（唯一双方民事法律行为，效力最优）
5. **限定继承**（遗产范围内还债）

继承编虽然只有两分，但知识量大。要想得高分，需要把这两分细致地分析清楚。
