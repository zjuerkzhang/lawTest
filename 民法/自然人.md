# 自然人

## 民事权利能力

- 做人的资格，法律意义上的人。始于出生，终于死亡
- 生前叫胎儿，身后叫死亡
- 原则上胎儿不是民事权利能力的人，除了以下三种情况（民法典第16条）（如果胎儿娩出时是死体，民事权利能力自始不存在，应当不当得利返还）
    - 遗产继承
    - 接受赠与（纯获法律上利益）
    - 侵权行为（侵权损害赔偿权）：妇产医院过错
        - 民法典186条
        - 医院把病人治坏了，涉及到违约责任和侵权责任，二者择一
        - 民法典1191条，医院替员工承担侵权责任，替代责任。对外责任是医院承担，只能内部追偿
        - 如果在治疗，分娩过程中，医院过失导致胎儿损伤，则医院需要承担侵权责任

## 民事行为能力

你是法律意义上的什么人，**独立**做事的资格，不同年龄不平等

- 无民事行为能力
- 限制民事行为能力
- 完全民事行为能力

影响**独立做事**资格的因素
- **年龄**
    - 818原则
        - 不满8周岁，无民事，民法典144条决定都要家长/监护人决定
        - =>8 并且 <18周岁，限制民事行为能力
        - 18周岁（大学入学）及以上
    - 特例：16周岁（9年义务教育完成，劳动法限制16周岁以下的劳动合同）以上视为完全民事行为能力人，民法典18条2款
        - 16周岁以上
        - 以自己劳动收入为主要生活来源
- 精神健康状况
    - 完全不能辨认自己行为：无民事行为能力，需要有监护人
    - 不完全

考点：限制民事行为能力人独立实施的民事法律行为的效力

一般理性人标准，民法典总则编司法解释第5条，民法典145条
- 与本人生活相关联程度
- 价款或报酬
    - 买卖合同
        - 效力待定，待家长/监护人定
    - 赠与合同
        - 原则上受赠人是限制行为能力人，如果只有受益，则赠与有效
        - 如果附义务的赠与，专款（物）专用。如果受赠人没有履行义务，赠与人可以撤销赠与合同


## 监护
抚养、监督（教育）和保护，顺着民事行为能力的思路往下

### 对象：
- 无民事行为能力人
- 限制民事行为能力人

### 监护制度三位一体
- 家庭监护为基础
- 社会监护为补充
- 国家监护为兜底

考点：
- 监护人的确定/选任（找家长）
    - 法定监护：法律直接给孩子找监护人，**五个顺序**，防止推卸责任
        - 父母：
            - 只要健在，并且有监护能力
                - 离婚导致直接抚养和间接抚养，不影响监护关系和侵权责任
            - 包括拟制关系的养父母
            - 有包括抚养关系的继父母
        - 兄姐（有监护能力）
        - 其他个人和组织（自愿，有关单位同意）
        - 公职监护人，民政部门
    - 遗嘱指定监护
        - **父母**（只有孩子父母可以通过遗嘱指定监护人，其他人无权）死亡前通过遗嘱确定孩子监护人，无顺序，不限制人数。如果被遗嘱指定监护人不同意承担监护责任，按照法定监护人顺序找下一顺位监护人
        - 祖父母，外祖父母
    - 协议监护
        - 主体具有监护资格，能力
        - 开家庭会议，指定监护人，不限顺序和人数
    - **指定监护**
        - 前提
            - 父母双亡或丧失监护能力
            - 协议监护无法达成（都想当，都不想当）
        - 四个单位为孩子找监护人：居委会，村委会，民政部门，人民法院
        - 两条路径
            - 行政指定
            - 直接法院指定
        - 法院指定无法拒绝
    - 临时监护人（新增）
        - 不允许自然人担任
        - 居委会，村委会，法律规定的有关组织（未成年人保护组织），民政部门
    - 成年人意定监护民法典33条
        - 成年完人
        - 书面委托合同
            - 任意解除权：委托人和受托人都有解除权，必须合同生效前
        - 自己给自己找
        - 意定监护优于法定监护
    - 临时委托监护
        - 背景：留守儿童
        - 通过委托合同，临时找人监护，无顺序人数限制，临时委托监护不导致监护职责的丧失，委托人依然是监护人，受托人存在过错的，适当承担相关责任



- 监护人职责
- 监护人资格的撤销
- 监护人资格的恢复
- 监护关系的终止
