# 用益物权

## 对照《民法典》物权编“用益物权”提取备考提纲


### 一、用益物权概述（民法典第323条）

**用益物权**：对他人所有的**不动产**，依法享有**占有、使用和收益**的权利。

用益物权单纯支配和利用的是标的物的**使用价值**（区别于担保物权支配的是交换价值）。

**用益物权的标的物仅限于不动产**——房和地。


### 二、土地承包经营权（民法典第十一章）

#### （一）概念与范围（民法典第331条）

土地承包经营权人依法对其承包经营的耕地、林地、草地等享有占有、使用和收益的权利，有权从事种植业、林业、畜牧业等农业生产。

#### （二）两种承包方式

| 承包方式 | 土地类型 | 是否有偿 | 主体 |
| :--- | :--- | :--- | :--- |
| **家庭承包** | 耕地、林地、草地 | **无偿**（福利） | **本集体经济组织农户**（以户为单位） |
| **招标、拍卖、公开协商** | 四荒土地（荒山、荒沟、荒丘、荒滩） | **有偿** | 本村村民优先；外村人需“同意+批准” |

#### （三）设立（民法典第333条）

**土地承包经营权自土地承包经营权合同生效时设立**。

- 登记机构发放证书，**确认**权利（登记是证权作用，不是设权作用）
- **无须登记即可设立**（减轻农民负担）

#### （四）流转（民法典第335条）

土地承包经营权互换、转让的，当事人可以向登记机构申请登记；**未经登记，不得对抗善意第三人**。

**两个环节总结**：

| 环节 | 当事人 | 设立时间 | 登记效力 |
| :--- | :--- | :--- | :--- |
| **第一环节** | 村集体 ↔ 农户 | **合同生效时** | **无须登记** |
| **第二环节** | 农户 ↔ 农户（转让） | **合同生效时** | **登记对抗**（未经登记不得对抗善意第三人） |

#### （五）土地经营权（民法典第341条）

流转期限为**五年以上**的土地经营权，自流转合同生效时设立。未经登记，不得对抗善意第三人。【注释：讲稿未展开此点，属于土地承包经营权的延伸制度，了解即可】


### 三、居住权（民法典第十四章）

#### （一）概念（民法典第366条）

居住权人有权按照合同约定，对他人的住宅享有**占有、使用**的用益物权，以满足生活居住的需要。

#### （二）设立（民法典第367条、第368条）

- **书面形式**订立居住权合同（可以是独立合同，也可以是合同中的条款）
- 居住权**自登记时设立**（**登记生效主义**）
- **遗嘱**也可以设立居住权

**公式**：有效居住权合同/条款 + 登记 = 居住权

#### （三）权利限制（民法典第369条）

| 权能 | 限制程度 | 内容 |
| :--- | :--- | :--- |
| **占有、使用** | 可以 | 居住权人可以在住宅中居住 |
| **出租（收益）** | **相对禁止** | 不得出租，但**当事人另有约定的除外** |
| **转让、继承（处分）** | **绝对禁止** | **不得转让、不得继承** |

#### （四）消灭（民法典第370条）

居住权期限届满或者居住权人死亡的，居住权消灭。

#### （五）居住权 vs 租赁权

| 对比项 | 居住权（物权） | 租赁权（债权） |
| :--- | :--- | :--- |
| 权利性质 | 物权（用益物权） | 债权（负担行为） |
| 期限 | 可以超过二十年 | 最长**二十年** |
| 登记 | **登记生效**（不登记不设立） | 备案登记（行政管理） |
| 有偿/无偿 | **可有偿可无偿**（原则无偿） | **一定有偿** |


### 四、地役权（民法典第十五章）

#### （一）概念（民法典第372条）

地役权人有权按照合同约定，利用他人的不动产，以提高自己的不动产的效益。

- **供役地**：他人的不动产（被利用的那块地）
- **需役地**：自己的不动产（需要提高效益的那块地）

#### （二）设立（民法典第374条）

**地役权自地役权合同生效时设立**。未经登记，**不得对抗善意第三人**（**登记对抗主义**）。

#### （三）地役权 vs 相邻关系

| 对比项 | 地役权 | 相邻关系 |
| :--- | :--- | :--- |
| 性质 | **约定的**、享受性权利 | **法定的**、最低限度容忍义务 |
| 是否需要合同 | **需要** | 不需要 |
| 是否有偿 | 可有偿可无偿 | 法定的，无偿 |
| 判断方法 | 题目中有**合同**→地役权 | 题目中无合同→相邻关系 |

#### （四）地役权的从属性（民法典第372条）

地役权是**从权利**，随需役地的所有权/使用权一并转让。

#### （五）三种转让情形

| 情形 | 规则 |
| :--- | :--- |
| **需役地转让** | 地役权随主权利**一并转让**，受让人取得地役权 |
| **供役地转让** | 地役权**未经登记**→**不得对抗善意第三人** |
| **两边都转让** | 叠加上述两种规则分析 |

### 第一段：用益物权编体系概览

还得技术流，每一科你都值得更好的。

好，接下来咱们来看专题八，叫做“用益物权”。用益物权这个专题呢，每年就考一道题目，一道单选题，一分。二零二五年的时候呢，也是这么来的。

用益物权，它是指用他人之物让自己获益的物权，它单纯支配和利用的只是那个标的物的**使用价值**。这里边他人之物的“物”到底是什么呢？咱们国家的民法典，用益物权的标的物仅限于**不动产**，而不动产就包括两类：一类是房，一类是地。

五个权利：书的118页上方它的体系——**土地承包经营权**说的是地，**建设用地使用权**说的还是地，**宅基地使用权**也是地，**居住权**说的是房，**地役权**还是地。这五个权利当中四个地，一个房。


### 第二段：土地承包经营权（一）——农村土地的分类与设立

先来看第一个叫土地承包经营权，它解决的就是农村的问题。农村的土地主要就两块：一块就是耕地、林地、草地。特别考试重点喜欢考**耕地**。这个耕地我们采用的是**家庭联产承包责任制**，它是家庭承包的，并且是无偿获得的，它是一个福利。这个福利只能是**本集体经济组织的农户**来才享有，它是以户为单位去获得的这个权利。

农村的土地除了耕地、林地、草地外，还有一些**四荒土地**——荒山、荒沟、荒丘、荒滩。那些四荒土地是通过招标、拍卖、公开协商等**有偿方式**而获得的。有些村民想去开发这个四荒土地，村外的其他单位和个人也想开发这个四荒土地，那咋办？在同等条件下，本村村民享有**优先承包权**。

如果你真的要把四荒土地给村外的其他单位或个人来开发，必须经过严格的程序：首先本集体经济组织成员的村民会议**三分之二以上**成员或者三分之二以上村民代表同意，同时还得报**乡镇政府批准**——同意加批准，双重限制。【修正：原文此处有重复】

**设立**：农村是熟人社会，大家伙都认识，这块承包地到底是谁的，不用去登记大家也都清楚。所以民法典为了减轻农民负担，采用**登记对抗主义**。

**第一个环节**：村民从村集体经济组织获得土地承包经营权——**土地承包经营权自土地承包经营权合同生效时设立**（民法典第333条）。合同一生效，农户就取得了土地承包经营权。政府颁发的土地承包经营权证只是一个确权的作用、证权的作用，证明有权利，而不是说权利来源于它。

**第二个环节**：村民与村民之间转让——**未经登记，不得对抗善意第三人**（民法典第335条）。【修正：原文此处有大量重复】

> **例题**：村民胡某承包了一块农村集体所有的耕地，订立了土地承包经营合同，未办理确权登记。胡某因常年在外，便与同村村民周某订立土地承包经营权转让合同，将土地交周某耕种，未办理变更登记。
>
> **问**：关于该土地承包经营权，下列哪一说法是正确的？
>
> A. 未经登记不得处分——**错**
>
> B. 自土地承包经营合同生效时设立——**正确**（民法典第333条）
>
> C. 其转让合同自完成变更登记时生效——**错**（物债两分，合同效力与物权变动效力分离）
>
> D. 其转让未经登记不发生效力——**错**（是不发生对抗效力，不是不发生效力）


### 第三段：居住权（一）——制度目的与设立

居住权说的是住宅和房屋，它是**民法典新增制度**（第十四章），从民法典2020年以来连续不断地考。

居住权要解决的问题有两个：

**第一，离婚经济帮助制度**。比如张三和李四离婚后，李四没地住、生活困难，法院可能要求张三在自己所有的房子上给李四设置居住权——居住到李四死亡，或者住两年，由法院根据案件具体情况裁判。

**第二，以房养老**。比如张大爷八十五周岁，在上海有一套房子，无儿无女。他可以把房子卖给养老院，价款用于给自己养老，自己继续住在原来的房子里，同时保留居住权。【补充：这是实践中的典型应用场景】

**设立方式**（民法典第368条）：
- **书面有效的居住权合同或者条款 + 登记 = 居住权**
- 居住权**自登记时设立**（**登记生效主义**）
- 物债两分：前面的合同是债权效力，只有办理了登记，才能取得居住权

**遗嘱也可以设立居住权**——合法有效的遗嘱 + 登记 = 居住权。【补充：讲稿提到“遗嘱设立居住权”是唯一还没考过的点】

> **例题（2024年真题改编）** ：张甲年过八十，与其侄子张乙签订房屋买卖合同，价款两百万元。买卖合同中包括居住权条款，并约定张甲有终身居住权，张甲每个月需支付一千元。但未办理居住权登记。
>
> **问**：关于本案，下列说法正确的是？
>
> A. 居住权生前可以转让，死后不能继承——**错**（既不能转让也不能继承）
>
> B. 居住权可以有偿设立，居住权条款有效——**正确**（居住权可有偿可无偿；条款有效）
>
> C. 买卖合同生效时居住权正式设立——**错**（居住权登记生效，未登记不设立）
>
> D. 居住权不能以条款设立，居住权条款无效——**错**（可以通过合同条款方式设立）


### 第四段：居住权（二）——权利限制与消灭

**权能限制**：

- **占有、使用**——可以（用益物权的核心）
- **收益（出租）** ——**相对禁止**：不得出租，除非当事人另有约定（民法典第369条）
- **处分（转让、继承）** ——**绝对禁止**：不得转让、不得继承（民法典第369条）

**消灭**（民法典第370条）：居住权期限届满或者居住权人死亡的，居住权消灭。

**居住权 vs 租赁权的区别**（121页上方）：

| 对比项 | 居住权（物权） | 租赁权（债权） |
| :--- | :--- | :--- |
| 期限 | 可以超过二十年（直至居住权人死亡） | 租赁期限最长不超过二十年 |
| 登记效力 | **登记生效主义**（不登记不设立） | 备案登记是行政管理问题 |
| 有偿/无偿 | 可有偿可无偿（原则无偿） | **一定是有偿的** |
| 权利性质 | 物权 | 债权（负担行为） |


### 第五段：地役权（一）——概念与相邻关系的区分

地役权——在他人的土地上设定负担的权利。

**地役权 vs 相邻关系**：

| 对比项 | 地役权 | 相邻关系 |
| :--- | :--- | :--- |
| 性质 | 约定的、**享受性**的权利 | **法定的**、最低限度的容忍义务 |
| 是否需要合同 | **需要合同** | 不需要合同 |
| 是否有偿 | 可有偿可无偿 | 法定的，无偿 |

**判断方法**：看题目当中有没有**合同**。有合同→地役权；没有合同→相邻关系。

> **例题（经典地役权题）** ：甲乙丙依次比邻而居。甲为了修房向乙提出在其院内堆放建材，乙不允。甲遂向丙提出在其院内堆放，丙要求甲付费两百块，并提出不得超过二十天，甲同意。修房过程中，甲搬运建材从乙家门前经过，乙阻拦。
>
> **分析**：
> - 甲与乙之间（堆放建材）→ 乙没有义务让甲堆放，这**不是**相邻关系（不属于最低限度的容忍义务）
> - 甲与丙之间（付费堆放）→ **地役权关系**（有合同、有偿）
> - 甲从乙家门前经过→ **相邻关系**（乙负有最低限度的容忍义务，无权阻拦）


### 第六段：地役权（二）——设立、登记与从属性

**设立**（民法典第374条）：地役权**自地役权合同生效时设立**。未经登记，**不得对抗善意第三人**（**登记对抗主义**）。

**与土地承包经营权的区别**：
- 土地承包经营权：第一个环节（村民与村集体）不需要登记，第二个环节（村民之间转让）才需要登记对抗
- 地役权：**设立时**就是登记对抗主义

**地役权的从属性**（民法典第372条）：地役权是**从权利**，需役地是主权利。用益物权项下只有两个从权利——地役权和担保物权。

**三种转让情形**：

**情形一：需役地转让**（需要看风景的那块地转让了）——地役权随主权利一并转让，受让人取得地役权。

> **例题**：甲公司与乙公司约定，甲公司向乙公司支付一百万，乙公司在二十年内不在自己厂区建造六米以上建筑。后来甲将全部房屋售出。乙在自己的厂区建造了一栋八米的厂房。
>
> **分析**：小区的业主们（丙）有两种救济途径——①追究甲（开发商）的**违约责任**；②追究乙的**侵权责任**（侵犯了地役权）。二者择一选择。

**情形二：供役地转让**（供别人看风景的那块地转让了）——地役权未经登记，**不得对抗善意第三人**。

> **例题**：甲与乙约定，乙不在自己土地上建造高层建筑，甲每年支付四千元。两年后乙将土地使用权转让给丙，丙在该土地上建造了一座高楼。
>
> **分析**：甲乙之间地役权合同生效时已设立，但**未登记**。丙是善意第三人（查了登记簿，没有地役权登记），丙不受地役权约束。甲不能追究丙的侵权责任。

**情形三：两边都转让**（需役地和供役地都转让了）——叠加前两种规则分析即可。


### 第七段：本章总结与考点分布

五个权利，每年一道单选题一分：

| 权利 | 标的物 | 登记制度 | 考试频率 |
| :--- | :--- | :--- | :--- |
| **土地承包经营权** | 地（农村） | 设立：合同生效（无须登记）；转让：**登记对抗** | **重点** |
| **居住权** | 房 | **登记生效** | 民法典新增，连续考，2025年未考 |
| **地役权** | 地 | **登记对抗**（设立时即登记对抗） | **重点**（2025年考了） |
| 建设用地使用权 | 地 | 登记生效 | 放到担保物权专题讲 |
| 宅基地使用权 | 地 | — | 法考从未考过（仅一个选项），**不得抵押** |

【补充】宅基地使用权：农村村民绝对不能把宅基地抵押出去，如果抵押，抵押合同直接**无效**。建设用地使用权可以融资担保，放到专题九抵押权部分讲。
