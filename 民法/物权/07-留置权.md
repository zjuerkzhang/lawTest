# 留置权

## 第一部分：对照《民法典》留置权相关条款提取备考提纲


### 一、留置权概述（民法典第447条）

债务人不履行到期债务，债权人可以留置已经合法占有的债务人的动产，并有权就该动产优先受偿。

**核心特征**：
- **法定担保物权**（没有担保合同）
- 标的物限于**动产**
- 以**占有**为公示方法（移转占有型担保物权）
- **不受三年诉讼时效限制**（标的物在债权人手里）


### 二、民事留置权的成立要件（民法典第448条）

| 要件 | 内容 | 法典依据 |
| :--- | :--- | :--- |
| **1. 债权已到期** | 未到期债权不能行使留置权 | — |
| **2. 合法占有** | 占有债务人或第三人的动产（非侵权行为、非占有辅助人） | 民法典第448条 |
| **3. 同一法律关系** | 动产的占有与债权的发生基于同一合同关系 | 民法典第448条 |
| **4. 不违法违约背俗** | 不违反法律规定、当事人约定、公序良俗；**留置权具有可排除性** | 民法典第449条 |


### 三、留置权的可分性（民法典第450条）

留置财产为**可分物**的，留置财产的价值应当**相当于债务的金额**。

【注释】这与抵押权、质权的不可分性不同。原因是留置权是法定担保物权，没有担保合同，法律只需保护债权人的现有合法权益，无需超出债权范围。


### 四、商事留置权（民法典第448条、担保制度解释第62条）

双方都是**企业**时适用商事留置权规则。

| 情形 | 是否需同一法律关系 | 是否需营业债权 |
| :--- | :--- | :--- |
| **标的物是债务人动产** | **不需要** | ✅ 需要（营业债权） |
| **标的物是第三人动产** | **需要**（与民事留置权相同） | ✅ 需要（营业债权） |

> **注意**：商事留置权只适用于“营业债权”，即企业之间因**营业关系**产生的债权。


### 五、留置权的实现——宽限期（民法典第453条）

留置权人与债务人应当约定留置财产后的债务履行期限；没有约定或约定不明的，留置权人应当给债务人**六十日以上**的履行期限（鲜活易腐烂物品除外）。

- 一般动产：**六十日以上**宽限期
- 鲜活易腐烂物品：**可立即**变卖（民法典第453条第2款）


### 六、留置权的排序（民法典第456条）

同一动产上已设立抵押权或者质权，该动产又被留置的，**留置权人优先受偿**。

**排序规则**：
1. **留置权**（永远是老大——法定担保物权）
2. **超级动产抵押权**（交付后十日内登记——千年老二）
3. **登记的动产抵押权 / 动产质权**（公示在先原则——先登记或先交付的优先）
4. **未登记的动产抵押权**（效力最弱，仅优先于普通债权）


### 七、留置权的消灭（民法典第457条）

留置权人对留置财产**丧失占有**的，留置权消灭。

**限缩解释**：限缩为**自愿丧失占有**（主动放弃占有或抛弃留置物）。非因留置权人意愿丧失占有的（如被盗抢），留置权不当然消灭。

【注释】与质权对比：质权人主动返还质物后，质权不当然消灭（仅不得对抗善意第三人）；留置权人自愿丧失占有后，留置权**消灭**。

### 第一段：留置权概述——唯一法定担保物权

好，接下来咱们来看第四节留置权。

留置权是担保家族当中**唯一的一个法定担保物权**。所谓的法定担保物权就是**没有担保合同**。从标的物的角度来看，它只限缩在了**动产**范围内，所以我们又管它叫动产留置权。

从当事人的角度来看，包括了两类：一类叫**民事留置权**，一类叫**商事留置权**。你就看看题目当中有没有一方是自然人（老百姓）。只要有一方是自然人的，都适用民事留置权规则。如果双方都是企业，那就是商事留置权。2025届客观题考试考的就是**商事留置权**。【修正：原文“二零二五届”】


### 第二段：民事留置权的四个成立要件（核心考点★★★★★）

作为法定担保物权的留置权，最核心的考点就是**成立要件**。书的一百五十八页下方，民事留置权一共**四个要件，必须同时满足，缺一不可**：

1. **债权已到期**
2. **合法占有债务人或第三人的动产**【修正：原文“合法债务债务人或第三的动产”】
3. **基于同一法律关系**
4. **不违法、不违约、不违背公序良俗**【修正：原文“三个不违法不违约不被诉”】


### 第三段：要件一——债权已到期

**债权已到期**很简单。从法律谚语的角度来看，“未到期的债权视为无债权”。债权未到期，债务人享有期限利益，你就不能扣押人家的动产。

> **例题**：孟某为马某运货，约定货物送到后一周内支付运费。孟某在货物运到后立刻请求马某支付运费，被拒绝。孟某不可以留置部分货物。
>
> **分析**：因为约定“一周内支付”，债权尚未到期。


### 第四段：要件二——合法占有债务人或第三人的动产（核心考点）

拆分成三个小考点：

#### （一）动产——具有可分性

**法定担保物权留置权具有可分性**。2025届客观题考的就是它。

> **例题**：孟某委托马某保管两千台电脑，每台价值一万元，保管费就两万块钱。到期后孟某不支付保管费两万块钱。
>
> **问**：马某是要留置两千台电脑，还是留置两台电脑就行？
>
> **答**：留置两台电脑即可。因为留置财产为可分物的，留置财产的价值应当相当于债务的金额。

**为什么抵押权、质权没有这个规则？** 因为抵押权、质权是意定担保物权（有担保合同），而留置权是法定担保物权（没有担保合同），法律只需保护债权人的现有合法权益，没必要超出其债权范围。

#### （二）合法——不能是侵权行为

如果欠钱不还，能不能把表从债务人手里抢走？**不能**，那是侵权行为。

> **例题**：方某将一行李遗忘在出租车上，立即发布寻物启事，言明以两千元现金酬谢返还行李者。出租车司机李某发现该行李及寻物启事后与方某联系。方某拒绝支付两千元给李某。
>
> **问**：李某可否行使留置权？
>
> **答**：**不可以**。拾得遗失物应当返还权利人（物权请求权），李某的占有是**无权占有**，不属于合法占有。

#### （三）占有 vs 持有（占有辅助人）

- **占有**：双方当事人**平等**（如承揽合同、保管合同）
- **持有（占有辅助人）** ：双方当事人**不平等**（如劳动合同，用人单位与劳动者）

> **例题**：徐某是甲公司总经理（法定代表人），甲公司为其配备了一辆轿车供其上下班使用。后徐某辞职，甲公司尚欠其十万块钱工资。徐某欲对轿车行使留置权。
>
> **问**：徐某可否对甲公司的轿车行使留置权？
>
> **答**：**不可以**。徐某对轿车的占有是**持有**（占有辅助人），而非占有。劳动合同双方当事人法律地位不平等。


### 第五段：要件三——同一法律关系（核心考点★★★★★）

**同一法律关系 = 同一合同关系**。

法律明文规定**只有六个合同的特定当事人**享有留置权：

| 合同类型 | 享有留置权的人 | 考试频率 |
| :--- | :--- | :--- |
| **承揽合同** | **承揽人** | **★★★★★**（修车、修表、修设备） |
| **运输合同** | **承运人** | ★★★ |
| **保管合同** | **保管人** | ★★★ |
| 仓储合同 | 仓储保管人 | ★（了解） |
| 委托合同 | 受托人 | ★（了解） |
| 行纪合同 | 行纪人 | ★（了解） |

**反面考点：租赁合同没有留置权！**

> **例题（反面）**：我把房子租给你，你欠一个月租金，我能不能开门进去把你的衣服、包包、法考复习资料搬走留置？
>
> **答**：**不可以**。租赁合同的出租人没有留置权。欠租金 → 找他要租金、起诉、解除合同，救济途径有的是。

> **例题（老马和小贝）** ：小贝的足球踢入邻居老马家水井，老马受惊打碎花瓶。老马捞出足球后，小贝要求归还（物权请求权），老马要求赔偿花瓶损失（债权请求权）。如小贝拒绝赔偿，老马可否对足球行使留置权？
>
> **答**：**不可以**。双方互负请求权，但不是**同一法律关系**（同一合同关系），不符合民法典第448条规定的同一法律关系要件。


### 第六段：要件四——不违法、不违约、不违背公序良俗

重点讲**不违约**——留置权**具有可排除性**。

当事人完全可以**约定排除留置权**。很多同学会答错（因为觉得“法定物权不能排除”）。

**为什么可以排除？** 因为标的物可能是**鲜活易腐烂的物品**（海鲜、水果、蔬菜、月饼、粽子、螃蟹等），如果被留置了，过两天就坏了。所以当事人可以在合同中约定“不得留置”。


### 第七段：商事留置权（2025年客观题考过★★★）

**商事留置权**：双方都是**企业**（B to B）。

**四个要件中**：要件一（债权已到期）和要件四（不违法不违约不背俗）与民事留置权完全相同。

**核心区别是要件二和要件三**：

| 情形 | 要件二（占有） | 要件三（同一法律关系） |
| :--- | :--- | :--- |
| **标的物是债务人的** | 合法占有**债务人**的动产 | **不需要**同一法律关系（**营业债权**即可） |
| **标的物是第三人的** | 合法占有**第三人**的动产 | **需要**同一法律关系（与民事留置权相同） |

> **例题（商事留置权一——债务人动产）** ：A公司（出租车公司）常年找B公司修车。2025年9月，A欠B修理费10万元（营业债权）。2025年10月，A又来修车（车是A公司的），B占有了该车。
>
> **问**：B公司能否因9月的修理费而留置10月的车？
>
> **答**：**可以**。标的物是债务人的动产，无需同一法律关系，只需是营业债权即可（商事效率原则）。

> **例题（商事留置权二——第三人动产，尚未考过★★★★★）** ：A公司欠B公司修理费10万元（营业债权）。2025年10月，A从C公司租了一台车来修（车是C公司的）。
>
> **问**：B公司能否因A欠的修理费而留置C公司的车？
>
> **答**：**不可以**。标的物是第三人的动产，**必须基于同一法律关系**。B与C之间没有承揽合同关系，不能留置C的车。否则A和B可能恶意串通损害C的利益。


### 第八段：留置权的实现——宽限期（六十日以上）

留置权的标的物在债权人手里（移转占有型担保物权），**不受三年诉讼时效限制**。

但行使留置权不能马上卖标的物——需要给债务人**六十日以上的宽限期**（民法典第453条）。

- **一般动产**：至少**六十日以上**宽限期
- **鲜活易腐烂物品**：可以**立即**变卖

> **例题**：孟某将其所有的汽车送至汽车修理厂修理，因孟某未依约支付修理费，修理厂遂依法将该车留置。
>
> **问**：修理厂留置汽车后，应当给孟某多长的支付修理费的期间？
>
> **答**：**六十日以上**（大于等于六十天）。


### 第九段：留置权的排序（★★★★★每年必考）

**同一动产上并存多个担保物权时的排序**：

| 顺位 | 权利类型 | 原因 |
| :--- | :--- | :--- |
| **老大** | **留置权** | 法定担保物权，且贡献力最大（修车保值增值） |
| **老二** | **超级动产抵押权** | 十日内登记，超级优先权 |
| **老三** | **登记的动产抵押权 / 动产质权** | **公示在先原则**（先登记的优先/先交付的优先） |
| **老四** | **登记的动产抵押权 / 动产质权** | （另一者） |
| **老五** | **未登记的动产抵押权** | 效力最弱（仅优先于普通债权） |

> **排序例题**：A公司将产车抵押给B公司，但**未登记**（未登记动产抵押权）。后来A将产车质押给C公司并**交付**（动产质权）。C占有期间产车坏了，送到D公司修车，修理费2000元，A不给，D将产车**留置**（留置权）。
>
> **排序**：**D（留置权）> C（动产质权）> B（未登记动产抵押权）**


### 第十段：留置权的消灭——自愿丧失占有（与质押权对比★★★）

留置权人对留置财产**丧失占有**的，留置权消灭（民法典第457条）。

**必须限缩理解为“自愿丧失占有”**：

- **质押权**：质权人主动返还质物后，质权**不当然消灭**（仅不得对抗善意第三人）
- **留置权**：留置权人**自愿丧失占有**（主动返还）后，留置权**消灭**，变成普通债权人

> **对比模型**：委托你修表，修好不给200块钱，你把表扣押了（留置权）。后来你心软把表还给我了，我拿表就跑了。
>
> **问**：你的留置权还在不在？
>
> **答**：**不在了**。自愿丧失占有 → 留置权消灭，你只能找我主张200元普通债权。
