
### 一、物权变动

#### （一）基于合同行为的物权变动

买卖合同和赠与合同

**1. 不动产物权变动（民法典第209条）**

不动产物权的设立、变更、转让和消灭，经依法登记，发生效力；未经登记，不发生效力。

**公式**：有效合同 + 登记 = 不动产物权变动

针对不动产，**登记生效主义**

**2. 动产物权变动（民法典第224条）**

动产物权的设立和转让，自交付时发生效力。

**公式**：有效合同 + 交付 = 动产物权变动

针对动产，**登记对抗主义**，对抗第三人

**3. 区分原则（民法典第215条）**

合同效力与物权变动效力相区分。未办理物权登记，**不影响合同效力**。

```
有权处分 + 有效合同 + 登记/交付 = 物权变动 = 继受取得
```

#### （二）非基于法律行为的物权变动

**没合同**

| 原因 | 物权变动时间 | 法典依据 |
| :--- | :--- | :--- |
| **政府征收** | 征收决定生效时 | 民法典第229条 |
| **法律文书** | 法律文书生效时（仅限形成判决） | 民法典第229条 |
| **继承** | 被继承人死亡（继承开始）时 | 民法典第230条 |
| **合法建造** | **事实行为**成就时 | 民法典第231条 |

**未经登记不得处分**（民法典第232条）：依照上述方式享有不动产物权的，处分该物权时，依照法律规定需要办理登记的，未经登记，不发生物权效力。

**未登记不影响签订合同**

法律文书-诉的分类
- 确认之诉
    - 确认一种状态或事实，比如确认无效婚姻
    - 不会发生物权变动
- 给付之诉
    - 给付一定财物的行为
    - 不会发生物权变动
- 形成之诉（变更之诉）
    - 会发生物权变动
    - 离婚案子中判决所有权

#### （三）不动产登记制度

| 制度 | 法典依据 | 功能 | 期限 |
| :--- | :--- | :--- | :--- |
| **变更登记** | 民法典第220条 | 登记簿记载的权利人统一变更登记 | - |
| **异议登记** | 民法典第220条 | 登记簿记载的权利人不同意变更登记，厉害关系人申请异议登记，登记错误时保护真实权利人 | **15日**内不起诉则失效 |
| **预告登记**（主观题） | 民法典第221条 | 期房买卖，防止一房二卖。预告登记保护了从债权（购买合同）到物权的转换，主要是防止二卖时产生物权效力 | **90日**内不申请登记则失效 |

#### （四）动产交付制度

分类
- 现实交付
- 观念交付
    - 简易交付
    - 指示交付
    - 占有改变

| 交付类型 | 法典依据 | 本质 | 举例 |
| :--- | :--- | :--- | :--- |
| **现实交付** | — | 实际转移占有 | - |
| **简易交付** | 民法典第226条 | 先占有后购买，第二个合同生效时发生物权变动 | 两个人，先借后买 |
| **指示交付** | 民法典第227条 | 自己所有、他人占有，第二个合同生效时发生物权变动 | 三个人，直接由占有人转交给购买方 |
| **占有改定** | 民法典第228条 | 先出卖后占有（租/借），第二个合同生效时发生物权变动 | - |


### 二、所有权取得的特别规定

#### （一）善意取得（民法典第311条）

**构成要件**（同时具备，缺一不可）：
1. 无处分权人处分他人财产
2. 受让人受让时**善意**（不知情）
3. 以**合理的价格**转让
4. 转让的财产应当登记的**已登记**，不需要登记的**已交付**

**法律效果**：
- 受让人取得所有权（**原始取得**）
- 原所有权人有权向无处分权人请求**损害赔偿**

**不适用善意取得的情形**：盗赃物、遗失物、埋藏物、隐藏物（占有脱离物）

#### （二）拾得遗失物（民法典第312条至第318条）

- 拾得人应当**返还**权利人【补充：民法典第312条】
- 应当**妥善保管**，因故意或重大过失致毁损灭失的，承担赔偿责任【补充：民法典第316条】
- 招领公告发布之日起**一年内**无人认领，归**国家**所有【补充：民法典第318条】
- 拾得人可主张**必要费用**；有**悬赏广告**的可主张报酬【补充：民法典第317条】
- **侵占遗失物**的，丧失必要费用请求权和报酬请求权【补充：民法典第317条】

#### （三）先占

- 法律无明文规定，但学理和司法实践承认
- 对象：**无主动产**
- 关键：原所有权人是否有丧失所有权的意思

#### （四）添附（复合、混合、加工）

- 法律无系统规定，学理分类
- 原理：**一个人取得物权，一个人取得债权**（物债两分）
- 有过错→**赔偿**；无过错→**补偿**（公平原则）


### 三、业主的建筑物区分所有权

#### （一）定义与构成（民法典第271条）

业主对建筑物内的住宅、经营性用房等专有部分享有所有权，对专有部分以外的共有部分享有共有和共同管理的权利。

三项权利：**专有权 + 共有权 + 成员权**

#### （二）专有权（民法典第272条）

业主对专有部分享有占有、使用、收益、处分的权利。行使权利**不得危及建筑物安全，不得损害其他业主合法权益**。

#### （三）共有权（民法典第273条至第276条）

- 共有部分：道路、绿地（除规定外）、物业服务用房、公共场所、公用设施等
- 不得以放弃权利为由不履行义务
- 专有权转让，共有权和成员权**一并转让**
- 车位车库：规划内的归开发商（通过出售、附赠、出租方式）；新增的归业主共有；**优先满足业主需要**
- 共有部分产生的收入（广告费等），扣除合理成本后**归业主共有**

#### （四）成员权与表决规则（民法典第278条）

**参与表决门槛**：双**三分之二以上**业主参与表决

| 事项类型 | 同意门槛 | 具体事项 |
| :--- | :--- | :--- |
| **一般事项** | 参与表决双**过半数** | 选聘/解聘物业、使用维修资金等 |
| **特殊事项** | 参与表决双**四分之三以上** | 筹集维修资金、改建重建、改变共有部分用途/经营 |

#### （五）住改商（民法典第279条）

业主将住宅改变为经营性用房，除遵守法律、法规以及管理规约外，应当经**有利害关系的业主一致同意**。

【补充：最高人民法院相关司法解释明确，“有利害关系的业主”包括本栋建筑物内的其他业主；本栋之外的业主，能证明其房屋价值、生活质量受到或者可能受到不利影响的，也应当认定为有利害关系的业主。】


### 四、共有

#### （一）按份共有与共同共有（民法典第297条至第299条）

- **按份共有**：按份额享有所有权
- **共同共有**：共同享有所有权（需特殊身份关系）
- 没有约定或约定不明的，除具有家庭关系等外，**视为按份共有**

#### （二）内部关系

| 事项 | 按份共有 | 法典依据 |
| :--- | :--- | :--- |
| **管理** | 有约从约；无约各共有人都有管理权利和义务 | 民法典第300条 |
| **重大事项**（修缮、变更性质/用途） | 占份额**三分之二以上**同意 | 民法典第301条 |
| **费用负担** | 按份额负担 | 民法典第302条 |
| **分割** | **可以随时请求分割** | 民法典第303条 |
| **收益分配** | 有约从约；无约按出资额；不能确定则等额 | 民法典第309条 |

#### （三）份额转让与优先购买权（民法典第305条、第306条）

- **转让自由**：按份共有人可以自由转让份额
- **继承、遗赠不属于转让**：其他共有人不得主张优先购买
- **对内转让**：其他共有人不得主张优先购买
- **对外转让**：应**通知**其他共有人；其他共有人在**同等条件下**享有**优先购买权**
- **同等条件**：综合考虑转让价格、价款履行方式及期限等
- **两个以上共有人主张**：协商；协商不成按**份额比例**行使
- **仅以优先购买权受侵害为由请求撤销合同或认定无效**→**不予支持**

#### （四）外部关系（民法典第307条）

因共有物产生的债权债务：
- **对外**：共有人承担**连带债权、连带债务**
- **对内**：按份共有人**按份额**承担
- 超出份额的，有权向其他共有人**追偿**


### 五、本专题知识体系总览

```
专题七：所有权
│
├── 一、物权变动（每年必考）
│   ├── 基于合同的物权变动
│   │   ├── 不动产：有效合同 + 登记（登记生效）
│   │   ├── 特殊动产：有效合同 + 交付（登记对抗）
│   │   └── 普通动产：有效合同 + 交付
│   ├── 区分原则（民法典第215条）——合同效力与物权变动效力分离
│   ├── 非基于合同的物权变动
│   │   ├── 政府征收（生效时）
│   │   ├── 法律文书（形成判决生效时）
│   │   ├── 继承（被继承人死亡时）
│   │   └── 合法建造（成就时）
│   ├── 不动产登记：异议登记（15天）、预告登记（90天）
│   └── 动产交付：现实交付 + 观念交付（简易、指示、占有改定）
│
├── 二、所有权取得的特别规定
│   ├── 善意取得（民法典第311条）——四个要件，缺一不可
│   ├── 拾得遗失物——返还义务 + 必要费用 + 悬赏报酬
│   ├── 先占——无主动产
│   └── 添附——复合、混合、加工（一个人拿物，一个人拿钱）
│
├── 三、业主的建筑物区分所有权（民法典第271-287条）
│   ├── 三项权利：专有权 + 共有权 + 成员权
│   ├── 住改商——利害关系业主一致同意（一票否决）
│   ├── 车位车库——规划内归开发商；新增归业主共有
│   └── 表决规则（民法典第278条）
│
└── 四、共有（民法典第297-310条）
    ├── 按份共有 vs 共同共有
    ├── 内部关系：管理、重大事项（2/3）、费用、分割（随时）、收益
    ├── 份额转让：自由 + 通知 + 优先购买权
    └── 外部关系：对外连带、对内按份
```

> **备考提示（来自讲稿）**：
> 1. **物权变动**——每年必考，核心是“看有没有合同”。有合同要登记/交付；没合同（征收、法律文书、继承、合法建造）不需要。
> 2. **善意取得**——看到“善意取得”第一反应是“错”，它是例外不是常态。四个要件少一个都不能善意取得。
> 3. **业主区分所有权**——核心是“住改商一票否决权”。谁去告？邻里纠纷→单个业主；公共事务→业主大会/业委会。
> 4. **共有**——核心是区分“处分共有物” vs “处分共有份额”。处分共有物看2/3；处分份额看优先购买权。
