# 抵押权3

## 第一部分：对照《民法典》抵押权相关条款提取备考提纲


### 一、超级动产抵押权（购买价金担保权）（民法典第416条）

**概念**：动产抵押担保的主债权是抵押物的价款，标的物交付后**十日内**办理抵押登记的，该抵押权人**优先于**抵押物买受人的其他担保物权人受偿，但是**留置权人除外**。

| 关键要素 | 内容 |
| :--- | :--- |
| **法律依据** | 民法典第416条 |
| **公式** | 一交易两合同，先卖后押 |
| **标的物** | **动产**（生产设备等） |
| **登记时限** | 标的物交付后**十日内** |
| **优先效力** | 超级优先（仅**劣后于留置权**，即“**千年老二**”） |
| **超出十日** | 适用民法典第414条，**正常排序** |


### 二、抵押权的实现与诉讼时效（民法典第419条）

**核心规则**：抵押权人应当在**主债权诉讼时效期间**行使抵押权；未行使的，人民法院不予保护。

| 权利类型 | 标的物是否在债权人手里 | 是否受诉讼时效限制 | 法律依据 |
| :--- | :--- | :--- | :--- |
| **抵押权** | ❌（在抵押人手里） | ✅ **受三年诉讼时效限制** | 民法典第419条 |
| **质押权** | ✅（在质权人手里） | ❌ 不受限制 | 后边讲 |
| **留置权** | ✅（在留置权人手里） | ❌ 不受限制 | 后边讲 |


### 三、最高额抵押权（民法典第420条至第424条）

**概念**：为担保债务的履行，债务人或者第三人对一定期间内将要连续发生的债权提供担保财产，债务人不履行到期债务或者发生当事人约定的实现抵押权的情形，抵押权人有权在**最高债权额限度内**就该担保财产优先受偿。

#### （一）四个特点

| 特点 | 说明 |
| :--- | :--- |
| **担保在先** | 先设担保，后发生债权 |
| **最高额确定** | 有一个最高限额（杯子容量） |
| **决算期** | 确定债权数额的时间点 |
| **债权不确定性** | 债权连续发生，数额随时变化（杯子里水量加减） |

#### （二）债权转入（民法典第420条第2款）

最高额抵押权设立**前**已经存在的债权，经当事人同意，**可以**转入最高额抵押担保的债权范围。

#### （三）债权转让的独立性（民法典第421条）——核心考点★★★★★

最高额抵押担保的债权**确定前**，部分债权转让的，**最高额抵押权不得转让**。

> **讲稿理解**：最高额抵押权担保的是整个“杯子”（最高限额内的所有债权），不是杯子里某一滴水（某一个具体债权）。债权转让相当于从杯子里往外倒水——倒出去的水没有担保，变成普通债权。

#### （四）债权确定（民法典第423条）

有下列情形之一的，最高额抵押权人的债权确定：
1. 约定的债权确定期间届满
2. 没有约定或约定不明，抵押权人或抵押人自最高额抵押权设立之日起**满二年**后请求确定债权
3. 新的债权不可能发生
4. 抵押权人知道或应当知道抵押财产被查封、扣押
5. 债务人、抵押人被宣告破产或解散
6. 法律规定债权确定的其他情形

**讲稿重点**：看到“**破产**”→ 加速到期，债权立即确定。


### 四、借新还旧（民法典担保制度解释第16条）——从未考过★★★

**概念**：为借新还旧而签订新的借款合同，新贷偿还旧贷。

**核心规则**：
1. 旧贷的担保人**继续**为新贷提供担保的 → 继续承担担保责任
2. 新贷的担保人**不知道**借新还旧事实的 → **不承担担保责任**（债权人负有法定风险告知义务）
3. 债权人未履行告知义务 → 构成**欺诈**，担保人可主张撤销担保合同

| 情形 | 是否承担担保责任 |
| :--- | :--- |
| 旧贷担保人继续担保 | **承担**（知道风险） |
| 新贷担保人**知道**借新还旧 | **承担**（知情同意） |
| 新贷担保人**不知道**借新还旧 | **免责**（债权人未履行告知义务） |

> **讲稿理解**：债权人负有**法定风险告知义务**。没告知 = 欺诈 = 担保人无过错 = 免责（责任为零）。

### 第一段：超级动产抵押权（购买价金担保权）——概述

接下来咱们来看第八个问题，叫做**超级动产抵押权**，又叫**购买价金担保权**。它是动产担保家族当中的老三【补充：前两个是动产抵押权和动产浮动抵押权】。

这个制度重点要解决什么问题呢？一家企业要去另一家企业买一台生产设备，设备可能高达一千万。买方没有办法一次性付清，出卖人害怕收不到尾款，就可以采用超级动产抵押权的模式来给自己的价款做担保。

**本质：一交易两合同，先卖后押。** 一个交易（买一台生产设备），做两个动作：先卖给你，然后让你把取得所有权的设备再反向抵押回去给出卖人。

【注释】讲稿提到“长得特别像占有改定”——占有改定是“先卖后租/借”，这里是“先卖后押”，都是第二个合同生效时发生物权变动，但具体规则不同。

**考点就一个：排序。** 看出卖人在标的物交付后的**十日内**是否办理了抵押权登记：
- **十日内登记** → 超级优先权（**千年老二**，仅劣后于留置权，民法典第416条）
- **十日外登记** → 正常排序（适用民法典第414条）

【修正】原文“千斤老二”系语音识别错误，应为“千年老二”。


### 第二段：超级动产抵押权——物债两分原理铺垫

做这种题目要先梳理**物债两分区分原则**。

> **铺垫案例**：A公司把一台产车卖给B公司，价款一千万。
>
> **情形一**：A公司先交付产车给B公司 → 产车所有权归B公司（物权效力），B公司还欠一千万没付（债权效力）。如果B公司不给钱，A公司只能追究违约责任——**A公司被动**。
>
> **情形二**：B公司先付一千万给A公司 → 一千万是A公司的（货币占有即所有），产车还是A公司的（未交付）。如果A公司不交付产车，B公司只能追究违约责任——**B公司被动**。
>
> **结论**：谁先履行谁被动。所以法律设计了“先卖后押”的规则。

> **超级动产抵押权模型**：A公司同意把产车交付给B公司（转移所有权），但B公司还欠一千万没付。于是B公司把刚刚取得所有权的产车再**抵押**给A公司——用公式表示：**先卖后押**。


### 第三段：超级动产抵押权——排序例题（民法典第416条）

> **例题**：2025年1月1日，众森公司和徐工集团签订产车买卖合同，价款200万元。徐工集团于1月2日将产车交付给众森公司。为担保价款的履行，1月3日双方签订书面产车抵押合同，众森公司将所购产车抵押给徐工集团，并于1月4日办理了抵押登记手续（**交付后十日内登记**）。
>
> 经查：1月2日下午，众森公司将产车抵押给工商银行贷款100万元，并于当日办理了抵押登记手续。1月3日，众森公司将产车再次抵押给农行贷款100万元，并于当日办理了登记手续。
>
> **问**：众森公司何时取得产车所有权？
>
> **答**：1月2日（交付时发生物权变动）。
>
> **问**：工商银行、农行、徐工集团的抵押权如何排序？
>
> **答**：**徐工集团排第一**（交付后十日内登记，适用民法典第416条超级优先权），工商银行排第二（1月2日登记），农行排第三（1月3日登记）。
>
> **如果徐工集团于1月14日才登记**（交付后超过十日）→ 正常排序：工商银行排第一（1月2日登记）> 农行排第二（1月3日登记）> 徐工集团排第三（1月14日登记，适用民法典第414条）。

**核心规则**：动产交付后**十日内**办理抵押权登记，出卖人享有超级优先权，**仅次于留置权**（千年老二）。


### 第四段：抵押权的实现——诉讼时效（民法典第419条）

第九个问题，抵押权的实现。

**核心区别**：抵押权**不移转标的物占有**（标的物在抵押人手里，不在债权人手里）。标的物不在债权人手里 → 法律催促债权人及时行权 → 受**三年诉讼时效期间**限制（民法典第419条）。

【注释】讲稿对比：后边会讲的质押权和留置权，标的物在债权人手里，行权不受诉讼时效期间限制。

**民法典第419条**：抵押权人应当在主债权诉讼时效期间行使抵押权；未行使的，人民法院不予保护。


### 第五段：最高额抵押权（一）——概念与比喻

第十个问题，**最高额抵押权**。

**本质**：担保一次，借款多次。提高效率。

> **水杯比喻**：银行给你一个杯子（最高额），容量就是最高限额（如600万）。你可以在杯子里任意加水（借款）、倒水（还款），在杯子里的水都有担保，倒出去的水没在杯子里就没有担保。杯子有容量，超过容量的水会溢出来（超出最高额的部分转为普通债权）。

**四个特点**：
1. **担保在先**（先办担保手续）
2. **最高额确定**（杯子容量）
3. **决算期**（多长时间结一次账）
4. **债权发生及其数额具有不确定性**（杯子里水量随时变化）


### 第六段：最高额抵押权（二）——债权转让的独立性（核心考点★★★★★）

**核心考点：债权转让的独立性**（民法典第421条）。

最高额抵押权担保的债权确定前，部分债权转让的，**最高额抵押权不得转让**——倒出去的水没有担保，变成**普通债权**。

> **例题**：众森公司以建设用地使用权在**200亿额度范围内**为未来两年内连续发生的贷款提供最高额抵押担保，并办理了抵押登记。
>
> **问一**：众森公司之前欠工商银行的2000万到期债权，能否转入最高额抵押担保范围？
>
> **答**：可以。最高额抵押权设立前已经存在的债权，经当事人同意，可以转入最高额抵押担保的债权范围（民法典第420条第2款）。
>
> **问二**：工商银行将其中的8000万债权转让给农行，农行能否优先受偿？
>
> **答**：**不可以**。债权转让后，该债权**不在最高额范围内**，转为普通债权（民法典第421条）。
>
> **问三**：如果把全部1.2亿债权都转让出去呢？
>
> **答**：**仍然不可以**。最高额抵押权不随某一具体债权转让而转让，除非基础法律关系（杯子）一并转让。
>
> **问四**（算账）：众森公司共借款280亿（超过最高额200亿）→ 200亿范围内有担保优先受偿，80亿转为普通债权；正好200亿→全部有担保；180亿→180亿全部有担保。
>
> **问五**（破产加速到期）：众森公司被宣告破产→债权立即确定（加速到期）。


### 第七段：借新还旧（民法典担保制度解释第16条）——从未考过，重点关注

第十一个问题，**借新还旧**。民法典担保制度解释新增制度，**还从来没有考过**。

**本质**：借债权人的钱，还债权人的钱。债务人根本没有偿债能力，只是为了账面上好看（银行应对坏账率考核）。

**核心考点：新债担保人的知情权**（民法典担保制度解释第16条）。

债权人（银行）负有**法定风险告知义务**，必须告知新债担保人这是一个“借新还旧”的债。

- **告知了** → 新债担保人继续承担担保责任（“想死就埋了”）
- **没告知** → 新债担保人**不知情** → 可基于**欺诈**主张撤销担保合同 → 担保人**无过错** → **免责**（责任为零）


> **例题**：众森公司向工商银行贷款2000万，期限一年，海马公司（全资子公司）以办公楼提供抵押担保并登记。到期后众森公司不能还款，双方签订借新还旧合同。海马公司答应继续为新债提供担保。同时，嘉银公司为新债提供保证担保，但对借新还旧的事实**不知情**。
>
> **问一**：借新还旧合同效力如何？
>
> **答**：**合法有效**（完全符合民法典第143条）。
>
> **问二**：仅签订借新还旧合同，并未实际还款，旧债的担保是否消灭？
>
> **答**：**消灭**（理论上旧债消灭，担保消灭从属性——皮之不存毛将焉附）。
>
> **问三**：海马公司是否应当继续承担担保责任？
>
> **答**：**应当**（它答应了继续为新债提供担保）。
>
> **问四**：嘉银公司是否有权基于欺诈主张撤销保证合同？
>
> **答**：**有权**。债权人未履行法定风险告知义务，构成欺诈（民法典第148条）。
>
> **问五**：嘉银公司是否有权拒绝承担担保责任？
>
> **答**：**有权**。担保合同被撤销，担保人无过错，**免责**。
>
> **问六**：工商银行的抵押权与后来海路公司的抵押权，何者优先？
>
> **答**：**工商银行优先**（登记在先）。虽然理论上旧债担保消灭，但**未办理注销登记**，登记簿上仍有权利外观（商事外观主义，工商银行顺位保留）【补充：这是讲稿的延伸分析，实践中登记未注销时后手抵押权人能看到先手登记】。


### 第八段：抵押权十一个问题总结

把书翻回一百三十四页，抵押权十一个问题全部讲完，做一个大的复盘总结：

| 问题 | 核心考点 | 一句话总结 |
| :--- | :--- | :--- |
| **1. 抵押财产** | 交换价值 | 公益法人的**公益设施**不能抵押（民法典第399条） |
| **2. 抵押权的设立** | 不动产：登记生效；动产：登记对抗 | 看标的物决定规则 |
| **3. 动产浮动抵押权** | 正常经营买受人 | 四个要件（已支付价款+已取得+营业范围+持续销售）（民法典第404条） |
| **4. 抵押权和租赁权冲突** | 公示在先原则 | 先租后押→**不破租赁**；先押后租→**破租赁**（民法典第405条） |
| **5. 抵押财产的转让** | 抵押合同约定 | 无约定→**无需同意，仅需通知**（民法典第406条）；有禁止约定→**抵除权** |
| **6. 抵押权的处分** | 放弃 | 放弃债务人物保→其他担保人相应**免责**（民法典第409条） |
| **7. 抵押权的顺位** | 清偿顺序 | 先登>后登；登>未登；均未登→按比例（民法典第414条） |
| **8. 超级动产抵押权** | 宽限期**十日** | 十日内登记→**千年老二**（仅劣后于留置权）（民法典第416条） |
| **9. 抵押权的实现** | 三年诉讼时效 | 抵押权行权受**三年**诉讼时效限制（民法典第419条） |
| **10. 最高额抵押权** | 债权转让**独立性** | 杯子里有担保，倒出去没担保→转为普通债权（民法典第421条） |
| **11. 借新还旧** | 新债担保人**知情权** | 债权人未告知→担保人**免责**（担保制度解释第16条） |
