# 质权


## 第一部分：对照《民法典》质权相关条款提取备考提纲


### 一、动产质权（民法典第425条至第439条）

**概念**：为担保债务的履行，债务人或者第三人将其动产出质给债权人占有的，债务人不履行到期债务或者发生当事人约定的实现质权的情形，债权人有权就该动产优先受偿。

#### （一）设立（民法典第427条、第429条）

- 设立质权，当事人应当采用**书面形式**订立质押合同
- 质权自**出质人交付质押财产时设立**

**公式**：书面质押合同 + 交付 = 动产质权

#### （二）不得约定不交付（民法典第429条）

质权自交付时设立。当事人不得约定不交付标的物——约定因违反法律的强制性规定而**无效**（物权法定）。

#### （三）交付方式限制

- 可以：**现实交付、简易交付、指示交付**
- 不可以：**占有改定**

#### （四）质权人返还质物的效力

质权人将质物返还于出质人后，**不得以其质权对抗善意第三人**。（仅双方当事人之间，质权不当然消灭。）

#### （五）质权人的权利（民法典第430条、第435条）

| 权利 | 内容 |
| :--- | :--- |
| **孳息收取权** | 质权人有权收取质押财产的孳息（但**所有权仍归出质人**） |
| **放弃质权** | 放弃质权的，其他担保人在放弃范围内**免责**（与抵押权规则相同） |

#### （六）质权人的义务（民法典第432条、第437条）

| 义务 | 内容 |
| :--- | :--- |
| **妥善保管义务** | 保管不善致毁损灭失的，**承担赔偿责任** |
| **及时行权义务** | 怠于行权致价格下跌的，**承担赔偿责任** |
| **返还义务** | 债务清偿后返还质物；拍卖后多退少补 |

**质权行权不受三年诉讼时效限制**（标的物在质权人手里）。


### 二、权利质权（民法典第440条至第446条）

可以出质的权利包括（民法典第440条）：
1. 汇票、本票、支票、债券、存款单、仓单、提单
2. 可以转让的基金份额、股权
3. 可以转让的注册商标专用权、专利权、著作权等知识产权中的财产权
4. 应收账款
5. 法律、行政法规规定可以出质的其他财产权利

#### （一）有价证券（民法典第441条）

- **有权利凭证的**：质权自**权利凭证交付质权人时**设立
- **无权利凭证的**：质权自**办理出质登记时**设立

#### （二）股权（民法典第443条）

以基金份额、股权出质的，质权自**办理出质登记时**设立。

#### （三）知识产权财产权（民法典第444条）

以知识产权中的财产权出质的，质权自**办理出质登记时**设立。

#### （四）应收账款（民法典第445条）

以应收账款出质的，质权自**办理出质登记时**设立。

#### （五）保证金账户质押权（担保制度解释第70条）——新增

设立专门的保证金账户并由债权人**实际控制**的，债权人可就账户内款项优先受偿。


### 三、转质（民法典第434条）

质权人在质权存续期间，未经出质人同意转质，造成质押财产毁损、灭失的，应当承担赔偿责任。

| 类型 | 要件 | 效力 |
| :--- | :--- | :--- |
| **承诺转质** | 经出质人同意 | 有效（有权处分 → 继受取得） |
| **责任转质** | 未经出质人同意 | 第三人可能**善意取得**质权；转质人承担赔偿责任 |

### 第一段：质权（质押权）概述——与抵押权的核心区别

好，接下来咱们来看第三节质权，又叫质押权。

它和前面的抵押权最大的区别就是：原则上它要**移转标的物的占有**，也就是以**交付**作为公示方法为原则。

从标的物角度来看，就两种：**动产质权**和**权利质权**。【修正：原文“一种叫动产质权，一种叫权利质权”表述重复，已合并】


### 第二段：动产质权（静态）——设立与物债两分

先来看第一种，一百五十三页下方，叫**动产质押权**（静态的）。

**最简单的模型**：老孟找马某借一万块钱，拿自己价值两万七千五的手表做质押，交付到马某手里去。

操作分成两步：
1. **书面质押合同**（债权效力）——民法典第427条要求**书面形式**
2. **交付标的物**（物权效力）——移转占有

**公式：书面有效质押合同 + 交付 = 动产质权**（民法典第429条）

> **例题**：甲为担保对乙的债务，于2023年3月1日与乙签订质押合同，承诺将自己的越野车质押给乙。同年4月1日，甲交付越野车，但未将随车工具箱交付给乙。
>
> **问**：质押合同何时生效？质押权何时设立？
>
> **答**：质押合同于3月1日生效（债权效力）；质押权于4月1日设立（物权效力，交付时设立）。


### 第三段：动产质权设立的三个考点

#### 考点一：不得约定不交付（物权法定）

当事人能不能特别约定“不需要交付标的物也能产生动产质权”？**不能**。

民法典第429条明文规定，动产质权自交付时设立。约定不交付的，因违反法律的强制性规定而无效（**物权法定**）。

#### 考点二：不能采用占有改定方式交付

能不能采用**占有改定**的方式设立质权（即约定由出质人继续占有标的物）？**不能**。

理由：（1）无法公示质权的存在，害及交易安全；（2）出质人仍直接占有质押财产，质权人无法行使留置效力。【修正：原文“原因二”表述重复】

**只能是现实交付、简易交付、指示交付，不能是占有改定。**

#### 考点三：质权人主动返还质物后的效力（主观题★★★★★）

> **模型**：出质人老孟把表质押给质权人马某并交付。后来老孟求马某把表还给他用两天，马某心软答应了。老孟拿到表后就不还了——只有两个人，没有第三人。
>
> **问**：马某的质权还在不在？
>
> **答**：**还在**。质权不会因为质权人主动返还质物而当然消灭（仅存在于双方当事人之间）。

**但是**，如果老孟拿着表又卖给了善意第三人张三并交付：

> **问**：马某能否对张三主张质权？
>
> **答**：**不能**。质权人主动返还质物后，**不得以其质权对抗善意第三人**（质权实质上消灭）。

**核心规则**：质权人将质物返还于出质人后，不得以其质权对抗第三人。只有两个人时质权不消灭；出现善意第三人时，质权不能对抗。


### 第四段：动产物权法定规则体系对比

物权法定，物权的种类和内容由法律规定。涉及三个权利：

| 权利 | 物权变动规则 | 法律依据 |
| :--- | :--- | :--- |
| **动产所有权** | 原则：交付；**例外**：动产所有权保留买卖合同（付款完毕才转移） | 民法典第224条 |
| **动产抵押权** | **合同生效时设立**，未经登记不得对抗善意第三人（登记对抗） | 民法典第403条 |
| **动产质权** | **交付时设立**（交付生效） | 民法典第429条 |


### 第五段：质权人的权利与义务

#### 权利

**1. 孳息收取权**（民法典第430条）

质权人有权收取质押财产的孳息，但**孳息所有权仍属于出质人**。

> **核心口诀**：标的物在谁手里，谁就收取孳息（收取权≠所有权）。担保物权不改变所有权。

**2. 处分权（放弃）**

质权人放弃质权的，其他担保人在质权人丧失优先受偿权益的范围内免除担保责任（与抵押权放弃规则相同，民法典第435条）。

#### 义务

**1. 妥善保管义务**（民法典第432条）

质权人负有妥善保管质押财产的义务；因保管不善致使质押财产毁损、灭失的，应当承担**赔偿责任**。

**2. 及时行权义务**（民法典第437条）

出质人可以请求质权人在债务履行期限届满后及时行使质权；质权人怠于行使权利致使质押财产价格下跌的，应当承担赔偿责任。

**质押权行权不受三年诉讼时效限制**（标的物在债权人手里）。

**3. 返还义务**（多退少补）

债务履行期限届满，出质人清偿债务后，质权人应当返还质押财产。拍卖质押财产后，价款超过债权数额的部分归出质人所有，不足部分由债务人清偿。


### 第六段：静态动产质权例题（鹦鹉案）

> **例题（2016年真题）** ：2016年3月3日，甲向乙借款10万元，约定还款日期为2017年3月3日。借款当日，甲将自己私有的市值5万元的名贵宠物鹦鹉质押交付给乙，作为债务到期不履行的担保。另外，第三人丙提供了连带责任保证。
>
> **问**：关于乙的质权，下列哪些说法是正确的？
>
> A. 2016年5月5日，鹦鹉产蛋一枚，市值2000元，应由甲处置——**错**（应由乙收取孳息，孳息所有权仍归甲）
>
> B. 因乙保管不善，2016年10月1日鹦鹉死亡，乙需承担赔偿责任——**正确**（妥善保管义务）
>
> C. 2017年4月4日，甲未偿还借款，乙未实现质权，甲可以请求乙及时行使质权——**正确**（及时行权义务）
>
> D. 乙可放弃该质权，并在乙丧失质权的范围内免除丙相应的保证责任——**正确**（质权放弃，保证人相应免责）
>
> **答案：B、C、D**


### 第七段：动态动产质押权（流动质押权）——民法典担保制度解释新增，从未考过★★★

**解决的问题**：生产型企业（打火机、毛绒玩具、鞋子等）拿大量货物去银行贷款，银行没地方放这些货物。于是引入**监管人**（物流公司）。

**三方关系（代理制度）** ：
- **债权人（银行）** ——被代理人
- **监管人（物流公司）** ——代理人
- **出质人（企业）** ——相对人

**设立时间**：质权自监管人**实际控制货物**时设立（民法典担保制度解释第55条）。

**监管人责任**：监管人违反约定向出质人或其他人**放货**（擅自放货）→ 债权人可请求监管人承担**违约责任**。

> **模型**：易迈达公司向工商银行贷款2000万，以自己生产的20万双鞋子提供流动质押担保。易迈达公司、工商银行及大陆公司（物流公司）三方签订委托监管协议，约定大陆公司受工商银行委托监管货物。易迈达公司将20万双鞋子交付于大陆公司。
>
> **问**：工商银行何时取得质权？
>
> **答**：大陆公司实际控制货物时，工商银行取得质权。
>
> **问**：大陆公司擅自放货，谁承担责任？
>
> **答**：大陆公司承担**违约责任**（违反委托监管协议）。


### 第八段：权利质权——主观题★★★★★

**标的物**：股权、债权、票据等（七大标的物中的后三类）。

**核心问题**：何时取得权利质权？——看标的物**有没有权利凭证**。

#### （一）有价证券（汇票、本票、支票、债券、存款单、仓单、提单）

- **有权利凭证**：**交付权利凭证时**设立
- **无权利凭证**：**办理出质登记时**设立

#### （二）股权（民法典第443条）

以有限责任公司、股份有限公司的股权出质的，质权自**办理出质登记时**设立。登记机关：目前由**市场监管局**办理，未来发展趋势是**中国人民银行征信中心**。

#### （三）知识产权中的财产权（民法典第444条）

以注册商标专用权、专利权、著作权等知识产权中的财产权出质的，质权自**办理出质登记时**设立。

#### （四）应收账款（债权）（民法典第445条）

以应收账款出质的，质权自**办理出质登记时**设立（登记机关：中国人民银行征信中心）。

**本质**：拿债权做担保。A欠B一百万（应收账款），B拿这一百万的应收账款质押给C借五十万。到期B不还钱，C可以直接找A要钱。

#### （五）保证金账户质押权（民法典担保制度解释第70条）——从未考过★★★★★

**本质**：银行账户质押。债务人设立专门账户，由债权人**实际控制**（掌握密码），账户内保持最低限额的资金。

**设立时间**：债权人**实际控制**账户时。

**标的物浮动不影响质权**：账户内资金可进可出，只要保持最低限额即可。


### 第九段：转质（民法典第434条）

**概念**：质权人为担保自己的债务，将占有的质物移交于自己的债权人而设定新质权的行为。

| 类型 | 是否经出质人同意 | 性质 | 效力 |
| :--- | :--- | :--- | :--- |
| **承诺转质** | ✅ 经同意 | **有权处分** | 质权有效设立，系继受取得 |
| **责任转质** | ❌ 未经同意 | **无权处分** | 第三人可能**善意取得**质权（责任转质人承担赔偿责任） |

> **例题（转质）** ：孟某向徐某借款1万元，将自己的手表出质并交付于徐某。徐某未经孟某同意，向马某借款2万元，将手表出质于马某。马某不知此手表为孟某所有。
>
> **问一**：徐某的转质是承诺转质还是责任转质？
>
> **答**：**责任转质**（未经孟某同意）。
>
> **问二**：徐某与马某的质押合同效力如何？
>
> **答**：**合法有效**（无权处分不影响合同效力）。
>
> **问三**：马某能否取得质权？
>
> **答**：**能**（**善意取得**动产质权）。
>
> **问四**：手表因不可抗力毁损灭失，谁承担赔偿责任？
>
> **答**：**徐某承担**（有过错——无权处分他人之物）；**马某不承担**（不可抗力，无过错）。


### 第十段：质权部分总结

一百五十三页翻回来，质权依然是**标的物**的问题。

**动产质权**：
- 静态动产质权：以交付为原则
- 动态动产质押权（流动质押权）：**新增制度，从未考过**，重点关注

**权利质权**（五个手指）：

| 权利类型 | 设立方式 | 登记机关 |
| :--- | :--- | :--- |
| 有价证券 | 交付凭证/登记 | — |
| 股权 | **登记** | 市场监管局/征信中心 |
| 知识产权财产权 | **登记** | 相关部门 |
| 应收账款（债权） | **登记** | 中国人民银行征信中心 |
| 保证金账户 | **实际控制** | — |

**抵押权 vs 质权核心区别**：
- 抵押权：**不转移占有**，登记/登记对抗
- 质权：**原则上转移占有**，交付/登记生效
- 相同点：都属于**意定担保物权**，都需要**书面担保合同**
