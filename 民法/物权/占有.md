# 占有

## 第一部分：对照《民法典》占有相关条款提取备考提纲


### 一、占有概述（民法典第五分编 第458-462条）

**占有**：占有人对不动产或者动产的实际控制与支配的事实状态。

**占有的性质**：**事实状态**（而非权利），但**准用物权规则**予以保护。


### 二、占有的分类

| 分类标准 | 类型 | 判断要点 |
| :--- | :--- | :--- |
| **是否有权源** | 有权占有 / 无权占有 | 是否存在本权或法律上的原因 |
| **无权占有的主观状态** | 善意占有 / 恶意占有 | 是否**误信且无怀疑**该物是自己的 |
| **是否以所有意思占有** | 自主占有 / 他主占有 | 是否以自己所有的意思占有 |
| **占有方式** | 直接占有 / 间接占有 | 是否存在**占有媒介关系** |

**核心考点**：
- 有权占有 **不分** 善意/恶意
- 善意占有/恶意占有是**无权占有的再分类**
- 遗失物的拾得人 = **无权占有 + 恶意占有**
- 有权占有与无权占有**可以转化**（如租期届满后拒不返还）


### 三、善意占有与恶意占有的区别（民法典第459-461条）

| 区别点 | 善意占有人 | 恶意占有人 | 法典依据 |
| :--- | :--- | :--- | :--- |
| **返还原物及孳息** | ✅ 应当返还 | ✅ 应当返还 | 第460条 |
| **必要费用请求权** | ✅ **有**（维护支出的必要费用） | ❌ **无** | 第460条 |
| **使用致损责任** | ❌ 不承担 | ✅ **承担赔偿责任** | 第459条 |
| **标的物毁损灭失** | 仅返还**现存利益**（保险金、赔偿金等） | 返还**所受利益**，不足部分**赔偿损失** | 第461条 |

【补充】民法典第460条：权利人应当支付善意占有人因维护该不动产或者动产支出的**必要费用**。

【补充】民法典第459条：恶意占有人因使用占有的不动产或者动产，致使该不动产或者动产受到损害的，应当承担**赔偿责任**。

【补充】民法典第461条：占有的不动产或者动产毁损、灭失，占有人应当将因毁损、灭失取得的保险金、赔偿金或者补偿金等返还给权利人；权利人的损害未得到足够弥补的，**恶意占有人还应当赔偿损失**。


### 四、占有保护请求权（民法典第462条）——核心考点★★★★★

占有的不动产或者动产被**侵占**的，占有人有权请求**返还原物**；对**妨害**占有的行为，占有人有权请求**排除妨害**或者**消除危险**；因侵占或者妨害造成损害的，占有人有权依法请求**损害赔偿**。

| 请求权类型 | 适用情形 | 行权期间 | 期间性质 |
| :--- | :--- | :--- | :--- |
| **占有返还请求权** | 占有被**侵占**（侵夺） | **侵占发生之日起一年内** | **除斥期间** |
| **排除妨害请求权** | 占有被**妨害** | 不受一年限制 | — |
| **消除危险请求权** | 占有**有被妨害的危险** | 不受一年限制 | — |
| **占有损害赔偿请求权** | 因侵占或妨害造成**损害** | 适用诉讼时效 | — |

【补充】民法典第462条第2款：占有人返还原物的请求权，自**侵占发生之日起一年内**未行使的，该请求权**消灭**。


### 五、占有的推定效力

占有**推定**为所有权，但该推定**可以被推翻**（权利人可通过证据证明真正的所有权归属）。

### 第一段：占有制度概述

好，接下来咱们来看专题十，占有制度。

占有又叫**准物权**，它只是**准用物权的规则来处理**【修正：原文“准用物权的规则来处理”表述无误】。占有本质上就是一个**事实状态或事实关系**，它要解决的是标的物的非所有权人对这个标的物占有过程当中所产生的法律关系。

每年考试就考一分，考点一共四个：**占有的分类**、**善意占有与恶意占有**、**占有的保护**、**占有的推定效力**。其中最重要的考点就是**占有的分类**。


### 第二段：占有的分类（一）——有权占有 vs 无权占有

一百六十三页下方第一个，占有的分类。占有从学理上分成四种：

1. **有权占有 vs 无权占有**
2. **善意占有 vs 恶意占有**
3. **自主占有 vs 他主占有**
4. **直接占有 vs 间接占有**

如果非得挑一个，考试考得最多的肯定是**善意占有和恶意占有**。

#### （一）有权占有 vs 无权占有

**判断标准**：占有人对于标的物的占有是否存在**本权或权源**（法律上的原因）。如果有，就是有权占有；如果没有，就是无权占有。

**有权占有的权源**可能来自多种基础法律关系：
- **监护关系**：小梦有十根金条，由爸爸妈妈保管——有权占有
- **财产代管关系**：宣告失踪后的财产代管人——有权占有
- **保管合同**：委托他人保管手表——有权占有

**无权占有**（考试重点）：没有本权或权源，又叫**非法占有**。
- 偷的、抢的——如把别人的表抢走了
- 捡的——如拾得遗失物

> **例题**：孟某的表丢了被马某捡了，马某把表卖给了曹某并交付。请问曹某对表的占有是有权占有还是无权占有？
>
> **分析**：看相对谁而言。相对马某（出卖人）——有权占有（有买卖合同）；相对孟某（所有权人）——无权占有（孟某没让曹某取得表）。考试中未特别强调时，一般**相对于所有权人**来认定，即**无权占有**。

**有权占有 vs 无权占有的区分意义**：
1. 只有**无权占有**时，所有权人才能请求返还（有权占有 > 所有权 > 无权占有）
2. 留置权要求**合法占有**——即有权占有
3. 善意取得要求**占有委托物**——即有权占有
4. 占有关系产生的收益归属问题

**有权占有和无权占有可以转化**：租期届满后承租人拒不返还，从有权占有变为无权占有。


### 第三段：占有的分类（二）——善意占有 vs 恶意占有（核心考点）

**善意占有和恶意占有是无权占有的再分类**【修正：原文强调“有权占有不分善恶”】。

**判断标准**：无权占有人主观上是否**误信且无怀疑**该物是自己的。

**善意占有**（误信且无怀疑）：
1. **盗赃物的买受人**：信赖出卖人有所有权而购买
2. **继承人不知有他人之物**：继承人误以为遗产中的某物是被继承人的
3. **拿错行李/书本**：误以为别人的东西是自己的

**恶意占有**：
- **小偷**：明知是别人的还偷——无权+恶意+自主
- **遗失物的拾得人**：捡了别人的东西——无权+恶意（主观上不可能误信且无怀疑）

> **重点强调**：遗失物的拾得人一定是**无权占有+恶意占有**。不要用道德评价理解“善意/恶意”——善意占有和恶意占有都不是什么好东西，都是无权占有的再分类。


### 第四段：占有的分类（三）——自主占有 vs 他主占有

**判断标准**：占有人是否**以自己所有的意思**而占有标的物。

- **自主占有**：以自己所有的意思占有（小偷、侵占遗失物的人）
- **他主占有**：不以自己所有的意思占有（想归还遗失物的拾得人）

**遗失物的拾得人**：
- 想还 → 他主占有
- 不想还 → 自主占有（**侵占遗失物，丧失债权请求权**）【补充：民法典第317条】


### 第五段：占有的分类（四）——直接占有 vs 间接占有

**直接占有和间接占有是相对概念**，需要同时满足三个要件：

1. **占有媒介关系**：双方之间存在合法有效的合同关系（质押合同、保管合同、租赁合同等）
2. **直接占有人以他主占有的心态占有**
3. **间接占有人对直接占有人享有返还请求权**

> **例题**：老孟把手表质押给马某并交付。在质押期间，马某对表的占有是**直接占有**，老孟对表的占有是**间接占有**。

**盗赃物和遗失物不存在直接占有和间接占有**——因为小偷和拾得人与所有权人之间不存在合法有效的占有媒介关系。


### 第六段：善意占有 vs 恶意占有的法律效果区别

| 区别点 | 善意占有人 | 恶意占有人 |
| :--- | :--- | :--- |
| **返还原物及孳息** | ✅ 必须返还 | ✅ 必须返还 |
| **必要费用请求权** | ✅ **有**（如包书皮的钱） | ❌ **无** |
| **耗损/损害赔偿责任** | ❌ 不承担 | ✅ **承担** |
| **标的物毁损灭失的责任** | 仅返还**现存利益** | 返还**所受利益**（更重） |

> **必要费用例题**：同学误拿了老师的民法典，还包了个书皮（花了5毛钱）。老师要不要给这5毛钱？**要**——善意占有人有必要费用请求权。但如果明知是老师的还拿走，包了50个书皮——**不给**，恶意占有人无必要费用请求权。


### 第七段：占有的保护（一）——占有保护请求权（核心考点★★★★★）

**禁止黑吃黑**——即使是无权占有人，其占有事实状态也受法律保护。

> **例题**：孟某的表被马某偷走了（马某无权占有、恶意占有）。后来曹某又把表从马某手里抢走了（黑吃黑）。
>
> **问**：马某能否请求曹某返还表？
>
> **答**：**可以**。马某的权源是**占有返还请求权**（民法典第462条）。

#### 占有返还请求权（民法典第462条）

**构成要件**：占有被**侵占**（侵夺）——如盗窃、抢夺。

**行权期间（核心考点★★★★★）** ：

| 权利人 | 请求权基础 | 行权期间 | 期间性质 |
| :--- | :--- | :--- | :--- |
| **所有权人（孟某）** | 返还原物请求权 | **三年** | 诉讼时效 |
| **占有人（马某）** | 占有返还请求权 | **一年** | **除斥期间**（有权不用，过期作废） |

【补充】民法典第462条第2款：占有人返还原物的请求权，自**侵占发生之日起一年内**未行使的，该请求权消灭。

#### 排除妨害请求权与消除危险请求权

与物权保护规则相同：
- **排除妨害请求权**：占有被妨害时，占有人有权请求除去妨害
- **消除危险请求权**：占有有被妨害的危险时，占有人有权请求消除危险

> **例题**：承租人住进房子后，邻居每天在门口堆垃圾。承租人能否要求邻居清理？**可以**——承租人作为占有人享有排除妨害请求权，不需要等到所有权人来主张。


### 第八段：占有的保护（二）——占有损害赔偿请求权

因侵占或妨害占有造成损害的，占有人有权依法请求**损害赔偿**（民法典第462条第3款）。

所有权人和占有人都可以请求赔偿，但最终赔偿款应归属于所有权人。


### 第九段：占有的推定效力

**占有推定所有权**——为了维护交易安全和社会秩序，占有的状态被推定为所有权。

**但推定可以被推翻**：权利人可以通过证据（购买票据、合同、证人等）证明真正的所有权归属。


### 第十段：占有部分总结与物权编总结

**占有专题总结**：
1. 占有是**非所有权人的占有**（所有权人另有更强的保护方式）
2. 考点核心是**占有的分类**，特别是**无权占有**及**善意占有/恶意占有**
3. 分类是**相对而言**的，可能**转化**

**物权编整体总结**（书第86页）：
- 物权是**静态财产权**，调整财产的归属和利用关系
- **四个权利**：所有权、用益物权、担保物权、准物权（占有）
- **两个规则**：**物权法定** + **物债两分区分原则**
- 主观题重点：**担保物权**（第四分编）
