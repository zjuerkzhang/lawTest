## 第1-1部分：对照《民法典》侵权责任特殊主体提取备考提纲


### 一、监护人责任（民法典第1188条；侵权责任编司法解释一第4条、第6条、第13条）

| 要点 | 规则 |
| :--- | :--- |
| **责任主体** | **监护人**（不是被监护人） |
| **归责原则** | **无过错责任原则**（3要件） |
| **免责事由** | 尽到监护职责可**减轻** |
| **被监护人有财产** | 执行上可先执行其财产，但**实体上不判其承担责任**；程序上可作**共同被告** |
| **侵权行为时不满18，起诉时已满18** | 原监护人**继续承担**侵权责任 |
| **父母离婚** | 离异夫妻承担**连带责任** |
| **无抚养教育关系的继父母** | **不承担**侵权责任 |
| **临时委托监护** | **监护人套受托人**（大圈套小圈）：监护人承担全部责任，受托人在过错范围内承担相应责任 |


### 二、单位用工责任（民法典第1191条；侵权责任编司法解释一第17条）

| 要点 | 规则 |
| :--- | :--- |
| **责任主体** | **用人单位** |
| **归责原则** | **无过错责任原则**（3要件） |
| **工作人员对外** | **不承担责任**（替代责任） |
| **内部追偿** | 工作人员**故意或重大过失**的，用人单位可**追偿** |
| **职务行为判断** | 以**执行职务的外在表现形态**为标准（表现侵权） |
| **工作人员犯罪** | **不影响**用人单位承担民事责任（司法解释一第17条，**新增，从未考过**） |


### 三、劳务派遣（民法典第1191条第2款；侵权责任编司法解释一第16条）

| 要点 | 规则 |
| :--- | :--- |
| **责任主体** | **实际用工单位**（大圈）+ 劳务派遣单位（小圈） |
| **实际用工单位** | 承担**全部责任** |
| **劳务派遣单位** | 在**过错范围内**承担**相应责任**（大圈套小圈） |


### 四、个人用工（民法典第1192条）

| 情形 | 规则 |
| :--- | :--- |
| **提供劳务一方自己受损** | 按双方**过错**承担责任 |
| **提供劳务一方造成第三人损害** | **接受劳务一方**（雇主）承担侵权责任 |
| **第三人造成提供劳务一方损害** | **择一选择**：向第三人请求侵权赔偿，或向接受劳务一方请求补偿（不真正连带责任） |

## 第1-2部分：对照《民法典》侵权责任特殊主体提取备考提纲


### 一、帮工责任（人身损害赔偿司法解释第4条、第5条）

| 情形 | 责任承担 |
| :--- | :--- |
| **帮工人自己受损** | 根据双方过错承担责任 |
| **帮工人造成第三人损害** | 被帮工人承担侵权责任 |
| **第三人造成帮工人损害** | **择一选择**：向第三人请求赔偿，或向被帮工人请求补偿（不真正连带责任） |


### 二、定作人责任（民法典第1193条）

- **责任主体**：**承揽人**承担主要责任（大圈）
- **定作人**：在**选任过失**范围内承担**相应的责任**（小圈，大圈套小圈）
- **归责原则**：**过错责任原则**（四要件）


### 三、用工责任 vs 定作人责任的区分

| 判断标准 | 用工责任 | 定作人责任 |
| :--- | :--- | :--- |
| **支配力** | **强**（让你干什么就干什么） | **弱**（你是专业的，我不懂） |
| **关系性质** | 劳动合同/劳务合同 | 承揽合同 |
| **典型案例** | 上班、请保姆 | 修车、定做婚纱 |


### 四、网络服务提供者责任（民法典第1194-1197条）

- **归责原则**：**过错责任原则**（四要件）
- **免责条件**：收到通知后及时**删除、屏蔽、断开链接**
- **未采取措施**：对**扩大部分的损失**承担**连带责任**


### 五、安保义务人责任（民法典第1198条）

| 要点 | 规则 |
| :--- | :--- |
| **责任主体** | 公共场所的经营者、管理者、组织者 |
| **归责原则** | **过错责任原则**（四要件） |
| **第三人侵权** | **类似一般保证人**：先执行第三人，不足部分由安保义务人承担**相应的补充责任** |
| **程序上** | 不能单独先告安保义务人（类似先诉抗辩权） |


### 六、教育机构责任（民法典第1199条、第1200条、第1201条）

| 学生年龄 | 归责原则 | 举证责任 |
| :--- | :--- | :--- |
| **不满8周岁（无民事行为能力人）** | **过错推定** | 学校自证无过错 |
| **8周岁以上（限制民事行为能力人）** | **一般过错** | 受害人举证学校有过错 |

**第三人侵权**：学校承担**相应的补充责任**，**类似于一般保证人**（程序上不能单独先告学校，实体上先执行第三人）。

### 第一段：七类特殊主体责任概述

好，接下来咱们来看专题四十六，七类特殊主体责任。这个专题是客观题和主观题都可能会考察到的，在旁边写上**主观题**。

从这个专题开始，考题变得比较简单。民法典及其人身损害司法解释一共类型化了七类特殊主体：**监护人责任、用工责任、帮工责任、定作人责任、网络服务提供者责任、安保义务人责任和教育机构责任**。

这七类特殊主体的核心考点实质性问题就是三个：
1. **责任主体**——谁来承担侵权责任？
2. **归责原则**——是三个要件还是四个要件？（考不考虑主观过错要件）
3. **免责事由**——除了共同的免责事由以外，有没有单独的特殊的免责事由？


### 第二段：监护人责任——基本规则（民法典第1188条）

监护人责任很容易和民法典总则编中的监护制度结合在一起。说的是我们家小梦六岁，他拿着板砖把马蜂给拍了，医药费一万块钱——由我跟刘某（爸爸妈妈）这两个监护人来承担侵权责任。

**第一，责任主体**：**监护人**。法院绝对不能判小梦来承担侵权责任，他不用承担侵权责任，是由监护人来承担。

**第二，归责原则**：**无过错责任原则**（三个要件）。只要侵害行为、损害后果、因果关系具备，至于监护人主观上有没有过错——不考虑。

**第三，免责事由**：尽到了监护职责可以**减轻**（不是免除）。除此之外没有其他特殊的免责事由。


### 第三段：监护人责任——特殊情形（民法典侵权责任编司法解释一）

#### （一）被监护人有财产的，责任如何承担？

即使小梦有十根金条（有自己的财产），也是谁来承担侵权责任？**监护人**。法院绝对不准判孩子来承担侵权责任。

**程序上**（司法解释一第4条）：无民事行为能力人、限制民事行为能力人可以**列为共同被告**。

**实体上**：被侵权人请求监护人承担侵权人应当承担的全部责任，法院予以支持。判决中明确：赔偿费用可以**先从被监护人财产中支付**，不足部分由监护人支付——这是**执行程序**的问题，不是审判程序中判孩子承担责任。

> **核心**：程序上可以当共同被告；实体上不判孩子承担责任；执行上可先执行孩子财产。

#### （二）侵权行为发生时不满18周岁，起诉时已满18周岁（司法解释一第6条）

以**侵权行为发生时**为准。打人的时候17岁，需要监护，监护人（父母）承担侵权责任。起诉时18岁不影响。

> 被侵权人仅起诉行为人（已满18岁的本人），法院应当**释明**申请追加**原监护人**为共同被告。

#### （三）父母离婚（民法典第1188条）

离婚不影响监护，也不影响侵权责任，仅影响抚养。父母离婚后，未成年子女造成他人损害，**离异夫妻共同承担侵权责任**——**连带责任**。

#### （四）无抚养教育关系的继父或继母

未与子女形成抚养教育关系的继父或继母，**不承担**监护人侵权责任。

#### （五）临时委托监护（监护人套受托人——大圈套小圈，司法解释一第13条）

父母（监护人）套爷爷奶奶（受托人）——**大圈套小圈**。

- **大圈（监护人）** ：承担**全部责任**（100%）
- **小圈（受托人）** ：在**过错范围内**承担**相应的责任**

> **临时委托监护例题**：孟某和刘某把小梦交给爷爷奶奶带，小梦把隔壁小王揍了，医药费1万。爷爷奶奶在场却不予理会（有过错）。
>
> **分析**：爸爸妈妈（监护人）承担100%责任（1万），爷爷奶奶（受托人）在其过错范围内承担相应责任（如法院判20%即2000元）→ **大圈套小圈**（1万套2000）。
>
> **注意**：受害人最终只能拿到1万（填补性损害原理），不能1万+2000=1.2万。监护人多承担的部分可以向受托人追偿。


### 第四段：用工责任概述（民法典第1191条、第1192条，主观题重点★★★★★）

用工责任的核心是**替代责任**——侵权行为人和赔偿义务人是分离的，一个人替另外一个人去承担责任。

**用工责任分两类**：
- **单位用工**（用人单位责任）——适用劳动法
- **个人用工**（个人劳务责任）——适用民法典第1192条


### 第五段：单位用工——劳动者造成第三人损害（核心考点★★★★★）

**民法典第1191条**：用人单位的工作人员因执行工作任务造成他人损害的，由**用人单位**承担侵权责任。

**三个要点**：

**1. 对外关系**：用人单位**单独承担责任**，工作人员**对外无需承担民事责任**。只要题目说员工对外承担侵权责任——**一律是错的**。

**2. 内部追偿**：工作人员有**故意或重大过失**的，用人单位承担后可以向其**追偿**（内部关系，不得对外抗辩）。

**3. 职务行为的判断**：以**执行职务的外在表现形态**为标准（表现侵权）。公车私用撞人——单位也得赔，只能说明内部管理不严格。


### 第六段：劳务派遣（民法典第1191条第2款，新增重点★★★★★）

劳务派遣——**大圈套小圈**：

- **大圈（实际用工单位）** ：承担**全部责任**（100%）
- **小圈（劳务派遣单位）** ：在**过错范围内**承担**相应的责任**（如不当选派、未依法培训等）

> **劳务派遣例题**：马峰被保安公司（劳务派遣单位）派到中国政法大学（实际用工单位）当保安。巡逻时把学生孟某暴揍一顿，医药费1万。
>
> **分析**：中国政法大学（实际用工单位）承担100%责任（1万）；保安公司（劳务派遣单位）有过错（如未培训）的，在过错范围内承担相应责任（如20%即2000元）→ **大圈套小圈**。

**工作人员承担刑事责任，不影响用人单位承担民事责任**（司法解释一第17条，**新增，从未考过**）——三种法律责任并行不悖。


### 第七段：个人用工（民法典第1192条）

个人用工是指接受劳务的一方（雇主）与提供劳务的一方（雇员）之间的个人劳务关系（如请保姆、家教）。

**三种情形**：

**1. 提供劳务一方自己受损**：根据双方**过错**承担责任。

**2. 提供劳务一方造成第三人损害**：由**接受劳务一方**（雇主）承担侵权责任（与单位用工相同，替代责任）。

**3. 第三人造成提供劳务一方损害**（不真正连带责任）：

> **例题**：保姆张阿姨骑电动车买菜回家路上被老王家的狗咬伤，医药费20万。
>
> **救济途径**：
> - 向**第三人（老王）** 请求**侵权损害赔偿**
> - 向**接受劳务一方（雇主）** 请求**补偿**（雇主补偿后可向老王追偿）
>
> → **择一选择**（不真正连带责任）


### 第八段：本部分总结

| 特殊主体 | 归责原则 | 责任主体 | 特殊规则 |
| :--- | :--- | :--- | :--- |
| **监护人责任** | 无过错（3要件） | **监护人** | 尽到监护职责可减轻；程序上被监护人可作共同被告；离婚不影响连带 |
| **单位用工** | 无过错（3要件） | **用人单位** | 工作人员对外不承担责任；故意重大过失可内部追偿；犯罪行为不影响单位承担责任 |
| **劳务派遣** | 无过错（3要件） | **实际用工单位**（大圈）+ 劳务派遣单位（小圈） | 大圈套小圈，劳务派遣单位在过错范围内承担相应责任 |
| **个人用工** | 无过错（3要件） | **接受劳务一方**（雇主） | 第三人侵权时，提供劳务方可择一选择（向第三人或雇主补偿） |

### 第一段：帮工责任

好，接下来咱们来看第三类，帮工责任。

帮工责任和前面的用工责任最大的区别就是：帮工责任是**无偿的**，用工责任是**有偿的**。帮工的本质其实是一个**好意施惠关系**。好意施惠不存在合同违约责任问题和缔约过失责任问题，但有可能会存在侵权责任问题。

帮工就三种问题，跟用工一模一样：
1. 帮工人自己受损
2. 帮工人造成第三人损害
3. 第三人造成帮工人损害

考试的重点是**第三人造成帮工人受损**——和前面保姆阿姨的问题一模一样，就是**择一选择**。

> **帮工责任例题（第三人侵权）** ：老孟结婚，发小张三来帮老孟迎亲（无偿帮工）。迎亲过程中，张三被第三人李四揍了。
>
> **分析**：帮工人张三有两种救济途径——
> - 向**实际侵权人李四**请求**赔偿**
> - 向**被帮工人老孟**请求**适当补偿**
>
> 被帮工人补偿后可以向李四追偿。最终责任是李四的。

> **核心**：帮工责任考得比较少，一年只考过两次，重点抓“第三人侵权”这一个考点就行。


### 第二段：定作人责任（承揽合同背景）

定作人责任旁边写上**承揽合同**。

定作人责任是**过错责任原则**（四要件），一定要**有过错**才承担。定作人的过错体现在**选任过失**——是否选了一个不专业的人。

> **定作人责任例题**：老孟去专业修理厂修车，修理厂在工作中造成第三人损害——老孟（定作人）**不用赔偿**（没有过错，选了专业的人）。
>
> 如果老孟随便找了个非专业人员修车，造成第三人损害——老孟**有过错**（选任过失），要承担赔偿责任。

**大圈套小圈**：
- **大圈（承揽人）** ：承担**全部责任**（100%）
- **小圈（定作人）** ：在**过错范围内**承担**相应的责任**

> **核心**：承揽人是专业的，不受定作人控制。定作人不懂修车、不懂做婚纱，对承揽人没有支配力。


### 第三段：用工责任 vs 定作人责任的区分（核心识别点）

历届同学的核心问题不是结论不会，是**识别不出来**——到底是个用工责任还是个定作人责任？

**判断标准：支配力大小**。

> **修车场景（定作人责任）** ：老孟去修车，对修车师傅说“你这么修那么拧”——师傅说“你行你上”。老孟对师傅**没有支配力**（不懂修车）。

> **上班场景（用工责任）** ：老板对员工说“去楼下拿打印纸”——员工乖乖去。老板对员工**支配力极强**（让你干什么就干什么）。

> **保姆场景（用工责任）** ：雇主对保姆说“先别做饭，去看孩子”——保姆不能拒绝。雇主对保姆**支配力极强**。

**五个参考因素**（任意一个都可判断）：
1. 是否存在控制、支配和从属关系（核心）
2. 是否由一方指定工作场所、提供劳务工具或设备
3. 是否限定工作时间
4. 报酬支付方式
5. 工作性质


### 第四段：网络服务提供者责任（民法典第1194-1197条）

网络服务提供者责任考得比较少。

**两个场景**：

**场景一：显而易见的侵权（红旗飘飘）** ——比如马峰把老孟的裸照发到新浪微博上。任何人一看都知道侵权，网络服务提供者必须马上**删除、屏蔽、断开链接**。做了就免责，没做就承担**连带责任**。

**场景二：需要通知的侵权**——马峰写了一篇五千字的文章，老孟觉得侵犯名誉权。网络服务提供者的工作人员不是学法律的，不知道哪些话是捏造的。老孟得**通知**新浪公司。新浪收到通知后必须采取必要措施，否则对**扩大部分的损失**承担**连带责任**。

**归责原则**：**过错责任原则**（四要件）——网络服务提供者有过错才承担侵权责任。过错体现在“有没有采取必要措施”。


### 第五段：安保义务人责任（主观题重点★★★★★）

安保义务人指的是**公共场所的经营者、管理者**（宾馆、银行、车站、机场、体育场所、娱乐场所、小区的物业公司等）。

**第一，责任主体**：只要对该场所具有**适当的控制力**即可，**不以双方有交易关系为必要**。

> 去商场只逛不买、只蹭厕所，滑倒摔伤——**商场也要赔**（不能说“你没买东西”）。

**第二，归责原则**：**过错责任原则**（四要件）。有过错才赔。

> 酒店洗手间地滑没有放置警示标志，顾客滑倒——酒店**有过错**，要赔。

**第三，免责事由**：证明自己没有过错。

**第四，第三人侵权（核心考点）**——**安保义务人类似于一般保证人**！

> **第三人侵权模型**：孟某在丽都酒楼吃饭，马峰抢了他的手机，酒楼保安呆若木鸡不管。
>
> **程序上（告谁）** ：
> - 可以单独告马峰
> - 可以把马峰和酒楼一起告（共同被告）
> - **不能单独先告酒楼**（类似一般保证人的先诉抗辩权）
>
> **实体上（怎么判）** ：
> - **先执行马峰**的财产
> - 不足部分再由**酒楼承担补充责任**（第二顺位）

> **安保义务人例题（流浪狗案）** ：某小区有流浪狗频频咬人，居民向物业公司反映未获回应。居民马大姐倒垃圾时遗洒厨余垃圾未清理，流浪狗抢食时咬伤了小学生小军。
>
> **分析**：
> - 受害人：小军
> - 往前倒：被流浪狗咬的
> - 流浪狗的“管理人”是谁？**物业公司**（物业有义务处理流浪狗，反映后未回应→有过错）
> - 马大姐没有过错（正常倒垃圾，清理是物业的职责）
>
> **结论**：由**物业公司**承担侵权责任。


### 第六段：教育机构责任（校园事故责任）

教育机构责任和安保义务人责任**原理完全相同**——都类似于一般保证人。

**第一，责任主体**：**学校、幼儿园或其他教育机构**。老师是学校的员工（用工责任），老师**不独立对外承担责任**。题目说老师对外承担侵权责任——**一律是错的**。

**第二，监护职责未转移**：家长把孩子送到学校，**监护职责并未转移给学校**。

**第三，在校期间**：包括进入校门、校车、学校组织的校外活动（春游、秋游等）。

**第四，归责原则**——**过错责任原则**，看孩子年龄决定举证责任：
- **不满8周岁（无民事行为能力人）** ：**过错推定**（学校自证清白，过错要件举证责任倒置）——民法典第1199条
- **8周岁以上（限制民事行为能力人）** ：**一般过错**（受害人举证，谁主张谁举证）——民法典第1200条

**第五，第三人侵权**——**教育机构类似于一般保证人**！

> **第三人侵权模型（校园）** ：马峰冲到A小学把小朋友小梦捅伤，医药费10万。
>
> **程序上**：
> - 可以单独告马峰
> - 可以把马峰和学校一起告（共同被告）
> - **不能单独先告学校**（类似一般保证人）
>
> **实体上**：
> - **先执行马峰**的财产
> - 不足部分再由**学校承担补充责任**（第二顺位）
>
> **仅起诉学校**：法院应当向原告**释明**申请追加第三人为共同被告。


### 第七段：七类特殊主体责任总结

| 特殊主体 | 归责原则 | 特殊规则 |
| :--- | :--- | :--- |
| **监护人责任** | **无过错**（3要件） | 尽到监护职责可减轻；被监护人程序上可作共同被告，实体上不承担责任 |
| **用工责任** | **无过错**（3要件） | 员工对外不承担责任；故意重大过失可内部追偿；犯罪行为不影响单位责任 |
| **帮工责任** | — | 第三人侵权时，帮工人可择一选择（向第三人或被帮工人补偿） |
| **定作人责任** | **过错**（4要件） | 大圈套小圈（承揽人套定作人）；选任过失 |
| **网络服务提供者责任** | **过错**（4要件） | 删除屏蔽断开链接即免责 |
| **安保义务人责任** | **过错**（4要件） | 第三人侵权→**类似一般保证人**（相应的补充责任） |
| **教育机构责任** | **过错**（4要件） | 不满8岁→**过错推定**；8岁以上→**一般过错**；第三人侵权→**类似一般保证人**（相应的补充责任） |

**判断规律**：**人员密集的地方**（定作人、网络服务提供者、安保义务人、教育机构）采用**过错责任原则**，否则天天被告，赔不过来。

**“相应的责任” vs “相应的补充责任”** ：
- **相应的责任**（大圈套小圈）：**连带责任**，不区分先后顺序（临时委托监护、劳务派遣、定作人责任、教唆帮助侵权）
- **相应的补充责任**：**类似一般保证人**，有先后顺序（安保义务人、教育机构）

**2025年考情**：2025年考的是**安保义务人责任**。
