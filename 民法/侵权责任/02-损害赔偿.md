## 第1部分：对照《民法典》损害赔偿提取备考提纲


### 一、人身损害赔偿（民法典第1179条、第1180条）

| 损害程度 | 赔偿项目 |
| :--- | :--- |
| **一般伤害** | 医疗费、护理费、交通费、营养费、住院伙食补助费、误工费 |
| **残疾** | 上述费用 + 辅助器具费 + 残疾赔偿金 |
| **死亡** | 上述相关费用 + 丧葬费 + 死亡赔偿金 |

**同命同价**（第1180条）：同一侵权行为造成多人死亡的，可以以相同数额确定死亡赔偿金。


### 二、精神损害赔偿（民法典第1183条）

#### （一）性质

**非财产损害**，须达到“**严重**”程度。

#### （二）主体

只有**自然人**可以主张精神损害赔偿（法人、非法人组织不能）。

#### （三）适用范围（第1183条）

| 侵害客体 | 法典依据 | 说明 |
| :--- | :--- | :--- |
| **人身权益**（人格权+身份权） | 第1183条第1款 | 人格权、身份权受到侵害+严重精神损害 |
| **具有人身意义的特定物** | 第1183条第2款 | 须**故意或重大过失**；典型：定情信物、结婚录像带、父母遗照 |

#### （四）身份权益的特殊保护（民法典侵权责任编司法解释一，**新增，从未考过**）

**非法使被监护人脱离监护**（拐卖儿童）：
- 财产损失（找孩子的费用）→ **所有监护人**可主张
- 精神损害赔偿 → **近亲属监护人**可主张（父母子女关系或其他近亲属关系受到严重损害）
- 人身损害赔偿 → **近亲属监护人**可主张（孩子死亡/残疾）


### 三、违约精神损害赔偿（民法典第996条，**新增，从未考过**）

因当事人一方的违约行为损害对方**人格权**，并造成**严重精神损害**，受害方选择请求承担**违约责任**的，不影响受害方请求**精神损害赔偿**。

**要件**：违约 + 人格权受损 + 严重精神损害


### 四、财产损害赔偿（民法典第1184条）

**全面赔偿原则**：损失发生时的**市场价格**。

### 第一段：损害赔偿专题概述

好，接下来咱们来看专题四十五，损害赔偿。损害赔偿这个专题重点就包括三个问题：**人身损害赔偿、精神损害赔偿、财产损害赔偿**。当然重点考试对象就是**精神损害赔偿**，它可以单独命题，也可以结合其他制度来做一个选择题中的多选题或不定项选择题。所以这个专题只要抓住精神损害赔偿就行了。


### 第二段：人身损害赔偿

四百三十八页，先来了解一下人身损害赔偿。人身损害赔偿是指行为人侵害他人的生命健康权益，造成致伤、致残、致死等法律后果，承担金钱赔偿责任的一种民事法律救济制度。

侵害他人造成人身损害的，应当赔偿**医疗费、护理费、交通费、营养费、住院伙食补助费**等费用以及**因误工造成的损失**（民法典第1179条）。如果变成残疾了，还有**辅助器具费、残疾赔偿金**。如果不小心把人撞死了，还有**丧葬费和死亡赔偿金**。

法考范围内不会让大家计算具体数额，会直接给出结局——“医药费两万块”就赔两万。前面讲过，张三想打李四就想打五百块钱的，一不小心打成五千块钱的，赔五百还是赔五千？**赔五千**，因为是法律规定的。

**同命同价**（民法典第1180条）：因同一侵权行为造成多人死亡的，可以以相同数额确定死亡赔偿金。比如高铁事故、飞机失事、煤矿坍塌，赔偿同命同价。


### 第三段：精神损害赔偿——性质与主体

精神损害赔偿是**非财产上的损害**，其实是一个**惩罚性赔偿**。每个人的痛苦系数不一样，所以必须题目当中要有“**严重**”两个字。

> **严重要件例题**：隔壁邻居装修房子导致你失眠，能不能要精神损害赔偿？**不能**，因为没说“严重”（失眠达不到严重程度）。

**主体**：只有**自然人**可以主张精神损害赔偿。法人、非法人组织无血无肉无情感，不能要精神损害赔偿。前面讲名誉权侵权时说过，自然人的名誉权受到侵害可以要精神损害赔偿，法人不能。


### 第四段：精神损害赔偿的适用范围（核心考点★★★★★）

#### （一）人身权益

包括**人格权益和身份权益**。权益=权利+利益（死者人格利益受到侵害，近亲属也可以主张）。

**身份权益——配偶权**：
- **结婚过错损害赔偿**：婚姻被认定无效或被撤销，无过错方可以请求**物质损害赔偿和精神损害赔偿**（侵犯配偶权，**新增制度，重点关注**）
- **离婚过错损害赔偿**：离婚时无过错方可以请求精神损害赔偿

**身份权益——亲子权（民法典侵权责任编司法解释一，**新增，从未考过★★★★★**）** ：

**非法使被监护人脱离监护**（典型：拐卖儿童，如于华英拐卖杨妞花案）：

| 赔偿类型 | 请求权主体 | 说明 |
| :--- | :--- | :--- |
| **财产损失**（找孩子的路费、食宿费） | **所有监护人**（包括民政部门、居委会、村委会） | 可以请求赔偿 |
| **精神损害赔偿** | **近亲属监护人**（父母子女关系或其他近亲属关系受到严重损害） | 可以请求精神损害赔偿 |
| **人身损害赔偿**（孩子被弄残疾或弄死） | **近亲属监护人** | 死亡赔偿金等 |

> **核心**：近亲属监护人才可以要精神损害赔偿；组织监护人不可以。

#### （二）财产——具有人身意义的特定物

**三个典型**：定情信物、结婚录像带、父母遗照。

**主观要件**：必须是**故意或重大过失**。一般过失不能主张精神损害赔偿。

> 一起出去玩，我一不小心把你的结婚戒指撞到河里去了——**一般过失**，不能要精神损害赔偿。
>
> 用打火机把别人的父母遗照点着了烧没了——**故意/重大过失**，可以要精神损害赔偿。

**注意**：财产权受到侵害原则上不能要精神损害赔偿，但“具有人身意义的特定物”除外（民法典第1183条第2款）。


### 第五段：违约精神损害赔偿（新增，从未考过★★★★★）

民法典第996条（**新增，从未考过**）：因当事人一方的违约行为损害对方**人格权**，并造成**严重精神损害**，受害方选择请求其承担**违约责任**的，不影响受害方请求**精神损害赔偿**。

**核心**：违约 + 人格权受损 + 严重精神损害 = 可以主张精神损害赔偿。

> **违约精神损害赔偿例题**：从京东买了一个苹果手机，打电话时**爆炸把耳朵炸掉了**。
>
> **分析**：京东既违约又侵权（违约+人格权受损+严重精神损害）。找京东追究**违约责任**的同时，也可以主张**精神损害赔偿**（民法典第996条）。

> **反面例题**：扫地拖地机器人爆炸把地板炸坏了。能不能要精神损害赔偿？**不能**——没有人格权受到侵害，只有财产损失。


### 第六段：违约精神损害赔偿例题

> **例题（父母遗照案）** ：赵某在唐山大地震中成为孤儿，将唯一一张和父母的合照视为珍宝，送照相馆翻拍（**承揽合同**）。照相馆雇员乱扔烟头引起火灾，照片被烧毁，赵某十分痛苦（严重）。
>
> **问**：下列说法正确的是？
>
> A. 赵某可以要求照相馆承担违约责任——**正确**（合法有效的承揽合同之约）
>
> B. 照相馆侵犯了赵某的健康权——**错**（健康权说的是机能正常运转和发挥，照片被烧没有侵犯健康权）
>
> C. 照相馆侵犯了赵某的肖像权——**错**（肖像权侵权六种形态：丑化、污损、伪造、制作、使用、公开，没有）
>
> D. 赵某有权请求照相馆赔偿精神损害——**正确**（具有人身意义的特定物：父母的遗照，照相馆是专业人员，可认定重大过失）
>
> **答案：A、D**


### 第七段：财产损害赔偿

四百四十一页，财产损害赔偿的重点就是**全面赔偿原则**——**填补性损害原理**。

计算标准：**损失发生时的市场价格**。

> **财产损害赔偿例题**：姚某在玉石市场试戴翡翠手镯，唐某报价18万，实际进货价8万，市价9万。姚某急忙取下时不慎将手镯摔断。
>
> **问**：姚某应当赔偿多少？
>
> **答**：**9万**（损失发生时的市场价格）。

翻回四百三十八页，损害赔偿体系：
