## 第1-1部分：对照《民法典》七类典型侵权责任提取备考提纲


### 一、产品责任（民法典第1202-1207条）

| 要点 | 规则 |
| :--- | :--- |
| **责任主体** | 生产者、销售者（被侵权人可选择） |
| **归责原则** | **无过错责任** |
| **责任性质** | **不真正连带责任**（多因一果，最终责任人只有一个） |
| **救济途径** | 侵权（可主张精神损害赔偿）；违约（加害给付，+精神损害赔偿） |
| **警示召回** | 未及时采取补救措施 → 对**扩大损害**承担责任 |
| **惩罚性赔偿** | 明知缺陷仍生产/销售 → 惩罚性赔偿 |


### 二、机动车交通事故责任（民法典第1208-1217条；道路交通安全法第76条）

| 情形 | 责任主体 | 归责原则 | 特殊规则 |
| :--- | :--- | :--- | :--- |
| **车撞人** | 使用人（驾驶人） | **无过错责任** | 受害人有过错可减轻；赔偿顺序：交强险→商业险→侵权人 |
| **车撞车** | 有过错方 | **过错责任** | 按过错比例承担责任 |
| **租赁/借用** | 使用人（大圈）；所有人有过错时承担相应责任（小圈） | 使用人无过错；所有人**过错责任** | 大圈套小圈 |
| **转让未过户** | **最后一次受让人** | 无过错 | 交付即物权变动 |
| **拼装/报废车** | 转让人+受让人 | — | **连带责任** |
| **盗抢车** | 盗抢人+使用人 | — | **连带责任** |
| **挂靠** | 挂靠人+被挂靠人 | — | **连带责任** |
| **未投交强险** | 使用人（大圈）+投保义务人（小圈） | — | 大圈套小圈（**新增，从未考过**） |


### 三、医疗损害责任（民法典第1218-1228条）

| 要点 | 规则 |
| :--- | :--- |
| **责任主体** | **医疗机构**（医生护士不对外承担责任） |
| **归责原则** | **过错责任原则**（一般过错；三种情形过错推定） |
| **过错推定情形** | 违反诊疗规范；隐匿/拒绝提供病历；遗失/伪造/篡改/销毁病历 |
| **免责事由** | 患者不配合；已尽合理诊疗义务；限于当时医疗水平 |
| **不真正连带责任** | 药品/血液缺陷致害 → 患者可告生产者/血站，也可告医院 |

### 第一段：七类典型侵权责任概述

好，接下来咱们来看专题四十七，七类典型侵权责任。这个专题客观题和主观题都会考察，旁边写上**主观题**。

民法典侵权责任编第四到第十章的章名规定了七类典型的侵权责任，分别是：**产品责任、机动车交通事故责任、医疗损害责任、环境污染和生态破坏责任、高度危险责任、饲养动物损害责任和建筑物和物件损害责任**。

考题类型跟前一个专题一模一样，就是**三个实质性问题**：第一，责任主体；第二，归责原则；第三，免责事由。


### 第二段：产品责任——基本规则与救济途径（主观题重点）

产品责任旁边写上**主观题**。

**责任主体**：**生产者或销售者**（民法典第1202条、第1203条）。因产品存在缺陷造成他人损害的，被侵权人可以向产品的生产者请求赔偿，也可以向产品的销售者请求赔偿——给了消费者**选择的权利**。

> **产品责任例题一（消费者受损）** ：孟某从京东商城购买了一台科思公司生产的扫地拖地机器人，在扫地拖地过程中，机器人因技术缺陷发生爆炸，造成孟某严重人身损害。
>
> **分析**：孟某作为消费者，有**三种救济途径**——
>
> 1. 向**生产者（科思公司）** 请求**侵权损害赔偿**，同时可主张**精神损害赔偿**（严重人身损害）
>
> 2. 向**销售者（京东商城）** 请求**违约责任**（加害给付，民法典第186条），同时可根据**民法典第996条**主张**精神损害赔偿**（违约+人格权受损，**新增，从未考过**）
>
> 3. 向**销售者（京东商城）** 请求**侵权责任**，同时可主张**精神损害赔偿**
>
> **三种救济途径择一选择**。

> **产品责任例题二（第三人受损）** ：机器爆炸造成的是孟某的妻子刘某严重人身损害。
>
> **分析**：刘某是**第三人**（不是消费者），与京东商城**没有合同关系**，**不能追究违约责任**。刘某只能向生产者或销售者主张**侵权责任**。

**不真正连带责任**：多因一果（生产者的缺陷 + 销售者的销售 = 损害结果），最终责任人只有一个。生产者赔偿后可向有过错的销售者追偿，销售者赔偿后可向有过错的生产者追偿（民法典第1204条）。


### 第三段：产品责任——警示召回与惩罚性赔偿

**警示与召回**（民法典第1206条）：产品投入流通后发现存在缺陷，生产者、销售者应当及时采取**警示、召回**等补救措施。未及时采取补救措施或补救措施不力造成损害扩大的，对**扩大的损害**承担侵权责任。

**惩罚性赔偿**（民法典第1207条）：明知产品存在缺陷仍然生产、销售，或者没有采取有效补救措施，造成他人死亡或者健康严重损害的，被侵权人有权请求相应的**惩罚性赔偿**。


### 第四段：机动车交通事故责任——车撞人（无过错责任）

机动车交通事故责任：两种情形——**车撞车**（过错责任）和**车撞人**（无过错责任）。考试重点肯定是**车撞人**。

车撞人是**无过错责任原则**（三个要件：侵害行为、损害后果、因果关系），**不考虑使用人的过错**。

> **车撞人例题**：老孟开车撞了左宁，医药费1万。老孟能不能说“我没有酒驾、有驾照、正常行驶”？**不能说**，不考虑过错。老孟的免责事由是证明**受害人左宁有过错**（骑电动车不戴头盔、驮着十个人、电动车没上牌照等），可以减轻老孟的责任。

**赔偿顺序**（道路交通安全法第76条）：
1. **交强险**先赔
2. 不够 → **商业险**再赔
3. 还不够 → **侵权人**（使用人）赔


### 第五段：机动车交通事故——特殊情形（危险控制理论）

一共十几种情形，不用背，掌握**危险控制理论**——谁最能控制风险，谁却没控制住，谁赔。

#### （一）租赁或借用机动车（民法典第1209条）——常考

> 老孟借李家的车开，撞了左宁。李家是**所有人**，老孟是**使用人**。
>
> **分析**：
> - 使用人（老孟）——**大圈**，承担全部责任
> - 所有人（李家）——**小圈**，有过错才承担相应责任
>
> **所有人的过错**（三种）：
> 1. 知道车辆有缺陷（如刹车不灵）还借出
> 2. 知道驾驶人**无驾照**还借出
> 3. 知道驾驶人**饮酒、吸毒**还借出

> **例题**：甲将自己的汽车借给乙使用，某日乙**酒后**驾驶该车撞伤丙。
>
> **问**：丙的损失由谁承担？
>
> **答**：由**乙**承担全部赔偿责任。甲把车借给乙时不知道乙会酒后开车，没有过错，不承担责任。

#### （二）转让并交付但未办理过户登记（民法典第1210条）

车辆是**登记对抗主义**——交付即发生物权变动。多次转让的，由**最后一次转让并交付的受让人**承担赔偿责任。

> **例题**：孟某卖车给马某并交付（未过户），马某又卖给曹某并交付（未过户），曹某开车撞伤徐某。
>
> **问**：徐某可以请求谁赔偿？
>
> **答**：**曹某**（最后一次转让并交付的受让人）。

#### （三）拼装或已达报废标准的车辆（连带责任）

转让拼装或已达报废标准的机动车发生交通事故的，**转让人和受让人承担连带责任**（民法典第1214条）。以“不知道”为由主张不承担责任的，法院**不予支持**（司法解释一第20条）。

#### （四）盗抢机动车（民法典第1215条）

盗窃人、抢劫人或者抢夺人与机动车使用人承担**连带责任**。

#### （五）挂靠（连带责任）

挂靠营运的，挂靠人和被挂靠人承担**连带责任**。

#### （六）未投保交强险（大圈套小圈，**新增，从未考过**）

> **例题**：李家有车但没投交强险，孟某开李家的车撞了左宁，医药费1万。
>
> **分析**：
> - **使用人（孟某）** ——**大圈**，承担全部责任（100%）
> - **投保义务人（李家）** ——**小圈**，在**交强险责任限额范围内**与使用人共同承担责任
>
> **大圈套小圈**：孟某套李家。


### 第六段：医疗损害责任（民法典第1218-1228条）

**责任主体**：**医疗机构**。医生、护士不对外承担责任（用工责任）。

**归责原则**：**过错责任原则**（四要件，人员密集）。举证责任根据情形不同：

- **一般过错**：患者/家属举证（谁主张谁举证）
- **过错推定**（民法典第1222条）：医疗机构**自证清白**——①违反诊疗规范；②隐匿或拒绝提供病历资料；③遗失、伪造、篡改或违法销毁病历资料

**免责事由**（民法典第1224条）：
1. 患者或其近亲属**不配合**医疗机构进行符合诊疗规范的诊疗
2. 医务人员在抢救生命垂危的患者等紧急情况下已经尽到**合理诊疗义务**
3. 限于**当时的医疗水平**难以诊疗

**不真正连带责任**（民法典第1223条）：因药品、消毒产品、医疗器械的缺陷，或者输入不合格的血液造成患者损害的，患者可以向生产者、血液提供机构请求赔偿，也可以向医疗机构请求赔偿。医疗机构赔偿后，有权向负有责任的生产者、血液提供机构追偿。
