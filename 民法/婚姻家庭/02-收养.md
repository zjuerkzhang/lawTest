
## 第1部分：对照《民法典》收养制度提取备考提纲


### 一、收养关系当事人（民法典第1093-1095条）

| 当事人 | 定义 | 典型例子 |
| :--- | :--- | :--- |
| **被收养人** | 未成年人（不满18周岁） | 小梦 |
| **送养人** | 生父母等有抚养义务的人 | 孟某和刘某 |
| **收养人** | 收养子女的人 | 王张教授夫妇 |


### 二、收养关系中的年龄规定（核心考点★★★★★）

| 序号 | 年龄规则 | 法典依据 |
| :--- | :--- | :--- |
| 1 | 被收养人不满**十八周岁** | 第1093条 |
| 2 | 收养人年满**三十周岁**（夫妻均年满三十周岁） | 第1098条第5项 |
| 3 | 八周岁以上被收养人，须**征得其同意** | 第1104条 |
| 4 | 无配偶者收养异性子女，年龄相差**四十周岁以上** | 第1102条 |


### 三、收养人的基本条件（民法典第1098条）

**概括：无子一子、有能力、无病、无罪、满三十**

1. 无子女或者只有**一名子女**
2. 有抚养、教育和保护被收养人的能力
3. 未患有在医学上认为不应当收养子女的疾病
4. 无不利于被收养人健康成长的违法犯罪记录
5. 年满**三十周岁**


### 四、特殊关系的收养（民法典第1099条、第1100条、第1103条）

| 特殊情形 | 放宽的条件 | 法典依据 |
| :--- | :--- | :--- |
| **收养三代以内旁系同辈血亲的子女（过继子女）** | **不受四十周岁年龄差限制** | 第1099条 |
| **华侨收养** | 不受收养人“无子女或只有一名子女”的限制 | 第1099条第2款 |
| **收养孤儿、残疾未成年人或福利机构儿童** | 不受收养人数限制（可以收养多名） | 第1100条 |
| **继父母收养继子女** | **只需生父母同意**，不受其他条件限制 | 第1103条 |

> **特别提示**：特殊关系的收养中，考试重点考察的是**过继子女**和**继父母收养继子女**。


### 五、收养关系的成立（民法典第1105条、第1111条）

- 收养应当向县级以上人民政府民政部门**登记**，收养关系自**登记之日**起成立
- 收养关系成立后，养子女与养父母形成**拟制血亲**关系，与生父母**不再具有法律上的**父母子女关系
- 养子女的姓氏：可以随养父或养母的姓，也可以**保留原姓氏**（第1112条）


### 六、收养关系的解除（民法典第1114条、第1115条、第1118条）

| 情形 | 解除方式 | 法典依据 |
| :--- | :--- | :--- |
| **未成年养子女** | 原则**不得解除**；例外：收养人与送养人**协议解除**（8周岁以上须征得同意） | 第1114条 |
| **成年养子女** | 关系恶化、无法共同生活 → **协议解除**或**诉讼解除** | 第1115条 |
| **法定解除** | 收养人虐待、遗弃养子女 → 送养人可**诉讼解除** | 第1114条第2款 |

**解除的法律效果**：均须去民政部门**办理解除登记**；成年养子女对养父母有虐待、遗弃行为的，应**补偿**抚养费（第1118条）。

### 第一段：收养制度概述

好，接下来咱们来看专题四十二，收养制度。收养制度大概是每隔两到三年考察一次。我们已经很久没有考过了，所以咱们二零二六届的同学呢，要重点关注这个收养制度。

收养制度就是形成一个**拟制血亲**（养父母子女关系）的过程。这里边的考试呢，其实掌握两点，大家就能够把这里的问题全部掌握清楚了。**第一**，就是把这里边涉及到**年龄**的部分全部把它抓出来。**第二**，就是把**特殊情形的收养**把它重点学习，剩下的就没什么了。


### 第二段：收养关系的当事人

首先我们来看四百零三页下方，收养关系的当事人其实就三个人。

我们前面给大家讲过例子还记得吗？我跟刘某，我们两个结婚以后生育了一子小梦。但是我们俩呢，可能生活困难，没有办法抚养这个小梦，于是呢，我们两个就把这个小梦送给了隔壁王张教授当孩子。那我跟刘某呢，我们两个人就叫**送养人**。那王张教授他们两个呢，叫**收养人**。而小梦呢，就叫**被收养人**。所以这里边就这么三方当事人。


### 第三段：收养关系中的年龄规定（核心考点★★★★★）

书的第四百零四页，大家请看，那么被收养人、送养人和收养人。我们重点先来看**年龄**，你只要把年龄搞定了，那大部分的问题就解决了。

**第一个年龄：被收养人**

书的第四百零四页的上方就可以看到，被收养人是不满**十八周岁**的未成年人。【修正：原文“年龄是不满十八”】

**第二个年龄：收养人**

第四百零四页大家请看，收养人要**年满三十周岁**。如果王张教授夫妇双方都要收养的话呢，他们俩一致同意，并且两个人都得达到三十周岁以上，所以叫**均年满三十周岁**。

**第三个年龄：八周岁以上征得同意**

如果小孩子是**八周岁以上**的，还必须要征得他的同意。也就是说他已经不再是小孩子了，你得听他的意见，包括我们前面爸爸妈妈离婚的时候，如果孩子八周岁以上，要尊重其真实意愿。这里不是“尊重其真实意愿”，而是要**征得他的同意**。

**第四个年龄：无配偶者收养异性**

如果是无配偶者收养异性，为了防止性侵害，我们要求二者必须相差**四十周岁以上**。这里包括无配偶的男性收养女性和无配偶的女性收养男性，都适用这个规则。

> **收养年龄总结**：
> 1. 被收养人：不满**十八周岁**
> 2. 收养人：年满**三十周岁**（夫妻双方均年满三十周岁）
> 3. 八周岁以上被收养人：须**征得其同意**
> 4. 无配偶者收养异性：年龄相差**四十周岁以上**


### 第四段：收养人的基本条件

收养人的基本条件，我们罗列了五种，但大家不用背，了解一下就行（民法典第1098条）：

1. 无子女或只有**一名子女**
2. 有抚养、教育和保护被收养人的能力
3. 未患有在医学上认为不应当收养子女的疾病
4. 无不利于被收养人健康成长的违法犯罪记录
5. 年满**三十周岁**

**概括**：**无子一子、有能力、无病、无罪、满三十**。

这个条件不需要大家背，因为我们接下来要看的考点是**特殊关系的收养**，往往采用排除法，大家只要抓住排除的那个要素就可以了。


### 第五段：特殊关系的收养（一）——收养三代以内旁系同辈血亲的子女（过继子女）

四百零四页下方的第三种情况，也就是**收养三代以内旁系同辈血亲的子女**。这啥意思呢？就是实践当中**过继子女**的意思。

过继子女就一点不受那个**四十周岁**年龄差的限制。你只要把它排除了就行了。为什么排除呢？因为过继子女一般都是在叔伯、姑姑、舅舅、姨妈之间，往往受到伦理道德的限制，一般不会存在性侵孩子的情况。

> **过继子女例题**：我哥孟献富（大伯），没有结婚也没有生子。他特别喜欢我们家小梦（侄子），想收养小梦当儿子。这就是过继子女（收养三代以内旁系同辈血亲的子女）。
>
> 如果我哥想收养我们家女儿小小梦（侄女）——无配偶者收养异性，但因为是过继子女，**不受四十周岁年龄差的限制**。我哥40岁，小小梦6岁，相差34岁，按规定本来不行，但因为是过继子女，**可以**。

大家请看四百零四页下方，收养三代以内旁系同辈血亲的子女，你就熟悉一下**过继子女**就行了，这在实践当中是比较普遍的。


### 第六段：特殊关系的收养（二）——华侨收养

四百零五页上方，**华侨**来过继子女的话，条件进一步放松。还不受收养人子女数量的限制，也就是不受“无子女或只有一名子女”的限制。

比如我哥孟献富是华侨，他和他妻子已经有了两个孩子，现在还想收养我们家小小梦，可不可以？**可以**。哪怕已经有了两名子女，也不受影响。


### 第七段：特殊关系的收养（三）——收养孤儿、残疾未成年人或福利机构儿童

第四百零五页，收养**孤儿、残疾未成年人**以及儿童福利机构抚养的、查找不到生父母的未成年人。考试重点考的是**孤儿**。不受人数限制，你收养几个都行，只要你有能力。

比如我带着小梦去北京儿童福利院，只要我有能力，想收养几个就收养几个。


### 第八段：特殊关系的收养（四）——继父母收养继子女（再婚重组家庭）

第四百零五页，**继父或继母**经继子女的**生父母同意**，去收养继子女的。考试考点就是**生父母同意**就行了。

再来看一下《家有儿女》这张图。夏东海和刘梅再婚重组了一个家庭。夏东海想去收养刘星，条件就是**胡一统（刘星的生父）同意**就行了。当然，如果刘星八周岁以上，还得征得刘星本人的同意。


### 第九段：特殊关系收养例题（2022年真题）

> **2022年真题**：沈某（男，29周岁）与邱某（女，31周岁）均为再婚。二人婚前均有异子一女，分别为沈俊（5周岁）、沈俏（4周岁）和冯亮（7周岁）、冯丽（3周岁）。经沈某前妻和邱某前夫同意，二人计划收养对方子女，组成新的六人家庭。
>
> **问**：关于本案，下列说法正确的是？
>
> A. 沈某前妻和邱某前夫是否有特殊困难不影响收养——**正确**（只要生父母同意就行）
>
> B. 沈某未满三十周岁，不可以收养冯亮和冯丽——**错**（继父母收养继子女，不受收养人条件限制）
>
> C. 邱某已满三十周岁，可以收养沈俊和沈俏——**正确**
>
> D. 沈某可以，但只能收养冯亮和冯丽中的一人——**错**（不受收养子女数量的限制）
>
> **答案：A、C**


### 第十段：特殊关系的收养总结

你就想想我哥哥去收养我们家的小小梦（我女儿），他不受年龄差的限制。第二，如果我哥是华侨，他和他妻子已经有两个孩子了，还想收养我们家小小梦，也可以。第三，收养孤儿、残疾未成年人以及儿童福利机构的孩子，只要有能力，想收养几个都不受限制。第四，继父或继母为了家庭更为融洽，去收养继子女，只要看看生父母同不同意就行了。


### 第十一段：收养关系的成立——登记制度

只要是身份关系，都要进行**登记**。收养关系自**登记之日**起成立（民法典第1105条）。

这里有很多同学会误解，认为收养关系的成立必须要签订收养协议。你可以签也可以不签，重点还是看民政部门有没有办理收养登记手续。

一旦收养关系成立了，养子女与养父母形成法律上的父母子女关系，与生父母的关系**不再具有法律上的关系**（民法典第1111条）。

关于子女的姓氏，可以随养父养母的姓，也可以**保留原姓氏**，看当事人自己协商确定（民法典第1112条）。【补充：讲稿原文“协商一致就可以”】

收养关系无效的，自始没有法律效力（考得比较少）。


### 第十二段：收养关系的解除

**原则**：收养人在被收养人成年以前**不得解除**收养关系（民法典第1114条）。

**例外**：
- **协议解除**：收养人、送养人双方协议解除（八周岁以上须征得被收养人同意）
- **法定解除**：收养人虐待、遗弃养子女的，送养人可以向法院起诉解除

**成年养子女**：关系恶化、无法共同生活的，可以协议解除；协议不成的，可以向法院起诉（民法典第1115条）。

不管是哪种解除方式，最终都得去民政部门**办理解除收养关系的登记**。

**解除的法律效果**（民法典第1116条、第1118条）：
- 成年养子女对养父母有虐待、遗弃行为的，解除时应**补偿**收养关系存续期间支出的抚养费
- “你陪我长大，我陪你养老”——陪不了了，那就补偿


### 第十三段：收养制度专题总结

翻回四百零三页，收养的体系一共就这四个问题：

1. **年龄规定**（被收养人不满18、收养人年满30、8周岁以上征得同意、无配偶者收养异性相差40周岁以上）
2. **特殊关系的收养**（过继子女、华侨、孤儿残疾儿童、继父母收养继子女）
3. **收养关系的成立**（以登记为准）
4. **收养关系的解除**（原则上不得解除、例外可以协议解除或法定解除）

收养部分没有新增的司法解释，所有已有的知识都已经考察过了。大家只要做题去熟悉就可以了。
