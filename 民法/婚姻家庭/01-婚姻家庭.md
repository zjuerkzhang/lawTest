## 第1-1部分：对照《民法典》婚姻家庭编提取备考提纲


### 一、结婚要件

| 要件类型 | 内容 | 法典依据 |
| :--- | :--- | :--- |
| **实质要件（积极）** | 男女双方完全自愿；男≥22周岁，女≥20周岁；双方均无配偶 | 第1046、1047条 |
| **实质要件（消极）** | 无直系血亲或三代以内旁系血亲 | 第1048条 |
| **形式要件** | 亲自前往民政部门办理结婚登记 | 第1049条 |

**登记瑕疵**：程序瑕疵 → **行政复议或行政诉讼**（非民事诉讼）


### 二、无效婚姻（民法典第1051条）

| 情形 | 能否治愈 | 宣告主体 |
| :--- | :--- | :--- |
| **重婚** | **绝对无效，无法治愈** | 当事人、近亲属、基层组织 |
| **未达法定婚龄** | **可以治愈** | 当事人、未达法定婚龄者的近亲属 |
| **禁止结婚的亲属关系** | 绝对无效 | 当事人、近亲属 |

**重婚绝对无效**（婚姻家庭编司法解释二第1条）：即使前婚已解除或配偶死亡，重婚仍无效。


### 三、可撤销婚姻（民法典第1052条、第1053条）

| 情形 | 撤销权人 | 除斥期间 |
| :--- | :--- | :--- |
| **重大疾病婚前未告知** | **当事人本人** | 知道或应当知道之日起**1年** |
| **胁迫** | **当事人本人** | 胁迫行为终止之日起**1年** |

**撤销机关**：**法院**（非民政部门）。

**特殊规则**：不适用民法典第152条关于5年最长除斥期间的规定（身份行为的特殊性）。


### 四、无效/可撤销的后果（民法典第1054条）

1. 自始无效
2. **不影响亲子关系**
3. 无过错方有权请求**损害赔偿**（物质 + 精神）


### 五、夫妻财产制

#### （一）法定共同财产（民法典第1062条）

| 类型 | 认定规则 |
| :--- | :--- |
| 工资、奖金、劳务报酬 | 婚姻关系存续期间取得 |
| 生产经营、投资收益 | 婚姻关系存续期间取得 |
| 知识产权收益 | 发表或获得稿酬之一发生在婚姻关系存续期间 |
| 继承或受赠财产 | 原则共同；**确定只归一方**的除外 |

#### （二）父母出资购房（婚姻家庭编司法解释二第8条）

| 情形 | 处理规则 |
| :--- | :--- |
| **一方父母全额出资** | 可判归**出资人子女一方**所有，酌情补偿另一方（协力理论） |
| **双方父母出资/一方父母部分出资** | 以**出资来源及比例**为基础，综合考虑多重因素，判归一方所有，另一方获合理补偿 |

#### （三）法定个人财产（民法典第1063条）

1. 婚前财产
2. 人身损害赔偿/补偿金
3. 遗嘱/赠与确定只归一方的财产
4. 专用生活用品
5. 其他

**婚前财产收益**：孳息和自然增值仍为个人财产；其他收益（如租金）为共同财产。

| 专用生活用品等 | 第1063条第4-6项 |


### 二、夫妻之间赠与房屋（婚姻家庭编司法解释二第5条）

| 情形 | 处理规则 |
| :--- | :--- |
| **未办理转移登记** | 综合考虑多重因素，判决房屋归其中一方所有，另一方获补偿（物债两分） |
| **已办理转移登记 + 婚姻存续时间短 + 给予方无重大过错** | 判决房屋归**给予方所有**（防止骗婚） |


### 三、婚内分割共同财产（民法典第1066条）

1. 一方有**隐匿、转移、变卖、毁损、挥霍**夫妻共同财产或伪造债务的行为
2. 一方负有法定扶养义务的人患重大疾病，另一方不同意支付医疗费


### 四、夫妻债务认定（民法典第1064条）

| 债务用途 | 性质 |
| :--- | :--- |
| 共同生活、共同经营、共同意思表示 | **夫妻共同债务**（连带清偿） |
| 个人挥霍（赌博、吸毒等） | **个人债务** |


### 五、离婚制度

#### （一）协议离婚（民法典第1077条）

30天冷静期 + 30天领证期；任何一方反悔可撤回；过期不领视为撤回。

#### （二）诉讼离婚（民法典第1079条）

调解前置；七种法定情形（感情破裂抽象标准 + 具体情形）。

#### （三）特殊保护（民法典第1082条）

女方**怀孕期间、分娩后一年内、终止妊娠后六个月内**，男方不得提出离婚（女方提出或法院认为有必要受理的除外）。


### 六、离婚过错损害赔偿（民法典第1091条）

**情形**：重婚、与他人同居、家暴、虐待遗弃、其他重大过错（通奸、卖淫、嫖娼、赌博、吸毒等）。

**赔偿**：物质损害赔偿 + 精神损害赔偿（侵犯配偶权）。

**限制**：仅**无过错方**可主张；双方都有过错 → 不支持；以**离婚**为前提。

| 年龄 | 规则 |

| 8周岁以上 | 尊重真实意愿 |

**抢孩子**（新增）：可申请**人身安全保护令**或**人格权侵害禁令**；抢孩子是不利认定因素。

### 第一段：婚姻家庭编概述

好，接下来咱们来看第六部分，婚姻家庭编。从这一编开始，咱们就正式进入到了身份权的学习。身份权一共包括三个：**配偶权、亲子权和亲属权**。考试的重点是配偶权，说的就是夫妻关系。

整个这一编分成两个专题：专题四十一婚姻家庭，专题四十二收养。这两个专题基本上在客观题中考察，大概占三分左右。**主观题中只关注一个点，就是夫妻债务的认定。**


### 第二段：结婚的实质要件与形式要件

把书翻到三百八十八页，先来看第一个问题——结婚。

结婚需要**实质要件**和**形式要件**，考试的重点是形式要件。

**实质要件**分为正面（积极要件）和反面（消极要件）：

- **积极要件**：①男女双方完全自愿；②双方均达法定婚龄（男**二十二周岁**，女**二十周岁**）；③双方均无配偶。
- **消极要件**：不能有直系血亲或三代以内旁系血亲的亲属关系（伦理道德问题）。

**形式要件**：身份关系的确立都需要**登记制度**。夫妻关系（配偶权）需要民政部门办理婚姻登记手续；收养关系（亲子权）也需要民政部门办理收养登记手续。身份关系看登记，**必须亲自前往，不得代理**。

**登记瑕疵的处理**（民法典婚姻家庭编司法解释一第17条）：当事人以结婚登记程序存在瑕疵为由提起民事诉讼主张撤销结婚登记的，告知其可以依法申请**行政复议或提起行政诉讼**（不是民事诉讼）。

> **登记瑕疵例题**：孟宪贵（双胞胎弟弟）因录课让哥哥孟宪富代去办理结婚登记。后来孟宪富要结婚时，民政部门因其已登记一次拒绝办理。孟宪富如何救济？
>
> **答**：提起**行政复议或行政诉讼**（结婚登记是行政确认行为），不能提起民事诉讼主张撤销婚姻。


### 第三段：效力有瑕疵的婚姻——无效婚姻

婚姻这类身份行为一共有三种效力状态：**有效婚、无效婚和可撤销婚**。考试重点考察效力有瑕疵的婚姻。

无效婚姻的情形（民法典第1051条）——**三种，且是闭合性的**：

| 序号 | 无效情形 | 能否治愈 | 宣告主体 |
| :--- | :--- | :--- | :--- |
| 1 | **重婚** | **绝对无效，无法治愈** | 当事人、近亲属、基层组织（居委会/村委会） |
| 2 | **未达法定婚龄** | **可以治愈**（长大了就符合了） | 当事人、**未达法定婚龄者的近亲属** |
| 3 | **有禁止结婚的亲属关系**（直系血亲或三代以内旁系血亲） | 绝对无效，无法治愈 | 当事人、近亲属 |

**重点规则**：

**重婚绝对无效，无法治愈**（民法典婚姻家庭编司法解释二第1条）：即使前婚已解除或配偶死亡，重婚的婚姻仍然无效——不能因李四离婚或死亡来弥补张三与王五的重婚效力。

**未达法定婚龄可治愈**：如果在诉讼时无效情形已经消失（如已满法定婚龄），法院不予支持。

> **可治愈例题**：甲（男22周岁）为与乙（女19周岁）结婚，故意隐瞒乙真实年龄办理登记。两年后，乙（女21周岁）以登记时未达法定婚龄为由起诉宣告婚姻无效。
>
> **答**：法院**不予支持**。无效情形已消失（女方已满20周岁）。

**诉讼程序**：宣告婚姻无效是**消极的确认之诉**。


### 第四段：可撤销的婚姻

可撤销的婚姻情形（民法典第1052条、第1053条）——**两种，且是闭合性的**：

| 序号 | 可撤销情形 | 撤销权人 | 除斥期间 |
| :--- | :--- | :--- | :--- |
| 1 | **重大疾病婚前未告知**（欺诈） | **当事人本人** | 知道或应当知道撤销事由之日起**1年**内（**新增，从未考过**） |
| 2 | **胁迫** | **当事人本人** | 胁迫行为终止之日起**1年**内 |

**重大疾病婚前未告知**（民法典第1053条）：一方患有**重大疾病**（严重遗传性疾病、指定传染病、有关精神病），应当在结婚登记前如实告知另一方；不如实告知的，另一方可以向人民法院请求撤销婚姻。

**注意**：不是所有疾病都能撤销，必须是“重大疾病”（艾滋病、淋病、梅毒、麻风病等）。普通疾病（如乙肝）或非疾病虚假信息（如假学历）不构成可撤销婚姻的理由。

> **重大疾病例题**：甲声称具有海外名校学历，乙与之结婚。半年后乙发现甲的毕业证书系伪造。乙可否请求撤销婚姻？
>
> **答**：**不可以**。欺诈婚姻不属于可撤销情形，乙可以请求**离婚**，但不能请求撤销婚姻。

**胁迫**（民法典第1052条）：因胁迫结婚的，受胁迫的一方可以向人民法院请求撤销婚姻。

**撤销权除斥期间的特殊规则**（不适用民法典第152条关于5年最长除斥期间的规定）：
- 重大疾病：从**知道或应当知道**撤销事由之日起1年
- 胁迫：从**胁迫行为终止之日**起1年
- 被非法限制人身自由的：从**恢复人身自由之日**起1年

**撤销机关**：**法院**（不是民政部门）。

> **胁迫可撤销例题（2024年真题）** ：李某父亲向张某借高利贷无力偿还，被张某殴打。张某称“要不还钱，要不让李某嫁给我”。李某无奈嫁给张某，并生育一女。六年后李某父亲死亡。
>
> **问**：下列说法正确的是？
>
> A. 李某母亲可作为利害关系人申请撤销婚姻关系——**错**（可撤销婚姻只有本人可以）
>
> B. 李某与张某离婚对其女儿的父母关系无影响——**正确**（离婚不影响亲子关系）
>
> C. 此婚姻关系虽然经过五年，仍可以撤销——**正确**（不适用5年最长除斥期间）
>
> D. 李某可以向张某要求损害赔偿——**正确**（胁迫方有过错，应赔偿，包括物质和精神损害赔偿）
>
> **答案：B、C、D**


### 第五段：无效/可撤销婚姻的法律后果

**无效或被撤销的后果**（民法典第1054条）：
1. **自始无效**（自始没有法律约束力）
2. **不影响亲子关系**（当事人所生的子女，适用本法关于父母子女的规定）
3. **无过错方有权请求损害赔偿**（包括**物质损害赔偿**和**精神损害赔偿**）

**无效婚 vs 可撤销婚的对比**（讲稿强调：无效情形严重违法或违背公序良俗，利害关系人可参与；可撤销只影响当事人自己利益，只有本人可以行使）


### 第六段：家庭关系——家事代理权

**日常家事代理权**（民法典第1060条）：夫妻一方因家庭日常生活需要而实施的民事法律行为，对夫妻双方发生效力。

**内部限制不得对抗善意相对人**：夫妻之间对一方可以实施的民事法律行为范围的限制，**不得对抗善意相对人**。

> **家事代理权例题**：夫妻约定“超过10万元需双方协商”，妻子花20万买黄金未与丈夫商量。
>
> **问**：丈夫能否主张该买卖合同无效？
>
> **答**：**不能**。内部限制不得对抗善意相对人（内外有别）。


### 第七段：亲子关系

**子女的范围**：婚生子女、非婚生子女、养子女、有抚养关系的继子女（最广义的子女概念）。

**亲子关系的确认（非婚生子女的认领）** ：非婚生子女的生父或生母可以向法院提起确认亲子关系之诉（民法典第1073条）。

**亲子关系的否认（婚生子女的否认）** ：对亲子关系有异议且有正当理由的，父或母可以向法院提起否认亲子关系之诉。

**成年子女的限制**：成年子女只能提起**确认**亲子关系之诉，**不能提起否认**亲子关系之诉（权利与义务相统一原则，民法典第1073条第2款，**新增，从未考过**）。

**试管婴儿**（**新增，从未考过**）：婚姻关系存续期间，夫妻双方一致同意进行人工受精，所生子女应**视为婚生子女**（民法典婚姻家庭编司法解释一第40条）。


### 第八段：双方均无配偶的同居关系（**新增，从未考过**）

民法典婚姻家庭编司法解释二第4条（**新增，从未考过**）：

**单纯的解除同居关系**：法院**不予受理**（前面讲过，同居关系本身不受民法调整）。

**财产纠纷处理**（法院应当受理）：
- **原则**：各自归各自所有（工资、奖金、劳务报酬、知识产权收益、继承或受赠财产、单独生产经营投资收益等）。
- **例外**：共同出资购置的财产或共同生产经营投资的收益 → 以**出资比例**为基础，综合考虑共同生活情况、有无共同子女、对财产贡献大小等因素分割（**协力理论**）。


### 第九段：夫妻财产制（核心考点★★★★★）

夫妻财产制分两类：**法定夫妻财产制**（绝大多数家庭适用）和**约定夫妻财产制**（书面约定，很少见）。考试重点考法定夫妻财产制。

#### （一）法定共同财产（民法典第1062条）

| 类型 | 认定规则 | 法典依据 |
| :--- | :--- | :--- |
| 工资、奖金、劳务报酬 | 婚姻关系存续期间取得 → 共同财产 | 第1062条第1款第1项 |
| 生产经营、投资收益 | 婚姻关系存续期间取得 → 共同财产（炒股赚的是共同财产） | 第1062条第1款第2项 |
| 知识产权收益 | 发表或获得稿酬之一发生在婚姻关系存续期间 → 共同财产 | 第1062条第1款第3项 |
| 继承或受赠的财产 | 原则上共同财产；**遗嘱或赠与合同确定只归一方**的除外 | 第1062条第1款第4项 |

> **知识产权收益例题**：老孟婚前写书，婚后第10天收到稿费5万——**夫妻共同财产**（获得稿酬发生在婚姻关系存续期间）。
>
> 老孟婚后写书发表，离婚后第5天收到稿费——**夫妻共同财产**（发表发生在婚姻关系存续期间）。

> **继承财产放弃（民法典婚姻家庭编司法解释二第11条，**新增，从未考过**）** ：夫妻一方以另一方放弃继承侵害夫妻共同财产利益为由主张放弃无效的，法院**不予支持**（原则）。但有证据证明放弃继承导致放弃方不能履行**法定扶养义务**的除外（例外，保护生存权）。

#### （二）父母为子女出资购房（**新增，从未考过**，民法典婚姻家庭编司法解释二第8条）

**三个基本原则**：
1. **协力理论**——看谁对家庭的贡献大
2. **伦理道德**——谁家的孩子谁的父母爱
3. **禁止不劳而获**——不能以结婚为名行骗财之实

**第一种：一方父母全额出资**
- 有约定从约定
- 无约定 → 法院可以判决房屋归**出资人子女一方所有**，综合考虑共同生活及孕育子女情况、离婚过错、家庭贡献大小、房屋市场价格等因素，确定是否补偿及补偿数额（**物债两分区分原则**：一个拿房，一个拿钱）。

> **一方父母全额出资例题**：刘某父母出资3000万在北京买房，登记在刘某名下，老孟和刘某结婚两年未生子即离婚。老孟能否分走一半？
>
> **答**：**不能**。房屋归刘某所有，老孟可酌情获得补偿（协力理论）。

**第二种：双方父母出资或一方父母部分出资**
- 有约定从约定
- 无约定 → 以**出资来源及比例为基础**，综合考虑共同生活及孕育子女情况、离婚过错、家庭贡献大小、房屋市场价格等因素，判决房屋归其中一方所有，另一方获得合理补偿（**物债两分区分原则**）。

> **双方父母出资例题**：张三父母出800万，李四父母出200万买房，婚后张三出轨导致离婚（张三有过错）。法院可判房屋归**李四**所有（出钱少的反而拿房），张三获得较少补偿或不补偿（协力理论 + 过错考量）。

#### （三）法定个人财产（民法典第1063条）

1. 婚前财产（不会因结婚而转化为共同财产）
2. 因人身损害获得的赔偿金或补偿金
3. 遗嘱或赠与合同中确定只归一方的财产
4. 一方专用的生活用品
5. 其他应当归一方的财产

**婚前财产在婚后产生的收益**（民法典婚姻家庭编司法解释一第26条）：除**孳息**和**自然增值**外，应当认定为夫妻共同财产。

> **注意**：这里的“孳息”不同于物权编中的孳息，限于**无需夫妻一方投入物力、劳动力**即可取得的孳息。房屋租金（婚后出租）属于**经营性收益**，可认定为夫妻共同财产（因为需要投入管理劳动）。

### 第一段：法定个人财产——婚前财产与夫妻之间赠与房屋（核心考点★★★★★）

法定个人财产，法律规定了六种情形，最重要的还是第一种——**一方的婚前财产**（民法典第1063条）。

一方的婚前财产不会因为结婚而自动转化为夫妻共同财产，但实践中不可避免可能对对方实施赠与。民法典婚姻家庭编司法解释二第5条，夫妻之间的赠与房屋问题，**2025年考过**。

**本质**：①**协力理论**（谁对家庭的贡献大）；②**禁止不劳而获**（不能以结婚为名骗婚）。基本规则：**物债两分区分原则**——一个拿房，一个拿钱。

**第一种情况：房屋尚未转移登记**

婚前或婚姻关系存续期间，当事人约定将一方所有的房屋转移登记至另一方或双方名下，离婚诉讼时**尚未办理转移登记**的，人民法院可以根据当事人的诉讼请求，结合给予目的，综合考虑婚姻关系存续时间、共同生活及孕育子女情况、离婚过错、对家庭贡献大小、房屋市场价格等因素，判决房屋归其中一方所有，并确定补偿数额。

> **未登记例题**：张三婚前有房，结婚后答应给李四加名，但一直未办理。双方共同生活十年，生育两个子女，李四付出较多。离婚时张三反悔不给房。
>
> **答**：法院可判决房屋归**李四**所有，由李四给张三适当补偿（协力理论，李四贡献大）。

**第二种情况：房屋已转移登记（2025年考过）**

婚前或婚姻关系存续期间，一方将其所有的房屋转移登记至另一方或双方名下，离婚诉讼中，如果**婚姻关系存续时间较短**且**给予方无重大过错**，法院可以判决该房屋归**给予方所有**（还回去）。

> **2025年真题（职业骗婚）** ：张南自购房屋，向王女求婚时承诺“若双方结婚即为王女加名”。婚后张南立即履行承诺，将房屋登记至双方名下。**三个月后**，张南发现王女系职业骗婚人，自己是其第六任丈夫，遂起诉离婚。
>
> **问**：下列哪些选项正确？
>
> A. 双方的婚姻可撤销——**错**（可撤销婚姻只有两种：重大疾病婚前未告知、胁迫）
>
> B. 张南可请求撤销为王女的加名行为——**正确**（把房子要回来）
>
> C. 双方的婚姻无效——**错**（无效婚姻只有三种：重婚、未达婚龄、禁婚亲属）
>
> D. 法院可判决该房屋归张南所有——**正确**（婚姻存续时间短+给予方无重大过错）
>
> **答案：B、D**


### 第二段：法定个人财产的其他类型

| 类型 | 说明 | 法典依据 |
| :--- | :--- | :--- |
| 一方婚前财产 | 不因结婚转化为共同财产 | 第1063条第1项 |
| 人身损害赔偿/补偿金 | 人身专属性（如被暴打的医药费） | 第1063条第2项 |
| 遗嘱/赠与确定只归一方的财产 | 有约定从约定 | 第1063条第3项 |
| 专用生活用品等 | 其他 | 第1063条第4-6项 |

**重点掌握前三项**。


### 第三段：约定夫妻财产制

民法典第1065条：约定夫妻财产制必须采用**书面形式**（要证据，否则容易坑外部第三人）。


### 第四段：三步判断财产归属法（方法论★★★★★）

题目问“这个财产是谁的”：

1. **第一步**：看有没有**书面约定**。有 → 按约定。考试一般没有。
2. **第二步**：判断是否属于**个人财产**——看有没有**人身专属性**。有 → 个人财产。
3. **第三步**：非个人财产 → 通通认定为**夫妻共同财产**。

> 人身专属性判断：人身损害赔偿金、遗嘱/赠与确定只归一方的财产等。


### 第五段：婚姻关系存续期间共同财产的处理和分割

**日常需要**：有平等的家事代理权（买盆买碗、柴米油盐）。

**非日常需要**（买房买车等重大财产处置）——须**夫妻双方一致同意**。一方擅自处分，构成**无权处分**，相对人可能**善意取得**。

**夫妻之间借款协议**（民法典婚姻家庭编司法解释二第6条、第7条）：以夫妻共同财产出借给一方从事个人经营或用于其他个人事务的，离婚时可以按照**借款协议**处理。

> **借款协议例题**：老公想炒股，跟老婆签借款协议借100万，结果亏光。离婚时怎么处理？
>
> **答**：按借款协议处理，老公须偿还老婆100万。

**婚内分割共同财产的例外**（民法典第1066条）：

1. 一方有**隐匿、转移、变卖、毁损、挥霍**夫妻共同财产，或伪造夫妻共同债务等严重损害夫妻共同财产利益的行为。
2. 一方负有法定扶养义务的人患有重大疾病需要医治，另一方不同意支付相关医疗费用。

**挥霍——网络打赏**（婚姻家庭编司法解释二第6条，**新增，从未考过**）：夫妻一方未经另一方同意，在网络直播平台用夫妻共同财产打赏，数额**明显超出其家庭一般消费水平**，严重损害夫妻共同财产利益的，可认定为**挥霍**。另一方可在不离婚的情况下分割夫妻共同财产，或在离婚时主张过错方**少分或不分**。

> **网络打赏例题**：马峰月收入1万，打赏网络主播100万。
>
> **问**：是否构成挥霍夫妻共同财产？
>
> **答**：**构成**。数额明显超出家庭一般消费水平，严重损害夫妻共同财产利益。

**转移——第三者赠与**（婚姻家庭编司法解释二第7条，**新增，从未考过**）：夫妻一方为重婚、与他人同居等目的，将夫妻共同财产赠与他人或明显不合理的价格处分，另一方主张该行为违背公序良俗无效的，法院**应予支持**。另一方可在不离婚的情况下分割夫妻共同财产，或在离婚时主张过错方**少分或不分**。


### 第六段：夫妻债务的认定（主观题重点★★★★★）

**主观题在婚姻家庭部分重点就考一个问题：夫妻债务的认定。**

**判断标准**：**客观判断标准**——看借来的钱**用于干什么**（目的论和功能论）。

| 债务用途 | 性质 | 后果 |
| :--- | :--- | :--- |
| **共同生活**（装修婚房、给孩子报辅导班、治病等） | **夫妻共同债务** | 连带清偿 |
| **共同经营**（夫妻店等） | **夫妻共同债务** | 连带清偿 |
| **共同意思表示**（共债共签） | **夫妻共同债务** | 连带清偿 |
| **个人挥霍**（赌博、吸毒等） | **个人债务** | 独自清偿，与配偶无关 |

> **夫妻债务例题**：薛女与江南离婚后，发现江南隐藏婚后股票100万，且江南平时喜好赌博欠款50万。
>
> **问**：50万赌博债务如何承担？
>
> **答**：**江南个人债务**，与薛女无关（用于个人挥霍）。


### 第七段：夫妻债务的清偿

夫妻共同债务 → 共同偿还（**连带责任**）。

离婚协议或法院判决/调解书关于债务承担的约定，**只在双方之间产生约束力**，对债权人没有法律约束力（**内外有别**）。

> **清偿例题**：王某以自己名义向赵某借款3000元给张某治病（夫妻共同债务），后王某与张某离婚，约定由张某负责偿还。赵某能否找王某要债？
>
> **答**：**可以**。内部分担约定对债权人无约束力，王某和张某承担连带责任。


### 第八段：协议离婚——冷静期制度（民法典第1077条）

**亲自前往民政部门**申请离婚登记（不得代理）。

**冷静期**：
1. 申请后**30天**冷静期。任何一方反悔可**撤回**（撤回=尚未生效的意思表示）。冷静期内双方仍是夫妻关系。
2. 冷静期届满后**30天内**，双方**亲自**前往领取离婚证。一方不配合或没去 → **视为撤回**（没离成）。


### 第九段：诉讼离婚

**调解前置**：法院审理离婚案件**应当进行调解**（民法典第1079条）。

**判决离婚的法定情形**（七种，“过不下去了”）：

| 序号 | 法定情形 | 说明 |
| :--- | :--- | :--- |
| 1 | 重婚或与他人同居 | 严重挑衅婚姻关系 |
| 2 | 实施家庭暴力、虐待、遗弃家庭成员 | — |
| 3 | 有赌博、吸毒等恶习屡教不改 | 强调“屡教不改” |
| 4 | 感情不和分居满**两年** | 重点：感情不和，不是分居本身 |
| 5 | 一方被宣告失踪 | 另一方提起离婚诉讼，应当准许 |
| 6 | 生育问题达不成一致 | 女方擅自终止妊娠等 |
| 7 | 判决不准离婚后，双方**又分居满一年** | **新增，从未考过**，再次起诉应当准许 |

**特殊婚姻保护**：
- **军婚**：了解即可，很少考。
- **女方三期保护**（民法典第1082条）：女方**怀孕期间、分娩后一年内、终止妊娠后六个月内**，男方**不准提离婚**（例外：女方提出；法院认为确有必要受理——如“喜当爹”）。


### 第十段：离婚过错损害赔偿（民法典第1091条）

**法定情形**：重婚、与他人同居、实施家庭暴力、虐待遗弃家庭成员、**其他重大过错**（通奸、卖淫、嫖娼、赌博、吸毒等严重违反婚姻家庭义务的行为）。

**责任人**：**过错方**（与第三人无关，第三人不用赔）。

**请求权人**：**无过错方**（双方都有过错 → 不予支持）。

**赔偿内容**：**物质损害赔偿 + 精神损害赔偿**（侵犯配偶权）。

**以离婚为前提**：法院判决不准离婚的，对损害赔偿请求不予支持。


### 第十一段：抢孩子问题（新增，从未考过★★★★★）

父母一方或其近亲属抢夺、隐匿未成年人子女，另一方可申请：
- **人身安全保护令** 或
- **人格权侵害禁令**（民法典第997条，预先保护制度）

**法律效果**：
- 抢孩子是**侵权行为**（侵犯另一方监护权），须停止侵权、把孩子还回来
- 法院可指定孩子临时照顾方案
- **谁抢孩子，对谁做不利认定**——优先考虑不抢孩子一方直接抚养


### 第十二段：子女抚养权确定

| 子女年龄 | 规则 |
| :--- | :--- |
| **不满2周岁** | **原则上随母亲**（哺乳期，例外：母亲有疾病等不宜抚养情形） |
| **2-8周岁** | 双方协商 → 协商不成，法院根据**最有利于未成年人**原则判决 |
| **8周岁以上** | **尊重其真实意愿**（询问意见，但不以其意见为准） |


### 第十三段：探望权

**性质**：探望是**权利**，不是义务。不探望不能告。

**协助义务**：直接抚养方有协助义务；拒不协助可采取**拘留、罚款**等强制措施，但**不得对孩子采取强制措施**。

**终止与恢复**：如一方染上吸毒等恶习，可**暂停**探望权；改好后可**恢复**。终止和恢复均由**法院**决定。


### 第十四段：父母子女关系——姓氏更改

离婚后一方擅自给孩子改姓，另一方**仍须支付抚养费**（抚养费保护弱势群体生存权，不因改姓而免除）。救济途径：请求法院**责令恢复原姓氏**。

> **姓氏更改例题（2023年真题）** ：甲乙离婚，丙随母亲乙生活。后乙嫁给丁，将丙名字改为随丁姓戊。甲发现后，丙欲上私立学校需增加抚养费。
>
> **问**：下列说法正确的是？
>
> A. 甲有权因乙为丙改了名字而不支付抚养费——**错**（抚养费不因改姓免除）
>
> B. 乙为丙改了名字，则甲丙之间的监护关系终止——**错**（监护关系不因改姓终止）
>
> C. 乙应为甲探望丙提供便利——**正确**
>
> D. 丙有权起诉请求甲增加抚养费——**正确**
>
> **答案：C、D**


### 第十五段：离婚后抚养费的给付

**离婚协议约定一方不负担抚养费的**（民法典婚姻家庭编司法解释二第16条，**新增，从未考过**）：
- 对双方具有法律约束力（当事人意思自治）
- 但若直接抚养方经济状况变化导致原生活水平显著降低，或子女生活教育医疗等必要费用增加，另一方可请求**支付抚养费**（参照情势变更）
- 抚养费**不受诉讼时效期间限制**


### 第十六段：离婚经济补偿与离婚经济帮助

**经济补偿**（民法典第1088条）：夫妻一方因抚养子女、照料老人、协助另一方工作等**付出较多义务**的，离婚时可请求**补偿**（道义性责任，协力理论应用）。

**经济帮助**（民法典第1090条）：离婚时一方**生活困难**（缺乏劳动能力、无生活来源、身体疾病），有负担能力的另一方应当给予**适当帮助**（居住权的应用场景之一——设置居住权直至去世）。

**经济补偿 vs 经济帮助**：
- 补偿：基于**对家庭的贡献**（协力理论）
- 帮助：基于**生活困难**（人道主义，一日夫妻百日恩）


### 第十七段：离婚过错财产分割

一方有**隐匿、转移、变卖、毁损、挥霍**夫妻共同财产或伪造夫妻共同债务的，可以**少分或不分**（民法典第1092条）。

**再次分割**：离婚后发现另一方有上述行为的，可请求**再次分割**（诉讼时效：**三年**，自**发现之日**起算）。


### 第十八段：夫妻股权分割——看出资不看登记（新增，从未考过）

夫妻以共同财产投资有限责任公司，并**均登记为股东**，离婚时一方请求按照股东名册或公司章程记载的各自出资额确定股权分割比例的，法院**不予支持**（民法典婚姻家庭编司法解释二第10条）。

**核心规则**：**夫妻之间看出资，不看登记**（内外有别：内部看出资，外部看登记）。


### 第十九段：附生效条件的离婚财产处理协议

当事人达成的以**协议离婚或到法院调解离婚为条件**的财产及债务处理协议，如果双方离婚未成，一方在离婚诉讼中反悔的，法院应当认定该协议**没有生效**（民法典婚姻家庭编司法解释一第69条）。

> **附条件离婚协议例题**：张三对李四说“如果你配合我协议离婚，我净身出户”。双方签字后，张三反悔了，转而诉讼离婚。
>
> **问**：李四能否要求张三净身出户？
>
> **答**：**不能**。条件未成就，协议未生效，依法分割财产。
