# 民法讲稿整理与提纲对照（民事法律关系基本原理·Markdown版）


## 第一部分：讲稿内容切分与修正

### 第一段：总则编体系概览

还得技术流，每一科你都值得更好的。

好，接下来咱们正式进入知识的学习，先来看第一部分——总则编。在书的专题一的前页，有一个总则编的体系。民法典总则编在考试当中，客观题一共考八分左右，一共是两个大的制度，六个小的制度。

这两个大的制度：
- **民事法律关系**，考三分左右
- **民事法律事实**，考五分左右

民事法律关系重点考察民事主体——自然人、法人和非法人组织，考试重点是自然人和法人。民事法律事实项下有三个制度：民事法律行为、代理和诉讼时效。

从主观题考试角度来看，总则编一共二百零四个条文，有十章。重点要关注的是第二章自然人、第六章民事法律行为和第七章代理制度。到了主观题阶段，重点是第六章民事法律行为和第七章代理，在旁边写上"主观题"。

### 第二段：民事法律关系——民法的调整对象（正面）

接下来我们先看总则编的第一个核心概念——**民事法律关系**。把书翻到专题一，民事法律关系的基本原理。这个专题一共就两个考点，一个重点掌握，一个了解。

考试方式：选择题，考**民事法律关系与非民事法律关系的区别**，也就是民法的调整对象。从法理学的角度来看，法律部门之间的区分标准是调整对象和调整方法，首要的是调整对象。我们要解决的第一个问题是：哪些事儿归民法管，哪些事儿不归民法管。

从正面而言，**民法典第二条**说得非常清楚：

> 民法调整**平等主体**的**自然人、法人、非法人组织**之间的**人身关系**和**财产关系**。

四个方面特点：
1. 平等主体
2. 之间的
3. 人身关系
4. 财产关系

### 第三段：考点一——平等性（不以主体论关系）

考试从来不考正面，考反面例外。一一对应四个特点。

**第一个考点：平等性。** 有关单位在从事管理活动时产生的社会关系，不受民法调整。

如何去区分民法和行政法？不能只看主体，而要看他的行为。考试常会给一个行政机关（公安局、税务局、交管局、城管局等），看起来像行政法，但其实从事的是民事活动。

**原则**：行政机关做出的行政管理活动，肯定是行政法律关系。比如违章停车被罚款、偷逃税款被税务机关罚款，都是行政法律关系。

**例外**：行政机关也可能会从事民商事活动。比如税务局、公安局、财政局去购买办公用品（买卖合同），或建办公大楼（建设工程施工合同）。如果产生纠纷，由民庭来管，这是民事活动。

**结论：不以主体论关系，而以行为论关系。** 千万不要看到行政机关就激动地说"行政法律关系"，不一定，得看他干了什么事儿。

### 第四段：考点二——"之间的"（自己内部的事不管）

**第二个考点：所谓的"之间的"。** 人自己的活动，民法不管。

从自然人的角度：几点起床、中午要不要午睡、下午要不要喝下午茶，法律不管，那是生活问题、习惯问题。

从法人的角度：总公司和分公司是一家公司，这是考试重点。比如中国工商银行和中国工商银行北京市分行是一家银行。总行对分行下达业务指导指令、放贷指标、行为规范，不具有可诉性，不归民法管。

旁边写上：**总分公司是一家公司**，伴随法考始终的重点。

### 第五段：考点三——不能产生司法效果的人身关系

**第三个考点：人身关系要能产生法律效力的才归民法管。**

自然人活在社会中，会和不同的人形成各种各样的社会关系，但不是所有的关系都是民法管的。比如师生关系、朋友关系、同事关系，往往受道德调整，不受法律调整。

考试重点是四个关系：**恋爱关系、同居关系、朋友关系、网友关系**。重点考察前两个。

**第一，恋爱关系。** 比如"你考上研究生我就嫁给你"，考上了对方不嫁，能不能去法院告？不可以，不具有可诉性。谈恋爱的事儿法院管不了，也没法强制执行。

**第二，同居关系（重点）。** 单纯的解除同居关系，不具有可诉性，民法不管，跟谈恋爱没啥区别。

**但有例外**：同居期间产生的**财产分割纠纷**或**子女抚养纠纷**，法院应当受理。孩子是无辜的，非婚生子女的抚养问题，法院得管。书上第六页，民法典婚姻家庭编司法解释一第三条，重点掌握。

**第三、第四（了解）**：朋友关系——朋友之间传递小道消息，买股票亏了，不能要求赔偿，风险自担。网友关系——网上交朋友、见面、打游戏、喝下午茶，法律不管。

### 第六段：考点四——不能产生司法效果的财产关系（好意施惠）

**第四个考点：不能产生司法效果的财产关系——好意施惠。**

**好意施惠**，又叫君子协议、情谊行为。通俗而言：**给你是情分，不给你是本分**。人家没有义务来给你。之所以实施这个行为，是为了增进同学关系、邻里关系、师生关系，是人际交往问题，是人情世故。

典型的好意施惠有七种（不完全列举）：

1. **邀请同看演出比赛或承诺陪同旅游**——放鸽子（爽约），不能要求赔偿。
2. **火车到站叫醒**——你没有义务叫我，叫了是人情，没叫不能要求赔偿。
3. **顺路投递邮件或顺便帮邻居清扫积雪**——顺带手的事儿，不构成无因管理，不产生赔偿问题。
4. **无偿指路**——指错了路，不能要求赔偿。我又不是高德地图。
5. **青年志愿者做帮工**——答应了后来爽约了，也不能要求赔偿。
6. **请客吃饭（重点）**
7. **搭便车（搭顺风车）（重点）**

### 第七段：请客吃饭的三段风险

请客吃饭分为前端、中端、后端。

**前端——爽约（放鸽子）**：约好了请客，结果爽约了。被爽约的人能不能要求赔偿？不能。不能追究违约责任，也不能追究缔约过失责任，也不能追究侵权责任。不管是哪一方爽约，都不能让对方赔偿。

**中端——劝酒**：去了，吃饭过程中极力劝酒、恶意劝酒，结果喝死了、喝伤了。**需要赔偿**。这是一般侵权，符合侵权行为四个要件：侵害行为、损害后果、因果关系、主观过错。

**后端——酒后照顾（考察概率更高）**：

- **一方醉酒，对方未照顾**：你跟别人喝酒，对方喝得烂醉，你有照顾他的义务（善良管理人的注意义务）。如果他出事了，你要承担**赔偿责任**（一般侵权）。
- **双方都醉酒，互相未照顾**：各自回家，一方出事了（比如掉河里淹死了）。此时是**补偿责任**，不是赔偿责任。补偿只是道义性责任，用的是公平原则，也就补偿几百、几千块钱。

### 第八段：搭便车的三段风险与过错责任

搭便车也分前端、中端、后端。

**前端——爽约**：答应载人结果爽约了，不能要求赔偿。

**中端/后端——出事故**：

- **驾车人单独过错**：比如超速行驶，急刹车导致乘车人受伤。**要赔偿**。但因为是无偿搭载，可以减轻责任（比如医药费两千，赔一千五）。
- **故意或重大过失**：比如玩漂移导致乘车人死亡。**全赔**，一毛钱责任都不能免。
- **驾车人与第三人都有过错**：驾车人超速，第三人违章变道，两车相撞，乘车人受伤。驾车人和第三人对乘车人承担**按份责任**，按过错大小分担。
- **乘车人也有过错**：比如乘车人指挥驾车人加速。受害人也有过错，相应减轻加害人责任。

总结一句话：**谁的过错谁担责**。三个主体（搭车人、驾车人、第三人），谁有过错谁承担。

### 第九段：戏谑行为

**戏谑行为**——说大话、吹牛皮、开玩笑，不产生法律后果。

经典例题：世界陶艺大师邢某在央视采访时拿出作品"五层吊球"，夸下海口："如果有人仿造出来，就把自己位于大连市中心的三层小楼，价值一千六百万的艺术中心送给他。"结果河南洛阳一陶瓷爱好者孙某真的仿造出来了，告到法院。

**结论**：这是戏谑行为，开玩笑说大话，不能要求履行，小楼不能给。

类似的：酒桌上说"你裸奔三圈，我的车就是你的"，当真了就属于不懂人情世故，不能要求履行。

### 第十段：民事法律关系的要素（了解级别）

第十页第二个考点，**民事法律关系的要素**。了解级别。

民事法律关系包括三个要素——**人、物、关系**：

- **人——民事主体**：自然人、法人、非法人组织
- **物——民事客体**：权利所指向的对象
- **关系——民事内容**：民事权利和民事义务

**民事权利**包括财产权（物权、债权）、人身权（人格权、身份权）、综合性权利（继承权、知识产权、股权等）。

**民事义务**：有权利就有义务。比如老孟对手表有所有权，你的义务就是不上来偷、不上来抢。

### 第十一段：民事责任优先原则

**民事责任**——违反法律义务产生的法律后果。

法律责任包括刑事责任、行政责任、民事责任。

**重点考点：民法典第一百八十七条——民事责任优先原则。**

如果同一个义务人既要承担刑事责任（没收财产），又要承担行政责任（罚款），还要承担民事责任（赔偿），结果钱不够咋办？

**刑事责任、行政责任的没收财产和罚款都是归国家的，民事赔偿是归受害人的。俗话说"国不与民争利"，在钱不够的情况下，民事责任优先。**

三个责任互相独立，互不干涉。承担了民事赔偿责任，不能免除刑事责任；承担了刑事责任，也不能免除民事责任。

### 第十二段：民事客体（了解）

民事客体——权利所指向的对象。

- **物权的客体**：特定化的物（如手表）。例外是权利（如股权质押）。
- **债权的客体**：**行为**。正常情况是积极行为（如交表、付钱）。例外是消极不作为（如竞业禁止协议）。
- 人身权的客体：人身利益。
- 知识产权的客体：智力成果。
- 继承权的客体：遗产（财产包，包括房子、车、股权、字画等）。

### 第十三段：本专题总结

至此，第一个专题"民事法律关系的基本原理"全部讲完。这个专题是入门级的，唯一能考到的重点就是——**民事法律关系与非民事法律关系的区别，也就是民法的调整对象**。

把民法典第二条背三遍：

> 民法调整平等主体的自然人、法人、非法人组织之间的人身关系和财产关系。

这个专题需要掌握的专业术语：
- **好意施惠**（重点掌握）
- **戏谑行为**（了解即可）


## 第二部分：对照《民法典》"民事法律关系的基本原理"提取备考提纲

> **说明**：本提纲基于讲稿内容，对照民法典总则编相关框架整理，**未自造法律概念**，超出讲稿范围或不确定处已标注。


### 一、总则编体系概览（讲稿第一段）

民法典总则编在法考客观题中约**8分**：

| 两大制度 | 分值 | 下设内容 |
| :--- | :--- | :--- |
| **民事法律关系** | 约3分 | 民事主体（自然人、法人、非法人组织） |
| **民事法律事实** | 约5分 | 民事法律行为、代理、诉讼时效 |

**主观题重点章节**：第二章自然人、第六章民事法律行为、第七章代理制度。


### 二、民法的调整对象——民法典第二条（讲稿第二至第九段）

**民法典第2条**：民法调整**平等主体**的**自然人、法人、非法人组织**之间的**人身关系**和**财产关系**。

法考考查方式：给出一个社会关系，判断是否受民法调整（民事法律关系 vs 非民事法律关系）。

#### （一）平等性——不以主体论关系（讲稿第三段）

| 情形 | 是否受民法调整 | 理由 |
| :--- | :--- | :--- |
| 行政机关作出行政管理活动（如罚款） | **否**（行政法律关系） | 非平等主体，管理/被管理关系 |
| 行政机关购买办公用品、签建设工程施工合同 | **是**（民事法律关系） | 从事的是民事活动，双方平等 |

> **核心判断标准（讲稿归纳）**：不以主体论关系，**以行为论关系**。看到行政机关≠行政法律关系，得看他干了什么事儿。

#### （二）"之间的"——自己内部的事不管（讲稿第四段）

| 情形 | 是否受民法调整 | 理由 |
| :--- | :--- | :--- |
| 自然人日常生活习惯（几点起床、喝下午茶） | **否** | 人自己的活动，法律不干预 |
| 总行对分行下达业务指导指令 | **否** | 总公司和分公司是一家公司，内部业务指导不具有可诉性 |

> **要点（讲稿归纳）**：总分公司是一家公司，自己内部的事不受民法调整。

#### （三）人身关系——要能产生司法效果（讲稿第五段）

| 情形 | 是否受民法调整 | 说明 |
| :--- | :--- | :--- |
| **恋爱关系**（如"考上了就嫁给你"反悔） | **否** | 不具有可诉性，法院管不了谈恋爱 |
| **同居关系**——单纯解除同居关系 | **否** | 跟谈恋爱没区别，分手就行了 |
| **同居关系**——财产分割或子女抚养纠纷 | **是（例外）** | 法院应当受理（婚姻家庭编解释一第3条） |
| **朋友关系**（传递小道消息导致投资亏损） | **否** | 风险自担，自己为自己的行为负责 |
| **网友关系** | **否** | 法律不管 |

> **法考提示（讲稿）**：重点掌握同居关系的例外——财产分割纠纷和子女抚养纠纷，法院应当受理。

#### （四）财产关系——好意施惠（讲稿第六至第八段）

**好意施惠**（又叫君子协议、情谊行为）：给你是情分，不给你是本分。不产生违约责任或缔约过失责任，但**可能产生侵权责任**。

| 典型情形 | 是否产生赔偿责任 | 说明 |
| :--- | :--- | :--- |
| **爽约（放鸽子）**——约好了请客/搭车，结果爽约 | **否** | 不产生违约责任、缔约过失责任、侵权责任 |
| **劝酒**——极力劝酒、恶意劝酒导致死伤 | **是（赔偿）** | 一般侵权，四要件齐全 |
| **酒后照顾**——一方醉酒，对方未照顾导致损害 | **是（赔偿）** | 有照顾义务（善良管理人注意义务），不履行则侵权 |
| **酒后照顾**——双方都醉酒，各自回家，一方出事 | **是（补偿）** | 补偿（道义性责任，公平原则），不是赔偿 |
| **搭便车**——驾车人有过错（如超速） | **是（赔偿），可减轻** | 因无偿搭载，可减轻驾车人责任 |
| **搭便车**——驾车人故意或重大过失 | **是（赔偿），不减轻** | 故意或重大过失不能减轻责任 |
| **搭便车**——驾车人与第三人都有过错 | **是（按份责任）** | 按过错大小分担 |
| **搭便车**——乘车人也有过错 | **减轻加害人责任** | 受害人有过错，减轻加害人责任 |

> **讲稿核心归纳**：谁的过错谁担责。好意施惠本身不产生合同责任，但可能产生侵权责任。

#### （五）戏谑行为（讲稿第九段）

**戏谑行为**：说大话、吹牛皮、开玩笑，**不产生法律后果**。

| 情形 | 是否产生法律后果 | 说明 |
| :--- | :--- | :--- |
| 酒桌上说"你裸奔三圈，我车归你" | **否** | 开玩笑，不能当真 |
| "我书里有一个错别字，给你十万" | **否** | 说大话，不是真实意思表示 |
| 央视采访中"夸下海口"说仿造出来就送两千平小楼 | **否** | 真实案例，认定为戏谑行为，不能要求履行 |

> **讲稿归纳**：戏谑行为"说者无意，听者有心"，但不产生法律约束力。


### 三、民事法律关系的要素（讲稿第十至第十二段）

民事法律关系包括三个要素：**主体、内容、客体**。

#### （一）民事主体（人）

**民法典第2条**：自然人、法人、非法人组织。

（后续专题二会全面展开，本讲仅作了解。）

#### （二）民事内容（权利和义务）

- **民事权利**：财产权（物权、债权）、人身权（人格权、身份权）、综合性权利（继承权、知识产权、股权等）
- **民事义务**：权利与义务相统一。没有无权利的义务，也没有无义务的权利。

#### （三）民事客体（物/权利所指向的对象）

| 权利类型 | 客体 | 说明 |
| :--- | :--- | :--- |
| **物权** | 特定化的**物** | 例外：**权利**（如股权质押） |
| **债权** | **行为** | 原则：积极行为（如交付物、支付价款）；例外：消极不作为（如竞业禁止协议） |
| **人身权** | 人身利益 | — |
| **知识产权** | 智力成果 | — |

### 四、民事责任优先原则（讲稿第十一段）

**民法典第187条**：民事主体因同一行为应当承担民事责任、行政责任和刑事责任的，承担行政责任或者刑事责任不影响承担民事责任；民事主体的财产不足以支付的，优先用于承担民事责任。

| 要点 | 讲稿解读 |
| :--- | :--- |
| **三种责任独立** | 刑事责任（刑庭）、行政责任（行政庭）、民事责任（民庭）各自独立，互不干涉。承担了民事责任≠免除刑事责任。 |
| **民事责任优先** | "国不与民争利"。钱不够时，刑事责任中的没收财产和行政责任中的罚款归国家，民事赔偿归受害人→**民事赔偿优先**。 |
