# 民法讲稿整理与提纲对照（诉讼时效专题·Markdown版）


## 第一部分：讲稿内容切分与修正

### 第一段：诉讼时效的基本概念——以借钱为例引入时间轴

还得技术流，每一科你都值得更好的。好，接下来咱们来看专题五，叫做诉讼时效。诉讼时效，就是一定时间的经过产生一定的法律效果。一个权利长时间不行使，到底会发生什么样的情况呢？

比如说举个例子，我现在找你借一万块钱，咱俩约定一年还。一年期届满以后，你三年研究生都毕业了，你都没找我要。那么这个时候法律是保护你还是保护我呢？

我们来看一个时间轴，你就明白了。比如说二零二一年一月一号，我找你借钱一万块钱，咱俩约定一年还，就是二零二二年的一月一号。那么这一年，叫做借款期间。这一年内，你不能找我要债，我也不需要还钱，因为咱俩约定一年还嘛，这还没到期呢。未到期的债权视为无债权。

那么接下来从二零二二年一月二号以后，就到期了吧，你就可以找我要债了。然后往后延三年，也就是我们的诉讼时效期间，普通的诉讼时效期间就是三年。到了二零二五年一月一号。那么这三年，我们管它叫诉讼时效期间。

### 第二段：诉讼时效期间内的法律状态——保护债权人

在这绿色的三年诉讼时效期间内，法律肯定要优先保护你权利人债权。你这三年内随时随地都可以找我要债，你只要找我要债，我就得还钱。我不还钱你把我告法院去就完了，法院肯定要支持你的诉讼请求。

但问题是，你太懒了，你怠于行权这三年，你啥也没干。你既不给我打电话，也不给我发短信，也不给我发邮件，也不去找村长解决纠纷，也不去镇政府的调解委员会，哪儿都不去，三年啥也不干。

### 第三段：诉讼时效期间届满后的法律后果——自然之债与保护债务人

结果麻烦了。从二零二五年一月二号以后，就是诉讼时效期间届满以后。法律前面已经优先保护你债权了三年，你都怠于行权。那这个时候，我欠的一万块钱，我们给它起了一个专业术语叫**自然之债**。

什么叫自然之债呢？债权本身依然存在，但是它**丧失了人民法院的强制执行力**。也就是说到蓝色的这个部分的时候，我们就开始**保护债务人利益**。为什么要保护债务人？是因为债权人怠于行权，所以法律要惩罚"睡眠在权利上的人"。

所以这个专题背后的法理与逻辑，就是因为权利人怠于行权，因此我们要去保护债务人的利益。大家每次看到"诉讼时效利益"的时候，你要明白它是保护债务人利益的。

### 第四段：诉讼时效的体系与法定性

书的七十五页下方有个诉讼时效的体系。诉讼时效就是时间的经过，它本质上是一个**自然事件**问题。立法背后为什么要保护债务人？是因为债权人怠于行权，惩罚睡眠在权利上的人。

在借款合同订立之时，债权人是处于优势地位的。既然你处于优势地位，你完全可能在订立借款合同的时候，排除法律对于我债务人的保护措施，就是这个诉讼时效。但是法律不允许你排除，不允许咱们做特别的约定。所以**诉讼时效一律法定**。

它一共包括五个问题：基本原理、种类、起算、中止、中断，一律都是法律规定。

### 第五段：诉讼时效的性质——强制性规定

七十六页我们先来看第一个问题，就是诉讼时效的基本原理。它的性质就是**强制性规定**，也就是一律法定的。既然是强制性规定，法律怎么规定的就按照怎么来。当事人如果敢自由创设、改变，都认定为无效。

诉讼时效的期间、计算方法以及中止、中断的适用由法律规定，当事人约定无效。当事人对诉讼时效利益的预先放弃无效。为什么都是无效？因为违反法律的强制性规定。比如法律规定的普通诉讼时效期间为三年，咱们俩能不能约定三十年？不行，超出法律规定的部分的约定是无效的。

### 第六段：适用范围（一）——债权请求权的原则

接下来看第一个重点考试对象，就是适用范围。哪些权利的形式要适用诉讼时效期间的限制呢？就是**请求权**。

所谓的请求权，是指特定权利人请求特定义务人为或不为特定行为的权利。请求权的权利人和义务人均是特定的。

从考试的角度来看，一共就这几种。第一个叫**债权请求权**，用我的话来说就是"要钱不要物"。赔偿要钱，原则上都得抓紧时间要钱，三年诉讼时效期间，原则上会受限制，但它有例外。

### 第七段：债权请求权的五个例外（一）——权利人未怠于行权

我们说的就是辩证思维，**一个原则，五个例外**。这五个例外大家不用背，我们讲背后的法理与逻辑。

诉讼时效制度是为了惩罚权利人怠于行权的。如果权利人没有怠于行权，就不能适用诉讼时效。

比如七十七页上方的一和二：

**第一，支付存款本金及利息请求权**（旁边写"银行"）。老孟把一万块钱存到银行里去，三年定期理财到期后不去取，不是因为怠于行使，而是因为想吃利息。所以权利人有怠于行使吗？没有，所以不适用诉讼时效期间的限制。

**第二，兑付国债、金融债券以及向不特定对象发行的企业债券本息请求权**。跟上面一模一样，债权人没有怠于行使权利。

### 第八段：债权请求权的五个例外（二）——缴付出资请求权与类推适用

**第三，基于投资关系产生的缴付出资请求权**（旁边写"股东"）。公司的股东，义务叫出资。按照公司法的规定，有限责任公司最长时间是五年就得把认缴的出资缴上去。假如八年过去了还没缴，公司要求缴付出资，股东能不能说"过了诉讼时效我就不缴了"？不能。

为什么？因为股东缴付出资才能保护公司利益，保护公司外部的债权人利益。股东绝对不能对于出资问题提诉讼时效抗辩权。

**类推适用：业主大会请求业主缴付公共维修资金或维修基金的权利**，可以类推适用投资关系的规则，也不适用诉讼时效的限制。为什么？因为电梯、屋顶、外墙等公共设施的维修改造，如果大家都不交，小区就没法维护了。

但要注意，**物业费请求权**属于普通的债权请求权，适用三年诉讼时效期间。物业公司每年催缴一次，诉讼时效中断一次，重新计算三年。所以实践中物业费一般不会过诉讼时效。

### 第九段：债权请求权的五个例外（三）——三费请求权与劳动报酬

**第五，三费——抚养费、赡养费、扶养费**。这是为了保护弱势群体的生存权。

比如父母离婚后，孩子跟妈妈生活，爸爸一直没给抚养费。六年后孩子能不能把爸爸告到法院要抚养费？可以。爸爸能不能提诉讼时效抗辩权？不可以。这是为了保护孩子的生存权——生活费、医疗费、教育费等。同样，赡养费也不能提诉讼时效抗辩权。

**注意：劳动者工资债权**。有同学会答错，劳动者虽然处于弱势地位，但劳动法有单独的保障机制。劳动者一旦跟用人单位产生纠纷，可以提劳动仲裁，费用非常低。所以劳动者的工资报酬请求权**适用诉讼时效期间限制**，不能躺在权利上睡觉。

总结一下：五个例外中——一和二是因为没有怠于行权；三是保护公司和债权人利益；四是保护国家利益（国有资产）；五是保护弱势群体利益（三费）。

### 第十段：适用范围（二）——物权请求权

**物权请求权**，用我的话来说就是"要物不要钱"。物权请求权是否适用诉讼时效期间限制？不可一概而论。**一个原则，两个例外**。

原则：物权请求权原则上不受诉讼时效期间限制。但有两个例外。

背后的法律与逻辑就两个字：**登记**。看标的物有没有登记制度。

**第一，不动产**。有登记制度。比如房子被霸占了五年，第六年能不能告法院要求"滚出去"？可以。因为多少年都可以去房管部门复印登记簿，证明所有权。所以不适用诉讼时效期间限制。

**第二，特殊动产**（机动车、船舶、航空器）。有登记制度。比如宝马车被抢走了，五年后发现，能不能请求返还？可以。因为多少年以后去车管所复印登记簿就能证明所有权。所以不适用诉讼时效期间限制。

**第三，未登记的动产物权**（普通动产，如书、手表、水杯、包包、衣服）。没有登记制度，一旦受到侵犯要抓紧时间要。为什么？因为没有登记制度，存在举证困难。如果长时间怠于行权，权利效力发生减损。所以**普通动产返还原物请求权适用三年诉讼时效期间**。用八个字概括："普通动产，返还原物"。

另外担保物权的问题，等到专题九（担保物权）再展开，先有个印象——抵押权受诉讼时效期间限制（民法典第四百一十九条）。

### 第十一段：适用范围（三）——人格权请求权与其他

**人格权请求权**。人格权受到侵害后，请求停止侵害、排除妨碍、消除危险，特别是消除影响、恢复名誉、赔礼道歉。这些为什么不适用诉讼时效期间限制？因为侵犯人格权时，侵权行为往往一直在持续。比如裸照被发到网上，侵权一直在持续。民法典第九百九十五条。

**离婚后的财产分割请求权**。夫妻离婚后，又发现还有财产没分。什么时候发现什么时候分。适用三年诉讼时效期间，起算点是自发现之日起起算。

总结：七十六页和七十七页最重要的就是那两处"蒙主点拨"——公共维修资金和劳动报酬，是常考点。

### 第十二段：诉讼时效经过的法律效果（一）——抗辩权发生说

七十八页，经过的法律效果。三年诉讼时效期间内权利人怠于行权，什么也没干，那产生了什么法律效果？

通过一个模型来感受。二零二一年十月十号，孟某向马某借款两千万，约定借款期限一年。

**第一问**：从二零二一年十月十号到二零二二年十月十号这借款期内，债权人可否请求还款？不行。未到期的债权视为无债权。

**第二问**：从二零二二年十月十一号到二零二五年十月十号这三年诉讼时效期间内，债权人可否请求还款？可以。

**第三问**：二零二五年十月十六号以后（诉讼时效期间届满后），债权人马某向法院起诉孟某还款。这时的两千万债权已经变成了**自然之债**。债权本身依然存在，但丧失了人民法院的强制执行力。除非债务人自愿还钱。

### 第十三段：诉讼时效经过的法律效果（二）——法院被动司法

那告到法院去以后，什么叫做**抗辩权发生说**？

**第一**：不影响权利人去起诉。人民法院应当受理，不得裁定不予受理。因为债权本身依然存在。

**第二**：法院审判庭的法官能不能主动向债务人提示："这个债过了诉讼时效了，你要不要提抗辩权？"**不可以**。民法典第一百九十三条。法官必须被动司法，不得主动释明，也不能主动适用。否则法院教债务人不还钱，有违诚信原则。

**第三**：如果债务人提了诉讼时效抗辩权，法院判决驳回债权人的诉讼请求。如果债务人不提（要么不懂，要么要脸），法院判决债权人胜诉。

所以，**诉讼时效期间届满后，只是发生了一个抗辩权给了债务人**——永久性的抗辩权。债务人行使还是不行使，完全由他自主决定。

**第四**：一审期间没提，二审期间能不能提？原则上不可以，除非有新证据（比如一审时借条没找到，二审找到了；或者证人一审没找到，二审出现了）。

### 第十四段：诉讼时效期间届满后的债务履行（一）——放弃抗辩权与同意履行

诉讼时效期间届满以后，债务人虽然享有诉讼时效抗辩权，但他可能会放弃。

看时间轴——红色部分（借款期间）：债务人主动还款，债权人不会构成不当得利。绿色部分（诉讼时效期间内）：主动还款，正常。蓝色部分（诉讼时效期间届满后，自然之债）：债务人主动还款，债权人同样不会构成不当得利。因为债权本身自始至终都是存在的。

**事后放弃**分为两种：

**第一，明示放弃（同意履行）**。比如过时效后，债务人说"还钱还钱，愿意还两千"。那么在两千范围内形成一个新的债权债务关系——**单方允诺之债**。这两千块钱诉讼时效重新起算。但剩下的八千块钱依然过了诉讼时效。**部分同意履行，部分新债发生**。

八十页：义务人在诉讼时效期间届满后同意履行的，原债权由不完整的债权转为完整的债权，诉讼时效期间重新起算。但重新起算的原因不是"诉讼时效中断"（中断必须发生在诉讼时效期间内），而是**新的债权债务关系的形成**——单方允诺之债。

例题：甲公司向乙公司催讨一笔已过诉讼时效期间的十万元货款。乙公司书面答复称"该债务已过诉讼时效，本公司本无义务偿还，但鉴于双方长期合作关系，可偿还三万块"。这三万块变成了单方允诺之债，诉讼时效重新计算三年。剩下的七万仍然过时效。

### 第十五段：诉讼时效期间届满后的债务履行（二）——自愿履行

**第二，默示放弃（自愿履行）**。债务人直接把钱还了。还了以后能不能反悔，以"不当得利"为由要求返还？不能。因为债务本身还存在，债权受领给付不构成不当得利。

八十页：诉讼时效期间届满后，债务人自愿履行视为放弃诉讼时效效力，不得反悔，债权受领给付的不构成不当得利。

### 第十六段：基本原理部分的总结

诉讼时效的基本原理就这五个问题，逻辑上是相通的：

- 诉讼时效是因为权利人**怠于行权**。
- 所适用的权利一般是一个**请求权**。
- 一旦过了时效，进入自然之债，核心就是**抗辩权发生说**——债务人行不行使诉讼时效抗辩权，用不用这个"挡箭牌"，由债务人自己说了算。
- 过时效后债务人自愿履行或同意履行，是诚信的表现，法律鼓励。履行后不得反悔。

欠债还钱，欠债就是要还钱。


## 第二部分：对照《民法典》第九章"诉讼时效"提取备考提纲

> **说明**：本提纲基于讲稿内容，对照民法典第九章（第188条至第199条）框架整理，**未自造法律概念**，超出讲稿范围或不确定处已标注。


### 一、诉讼时效的基本概念与制度功能（讲稿第一至第三段）

| 概念 | 含义 | 讲稿关键词 |
| :--- | :--- | :--- |
| **诉讼时效** | 一定时间的经过产生一定的法律效果。权利长时间不行使，发生效力减损。 | 时间经过、自然事件 |
| **制度功能** | 惩罚"睡眠在权利上的人"（怠于行权的权利人），保护债务人利益。 | 权利人怠于行权→保护债务人 |
| **诉讼时效期间届满后的债权** | 称为**自然之债**——债权本身依然存在，但**丧失人民法院的强制执行力**。 | 自然之债、丧失强制执行 |
| **三个时间段** | ①借款期间（未到期）→ ②诉讼时效期间（三年，保护债权人）→ ③时效届满后（保护债务人） | 时间轴 |

> **提示（讲稿核心观点）**：诉讼时效制度的本质，是债权"效力减损"而非"权利消灭"。债权本身始终存在。


### 二、诉讼时效的法定性（讲稿第四至第五段）

**民法典第197条**：诉讼时效的期间、计算方法以及中止、中断的事由由法律规定，当事人约定无效。当事人对诉讼时效利益的预先放弃无效。

| 要点 | 讲稿解读 |
| :--- | :--- |
| **一律法定** | 当事人不能约定延长或缩短诉讼时效期间，不能预先放弃时效利益。 |
| **强制性规定** | 违反法定约定无效。 |
| **为什么法定** | 债权人在借款关系中处于优势地位，为防止强势方排除对弱势方（债务人）的保护，法律强制规定。 |


### 三、诉讼时效的适用范围——请求权（讲稿第六至第十一段）

诉讼时效仅适用于**请求权**（特定权利人请求特定义务人为或不为特定行为的权利）。不是所有请求权都适用，需分类讨论。

#### （一）债权请求权（"要钱不要物"）——一个原则，五个例外

| 类型 | 是否适用诉讼时效 | 理由（讲稿归纳） |
| :--- | :--- | :--- |
| **原则：债权请求权** | **适用**（三年） | 一般要钱的权利，都受时效限制。 |
| **例外一：支付存款本金及利息请求权** | **不适用** | 权利人未怠于行权（存钱是为了吃利息）。 |
| **例外二：兑付国债、金融债券、企业债券本息请求权** | **不适用** | 权利人未怠于行权。 |
| **例外三：基于投资关系产生的缴付出资请求权** | **不适用** | 保护公司及公司债权人利益（股东出资义务不能因时效而免除）。 |
| **例外四：国有资产受到侵犯的请求权** | **不适用** | 防止国有资产流失（考试基本不考）。 |
| **例外五：抚养费、赡养费、扶养费请求权（"三费"）** | **不适用** | 保护弱势群体生存权。 |

**讲稿特别提示（均为常考点）**：

1. **公共维修资金**（业主大会请求业主缴付）：**不适用诉讼时效**，类推适用股东出资规则。
2. **物业费**：**适用诉讼时效**（三年），属于普通债权请求权。
3. **劳动者工资报酬**：**适用诉讼时效**，劳动法有单独保障机制，不能躺在权利上睡觉。

#### （二）物权请求权（"要物不要钱"）——一个原则，两个例外

| 类型 | 是否适用诉讼时效 | 理由（讲稿归纳） |
| :--- | :--- | :--- |
| **原则：物权请求权** | **不适用** | — |
| **例外一：未登记的动产物权返还请求权**（普通动产，如书、手表） | **适用**（三年） | 无登记制度，存在举证困难，需及时行权。 |
| **例外二：担保物权（抵押权）** | **适用**（具体规则见民法典第419条，讲稿未展开） | 标的物不在债权人手里，需及时行权。 |

> **提示（讲稿归纳）**：看标的物有无**登记制度**。有登记制度（不动产、特殊动产）→不适用时效；无登记制度（普通动产）→适用时效。登记制度解决了举证困难的问题。

#### （三）其他不适用诉讼时效的请求权

| 类型 | 理由（讲稿归纳） |
| :--- | :--- |
| **人格权请求权**（停止侵害、排除妨碍、消除危险、消除影响、恢复名誉、赔礼道歉） | 侵权行为往往持续存在（民法典第995条）。 |
| **离婚后再次分割夫妻共同财产的请求权** | 什么时候发现什么时候分；适用三年时效，起算点为发现之日。 |


### 四、诉讼时效期间届满的法律效果——抗辩权发生说（讲稿第十二至第十三段）

**民法典第192条**：诉讼时效期间届满的，义务人可以提出不履行义务的抗辩。诉讼时效期间届满后，义务人同意履行的，不得以诉讼时效期间届满为由抗辩；义务人已经自愿履行的，不得请求返还。

**民法典第193条**：人民法院不得主动适用诉讼时效的规定。

| 要点 | 讲稿解读 |
| :--- | :--- |
| **抗辩权发生说** | 时效届满后，**实体权利、诉权都不消灭**，义务人取得**拒绝履行的抗辩权**。义务人可以行使，也可以不行使，完全自主决定。 |
| **法院不得主动适用** | 法官不能主动向债务人提示"时效过了"（被动司法），否则有违诚信原则。 |
| **起诉权不受影响** | 法院应当受理时效届满的案件，不得裁定不予受理。 |
| **债务人提抗辩** | 法院判决驳回原告诉讼请求。 |
| **债务人不提抗辩** | 法院判决债权人胜诉。 |
| **二审提抗辩** | 原则上不允许，除非有新证据。 |

> **讲稿比喻**："诉讼时效抗辩权就是一个挡箭牌，债务人用还是不用，由他自己说了算。"


### 五、诉讼时效期间届满后的债务履行（讲稿第十四至第十六段）

#### （一）同意履行（明示放弃抗辩权）

| 要点 | 讲稿解读 |
| :--- | :--- |
| **法律效果** | 原债权由不完整的债权（自然之债）转为完整的债权。 |
| **时效重新起算** | 但不是"时效中断"（中断需发生在时效期间内），而是形成**新的债权债务关系**——**单方允诺之债**。 |
| **同意履行部分** | 仅在同意履行的范围内形成新债，重新计算时效；剩余部分仍为自然之债。 |

**例题**：十万元货款已过诉讼时效，债务人书面答复"可偿还三万块"——三万块变为单方允诺之债，诉讼时效重新计算三年；七万块仍然过时效。

#### （二）自愿履行（默示放弃抗辩权）

| 要点 | 讲稿解读 |
| :--- | :--- |
| **法律效果** | 视为放弃诉讼时效效力，不得反悔。 |
| **债权受领** | 债权受领给付的，**不构成不当得利**。 |

> **讲稿核心观点**：无论在哪个时间段（借款期间、时效期间内、时效届满后），债务人主动还款，债权人都不会构成不当得利。因为债务本身始终存在。


### 六、诉讼时效的中止与中断

> **说明**：讲稿在本节音频中仅提及"诉讼时效的起算、中止、中断属于法定事项"，未在本次音频中展开具体规则，故提纲暂空，待后续内容补充。

**民法典相关条文索引**（供后续学习参考）：
- **第188条**：普通诉讼时效期间为三年；起算规则。
- **第194条**：诉讼时效中止（最后六个月内因不可抗力等障碍不能行使请求权的，中止）。
- **第195条**：诉讼时效中断（权利人主张权利、义务人同意履行等，中断后重新计算）。
