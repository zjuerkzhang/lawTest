# 民法讲稿整理与提纲对照（Markdown版）

## 第一部分：讲稿内容切分与修正

### 第一段：章节总述与核心概念

民事还得技术流，每一科你都值得更好的。

好，接下来咱们来看第二节，叫民事法律行为的效力。那么这一节是绝对重点，可以说每年绝对必考事项，并且主观题也会考查这一节，所以在这一节旁边写上“主观题”。

那么这个民事法律行为的效力是个啥意思呢？其实就是对民事法律行为进行的一个价值判断。很多同学说这里的民事法律行为记起来比较困难，其实咱们民法典总则编当中的民事法律行为的效力，专指的其实就是合同的效力。所以，在民事法律行为旁边你可以写两个字“合同”，你就按照合同的效力来学习就好了。

### 第二段：成立与效力的区分（事实判断 vs 价值判断）

那么合同的效力，涉及第一阶段和第二阶段，就是成立和效力的问题。成立的民事法律行为，它效力如何呢？这俩完全不是一回事儿。一个是**事实判断问题**。

比如说现在我把我的表卖给你，十二万要不要？你说不要。那我卖你不买，合同**没成立**。合同不成立，要不要去问效力问题？第一关都没过去，不需要去看第二关。

那如果现在我把我手上的五克毒品卖给了你，你要五万块钱，那咱们俩的合同**成立了吗？成立了**。但效力如何啊？**无效的**。所以这不就进行了一个价值判断吗？

所以，合同的成立问题，它是个事实判断问题，回答“你我之间有没有合同关系”；而合同的效力问题，是价值判断问题，回答“咱们的合同经不经得住法律的考量”。

### 第三段：自愿原则与法律限制

法律先要给大家画个圈圈。我们民法讲究意思自治，但这不是绝对的自由。为什么民法典当中不叫“自由原则”而叫“自愿原则”呢？因为老百姓会觉得自由就是我想干啥干啥，其实不是。自愿的意思就是你在法律规定的范围内可以自愿，是相对的自由。你去买本民法典，可以；但卖毒品，不可以。

所以第二件事就是问：你签的这个合同，法律同不同意？你当然愿意卖毒品，我也愿意买，但法律不同意。法律会进行否定性的价值评价。

### 第四段：民事法律行为效力的四种形态概述

有了大体印象以后，翻过来四十八页上方，我们来看民事法律行为的效力，它一共有四种：**有效、无效、待定、撤销**。

在司法考试年代（2018年以前），比较喜欢考合同以外的民事法律行为。但民法典颁布以后，法考以来基本考的都是合同。不过为了全面掌握，大家要了解：合同有四种效力状况；婚姻行为只有三种（有效婚、无效婚、撤销婚），没有待定；遗嘱遗赠、动产所有权的抛弃是典型的单方民事法律行为，非黑即白，只有有效和无效两种。遗嘱嘛，要么有效要么无效，总不能让死了的人去追认吧？

### 第五段：有效的民事法律行为

什么叫有效？有效的民事法律行为，通俗而言，就是**能够发生当事人意欲发生、主动追求或内心期待的法律效果**。说白了，就是“求仁得仁”，法律认可你。

要构成有效，得符合民法典第一百四十三条最基本的有效要件，我给它总结为 **“强公主义”** （谐音记忆：强=不违反法律行政法规的强制性规定；公=不违背公序良俗；主=主体具有相应民事行为能力；义=意思表示真实）。其中最重要的是意思表示真实，旁边写上“自愿原则”。

注意！意思表示不真实不一定就是无效的，也可能是**可撤销**的（比如欺诈、胁迫）。所以做题时看到“意思表示不真实因此无效”这种表述，肯定是不对的。

举例：孟某与培训机构签协议，交五万块保证上二本，否则退费。这种对赌协议不违法、不背俗，主体合格，意思真实，合同合法有效。考不上就退费，承担违约责任，违约也得有约可违嘛。

### 第六段：无效的民事法律行为（含义）

无效的意思，是**不发生**当事人意欲发生、主动追求或内心期待的法律效果。比如买毒品，法律不让你这么干。但注意：虽然不发生你想要的后果，却**会发生法律意欲发生的法律后果**。这个后果就是后面会学到的——**恢复原状、缔约过失责任**。如果卖毒品，还得承担刑事责任。

无效是效力瑕疵中最严重的一种，一定要慎之又慎，绝对例外。百分之九十五的合同都不会是无效的。不要动不动就选无效。

### 第七段：无效的五种法定情形（强公主义恶）

法律严格限制无效的认定，在民法典总则编中抽象出五种无效情形（统称“强公主义恶”）：

1.  **违反法律、行政法规的强制性规定**（如买卖毒品、枪支弹药）。
2.  **违背公序良俗**。
3.  **无民事行为能力人实施的民事法律行为**一律无效（民法典第144条）。
4.  **双方通谋虚假意思表示**（明实不符）无效，主观题常考。
5.  **恶意串通损害他人合法权益**的民事法律行为无效。

重点掌握前三项：强、公、义。但注意，违法不一定无效！

### 第八段：重点展开——违法不一定无效（附司法解释案例）

旁边写上“违法不一定无效”。据不完全统计，我国民商事法律有三千八百多件，老百姓动不动就会“犯法”，但法官绝不能认为违法就一律无效。

举例：民法典婚姻家庭编司法解释（二）第十五条（新增，还没考过）。父母用夫妻共同财产买房登记在未成年子女名下，后来以家长身份把房卖了，房价涨了又反悔，以“违反民法典第三十五条（监护人不得处分被监护人财产）”为由主张合同无效。法院**不予支持**。为啥？因为如果支持，就破坏了诚信原则和交易安全。这恰恰印证了“违法不一定无效”的理念。

### 第九段：重点展开——违背公序良俗（善良风俗的典型情形）

啥叫公序良俗？包括“公共秩序”（国家安全、社会公共利益）和“善良风俗”（社会公德、家庭伦理、人格尊严）。重点考善良风俗，这本质是道德问题，用一般理性人标准判断“缺不缺德”。

法考常考的几种违背善良风俗的情形：

1.  **结婚自由**：强迫、包办买卖婚姻，无效。例题：约定付保证金强迫结婚，否则不得返还，该约定因违背公序良俗无效。
2.  **生育自由**：离婚协议约定女方再婚后不得再生育，该约定无效。
3.  **买卖人体器官**：买卖人体细胞、组织、器官、遗体的行为无效（民法典第1007条）。
4.  **姓名权行使**：随意起名违背文化传统和公序良俗，公安机关不予登记（如“北燕云依”案）。
5.  **代孕**：代孕合同在境内因将人物化而违背公序良俗，无效。
6.  **第三者赠与**（重点）：夫妻一方为重婚、与他人同居等目的，将夫妻共同财产赠与他人（房车钱），另一方主张该赠与违背公序良俗无效的，法院**应予支持**。无效后果就是恢复原状（全部返还），受骗一方可另案主张缔约过失责任。


## 第二部分：对照《民法典》第六章“民事法律行为”提取备考提纲

> **说明**：本提纲基于讲稿内容，对照民法典第六章（第133条至第160条）框架整理，**未自造法律概念**，超出讲稿范围或不确定处已标注。

### 一、民事法律行为的成立与效力（讲稿第二段）

| 对比维度 | **成立（事实判断）** | **效力（价值判断）** |
| :--- | :--- | :--- |
| **核心问题** | 当事人之间是否存在民事法律行为（合同关系）？ | 已成立的民事法律行为能否获得法律认可？ |
| **判断依据** | 是否存在双方（或多方）的合意/单方意思表示。 | 是否符合法律规定的**有效要件**（民法典第143条）。 |
| **逻辑关系** | **前提**。行为不成立，则无需探讨效力。 | **后续**。行为成立后，才判断其效力状态。 |
| **举例** | 我卖表你拒绝，合同不成立，不谈效力。 | 双方合意买卖毒品，合同成立，但法律评价为无效。 |

> **提示（讲稿核心观点）**：成立看“有没有”，效力看“好不好”。“好”的法律认（有效），“不好”的法律否定（无效、可撤销、效力待定）。

### 二、民事法律行为效力的四种形态（讲稿第四段）

依据讲稿，民事法律行为的效力状态分为四类：

1.  **有效**
2.  **无效**
3.  **可撤销**
4.  **效力待定**

> **补充说明（讲稿第四段）**：不同类型的民事法律行为适用的效力状态不完全相同：
> - **合同**：四种状态均有。
> - **婚姻行为**：仅三种（有效、无效、可撤销），无“效力待定”。
> - **遗嘱/遗赠（单方行为）**：仅两种（有效、无效），无“可撤销”与“效力待定”。

### 三、有效的民事法律行为（讲稿第五段）

**1. 核心含义**
能够发生当事人**意欲发生、主动追求或内心期待的法律效果**（讲稿语，即“求仁得仁”）。

**2. 法定有效要件（民法典第143条）——讲稿总结为“强公主义”**

| 要件（讲稿助记） | 民法典对应内容 | 讲稿重点提示 |
| :--- | :--- | :--- |
| **主**（主体） | 行为人具有相应的民事行为能力。 | 无民事行为能力人独立实施的行为无效（例外情形见“无效”部分）。 |
| **义**（意思表示） | 意思表示真实。 | **最核心**。但意思表示**不真实不一定无效**，也可能属于**可撤销**（如欺诈、胁迫）。 |
| **强**（强制） | 不违反法律、行政法规的**强制性规定**。 | **“违法不一定无效”** （重要！违反管理性强制性规定不当然导致无效，仅违反效力性强制性规定才无效，讲稿未深入区分，但强调不能一概而论）。 |
| **公**（公序） | 不违背**公序良俗**。 | 违反公序良俗——>**无效**（最严厉否定）。 |

### 四、无效的民事法律行为（讲稿第六、七、八、九段）

**1. 核心含义**
**不发生**当事人意欲发生的法律效果，但**会发生法律意欲发生的法律后果**（如恢复原状、缔约过失责任等）。

> **无效是效力瑕疵中最严重的一种（讲稿强调：慎之又慎，绝对例外）。**

**2. 无效的法定情形（民法典第144、146、153、154条）——讲稿总结为“强公主义恶”**

| 类型（讲稿助记） | 民法典对应条款 | 讲稿示例/提示 |
| :--- | :--- | :--- |
| **强**（违法） | 第153条第1款：违反法律、行政法规的**强制性规定**的民事法律行为无效。但是，该强制性规定不导致该民事法律行为无效的除外。 | 买卖毒品、枪支弹药无效。但**违法不一定无效**（结合婚姻家庭编解释二第15条案例理解）。 |
| **公**（背俗） | 第153条第2款：违背**公序良俗**的民事法律行为无效。 | 强迫结婚约定、限制生育约定、买卖器官、代孕、第三者赠与等（详见下面“公序良俗典型情形”）。 |
| **主**（主体） | 第144条：**无民事行为能力人**实施的民事法律行为无效。 | 讲稿提及，但非重点。 |
| **义**（虚假） | 第146条：行为人与相对人以**虚假的意思表示**实施的民事法律行为无效。 | 双方通谋虚假意思表示（明实不符），主观题常考。 |
| **恶**（恶意串通） | 第154条：行为人与相对人**恶意串通**，损害他人合法权益的民事法律行为无效。 | 考的很少，慎选。 |

**3. 关于“违反强制性规定”的深入理解（讲稿第八段）**

- **核心原则**：**违法 ≠ 无效**。不能因行为触犯了某条法律就当然认定无效。
- **案例说明**：父母将登记在未成年子女名下的房产出售，后以“违反民法典第35条（监护人不得处分被监护人财产）”为由主张合同无效。法院**不予支持**。该案体现了“违法不一定无效”的价值判断，旨在维护交易安全和诚信原则。
- **法考提示（讲稿）**：此为2025年司法解释新增考点，尚未考查过，需重点关注。

**4. 违背公序良俗的典型情形（讲稿第九段）**

讲稿列举了法考常见的“善良风俗”案例：

| 类别 | 典型行为 | 法律效果 |
| :--- | :--- | :--- |
| **结婚自由** | 以保证金、协议等方式强迫他人结婚。 | 约定无效。 |
| **生育自由** | 协议限制他人再婚后生育。 | 约定无效。 |
| **人体器官** | 买卖人体细胞、组织、器官、遗体。 | 买卖行为无效（民法典第1007条）。 |
| **姓名权** | 随意起名，不随父姓/母姓或直系长辈血亲姓氏，违背文化传统。 | 公安机关不予登记（民法典第1015条）。 |
| **代孕** | 签订代孕协议，支付代孕费用。 | 代孕合同无效。 |
| **第三者赠与**（重点） | 夫妻一方为重婚、与他人同居等目的，将**夫妻共同财产**赠与他人（房、车、钱）。 | 另一方主张无效的，法院**应予支持**，赠与物**全部返还**（民法典婚姻家庭编解释二第7条）。 |

### 五、其他效力瑕疵形态（讲稿提及，但未深入展开）

讲稿在本节音频中重点讲解了“有效”与“无效”，对于**可撤销**（如欺诈、胁迫、重大误解、显失公平）和**效力待定**（如限制民事行为能力人实施的行为、无权代理行为）仅作为效力类型列出，未在本次音频中详细展开，故提纲暂空，待后续内容补充。

> **附注**：讲稿中提及“无效的合同不是没有法律后果，无效后会涉及恢复原状、缔约过失责任”，此为民法典第157条关于民事法律行为无效、被撤销或确定不发生效力后的法律后果，属于后续章节（民事责任）范畴，在此作为理解“无效含义”的补充。
