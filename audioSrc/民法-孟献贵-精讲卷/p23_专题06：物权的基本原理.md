### 第一段：物权编体系概览

还得技术流，每一科你都值得更好的。

好，接下来咱们来看第二部分，叫做物权编。物权编在客观题考十分左右，重点考察四个权利。书翻到八十七页，专题六物权的基本原理的前页，有一个物权编的体系。

物权编一共有四个权利，分别是**所有权、用益物权、担保物权**和【修正：原文为“准物权占有”，物权编体系一般为“占有”与前三者并列，此处调整为】**占有**。

物权制度开始，我们就展开了各具体的民事权利，学起来会比总则编相对轻松一点，因为它相对更为具体，跟我们的生活更为接近。

### 第二段：物权编的两驾马车

整个物权编的学习，要掌握**两驾马车**：

**第一，物权法定。** 物权的种类和内容由法律规定，原则上不允许当事人自由创设或改变，跟我们前面讲过的诉讼时效有相似之处。既然一切都是法律规定的，我们就按照法律规定来理解就好了，这个分是比较容易拿的。

**第二，物债两分区分原则。** 说白了就是合同是合同，物权是物权。我们要正式进入财产权的两大支柱的学习了。把它进行一个区分原则的认知，就能够把这些法律制度摘清楚了。

带着这四个权利、两驾马车，我们来学习物权编的第一个专题——物权的基本原理。

### 第三段：物的分类——不动产与动产

我们前面介绍过，物权的客体以物为原则，以权利为例外。于是要先看物的分类。

八十七页第一个问题叫物的分类。我们把物从不同角度进行了分类：不动产和动产、原物和孳息、主物和从物、货币、以及数据和网络虚拟财产。

先来看**不动产和动产的区分**，这是我们后边会一以贯之的制度。区分标准就是看标的物能否移动，以及移动以后是否会影响经济价值。

方法很简单：先定义不动产，除了不动产以外的通通都是动产。不动产就是不能移动或移动以后会影响经济价值的物。从考试角度来看，就两个重点——**房屋**和**土地**，特别是房屋。

除了不动产以外的都是动产。动产相对复杂一点，包括**特殊动产**和**普通动产**。特殊动产就三个：**机动车、船舶、航空器**，考试一般考机动车。普通动产包括自行车、照相机、机械设备等。特别注意：挖掘机、推土机、铲车、吊车等工程机械设备，都是**普通动产**，不是特殊动产。

区分不动产和动产的意义：

1. **公示方法不同**：不动产——**登记**；动产——**交付**（交付以后的结果是占有）。
2. **担保规则不同**：房屋只能做**抵押担保**；车和机械设备既可以抵押也可以质押，还能留置。

### 第四段：物的分类——原物与孳息

**原物和孳息**——两个物的关系：**出产和分离**的关系。

出产的原因有两种：
- **天然孳息**：基于自然原因而出产。包括动物（母鸡下的蛋——要分离，母鸡肚子里的鸡蛋不是孳息，是一个物）和植物（树上掉落的果实——掉落才分离，树上成熟的果实不是孳息，是一个物）。
- **法定孳息**：基于法律原因而出产。比如租金、存款利息。

**射幸孳息**（相对专业）：发生与否不确定。比如彩票中奖所得奖金、抽奖获得的奖品、保险合同产生的保险金。保险事故的发生与否是不确定的。

### 第五段：物的分类——主物与从物（重点）

**主物和从物**——两个物的关系：**辅助关系**。

构成要件：
1. **两个物**（不能是一个物的组成部分）。轮胎和汽车不是主物从物，是一个物；房屋和房门不是主物从物，是一个物。
2. **辅助关系**：没有从物，主物也能发挥功能，但发挥得不太好。手表和手表带、手机和手机壳、电视机和遥控器、汽车和备胎。
3. **属于一个人**：我的手机和你的充电器，不构成主物从物。

**区分的意义——从随主**：
- 主物转让，从物随之转让（除非当事人特别约定）。
- 主物做担保，从物随之做担保（除非特别约定）。

总结：两个物+一个人+辅助功能=主物与从物；核心规则：**从随主**。

### 第六段：物的分类——货币与网络虚拟财产

**货币**——一般等价物，原则上**占有即所有权**。比如把钱存银行，钱就是银行的了，存款人变成债权人。

但有例外：货币被**特定化**后，不能适用占有即所有。比如去银行取的一捆钱，上面有标记；或信封上写着“某某的私房钱”。

**数据和网络虚拟财产**——紧跟时事。手机流量、网络游戏装备、打赏网络主播的虚拟礼物等，都属于民法上的物。

总结：物的分类中，最重要的：不动产和动产（永久考点）、主物和从物（近年重点）。

### 第七段：物权三大效力（一）——追及效力

八十九页看第二个问题——**物权的三大效力**。

物权作为支配权，有三个效力：

**第一，追及效力。** “物在呼唤主人”——不管标的物辗转于何人之手，权利人均可主张所有权，要求返还。

例如：我的手表丢了，被你捡了，后被马蜂偷走，又被白浪涛捡到。我发现表在白浪涛手里，能不能要回来？可以。这叫物在呼唤主人。

**例外：善意取得**（每年必考）。善意取得为物权追及效力的例外。

### 第八段：物权三大效力（二）——排他效力与优先效力

**第二，排他效力。** 重点掌握**一物一权**——一个物只能有一个所有权，但可能有多个所有权人（共有）。比如我和刘某共同共有一套房子，一个所有权，两个所有权人。

**第三，优先效力。** 核心是**排序**，主要体现在担保制度中。原则：**物权优于债权**。

例题：孟某向白某借款两万（普通债权人，没做担保），又向马某借款两万，将自己的手表质押给马某并交付。到期无法归还。手表拍卖两万七千五——马某（质权人）优先受偿两万，剩下七千五归白某。白某剩下的一万两千五转为普通债权，继续追偿。

为什么马某优先于白某？**物权优于债权**。

三大效力总结：追及效力（例外：善意取得）、排他效力（一物一权）、优先效力（排序，物权优于债权）。

### 第九段：物权的保护（一）——物权请求权与债权请求权

**物权的保护**——我手上有一块表，被你抢走了，我有哪些救济途径？

1. 要物不要钱——请求返还表（**物权请求权**）
2. 要钱不要物——请求赔偿两万七千五（**债权请求权**）
3. 既要物也要钱——表被摔裂了，请求返还同时请求赔偿（可以并用）

**物权请求权**：只有当物权受到侵害时才会产生，是一种救济性权利。只有物权人才能行使物权请求权。

### 第十段：物权请求权的三种类型

**第一种：返还原物请求权（民法典第235条）**

核心要点：**无权占有**。

合法占有（有权占有）> 所有权 > 非法占有（无权占有）。

举例：我把表借给你用七天。七天内你是有权占有，我是所有权人也不能要求你返还；七天后你变成无权占有，我就可以要求返还。考试重点考借用合同、租赁合同、保管合同——看看有没有在借期/租期内。

**第二种：排除妨害请求权（邻里纠纷）**

邻居在门口堆垃圾、装修噪音等，影响了房屋所有权的圆满支配状态。既可以请求行为妨害人，也可以请求状态妨害人。

**注意**：排除妨害请求权**不适用诉讼时效期间限制**（不动产）。

**第三种：消除危险请求权**

危险还没发生，但有发生的可能性。比如邻居家的墙摇摇欲坠，还没倒，但有可能倒。

**精神损害赔偿**：必须是达到“严重”程度才行（普通失眠不行，严重失眠才可以）。

### 第十一段：物权请求权与债权请求权的区分

| | 物权请求权 | 债权请求权 |
| :--- | :--- | :--- |
| **目的** | 恢复对物的圆满支配状态 | 填补损害 |
| **关键词** | 返还原物、排除妨害、消除危险——“要物不要钱” | 赔偿——“要钱不要物” |
| **前提** | 以物的存在为前提（物没了就变成债权请求权） | 以债权的存在为前提 |
| **时效** | 不适用诉讼时效（不动产） | 适用诉讼时效 |

**可以并用**：比如表被抢了又被摔裂了，既请求返还原物（物权请求权），又请求赔偿损失（债权请求权）。

### 第十二段：经典例题与总结

例题：小贝的足球踢入邻居老马家水井，老马受惊打碎花瓶。小贝要求老马归还足球（物权请求权——返还原物）；老马要求小贝赔偿花瓶损失（债权请求权——赔偿）。

**总结专题六**：
1. 物的分类——不动产/动产、主物/从物（从随主，近年重点）
2. 物权三大效力——追及（例外善意取得）、排他（一物一权）、优先（物权优于债权）
3. 物权的保护——物权请求权（要物不要钱）、债权请求权（要钱不要物）、可以并用


## 第二部分：对照《民法典》物权编提取备考提纲

> **说明**：本提纲基于讲稿内容，对照民法典物权编相关框架整理，**未自造法律概念**，超出讲稿范围或不确定处已标注。


### 一、物权编体系概览

物权编重点考察**四个权利**：所有权、用益物权、担保物权、占有。

**两个基本原则**：
1. **物权法定**（民法典第116条）：物权的种类和内容由法律规定，不允许当事人自由创设或改变。
2. **物债两分区分原则**：合同与物权相区分。


### 二、物的分类

#### （一）不动产与动产

| 类型 | 定义 | 考试重点 |
| :--- | :--- | :--- |
| **不动产** | 不能移动或移动后影响经济价值 | 房屋、土地 |
| **动产** | 不动产以外的 | 特殊动产（机动车、船舶、航空器）与普通动产（机械设备等） |

**区分意义**：
- 公示方法：不动产**登记**，动产**交付**（→占有）
- 担保规则：不动产只能抵押；动产可抵押、质押、留置

> 【修正提示】原文中“挖掘机、推土机、铲车、吊车”被老师特别强调为**普通动产**而非特殊动产，这一点在考试中容易出错。

#### （二）原物与孳息

| 类型 | 含义 | 示例 |
| :--- | :--- | :--- |
| **天然孳息** | 基于自然原因出产并分离 | 母鸡下的蛋（非肚子里的蛋）、树上掉落的果实（非树上成熟的果实） |
| **法定孳息** | 基于法律原因产生 | 租金、存款利息 |
| **射幸孳息**（了解） | 发生与否不确定 | 彩票奖金、抽奖奖品、保险金 |

> **核心**：必须与原物分离，成为**两个物**。

#### （三）主物与从物（近年重点）

| 要件 | 说明 |
| :--- | :--- |
| 两个物 | 不能是一个物的组成部分（轮胎和汽车×，房屋和房门×） |
| 辅助关系 | 没有从物，主物也能发挥功能（手表+表带√，手机+手机壳√） |
| 属于一个人 | 我的手机+你的充电器× |

**核心规则——从随主**：
- 主物转让，从物随之转让（除非特别约定）
- 主物做担保，从物随之做担保（除非特别约定）

#### （四）货币

- **原则**：占有即所有权（一般等价物）
- **例外**：货币被**特定化**后，不适用占有即所有

#### （五）数据和网络虚拟财产

手机流量、网络游戏装备、打赏网络主播的虚拟礼物等——属于民法上的物。


### 三、物权的三大效力

| 效力 | 含义 | 例外/补充 |
| :--- | :--- | :--- |
| **追及效力** | “物在呼唤主人”——不管辗转何人之手，均可追回 | 例外：**善意取得**（每年必考） |
| **排他效力** | 一物一权——一个物只能有一个所有权 | 但可有多个所有权人（共有） |
| **优先效力** | 排序——**物权优于债权** | 主要体现在担保制度中 |

> 【修正提示】原文中出现的“追及效力”与“排他效力”在民法典中并非明确条文用语，而是学理归纳。其中“优先效力”的具体体现为担保物权人的优先受偿权。


### 四、物权的保护

#### （一）物权请求权 vs 债权请求权

| | 物权请求权 | 债权请求权 |
| :--- | :--- | :--- |
| **目的** | 恢复对物的圆满支配状态 | 填补损害 |
| **关键词** | 返还、排除妨害、消除危险（要物不要钱） | 赔偿（要钱不要物） |
| **前提** | 以物的存在为前提 | 以债权的存在为前提 |
| **时效** | 不适用诉讼时效（不动产） | 适用诉讼时效 |

**可以并用**：物权请求权与债权请求权可合并适用。

#### （二）物权请求权的三种类型

| 类型 | 要件 | 法典依据 | 讲稿要点 |
| :--- | :--- | :--- | :--- |
| **返还原物请求权** | **无权占有** | 民法典第235条 | 有权占有可对抗所有权人（借用/租赁/保管期间内） |
| **排除妨害请求权** | 妨害正在进行 | 民法典第236条 | 邻里纠纷，可请求行为妨害人或状态妨害人；不适用诉讼时效 |
| **消除危险请求权** | 危险尚未发生，但有可能性 | 民法典第236条 | 不适用诉讼时效 |

> 【修正提示】原文“合法占有大于所有权大于非法占有”表述不够准确，修正为：**有权占有（合法占有）可以对抗所有权人的返还原物请求权；无权占有（非法占有）则不能对抗**。

#### （三）精神损害赔偿

需达到“**严重**”程度（如严重失眠），一般程度的损害不支持。
