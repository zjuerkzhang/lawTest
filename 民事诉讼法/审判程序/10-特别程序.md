# 第1部分：备考提纲（对照民事诉讼法条文）


## 特别程序备考提纲

> 📌 **适用对象**：本法考备考提纲适用于非法学背景的考生，尽量使用通俗语言解释法律概念和制度。


### 一、什么是特别程序？

**特别程序**是民事诉讼法中专门规定的一类程序，用于审理**选民资格案件、宣告失踪宣告死亡案件、指定遗产管理人案件、认定公民无民事行为能力或限制民事行为能力案件、认定财产无主案件、确认调解协议效力案件、实现担保物权案件**。

**核心特点**（必须记住）：

| 特点 | 说明 |
|------|------|
| **不是诉讼程序** | 不解决民事权益争议，只是借用法院的审判组织来解决特殊问题 |
| **不讨论诉的分类** | 不能说这是确认之诉、形成之诉或给付之诉 |
| **不搞辩论** | 辩论原则只适用于诉讼程序 |
| **不搞调解** | 调解制度只适用于诉讼程序 |
| **没有人民陪审员** | 人民陪审员只参加诉讼程序 |
| **一审终审** | 不能上诉、不能申请再审、不能提第三人撤销之诉 |
| **错了只能提异议** | 向作出裁定的法院提出异议 |


### 二、特别程序的共同规则

**1. 审判组织**

| 情形 | 审判组织 |
|------|----------|
| 原则上 | 审判员一人**独任**审理 |
| 选民资格案件 | 审判员组成**合议庭** |
| 重大疑难案件 | 审判员组成**合议庭** |

⚠️ 合议庭只能由**审判员**组成，**人民陪审员不能参加**。

**2. 审限**

- 一般案件：立案之日起**三十日内**审结，或公告期满后**三十日内**审结
- 选民资格案件：**选举日前**审结

**3. 救济方式**

| 情形 | 救济方式 |
|------|----------|
| 一般情况 | 不能上诉、不能再审、不能三撤 |
| 裁定错了 | 向作出裁定的法院提出**异议** |


### 三、各类特别程序详解

#### （一）宣告失踪案件——财产代管人变更（⚠️ 重点考点）

**制度要点**：法院判决宣告失踪时，要同时指定财产代管人。

**变更代管人的两种情形**：

| 谁要求变更 | 有无纠纷 | 适用程序 |
|------------|----------|----------|
| **被指定的代管人**本人说“我不干了” | ❌ 无纠纷 | **特别程序** |
| **其他利害关系人**说“换人” | ✅ 有纠纷 | **普通程序**（起诉） |

**【真题模型】**

> 法院宣告赵某失踪，指定其妻黄某为财产代管人。后赵母李某有异议，黄某也想换人。
>
> - 黄某申请变更 → **特别程序**（无纠纷）→ ✅
> - 李某申请变更 → 以黄某为被告**起诉** → **普通程序** → ✅
> - 李某以赵小某为被告起诉 → ❌ 应以**原指定代管人黄某**为被告


#### （二）指定遗产管理人案件（⚠️ 2023年新增，重点）

**1. 实体法背景（民法典规定）**

遗产管理人按以下顺序确定：
1. **遗嘱执行人** → 2. 继承人**推选** → 3. 全体继承人**共同担任** → 4. 没有继承人的：**民政局**（城镇居民）/ **村委会**（农村居民）

有争议的，可以申请法院指定。

**2. 程序要点**

| 项目 | 内容 |
|------|------|
| **申请人** | **利害关系人**（所有对遗产主张权利的人：继承人、受遗赠人、债权人） |
| **管辖法院** | **被继承人死亡时住所地**或**主要遗产所在地**的**基层法院** |
| **法院指定原则** | **有利于遗产管理**的原则 |
| **变更（客观原因）** | 被指定人无法履职 → 本人**或**利害关系人都可申请另行指定 |
| **变更（主观原因）** | 违反职责严重侵害权益 → **只有**利害关系人可申请撤销并另行指定 |

**3. 实务意义（两个经典例子）**

- **例1**：继承人放弃继承，债权人无法主张债权 → 申请指定遗产管理人（民政局）→ 债权人向管理人主张
- **例2**：无继承人，对被继承人尽了抚养义务的人无法主张权利 → 申请指定遗产管理人 → 起诉管理人（**民事诉讼**，不是行政诉讼）


#### （三）确认调解协议效力（⚠️ 传统重点）

**1. 基本概念**

人民调解委员会制作的调解协议**没有强制执行力**。双方申请法院确认后，法院作出裁定，**裁定有强制执行力**。

**2. 申请条件**

| 项目 | 内容 |
|------|------|
| **申请人** | **双方当事人共同**申请 |
| **申请期限** | 调解协议**生效之日起三十日内** |

**3. 管辖法院**

| 情形 | 管辖法院 |
|------|----------|
| 法院**邀请**调解组织调解的 | **作出邀请的法院** |
| 当事人**自行**找调解组织调解的 | **当事人住所地** / **标的物所在地** / **调解组织所在地**的法院（基层；若纠纷本应由中院管辖的则找中院） |

**4. 不予受理的情形**（重点记第3、第5项）

1. 不属于法院受理范围的
2. 不属于该法院管辖的
3. ⚠️ **申请确认婚姻、亲子、收养等身份关系无效、有效或解除的**
4. 涉及其他特别程序、公示催告程序、破产程序的
5. ⚠️ **调解协议内容涉及物权、知识产权确权的**

**为什么这两类不予受理？** 身份关系涉及**公序良俗**；物权知识产权是**对世权**，对社会影响大。都不能用“水”程序（人民调解+特别程序确认）来处理，必须严格依法审查。

**5. 审查处理与救济**

| 裁定类型 | 内容 | 救济方式 |
|----------|------|----------|
| **肯定性裁定** | 确认调解协议效力（**可强制执行**） | **异议**：当事人**15日**内；利害关系人**6个月**内 |
| **否定性裁定** | 驳回申请 | **无救济**（法院什么都没干），重新解决纠纷 |


#### （四）实现担保物权（⚠️ 传统重点）

**1. 制度背景**

- 以前：必须先**起诉** → 判决 → 执行（繁琐）
- 现在：直接**申请**法院拍卖变卖担保财产 → 裁定 → 执行（快捷）

**2. 申请人**

- **担保物权人**：抵押权人、质押权人、留置权人等
- **其他有权人**：如留置/质押中的**债务人**（债权人占有担保物不着急，债务人着急）

**3. 管辖法院**

**担保财产所在地**或**担保物权登记地**的**基层人民法院**。

⚠️ **一律基层法院管辖**，没有中院管辖的问题！

**4. 审判组织**

| 情形 | 审判组织 |
|------|----------|
| 原则上 | 审判员一人**独任** |
| 担保财产标的额**超过**基层法院级别管辖范围 | 审判员组成**合议庭** |

⚠️ **易混淆**：标的额超基层管辖范围 → 只是从独任变**合议庭**，**管辖法院仍然是基层法院**，不会变成中院！

**5. 审查处理与救济**

| 裁定类型 | 内容 | 救济方式 |
|----------|------|----------|
| **肯定性裁定** | 准许拍卖变卖（**可强制执行**） | **异议**：当事人**15日**内；利害关系人**6个月**内 |
| **否定性裁定** | 驳回申请 | **无救济**（法院认为有纠纷），重新解决纠纷 |

**6. ⚠️ 仲裁条款问题（重要补充）**

> 抵押合同约定了仲裁条款，还能不能申请实现担保物权？

**能**。仲裁协议只是**禁止诉讼**，不禁止**特别程序（非讼程序）**。

- 申请时认为**没有纠纷** → 法院受理 → 与仲裁无关
- 法院审查认为**没有纠纷** → 作出肯定性裁定 → 与仲裁无关
- 法院审查认为**有纠纷** → 裁定驳回 → 此时**有纠纷了**，仲裁协议才发挥作用（只能仲裁，不能起诉）

## 专题二十二 特别程序

### 一、特别程序概述

好，我们讲专题二十二——特别程序。

来，大家把讲义翻到二百七十六页，先简单看一下概述。特别程序包括这么几类案件：选民资格案件、宣告失踪宣告死亡案件、指定遗产管理人案件、认定公民为无民事行为能力或限制民事行为能力案件、认定财产无主案件、确认调解协议效力案件，以及实现担保物权案件。

这些程序当中，重点需要大家掌握这么几个问题：第一，宣告失踪死亡案件当中有一个**财产代管人的变更问题**，这是重点。第二，**指定遗产管理人案件**，这个是2023年民诉法新增的一个程序，虽然内容很简单，但属于新增内容，大家注意一下。以及最后两个——**确认调解协议效力**和**实现担保物权**，这两个是传统的重点考点。这几个掌握一下就好了。


首先，我们现在讲的特别程序，它们都不是诉讼程序——除了选民资格之外，其余的叫**非诉讼程序（非讼程序）** 。那既然它们不是诉讼程序，为什么要把它们规定在民事诉讼法当中呢？它们只是借用民事诉讼法所规定的审判组织以及审判程序来解决一些特殊问题，所以把它规定在民事诉讼法中，专门给了一章叫特别程序。

那么，既然这些案件都不是诉讼程序，不是诉，那能不能去讨论**诉的分类**（确认之诉、形成之诉、给付之诉）？**不能**。考试的时候，在这些程序里边随便挑一个，说“这是什么确认之诉”“这是形成之诉”——都不对。诉的分类的前提是它得是一个诉，特别程序不是诉，不讨论诉的分类。

第二，既然它们都不是诉讼程序，那我们之前总结的那些**只能适用于诉讼程序的制度**，在这里就不能用了。我们总结过，哪些制度只适用于诉讼程序？

- **辩论原则**只适用于诉讼程序，所以特别程序不能辩论。
- **调解制度**只适用于诉讼程序，所以在特别程序当中不能调解。
- **人民陪审员**只参加适用一审程序审理的诉讼程序，所以特别程序当中不能有人民陪审员。

所以特别程序不是诉讼程序——**不搞辩论，不搞调解，不搞人民陪审员**。这个记住了。


### 二、特别程序的共同特点

看到二百七十七页，共同特点：

**（一）审判组织**

选民资格案件或者其他重大疑难案件，由审判员组成合议庭；其他案件由审判员一人独任审理。

记住了：**特别程序原则上独任制**。例外：选民资格案件和重大疑难案件要搞合议制。而且这个合议制**只能由审判员组成，人民陪审员不能参加**。

**（二）一审终审**

特别程序的判决裁定送达即生效，**不能上诉**。同时，生效的判决裁定错了，**也不能搞再审，不能搞第三人撤销之诉**。

那特别程序的判决裁定错了怎么办？可以向法院提出异议，法院适用特别程序审查之后作出新的判决裁定，撤销原判决裁定。

**（三）审限**

立案之日起三十天之内审结，或者公告期满三十天之内审结。选民资格案件在选举日前审结。

### 四、宣告失踪死亡案件——财产代管人变更问题

翻过来二百七十八页，三、宣告失踪死亡案件。宣告失踪死亡案件的具体程序大家看一看就好了，我们讲一个**重点问题**。

宣告失踪案件，法院在判决宣告失踪时，要为失踪人指定财产代管人。指定财产代管人之后，就有可能出现**财产代管人变更**的问题。

来举个例子：现在法院判决宣告张三失踪，同时指定张三的妻子李四担任财产代管人。那李四担任财产代管人之后，有可能出现要变更代管人的问题。主要会出现**两种情形**：

**第一种情形**：被指定的财产代管人李四说：“法官我不干了，你们重新指定吧。”——被指定代管人李四要求变更代管。

**第二种情形**：张三的爸爸老张说：“不要让李四干了，李四担任财产代管人会损害我儿子张三的利益。”——其他利害关系人要求变更代管人。

这两种情形有没有区别？**当然有区别**。

- 如果被指定的代管人李四说“我不干了”——没有实体的纠纷。法院适用**特别程序**撤销她代管人身份，另行指定就好。

- 如果是其他利害关系人老张说“不要让李四干了”，李四说“我还要干”——**有纠纷**。特别程序不解决纠纷，那怎么办？告知利害关系人老张做原告，以原指定的代管人李四做被告，提起诉讼，法院适用**普通程序**审理解决纠纷。

所以看清楚**是谁要求变更代管人**。


**【真题实例】**

二百七十八页最底下：

赵某下落不明八年，赵某的妻子黄某申请法院宣告赵某失踪，法院同时指定黄某为财产代管人。后来赵某的母亲李某对此有异议，希望自己担任财产代管人。黄某呢？希望指定自己的成年儿子赵小某为财产代管人。

这里有几个人想变更代管人？赵某的母亲李某希望变更代管人，原指定代管人黄某也希望变更代管人。有区别：

- **黄某**想变更代管人——没有纠纷，法院适用**特别程序**撤销她代管身份，另行指定就好。**（A选项正确）**
- **李某**要变更代管人——有纠纷。李某做原告，黄某（原指定代管人）做被告，提起诉讼，法院适用**普通程序**审理。**（B选项正确）**
- C. 李某可以黄某为被告提起诉讼，法院适用简易程序审理——**不对，应当适用普通程序**。
- D. 李某可以赵小某为被告提起诉讼——**不对，应当以原指定代管人黄某为被告，不能以赵小某为被告。**


### 五、指定遗产管理人案件

接下来二百七十九页，第四、指定遗产管理人案件。这是2023年民诉法新增的内容。在民事诉讼当中是比较简单的问题，大家需要结合民法典当中的遗产管理人制度来掌握。

**（一）民法典中遗产管理人的确定规则**

回顾一下，在民法典当中遗产管理人怎么确定：

1. 继承开始后，首先由**遗嘱执行人**担任遗产管理人。
2. 没有遗嘱执行人的，由**继承人推选**。
3. 继承人推选不出来的，由**全体继承人共同担任**。
4. 没有继承人、继承人放弃继承的，由**当地的民政局或者村委会**担任（被继承人是城镇居民的→民政局；农村居民的→村委会）。

对遗产管理人的确定有争议的，可以申请法院指定遗产管理人。

**（二）指定遗产管理人程序的现实意义**

为了配合民法典规定的遗产管理人制度，民事诉讼法在特别程序当中增加了指定遗产管理人的程序。这个程序有什么现实意义呢？

**例1：继承人放弃继承，债权人无法主张债权**

> 张三和李四结婚后育有一子张小三，张三父母双亡。张三突发疾病死亡，留下800万遗产。继承人李四和张小三准备继承。但突然出现债权人甲、乙、丙、丁，主张债权共计1000万。李四和张小三一算，800万还不够清偿债务，于是决定放弃继承。债权人懵了——债权真实存在，张三确实留下800万遗产，但继承人放弃了，找谁主张？

> 此时，债权人甲乙丙丁也好，继承人李四和张小三也罢，都可以向法院**申请为张三指定遗产管理人**。继承人放弃继承的，指定当地民政局担任遗产管理人。民政局接管遗产后，用遗产清偿债务。债权人要主张权利去找民政局，不满意可以起诉告民政局。

**例2：无继承人，对被继承人尽了抚养义务的人无法主张权利**

> 小熊自幼父母双亡，体弱多病，没有其他近亲属。远房姑妈赵女士一直抚养小熊成人，出全款给小熊买了一套房子登记在小熊名下。小熊突发疾病死亡。姑妈赵女士想把房子要回来。小熊没有留遗嘱，赵女士不是法定继承人。民法典规定“对被继承人尽了较多抚养义务的人，可以适当分得遗产”——但赵女士找谁去主张？打官司告谁？没有明确被告。

> 现在有了指定遗产管理人制度——赵女士可以作为利害关系人，向法院申请为小熊指定遗产管理人。法院指定当地民政局担任遗产管理人。民政局接管遗产后，把房子适当分给赵女士。赵女士对分配不满意，可以**起诉遗产管理人（民政局）** 。

> ⚠️ **注意**：起诉遗产管理人民政局，这是**民事诉讼**还是**行政诉讼**？——这是**典型的民事诉讼**。因为民政局不是行使行政管理职能，而是作为遗产管理人。赵女士与遗产管理人之间因财产权利关系发生纠纷，是民事诉讼。

**（三）指定遗产管理人案件的考点**

**1. 申请主体——利害关系人**

谁可以作为利害关系人申请指定遗产管理人？**所有可以对遗产主张权利的人**：
- **继承人**（包括法定继承人、遗嘱继承人，以及对被继承人尽了较多抚养义务或主要依靠被继承人抚养的人）
- **受遗赠人**
- **被继承人的债权人**

**2. 管辖法院**

由**被继承人死亡时住所地**或者**主要遗产所在地**的**基层人民法院**管辖。

为什么是这两个地方？遗产继承纠纷也是由被继承人死亡时住所地和主要遗产所在地法院专属管辖。指定遗产管理人虽然本身不是遗产继承纠纷，但申请指定遗产管理人往往集中发生在遗产继承纠纷中，所以由这两个地方的法院管辖，便于后续遗产继承纠纷的展开。

**3. 法院审查与指定**

法院受理申请后应当审查核实，依照**有利于遗产管理的原则**，判决指定遗产管理人。

**4. 遗产管理人的变更**

- **客观原因**：被指定的遗产管理人死亡、终止、丧失民事行为能力或者存在其他无法继续履行遗产管理职责情形的——被指定遗产管理人本人**或者**利害关系人都可以申请法院另行指定遗产管理人。
- **主观原因**：遗产管理人违反遗产管理职责，严重侵害利害关系人合法权益的——**只有利害关系人**可以申请撤销其遗产管理资格，另行指定新的遗产管理人。


**【例题】**

二百八十页例题：

> 老张住南山市A区，妻子去世多年，有两个儿子张甲、张乙。2024年1月老张去世。老张的生前好友李四向张甲张乙出示一份遗嘱，遗嘱载明：南山市B区的房屋归张甲所有，B区某银行的存款200万在清偿债务后分配给张乙，同时指定李四为遗嘱执行人。该遗嘱经南山公证处公证。
>
> 张甲认为老张不会指定一个外人担任遗嘱执行人，怀疑李四与公证处工作人员串通虚构遗嘱，遂向法院申请指定遗产管理人。

**问题1：张甲向哪个法院申请？**

张甲是继承人，可以作为利害关系人申请指定遗产管理人。向被继承人死亡时住所地（A区）或主要遗产所在地（B区）申请——**A区、B区都可以**。

**问题2：法院判决指定李四为遗产管理人，对不对？**

**对**。理由：（1）公证遗嘱所载内容推定为真实，张甲虽有怀疑但无相反证据足以推翻；（2）公证遗嘱指定了李四为遗嘱执行人，有遗嘱执行人的应当由遗嘱执行人担任遗产管理人。

**问题3：盛利公司因装修合同，老张欠其5万块钱，要起诉要钱，谁做被告？**

**李四**。李四虽然不是装修合同的当事人，但对被继承人老张的遗产享有管理权，所以应当以李四为被告提起诉讼。

**问题4：李四在验收房屋装修后忘记关闭电源，导致房屋失火，造成损失60万元。张甲张乙可以要求李四赔偿损失吗？**

**可以**。遗产管理人因故意或重大过失造成利害关系人损失的，应当承担民事赔偿责任。同时，利害关系人可以申请法院撤销李四的遗产管理资格，另行指定遗产管理人。

### 八、确认调解协议效力（重点）

**（一）基本概念**

确认调解协议效力，确认的是**人民调解委员会主持制作的调解协议**。

人民调解委员会主持制作的调解协议有没有强制执行效力？**没有**。当事人希望它有强制效力，可以**申请法院确认调解协议效力**。法院适用特别程序审查后下达裁定，确认调解协议效力，这个**裁定就可以强制执行**。

**（二）申请**

- **谁去申请**：**双方当事人共同申请**。
- **什么时候申请**：调解协议**生效之日起三十日之内**。

**（三）管辖**

分两种情况：

1. **法院邀请调解组织开展先行调解的**——向**作出邀请的法院**提出申请。

2. **调解组织自行开展调解的**——向**当事人住所地、标的物所在地、调解组织所在地**的基层法院提出。如果调解协议所涉纠纷本身应当由中院管辖的，就找上述三地的**中院**。


**【管辖例题】**

> A区甲公司与B区乙公司签订建设工程施工合同，约定甲公司承建乙公司位于C区的工程，同时约定纠纷由合同签订地D区法院管辖。

**情况一：甲公司起诉乙公司工程款800万**

这是建设工程施工合同纠纷，**不动产专属管辖**，由C区法院专属管辖。

C区法院在立案前邀请位于E区的建设工程施工合同纠纷人民调解委员会来组织调解，达成调解协议。要申请确认调解协议效力，向**C区法院**（作出邀请的法院）申请。

**情况二：甲乙公司自行到E区人民调解委员会调解**

达成调解协议后，向**当事人住所地（A区、B区）、调解组织所在地（E区）、标的物所在地（C区）** 的法院申请。

**（四）不予受理的情形**

总共五种情形：

重点讲两个：

1. **申请确认婚姻、亲子、收养等身份关系无效、有效或解除的**——法院不予受理。因为身份关系涉及社会公序良俗，法院必须依职权调查，不能说“你们开心就好”。

2. **调解协议内容涉及物权、知识产权确权的**——法院不予受理。物权是对世权，对社会有较大影响；知识产权也具有主物权的效力，必须严格依法审查。

> ⚠️ 为什么这两类不予受理？人民调解的调解程序比较灵活，不要求调解员严格懂法；确认调解协议效力的特别程序也不进行实质审查。两者都很“水”，但涉及身份关系（公序良俗）和物权知识产权确权（对世权）的，必须严格依法审查，不能用这种“水”程序。

**（五）审查处理与救济**

法院审查后作出两种裁定：

**1. 肯定性裁定——裁定确认调解协议效力**

这个裁定可以强制执行。如果错了会影响当事人和利害关系人的利益，所以要给救济。

- 不能搞第三人撤销之诉（特别程序不搞三撤）
- 不能搞再审（特别程序不搞再审）
- **只能提出异议**：
  - **当事人**：收到裁定之日起**十五日**内提出异议
  - **利害关系人**：知道或者应当知道权利受损之日起**六个月**内提出异议

**2. 否定性裁定——裁定驳回申请**

法院什么都没干，不影响当事人利害关系人的利益，**没必要给救济**。

驳回申请说明法院否认了调解协议的效力，当事人需要**重新解决纠纷**——可以和解、可以重新调解、可以仲裁、可以起诉。


**【例题】**

二百八十三页：

> 林某与刘某因交通事故赔偿纠纷，经人民调解委员会调解达成协议后，申请法院确认调解协议效力。某区法院审查认为符合规定，裁定确认调解协议效力。林某与刘某当即履行完毕。后来林某认为该调解协议中约定的残疾赔偿金数额远小于自己遭受的损失，林某可以何种方式救济？

林某是**当事人**，应当在收到裁定之日起**十五日之内**提出异议。

- A. 向上级法院复议——**错**
- B. 向该区法院或上级法院申请再审——**错**（特别程序不搞再审）
- C. 提出上诉——**错**（特别程序一审终审不能上诉）
- D. 向该区法院提出异议——**对**


### 九、实现担保物权（重点）

**（一）制度背景**

这个程序怎么来的？

以前根据担保法，银行想就抵押的房子优先受偿，需要就担保权利纠纷去法院**起诉**，拿到生效判决，再申请执行。但担保物权是**物权（支配权）** ，担保法却把它搞成了**请求权**。后来物权法改了——银行可以直接申请法院拍卖变卖担保财产，实现担保物权，不用再起诉。

但物权法生效后，民事诉讼法**没有配套的程序规定**。法院很头疼——直接让执行局卖？不行，执行必须要有生效法律文书。拿到审判庭判一下出个判决？那跟担保法有什么区别？由于缺乏程序法配套，物权法的规定一直被搁置到**2012年民诉法修改**。

2012年民诉法增加了**实现担保物权程序**——不走诉讼程序、不解决纠纷，但可以做出一个强制执行的文书，刚好放在特别程序当中。

**（二）申请人**

- **担保物权人**：抵押权人、质押权人、留置权人等
- **其他有权请求实现担保物权的人**

> 谁是“其他有权的人”？举个例子：我开着一辆10万的车去修理，欠修理费2万，修理铺老板把车留置了。修理铺老板是留置权人，可以去申请拍卖变卖——但他会去吗？不会，车在他手里押着，他不着急。**我着急了**——我欠2万，车值10万，卖了之后他拿2万走，剩8万得还我。车贬值到5万，他拿2万走，剩3万还我，损失我承担。所以**债务人**也可以申请。这就是“其他有权请求实现担保物权的人”——留置或质押中的**债务人**。

**（三）管辖**

**担保财产所在地**或**担保物权登记地**的**基层人民法院**。

⚠️ **注意**：实现担保物权案件一律由**基层法院管辖**，没有中院管辖的问题。

**（四）审判组织**

实现担保物权是特别程序，所以**原则上由审判员一人独任审查**。

但如果**担保财产标的额超过基层法院级别管辖范围**的，视为特别程序当中的**重大疑难案件**，应当由**审判员组成合议庭**审查。

> ⚠️ **易混淆考点**：说“实现担保物权案件由基层法院管辖，但如果担保财产标的额超过基层法院级别管辖范围的，应当由中院管辖”——**错**！实现担保物权案件一律由基层法院管辖。我们只是借用了基层法院的级别管辖标准，作为判断是独任制还是合议制的标准，跟级别管辖本身没关系。

**（五）审查处理与救济**

法院审查后作出两种裁定：

**1. 肯定性裁定——裁定准许拍卖变卖**

这个裁定可以强制执行。如果错了会影响当事人和利害关系人的利益，所以要给救济。

- 不能搞第三人撤销之诉（特别程序不搞三撤）
- 不能搞再审（特别程序不搞再审）
- **只能提出异议**：
  - **当事人**：收到裁定之日起**十五日**内提出异议
  - **利害关系人**：知道或者应当知道权利受损之日起**六个月**内提出异议

**2. 否定性裁定——裁定驳回申请**

说明法院认为担保物权**有纠纷**。特别程序不解决纠纷，所以**没必要给救济**。

当事人需要**重新解决担保物权纠纷**——可以和解、可以调解、可以仲裁、可以起诉。

**（六）⚠️ 仲裁条款与实现担保物权**

> 张三把房子抵押给甲银行贷款100万，抵押贷款合同中约定：“因履行本合同发生的任何纠纷，由A仲裁委仲裁。”还款期限届满，张三不还钱，甲银行向法院申请实现担保物权。**法院是否受理？**

**受理**。

仲裁协议作为纠纷解决条款，只是**禁止当事人通过诉讼方式解决纠纷**，并不禁止当事人通过**特别程序（非讼程序）** 实现权利。

- 甲银行申请实现担保物权，是因为认为**没有纠纷**——跟仲裁协议没关系。
- 法院受理后审查，如果认为**没有纠纷**，作出肯定性裁定——跟仲裁协议没关系。
- 如果法院审查认为**有纠纷**，裁定驳回申请——此时担保物权**有纠纷**了，要解决纠纷，仲裁协议**才发挥作用**。因为仲裁协议排斥诉讼，所以不能起诉，只能根据仲裁协议申请仲裁。


### 十、特别程序总结

回顾一下特别程序这一章讲了哪些问题：

**（一）特别程序概述**

1. 特别程序不是诉讼程序——**不讨论诉的分类**。
2. 只能适用于诉讼程序的制度不能用——**不搞辩论、不搞调解、不搞人民陪审员**。
3. 一律**一审终审**——不能上诉、不能三撤、不能再审。错了只能提出异议。
4. 审判组织——**原则上独任制**；选民资格案件和重大疑难案件由审判员组成**合议庭**（无人民陪审员）。

**（二）指定遗产管理人**

1. 申请人：**所有对遗产主张权利的人**（继承人、受遗赠人、债权人）
2. 管辖：**被继承人死亡时住所地**或**主要遗产所在地**基层法院
3. 变更：客观原因→本人和利害关系人都可申请；主观原因→**只有**利害关系人可申请

**（三）确认调解协议效力**

1. 申请：**双方当事人共同**，调解协议生效之日起**三十日内**
2. 管辖：法院邀请的→邀请法院；自行调解的→当事人住所地/标的物所在地/调解组织所在地
3. 不予受理：涉及**身份关系**的、涉及**物权知识产权确权**的
4. 救济：肯定性裁定→**异议**（当事人15日，利害关系人6个月）；否定性裁定→**无救济**，重新解决纠纷

**（四）实现担保物权**

1. 申请人：**担保物权人** + **其他有权人**（如债务人）
2. 管辖：**担保财产所在地**或**担保物权登记地**基层法院（**一律基层**）
3. 审判组织：原则上**独任**；标的额超基层管辖范围的→**合议庭**（但仍是基层法院管辖）
4. 救济：肯定性裁定→**异议**（当事人15日，利害关系人6个月）；否定性裁定→**无救济**，重新解决纠纷
5. 仲裁条款：**不阻止**申请实现担保物权
