# 第1步：法考备考提纲（民事诉讼法·再审审理程序）

> **说明**：以下提纲基于讲稿内容提取，结合《民事诉讼法》相关条款整理，便于非法科背景考生备考。


## 一、再审程序的两个阶段

| 阶段 | 起点 | 终点 | 核心任务 |
|---|---|---|---|
| **启动程序** | 判决生效 | 法院裁定再审 | 审查原裁判是否可能错误 |
| **重新审理程序** | 法院裁定再审 | 作出新判决 | 重新审理、作出新判决 |

**分界点**：法院裁定再审


## 二、重新审理的法院（由谁审）

### （一）法院启动再审 → 谁启动谁审

### （二）检察院抗诉启动再审
- **原则**：接受抗诉的法院提审
- **例外（事实证据问题）** ：可交下级法院再审；但该下级法院已再审过的除外

### （三）当事人申请启动再审（★考试重点）

**法条依据**（讲稿所述，具体条文号以现行《民事诉讼法》为准）：
- 因当事人申请而裁定再审的案件，由**中级以上法院**审理；当事人依法选择向基层法院申请再审的除外
- 最高院、高院裁定再审的，可由本院再审、交其他法院再审或交原审法院再审

**表格速记**：

| 原审法院 | 向谁申请 | 裁定再审后由谁审 |
|---|---|---|
| 基层法院 | 中院 | **只能由该中院审**（提审） |
| 基层法院★ | 可选择基层法院 | 该基层法院审 |
| 中院 | 高院 | 高院审（提审）或指令原中院/其他中院 |
| 中院★ | 可选择中院 | 该中院审 |
| 高院 | 最高院 | 最高院审（提审）或指令原高院/其他高院 |
| 高院★ | 可选择高院 | 该高院审 |

★ = 一方人数众多或双方均为公民


## 三、中止执行

**规则**：裁定再审时 → **应当裁定中止执行**

**例外（可以不中止）** ：追索**赡养费、扶养费、抚育费、抚恤金、医疗费、劳动报酬**的案件

**注意**：申请再审时 **不停止** 执行；裁定再审时才中止执行。


## 四、审理程序（一审还是二审）

| 情形 | 适用程序 | 判决效力 |
|---|---|---|
| 原一审生效 | 一审程序 | 可上诉 |
| 原二审生效 | 二审程序 | 终审，不可上诉 |
| **上级法院提审** | **一律二审程序** | **终审，不可上诉** |

**★ 核心考点**：问再审适用一审还是二审 → **先看有无提审，有提审直接二审；无提审再看原审是一审还是二审。**


## 五、再审范围有限原则

**原则**：再审以原审范围为限，**不准**增加变更诉讼请求，**不准**反诉。

**例外**（发回重审 + 四种情形之一）：

1. 原审未合法传唤即缺席判决
2. 追加新的当事人
3. 诉讼标的灭失或变化
4. 无法通过另诉解决

**判断逻辑**：再审中看到增加/变更/反诉 → 原则上不准 → 找“发回重审”四个字 → 有则找四种情形 → 找到则允许


## 六、结案方式（四种）

| 方式 | 要点 |
|---|---|
| **裁判** | 作出新判决时才能撤销原判；裁定再审时只能中止执行 |
| **调解** | 二审/再审调解必须制作调解书，送达后原判视为撤销 |
| **撤回起诉** | 二审/再审撤回起诉须对方同意，且撤后**不能**再起诉 |
| **终结** | 当事人达成和解并履行完毕 → 终结审查（启动阶段）/终结再审程序（审理阶段） |


## 七、原判效力时间轴（★高频考点）

| 时间点 | 原判效力 |
|---|---|
| 申请再审时 | 有效 + 继续执行 |
| 裁定再审时 | 有效 + 中止执行（**不能**撤销） |
| 重新审理中 | 有效 + 中止执行 |
| 作出新判决时 | 才能撤销/改变/维持 |


## 八、小额诉讼再审

- 可再审，向**原审法院**申请
- 必须组成**合议庭**
- 适用**一审程序**审理
- 能否上诉：程序错误理由 → 可上诉；实体错误理由 → 仍一审终审

# 第2步：讲稿切分段落（按逻辑顺序）

> **说明**：以下内容根据音频转文字整理，对重复、口语化及识别错误进行了适当修改。修改处用 **【修改】** 标注；原文保留但逻辑不清处用 **【调整】** 标注；超出主题范畴的概念用 **【注释】** 标注。未标注处为原文保留。


## 第一部分：再审程序概述——两个程序的划分

那好，不管是法院自己决定再审，还是检察院提出抗诉、检察建议，还是当事人申请、法院认为理由成立裁定再审的——反正最后法院一致裁定再审——于是进入第二个程序，叫**重新审理程序**。

在重新审理程序当中，我们要做的工作是：用原来的一审或二审程序对案件进行重新审理，作出新的判决。

**【调整】** 再审分为两个独立的程序：
- **启动程序**：解决原生效裁判是否可能错误、是否需要重新审理的问题。
- **重新审理程序**：裁定再审后，对案件进行重新审理、作出新判决的程序。

两个程序的分界点：**法院裁定再审**。


## 第二部分：重新审理的法院（审理法院）

由谁来重新审理？肯定是由法院来重新审理。那由哪个法院来重新审理呢？根据再审的不同启动方式分开讨论。

### （一）法院启动再审

谁启动谁来审：
- 本院启动的 → 本院自己审
- 上级法院启动的 → 上级法院自己提审

**口诀：谁启动谁来审，谁惹麻烦谁受惩罚。**

### （二）检察院启动再审（抗诉）

**原则**：由接受抗诉的法院提审。

**例外（事实证据问题）** ：可以交给下级法院审理，但**经该下级法院再审审理过的除外**。

**【修改】举例说明**：假设甲省有A、B、C、D四个市。A市中院的生效判决错了，应由甲省检察院向甲省高院抗诉。

- **原则**：省高院提审。
- **例外**：如果是事实证据问题（需要重新查事实），可以交给下级法院再审——可以是原审A市中院，也可以是B、C、D市中院。
- **但**：如果A市中院已经再审审过一次，省检再次抗诉的，**不管是不是证据问题，都不准再交给A市中院**——因为已经再审过一次，再抗诉说明“屡教不改”。

### （三）当事人申请再审——最重要、考试最常考

**【修改】** 两个法条依据：

1. 因当事人申请而裁定再审的案件，由**中级以上法院**审理；但当事人依法选择向基层法院申请再审的除外。
2. 最高院、高院裁定再审的案件，由本院再审，或者交由其他法院再审，也可以交由原审法院再审。

**核心规律**（以A市中院生效判决、向省高院申请再审为例）：

省高院裁定再审后：
- 省高院自己审 → 可以
- 指令别人再审 → 可以
  - 向上（最高院、乙省高院）→ **不行**（指不动）
  - 向下（A市中院、B市中院、C市中院、D市中院）→ **可以**
  - 指令本省某基层法院 → **不行**（级别低于原终审法院级别）

**规律总结**：被指令的法院，**级别不能低于原终审法院的级别**。

**【调整】分情形列表（表格形式）** ：

| 原审法院 | 向谁申请再审 | 裁定再审后由谁审理 | 备注 |
|---|---|---|---|
| 基层法院 | 中院 | **只能由该中院重新审理** | 不能指令原基层法院或其他基层法院 |
| 基层法院（一方人数众多或双方均为公民★） | 可选择向基层法院申请 | 由该基层法院重新审理 | 法条第一句后半段的例外 |
| 中院 | 高院 | 高院自己审（提审）或指令原中院/其他中院 | — |
| 中院（一方人数众多或双方均为公民★） | 可选择向该中院申请 | 由该中院重新审理 | — |
| 高院 | 最高院 | 最高院自己审（提审）或指令原高院/其他高院 | — |
| 高院（一方人数众多或双方均为公民★） | 可选择向该高院申请 | 由该高院重新审理 | — |

**★ 特别提示**：讲稿中提到“一方人数众多或者双方都是公民，我用星号表示”——**【注释】** 这里的“双方都是公民”对应《民事诉讼法》第206条（原第199条）规定的例外情形。


## 第三部分：中止原判的执行

法院决定再审后，**应当裁定中止**原判决、裁定、调解书的执行。

**为什么？** 当事人申请再审时不停止执行（还在启动阶段，法院没表态）；现在法院决定再审了（进入重新审理阶段），说明法院认为可能确有错误，为防止继续执行造成难以弥补的损害，应当中止执行。

**例外（可以不中止执行）** ：追索**赡养费、扶养费、抚育费、抚恤金、医疗费、劳动报酬**的案件。

**【修改】** 讲稿特别提示：这些费用中**医疗费**最常考，因为其他费用容易引起警觉，医疗费可能隐藏在看似普通的侵权纠纷中不易察觉。


## 第四部分：重组合议庭

再审法院审理再审案件，应当**另行组成合议庭**。再审必须用合议制。


## 第五部分：审理程序（适用一审还是二审程序）

再审没有独立的审理程序，而是使用**原来的一审或二审程序**对案件进行重新审理。

- 原来是一审终审的 → 按一审程序重新审理 → 所作判决为一审判决，**可以上诉**
- 原来是两审终审的 → 按二审程序重新审理 → 所作判决为终审判决，**不能上诉**
- **上级法院提审的** → **一律按二审程序重新审理** → 终审判决，不能上诉

**核心考点**：遇到再审适用一审还是二审程序的问题——**先看有没有提审，有提审直接适用二审程序；没有提审再看原来是一审还是二审。**


## 第六部分：例题——提审的考查

**【例题】**
甲公司起诉乙公司合同纠纷，甲市A区法院一审判决后，上诉期满当事人没上诉，判决生效。判决生效后甲公司发现新证据，申请再审。

**问**：关于本案再审的表述正确的是？
- A. 甲公司应当向甲市中院申请再审
- B. 甲公司可以向A区法院申请再审
- C. 本案应当适用一审程序重新审理
- D. 本案应当适用二审程序重新审理

**【解析】**
- 甲公司、乙公司没有一方人数众多，也没有双方都是公民 → 只能找上级法院 → A对，B错
- 向甲市中院申请再审 → 中院裁定再审后由中院重新审理 → **这是提审**（原来A区法院审，现在中院审）
- 提审 → **一律适用二审程序** → C错，D对

**答案：AD**


## 第七部分：再审范围有限原则

再审是**纠错**程序——原来处理过的问题处理错了，专门纠错。**新问题不解决**。

**原则**：再审以原审范围为限，当事人超出原审范围增加、变更诉讼请求的，不属于再审范围，再审法院不予审理。

**具体规则**：
- 再审中**不准**增加、变更诉讼请求
- 再审中**不准**提反诉
- 对增加变更的诉讼请求和反诉——**不调、不审**，直接告知另行起诉


## 第八部分：再审范围有限原则的例外——发回重审

**例外情形**：再审中法院裁定**撤销原判、发回重审**的案件，在重审中出现以下四种情形之一的，**允许**增加变更诉讼请求、提出反诉：

1. 原审没有合法传唤即缺席判决，影响当事人行使诉讼权利
2. 追加新的当事人
3. 诉讼标的灭失或发生变化，致使原诉讼请求无法继续实现
4. 增加变更的诉讼请求或反诉，无法通过另诉方式解决

**判断逻辑（两步法）** ：
- **第一步**：再审中看到增加变更诉讼请求或反诉 → 想到“再审范围有限原则”→ 原则上不准
- **第二步**：找有没有 **“发回重审”** 四个字 → 有的话，再找有没有四种情形之一 → 找到了，**允许**


## 第九部分：例题——再审范围（发回重审）

**【例题1】**
元丰公司依据供货合同要求神木公司支付货款，法院一审、二审神木公司均败诉。神木公司向法院申请再审，上级法院认为事实不清，**指令下级法院再审**（未发回重审）。再审期间，元丰公司增加违约金，神木公司以货物质量不合格为由主张解除合同。法院如何处理？

**【解析】**
- 再审中原告增加诉讼请求、被告提反诉 → 再审范围有限原则 → 原则上不准
- 找“发回重审”四个字 → **没有** → 不准
- 对增加的诉讼请求和反诉不予审理，直接告知另行起诉

---

**【例题2】**
全某起诉支付借款利息两万，法院一审判决驳回请求。市检察院以一审程序违法为由提出抗诉，中院裁定再审后认为一审程序错误，**撤销原判、发回一审法院重审**。重审中，全某明确诉讼请求为支付利息两万，并要求追加保证人郑某承担连带责任；吴某提出反诉要求全某返还三万不当得利。

**【解析】**
- 再审中吴某提反诉 → 原则上不准
- 找“发回重审”四个字 → **有**（撤销一审判决，发回一审法院）
- 找四种情形 → 重审中**追加了新的当事人**（郑某）→ 符合第二种情形 → **允许反诉**
- 选项：A（仅限抗诉范围）错；B（追加郑某为共同被告）对；C（审理反诉请求）对；D（追加郑某共同被告）对
- **答案：BCD**

---

**【例题3】**
甲公司起诉乙公司买卖合同纠纷，法院判决乙公司败诉。判决生效后乙公司发现新证据申请再审。再审中，法院发现遗漏了必须参加诉讼的丙公司，遂**裁定发回重审**。重审中法院追加丙公司参加诉讼，乙公司提出了反诉。

**【解析】**
- 再审中乙公司提反诉 → 原则上不准
- 找“发回重审”四个字 → **有**
- 找四种情形 → 重审中**追加了新的当事人**（丙公司）→ **允许反诉**
- 允许反诉后，本诉（甲公司起诉）和反诉（乙公司反诉）相互独立
- 甲公司撤诉（撤本诉）→ 反诉继续进行
- **答案：ABD**（A准许反诉；B适用普通程序；C驳回乙公司再审申请错；D就反诉进行审理对）


## 第十部分：一审、二审、再审中增加变更诉讼请求与反诉的对比总结

**【调整】** 讲稿在此做了全诉讼阶段的对比总结：

| 阶段 | 规则 | 理由 |
|---|---|---|
| **一审** | 法庭辩论终结前，可增加变更诉讼请求、可反诉 | 方便法院审理 |
| **二审** | 增加独立诉讼请求或反诉 → 只能调解，调解不成告知另行起诉；但当事人同意一并审理的除外 | 保护两审终审制 |
| **二审发回重审** | 可以增加变更诉讼请求、可以反诉 | 适用一审程序 |
| **再审** | **不准**增加变更诉讼请求、**不准**反诉 | 再审范围有限原则 |
| **再审发回重审**（有四种情形之一） | **允许**增加变更诉讼请求、**允许**反诉 | 例外 |


## 第十一部分：再审的结案方式（四种）

### （一）裁判（作出新判决）

重新审理后作出新判决，在新判决中确定是否**撤销、改变或维持**原判决。

**关键区别**：
- **裁定再审时**：只能裁定**中止执行**，不能撤销原判
- **作出新判决时**：才能**撤销原判**

### （二）调解

一审、二审、再审都可以调解。

- **一审调解**：原则上做调解书；四种情形可不做（调解和好的离婚、调解维持收养关系、能够及时履行、当事人同意签字生效）
- **二审和再审调解**：**必须**制作调解书，调解书送达后原判决**视为撤销**

### （三）撤回起诉

一审、二审、再审都可以撤回起诉，但规则不同：

| | 一审 | 二审/再审 |
|---|---|---|
| 是否需要对方同意 | 法庭辩论终结前不需要；之后对方不同意的可不予准许 | **必须**经对方当事人同意 |
| 撤回后能否再次起诉 | **可以** | **不能** |

### （四）终结（终结审查/终结再审程序）

当事人自行达成和解协议并**已履行完毕**的：
- 在**审查再审申请期间**发现 → **终结审查**
- 在**重新审理过程中**发现 → **终结再审程序**


## 第十二部分：例题——终结

**【例题1】**
甲乙公司合同纠纷，A市地区法院判决甲公司胜诉，双方均未上诉。判决生效后乙公司拒不履行，甲公司向B市法院申请强制执行。执行中甲乙达成和解协议并当即履行完毕，B市法院裁定执行终结。后来乙公司发现新证据，向A市中院申请再审。A市中院应当如何处理？

**【解析】**
- 当事人已达成和解协议并履行完毕 → 权利义务已实现
- 在**审查再审申请期间** → 终结审查
- **答案：裁定终结再审申请的审查**

---

**【例题2】**
张某起诉李某要求返还借款，一审判决李某败诉，双方均未上诉。判决生效后李某向法院申请再审，法院决定再审（进入重新审理程序）。再审中，发现张某李某达成和解协议并已支付完毕。应当如何处理？

**【解析】**
- 已达成和解协议并履行完毕
- 在**重新审理过程中** → 终结再审程序
- **答案：终结再审程序**


## 第十三部分：一张图读懂再审程序（核心考点——原判效力问题）

**图示**：

```
O —————— A —————— B
|   启动程序   | 重新审理程序 |
O = 判决生效
A = 法院裁定再审（中止执行）
B = 法院作出新判决（撤销/改变/维持原判）
```

**四个时间点的效力**：

| 位置 | 行为 | 原判效力 |
|---|---|---|
| OA之间 | 当事人申请再审 | 原判**有效**且**继续执行** |
| A点 | 法院裁定再审 | 原判**有效**但**中止执行**（不能撤销） |
| AB之间 | 重新审理过程中 | 原判**有效**但已中止执行 |
| B点 | 作出新判决 | 才能**撤销原判** |

**【调整】讲稿提到的连续考了四次的四个错误表述**：
1. “法院在裁定再审时应当裁定撤销原判” → **错**（A点只能中止执行）
2. “法院在裁定再审时可以裁定撤销原判” → **错**（不是应当/可以的问题，而是根本不能）
3. “高院在指令中院重新审理时应当撤销原判” → **错**（指令再审在A点，不能撤销）
4. “法院裁定再审时应当终结对原判的执行” → **错**（A点只能中止执行，B点才能终结）


## 第十四部分：小额诉讼程序的再审（补充细节）

小额诉讼程序一审终审，判决生效后**可以再审**。

**四个要点**：
1. **向谁申请**：向**原审法院**（原基层法院）申请再审
2. **组成合议庭**：再审必须组成合议庭
3. **适用程序**：原来一审终审，现在适用一审程序重新审理
4. **能否上诉**：**不一定**——取决于申请再审的理由
   - 理由是不该用小额诉讼程序（程序错误）→ 不再适用一审终审 → **可以上诉**
   - 理由是事实或法律错误（实体错误）→ 仍满足一审终审条件 → **不能上诉**


## 第十五部分：例题——小额诉讼再审

**【例题】**
李某诉谭某返还借款一案，M区法院按照小额诉讼程序审理，判决谭某返还借款。判决生效后，谭某认为借款数额远高于小额诉讼案件数额标准，不应按小额诉讼程序审理，向法院申请再审。法院裁定再审。

**问**：关于本案再审表述正确的是？
- A. 谭某应当向M市中院申请再审
- B. 法院应当组成合议庭审理
- C. 对所作再审判决当事人可上诉
- D. 所作再审判决仍为一审终审

**【解析】**
- A错：小额诉讼应向**原审法院**（M区法院）申请再审
- B对：再审必须组成合议庭
- C对、D错：申请再审理由是“不该用小额诉讼程序”（程序错误）→ 不再适用一审终审 → **可以上诉**
- **答案：BC**


## 第十六部分：本章总结——再审程序完整回顾

**【调整】** 讲稿最后做了完整总结：

**一、再审分为两个程序**
1. **启动程序**：解决原生效裁判是否可能错误、是否需要重新审理
2. **重新审理程序**：用原来的一审或二审程序重新审理，作出新判决

**二、启动主体**
- **法院**：本院（院长提交审委会）或上级法院
- **检察院**：抗诉或检察建议
- **当事人**：向法院申请再审；也可向检察院申请检察建议或抗诉（**断后性** + **终局性**）

**三、重新审理程序要点**
- **审理法院**：法院启动（谁启动谁审）；检察院抗诉（原则提审，事实证据问题可交下级）；当事人申请（中级以上法院审理，有例外）
- **中止执行**：裁定再审时中止，但六类案件可以不中止
- **合议庭**：另行组成
- **审理程序**：按原审程序；提审一律用二审程序
- **再审范围**：有限原则，不准增加变更诉讼请求、不准反诉（发回重审+四种情形例外）
- **结案方式**：裁判、调解、撤回起诉、终结
