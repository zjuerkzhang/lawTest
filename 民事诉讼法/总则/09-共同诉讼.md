# 第1部分：结合《民事诉讼法》的提纲（适用于非法学背景考生）


## 一、共同诉讼概述

### 1. 什么是共同诉讼？

一方或者双方为两人以上，一起打官司。

**法条依据：** 《民事诉讼法》第五十二条（原第五十三条）

### 2. 共同诉讼的两种类型

| | **普通共同诉讼** | **必要共同诉讼** |
|---|---|---|
| **诉讼标的** | 同种类（多个） | 同一（一个） |
| **诉的性质** | 可分之诉（可拆开审） | 不可分之诉（必须合并审） |
| **审理方式** | 可合并也可分开 | 必须合并审理、合一判决 |
| **当事人关系** | 相互独立 | 一人行为经他人同意后对他人生效 |

**法条依据：** 《民事诉讼法》第五十二条

**【区分口诀】** 数诉讼标的的个数——多个标的是普通，一个标的是必要。

**【简便判断法】** 问：被告赔钱是加起来赔一份，还是每人各赔一份？
- 加起来赔一份 → 一个事儿 → **必要共同诉讼**
- 每人各赔一份 → 多个事儿 → **普通共同诉讼**


## 二、常考的必要共同诉讼人

### 1. 追索赡养费案件

**规则：** 应当将**所有赡养义务人**作为共同被告，一个都不能少。

**法条依据：** 《民诉法解释》相关规定

**背后的道理：** 赡养老人是所有子女不可分割的共同义务。只要老人没有受到完全赡养，所有子女都视为没有尽到赡养义务。

**【注意】** 这不是纯粹的诉讼法理论问题，而是生活经验问题——防止老人偏心只告一个子女，导致子女间的矛盾无法解决。

### 2. 遗产继承纠纷

**规则：** 部分继承人起诉的，法院应当通知其他继承人作为**共同原告**参加诉讼。

**法条依据：** 《民诉法解释》

| 被通知的继承人态度 | 处理方式 |
|---|---|
| 愿意参加 | 列为共同原告 |
| 不愿参加，但**未放弃**实体权利 | **仍列为共同原告** |
| 不愿参加，且**明确放弃**实体权利 | **不列为共同原告** |

**【延伸考点】** 共同原告在法庭上的陈述 = **当事人陈述**；案外人的陈述 = **证人证言**。

### 3. 其他常考的必要共同诉讼情形

- 挂靠经营中的挂靠人和被挂靠人
- 个体工商户业主与实际经营者不一致
- 个人合伙的全体合伙人
- 企业法人分立后的各企业
- 借用业务介绍信、合同专用章等的出借单位和借用人
- 保证合同纠纷中的保证人和被保证人（连带保证）
- 共同侵权的行为人


## 三、当事人适格（结合民法典）

### 1. 三个基本概念

| 概念 | 含义 | 备注 |
|---|---|---|
| **诉讼权利能力** | 以自己的名义起诉、应诉的资格 | 自然人：始于出生终于死亡；法人：始于成立终于注销 |
| **诉讼行为能力** | 亲自参加诉讼的能力 | 法人：与权利能力同时产生消灭；自然人：仅完全民事行为能力人有 |
| **当事人适格** | 在具体案件中是否是正当当事人 | 原则：实体法律关系双方当事人 |

### 2. 结合民法典确定适格被告

民法典规定赔偿义务人的三种方式：

| 民法典规定方式 | 民诉中如何列被告 |
|---|---|
| **A承担责任** | A为被告 |
| **A、B承担连带责任** | 权利人选择：可只告A、只告B、或A、B为共同被告 |
| **A承担责任，B满足条件才承担** | A一定为被告；权利人主张B承担责任则A、B为共同被告，否则只告A |


## 四、代表人诉讼

### 1. 什么是代表人诉讼？

一方或双方人数**众多**（十人以上）时，推选**两到五人**作为代表人参加诉讼。

**本质：** 代表人诉讼本质上一定是共同诉讼（人数众多本身就是两人以上）。

### 2. 代表人的诉讼权利

| 行为类型 | 是否需要被代表当事人同意 |
|---|---|
| **一般诉讼行为**（程序性事项） | **不需要** |
| **处分实体权利**（承认、放弃、变更诉讼请求、和解、调解等） | **必须经同意** |

### 3. 代表人诉讼的分类

#### （1）人数确定的代表人诉讼

**定义：** 起诉时人数众多一方人数确定。

**可能是普通共同诉讼，也可能是必要共同诉讼。**

**代表人如何确定：**
- 全体当事人推选共同的代表人
- 部分当事人推选自己的代表人
- 选不出代表人的：普通共同诉讼→另行起诉；必要共同诉讼→亲自参加

**法条依据：** 《民诉法解释》第七十六条

#### （2）人数不确定的代表人诉讼

**定义：** 起诉时人数众多一方人数不确定。

**一定是普通共同诉讼**（不可能是必要共同诉讼，因为必要共同诉讼要求有明确的被告）。

**程序：**
1. 法院**确定权利人范围**
2. 法院**发布公告**（敦促权利人登记）
3. 权利人**向法院登记**
4. 公告期满，**确定代表人**（推选→协商→指定）
5. 代表人**参加诉讼**
6. 法院**作出判决**

**判决效力：**
- 对**参加登记的权利人**：**直接约束力**
- 对**未参加登记而另行起诉的人**：**预决效力**（法院认定其请求成立的，裁定适用之前判决）

**法条依据：** 《民事诉讼法》第五十七条（原第五十五条）

#### （3）证券纠纷特别代表人诉讼

**定义：** 依据《证券法》第九十五条第三款提起的诉讼。

**与普通代表人诉讼的区别：**

| | **普通人数不确定的代表人诉讼** | **特别代表人诉讼** |
|---|---|---|
| **代表人** | 权利人中的代表 | **投资者保护机构**（投保机构） |
| **权利人参加方式** | **主动登记** | **默示加入、明示退出**（不声明退出即参加） |
| **判决效力** | 只对**主动登记**的人 | 对**没有声明退出**的所有人 |

**两个痛点如何解决：**
1. **实力不对等** → 投保机构（注册资本数十亿）对抗上市公司
2. **维权成本高** → 投资者什么都不用干，躺着等判决

**法条依据：** 《证券法》第九十五条第三款；《最高人民法院关于证券纠纷代表人诉讼若干问题的规定》

## 【专题六】共同诉讼

### 一、共同诉讼的基本概念

好，我们讲专题六——共同诉讼。来，大家把讲义翻到第七十九页。

那首先什么叫共同诉讼啊？一方或者双方为两人以上，咱们一起打官司嘛，这个就叫共同诉讼。


### 二、普通共同诉讼与必要共同诉讼的区分

首先第一个问题，共同诉讼呢，我们区分为**普通共同诉讼**和**必要共同诉讼**。

什么叫普通共同诉讼？什么叫必要共同诉讼呢？来看好啊——

**普通共同诉讼**是一方或者双方为两人以上，诉讼标的**同种类**。什么叫同种类啊？来同学们看好了，老师桌上的两瓶水同种类，几瓶？两瓶嘛。那诉讼标的同种类有几个诉讼标的？两个以上嘛。那既然有两个以上诉讼标的，说明在本质上就是两个以上独立的**可分之诉**。那能不能拆开一个一个审啊？可以啊。只不过在诉讼标的相关似的情况下，为了提高诉讼效率，我们征得当事人同意将其合并审理，这个叫做普通共同诉讼。那当然，可以拆开一个一个审啊，可以没问题。

那再来，我们翻过来看到讲义的第八十页——**必要共同诉讼**。什么叫必要共同诉讼呢？一方或者双方为两人以上，诉讼标的**同一**。同一同一说明几个诉讼标的？一个。既然只有一个诉讼标的，说明只有几个诉——一个诉。所以法院必须合并审理，**合一判决**，这个叫什么？叫必要共同诉讼。

**【区分方法】** 那所以首先我们要区分啊，普通共同诉讼和必要共同诉讼怎么区分呢？很简单，**去数诉讼标的的个数**。


### 三、区分普通共同诉讼与必要共同诉讼的例题

来咱们举例子，听好了。

**【例题1·普通共同诉讼】**

某旅游公司的大巴车载着四十名乘客外出旅游，发生交通事故，车上四十名乘客不同程度受到损害，纷纷起诉旅游公司要求赔偿。请问本案有几个诉讼标的啊？

几个啊？四十个不同的侵权或者违约法律关系，对不对？这四十名乘客能不能分别起诉旅游公司啊？可以啊。只不过这诉讼标的相类似嘛，为了提高诉讼效率，我们征得当事人同意后把它合并审理了嘛，这个叫普通共同诉讼。那当然可以拆开一个一个审啊，可以啊，没问题，对吧？

**【例题2·必要共同诉讼】**

旅游公司的大巴车载着四十名乘客外出旅游，这一路上呢，乘客们和旅游公司相处得不太愉快。于是呢，趁着一次汽车抛锚了，乘客们一起走下大巴车，把大巴车围了起来，一起喊一二三，就把大巴车抬起来扔下了悬崖。那这个时候旅游公司起诉这四十名乘客赔偿，那么请问本案有几个诉讼标的啊？

几个啊？一个——这四十名乘客对旅游公司的共同侵权。由于只有一个诉讼标的，所以只有一个诉，所以法院必须合并审理和判决，这个叫什么？叫必要共同诉讼，对吧？

**【例题3·必要共同诉讼】**

再一组，两个例子体会一下——

左林在网上发帖子，对我和李家的名誉进行了毁损。那这个时候李家找到我说：“左林这小子到处发帖子骂我们，走收拾他。”我说：“行。”于是我和李家一起来到左林家里边，把左林打了一顿，腿打断一条。那这个时候左林起诉我和李家赔偿，请问本案有几个诉讼标的啊？

有几个诉讼标的啊？一个——我和李家对左林的共同侵权。由于只有一个诉讼标的，所以只有一个诉，所以法院必须合并审理和判决，这个叫必要共同诉讼，对吧？

**【例题4·普通共同诉讼】**

那再来，李家找到我说：“左林这小子到处在网上发帖子骂我们，走收拾他。”我说：“不行，现在是法治社会，要讲法治。我们去法院告他吧。”于是我和李家来到法院起诉左林名誉侵权。请问法院有几个诉讼标的啊？

几个啊？两个——左林对李家的名誉侵权，左林对我的名誉侵权。那我和李家能不能分别去起诉左林啊？可以啊。只不过诉讼标的相似嘛。为了提高诉讼效率，征得当事人同意，我们合并审理没问题。那能不能拆开一个一个审啊？可以没问题啊。

**【例题5·连带保证中的共同诉讼】**

再举一个例子——

甲公司是一个小贷款公司，跟张三签订了借款合同，约定甲公司向张三提供借款一百万。同时，李四跟甲公司签订担保合同，约定由李四为张三应当偿还的本金利息承担连带保证责任。好，连带保证。

到了还款期限，张三拒不还款，那于是甲公司就起诉债务人张三以及担保人李四，要求他们连带清偿这一百万。那么请问本案有几个诉讼标的？

几个？两个——甲公司和张三的借款法律关系，以及甲公司和李四的担保法律关系。这是两个独立的法律关系吗？是的吧？所以只有两个法律关系，两个诉，这叫什么？**普通共同诉讼**。〖注：讲稿中此处讲授者原话为“必要共同诉讼”，结合前后文的法律关系分析（两个独立法律关系）及讲授者随后提供的“简单执法方法”，此处应为讲授者口误，实际应归类为普通共同诉讼。〗


**【简便判断方法】**

你实在不会判断，我给你教个简单的执法方法——

**问：在本案当中，被告向原告赔偿，是加起来赔一份就行了，还是每人各赔一份？**

- 加起来赔一份就行了 → **一个事儿** → **必要共同诉讼**
- 每人各赔一份 → **多个事儿** → **普通共同诉讼**

来试试——

**【例题1回顾】** 某旅游公司大巴车载着四十名乘客外出旅游，发生交通事故，车上四十名乘客不同程度受到损害，纷纷起诉旅游公司赔偿。你告诉我，这个旅游公司对这四十名乘客进行赔偿怎么赔？是给这四十个人每人赔一份，还是给这四十个人加起来赔一份就行了？**给每人赔一份嘛，赔四十份嘛，四十个事儿嘛**——这个叫普通共同诉讼，对吧？

**【例题2回顾】** 那这四十名乘客把旅游公司大巴车抬起来扔下悬崖，旅游公司起诉这四十名乘客赔偿。那告诉我，在这个案件当中，这四十名乘客向旅游公司赔偿怎么赔啊？是每人赔一辆车赔四十辆，还是加起来赔一辆就行了？**加起来赔一辆就行了**，一个事儿——叫必要共同诉讼，对吧？

**【例题3回顾】** 那我和李家去把左林打了一顿腿打断一条。那这个时候左林起诉我和李家赔偿，你告诉我，我和李家对左林赔偿怎么赔？是每人赔一条腿总共赔两条，还是加起来赔一条就行了？**加起来赔一条就行了**——一个事儿，叫必要共同诉讼，对吧？

**【例题4回顾】** 那我和李家去法院起诉左林名誉侵权，那这个时候左林向我和李家赔偿怎么赔？给李家一份吧，给我一份吧，赔几份？**赔两份啊，两个事儿**——叫普通共同诉讼。

哎，这么区分，就没有问题了吧？


### 四、共同诉讼人之间的关系

好，这是普通共同诉讼、必要共同诉讼会区分。来，接下来——

**普通共同诉讼**当中，各共同诉讼人之间是什么关系？你想，普通共同诉讼反正是若干独立可分之诉，那这官司怎么打呀？你打你的，我打我的——咱们叫**相互独立，互不干扰**。

那**必要共同诉讼**呢？那你想，必要共同诉讼，我们俩跟对方产生一个法律关系，说明我们俩之间法律关系紧密不紧密啊？紧密嘛。那我们俩之间什么关系？**凡事商量着来嘛，一人行为经其他共同诉讼人同意，对其他共同诉讼人发生法律效力**，对不对？

所以——
- **普通共同诉讼**：共同诉讼人之间相互独立
- **必要共同诉讼**：共同诉讼人当中的一人的行为经其他共同诉讼人同意，对其他共同诉讼人生效


### 五、常考的必要共同诉讼人

好，那接下来我们看到第八十页，下面第四——**常考的必要共同诉讼人**。这里有这么多啊，其实我们基本都是讲过的没问题。来我重点强调两个问题啊——第七和第八。


#### （一）追索赡养费案件

**第七，追索赡养费案件中应当将所有的赡养义务人作为共同被告。**

什么意思？举个例子——

**【例题6·追索赡养费】**

现在张老头老了，有四个儿子：老大张甲在老家种田照顾张老头起居生活；老二张乙在城里工作每个月给赡养费三千；老三张丙在外地创业工作很忙没有时间照顾老人也没有给钱；老四张丁呢刚刚大学毕业还没有时间没有精力没有能力来赡养老人。

那张老头的生活状态：老大张甲出力，老二张乙出钱，老三老四不管。张老头呢，每个月有人照顾自己的起居生活，每个月还有三千块钱花，其实过得挺爽的——每天早上出门遛鸟，中午回家吃饭，下午打麻将，晚上跳舞回家睡觉。

但是呢，平淡的生活总会发生一些波澜。那一天晚上，张老头出去跳舞的时候，就跳回来一个老太太，于是张老头就谈恋爱了。这个恋爱一谈，一个月三千块钱够不够花？不够。于是张老头就想找孩子们要钱——找谁呢？老大张甲没钱；老二张乙有钱但是一个月给了三千，再要不合适；老三张丙有钱没给。

于是张老头一纸诉状就把老三张丙告上法庭，要求张丙每个月给赡养费三千。你看到，张老头起诉老三张丙——那本案谁作为被告？张甲、张乙、张丙、张丁四个孩子**一个都不能少，全都是共同被告**。

哎，有同学说老师，这个张乙给了钱了，为什么还要作为共同被告呢？这个不公平啊？

没什么不公平的——我们讲**赡养老人是所有子女不可分割的共同义务**，只要老人没有受到完全赡养，所有子女都叫没有尽到赡养义务，一定作为共同被告，一个都不能少。再说了，谁告诉你给了钱就叫尽到赡养义务了？我们唱歌是怎么唱的？唱“回家看看，回家看看”，有谁唱过“要给五万”吗？没这么唱的吧，对不对？

所以**追索赡养费案件中，应当将所有赡养义务人作为共同被告，一个都不能少**。

有同学说老师，这张老头有处分权啊，我只找老三张丙要啊，我没找其他几个要啊，那为什么要把其他几个作为共同被告呢？

注意同学们啊——这个地方你不要试图用民事诉讼的基本原理、基本理论去解释这个问题，解释不通啊。这个其实是一个**生活常识、生活经验**的问题。

什么意思呢？我先说一个结论——这个追索赡养费的纠纷，表面上看是老人找孩子要赡养费，表面上是老人和孩子间的矛盾，但其实在很多情况下**是子女之间的矛盾**。

来举个例子——假设你父亲年龄大了，父亲就你一个孩子，那这个时候要求你每个月给赡养费三千块钱，请问你有意见吗？没有嘛，对不对？没有意见，该给对不对？

那现在问题是，你父亲年龄大了，有两个孩子——有你，还有你一个哥哥。那这个时候你父亲每个月就找你要三千块钱，但从来不找你哥哥要，请问你有意见吗？可能有意见了吧，对不对？

你说还没有意见——那这样，每个月就找你要三千块钱，老爷子很节约啊，一个月三千块钱花不完啊，于是只花了两千五，每个月还能攒下五百块钱，攒下这五百块钱呢，每个月给哥哥家补贴家用——请问你是不是有意见了？一肚子火了吧？

那你生气，桌子一拍说我不给了。那你不给了，老爷子也生气了，于是一纸诉状就把你告上法庭，要求你每个月给赡养费三千块钱。

那如果这个案件是老头做原告，你做被告，那就会出现什么问题呢？到了法庭上，法官问：“哎，你当孩子的，每个月给你爸三千块钱，你愿不愿意给？”你怎么说？愿意。那法官问：“哎，老爷子啊，你孩子愿意每个月给你三千块钱，你愿不愿意要啊？”老爷子说愿意。那这个法官就懵了——你们一个愿意给，一个愿意要，你们都没纠纷啊，你们打什么官司啊？就把你们轰出去。

轰到一半儿你就喊：“法官不对！我给三千没问题，但我还有个哥哥，我哥哥也得给！”

那这个时候法官对你嘿嘿一笑：“本案只有你爸做原告，你做被告，就你们俩的事，我就判你给就行了呀。至于你哥该不该给，与本案无关。你觉得你哥该给，那好啊，你另行起诉去。”

我另行起诉我哥要赡养费——我是适格原告吗？不是吧？我没法起诉。我就只能动员我爸去起诉我哥呀，对不对？那问题不就出在这里吗？我爸偏心，不找我哥要啊。那请问这个事儿能解决吗？永远解决不了，对不对？

所以司法解释规定——老头起诉要赡养费时，**法院应当追加我哥作为共同原告**。那么在本案中判决我们俩每人给三千，这事儿不就解决了吗？对不对？

哎，所以你看到——这是生活经验、生活常识啊。


#### （二）遗产继承纠纷

**第八，在遗产继承纠纷中，被部分继承人起诉的，人民法院应当通知其他继承人作为共同原告参加诉讼。被通知的继承人不愿参加诉讼，又不明确表示放弃实体权利的，人民法院仍然将其列为共同原告。**

那言下之意是什么？被通知的继承人不愿参加诉讼，但人家**明确表示放弃实体权利**，那还能不能将其作为共同原告呢？**不能。**

**【例题7·遗产继承】**

来举个例子，听好了——

张老头死了，有四个孩子：老大张甲、老二张乙、老三张丙、老四张丁。

现在呢，老大张甲起诉老二张乙争遗产。在遗产继承纠纷中，法院得去通知老三张丙、老四张丁参加诉讼，通知他们作为共同原告参加诉讼，对吧？

好，那法院去通知张丙参加诉讼：“张丙啊，你爸死了，大哥起诉二哥争遗产，官司打到了法院，我们通知你作为共同原告参加诉讼，请问你来吗？”

张丙一听，嚎啕大哭，边哭边骂：“父亲尸骨未寒，你们亲兄弟居然对簿公堂！这个官司我就不来了！”法官说：“不来可以啊，但这个钱要吗？”张丙一算：“哎，亲兄弟还是要明算账，这个钱还是要的。”

好，那法官去通知张丁。张丁同样的态度：“父亲尸骨未寒，亲兄弟对簿公堂，诉讼不来不来！”法官说：“不来可以啊，但这个钱要吗？”张丁一算没几个钱：“算了，不要了。”

那同学们分析——在本案中，原告张甲，被告张乙。我们法院去通知张丙来，张丙不愿意参加诉讼，但他有没有放弃实体权利啊？**没有**——没有放弃实体权利，**依然将其列为共同原告**，不到庭不影响审理。

张丁不愿意参加诉讼，而且**明确表示放弃实体权利**——还要不要追加为共同原告？**不用了**。

所以本案张甲和张丙做共同原告，张乙做被告，跟张丁没关系，张丁不做本案当事人，对吧？

**【延伸考点·证据类型】**

好，那题目接着问啊——张丙、张丁都说了开庭是不来。到了开庭那一天，张丙、张丁不约而同地来到了法院，分别向法官陈述当年那个房子的修建状况、出资状况。那么请问张丙、张丁说的话分别属于民事诉讼中的哪种证据？

哎，一个人说的话是什么证据？你得看他的身份和地位吧——

- **张丙**是什么人？**共同原告**啊。原告说的话叫什么？叫**当事人陈述**。
- **张丁**是什么人？**不是本案的当事人**，是**案外人**。那案外人说的话叫什么？叫**证人证言**。

对不对？好，这是常考的——我们就强调这两个问题啊，其他的大家了解一下就行了。


### 六、代表人诉讼

#### （一）代表人诉讼的基本概念

那么看到第八十一页——第四，**代表人诉讼**。

那代表人诉讼什么意思呢？我们刚才讲共同诉讼是一方或双方为两人以上，那好，在两人以上的基础之上，人数进一步增多，那么就达到了一个叫**人数众多**——什么叫人数众多呢？**十人以上**。

那此时就出现了一个问题——你们一方或者双方人数众多，十人以上，你们都到法庭上来，法庭上坐不下呀，对吧？就算坐下了，书记员也疯了——大家都在七嘴八舌地说，我怎么记呀，我记谁的，对不对？

所以那这个时候怎么办？你们这么多人数众多的一方当事人，你们不要都来了，**你们选两到五个代表人**，代表你们参加诉讼就行了。

**代表人诉讼**，是指一方或双方人数众多（十人以上），有人数众多的一方当事人推举代表**两到五人**，代表本方当事人进行诉讼活动。

那首先我们一个前提——一方或双方人数众多，十人以上，首先满不满足两人以上的条件啊？满足嘛。所以**代表人诉讼本质上一定也是共同诉讼**，只不过在共同诉讼的基础之上，人数进一步增多，达到了十人以上，对吧？


#### （二）代表人的诉讼权利

好，那接下来，有人数众多的一方当事人推举代表两到五人，代表本方当事人参加诉讼——那这个推举出来的代表人有什么诉讼权利呢？

来注意，**关键看这个代表人干的什么事儿**——

- 如果代表人所为的是一些**一般的诉讼行为**（程序性事项），直接对所代表的当事人生效，**不需要经过被代表的当事人同意**。
- 如果是代表人要**承认、放弃、变更诉讼请求、和解、调解**等涉及到处分当事人实体权利的事情，那必须**经过被代表的当事人同意**——谁同意对谁生效，谁不同意对谁就不生效。

所以考试关键看清楚——这代表人干的什么事儿啊。


#### （三）代表人诉讼的真题

**【真题1·在线诉讼】**

下面有两个真题啊，八十一——

甲公司对二十名债务人提起借款合同纠纷诉讼，二十名债务人集体推选罗某、胡某担任代表人参加诉讼。这个案件原告是甲公司，被告是二十名债务人——被告一方人数众多吧。那这个时候这二十个债务人就推选了罗某、胡某担任代表人代表他们进行诉讼。

合议庭由审判长付某、法官赵某和人民陪审员宋某组成。后来法院拟采取在线方式开庭。同学们，在线开庭我们之前讲过啊——**在线诉讼必须经过当事人同意**，谁同意对谁用。

那么关于在线开庭的条件表述正确的是——

**A.** 必须经过甲公司同意后可以对甲公司在线开庭。（对甲公司在线开庭必须经过甲公司同意——**A选项对的**）

**B.** 必须经二十名债务人全部同意后才能对被告方在线开庭。（还是说必须经罗某、胡某同意后对被告方在线开庭？同学们，同意在线开庭这只是一个诉讼行为吧？没有去处分当事人实体权利吧？不涉及什么承认放弃变更诉讼请求、和解调解上诉反诉等处分当事人实体权利的事情。那代表人所谓的诉讼行为直接对所代表的当事人生效——那代表人罗某、胡某同意在线诉讼的行为直接就对被代表的这二十名债务人生效。所以对被告方在线诉讼只需要经过代表人罗某、胡某同意就行，不需要经过这二十名债务人同意啊——**B选项错误**）

**C.** 正确〖注：原文此处内容缺失，根据讲授者论述逻辑，C选项应为“经罗某、胡某同意后可以对被告方在线开庭”〗。

**D.** 必须经过宋某同意后才能在线开庭。（宋某是什么人？人民陪审员。在线开庭要经过人民陪审员同意？不需要嘛）

所以这个题选什么？**A和C**。


**【真题2·代表人调解】**

再翻过来，八十二——

盛发公司向消费者销售礼品券，因产品质量与承诺不符，共有包括甲在内的一百多名消费者起诉盛发公司。诉讼中，原告选定彭某、张某担任代表人，代为进行诉讼。被告方是盛发公司。

诉讼中法院组织代表人彭某、张某与盛发公司达成调解协议，多数消费者均同意调解，甲不同意调解，仅要求法院作出判决。

同学们看到——代表人彭某、张某是在干什么？跟对方调解——调解达成调解协议，要不要处分实体权利啊？**要**。那此时代表人所谓的行为必须经过被代表的当事人同意后有效。

那这个时候多数消费者均同意调解——那这个调解对多数消费者生效没问题。甲不同意调解——那么代表人彭某、张某与对方达成的调解协议对甲**不生效**，因为甲不同意。

既然调解对甲不生效，那法院应该怎么办啊？那就应当**对甲继续审理后作出判决**嘛，对吧？

所以看到——

**A.** 法院应当制作调解书，这调解书只对那个同意调解的多数消费者生效，没问题。告知甲另行起诉——哎，我已经委托代表人彭某、张某代表我进行诉讼了，我已经提起诉讼了，凭什么告知另行起诉呀？没道理嘛。我就要在这个诉讼中，你法院就得给我审理，就得给我做出判决，凭什么让我另行起诉？——**A选项错误**。

**B.** 法院应当制作调解书，应当依法审理并作出判决——错。多数消费者同意调解了，那调解对多数消费者是生效的，你为什么不能做调解书呢？——**B选项错误**。

**C.** 法院应当制作调解书，对全体当事人生效——错。调解涉及到当事人实体权利，甲不同意，对甲不生效啊——**C选项错误**。

**D.** 法院应当制作调解书，但调解书对甲不生效——对。那法院对甲应当怎么办？继续审理，作出判决嘛。

这个题选**D**。


#### （四）代表人诉讼的分类

那接下来，代表人诉讼的分类——我们根据被代表的当事人一方人数众多，人数能否确定，把它分为**人数确定的代表人诉讼**和**人数不确定的代表人诉讼**。

- 起诉时人数众多一方当事人人数确定的，叫**人数确定的代表人诉讼**。
- 起诉时人数众多一方当事人人数不确定的，叫**人数不确定的代表人诉讼**。

哎，哪有起诉人数不确定的？当然有啊——比如说长江上游的化工厂排污污染长江，导致长江中下游的养殖户纷纷受到损害，纷纷来起诉。那么请问有多少养殖户来起诉呀？不知道呀。这个就叫人数不确定的代表人诉讼。


#### （五）人数确定的代表人诉讼

来，一个一个说啊——

我们先讲**人数确定的代表人诉讼**。在起诉时人数众多一方当事人人数确定的，这个叫人数确定的代表人诉讼。

我们刚才说代表人诉讼的一定是共同诉讼吧？那人数确定的代表人诉讼是普通共同诉讼还是必要共同诉讼？**都有可能**。

同学们还记得吗？我们之前举的那个四十个人坐大巴车的例子——

四十个人乘坐旅游公司的大巴车外出旅游，结果发生交通事故，车上四十名乘客不同程度受到损害，纷纷起诉旅游公司要求赔偿。本案原告方几个人？四十个人——这叫**人数确定的代表人诉讼**，对不对？这是普通共同诉讼还是必要共同诉讼？**普通共同诉讼**，对吧？

那再来——四十名乘客把这大巴车抬起来扔下了悬崖，旅游公司起诉这四十名乘客赔偿。本案被告方四十个人吧——**人数确定的代表人诉讼**，对不对？这是普通共同诉讼还是必要共同诉讼？**必要共同诉讼**，对吧？

所以都有可能。

**代表人怎么确定？**

由当事人推选。怎么推选的？来——

**第一种**：这四十个乘客一致同意张三、李四、王五担任我们的代表人——可不可以？可以。张三、李四、王五就代表这四十个人参加诉讼就行了。这个叫**全体当事人推选共同的代表人**，没问题。

**第二种**：这四十个人里边分了帮派——前十三个人选张三，中间十三个人选李四，后边十三个人选王五——可不可以啊？可以。张三、李四、王五分别代表选他们的那部分人参加诉讼。这个叫**部分当事人推选自己的代表人**，没问题。（哎，每组十三个人，三组三十九个——还差一个，是的。还差一个就是那种你让他选张三他不同意，你让他选李四他不同意，你让他选王五他也不同意，选不出代表来的人。）

那**选不出代表人**怎么办？

选不出代表人——那这个人是什么？自己上啊？不一定哦，要分情况讨论——

- 如果是**普通共同诉讼**，反正是若干独立可分之诉，你选不出代表人——**另行起诉**就好了。
- 如果是**必要共同诉讼**，那必须一次性搞定——那就**亲自参加诉讼**。

所以人数确定的代表人诉讼中，代表怎么确定？可以选——全体的选全体的，部分的选部分的，都行。选不出代表人的分情况讨论——普通共同诉讼另行起诉，必要共同诉讼亲自参加。


#### （六）人数不确定的代表人诉讼

那接下来——**人数不确定的代表人诉讼**。

起诉时人数众多一方当事人人数不确定的，这个叫人数不确定的代表人诉讼。

那首先——人数不确定的代表人诉讼，它是共同诉讼吗？是普通共同诉讼还是必要共同诉讼啊？

注意——**人数不确定的代表人诉讼一定是普通共同诉讼，不可能是必要共同诉讼**。

这个为什么呢？我们之后会给大家讲，起诉的时候需要满足一个起诉条件——其中有一个要求是**有明确的被告**。

假如这个人数不确定的代表人诉讼是必要共同诉讼，那就必须一次性搞定——原告来法院起诉，法官问原告：“你告谁呀？”原告说：“不知道，还没确定呢。”本案有明确被告吗？没有啊，你还没确定呢，对不对？那就不符合起诉条件，法院就裁定不予受理了。

那如果人数不确定的代表人诉讼是普通共同诉讼——那无所谓，反正是若干独立的可分之诉，你确定的几个告几个，没确定的以后再说。

所以**人数不确定的代表人诉讼一定是普通共同诉讼，不可能是必要共同诉讼**。


#### （七）人数不确定的代表人诉讼的程序

好，那人数不确定的代表人诉讼，这个诉讼程序里边有些特殊程序——来我们给大家捋一遍。

比如说这个化工厂排污，污染河流，导致河流下游的一些养殖户受到损害。你觉得这些受到损害的养殖户会手拉手一起到法院来起诉吗？那肯定不会嘛，一般都是陆陆续续一个一个来。

首先张三来起诉化工厂了——法院觉得就是一个普通的侵权案件嘛，也就受理了。过段时间李四又来起诉，再过段时间王五又来起诉——法院发现本案潜在的受害人可能很多，这可能是一个人数不确定的代表人诉讼。

那我一个一个受理、一个一个审，这案件能审到后面吗？没法搞，对不对？于是法院就准备把这个案件做成一个人数不确定的代表人诉讼来处理。

**第一步：确定权利人范围**

法院首先去确定一下权利人范围——河流上游某处的化工厂排污，那只有这个地方往下游多少公里范围之内的养殖户才属于潜在的受害者吧。所以法院确定权利人范围——某某河流上游某处及其往下游多少公里范围之内的养殖户，可能是本案的潜在受害者。

**第二步：发布公告**

法院就发布公告。同学们告诉说，法院发布公告的目的是什么？是**敦促这些所有的潜在受害者向法院来登记权利**，对不对？

**第三步：权利人登记**

公告期间就陆陆续续有一些权利人向法院来登记权利了。

**第四步：确定代表人**

公告期间届满，法院就组织这些参加登记的权利人确定代表人参加诉讼。代表人怎么确定呢？

第一，参加登记的权利人可以推选代表人——全体的选全体的，部分的选部分的，都行。

那如果推选不出来——河流下游的养殖户之间相互认识吗？不认识啊。我都不认识你，我敢让你代表我参加诉讼、我来承担后果吗？所以一般大家不认识不了解，推选不出来。

那法院理解说——你们相互不认识不了解，推选不出来。那这样啊，本院认为张三李四王五水平不错，推荐张三李四王五担任大家代表人，大家觉得怎么样啊？这是**法院提出人选与当事人协商**。

协商不成——你想啊，让你们自己选选不出来，法院提出张三李四王五跟你们协商，你们不同意——得了，别选了，别协商了，本院依法指定张三李四王五担任代表人，代表大家参加诉讼。这个叫什么？**指定**。

所以大家发现，在人数不确定的代表人诉讼中**一定会出现代表人**——你们选吗？选不出来；法院提出人选协商嘛；协商不成，法院就指定嘛。

**第五步：代表人参加诉讼**

确定代表人之后，就由代表人代表参加登记的权利人参加诉讼。

**第六步：法院作出判决**

法院经过审理后对本案作出判决。

那我的问题是——法院经过审理后作出判决，这个判决对哪些人具有直接约束力？是不是只对代表人？

**不是**——代表人是代表参加登记的权利人参加诉讼，所以**法院的判决对所有参加登记的权利人具有直接约束力**。没问题。

**第七步：未参加登记的权利人**

那有没有一些权利人他没有去参加登记啊？有。

未参加登记的权利人怎么办？反正是普通共同诉讼嘛，若干独立可分之诉——那没有参加登记的权利人，可以**另行起诉**。

那另行起诉的，法院经过审理，认为其诉讼请求能够得到支持的——怎么办？**直接下达裁定，裁定适用之前的判决**。

也就是说，人数不确定的代表人诉讼，法院所作生效判决——
- **对参加登记的权利人具有直接约束力**
- **对没有参加登记而另行起诉的人，虽然没有直接约束力，但有个预决效力**

这个叫人数不确定的代表人诉讼。听清楚没有？


### 七、证券纠纷特别代表人诉讼

好，那么接下来我们补充一个小专题——来大家把讲义看到第八十四页，八十四页——**证券纠纷特别代表人诉讼**。

这是证券法规定的一个特别的代表人诉讼。

#### （一）背景举例：康某药业案

我们看到下面有个原理逻辑——比如说，康某药业因为信息披露违法违规被证监会处罚。顾某、刘某等十一名投资者向广州中院提起诉讼。

广州中院对侵权事实性质进行审查后，确定权利人范围后，发布权利登记公告。

首先可能是有几个投资者陆陆续续来起诉康某药业。法院开始当作普通的纠纷受理了，但是权利人陆陆续续来起诉，法院就发现这个案件的受到损害的投资者人数可能比较多——那可能是一个人数不确定的代表人诉讼，对吧？

#### （二）普通人数不确定的代表人诉讼的痛点

那法院怎么办呢？来——

**第一种情况：** 如果按照正常的人数不确定的代表人诉讼来处理——

法院确定权利人范围（某年某月某日买入了康某药业股份，至某年某月某日止仍然持有的投资者），然后法院发布公告，公告期间陆陆续续有些权利人到法院来登记权利。公告期满，法院就组织参加登记的权利人确定代表人参加诉讼——推选、协商、指定，反正要搞出代表人来。代表人代表参加登记的权利人参加诉讼，法院作出判决——这个判决对所有参加登记的权利人具有直接约束力。未参加登记的权利人可以另行起诉。

这个没问题——这是我们讲的一般的、正常的人数不确定的代表人诉讼。

但是大家发现啊——如果按照这种正常的人数不确定的代表人诉讼进行，它会有几个**痛点**：

**痛点一：实力不对等**

参加登记的权利人就是这些投资者——股民嘛。这帮投资者都是什么人呢？要么像老师这种几百万几百万买的“大韭菜”，要么像戴老师这种买了一两万块钱的“小韭菜”——甭管大韭菜小韭菜，都是韭菜呀。对手是谁？对手是拥有上亿、上十亿市值的上市公司。你看这帮韭菜担任代表人，跟这个拥有上十亿市值的上市公司——不论人力、物力、财力上能够抗衡吗？**抗衡不了**。所以投资者的权利就很难得到维护。

**痛点二：维权成本高**

尤其是对于小投资者——投了两万块钱进去，受损也就是几千块钱。要维护权利，需要天天盯着法院的公告，按照公告要求的时间去登记权利，登记之后还得参加推选代表人——搞了半天最后法院下判决。为了维护这几千块钱的权利，费时费力。不维护吧，又心有不甘。

就这么两个痛点。

#### （三）特别代表人诉讼的解决方式

于是证券法就规定了一种叫**证券纠纷特别代表人诉讼**。

**情形二：** 法院不是确定权利人范围、发布公告了吗？好，那此时有一帮投资者找到了**投资者保护组织**（叫**投保机构**）。

投资者保护组织有哪些呢？大家看到讲义的第八十四页下面一个脚注——目前来说有两个：第一个叫**投资者服务中心**，第二个叫**投保基金公司**。这两家投资者保护机构的实力有多强呢？注册资本大几十个亿吧。

那这个时候有一帮投资者——比如**五十名以上**的投资者——去找到了这个投资者保护机构，委托这个投资者保护机构作为**特别代表人**，代表我们去参加诉讼。

那投资者保护机构受五十名以上投资者委托，就向法院申请作为特别代表人参加诉讼。

法院收到这个申请之后，就把这个投资者保护机构作为特别代表人代表大家进行诉讼的情况**再次进行公告**。公告的目的呢，就是看看你们一方投资者同不同意这个投资者保护机构作为特别代表人代表你们参加诉讼。

公告期间——如果有投资者**不同意**这个投资者保护机构代表你参加诉讼的，那你就**声明退出**（主动声明退出）。

如果公告期间届满——此时投资者保护机构就直接**为没有声明退出的权利人登记权利**。

那怎么确定权利人呢？很简单——法院不是确定了权利人范围吗？投资者保护机构就拿这个权利人范围去找证券登记结算机构把名单调出来不就完了吗？

投资者保护机构直接为那些未声明退出的权利人向法院登记权利。然后，投资者保护机构作为特别代表人，就代表那些**没有声明退出的所有权利人**参加诉讼。

法院经过审理后作出判决——这个判决对谁生效？对参加登记的权利人生效。参加登记的权利人是哪些人？**只要没有声明退出的所有投资者**，投资者保护机构都会为你去登记权利。所以法院的判决对那些没有声明退出的所有投资者、所有权利人具有**直接约束力**。

而声明退出的权利人，可以**另行起诉**。

#### （四）特别代表人诉讼解决了什么痛点？

大家发现——在这个证券纠纷特别代表人诉讼当中，它就解决了我们刚才说的两个痛点：

**第一个痛点——实力不对等：** 代表这些投资者进行诉讼的是什么人？**投资者保护机构**。注册资本动辄大几十个亿——实力足以跟上市公司抗衡。所以能够更加有效地维护投资者的权利。

**第二个痛点——维权成本高：** 比如说对我这种小投资者，我什么都不需要干——我只要不去行使声明退出，我什么都不需要干，投资者保护机构就会主动为我登记权利，然后代表我参加诉讼，法院判决就直接对我生效。我什么都不要干，只要不声明退出——我在家里面躺着，钱就到账了。所以更有利于中小投资者维护权利，减少他们维权的成本。

#### （五）特别代表人诉讼与普通代表人诉讼的区别

那么同学们比较一下——这种证券纠纷特别代表人诉讼相较于普通的（人数不确定的）代表人诉讼，有几个区别：

| | **普通人数不确定的代表人诉讼** | **特别代表人诉讼** |
|---|---|---|
| **代表人** | 在权利人当中推选、协商、指定 | 投资者保护机构（五十名以上投资者委托） |
| **权利人参加方式** | 权利人需要**主动登记**权利 | 权利人**只要不声明退出**，投保机构主动为你登记 |
| **判决效力** | 只对**主动登记**的权利人生效 | 对**没有声明退出**的所有权利人生效 |

#### （六）真题

来，我们看到下面一个题目——八十五页。

甲公司是一家上市公司，在公募中虚构业绩，导致大量投资者损失，众多投资者纷纷提起诉讼。法院经过审查后确定有五千八百零八名投资者受到甲公司虚报业绩的损害。

某证券投资者保护基金公司受八十名投资者委托（满足五十名以上）担任特别代表人向法院提起诉讼。

郭某一人声明退出——那投资者保护机构就会为除郭某之外的其他五千八百零七名投资者向法院登记权利，然后就代表除郭某之外的五千八百零七名投资者进行诉讼。

法院所作判决——对没有声明退出的另外五千八百零七名投资者生效，对郭某**不生效**。那郭某可以怎么办？**另行起诉**。

看选项——

**A.** 如果代表人胜诉，判决约束八十名投资者——对不对？**不是**，约束除郭某外的其他五千八百零七名投资者。——**A选项错误**

**B.** 如果代表人胜诉，判决约束除郭某外的五千八百零七人——对不对？**对**。——**B选项正确**

**C.** 如果代表人胜诉，胜诉判决约束全部五千八百零八名投资者——对不对？**错**，郭某声明退出了，不约束郭某。——**C选项错误**

**D.** 如果代表人败诉，判决只约束投保基金，所有投资者都可以另行起诉——对不对？**错**，那他还代表什么啊？——**D选项错误**

所以这个题选什么？选正确的——**B**。


### 八、本章回顾：当事人与共同诉讼

好同学们，在这个地方呢，我们把第五章的“当事人”以及第六章的“共同诉讼”一起给大家回顾一遍——


#### （一）当事人

首先我们讲了第一个概念叫**当事人**。这一章当中我们讲了三个概念——

**1. 诉讼权利能力**——以自己的名义起诉、应诉的资格。

哪些主体有诉讼权利能力？
- 自然人：始于出生，终于死亡
- 法人：始于成立，终于终止（终止的标志是什么？**注销**）
- 符合条件的其他组织：始于成立，终于终止

**2. 诉讼行为能力**——不仅要以自己的名义起诉应诉，而且还得亲自参加诉讼、行使权利、履行义务。
- 法人和其他组织的诉讼行为能力跟诉讼权利能力**同时产生、同时消灭**（始于成立，终于终止）
- 自然人：只有民法中的**完全民事行为能力人**才具有诉讼行为能力；**无民事行为能力人和限制民事行为能力人**都不具有诉讼行为能力，都不能亲自参加诉讼，都需要**法定代理人**代为诉讼。

**3. 当事人适格**——最重要的问题。

**原则：** 本案所争议的**实体法律关系的双方当事人**，就是适格当事人。

**例外：** 有些主体虽然不是争议的实体法律关系双方当事人，但依然是适格当事人——三类例外：
- 确认之诉中对诉讼标的享有**确认利益**的人
- **依法或者依照他人授权**对他人民事法律关系、民事权利享有管理权的人
- 依法有权提起**公益诉讼**的机关和组织

重点我们讲的是**原则**——原则上本案所争议的实体法律关系双方当事人就是适格当事人。

所以这个地方我们需要**结合民法典的规定**来判断民诉中的适格当事人。怎么结合呢？民法典规定的**实体的赔偿义务人**，那就是诉讼中的**适格被告**。

民法典当中规定赔偿义务人总共有几种规定方式？**三种**——
1. 什么什么事儿由**A承担责任**（直接规定赔偿义务人）
2. 什么什么事儿由**A、B承担连带责任**
3. 什么什么事儿由**A承担责任**，**B呢要满足一定实体条件**（有过错呀、违约呀、管理职责呀等等）才承担相应责任或补充责任

在这三种规定之下，我们如何确定民诉中的适格当事人呢？

**第一种：** A承担责任——诉讼中直接**A为被告**。

**第二种：** A、B承担连带责任——连带责任就意味着**权利人有选择权**：
- 权利人主张A承担责任 → A为被告
- 主张B承担责任 → B为被告
- 主张A、B都要承担责任 → A、B作为**共同被告**

**第三种：** A承担责任（A一定为被告，跑不了），B满足一定实体条件才承担责任——B满不满足实体条件，起诉时不知道呀，所以**由权利人主张而定**。两种搞法：
- 权利人主张B承担责任 → A、B做**共同被告**
- 权利人没有主张B承担责任 → 只能以**A为被告**，B不做被告


#### （二）共同诉讼

讲完当事人之后——好，如果一方或双方人数众多，一方或双方为两人以上，这就产生了**共同诉讼**。

共同诉讼我们区分为**普通共同诉讼**和**必要共同诉讼**。

**区分方法：**
- **普通共同诉讼**：一方或双方两人以上，诉讼标的**同种类**——说明有好几个诉讼标的，本质上就是若干**独立可分之诉**。只不过为了提高诉讼效率，征得当事人同意合并审理。
- **必要共同诉讼**：一方或双方两人以上，诉讼标的**同一**——只有一个诉讼标的，只有一个诉，法院必须**合并审理、合一判决**。

**共同诉讼人之间的关系：**
- **普通共同诉讼**：共同诉讼人之间**相互独立**
- **必要共同诉讼**：共同诉讼人当中**一人行为经其他共同诉讼人同意后，对其他共同诉讼人生效**


#### （三）代表人诉讼

在共同诉讼的基础之上，人数进一步增多，就出现了一种诉讼叫做**代表人诉讼**。

代表人诉讼是一方或双方人数众多——人数众多指的是**十人以上**。有人数众多的一方当事人推举代表**两到五人**，代表当事人参加诉讼。

**代表人的诉讼权利：**
- 一般的程序性行为 → 直接对所代表的当事人生效，**不需要**经过被代表的当事人同意
- 承认、放弃、变更诉讼请求、和解、调解等处分实体权利的行为 → 必须**经过被代表的当事人同意**后有效

**代表人诉讼的分类：**

**1. 人数确定的代表人诉讼**

代表人诉讼一定是共同诉讼。人数确定的代表人诉讼**有可能**是普通共同诉讼，**有可能**是必要共同诉讼。

代表人怎么确定？由当事人推选——全体的选全体的，部分的选部分的，都行。

选不出代表人怎么办？分情况讨论——
- 必要共同诉讼 → **亲自参加**
- 普通共同诉讼 → **另行起诉**

**2. 人数不确定的代表人诉讼**

形式上**必须是普通共同诉讼**，不可能是必要共同诉讼。

**程序：**
1. 法院确定权利人范围
2. 法院发布公告（敦促权利人登记权利）
3. 权利人向法院登记权利
4. 公告期满，组织参加登记的权利人确定代表人（推选→协商→指定）
5. 代表人参加诉讼
6. 法院作出判决——**对参加登记的权利人具有直接约束力**
7. 未参加登记的权利人——可以**另行起诉**（法院经审理认为其诉讼请求成立的，**裁定适用之前的判决**——预决效力）

**3. 证券纠纷特别代表人诉讼**

相较于普通人数不确定的代表人诉讼的区别——
- **代表人**：投资者保护机构（而非权利人中的代表）
- **权利人参加方式**：**默示加入、明示退出**（不声明退出即视为参加，投保机构主动登记）
- **判决效力**：对**没有声明退出**的所有权利人直接生效
