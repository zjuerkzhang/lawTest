# 第1部分：法考备考提纲


## 一、诉的要素

### 1. 核心概念

| 概念 | 含义 | 关键词 |
|------|------|--------|
| **诉的要素** | 诉由三个要素构成：主体（当事人）、客体（诉讼标的）、理由 | 三要素 |
| **诉讼标的** | 双方当事人发生争议，并请求法院作出裁判的**实体法律关系** | 核心：**关系** |
| **诉讼请求** | 原告基于诉讼标的提出的具体权利要求 | 基于诉讼标的的具体要求 |

### 2. 诉讼标的 vs 诉讼请求——区分方法

| 对比项 | 诉讼标的 | 诉讼请求 |
|--------|----------|----------|
| 本质 | 实体法律关系 | 基于法律关系提出的具体要求 |
| 判断方法 | 看选项带不带“关系”二字 | 看原告具体要什么 |
| 数量关系 | 一个诉讼标的可对应多个诉讼请求 | 多个诉讼请求可基于同一诉讼标的 |

**【例题1】** 相邻权纠纷：李四起诉张三要求清除建筑材料——诉讼标的是相邻权法律关系；诉讼请求是清除建筑材料。

**【例题2】** 借款纠纷：张三起诉李四要求归还本金、支付利息、支付违约金——**一个**诉讼标的（借款法律关系）；**三个**诉讼请求。

**【例题3】** 租赁合同纠纷：张三从“要租金”变为“要解除合同”——诉讼请求变了，诉讼标的（租赁合同关系）没变。


## 二、诉的分类

### 1. 三种诉的类型——对比表

| 类型 | 定义 | 原告的诉讼请求 | 典型例子 |
|------|------|----------------|----------|
| **确认之诉** | 请求确认与被告之间的某种法律关系是否存在、是否有效 | 请求**确认** | 确认合同有效/无效、确认婚姻关系有效/无效 |
| **给付之诉** | 请求判令被告履行一定的义务 | 请求**给付**（财物/行为） | 要求还钱、要求赔礼道歉、要求停止侵权 |
| **形成之诉**（变更之诉） | 请求变更或者消灭与被告之间的某种法律关系 | 请求**变更/消灭** | 离婚、解除收养关系 |

### 2. 确认之诉——重点考点

**确认之诉的确认对象：必须是一种法律关系，不能是事实或行为。**

| 情形 | 是否确认之诉 | 理由 |
|------|-------------|------|
| 请求确认合同有效 | ✅ 是（积极确认之诉） | 确认的是法律关系 |
| 请求确认合同无效 | ✅ 是（消极确认之诉） | 确认的是法律关系 |
| 请求确认存在违规操作行为 | ❌ 不是 | 确认的是**行为**，不是法律关系 |
| 请求确认债权已过诉讼时效 | ❌ 不是 | 确认的是**事实状态**，不是法律关系 |

### 3. 给付之诉——给付的内容

| 给付内容 | 示例 |
|----------|------|
| 财物的给付 | 赔钱、返还财物 |
| 行为的给付（积极作为） | 赔礼道歉 |
| 行为的给付（消极不作为） | 停止侵权 |

### 4. 形成之诉（变更之诉）——重点考点

**形成之诉 = 要求变更/消灭一个既存的法律关系。一个既存的法律关系包含主体、客体、内容三个方面——变更三者之一即构成形成之诉。**

| 变更内容 | 示例 | 类型 |
|----------|------|------|
| 变更**客体** | 张三起诉李四要求解除婚姻关系 | 形成之诉 |
| 变更**主体** | 张三起诉前妻李四，请求将儿子判归自己抚养 | 形成之诉 |
| 变更**内容** | 张三代理儿子起诉前妻，要求将抚养费从两千增至三千 | 形成之诉 |

### 5. 确认之诉 vs 形成之诉——核心区别

| 对比项 | 确认之诉 | 形成之诉 |
|--------|----------|----------|
| 待处理的法律关系状态 | 是否存在/是否有效——**正在争议** | **既存且有效**——等待变更或消灭 |
| 法院的作用 | 确认该关系是否存在/有效 | 通过判决变更或消灭该关系 |


## 三、民法中的形成权与民事诉讼中的形成之诉（民诉×民法交叉考点）

### 1. 核心概念

| 概念 | 含义 |
|------|------|
| **形成力** | 生效判决具有直接变更、消灭既存法律关系的效力（判决生效之日=法律关系变更消灭之时） |
| **普通形成权** | 可通过单方意思表示（通知）变更消灭法律关系 |
| **形成诉权** | 必须通过诉讼/仲裁，由法院判决/仲裁裁决来变更消灭法律关系 |

### 2. 形成权 vs 形成之诉——关系对照表

| 形成权类型 | 变更方式 | 法律关系何时变更 | 生效判决有无形成力 | 提起的诉讼类型 |
|------------|----------|------------------|-------------------|----------------|
| **普通形成权**（如解除合同） | 通知到达/起诉状副本送达 | 通知到达/副本送达时 | ❌ 无 | **确认之诉** |
| **形成诉权**（如撤销权） | 必须提起诉讼/申请仲裁 | 法院生效判决作出时 | ✅ 有 | **形成之诉** |

### 3. 解除合同之诉——为什么是确认之诉？

根据**民法典第565条**：
- 通知解除：合同自通知到达对方时解除
- 直接起诉解除：合同自起诉状副本送达对方时解除

→ 合同都不是被**生效判决**解除的，所以生效判决**没有形成力**，不是形成之诉，而是**确认之诉**。

### 4. 撤销权之诉——为什么是形成之诉？

根据**民法典第147-151条**（重大误解、欺诈、胁迫、显失公平等）：
- 撤销民事法律行为**必须**请求人民法院或仲裁机构予以撤销
- **不能**通过单方意思表示撤销

→ 合同是被**法院生效判决**撤销的，生效判决**有形成力**，是**形成之诉**。

### 5. 债权人撤销权之诉——形成之诉

根据**民法典第538条、第539条**：
- 债权人撤销债务人与第三人的行为，**必须**请求法院撤销
- **不能**通过通知撤销

→ 法院生效判决具有形成力，是**形成之诉**。


## 四、反诉

### 1. 反诉的构成要件——五要素

| 要件 | 内容 | 注意事项 |
|------|------|----------|
| **主体** | 本诉的被告→本诉的原告 | 诉讼地位相反 |
| **牵连关系** | 本诉与反诉有牵连 | **不要求**基于相同法律关系 |
| **时间** | 一审法庭辩论终结前 | 二审可提但只准调不准判 |
| **管辖** | 向受理本诉的法院提出 | 牵连管辖（专属管辖除外） |
| **程序** | 适用同一程序 | — |

### 2. 二审中的反诉——特殊规则

| 规则 | 内容 |
|------|------|
| 能否提起 | ✅ 可以 |
| 处理方式 | 二审法院**只准调解，不准判决** |
| 调解不成 | **告知另行起诉**（保护上诉权） |
| 例外 | 当事人同意由二审法院一并审理作出判决的→**可以**判决 |

### 3. 反诉 vs 反驳——核心区分方法 ⭐⭐⭐（高频考点）

**【三步判断法】**
1. 找到**被告的主张**是什么
2. **假设去掉原告的起诉**（没有原告起诉时）
3. 被告就自己的主张能否**单独直接向法院起诉**？
   - ✅ 能 → **反诉**（独立的诉）
   - ❌ 不能 → **反驳**（不是独立的诉）

**【绝对禁止】** 不准去看“抵销”“折抵”等法律效果词——反诉和反驳都可能产生抵销效果，无法据此区分。

### 4. 反诉的独立性——不因本诉消灭而消灭

| 情形 | 处理 |
|------|------|
| 原告撤诉 | 本诉撤诉，**反诉继续进行** |
| 原告不到庭 | 本诉按撤诉处理，反诉中原告是被告→**缺席判决** |


## 五、诉的合并

### 1. 诉的合并——分类总览

```
诉的合并
├── 主体合并（一方或双方为两人以上）
└── 客体合并（多个法律关系合并审理）
    ├── 单纯合并（无牵连 → 分别审理分别判决）
    ├── 预备合并（主请求 + 预备请求）
    └── 重叠合并（多个法律关系 → 同一效果）
```

### 2. 单纯合并

| 特征 | 内容 |
|------|------|
| 法律关系之间 | **没有任何牵连** |
| 审理方式 | 合并审理，但**分别审理、分别判决** |
| 典型例子 | 同一原告告同一被告：租金纠纷 + 借款纠纷 + 侵权纠纷（三个独立的法律关系） |

### 3. 预备合并 ⭐

| 类型 | 条件 | 例子 |
|------|------|------|
| **类型一** | 预备请求的审理以**主请求得到支持**为前提 | 离婚诉讼：先审离婚，离婚成立才审财产分割 |
| **类型二** | 预备请求的审理以**主请求没有得到支持**为前提 | 确认股权之诉：确认股权不成→退一步审返还投资款 |

### 4. 重叠合并（选择合并）

| 特征 | 内容 |
|------|------|
| 效果 | 原告只想实现**一个法律效果** |
| 依据 | 可基于**若干不同的法律关系**实现同一效果 |
| 审理方式 | 法院对几个法律关系**择一或者全部**审理 |
| 典型例子 | 要求返还房屋：可基于租赁合同关系 + 可基于所有权关系 |


## 六、诉的分离

| 概念 | 内容 |
|------|------|
| **诉的分离** | 诉的合并后发现案件更复杂、效率降低→再拆开 |
| **效力** | 分离前已进行的诉讼行为，分离后**依然有效** |

## 一、诉的基本理论——概述

接下来我们看到专题二：诉的基本理论。诉的基本理论在理论学习当中很复杂，但是在考试当中考点很明确，总共四个考点：第一个叫诉的要素；第二个叫诉的分类；第三个叫反诉；第四个叫诉的合并与分离。


## 二、诉的要素

首先看到第一个问题：诉的要素。

一个诉有三个要素构成：第一，主体，叫当事人；第二，诉的客体，叫诉讼标的；第三，诉的理由。

重点考点在于诉讼标的，考诉讼标的的区分和判断。

**什么叫诉讼标的？** 诉讼标的是双方当事人发生争议，并请求法院作出裁判的一种实体法律关系。记住：诉讼标的的本质是一种实体法律关系。所以考试中很简单——问你本案的诉讼标的是什么，四个选项一看，哪个选项带“关系”二字的你就选，不带“关系”二字的绝对不能选。

**【例题1】** 张三和李四是邻居。张三买了堆建筑材料堆放在自家门口，严重影响了邻居李四家的通行。于是李四起诉张三，要求判令张三清除这堆建筑材料，保障自家通行便利。

请问：本案的诉讼标的是什么？

本案发生争议的是张三和李四之间的相邻权法律关系。所以本案的诉讼标的是张三李四之间的相邻权法律关系。而至于“清除建筑材料”“保障自家通行便利”，这是原告李四基于这个诉讼标的提出的具体的要求，这个叫**诉讼请求**。

**【例题2】** 张三李四签订了借款合同，约定张三向李四提供十万借款，李四于一年后偿还。后来逾期李四不还钱，于是张三起诉李四，要求：①归还借款本金十万；②支付利息两万；③支付逾期还款的违约金三万。

请问：本案有几个诉讼标的？

张三要求李四归还借款本金、支付利息、支付违约金，都是基于他们之间的借款法律关系。所以本案只有**一个**诉讼标的——张三李四之间的借款法律关系。而支付本金、支付利息、支付违约金，都是基于这个诉讼标的提出的三个不同的**诉讼请求**。

**【例题3】** 张三与李四签订了租赁合同，约定张三将自己家的房子租赁给李四。后来李四逾期支付租金，张三起诉李四要求李四支付租金一万。诉讼中张三说：“算了算了，租金我不要了，我要求解除租赁合同。”

本案中，原告张三变了一个东西——变的是诉讼请求还是诉讼标的？

“要租金一万”是诉讼请求，“要解除合同”也是诉讼请求，二者不一样，所以**诉讼请求变了**。但无论是要租金还是要解除合同，都是基于租赁合同关系，**诉讼标的没变**。


## 三、诉的分类

### （一）诉的分类概述

诉的分类，根据原告的诉讼请求把诉分为三类：**确认之诉、给付之诉、形成之诉**（又叫变更之诉）。

### （二）确认之诉

**定义**：原告的诉讼请求是请求法院判决确认与被告之间的某种法律关系是否存在、是否有效。

进一步区分为：
- **积极的确认之诉**：请求确认某种法律关系存在、有效。例如：张三起诉李四请求确认与李四之间的合同有效、婚姻关系有效。
- **消极的确认之诉**：请求确认某种法律关系不存在、无效。例如：张三起诉李四请求确认与李四之间的合同关系无效、婚姻关系无效。

**【重要考点——确认之诉的确认对象】**

确认之诉中待确认的**一定是一种法律关系**，而不能单纯是某种事实状态或者某种行为。

**【例题4】** 甲公司和乙公司签订了租赁合同，约定将甲公司一台设备租赁给乙公司使用。在合同履行中，甲公司发现乙公司存在严重的违规操作设备的行为。于是甲公司起诉乙公司，请求确认乙公司存在违规操作设备的行为。

请问：这是不是确认之诉？

**不是。** 甲公司请求确认的是“一种行为”，不是法律关系，不构成确认之诉。

**【例题5】** 张三是债权人，李四是债务人。债务人李四起诉债权人张三，请求确认张三对自己的债权已经过了诉讼时效。

请问：这是不是确认之诉？

**不是。** 李四确认的是“债权过没过诉讼时效”这一事实状态，不是去确认一种法律关系是否存在、是否有效，不构成确认之诉。

> **【讲授者强调】** 确认之诉确认的对象是**一种法律关系**，而不能是事实、不能是行为。甲公司起诉乙公司请求确认乙公司存在违规操作设备的行为——这不是确认之诉，这什么都不是，实践中就没有这么起诉的。这只是命题老师为了考这个考点而营造的案型。

### （三）给付之诉

**定义**：原告的诉讼请求是请求法院判令被告履行一定的义务。

履行义务的方式包括：
- 财物的给付
- 行为的给付（包括积极的作为和消极的不作为）

只要原告的诉讼请求是请求判令被告履行义务，都叫给付之诉。

**【例题6】** 左宁写了一本“小黄书”，对李家的私生活进行了详细描述，在同学之间广为流传。李家认为左宁侵犯了自己的名誉权，起诉左宁名誉侵权。李家提出的诉讼请求：①赔偿精神抚慰金十万；②公开赔礼道歉；③停止侵权行为。

分析：
- 赔钱十万——给付内容：**财物**
- 赔礼道歉——给付内容：**行为（积极的作为）**
- 停止侵权——给付内容：**行为（消极的不作为）**

这是一个典型的**给付之诉**。

### （四）形成之诉（变更之诉）

**定义**：原告的诉讼请求是请求法院变更或者消灭与被告之间的某种法律关系。

**【例题7】** 张三起诉李四请求解除婚姻关系（离婚诉讼）——形成之诉。张三和李四之间本来存在一个合法有效的婚姻关系，原告的诉讼请求是把这婚姻关系予以消灭。

**【例题8】** 张三起诉李四要求解除收养关系——形成之诉。

### （五）诉的分类——深度拓展

#### 拓展一：诉的分类的前提——必须是一个“诉”

**【例题9】** 张三李四侵权纠纷，经人民调解委员会调解达成协议，约定李四赔偿张三五千块钱。达成协议后，张三李四向法院申请确认调解协议效力。法院受理后适用特别程序审查，符合条件的下达裁定确认调解协议效力。

请问：这是个什么诉？

**注意：** 诉的分类的前提是——它得**是一个诉**。而确认调解协议效力是**特别程序**，不是诉讼程序，不是一个诉，所以**不能说是确认之诉**。

> **【拓展结论一】** 诉的分类的前提是它得是一个诉。考试中经常用特别程序（确认调解协议效力、确认财产无主、确认公民为无限制民事行为能力人等）来干扰，这些都不是诉讼程序，不是一个诉，不准讨论诉的分类。

#### 拓展二：诉的分类的唯一依据是原告的诉讼请求

**【例题10】** 张三起诉李四要求归还借款十万，李四说：“没借过钱，借款关系不存在。”

请问：这是个什么诉？

**分析角度一**：诉的分类的唯一依据是**原告的诉讼请求**。本案中原告张三的诉讼请求是“还钱”，还钱是典型的给付请求，所以是**给付之诉**。被告的答辩意见是干扰信息，不准去看。

**分析角度二**：给付之诉、形成之诉中往往可能包含确认的内容——给付之诉履行合同，前提得确认合同有效；形成之诉解除婚姻关系，前提得确认婚姻关系有效。但只要原告提出了给付请求，一定是给付之诉；只要原告提出了变更、消灭的请求，一定是形成之诉，不可能再是确认之诉。

> **【拓展结论二】** 诉的分类的唯一依据是原告的诉讼请求，不准去看被告的答辩意见。
>
> **【拓展结论三】** 给付之诉、形成之诉中往往可能包含确认的内容，但只要原告提出了给付请求，一定是给付之诉；只要原告提出了变更、消灭的请求，一定是形成之诉，不可能再是确认之诉。

#### 拓展三：消极的确认之诉与形成之诉的区别

**【例题11】** 张三起诉李四请求确认婚姻关系无效——这是什么诉？**确认之诉（消极的确认之诉）** 。

**【例题12】** 张三起诉李四要求解除婚姻关系（离婚）——这是什么诉？**形成之诉**。

**区别**：
- **确认之诉**中，待确认的那个法律关系是否存在、是否有效，是本案正在争议的问题，等待法院通过判决来确认。
- **形成之诉**中，待变更、待消灭的那个法律关系本身**一定是既存而有效的**，只是希望法院通过判决把它消灭、变更掉。

#### 拓展四：形成之诉——变更法律关系的主体、客体或内容

**【例题13】** 张三起诉前妻李四，请求将婚生子张小三判归自己抚养。

请问：这是什么诉？

有同学说是给付之诉，但给付的内容是什么？财物？把自己的儿子当财物不合适吧？

本来在前妻李四与儿子张小三之间存在一个既存的抚养权利义务关系。现在张三的诉讼请求是把这个抚养权利义务关系**变为自己与儿子之间存在**——这是要求对这个既存法律关系的**主体进行变更**。这叫**形成之诉（变更之诉）** 。

**【例题14】** 张三代理儿子张小三，起诉前妻李四，请求将张小三的抚养费从每月两千增加到每月三千。

请问：这是什么诉？

本来在前妻李四与儿子张小三之间存在一个抚养权利义务关系，每月两千。现在原告的诉讼请求是把两千变为三千——这是对这个既存法律关系的**内容（数额）进行变更**。这叫**形成之诉（变更之诉）** 。

> **【拓展结论五】** 形成之诉（变更之诉）中，原告的诉讼请求是要求变更、消灭一个既存的法律关系。一个既存的法律关系包含**主体、客体、内容**三个方面。只要原告的诉讼请求是要求将这三者之一进行变更，就是要求对这个既存法律关系进行变更，就叫形成之诉（变更之诉）。


## 四、民法中的形成权与民事诉讼中的形成之诉

> **【注释】** 以下内容涉及民法实体法（民法典）与民事诉讼法程序法的交叉融合，属于民诉法中的重要考点。

### （一）形成力的概念

形成之诉中，原告胜诉的判决应当具有**形成力**——即生效判决具有直接变更、消灭既存法律关系的效力。判决生效之日，即为法律关系变更、消灭之时。

例如：离婚诉讼中，法院判决解除婚姻关系，判决生效之日婚姻关系即告消灭——生效判决直接消灭了既存的婚姻关系，这就是形成力。

### （二）解除合同之诉——为什么不是形成之诉？

**【例题15】** 张三和李四签订了一个买卖合同，李四存在严重违约行为，张三起诉李四请求解除合同。

请问：这是什么诉？

很多人会认为这是形成之诉，但**不对**。

根据民法典第565条的规定【**校正：原文为“民法典五百六十五条”** 】，解除合同有两种方式：

**方式一：通知解除。** 当事人一方依法主张解除合同，应当通知对方。合同自通知到达对方时解除。对方对解除合同有异议的，任何一方当事人均可以请求法院或者仲裁机构确认解除行为的效力。

→ 合同是被**通知**解除的，不是被生效判决解除的。生效判决没有形成力。之后提起的诉讼只是对合同是否解除进行确认——这是**确认之诉**。

**方式二：直接起诉解除。** 当事人一方未通知对方，直接以起诉方式主张解除合同，法院确认该主张的，合同自起诉状副本送达对方时解除。

→ 合同是被**起诉状副本送达**解除的，不是被生效判决解除的。生效判决没有形成力。之后进行的诉讼只是对合同是否解除进行确认——这是**确认之诉**。

> **【结论】** 基于民法典对解除合同一律采取“通知到达主义”（通知到达或起诉状副本到达时合同解除），并非生效判决直接解除合同，所以解除合同的生效判决**没有形成力**，**不是形成之诉**，而是**确认之诉**。

### （三）形成权与形成之诉的关系

民法中的**形成权**，区分为两种：

1. **普通形成权**：当事人可以通过单方意思表示（如通知）来变更、消灭法律关系。例如：解除合同（通知解除）。
   - 法律关系自单方意思表示到达对方时即已变更、消灭。
   - 不是由生效判决变更、消灭的，生效判决没有形成力。
   - 基于普通形成权提起的诉讼，一般是**确认之诉**。

2. **形成诉权**：不能通过单方意思表示来变更、消灭法律关系，必须通过诉讼或仲裁，由法院判决或仲裁裁决来变更、消灭法律关系。
   - 法律关系是根据法院生效判决变更、消灭的，生效判决具有形成力。
   - 基于形成诉权提起的诉讼，是**形成之诉**。

**典型例子——撤销权（形成诉权）** ：

民法典第147条（重大误解）、第148条（欺诈）、第149条（第三人欺诈）、第150条（胁迫）、第151条（显失公平）规定了撤销权【**校正：原文为“民法典一百四十七、一百四十八、一百四十九、一百五十、一百五十一”** 】。这些条文都规定“当事人有权请求人民法院或者仲裁机构予以撤销”——当事人想要撤销民事法律行为，**不能通过单方意思表示**，只能提起诉讼或申请仲裁。

例如：张三与李四签订合同后，张三认为受到了欺诈，起诉李四请求撤销合同。

→ 合同是根据法院生效判决消灭的，生效判决具有形成力，这是**形成之诉**。

**另一个例子——债权人的撤销权**：

民法典第538条、第539条规定的债权人撤销权【**校正：原文为“民法典五百三十八条、五百三十九条”** 】。债权人张三对债务人李四享有一百万到期债权，李四无力清偿，但将一辆价值一百万的车无偿赠予了王五。张三可以请求法院撤销李四和王五之间的赠与合同。

→ 张三**不能**通过发通知来撤销，只能向法院起诉。赠与合同是被法院生效判决撤销的，生效判决具有形成力，这是**形成之诉**。

> **【结论】** 基于民法典中的**普通形成权**提起的诉讼，一般是**确认之诉**；只有基于民法典中规定的**形成诉权**提起的诉讼，才叫**形成之诉**。


## 五、反诉

### （一）反诉的定义

**反诉**：在本诉诉讼中，本诉的被告向本诉的原告提出的一个独立的反请求。

**【例题16】** 张三和李四打架，李四把张三打了。张三起诉李四要求赔偿医疗费八千（本诉）。诉讼中李四说：“张三你骂了我，我才打你的，我要求你张三向我赔五千。”（反诉）

→ 被告李四对原告张三提出的赔偿五千的请求，构成**反诉**。

### （二）反诉的构成要件

**1. 主体**：反诉必须由本诉的被告向本诉的原告提出。本诉与反诉的主体具有同一性，但诉讼地位正好相反（原告→被告，被告→原告）。

**2. 牵连关系**：本诉与反诉之间要有牵连关系。但**不要求**基于相同的法律关系。

**【例题17】** 张三把自己的房子租给李四，李四不给租金。张三起诉李四要求支付房屋租金五千（本诉——基于租赁合同关系）。诉讼中李四说：“张三你没有尽到修缮义务，天花板掉下来把我砸了，产生医疗费八千，赔吧。”（反诉——基于侵权法律关系）

→ 本诉基于租赁合同关系，反诉基于侵权法律关系，二者不同，但**有牵连**（都源于租赁关系）。牵连关系不要求基于相同法律关系。

**3. 时间**：反诉必须在**一审法庭辩论终结前**提出。

> **【注释】** “法庭辩论终结”是指法庭调查结束后，双方当事人围绕争议焦点发表辩论意见，辩论结束后法官宣布休庭评议。法庭辩论终结后就只剩下合议庭评议和宣判，当事人没有参与和表达的机会了，所以要提反诉最迟在法庭辩论终结前提出。

**二审中能否提反诉？**

在二审中可以提反诉。但二审法院**只准调解，不准判决**。

为什么？如果二审法院直接对反诉作出判决，那就是终审判决，对反诉而言只经过了一次审理，剥夺了反诉当事人的上诉权。

- 调解不成→**告知另行起诉**（目的是保护反诉当事人的上诉权）。
- 但如果当事人同意由二审法院一并审理作出判决的，二审法院**可以**一并审理作出判决。

**4. 管辖**：反诉直接向受理本诉的法院提出即可。受理本诉的法院基于受理本诉及本反诉之间的牵连关系，当然取得对反诉案件的管辖权——这称为**牵连管辖**。但反诉专属于其他法院管辖的除外（专属管辖除外）。

**5. 程序**：本诉与反诉应当适用同一程序审理。

### （三）反诉的理论——反诉是一个独立的诉

反诉是一个**独立的诉**。这一理论从两个角度考察：

#### 角度一：反诉与反驳的区分

**【核心方法】** ：
1. 绝对**不准**去看法律效果（即不准看“抵销”“折抵”这样的词），因为反诉和反驳都有可能产生抵销、折抵的效果，无法区分。
2. **关键区分方法**：找到被告的主张是什么，然后**假设去掉原告的起诉**——没有原告起诉时，被告就自己的主张能否单独直接向法院起诉？
   - 如果能 → 是独立的诉 → **反诉**
   - 如果不能 → 不是独立的诉 → **反驳**

**【例题18】** 原告起诉被告要求归还借款十万，被告主张“借款已经归还”。

请问：被告的主张是反诉还是反驳？

被告的主张是“借款已经归还”。假设没有原告起诉被告还钱，被告能不能直接去法院起诉原告说“我这钱已经还了”？——不能。所以这不是独立的诉，是**反驳**。

**【例题19】** 原告起诉被告归还借款十万，被告主张“借款已经过了诉讼时效”。

被告的主张是“借款已经过了诉讼时效”。假设没有原告起诉被告还钱，被告能不能直接起诉原告说“你那钱已经过了诉讼时效”？——不能。所以是**反驳**。

**【例题20】** 原告起诉被告要求履行合同，被告主张“确认合同无效，要求判令原告承担缔约过失责任”。

被告的主张是“确认合同无效，判令原告承担缔约过失责任”。假设没有原告起诉被告履行合同，被告能不能直接去法院起诉原告请求确认合同无效、判令承担缔约过失责任？——**可以**。这是一个独立的诉，是**反诉**。

**【例题21】** 原告起诉被告要求归还借款，被告主张“一个月前我卖给原告一块玉石，并言明折抵借款，现在要求用玉石价款折抵借款”。

请问：被告的主张是反诉还是反驳？

**注意**：不准去看“折抵”这个词。被告**真正的主张**是“基于玉石买卖合同关系，找原告要玉石价款”。假设没有原告起诉被告要求还钱，被告能不能直接起诉原告要求给玉石价款？——**可以**。这是一个独立的诉，是**反诉**。

#### 角度二：反诉的独立性——不因本诉消灭而消灭

既然反诉是一个独立的诉，那么案件中就有两个诉：本诉（原告告被告）和反诉（被告告原告）。本诉与反诉相互独立，反诉**不会**因为本诉的消灭而消灭。

**【情形一】** 原告起诉被告，被告反诉原告。诉讼中原告撤诉了——可不可以？可以。撤的是什么诉？**本诉**。本诉撤了，**反诉继续进行**。

**【情形二】** 原告起诉被告，被告反诉原告。诉讼中原告经传票传唤没有正当理由拒不到庭——原告有两个身份：在本诉中是原告（不到庭→按撤诉处理）；在反诉中是被告（不到庭→缺席判决）。所以：**本诉按撤诉处理，反诉缺席判决**。


## 六、诉的合并与分离

### （一）诉的合并概述

诉的合并，就是把两个诉合到一起来审理。

诉的合并分为：
- **主体合并**（当事人的合并）
- **客体合并**（诉讼标的的合并）

### （二）主体合并

只要原告一方或者被告一方（双方当事人中一方）为两人以上，就构成诉的**主体合并**。

### （三）客体合并

如果在一个案件中将好几个法律关系（诉讼标的）合并到同一程序中审理，就叫诉的**客体合并**。

根据被合并的几个法律关系之间是否存在牵连关系、存在何种牵连关系，进一步区分：

#### 1. 单纯合并

被合并的几个法律关系之间**没有任何牵连**，完全是几个独立的诉。为了提高诉讼效率，可以将它们合并在同一程序中，但法院必须**分别审理、分别判决**。

**【例题22】** 张三是房东，李四是租客。租赁合同履行期间，李四拖欠租金；张三又借给李四两万块钱；后来张三去找李四交涉时被李四家养的狗咬了。张三起诉李四要求：①支付租金一万；②归还借款两万；③赔偿医疗费八千。

三个法律关系（租赁合同关系、借款法律关系、侵权法律关系）之间没有任何牵连，完全是三个独立的诉。这是**单纯的合并**，法院必须分别审理、分别判决。

#### 2. 预备合并

原告向法院提出两个请求——**主请求**和**预备请求**。主请求法院一定要审；预备请求审不审取决于主请求的审理情况。

**类型一：预备请求的审理以主请求得到支持为前提**

主请求得到支持→进一步审理预备请求；主请求没有得到支持→不必审理预备请求。

**【例题23】** 离婚诉讼。张三起诉李四离婚，提出两个请求：①要求解除婚姻关系（离婚）；②要求分割夫妻共同财产。

如果法院判决支持离婚请求→进一步审理分割财产的请求；如果法院判决不准离婚→不必审理分割财产的请求。分割财产的请求以离婚请求得到支持为前提。这是**预备合并**。

**类型二：预备请求的审理以主请求没有得到支持为前提**

主请求得到支持→不必审理预备请求；主请求没有得到支持→退一步审理预备请求。

**【例题24】** 张三与甲公司签订投资入股合同，约定张三出资一百万购买甲公司百分之二十的股权。张三支付投资款后，甲公司一直没有完成股权过户登记。张三起诉甲公司，最想实现的是“确认我对甲公司享有百分之二十的股权”（主请求）。但张三担心法院认定投资合同无效，于是提出预备请求：“如果法院不支持我确认股权的请求，则请求法院判令甲公司返还投资款一百万。”

如果确认股权的请求得到支持→判决确认股权，不审返还投资款；如果确认股权的请求不能得到支持→退一步审理返还投资款的请求。这是**预备合并**。

**【例题25】** 张三、李四、王五、赵六共同出资成立甲公司，赵六持股20%。大股东张三操纵股东会作出公司增资决议。小股东赵六起诉甲公司请求撤销公司增资决议（主请求）。但赵六担心撤销请求得不到支持，于是提出预备请求：“如果撤销公司决议的请求不能得到支持，则请求法院确认我对增资的一千万中的20%享有优先认购权。”

如果撤销请求得到支持→公司增资决议被撤销，不审优先认购权；如果撤销请求不能得到支持→退一步审理优先认购权的请求。这是**预备合并**。

#### 3. 重叠合并（选择合并）

原告对被告只想实现**一个法律效果**，但可以基于**若干不同的法律关系**来实现同一个效果。这若干不同的法律关系产生的效果是一样的，所以称为“重叠合并”。法院根据当事人的主张，对这几个法律关系**择一或者全部**进行审理。

**【例题26】** 张三把自己的房子租给李四，租赁期限届满，李四拒不返还房屋。张三起诉李四要求返还房屋。

张三可以基于哪些法律关系来实现“返还房屋”这个效果？
- 基于租赁合同关系（租赁期限届满，李四应返还房屋）
- 基于所有权关系（张三对房屋享有所有权，李四应返还房屋）

不论基于哪个法律关系，实现的效果都是“返还房屋”。这是**重叠合并**。

**【例题27】** 甲公司承建乙公司的建设工程，工程完工后乙公司向甲公司背书转让了一张五百万的汇票作为付款。甲公司持汇票找银行承兑时被拒绝承兑。甲公司起诉乙公司要这五百万。

甲公司可以基于哪些法律关系来实现“要这五百万”这个效果？
- 基于票据上的追索权
- 基于建设工程施工合同（乙公司的付款义务）

不论基于哪个法律关系，实现的效果都是“乙公司给五百万”。这是**重叠合并**。

### （四）诉的分离

诉的合并的目的是提高诉讼效率。但合并后发现案件变得更加复杂、效率反而不利于提高，就再拆开——这叫**诉的分离**。

先合并再分离的，分离之前已经进行的诉讼行为，在分离之后**依然有效**，不需要重新进行。

### （五）真题示例

**【2022年真题】** 乙欠甲三十万。乙以四十五万的价格从丙处购买了一辆价值二十万的轿车（以明显不合理的高价购买），影响了甲的债权的实现。甲认为乙以明显不合理的高价从丙处购买财产影响其债权实现，遂向法院起诉，请求法院判决撤销乙丙之间的买卖合同，同时请求将乙已支付给丙的款项返还给乙。

**分析**：
- 甲请求撤销乙丙之间的买卖合同——撤销权诉讼
- 同时请求丙将款项返还给乙

这两个请求的关系：如果法院支持了撤销买卖合同的请求（买卖合同被撤销）→才存在丙向乙返还财产的问题；如果买卖合同没有被撤销→不需要返还。返还财产的请求以撤销买卖合同的请求得到支持为前提——这是**诉的预备合并**。

**选项分析**：
- A. 属于诉的主体合并——**对**。撤销权诉讼中，债权人甲做原告，债务人乙和相对人丙做共同被告，被告方两人以上，构成主体合并。
- B. 属于诉的从别合并【**校正：原文“从别合并”应为错别字，标准术语无此概念**】——**错**。
- C. 属于诉的预备合并——**对**。
- D. 属于诉的客体合并——**对**。预备合并本身就是客体合并的一种。

**答案：ACD**


## 七、课程回顾

> **【注释】** 以下为讲授者对已学内容的总结回顾，涉及“诉讼与非讼”的区分，属于民事诉讼基本理论范畴，与诉的基本理论密切相关，予以保留。

**（一）民事审判程序的区分**

民事审判程序区分为**诉讼**和**非讼**：
- **诉讼**：在民事审判程序中解决纠纷的，如一审、二审、再审。
- **非讼**：在民事审判程序中不解决纠纷的，如除选民资格之外的特别程序、督促程序、公示催告程序。
- 选民资格案件既不是诉讼，也不是非讼（根本就不是民事审判）。

**（二）诉讼标的**

诉讼标的是双方当事人发生争议、并请求法院作出裁判的那种实体法律关系。基于这个诉讼标的，原告可以提出若干不同的诉讼请求。

**（三）诉的分类（根据原告的诉讼请求）**

| 类型 | 内容 |
|------|------|
| 确认之诉 | 请求确认某种法律关系是否存在、是否有效 |
| 给付之诉 | 请求判令被告履行义务 |
| 形成之诉（变更之诉） | 请求法院变更或者消灭某个既存的法律关系 |

**（四）反诉**

**构成要件**：
1. 主体：本诉的被告向本诉的原告提出，诉讼地位相反
2. 牵连关系：不要求基于相同法律关系
3. 时间：一审法庭辩论终结前（二审可提但只准调不准判）
4. 管辖：牵连管辖（专属管辖除外）
5. 程序：适用同一程序

**理论**：反诉是一个**独立的诉**。
- 反诉与反驳的区分：去掉原告起诉，看被告的主张能否单独起诉
- 反诉不因本诉的消灭而消灭

**（五）诉的合并**

| 类型 | 内容 |
|------|------|
| 主体合并 | 一方或双方为两人以上 |
| 客体合并 | 单纯合并（无牵连，分别审理分别判决） |
|  | 预备合并（主请求+预备请求） |
|  | 重叠合并（多个法律关系实现同一效果） |
