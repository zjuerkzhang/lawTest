# 第1部分：备考提纲（针对法考备考）

> **说明**：以下提纲基于讲稿内容整理，对照《民事诉讼法》及相关司法解释。**加粗**为重点内容，⚠️标注特别注意事项。法律条款引用基于讲稿中提及的内容，部分条款序号为讲稿所述，建议以最新法条为准核对。


## 一、鉴定意见

### 1. 鉴定的启动
| 启动方式 | 适用情形 | 鉴定人确定 |
|---------|---------|-----------|
| **当事人申请**（原则） | 一般专业事实鉴定 | 法院组织当事人协商 → 协商不成法院指定 |
| **法院依职权**（例外） | 涉及国家利益/社会公共利益/第三人合法权益、身份关系、程序性事实 | 法院询问当事人后指定 |

**【核心公式】谁申请鉴定 = 谁承担证明责任**

### 2. 鉴定前的准备
- 鉴定人**签署承诺书**
- 鉴定材料**必须经质证**，未经质证不得作为鉴定根据

### 3. 鉴定意见的形式
- 鉴定人**个人签名** + 附资格证明
- 委托机构鉴定的：鉴定人签名 + **机构盖章**
- 多名鉴定人意见不一致 → **如实注明**，不能少数服从多数

### 4. ⚠️ 鉴定人出庭（高频考点）

**出庭情形**（二选一）：
1. 当事人对鉴定意见有异议
2. 法院认为鉴定人应当出庭

**出庭程序**（2019年证据规定）：
鉴定人提交鉴定书 → 法院送副本给当事人 → 当事人有异议提**书面异议** → 鉴定人**书面答复** → 仍有异议 → **通知出庭**

**费用**：由**败诉方**承担（因鉴定意见瑕疵导致出庭的除外）

**拒不出庭后果**：
1. 鉴定意见不得作为定案根据
2. 返还鉴定费用
3. 建议行业主管部门/组织处罚

### 5. ⚠️ 当事人自行委托鉴定的效力（易错点）

| 问题 | 答案 |
|------|------|
| 是否属于民诉法中的“鉴定意见”？ | **不属于**（无启动程序、无协商/指定、材料未经质证） |
| 能否作为证据使用？ | **可以**，作为**书证（私文书证）** |
| 对方反驳并申请鉴定？ | 法院**应当准许** |


## 二、有专门知识的人（专家辅助人）

| 考点 | 内容 |
|------|------|
| **表述** | 可称“有专门知识的人”或“专家辅助人”，**绝不能称“专家证人”** |
| **出庭方式** | 当事人**申请**（不能依职权），举证期限内，1-2名 |
| **作用** | ①对鉴定意见提意见 = **质证意见**；②对专业问题提意见 = **当事人陈述** |
| **费用** | **申请方自己承担**（不是败诉方） |
| **是否回避** | **不适用回避制度**（不期待中立） |


## 三、鉴定人 vs 有专门知识的人 vs 司法技术人员

| | 鉴定人 | 有专门知识的人 | 司法技术人员 |
|--|--------|---------------|-------------|
| **谁来请** | 法院启动程序后委托 | 当事人申请 | 法院指派/聘请 |
| **帮谁** | 法庭 | 当事人 | 法官 |
| **是否中立** | ✅ 中立 | ❌ 不中立 | ✅ 中立 |
| **适用回避** | ✅ 适用 | ❌ 不适用 | ✅ 适用 |


## 四、当事人陈述

### 1. 出庭义务
法院认为必要，可要求当事人本人到庭接受询问，须**签署并宣读保证书**。

### 2. 不出庭后果
负有证明责任的当事人拒绝到庭/拒绝签署宣读保证书，待证事实欠缺其他证据证明 → **不予认定**。

### 3. 当事人出庭的两方面作用

| 作用 | 委托代理人 | 不委托也不来 |
|------|-----------|-------------|
| 行使诉讼权利 | 代理人行为=本人行为 | 原告→按撤诉处理；被告→缺席判决 |
| 提供当事人陈述（证据） | 代理人不能替代本人陈述 | 视为无此证据，依现有证据+证明责任判 |

## 段落一：鉴定意见的概念与启动

**（一）鉴定意见的概念**

在诉讼案件审理过程中，可能需要对一些专业事实进行鉴定。这个专业事实，法官也不懂，双方当事人也不懂，那怎么办？可由当事人向法院提出申请，或者由法院依职权决定对这个事进行鉴定。咱们就把这个事交给鉴定人，由鉴定人通过专业的程序、专业的知识进行分析之后给出一份意见，这个意见叫做**鉴定意见**。鉴定意见是一种法定的证据种类。

**（二）鉴定的启动方式**

鉴定的启动，**原则上应当由当事人在法院指定的期限内申请鉴定**。如果待鉴定事实属于法院应当依职权调查的事实的，可由法院依职权决定鉴定。

【修正说明：原文“原则上应当由当事人在法院指定的期限内申请鉴定。如果待鉴定事实属于法院应当依职权调查的事实的，可由法院依职权决定鉴定。”——保留原意。】

哪些事实属于法院应当依职权调查的事实？三类：**国家利益、社会公共利益、第三人合法权益**，**身份关系**，**程序性事实**。如果待鉴定事实涉及这些，法院可以依职权决定鉴定。除此之外，鉴定的启动要由当事人申请。

**（三）谁申请鉴定 = 谁承担证明责任**

双方当事人由谁来申请鉴定？**谁对待鉴定事实承担证明责任，谁申请鉴定**。所以考试问“谁来申请鉴定”，说白了就是问“谁承担证明责任”。

道理很简单：如果这个事由我承担证明责任，我提供证据证明，证明不了就申请鉴定。如果我不申请鉴定，最后事实查不清楚，就应当由我承担证明责任的不利后果。


## 段落二：例题——环境污染侵权纠纷中的鉴定申请

**【例题一】**
甲工厂排污污染河流，导致河流沿岸的养殖户张某养殖的鱼虾死亡，张某起诉甲工厂赔偿损失12万元。甲工厂对张某主张的损失不予认可，同时主张自己的排污行为与张某鱼虾死亡之间不存在因果关系。关于本案死亡鱼虾的价值和排污行为与损害结果之间的因果关系产生争议，需要鉴定。请问：应当由谁提出鉴定申请？

**【分析】**
- 本案为侵权纠纷，具体是**环境污染致人损害**，属于**无过错责任**。
- 原告张某应证明：侵权行为、损害结果。【注释：环境污染侵权中，因果关系的举证责任分配较为特殊，依《民法典》第1230条，行为人应当就法律规定的不承担责任或者减轻责任的情形及其行为与损害之间不存在因果关系承担举证责任。讲稿中老师表述为“因果关系导给了甲工厂”，即由被告对不存在因果关系承担证明责任。】
- 被告甲工厂应证明：免责事由、不存在因果关系。
- **死亡鱼虾的价值**属于**损害结果**，由张某承担证明责任 → **由张某申请鉴定**。
- **排污行为与损害结果之间的因果关系**，由甲工厂承担证明责任 → **由甲工厂申请鉴定**。

**【答案】** 死亡鱼虾价值由张某申请鉴定；因果关系由甲工厂申请鉴定。


## 段落三：例题——合同纠纷与借款纠纷中的鉴定申请

**【例题二】**
在合同纠纷中，原告主张合同成立生效，向法院提交合同原件，被告主张合同是伪造的，关于合同的真实性需要鉴定。请问：谁申请鉴定？

**【分析】**
- 争议的是合同的**形式真实性**（被告说合同是伪造的）。
- **谁主张以私文书证证明案件事实，谁对私文书证的真实性承担证明责任。**
- 本案中，原告以合同原件证明合同成立生效 → 原告对合同真实性承担证明责任 → **由原告申请鉴定**。

**【例题三】**
借款纠纷中，被告主张借款已经归还，提供了一张收条，原告主张收条是伪造的，关于收条的真实性需要鉴定。请问：谁申请鉴定？

**【分析】**
- 收条是**私文书证**，争议的是**形式真实性**。
- 被告希望通过收条证明还款事实 → 被告对收条真实性承担证明责任 → **由被告申请鉴定**。


## 段落四：真题实例——收货确认单签名的真实性鉴定

**【真题实例】（2025年真题）**
王某向李某供货，后李某拒不支付货款，王某起诉李某要求支付货款。诉讼中，王某向法院提供了23张由李某签名的收货确认单。李某主张其中9张收货确认单上签名是伪造的，法院经审查，该9张收货单上的签名笔迹确实与其他14张明显不同。关于本案说法正确的是？

**A. 法院应当依职权决定对该9张收货确认单上签名的真实性进行鉴定**

【分析】鉴定的启动原则上由当事人申请，除非待鉴定事实涉及国家利益、社会公共利益、第三人合法权益、身份关系、程序性事实。本案中9张收货单签名的真实性**不涉及**上述情形，法院**不能**依职权决定鉴定。

**B. 应当由王某申请对该9张收货确认单上签名的真实性进行鉴定** ✅

【分析】谁申请鉴定 = 谁承担证明责任。王某希望通过收货确认单证明已交付货物 → 王某对私文书证的真实性承担证明责任 → **由王某申请鉴定**。

**C. 王某应当对该9张收货确认单上的签名为真实承担证明责任** ✅

**D. 李某应当对签名为虚假承担证明责任** ❌


## 段落五：鉴定前的准备——签署承诺书与鉴定材料质证

**（一）签署承诺书**

鉴定开始之前，法院应当要求鉴定人签署承诺书。

**（二）对鉴定材料的质证**

法院应当组织当事人对鉴定材料进行质证，**未经质证的材料不得作为鉴定根据**。

**【举例说明】**
左林【此为虚构举例，为便于理解】起诉妻子离婚，关于孩子抚养权发生争议，左林说“孩子不是我的”，妻子说“孩子就是你的”。需要进行亲子鉴定。

- 这个鉴定**可以**由当事人申请进行。
- 法院**也可以**依职权决定鉴定，因为涉及**身份关系**，属于法院应当依职权调查的事实。
- 鉴定材料（如头发等检材）**必须经过质证**。如果送检的头发不能确保是左林和孩子的，鉴定意见就没有意义。鉴定意见通常有一句话：“本鉴定意见只对送检检材负责。”所以法院必须组织当事人对鉴定材料进行质证，确保检材的真实性。


## 段落六：鉴定人的确定

**（一）当事人申请鉴定的**

当事人对鉴定很关心，对鉴定人的选定也很关心 → 法院组织**当事人协商**确定鉴定人，协商不成，**法院指定**。

**（二）法院依职权决定鉴定的**

法院依职权决定鉴定的都是涉及国家利益、社会公共利益、第三人合法权益、身份关系、程序性事实的事项，当事人往往不太关心鉴定人的选定 → 由**法院指定**鉴定人（指定前要询问当事人意见）。


## 段落七：鉴定意见的形式

**（一）按时完成鉴定**

**（二）鉴定书的签名与盖章**

- 鉴定书应当由**鉴定人签名或者盖章**，并附有鉴定资格证明。
- 鉴定意见由**鉴定人个人负责**，所以鉴定人必须个人签名。
- 委托机构鉴定的，鉴定书应当由**鉴定机构盖章**，并有从事鉴定的人员签名。

**（三）多名鉴定人意见不一致的处理**

如果多名鉴定人对同一问题进行鉴定，出现不同意见，**不能搞少数服从多数**，应当**如实注明**——每个意见都写清楚，包括理由。到底采信哪个意见，交给法庭根据本案其他证据综合审查判断。


## 段落八：鉴定人出庭——重点

**（一）鉴定人出庭的情形（民诉法规定）**

出现以下两种情形之一，鉴定人应当出庭：
1. 当事人对鉴定意见有异议；
2. 法院认为鉴定人应当出庭。

**（二）实践中出现的问题与2019年证据规定的解决**

【修正说明：原文中“德，啊，你们委托我鉴定的时候……”等口语化表达予以简化，保留核心意思。】

实践中出现问题：
1. 当事人提出的异议往往是“有错别字”“有歧义”等非实质性异议，导致鉴定人频繁出庭，浪费司法资源。
2. 鉴定人出庭的费用没有明确规定补助，鉴定人不愿出庭；不出庭又要退费，导致鉴定人提前多收费用，造成鉴定费普遍涨价。

**2019年《最高人民法院关于民事诉讼证据的若干规定》的解决方案：**

增设**书面答复程序**：
1. 鉴定人完成鉴定后，将鉴定书提交法院，法院将副本送达双方当事人。
2. 当事人收到后**没有异议** → 程序结束。
3. 当事人**有异议** → 向法院提交**书面异议**。
4. 法院将书面异议交鉴定人，鉴定人进行**书面答复**。
5. 书面答复后当事人**没有异议** → 程序结束。
6. 书面答复后当事人**仍有异议** → **通知鉴定人出庭**。

【修正说明：原文中“一来，只要当事人坚持提出异议……”等重复表述予以精简。】

这个程序既保障了当事人提出异议让鉴定人出庭的权利（民诉法的规定），又通过书面答复解决了绝大多数非实质性异议，节约了司法资源。

**（三）鉴定人出庭的费用**

鉴定人出庭的费用跟证人出庭的费用一样，**由败诉方当事人承担**。但因鉴定意见不明确或有瑕疵导致需要出庭的，由鉴定人自行承担。


## 段落九：鉴定人拒不出庭的后果

经通知，鉴定人拒不出庭的：
1. **鉴定意见不得作为定案根据**；
2. 支付费用的当事人有权要求**返还鉴定费用**；
3. 法院可以建议行业主管部门或行业组织**对拒不出庭的鉴定人进行处罚**。


## 段落十：拓展——当事人自行委托鉴定的效力

【注释：此部分内容虽涉及鉴定主题，但讲的是“非官方鉴定”的效力问题，属于重要考点拓展。】

**问题**：当事人在起诉前或诉讼中，自行委托专业机构/人士对专业问题进行的所谓“鉴定”，出具的意见/报告，是否属于民诉法中的鉴定意见？

**回答**：**不属于**。原因：
1. 没有启动法定的鉴定程序（没有申请法院决定，也没有法院依职权决定）；
2. 鉴定人不是由法院组织协商或指定确定的；
3. 鉴定材料没有经过质证。

**但是**：这种材料**可以作为证据使用**，适用**书证（私文书证）** 的规则。如果对方当事人有证据或理由予以反驳并申请鉴定的，法院**应当准许**。


## 段落十一：有专门知识的人（专家辅助人）

**（一）什么是有专门知识的人**

【举例】汽车质量问题可以委托鉴定人鉴定，但路边修了20年车的老师傅，一听发动机声音就知道问题所在——他没有鉴定资格，但有专业知识，可以请来帮助认定事实，这叫**有专门知识的人**（也称**专家辅助人**）。

【⚠️ 重要提示】**不能表述为“专家证人”** ，我国不承认专家证人。

**（二）出庭程序**

当事人**在举证期限内申请**，由法院通知**一到两名**有专门知识的人出庭。**法院不能依职权通知**。

**（三）作用**

有专门知识的人出庭的作用有两个：
1. **对鉴定意见提出意见** → 这是在帮当事人**质证**，视为当事人的质证意见；
2. **对专业问题提出意见** → 这是在帮当事人**陈述**，视为当事人的陈述意见。

**（四）三个核心考点**

1. **出庭方式**：只能由当事人申请，不能依职权 → 因为只涉及当事人的质证权、陈述权。
2. **费用承担**：**由申请方自己承担**（不是败诉方）→ 因为他是帮一方打另一方，不中立。
3. **是否适用回避**：**不适用回避制度** → 因为不期待他中立。

【注释】作为对比：诉讼费、证人出庭费、鉴定人出庭费由败诉方承担；鉴定人适用回避制度；证人不适用回避制度（因具有不可替代性）。


## 段落十二：例题——有专门知识的人的身份认定

**【例题】**
甲公司因工程质量问题起诉乙公司，向法院提交了甲公司在起诉前委托某鉴定中心的鉴定人刘某对工程质量问题进行鉴定所出具的鉴定意见书。

**问题一：这份鉴定意见书属于什么证据？**
- 没有申请法院决定鉴定，没有法院依职权决定鉴定；
- 鉴定人不是法院组织协商或指定确定的；
- 鉴定材料没有经过质证。
→ **不属于鉴定意见**，但**可以作为证据使用**，属于**书证**。

**问题二：刘某的身份是什么？**
- 本案没有启动鉴定程序，刘某不是本案的鉴定人。
- 他是利用自己的专业知识帮助认定事实 → **有专门知识的人**。

**问题三：乙公司申请法院通知某大学张教授出庭发表意见，张教授的身份是什么？**
- 也是**有专门知识的人**。
- 刘某和张教授可以当庭**对质**（双方请的有专门知识的人可以就专业问题进行对质）。

**问题四：刘某发表的意见属于什么？**
- 不属于鉴定意见。
- 本案没有鉴定，刘某只是对专业问题发表意见 → 视为**当事人的陈述**。

**问题五：法院能否直接认定鉴定意见书的证明力？**
- 不能。证明力的有无和大小，应由法官根据法律规定、遵循职业道德、运用生活经验、逻辑推理，具体问题具体分析。


## 段落十三：鉴定人、有专门知识的人、司法技术人员——三者的区分

在诉讼中涉及专业知识时，有专业知识的人可以以不同身份参与诉讼：

| 身份 | 谁来请/指定 | 帮谁 | 是否中立 | 是否适用回避 |
|------|------------|------|---------|-------------|
| **鉴定人** | 法院启动程序后委托 | 法庭 | **中立** | **适用** |
| **有专门知识的人** | 当事人申请 | 当事人（质证/陈述） | **不中立** | **不适用** |
| **司法技术人员** | 法院指派或聘请 | 法官（审查专业证据） | **中立** | **适用** |

**司法技术人员**：法院指派或聘请来帮助法官认定事实、审查判断专业证据的人。由法院指派或聘请，需要中立，适用回避制度（属于“其他人员”，由审判长决定回避）。


## 段落十四：例题——技术调查官的身份与回避

**【真题实例】**
甲乙公司就专利侵权事项发生纠纷诉至法院，法院指派该领域专家王某担任该案**技术调查官**，参与诉讼活动。甲公司以王某是乙公司大股东为由申请王某回避。下列说法正确的是？

**【分析】**
- 技术调查官属于**司法辅助人员**（即民事诉讼中的**司法技术人员**）。
- 司法技术人员适用回避制度（属于“其他人员”，由审判长决定回避）。

**【答案】** 王某是司法技术人员，应当适用回避制度（由审判长决定）。

【注释】本题可直接依据《最高人民法院关于技术调查官参与知识产权案件诉讼活动若干规定》作答；也可从原理推导：帮法官的、中立的 → 司法技术人员 → 适用回避。


## 段落十五：当事人陈述

**（一）当事人陈述是一种证据**

**（二）出庭义务**

法院认为必要的，可以要求当事人本人到庭，就案件有关事实接受询问。法院应当在询问前责令当事人**签署并宣读保证书**。

**（三）当事人不出庭的后果**

负有证明责任的当事人拒绝到庭、拒绝宣读保证书或拒绝签署保证书，待证事实又欠缺其他证据证明的，法院对其主张的事实**不予认定**。

**（四）当事人出庭的两方面作用**

1. **行使诉讼权利** → 可以委托代理人代为行使。代理人（在法律规定范围内）的诉讼行为与当事人本人的诉讼行为效力相同。
   - 原告不来也不委托代理人 → 视为放弃诉讼权利 → **按撤诉处理**。
   - 被告不来也不委托代理人 → 视为放弃陈述与申辩的权利 → **缺席判决**。

2. **提供当事人陈述这一证据**（也可理解为行使提供证据的权利） → 当事人不来 → 视为没有这个证据 → 法院根据**现有证据 + 证明责任规则**认定事实并作出判决。


## 段落十六：例题——当事人出庭问题（2023年真题）

**【真题实例】（2023年真题）**
甲起诉乙归还借款10万元，甲委托张律师代理诉讼。法院经审理后认为现有证据无法证明债权成立，需要甲到庭陈述事实，遂通知甲到庭接受询问。甲以工作繁忙为由拒绝到庭，法院决定延期审理。第二次开庭时甲仍未出庭，委托张律师出庭。关于本案表述正确的是？

**A. 可以对甲拘传** ❌
**B. 可以对甲缺席判决** ❌
**C. 按甲撤诉处理** ❌
**D. 如果现有证据无法证明债权成立，法院可以对甲主张的债权不予认定** ✅

**【解析】**
- 甲委托了代理人出庭 → 诉讼权利已由代理人行使，不能按撤诉处理，也不能缺席判决，更不能拘传。
- 甲本人不来 → 只是不提供“当事人陈述”这一证据 → 法院根据现有证据和证明责任认定事实即可。

**【⚠️ 重要提示：拘传的适用条件】**

民事诉讼中**拘传适用极少**。不能因为“必须到庭”就轻易适用拘传。

拘传只适用于**超越了双方当事人私权利之上、值得法院通过剥夺人身自由来保障的价值**的情形，例如：
- 负有**赡养、抚养、扶养义务**的被告拒不到庭（保障老人/被抚养人基本生存需要 + 法制教育）；
- 当事人**恶意串通虚假诉讼**，不到庭规避处罚（查清违法事实 + 处罚）；
- **执行环节**中，被执行人拒不履行生效判决、妨碍执行。

**【注释】** 讲稿中老师特别强调：实践中很多民事法官职业生涯从未签发过拘传票。考试中看到“拘传”选项要非常慎重。


## 段落十七：勘验笔录

勘验笔录是一种证据。**勘验的主体是司法工作人员**（法院工作人员），由法院工作人员进行现场勘验后形成的笔录叫勘验笔录。


## 段落十八：证据章节总结

证据这一章讲了两个大问题：

**一、证据的理论分类**

| 分类标准 | 类别 |
|---------|------|
| 证据来源 | **原始证据**（直接来源于案件事实，未经复制转述） vs **传来证据**（经过复制转述） |
| 证据内容 | **直接证据**（内容上能完整证明一件事） vs **间接证据**（内容上只能证明一个部分/片段）——只看内容的完整性，与来源、证明力大小无关 |
| 证据与证明责任的关系 | **本证**（承担证明责任的当事人提供） vs **反证**（不承担证明责任的当事人提供） |

**二、证据的法定种类（八种）**

1. **物证、书证、视听资料、电子数据**
   - 书证、视听资料、电子数据本质相同（都用内容证明案件事实），书证的证据规则同样适用于视听资料和电子数据。
   - 书证的核心规则：
     - **最佳证据规则**：书证应当提供原件，提供原件困难的可以提供复印件。
     - **公文书证 vs 私文书证**：公文书证（公权力机关行使法定职权制作）记载内容直接推定为真实，对方有相反证据足以推翻的除外；私文书证的真实性由主张以该文书证证明案件事实的一方承担证明责任。私文书证的形式真实性需要证明；内容真实性推定（有签名盖章捺印即推定是制作者真实意思表示），反对者承担证明责任。
     - **文书提出命令**：有证据证明对方持有某书证，可申请法院责令对方提交。对方不提交 → 推定申请人主张的书证内容为真实；对方毁损书证 → 推定申请人主张的以该书证能够证明的案件事实为真实，并可罚款拘留乃至追究刑事责任。
     - **域外书证**：域外公文书证需经所在国公证或履行条约约定手续；域外身份关系证据还需经我国驻该国使领馆认证。

2. **证人证言**
   - **证人资格**：两个条件——①知晓案件事实；②能够正确表达。无/限制民事行为能力人、与案件有利害关系的人，只要满足这两个条件，都可以作为证人（证人不适用回避制度）。
   - **证人出庭方式**：原则上由当事人申请；涉及法院应依职权调查的事实的，法院可依职权通知。
   - **证人必须出庭**：无正当理由未出庭的证人提供的书面证言**不得作为定案根据**（无证据能力）。
   - **不出庭的例外**：①视为出庭——已在证据交换/调查询问中陈述证言；②法定原因（健康原因、路途遥远、不可抗力等）经法院许可；③一定原因（双方当事人同意）经法院准许。

3. **鉴定意见**（详见前述段落一至十）

4. **有专门知识的人/专家辅助人**（详见段落十一至十三）

5. **当事人陈述**（详见段落十五至十六）

6. **勘验笔录**（详见段落十七）
