# 第1部分：法考备考提纲

> 【说明】本提纲基于讲稿内容整理，对照《民事诉讼法》相关条款。部分内容（如"审判委员会"）属于超出讲稿主题但讲授者提及的概念，以注释形式标注。


## 一、民事诉讼基本制度概述

民事诉讼法总则编中的四个基本制度：
1. **合议制**（审判组织）
2. **回避制度**
3. **公开审判制度**
4. **两审终审制度**


## 二、审判组织（合议制与独任制）

### （一）审判组织的三种形式

| 审判组织 | 组成方式 | 说明 |
|---------|---------|------|
| **独任庭** | 审判员1人 | 独任制 |
| **合议庭** | 审判员3人，或审判员+人民陪审员共3人 | 合议制 |
| **审判委员会** | 院长、副院长、资深审判员 | 最高业务决策机构，讨论重大疑难案件 |

**【注释】** 审判委员会不是案件的审判组织，而是法院内部的业务决策机构。


### （二）2021年《民事诉讼法》修改——扩大独任制适用

**修法背景**：案多人少→提高诉讼效率→扩大独任制适用。

**修法的两个重要考量**：
1. **司法民主**——防止压缩人民陪审员参与机会
2. **判决公信力**——防止公信力降低导致上诉/再审激增


### （三）一审的审判组织（《民事诉讼法》第40条）

| 程序 | 审判组织 | 可独任制的条件 |
|------|---------|---------------|
| **简易程序** | **必须独任制** | — |
| **普通程序** | 原则上**合议制** | ①**基层法院**审理；②**基本事实清楚、权利义务关系明确** |

**【法条依据】** 《民事诉讼法》第40条："适用简易程序审理的民事案件，由审判员一人独任审理。基层人民法院审理的基本事实清楚、权利义务关系明确的第一审民事案件，可以由审判员一人适用普通程序独任审理。"


### （四）二审的审判组织（《民事诉讼法》第41条）

| 程序 | 审判组织 | 可独任制的条件（四个同时满足） |
|------|---------|------------------------------|
| **二审** | 原则上**合议制** | ①**中院**的二审；②一审**简易程序**或**不服裁定**上诉；③**事实清楚、权利义务关系明确**；④**双方当事人同意** |

**【法条依据】** 《民事诉讼法》第41条第2款："中级人民法院对第一审适用简易程序审结或者不服裁定提起上诉的第二审民事案件，事实清楚、权利义务关系明确的，经双方当事人同意，可以由审判员一人独任审理。"


### （五）不得适用独任制的案件（《民事诉讼法》第42条）

1. 涉及**国家利益、社会公共利益**的案件；
2. 涉及**群体性纠纷**，可能影响社会稳定的案件；
3. **人民群众广泛关注**或者其他社会影响较大的案件；
4. 属于**新类型**或者**疑难复杂**的案件；
5. **法律规定应当组成合议庭**审理的案件；
6. **其他**不宜独任审理的案件。

**【记忆】** 要么**影响大**，要么**疑难复杂**。


### （六）独任制转合议制（《民事诉讼法》第43条）

- 法院发现不宜独任的→**裁定**转为合议庭
- 当事人提出异议→法院审查→异议成立则**裁定**转为合议庭


### （七）其他程序的审判组织

| 程序 | 审判组织 |
|------|---------|
| **再审** | **必须合议制**（按一审或二审程序另行组成合议庭） |
| **特别程序** | 原则上**独任制**；选民资格/重大疑难案件→**合议制**（仅审判员） |
| **督促程序** | **独任制** |
| **公示催告程序** | 公示催告阶段**独任制**；除权判决阶段**合议制**（仅审判员） |


### （八）人民陪审员参加合议庭的范围

**核心结论**：人民陪审员**只参加适用一审程序审理的诉讼程序**。

- ✓ 一审诉讼程序→**可以有**人民陪审员
- ✗ 二审→**不能有**（不适用一审程序）
- ✗ 特别程序→**不能有**（非诉讼程序）
- 再审→看情况：适用一审的再审**可以有**；适用二审的再审**不能有**


## 三、回避制度

### （一）回避制度的含义

所有与案件有利害关系的人必须退出本案的审理工作，以保证审判公正。

### （二）适用对象

| 适用回避 | 不适用回避 |
|---------|-----------|
| 审判人员（审判员、人民陪审员） | 诉讼代理人（不期待中立） |
| 法官助理 | **证人**（虽期待中立，但具有不可替代性） |
| 书记员 | |
| 司法技术人员 | |
| 翻译人员、鉴定人、勘验人 | |

### （三）回避的法定原因

1. **与案件有利害关系**（近亲属、利害关系、曾担任本案证人/代理人等、其他可能影响公正的关系）
2. **存在不正当行为**（接受请客送礼、违规会见等）
3. **参与过前序程序的审判**（但发回重审后原二审法官除外）

### （四）回避的方式

1. **自行回避**（法官自觉）——实践中最常见
2. **申请回避**（当事人申请）
3. **指令回避**（院长决定）

### （五）申请回避的程序要点

| 事项 | 规则 |
|------|------|
| **申请形式** | 口头/书面均可 |
| **申请理由** | 必须说明理由（**无因回避**不被承认） |
| **申请时间** | 开始审理时提出；事由后知道的，**法庭辩论终结前**提出 |
| **申请效力** | 申请后、决定作出前→被申请回避人**暂停**工作（紧急措施除外） |
| **决定权** | 判决书**署名**的→**院长**决定；**不署名**的→**审判长**决定；院长的回避→**审委会**决定 |
| **救济** | 驳回回避申请→可**申请复议一次**（复议期间不停止工作）；被决定回避的**不能**申请复议 |
| **回避效力** | 审判人员更换后，诉讼程序**继续进行**，原程序**继续有效** |


## 四、公开审判制度

### （一）原则与例外

**原则**：公开审理

**例外**：

| 类型 | 情形 |
|------|------|
| **法定不公开** | 国家秘密、个人隐私、法律规定的其他案件 |
| **申请不公开** | 离婚诉讼、涉及商业秘密的案件 |

### （二）评议与宣判

- 合议庭评议→**一律保密**
- 宣判→**一律公开**（不论是否公开审理）


## 五、两审终审制度

### （一）基本含义

一个案件经**两级法院**审理后即告终结。二审判决为生效判决、终审判决。

### （二）例外情形

1. 适用**特别程序、督促程序、公示催告程序**审理的案件（一审终审）；
2. **最高人民法院**审理的一审案件（一审终审）。


## 六、四个基本制度的逻辑关系

| 制度 | 作用 |
|------|------|
| **审判组织**（合议制/独任制） | 最基本的制度——谁来审 |
| **回避制度** | 内部制约——保证中立 |
| **两审终审制度** | 外部制约（上级法院）——上诉监督 |
| **公开审判制度** | 外部制约（公众舆论）——社会监督 |

# 第2部分：讲稿整理（按逻辑顺序分段）


## 一、基本制度概述

我们看到第二节民事诉讼的基本制度。在基本制度这一节呢，我们讲四个基本的制度：第一个叫**合议制**，讲的是法院的审判组织；第二个叫**回避制度**；第三个叫**公开审判制度**；第四个叫**两审终审制度**。


## 二、合议制——法院的审判组织

### （一）审判组织的三种形式

来看到第一个问题，合议制讲法院的审判组织。法院呢是国家的审判机关，在法院里边有几种审判组织啊，总共是三种：

**第一种叫独任庭**，有审判员一人独任审理，对应的概念叫独任制。

**第二种叫合议庭**，有审判员三人（就是大家说的法官）或者法官与人民陪审员三人组成合议庭，对案件进行审理，这个叫做合议制。

**第三种叫审判委员会**。在法院里边，法院的院长当然是审委会的一把手，然后还有副院长也是审委会的成员，那还有一些资深的审判员，我们一起组成一个组织叫审判委员会，作为法院的最高业务决策机构，负责对法院里边的重大疑难案件进行讨论，作出决定。

**【注释】** 根据《民事诉讼法》，审判委员会的主要职能包括：总结审判工作经验；讨论决定重大、疑难、复杂案件的法律适用；讨论决定本院已经发生法律效力的判决、裁定、调解书是否应当再审等。


### （二）独任制与合议制的适用——2021年修法重点

#### 1. 修法前的简单规定

那之前民诉法对这个审判组织的规定很简单。我们来先讲三个重要的程序：一审（分为简易程序和普通程序）、二审。

- **简易程序**：案件简单，由审判员一人独任审理。
- **普通程序**：案件不简单，搞合议制，由审判员三人或者审判员与人民陪审员三人组成合议庭。
- **二审**：案件肯定不简单，搞合议制，由审判员三人组成合议庭审理。

#### 2. 2021年修法的背景——扩大独任制适用

但是遗憾的是，2021年民诉法修改的时候，把这个问题改得相对还是比较复杂的。那2021年民诉法修改一个重大的修改方向是什么呢？**扩大独任制的适用**。

为什么呢？大家如果身边有做法官的朋友，你应该经常听到他们抱怨一个词叫 **"案多人少"** ——案子太多，人手不够。一个基层法院的民事法官每年要审结五百多起民事案件，一年工作时间才二百多天，意味着一天至少要审结两个案件。要完成文书的送达、开庭、写判决书、送达判决书，压力特别大。

那为了解决案多人少的问题：减少案件数量？做不到，要保障当事人的起诉权。增加人手？增加编制意味着国家要多出钱。所以只有一条路——**提高诉讼效率**。怎么提高？**扩大独任制的适用**——以前三个人才能搞定的案件，现在一个人就搞定了。

#### 3. 扩大独任制适用的两个重要考量

**第一，司法民主。** 法院的判决不能只体现职业法官的专业法律思维，还要体现老百姓的生活经验和常识。怎么体现？通过人民陪审员制度——让人民陪审员和职业法官一起组成合议庭审理案件。但人民陪审员参与审判的组织载体是**合议制**，独任制只有一名法官，没有人民陪审员参加的机会。所以盲目扩大独任制适用，会压缩人民陪审员参与审判的机会，破坏司法民主。

**第二，司法判决的公信力。** 同样一个案件，三个人合议审理后作出的判决，公信力强；一个人独任审理后作出的判决，公信力弱一些。如果一审搞合议制，公信力强，当事人服判不上诉；如果搞成独任制，公信力降低，当事人上诉的可能性增加——一审效率提高了，压力给了二审。同理，二审搞独任制，当事人申请再审的可能性增加，会破坏两审终审制度。

在这两个考量之下，我们来看立法者做出的决定。

#### 4. 一审的审判组织（2021年修法后）

**（1）简易程序**——本来就简单，**一定独任制**。

**（2）普通程序**——**原则上合议制**，由审判员组成合议庭，或者审判员与人民陪审员共同组成合议庭。但**符合条件可以独任制**。

普通程序搞独任制需要满足**两个条件**：
- ① **基层法院**审理的；
- ② **基本事实清楚、权利义务关系明确**。

> 【修改说明】原文中"权利关系明确"应为"**权利义务关系明确**"，已统一修正。

为什么普通程序搞独任制要求"基本事实清楚、权利义务关系明确"？因为人民陪审员参与审判最擅长的是事实认定，这个案件基本事实清楚，就没有太多事实认定的必要，人民陪审员来了也没有用武之地。搞独任制不会压缩人民陪审员参与审判的机会。

#### 5. 二审的审判组织（2021年修法后）

二审——**原则上合议制**，由审判员组成合议庭审理。但**符合条件可以独任制**。

二审搞独任制需要同时满足**四个条件**：
- ① **中级人民法院**的二审；
- ② 一审是**简易程序**审结，或者**不服一审裁定**上诉（如不予受理、驳回起诉、管辖权异议等程序性裁定）；
- ③ **事实清楚、权利义务关系明确**；
- ④ **经双方当事人同意**。

**为什么二审独任制要"经双方当事人同意"？** 我国实行两审终审制度，二审判决是终审判决、生效判决。特别要注意二审判决的公信力问题。如果当事人没有同意就搞独任制，判决公信力降低，当事人不服二审判决的可能性增加，就会有大量当事人申请再审——给再审带来压力，同时破坏两审终审制度，导致"终审不终"。所以二审搞独任制必须经过当事人同意。

#### 6. 一审二审审判组织总结

| 程序 | 原则 | 例外（可独任制的条件） |
|------|------|----------------------|
| 一审·简易程序 | **必须独任制** | — |
| 一审·普通程序 | 原则上合议制 | ①基层法院；②基本事实清楚、权利义务关系明确 |
| 二审 | 原则上合议制 | ①中院；②一审简易程序或不服裁定上诉；③事实清楚、权利义务关系明确；④双方当事人同意 |


### （三）例题演练

**【命题角度一】**

**例1：** 某中院受理的甲公司诉乙公司合同纠纷案，法院认为本案基本事实清楚、权利义务关系明确，由审判员张某独任审判。——**判断：错误。**

> 【解析】首先判断是一审。一审有可能是简易程序或普通程序。简易程序只适用于基层法院及派出法庭，本案是中院，所以一定是普通程序。普通程序搞独任制要求基层法院审理，中院不符合条件，不能独任制。

**【总结】** 中院审理的一审民事案件**必须搞合议制，不可能独任制**。

**例2：** 某区法院受理张某与李某合同纠纷一案，依法适用普通程序审理后，认为本案原告不是适格当事人，裁定驳回起诉。张某不服该裁定，提起上诉。中院认为本案事实清楚、权利义务关系明确，经张某李某同意后，由审判员王某独任审理。——**判断：正确。**

> 【解析】二审。二审独任制四个条件：①中院✓；②不服一审裁定上诉✓；③事实清楚、权利义务关系明确✓；④双方当事人同意✓。全部满足，可以独任制。

**例3：** 某区法院受理张某诉李某侵权纠纷案，依法由审判员一人适用普通程序审理后，判决驳回张某诉讼请求。张某不服一审判决，提起上诉。中院认为本案事实清楚、权利义务关系明确，经张某李某同意后，由审判员一人独任审理。——**判断：错误。**

> 【解析】二审。四个条件中，第二个条件"一审是简易程序或者不服一审裁定上诉"不满足——一审是普通程序，且是对判决上诉，不是对裁定上诉。所以不能独任制。

**例4：** 某中院受理甲公司诉乙公司合同纠纷案，乙公司提出管辖权异议，该中院驳回乙公司的异议，乙公司不服该裁定，提起上诉。高院认为本案事实清楚、权利义务关系明确，经甲乙公司同意后，由审判员张某独任审理。——**判断：错误。**

> 【解析】二审。二审独任制要求"中院的二审"，本案是高院的二审，不符合条件，不能独任制。

**【进一步总结】** 凡是由中院一审的民事案件，这个案件**永远不可能独任制**——一审不能独任，二审（到高院）也不能独任。因为中院一审的案件肯定是重大复杂案件。


### （四）其他程序的审判组织

#### 1. 再审程序
再审没有独立的审理程序：
- 原来是一审的，按照**一审程序另行组成合议庭**——可由审判员组成，也可审判员与人民陪审员共同组成。
- 原来是两审终审，或者现在上级法院提审的，按照**二审程序组成合议庭**——只能由审判员组成，人民陪审员不能参加。
- **再审必须搞合议制，不准独任制。**

#### 2. 特别程序
- **原则上**由审判员一人**独任审理**。
- **选民资格案件**或者**重大疑难案件**，由审判员组成**合议庭**（人民陪审员不能参加）。

#### 3. 督促程序（支付令程序）
由审判员一人**独任审查**。

#### 4. 公示催告程序
分两个阶段：
- **公示催告阶段**：**独任制**。
- **除权判决阶段**：**合议制**，由审判员组成合议庭。


### （五）人民陪审员参加什么合议庭？

**核心结论：人民陪审员只参加适用一审程序审理的诉讼程序。**

> 【修改说明】原文多次重复"人民陪审员只参加适用一审程序审理的诉讼程序"，已整合精简，内容未遗漏。

**判断标准**：
- **审级**：适用一审程序审理 ✓
- **案件性质**：诉讼程序 ✓

**例题判断**：
- 一审中有没有可能有人民陪审员？**有可能**（适用一审程序审理的诉讼程序）。
- 二审中有没有可能有人民陪审员？**不可能**（不是适用一审程序审理）。
- 特别程序中有没有可能人民陪审员？**不可能**（不是诉讼程序）。
- 中院审案件可不可以有人民陪审员？中院是一审法院则**有可能**，中院是二审法院则**不可能**。
- 再审程序中有没有可能有人民陪审员？适用一审的再审**有可能**，适用二审的再审**不可能**。


### （六）不得适用独任制的六类案件

下列案件**不允许适用独任制**：

1. 涉及**国家利益、社会公共利益**的案件；
2. 涉及**群体性纠纷**，可能影响社会稳定的案件；
3. **人民群众广泛关注**或者其他社会影响较大的案件；
4. 属于**新类型**或者**疑难复杂**的案件；
5. **法律规定应当组成合议庭**审理的案件；
6. **其他**不宜由审判员一人独任审理的案件。

**【记忆方法】** 要么**影响大**（1、2、3），要么**疑难复杂**（4），让你独任你也不敢。

> 【修改说明】原文中"不中到独任制"修正为"**不允许适用独任制**"。


### （七）独任制转合议制

法院在独任制审理过程中，发现案件不宜搞独任制的，应当**裁定转为合议庭**审理。

当事人认为案件由审判员一人独任审理违反法律规定的，可以向人民法院提出**异议**。法院审查认为异议成立的，**裁定转为合议庭**审理；异议不成立的，**裁定驳回**。

**注意：用裁定书，不是决定书。**


### （八）真题实例（2022年题目）

**题目：** 甲公司为合同纠纷起诉乙公司，某县人民法院适用简易程序审理后，判决甲公司败诉。甲公司不服一审判决，提起上诉。二审法院指定一名法官对本案独任审理。乙公司当庭提出异议。关于本案说法正确的是？

**判断思路：** 二审搞独任制需要四个条件：
- ①中院？一审是县法院，二审是中院 ✓
- ②一审是简易程序 ✓
- ③事实清楚、权利义务关系明确？**题目没说** ✗
- ④双方当事人同意？乙公司当庭提出异议 = **不同意** ✗

所以本案**不能搞独任制**，应当**裁定转为合议庭**。

**选项分析：**
- A. 二审程序只能组成合议庭审理，故本案二审不能独任制。——**错误**（二审符合条件可以独任制）
- B. 简易程序一审终审不能上诉。——**错误**（简易程序可以上诉）
- C. 二审法院有权直接指定法官适用独任制。——**错误**（二审独任制需经当事人同意）
- D. 二审应当裁定转合议庭审理。——**正确** ✓


## 三、回避制度

### （一）回避制度的含义

什么叫回避制度？审判要公正，所有与这个案件有利害关系的人必须退出本案的审理工作，这个制度就叫回避制度。

### （二）回避制度的适用对象

适用回避的人员包括：**审判人员**（包括审判员和人民陪审员）、**法官助理**、**书记员**、**司法技术人员**、**翻译人员**、**鉴定人**、**勘验人**。

**【记忆方法】** 看我们是否期待这个人保持中立。期待他中立→有利害关系就无法中立→要回避。

- **要回避**：审判人员、书记员、翻译人员、鉴定人、勘验人员等（期待他们中立）。
- **不回避**：诉讼代理人（不期待中立，本来就是帮当事人说话的）。
- **例外——证人不适用回避**：虽然期待证人如实作证、保持中立，但证人不适用回避制度。因为上述人员都是"凭本事吃饭"——有这本事的人不止一个，回避了别人能顶上。但证人是"凭缘分吃饭"——具有**不可替代性**，让证人回避了就没证人了。

### （三）回避的法定原因（三大类）

**第一类：与案件有利害关系**

包括但不限于：
- 是本案当事人或者当事人、诉讼代理人近亲属的；
- 与本案有利害关系的；
- 担任过本案的**证人、鉴定人、辩护人、诉讼代理人、翻译人员**的（注意：不是说证人要回避，而是说**曾经担任过本案证人的人，现在不能担任本案的审判人员、书记员等**——是审判人员等要回避）；
- 与本案当事人或诉讼代理人有**其他利害关系**，可能影响公正审理的（如同学关系、朋友关系、师生关系、远房亲戚、战友关系、债权债务关系等）。

> 【修改说明】原文中"与本案的当事人或诉讼代理人有其他利害关系，可能影响公正审理的"，修正为"**其他利害关系**"。

**关于"其他利害关系"的考试技巧：** 看前后有无修饰性词语提示可能影响公正审理。例如"多年出生入死的战友关系"——"出生入死"就是在提示可能影响公正审理，要回避。

**第二类：存在不正当的行为**

如接受当事人、诉讼代理人请客送礼，违反规定会见当事人、诉讼代理人等。

**注意：** 法官可以会见当事人、诉讼代理人，但必须是**合规会见**——工作时间、工作场合的会见。非工作时间、非工作场合一律是**违规会见**（如早上跑步时在小区花园里交流案情）。

**第三类：参与过前序程序的审判**

在一个审判程序中参与过本案审判工作的审判人员，**不得再参与该案其他程序的审判**。

**举例：** 张法官在白云区法院参与过本案一审，后调任广州中院，不能担任本案的二审法官——因为参与过一审，已产生先入为主的偏见和预断。

**但有一个例外：** 发回重审的案件，在一审法院作出裁判后又进入二审程序的，**原二审程序中的审判人员不受上述规定限制**。

**举例：** 李法官作为中院法官，处理二审时撤销原判、发回重审。重审后当事人再上诉到中院，李法官可以继续担任二审法官——因为上次二审他没有对案件进行实质性审理（直接发回重审了），没有产生偏见和预断。

### （四）回避的方式

三种方式：
1. **自行回避**：法官自觉有利害关系，主动提出回避。
2. **申请回避**：法官不自觉，当事人发现后申请回避。
3. **指令回避**：法官不自觉、当事人没发现，院长发现后直接决定回避。

**实践中最常见的是自行回避。** 回避制度既是对法官的约束，也是对法官的保护——避免法官陷入"坚守公平正义 vs 向亲情妥协"的两难境地。

### （五）申请回避的程序

**1. 申请形式**：口头、书面都可以。

**2. 申请理由**：必须说明理由——我国**不承认无因回避**，一定是有因回避。

**3. 申请时间**：
- 在**案件开始审理时**提出；
- 回避事由在开始审理后知道的，也可以在**法庭辩论终结前**提出。

**为什么最迟是"法庭辩论终结前"？** 法庭辩论终结后只剩两个工作：合议庭评议（秘密进行，你不知道谁参与了）和宣判（结果已定，申请回避没意义）。

**4. 申请的效力**：
- 申请后、回避决定作出前，被申请回避的人应当**暂停**本案的工作。
- **但**案件需要采取紧急措施的除外。

**5. 回避的决定权**：

**【核心口诀】** 判决书最后**署名**的 → **院长**决定；判决书最后**不署名**的 → **审判长**决定。

> 【修改说明】原文中关于决定权的表述有重复和口语化，已精简整理，内容未遗漏。

判决书最后署名的人员（审判长、审判员、人民陪审员、书记员）——**院长**决定。

判决书最后不署名的人员（翻译人员、鉴定人、勘验人）——**审判长**或**独任审判员**决定。

**院长**担任审判长或者独任审判员时的回避——**审判委员会**决定。

**注意：** 证人不适用回避，所以不存在"证人的回避谁决定"的问题。

**6. 救济**：
- 申请人对**驳回回避申请**的决定不服的，可以申请**复议一次**。
- **复议期间**，被申请回避的人员**不停止**办案工作。
- **被决定回避的人不能申请复议**（开心都来不及，复议干什么？）。

**【对比记忆】** 申请后、决定作出前→**暂停**工作；复议期间→**不停止**工作。

### （六）回避的效力——诉讼程序继续进行

在民事诉讼中，审判人员回避导致审判人员更换后，诉讼程序**继续进行**，原来已经进行的诉讼程序**继续有效**，**不需要重来**。


### （七）例题演练

**例1：** 某区法院审理原告徐某与被告某饭店食物中毒纠纷一案，审前法院书面告知徐某合议庭组成人员及人民陪审员丙，徐某没有提出回避申请。开庭后，徐某指认人民陪审员丙与被告的法定代表人是亲兄弟，遂提出回避申请。

**选项分析：**
- A. 徐某可以在知道丙与被告法定代表人是亲兄弟时提出回避申请。——**正确** ✓（在法庭辩论终结前知道即可申请）
- B. 法院对回避申请作出决定前，丙不停止参与本案的审理。——**错误**（申请后、决定作出前应当暂停）
- C. 应当由审判长决定丙是否回避。——**错误**（人民陪审员是审判人员，署名，由院长决定）
- D. 法院作出回避决定后，徐某可以对此上诉。——**错误**（申请人不服可申请复议，不能上诉）

**答案：A**

**例2：** 王某诉马某侵权纠纷损害赔偿一案，经开庭审理后，合议庭经评议，准备支持王某的诉讼请求。法院通知双方当事人8月1日进行宣判。马某发现本案的书记员张某是原告的表妹夫，于7月31日向法院提出书记员张某回避的申请。

**考点：** 申请回避最迟在**法庭辩论终结前**。本案法庭辩论已经终结，合议庭已经评议，**过了申请期限**，法院对申请**不予审查**。

- A. 以书记员不属于应当回避的人员为由驳回——**错误**（书记员属于应当回避的人员）
- B. 以马某的申请不符合法定回避情形为由驳回——**错误**（不审查，不是因为不符合情形）
- C. 因为马某超过了法定申请回避的期限，法院对马某的申请不予审查。——**正确** ✓
- D. 法院应当延期宣判，对马某提出的回避申请予以审查。——**错误**（不审查）

**答案：C**

**例3：** 甲公司诉赵某违约一案，由包法官独任审理。赵某提出管辖权异议被驳回。赵某怀疑包法官有偏见，申请包法官回避被驳回。甲公司被收购，法定代表人变更为薛某。赵某发现薛某与包法官是同学，再次提出回避申请。

**选项分析：**
- A. 赵某不能再次提出回避申请，只能申请复议。——**错误**（以新的事实理由可以再次申请）
- B. 包法官应当暂停本案的审理工作。——**正确** ✓（申请后、决定作出前应当暂停）
- C. 法院可以裁定驳回赵某的回避申请。——**错误**（回避用**决定**，不用裁定）
- D. 如果法院决定包法官回避，赵某可以再次提出管辖权异议。——**错误**（回避后程序继续进行，原裁定依然有效，不能再次提出）

**答案：B**


## 四、公开审判制度

### （一）公开审判的含义

公开审判制度是指人民法院审理民事案件，审判过程和结果应当向群众和社会公开，允许民众旁听，允许新闻媒体报道，以便接受社会监督。

### （二）不公开审理的例外

以**公开为原则**，不公开为例外。

**第一类：法定不公开**（法律规定绝对不能公开）
- 涉及**国家秘密**的案件；
- 涉及**个人隐私**的案件；
- **法律规定的其他案件**。

**第二类：申请不公开**（原则上应公开，当事人申请则不公开）
- **离婚诉讼**；
- **涉及商业秘密**的案件。

> 【修改说明】原文中"经申请不公开"修正为"**申请不公开**"。

**特别注意：** 开庭时当事人是**未成年人**的案件，在民事诉讼中**不属于不公开的情形**！不要和刑事诉讼法搞混。刑事诉讼中未成年人犯罪不公开审理是为了保护未成年人自尊心；民事诉讼中未成年人当原告或被告不是什么不光彩的事情。

### （三）评议与宣判

- **合议庭评议**：**一律保密**，不准公开。
- **宣判**：**一律公开进行**。涉及国家秘密、个人隐私的案件也要公开宣判——宣判的是判决结果，不会泄露国家秘密或个人隐私。


## 五、两审终审制度

两审终审制度是指一个案件经两级法院审理后，即告终结。当事人对一审判决不服可以上诉，二审法院所作判决为生效判决、终审判决。

**两审终审制度的例外**：
1. 适用**特别程序、督促程序、公示催告程序**审理的案件（非诉案件），实行一审终审；
2. **最高人民法院**所作的一审判决、裁定，为终审判决、裁定。


## 六、四个基本制度的逻辑关系

我们总共讲了四个制度：

**审判组织（合议制/独任制）** ——最基本的制度。哪些案件搞合议制，哪些搞独任制。

为了保证审判组织公正行使审判权，需要进行制约：

1. **内部制约——回避制度**：所有参与审判工作的人员必须保持中立，有利害关系的要回避。

2. **外部制约之一——两审终审制度**：来自上级法院的制约。当事人对一审判决不服可以上诉，二审法院对一审进行监督。

3. **外部制约之二——公开审判制度**：来自公众舆论的制约。审判应当公开进行，让群众旁听、媒体报道，接受监督。
